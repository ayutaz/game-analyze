# 体験型 (EXP) ゲーム — 「他者軸」分析

「遊ぶ過程そのもの」が面白さの中核となるゲーム群。とりわけ「他者（人間）との相互作用」が価値の中心にあるタイトルを重視して整理する。

---

## 1. 他者軸サブタイプの定義（再掲）

| サブタイプ | 説明 |
|---|---|
| **対戦 (PvP)** | 同期型の対人対戦 |
| **協力 (Co-op)** | 同期型の協力プレイ |
| **非対称 (Asym)** | 役割の違うチーム間対戦 |
| **非同期 (Async)** | 時間差での相互作用（メッセージ・島訪問・道共有等） |
| **観戦／配信 (Spectate)** | 大会・実況・配信視聴の価値が高い |
| **ソロ (Solo)** | 体験型だが他者相互作用が薄い |

## 3. 「他者軸」全体観察

### 3-1. 日本コンソール市場 = 「対戦＋協力＋非同期」が三本柱
- Top10 売上を握るのは ほぼ EXP 群（あつ森、マリカ、スプラ、スマブラ、MH）。
- 「他者」との関わり方が複層的（同期・非同期どちらも）。

### 3-2. 日本モバイル市場 = 「報酬型」優位だが、ラストウォー／モンスト等 EXP 要素混在
- ラストウォー：サバイバルは「同盟戦」という対戦・協力ハイブリッド。
- モンストは協力＋報酬。

### 3-3. インディー文化 = 「配信×他者観測」が広告チャネル化
- 8番出口、Phasmophobia、Buckshot Roulette、Vampire Survivors。
- 「友人がやっているのを見たから買う」フローが定型化。

### 3-4. From Software 系の非同期独自モデル
- メッセージ・血痕・召喚＝「他者は見えるが直接干渉しない」設計が国内・世界双方で評価。
- DEATH STRANDING も同思想（道・梯子の共有）。
