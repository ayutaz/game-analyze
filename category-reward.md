# 報酬型 (REW) ゲーム

ゲーム内報酬 — ガチャ、キャラ収集、ランキング、経験値、アイテム、レアドロップ — を得ることそのものが目的化しているゲーム群。

---

## 1. サブタイプ分類

| サブタイプ | 内容 | 代表例 |
|---|---|---|
| **ガチャRPG** | 確率排出キャラを集めて編成・育成 | FGO、モンスト、原神、スタレ |
| **デジタルTCG／カード収集** | カードを引き集める | ポケポケ、シャドバ、ハースストーン |
| **育成型** | 育成サイクル＋因子継承等の「最適解探索」 | ウマ娘、プロセカ、パワプロ |
| **位置情報ARPG** | 屋外でキャラ／報酬を獲得 | ポケモンGO、ドラクエウォーク |
| **コンソールRPG収集系** | キャラ／モンスター収集が中核 | ポケモンSV/SS/X・Y/赤緑、レジェンズZ-A |
| **SLG（同盟・国家戦）** | 課金で軍備拡張、同盟戦で順位 | ラストウォー、ホワイトアウト、Clash of Clans |

## 3. 報酬型のメカニクス分析

### 3-1. 「3つのループ」
報酬型ゲームに普遍する設計ループ:
1. **取得ループ** — ガチャ／ドロップ／カード開封で新キャラを得る
2. **成長ループ** — 育成・継承・限界突破で強化
3. **発露ループ** — クエスト／対戦／ランキングで使う

→ ポケポケは「2」を最小化し「1」と「3（対戦・図鑑）」に集中したことが軽量プレイで広く受けた要因。

### 3-2. IP × 軽量化 = 2024-2026 のメガヒット式
- **ポケモンTCGポケット** = ポケモンIP × 1日2パック開封の超軽量ループ → 6ヶ月で世界 $1bn 達成。
- **ウマ娘** = 競馬実名IP × 育成・SNS連携。

### 3-3. SLG「真の収益構造」
- ラストウォーやホワイトアウトは **ハイパーカジュアル風広告で集客 → SLG課金で回収** モデル。
- 同盟戦のため自然と「他者軸：対戦／協力」も持つが、課金駆動性が極めて高いため主分類は REW。

### 3-4. コンソールでのREW構造
- ポケモン本編は「ストーリー消費（NAR）」と「図鑑コンプ／対戦育成（REW）」の二層構造。
- 副分類で NAR を持つ場合が多い。

## 4. 報酬型タイトル数（主分類）

約 **25 タイトル**（カタログより集計）。

## 5. 市場・社会的観察

- **2025-2026 の国内モバイル収益 Top はラストウォー・モンスト・ポケポケがリード** (4Gamer/Sensor Tower)。
- 旧来の Cygames・サイゲ系・KLab 系の伝統 RPG ガチャは健在だが、**新規参入は海外発（HoYoverse、First Fun、Century）と Pokemon TCG Pocket が席巻**。
- **報酬型は「他者軸：観戦／配信」を取り入れた IP** が今後の差別化要因（ポケポケのコレクション SNS シェア、ウマ娘の動画文化）。
