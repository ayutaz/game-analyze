# ポケットモンスター 赤・緑

## なぜ REW 主分類なのか

このゲームを遊ぶ子どもがやっていたことを 1 文で言えば **「151 匹を集めて、友達と交換して、図鑑を埋める」** だった。戦闘も物語もあるが、子どもたちの会話の中心は常に「どこで何が捕れる」「あいつは持ってる？」の収集情報交換であり、主動機は **集めること** にある。

concept にある「通信ケーブルで友達のソフトとモンスターを交換する」という設計は、収集をソロ完結させずに **社交を経由してしか完成しない構造**にしている。赤・緑どちらか片方だけでは図鑑は埋まらず、ソフト分断 × 通信交換が「集め切る」ことを社会的タスクに昇格させた。この時点で「報酬獲得＝図鑑コンプ＝他者との接触」が一体化しており、典型的な REW 設計だと言える。

`secondary: NAR` が効くのは、ジムリーダー / 四天王 / ライバル / マサラタウン出発という古典的 RPG 文法が「集めながら旅する」ための背骨として最小限機能している点である。物語は薄いが、移動の動機としては十分に作用している。

## 報酬ループの組成

赤・緑の報酬構造は、後年のソーシャルゲームのガチャ収益モデルとは別系統の、**「希少性 × 交換可能性」を物理メディアで担保した**初期型の収集ループとして読める。

| 層 | 要素 | 効果 |
|---|---|---|
| 基礎報酬 | 野生捕獲・進化・レベル上げ | 短サイクルの達成感、図鑑カウントが進む可視的進捗 |
| 希少性層 | バージョン限定（赤＝ガーディ / 緑＝ロコン 等）、進化キー（カイリキー・ゲンガー等は通信進化）、伝説（ミュウツー 1 体） | 「ソロでは絶対に埋まらない」状態を意図的に作る |
| 交換層 | 通信ケーブル + 友達との対面交換 | 報酬獲得を社会的接続コストとセットにし、コミュニティ形成を強制 |
| メタ層 | 図鑑完成 / 殿堂入り / ミュウ配布イベント | 長期目標と公式運営の上層報酬 |
| 物語層 (secondary) | ジムバッジ 8 個・四天王・ライバル戦 | 移動と探索の動機付け、収集の道筋を強制誘導 |

ガチャは存在しないが、**「バージョン分割 × 通信進化 × 通信交換」の三点セットが、現代のガチャの希少性設計と機能的に等価**だと整理できる。希少枠は確率ではなく「他者を経由する手間」で生成されている。

## バズ要因 / 人気の根拠

popularity の「シリーズ原点、通信交換」を分解すると、当時の社会現象化はおよそ次の 4 要因に整理できる。

- **物理メディア時代のソーシャルゲーム化**: 通信ケーブルを介した対面交換が、教室・公園・友達の家を「ゲームを進める場」に変えた。ゲームを進めるための友達訪問という行動様式自体が新しかった。
- **バージョン分割の発明**: 赤・緑を別商品として売り、片方では完成しない設計を取ったことで、家族内・友達間で複数本買う動機が自然発生し、国内 822 万本・世界累計 3,100 万本超の販売規模に到達した。
- **発売後の継続的話題供給**: ミュウの裏技配布、コロコロコミック連載、アニメ放映（1997 開始）、トレーディングカードゲーム、後の青／ピカチュウ版へと、メディアミックスがリアルタイムで重ねられた。
- **長期 IP 化への足場**: ここで定義された「捕獲 → 育成 → 交換 → 対戦」の四点セットが、その後 30 年に渡る本編・派生（GO、ユナイト、SV、ポケポケ等）に継承され、シリーズ累計の世界販売・配信本数は他 IP を圧倒する水準に達している。

## 企画への示唆

- **「片方では完成しない」を商品分割で表現すると、ソフトが買われた瞬間にコミュニティが必要になる**。確率ガチャを使わずに希少性を作りたい場合、配布チャネル自体を分割する手は今も有効である。
- **収集対象の進化に「他プレイヤー経由」を 1 段だけ挟むと、SNS / コミュニティが自走する**。通信進化はその最小実装で、現代では「フレンド交換でのみ取れる」「2 人いないと進化しない」等に翻訳できる。
- **薄い物語 × 強い収集**でも、移動と探索の理由付けとして物語が機能していれば収集の単調さは消える。報酬型でも世界とキャラクターの最小限の文脈は割に合うコストになる。
- **発売後のメディアミックスを前提に IP 設計すると、ゲーム本体のライフサイクルが長期化する**。アニメ・カード・グッズ・続編の入る余白を最初の図鑑とキャラデザで確保しておくと、後段の収益柱が立つ。
- **「全部を集める」目標は数字で見える形にすると強い**。図鑑の埋まり具合という単純な進捗 UI が、報酬獲得の主観的価値を増幅している。

## 比較で見るべきタイトル

- **デジタルモンスター（バンダイ）** — 同時代の収集 × 対戦型携帯デバイスとして、対面通信の社交装置化を別アプローチで実装。
- **ポケットモンスター 金・銀（1999）** — 通信交換軸を引き継ぎつつ、時計機能・呼び戻し電話で「時間軸」を収集動機に追加した直接の後継。
- **どうぶつの森（任天堂）** — 「非同期 / 通信 / 収集」を別ジャンルで再構成した任天堂内の隣接系譜。
- **遊戯王 オフィシャルカードゲーム** — 物理メディア × 希少性 × 交換 × 対戦の四点セットを TCG として並走させた IP。
