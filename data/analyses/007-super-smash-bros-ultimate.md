# 大乱闘スマッシュブラザーズ SPECIAL

## なぜ EXP 主分類なのか

プレイヤーの主動機は「好きな IP のキャラで他人と殴り合う」こと、つまり**競うこと**である。concept が示す「オールスター IP 集約装置」は入口の比喩であって、実際の遊び時間の大半はローカル／オンライン対戦のフレーム単位の読み合いに費やされる。social_axis が「対戦」であることがそのまま主分類の根拠となる。

target に並ぶ「ローカル対戦パーティ需要」「競技プレイヤー」「IP ファン全般」は、同じ対戦システムを 3 つの異なる強度で消費する層を意味する。party 層は乱闘・アイテム・ステージギミックで体験を消費し、competitive 層は 1on1・アイテム無し・終点でフレーム読みを行う。同じソフトが両端を抱えるのは、ルールセットを切り替え可能にする設計の帰結である。secondary に REW が入っているのは、89 体のファイター解放・スピリッツ収集・フィギュア的ロスター完成という収集動機が副ループとして機能しているためで、対戦に飽きた時間を埋める受け皿になっている。

## 体験の核 / 設計の発明

中核メカニクスは「HP ではなく蓄積ダメージ％で吹っ飛ばす」吹っ飛ばし型格闘である。これにより**残体力の不可視化が消え、誰でも逆転可能性を体感できる**点が初心者と熟練者を同じ卓に座らせる装置になっている。

| 設計レイヤ | 内容 | 観客／プレイヤーへの効果 |
|---|---|---|
| ルール層 | 蓄積％・場外撃墜・ストック制 | HP バー型と違い、終盤の 1 ヒットで逆転が起こる |
| キャラ層 | 89 体（参戦キャラ史上最大級）＋ DLC | IP 露出が即「使いたい動機」になる |
| ステージ層 | 100 以上 + 終点化トグル | カジュアル／競技の両ルールセットを同居 |
| 操作層 | ボタン少・コマンド技ほぼ無し | 初手の手触りが軽く、深さは間合いとベクトルに移譲 |

前世代（for 3DS/Wii U）との差分は、参戦キャラを「全員参戦」に振り切ったこと、ストーリーモード「灯火の星」でスピリッツによる縦の収集軸を作ったこと、そしてオンライン対戦をデフォルト動線に格上げしたことの 3 点である。social_axis「対戦」は同部屋ローカルと世界戦の双方で発火し、Switch のハードウェア性質（Joy-Con おすそわけ）が party 側の体験を底上げしている。

## バズ要因 / 人気の根拠

popularity に書かれた「キャラクロスオーバー集大成」を分解すると、

- 任天堂・サードパーティを横断する **89 体の参戦キャラ**（バンナム／カプコン／スクエニ／SEGA／コナミ等）が IP ニュースとして機能し、DLC 発表ごとに「次は誰か」がメディア化された
- 国内 574 万本 / 世界 3,500 万本超という売上は、Switch 本体世代の対戦アクションとして突出して厚く、**ローカル多人数需要を Switch の据置／携帯両モードが受けた**ことが追い風になった
- 競技シーンが EVO 等の国際格ゲー大会の枠で扱われ、配信視聴を通じて非プレイヤーの観戦動機を生んだ
- ストーリーモード「灯火の星」のスピリッツ収集が、対戦に乗らない層にもプレイ理由を提供した

## 企画への示唆

- **蓄積型ダメージで残量を不可視化すると、逆転可能性が常に画面に残る** → 観戦・配信時の最終局面まで視聴維持率が落ちにくい
- **キャラを増やすほどリリース毎にニュースサイクルが伸びる** → DLC ロードマップ自体が継続的なマーケ装置になる
- **同じ対戦システムにルールトグルを 2 つ（party / competitive）用意すると、コミュニティが自然に二分化して両方を回す** → 上層と下層の両方からプレイ時間が積み上がる
- **対戦コアに収集（スピリッツ／ファイター解放）を貼ると、対戦疲れの離脱時間をソロループで吸収できる** → secondary に REW を持たせる設計が retention を伸ばす
- **既存 IP を横断する横スクロール格闘の文法は、版権交渉さえ突破すれば営業資料が要らない** → IP 自体が target を連れてくる
