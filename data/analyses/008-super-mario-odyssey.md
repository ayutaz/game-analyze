# スーパーマリオオデッセイ

## なぜ EXP 主分類なのか

プレイヤーの主動機は「キャプチャーで世界の物理を切り替えながら、立体空間を踏破する」こと、つまり**体験すること**そのものにある。`concept` が示す通り、本作の中核は帽子「キャッピー」を投げて敵やオブジェクトに憑依する発明であり、これは物語を消費する装置でも、報酬を蓄積する装置でもなく、操作の手触りを更新するための装置である。クッパ討伐という物語フレームは存在するが、それは各国（クニ）を巡る背骨に過ぎず、プレイヤーが立ち止まって楽しむのは「ノコノコでスロープを滑る」「クリボーを積む」「Tレックスで岩を砕く」といったキャプチャー固有のミニ体験である。

`social_axis` は **ソロ**。2人プレイ（キャッピー操作）はあるが、競争・協力・観戦のいずれも中核ではなく、原則として一人称的な探索体験として設計されている。`target` が「3D マリオの伝統を愛する 20-40 代と、新しい操作体験を求めるアクションファン」と二層に分かれている点も EXP 分類を補強する：旧来層には 64／サンシャイン譜系の自由探索を、新規層には「触ったことのない操作」を提供する設計になっている。`secondary` は空で、収集（パワームーン 880 個）はあくまでも探索体験の燃料として機能している。

## 体験の核 / 設計の発明

キャプチャーが「再発明」である理由は、3D マリオが従来抱えていた「マリオの動きは固定、世界の方が応答する」という非対称を反転させ、**世界側のアクター語彙をプレイヤー側に取り込んだ**点にある。これにより、1 つのステージは「マリオで遊ぶ場」から「複数のキャラクターで遊ぶ場」に多重化された。

| レイヤー | 64／サンシャイン | オデッセイ |
|---|---|---|
| プレイヤー動詞 | ジャンプ・ヒップドロップ・ポンプ放水 | ジャンプ＋帽子投げ＋約 50 種のキャプチャー固有動詞 |
| 場の再利用 | ステージ内ミッション切替 | 同ステージで操作主体ごとに別ルートが開く |
| 学習コスト | マリオ操作の習熟 | キャプチャー対象ごとの操作チュートリアルが分散投下 |
| クリア条件 | スター（収束的目的） | パワームーン（拡散的目的、密度設計） |

社会軸ソロでありながら飽きが来にくいのは、キャプチャー対象ごとに「操作チュートリアル → 短い達成」のループが数十秒〜数分単位で回るからで、これは前世代の「中目的（スター 1 個 5-10 分）」より粒度を一段細かくした構造である。

## バズ要因 / 人気の根拠

`popularity` 表記は「キャプチャーで再発明」。これを分解すると：

- 世界累計 **28M 本超**。Switch 本体の初年度キラータイトルとして本体販売を牽引した。
- 発売直後の Game Awards 2017 で複数部門ノミネート（GOTY 候補）入りし、3D アクションのレビュー平均でシリーズ最上位帯に位置する評価を獲得。
- 「キャプチャー」という単語自体がミーム化し、Tレックス／電線ヒョロロ／ジュゲム等の SNS スクリーンショットが拡散、発売前 PV のニュードンク市の実写都市表現も話題化した。
- スピードラン文脈でも長期間継続的に走られており、Any%／100% それぞれにコミュニティが形成され、配信視聴のロングテールを生んだ。

## 企画への示唆

- 既存 IP を更新したいなら **「主人公の動詞を増やす」より「世界側の動詞を主人公に取り込ませる」** ほうが拡張性が高い（オデッセイ＝マリオに約 50 体分の動詞を後付け）。
- 収集ターゲットを **収束目的（スター）から拡散目的（ムーン）に切り替える** と、同じ空間の再利用率が跳ね上がり、開発コストあたりのプレイ時間が伸びる。
- 一人プレイ前提でも、**操作主体を切り替え続ける設計**にすると単調感を回避でき、ソロ EXP が持続する。
- 新メカニクスは **既存ファンの郷愁（64 リスペクトのパーティクル／BGM）と並走させる** と、新規・既存どちらの target にも刺さる二層設計になる。
- チュートリアルを集中投下せず、**キャプチャー対象ごとに分散投下**することで、ゲーム全体を学習曲線で覆える（プレイ後半でも新しい動詞が登場し続ける）。

## 比較で見るべきタイトル

- **スーパーマリオ 64**（1996）— 自由探索 3D アクションの原型、収束目的（スター）の世代。
- **ゼルダの伝説 ブレス オブ ザ ワイルド**（2017）— 同年・同 social_axis ソロ、物理ルールの一貫性で世界を再利用するアプローチ。
- **A Hat in Time**（2017）— インディー版オマージュ、帽子＝能力切替モチーフ。
- **マリオ ワンダー**（2023）— 「ワンダー効果」で世界側の動詞を変化させる設計、2D 文脈での再発明。
