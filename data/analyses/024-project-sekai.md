# プロジェクトセカイ カラフルステージ! feat. 初音ミク

## なぜ REW 主分類なのか

プレイヤーの主動機は「推しユニット／推しメンバーのカードを集めて育てる」こと。リズムゲームの腕前を磨くこと自体ではなく、ライブで稼いだスコアを通貨に変えてガチャ・ランクマッチ・ランキングイベントに突っ込む経路が中心にある。

concept にある通り本作は VOCALOID シーンとリズムゲームの結節点で、5 ユニット × 各 4 名 + バーチャル・シンガー 6 名のキャラクター IP を、ストーリー → 楽曲 → カード → ガチャの順で換金する構造になっている。target が「中高生〜20 代の VOCALOID 文化育ち層」である以上、課金動機は「曲が好き」より「キャラと曲のセットを揃えたい」に寄る。social_axis が「協力」なのはイベント期間中のチーム編成・協力ライブ（マルチプレイ）が貢献度ランキングを決定づけるからで、これは報酬獲得の手段として機能している。

secondary に EXP が立つのは、リズムゲーム部分が単独で完結する操作快感を持っているため。譜面の難易度（EASY〜MASTER、後に APPEND）と判定の厳しさは音ゲー単体として成立する強度を持つが、収益・継続の主軸はカード収集にある。

## 報酬ループの組成

| 層 | 通貨 / メカニクス | 役割 |
|---|---|---|
| 楽曲プレイ | スコア・経験値・マイセット素材 | 日次の燃料供給。スタミナ消費でランク上昇 |
| ガチャ | クリスタル（有償／無償）→ 限定★4 カード | キャラ獲得の主経路。スキル値とサイドストーリー解放権 |
| カード育成 | マスターランク・サイドストーリー | カードを「読み物」にし、★4 を最大解放するまで囲い込む |
| イベント（ワールドリンク／マラソン） | イベント P・ボーダー | 期間限定ランキングで上位報酬を分配。協力ライブが効率上昇 |
| バーチャルライブ | リアルタイム同期視聴 | アバター衣装の販売動線。コラボ告知の場として機能 |

ガチャ→育成→イベント→再課金の循環の上に、**サイドストーリーが「ガチャ景品を物語として消費させる」**役割で重なっているのが特徴。カードを引いただけでは終わらず、マスターランクを上げて読了するまでがワンセットになっており、これが LTV を引き伸ばす。

## バズ要因 / 人気の根拠

- 世界累計 7,000 万 DL+。日本モバイル収益ランキング上位の常連で、リズム＋ガチャ系として FGO / モンスト級の収益帯に届いている
- VOCALOID コミュニティの楽曲投稿エコシステムを直接取り込んだ。書き下ろし新曲を継続供給する仕組みで、ニコニコ / YouTube のヒット曲が遅延ゼロで実装される
- バーチャルライブ機能で初音ミクら IP の「ライブを観る」体験をアプリ内に常設し、リアルライブ・コラボグッズと往復する物販動線を作った
- 5 ユニット並列構造（Leo/need、MORE MORE JUMP!、Vivid BAD SQUAD、ワンダーランズ×ショウタイム、25 時、ナイトコードで。）で、推しジャンルを分散させて間口を広げた

## 企画への示唆

- ガチャ景品にサイドストーリーを紐づけると、引いた後の「読了 → マスター解放」までが課金理由になる
- 既存 UGC コミュニティ（VOCALOID 投稿文化）の上に乗ると、新曲制作コストを外部化しつつ「界隈のヒット曲が即実装される」鮮度感を演出できる
- ユニット数を 5 並列にすると、ファン層をジャンル別に分割でき、新ユニット追加で休眠ユーザーを呼び戻せる
- リアルタイム同期の「バーチャルライブ」を実装すると、リアルライブ・グッズ・コラボ告知の発射台として機能し、アプリ外の経済圏と接続できる
- 協力ライブを「上位報酬への近道」として設計すると、ソロ音ゲーに社交圧を持ち込めるが、ボーダー疲弊と表裏一体になる点に注意
