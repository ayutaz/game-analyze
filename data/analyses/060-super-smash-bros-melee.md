## なぜ EXP 主分類なのか

このタイトルの主動機は「人間を相手に競う」ことであり、操作技術を磨いて勝つ過程そのものが報酬になっている。キャラクターや IP は入口に過ぎず、20 年以上にわたり国際大会が続いている事実が示すのは、プレイヤーがコレクションでも物語でもなく対人技術の深さを消費しているということだ。social_axis が「対戦」である根拠もここにあり、コンセプトに記された「移動キャンセル（ウェイブダッシュ）発見で深い競技性を持つ」という設計外で生まれた挙動が、競技層の主動機を支えている。

target は 30–40 代の格闘ゲーム熟練層、特に「上限の高さ」を求める競技者で、これは ID 060 を REW（キャラ収集ゲーム）ではなく EXP として読むべき決定的な手がかりになる。secondary が空であることも、本作が単一の動機—対人で勝つ—に絞って遊ばれていることを補強している。

## 体験の核 / 設計の発明

GC 1 本だけで稼働し続ける理由は、操作系の偶発的な深さにある。仕様の意図と実プレイの乖離が「攻略の余白」として残り、それを発見・最適化していく行為自体が体験になっている。

| 層 | 仕様 | 競技層の使い方 |
|---|---|---|
| 移動 | ダッシュとシールドの組み合わせ | ウェイブダッシュで全方向に滑走、立ち回りの速度を底上げ |
| 復帰 | %蓄積による吹っ飛び | エッジ周りの 1 フレーム単位の駆け引き（メテオ・エッジガード） |
| 操作 | アナログ入力 + L キャンセル | 着地硬直を半減、コンボ密度の確保 |
| キャラ | 26 体の多軸バランス | Fox / Falco / Marth / Sheik 等の Tier 固定だが選択の余地は維持 |

前世代の N64 版との差分は、フレーム数の高速化と空中行動の自由度であり、これが「キャラゲー」を「対戦技術ゲー」へ転位させた。social_axis「対戦」が機能しているのは、勝敗の説明変数が乱数ではなく操作精度に寄っているためだ。

## バズ要因 / 人気の根拠

- 全世界 700 万本超を売り上げた GC 屈指のキラータイトルで、本体牽引力を持っていた
- EVO や Genesis など海外大会で 20 年以上トーナメントが組まれ続け、配信視聴数が新作リリース時の指標になる
- メーカー想定外の挙動（ウェイブダッシュ、L キャンセル）がコミュニティ主導で「正典化」され、攻略文化が長期化した
- ニンテンドーオールスター IP の入口性と、深掘りで現れる格闘ゲーム的精密さが、ライトとハードコアの両方に刺さる二層構造

## 企画への示唆

- キャラクター IP を入口にしつつ操作精度に勝敗を寄せると、ライト層とコア層の二層を同居させられる
- 仕様の意図外挙動を削らず競技層に渡すと、発売後 20 年スケールの長期コミュニティが立ち上がる余地が生まれる
- ストック制 + %蓄積のような、勝敗が見えやすく逆転余地もあるルールにすると観戦コンテンツとして成立する
- 既存 IP のオールスター化は単発の話題ではなく、対人軸を設計すれば「常設の競技種目」に育つ
- アップデート停止後も成立する設計（GC 1 本で完結）は、運営型と対極にある「凍結された競技資産」というモデルを示す

## 比較で見るべきタイトル

- スマブラ SP（ID 058 系）— 同シリーズ後継、競技性と量の関係の対照
- ストリートファイター 6 — 対戦アクションの近年指標、ロールバック前提の競技配信モデル
- 大乱闘スマッシュブラザーズ for Wii U / 3DS — DX とは別のバランス哲学を取った中間世代
- 鉄拳 8 — 3D 格闘の側で「凍結された競技資産」とは逆の運営型アップデート路線
