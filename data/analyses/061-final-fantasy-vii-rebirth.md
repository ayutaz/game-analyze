# ファイナルファンタジーVII リバース

## なぜ NAR 主分類なのか

このゲームを遊ぶプレイヤーの主動機は「1997 年版『FF7』の中盤＝ミッドガル脱出からフォルゴットンキャピタル到達までを、現代の映像演出と尺で**追体験する**」ことに集約される。concept にある通り、原作の冒険パートを開放型マップで再構築した「リメイク 3 部作の第 2 部」であり、ストーリーの結末（あの場面）を知っている前提で、その既知の物語をどう再解釈・改変して見せるかが商品価値の中心になっている。target が「30-50 代の FF7 オリジナル世代 + 若年層 JRPG ファン」に明確に二極化しているのも、物語消費を駆動力にした作品設計の帰結である。

secondary に EXP が入っているのは妥当で、コマンド＋アクションのハイブリッド戦闘、ミニゲーム（クイーンズブラッド、チョコボレース、ピアノ）、広域マップの探索など、体験型要素の総量は前作 Remake より明確に増えている。ただしこれらは「物語を分割して進めるための章間アクティビティ」として配置されており、戦闘や探索を回し続けることがゲームの目的にはなりにくい。

## 物語の駆動構造

3 部作中盤という構造上、Rebirth は「導入と結末を持たない長い第二幕」を成立させる必要があり、原作の地理を章割りに変換した移動劇として再設計されている。

| 駆動レイヤ | 仕掛け | 効果 |
|---|---|---|
| 章立て | 原作の地名（カーム→ジュノン→コスモキャニオン→ニブルヘイム→ゴールドソーサー→フォルゴットン）を章境界に流用 | オリジナル世代に「次はあの街」という予測可能性を与える |
| 改変 | エンディングと一部キャラの運命を原作から逸脱させる | 既知ファンに「今作で何が変わるか」の謎を持たせ、配信・考察を駆動 |
| 関係性 | デート相手選択（エアリス／ティファ／ユフィ／バレット／レッド XIII／ケットシー）と好感度パラメータ | キャラ消費を周回動機に変換 |
| 章間アクティビティ | クイーンズブラッド、チョコボ系ミニゲーム、サブクエ | 物語密度の希薄なフィールド移動を退屈にさせない緩衝材 |

原作既プレイ層には「結末改変の意図を読む」観賞動機、未プレイ層には「次の街と次の仲間に会う」純粋な続き読み動機を、同じ章構造で同時に供給している。

## バズ要因 / 人気の根拠

- 1997 年の原作プレイヤーが現在 30-50 代の購買力ピーク世代になっており、世代横断 IP として再起動された
- PS5 独占という大型タイトルとしては珍しい絞り込みで、ハード牽引タイトルとして注目を集めた
- 原作の象徴的シーン（特定キャラの運命）を改変する/しないという論点が SNS と動画プラットフォームでの考察コンテンツを生んだ
- ミニゲーム「クイーンズブラッド」が独立した遊びとして評価され、本編の物語消費と切り離してバズる二次的経路を獲得した

## 企画への示唆

- 既知 IP を分割販売する場合、各章を**地理＝原作の地名**で切ると、ファンに「次はあそこ」の予測を与えつつ商品単位として成立させやすい
- 結末を知っている素材を売るときは、「結末が変わるかもしれない」という揺らぎを 1 つ仕込むと、既知ファンの観賞動機を再点火できる
- 物語密度が薄くなる移動パートには、独立して遊べるミニゲーム（カードゲーム等）を埋め込むと、本編バズの外側に二次的話題性を作れる
- 好感度＋デート相手選択は、キャラ単位での周回・推し活動機に変換できる古典装置として現役で機能する
- 主分類が物語型でも、コマンド＋アクションの戦闘やオープンフィールドを secondary として実装することは、若年層 JRPG ファンの取り込みに有効
