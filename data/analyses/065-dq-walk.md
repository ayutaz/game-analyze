# ドラゴンクエストウォーク

## なぜ REW 主分類なのか

このゲームの主動機は「歩いて素材・装備・キャラを集めて育てる」こと、つまり**集める／育てる**であり、戦闘体験そのものではなく蓄積の手応えが中心に置かれている。コンセプトは「ポケモンGOの位置情報フォーマットを、ドラクエの『町から町へ歩く』物語に当てはめた歩行 RPG」で、ターゲットは 30〜50 代の通勤・散歩ユーザー、特に DQ ファンの社会人層。歩数や移動距離は経験値・素材・ふくびき券に変換され、毎日の通勤が「ガチャ通貨を貯める作業」になる構造を持つ。

`social_axis: 非同期` は、フレンドのおみやげ／ほこら共有／全国ランキングなど「他人と直接対戦はしないが、他人の足跡が自分の盤面に流れ込む」設計を指す。secondary に EXP が立つのは、ふくびき装備と転職で組んだパーティでメガモンスター・強敵を倒すバトル体験が副次動機として効いているためで、報酬を取りに行く動機が主、戦闘の手応えが従、という重みづけになる。

## 報酬ループの組成

DQ ウォークの報酬ループは「歩行＝資源生産」を最下層に置いた多層構造になっている。

| 層 | 報酬源 | リテンション機構 |
|---|---|---|
| 日次 | 歩数・移動距離・スタミナ消費 | ログインなしでも歩けば素材が貯まる |
| 週次 | こころ収集（モンスターのドロップ装備に近い） | こころグレード S/A/B の埋め直し |
| 月次 | 章追加・季節イベント・コラボ | 期間限定ふくびき・限定こころ |
| 縦軸 | ストーリー章クリアによる新地域開放 | 物語進行が新ふくびきの解禁条件 |
| 横軸 | ふくびき（武器ガチャ）と転職 | キャラ枠ではなく装備＋ジョブで差別化 |

ポケモン GO と異なり、ガチャの主対象は「キャラ」ではなく**ドラクエ伝統の武器**である点が肝で、IP の象徴である「ロトの剣」「天空のつるぎ」を月替わりで投入できる弾切れの起きにくい設計になっている。

## バズ要因 / 人気の根拠

- 国内モバイル収益ランキングで配信初年度から上位常連。位置情報ゲームとしては Pokémon GO に次ぐ規模。
- ドラクエ本編の世界観・楽曲・モンスターをそのまま流用できる IP 力で、初動から既存ファン層を吸引した。
- スマホ位置情報 × ドラクエ本編の「町を順番に攻略していく」フォーマットの噛み合わせが良く、通勤導線がそのままプレイ動線になる。
- メガモンスター（広域 raid 的な強敵）と季節ごとのご当地クエスト・コラボ章で、長期運用のネタ切れを地理拡張で吸収している。

## 企画への示唆

- **既存生活動線を資源生産にマップすると、ゲーム時間の確保コストがゼロになる**（通勤＝経験値生産）。
- **IP に既にあるアイコン（武器・呪文・モンスター）をガチャ排出対象にすると、キャラ追加の弾切れを構造的に回避できる**。
- **位置情報 × IP の組み合わせは、IP 側に「地理・移動」のモチーフが既にあるほど自然に接続する**（DQ は「旅」が原作モチーフ）。
- **30〜50 代の社会人層を取りに行くなら、片手・短時間・スタミナ消費型の UI を採用すると、課金可処分が高い層を継続させやすい**。
- **非同期マルチ（ほこら共有・おみやげ）は、対戦ストレスを避けつつ「他人の存在感」だけを抽出できる**ので、ライト層リテンションに効く。
