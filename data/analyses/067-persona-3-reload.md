## なぜ NAR 主分類なのか

プレイヤーの主動機は「2006 年に提示された SEES の物語を、現代的な解像度で読み直す」ことに集約される。concept にある通り本作は原作のフルリメイクであり、target も「ペルソナ5 以降の新規層 + 原作プレイヤー」と明示されている。つまり購入動機は新規メカニクスの発見ではなく、確定したキャラクター・確定した結末を、現行ハードのカットイン演出・フルボイス・心の声システムで再体験すること自体にある。social_axis はソロで、進行は単線（暦は分岐するが本筋は固定）、コミュは仲間ごとに用意された短編連作として消費される。

副次の REW は、ペルソナ合体・コミュ Max・タルタロス踏破などの収集／育成ループが背景で回り続けることを指す。物語消費を維持する伴走者であり、戦闘の手応えやペルソナ図鑑コンプは主目的ではなく「次の暦・次の物語イベントに到達するための燃料」として機能している。

## 物語の駆動構造

ペルソナ 3 の物語は「カレンダー × 満月作戦 × コミュニティ」という三層で駆動する。Reload は原作の構造に手を加えず、提示密度のみを上げた。

| 層 | 役割 | Reload での更新 |
|---|---|---|
| カレンダー（1年） | 死へのカウントダウンを可視化、章立てに代わる時間軸 | UI を 5 譲りのモダン化、日めくり演出を強化 |
| 満月作戦 | ボス戦をマイルストーン化、物語の節目を固定 | ボスデモを再演出、戦闘 BGM 追加 |
| コミュニティ | 仲間・脇役の個別短編をプレイヤーが選択して進める | 男性仲間コミュを新規実装、Linked Episode で側面補完 |
| 心の声 / Theurgy | 主人公・仲間の内面独白と必殺技を直結 | 新規システムとして追加、キャラクターの内面に音声を割り当て |

カレンダーが終端を確定させるため、プレイヤーは「あと何回この日常があるか」を毎日意識する。コミュは並列スレッドで、未消化のまま終わると喪失として残る。この構造的アイロニーが物語消費を強く駆動する。

## バズ要因 / 人気の根拠

- アトラス・セガ全体でも上位の立ち上がり。発売 1 週間でシリーズ最速 100 万本到達、初年度でシリーズ累計売上を押し上げた話題作として国内外メディアに広く取り上げられた。
- ペルソナ 5 で新規参入した層に対し「シリーズの原点」を遊べる導線になった。target に明記された通り 5 以降のファン回収が成立した。
- Xbox / PC Game Pass day-one 配信で英語圏のディスカバリーが拡大、初週レビュー本数で過去のリメイク JRPG を上回る規模に達した。
- 心の声 / Theurgy / 男性コミュ拡張など「原作で物足りなかった部分」を狙い撃ちした追加が、原作プレイヤーの SNS 上での比較考察を活性化させた。

## 企画への示唆

- 完結済み IP のリメイクでは、ストーリー本筋を改変せず「提示の密度」だけを引き上げると、原作層と新規層の両方を取りに行ける。
- カレンダー＋締切のあるシナリオ構造に、並列の短編コミュを乗せると、未消化が喪失感に転化してリプレイ動機が生まれる。
- 内面独白（心の声）をシステム化すると、ボイス収録分の追加コストでキャラクターの厚みが線形以上に増す。
- リメイクで追加要素を入れるなら「原作で議論された欠落」を狙うと、SNS 上の比較考察そのものが宣伝資産になる。
- Game Pass / 多言語同時配信は、JRPG リメイクのディスカバリーで最大のレバレッジになる。day-one 投入の判断は売上ではなく「英語圏での議論量」で評価する。
