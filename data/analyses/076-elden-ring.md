## なぜ「EXP 主分類」なのか

エルデンリングは物語的に深く（NAR 副分類）、報酬要素も豊富（武器・霊薬・ルーン）だが、**プレイヤーの主動機は「戦闘・探索という体験そのもの」** にある。フロム・ソフトウェアのソウル系列の延長で、 **「死を学習に翻訳する高難度ループ」** が主動機。

物語は「読む」ものではなく「断片を発見し、考察する」もの（Item Description Lore）。これは典型的な物語型ではなく、**プレイヤーの能動的発見そのものが体験**になっているため、EXP 分類が妥当。

## 「他者軸」の特殊性：マルチではない他者

エルデンリングの社会軸は **非対称 + 非同期** という稀有な組み合わせ：

| 機能 | 何をするか |
|---|---|
| **メッセージ機能** | 他プレイヤーが地面に残したヒント / トラップ警告 / 罠が、自分の世界に表示される |
| **血だまり** | 他プレイヤーの死亡演出を観察できる（学習素材） |
| **召喚サイン** | 他プレイヤーが「呼ばれて手伝う」協力プレイ |
| **侵入** | 他プレイヤーが侵入してきて殺しに来る（PvP） |

**「他人の足跡が世界に残っている」** という設計は、孤独な探索体験を「孤独でないように感じさせる」発明。SNS / Reddit / 攻略 Wiki でのコミュニティ熱狂もこの延長線上。

## バズ要因の分解

1. **フロム・ソフトウェアの IP**：ソウル → ブラッドボーン → SEKIRO の系譜
2. **ジョージ・R・R・マーティン**との共作（世界観構築）
3. **オープンワールド化**：従来のソウル系の閉鎖型から開放型へ
4. **配信文化との相性**：1ボスに何時間も挑む配信が成立する
5. **SHADOW OF THE ERDTREE（DLC）**：2024年 GOTY 受賞、本編から2年後の追加販売モデル

## エルデンリングの「設計の発明」

| 設計 | 効果 |
|---|---|
| 死亡時のルーン落とし＆回収 | 「もう一回挑戦したい」を強制する報酬構造 |
| 召喚サイン | 詰まったら助けを呼べる → 離脱率低下 |
| メッセージ機能 | 他者の足跡が「ヒント or 罠」両方になる |
| 戦灰 / 遺灰 | プレイスタイルを後から大幅に変えられる柔軟性 |
| オープンワールド + ダンジョン | 「詰まったら別のことをする」逃げ道 |

## 企画への示唆

- **高難度ゲームの真髄は「失敗の学習価値」**：死亡ペナルティをやり直しの動機に変換する
- **非同期マルチ（メッセージ・血だまり）はソロゲームに社会性を持ち込める**
- **物語を「読ませる」のではなく「発見させる」**ことで、能動的体験を作れる（IP のレア感も増す）
- **オープンワールドの「逃げ道」設計**は、初心者でも進める道を作る（線形ソウルではなかった）
- **DLC モデル**（数年後の大型追加）は、コアユーザーへのリテンションとして強力
