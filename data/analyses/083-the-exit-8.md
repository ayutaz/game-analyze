# 8番出口 / The Exit 8 — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「観察して、異変を見分けて、進むか引き返すかを判断する」こと。物語の追体験ではなく、知覚と意思決定の体験そのものが商品である。

舞台は無限に続く地下通路の単一区画。プレイヤーはひたすら歩きながら、ポスター、照明、広告、おじさん（通行人NPC）、扉、案内表示など数十の要素を毎ループ照合し、「異変があれば引き返す／なければ進む」の二択を繰り返す。`concept` にある通り、ゲームメカニクスは観察と判断の 2 動詞だけで構成されており、プレイヤーが楽しんでいるのは「自分の知覚精度が試される瞬間」そのものだ。これは典型的な体験型（EXP）の構造である。

`secondary: NAR` は、異変そのものに含意される地下通路の不穏さ、結末の解釈余地、KOTAKE CREATE 個人作家の作家性が物語的余韻を残す点を指す。ただし物語はメカニクスの背景としてのみ機能しており、プレイヤーは物語を消費しに来ているのではなく「異変を当てに来ている」。`social_axis` 未設定だが、本質的にソロ体験であり、後述の通り「配信観戦による疑似マルチプレイ化」が拡散の主動力になった。

## 体験の核 / 設計の発明

本作の発明は、ホラーを「ジャンプスケア」ではなく「間違い探し」に再定義した点にある。要素を分解すると以下のようになる。

| 設計要素 | 内容 | 効果 |
|---|---|---|
| ループ構造 | 同一区画を 0〜8 番出口まで反復 | 学習コストゼロ、即理解可能 |
| 二択判断 | 異変あり→引き返す／なし→進む | 操作 1 つに緊張が集約 |
| 異変カタログ | 約 30 種類の細部差異 | 観察対象の網羅が遊びの目標になる |
| 失敗ペナルティ | 0 番に戻されるだけ（死亡演出なし） | ホラー耐性が低い層にも届く |
| プレイ時間 | クリア 30 分〜数時間 | 配信尺・SNS 共有尺に最適化 |

特筆すべきは「失敗してもジャンプスケアで脅さない」設計判断である。前世代の P.T.（2014）や Visage が「怖がらせて止める」方向だったのに対し、8番出口は「気づいたかどうかをじわっと突きつける」方向に振った。結果、ホラー実況の主要視聴者であるホラー苦手層に対して入口が広がり、観戦コンテンツとしての射程が伸びた。`social_axis` は形式上ソロだが、実態は「配信者と視聴者で異変を一緒に探す」非同期協力型に近い。

## バズ要因 / 人気の根拠

- KOTAKE CREATE は実質一人開発のインディー。最小コストで「短尺・観察主体・配信向き」の三拍子を成立させた点が、AAA との差別化として機能した
- TikTok・YouTube Shorts で「異変だけ抜き出した数十秒クリップ」が拡散しやすい構造を持ち、`target` の通り非ゲーマー層まで到達した
- 任天堂 Direct（2024）で Switch 版が紹介された後、Steam / PS5 / Switch のマルチプラットフォーム展開で間口がさらに広がった
- 続編『8番のりば』(2024)、実写映画化（2025）、書籍化など IP 拡張が連鎖し、「日本発インディー → 国際的ミーム」の経路が成立した

## 企画への示唆

- ホラーを「驚かせる」ではなく「気づかせる」設計に振ると、ホラー苦手層と配信視聴層を同時に獲得できる
- ループ構造をコアにすると、学習曲線が極端に低くなり、配信者と視聴者の間で前提知識を共有しなくて済む
- プレイ時間 30 分〜数時間に絞ると、配信 1 回で完結し、SNS のクリップ尺とも整合する
- メカニクスを 2 動詞（観察・判断）まで圧縮すると、操作説明が不要になり、言語・年齢の壁を越える
- 失敗ペナルティを軽くすると（死亡演出なし／即リトライ）、緊張は残しつつ離脱を防げる
- 一人作家の作家性を前面に出すと、IP 拡張時に「KOTAKE CREATE の新作」というブランドが効く

## 比較で見るべきタイトル

- P.T.（2014）— ループ × 一人称ホラーの前世代。ジャンプスケア寄り
- Anatomy / Faith（インディーホラー）— 観察主体・低予算・配信向きの先行例
- 8番のりば（2024、同作者の続編）— 同じ二択ループを「動く電車」に展開
- The Stanley Parable（2013）— ループ構造を物語的に転用した先行例
- Pokémon Sleep / Strange Horticulture — 「観察」を主動詞に置いた異ジャンル比較対象
