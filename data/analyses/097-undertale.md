# UNDERTALE

## なぜ NAR 主分類なのか

プレイヤーの主動機は「自分が下した選択の意味を読み解く」ことにある。RPG の外形をまといながら、戦闘システムの中核に置かれているのは敵を倒すための最適化ではなく、「ACT で逃がすか、FIGHT で殺すか」という二者択一を全モンスターに対して延々と問い続ける構造である。concept にある通り、戦闘は数値ではなく「価値観そのものを問う」ための舞台装置として設計されており、プレイヤーは結果を物語の分岐として消費する。

主分類が NAR である根拠は、(a) ルート分岐（Pacifist / Neutral / Genocide）が章単位の物語消費を駆動していること、(b) target にある「ナラティブ重視層」「SNSミーム経由で入る層」が想定買い手であり、彼らが期待するのは戦闘習熟ではなく終盤の語りであること、の二点に集約される。secondary に EXP が入るのは、弾幕回避とアクションコマンドという体験面の発明があるためで、特に Sans 戦のフレーム単位回避は「体験の核」として独立して語られる。ただしそれは終盤の一部に集中しており、20 時間規模の動機としては物語が主、体験が副となる。

## 物語の駆動構造

| 駆動要素 | 仕掛け | 物語消費への効果 |
|---|---|---|
| ルート分岐 | 全モンスターを助ける / 中立 / 全滅させる の 3 系統 | 1 周目では完結せず、必ず複数周回前提で「真の物語」を取りに行く動機を作る |
| メタ参照 | セーブデータを世界が記憶し、キャラがプレイヤーに直接話しかける | プレイヤー自身を物語の登場人物に組み込み、選択の罪悪感を外部化させない |
| キャラクター単位の戦闘 | 雑魚敵にも固有の台詞と弱点 ACT がある | 「倒す対象」を「対話する個人」に変換し、ルート選択の重みを担保 |
| カウンター式の終盤戦 | Genocide 進行に応じてラスボスが変質 | 行動の累積を物語の文法で報酬化する |

前世代の JRPG が「物語 → 戦闘 → 物語」を直線で繋いでいたのに対し、Undertale は戦闘画面そのものを物語の意思決定装置に再定義した。social_axis はソロだが、ルート分岐の解釈は配信・実況・考察コミュニティで集合的に消費される設計になっている。

## バズ要因 / 人気の根拠

- toby fox 一人制作（音楽・脚本・プログラム）という制作体制が SNS で物語化され、インディー文化の象徴として参照され続けている
- 楽曲（Megalovania 等）が TikTok / YouTube / アレンジ動画経由で原作未プレイ層にも拡散し、Sans / Papyrus のキャラクター画像がミーム化
- 実況配信との相性が極めて高い（短いボス戦、明確な分岐選択、声に出して読み上げられる台詞密度）
- Steam / Switch / PS / モバイルへの段階的マルチ展開で長期テールを確保。日本では Switch 版以降に二次的な拡散があり、続編 DELTARUNE への期待が母数を維持

## 企画への示唆

- 戦闘システムに「倒さない」選択肢を常設すると、メカニクスがそのまま倫理的選択装置に変わり、プレイヤーの感情が数値ではなく自己評価に紐づく
- 全敵キャラクターに固有名・固有台詞・固有 ACT を与えると、雑魚処理コストが「人間関係コスト」に変換され、ルート選択の重みが自動的に上がる
- セーブデータをゲーム世界の記憶として扱うと、リセットや周回が物語の一部になり、メタ構造が考察ネタとして外部に拡散しやすい
- 一人制作・低予算という制作背景を作品テーマ（小さな個の選択）と一致させると、SNS で語られる際の物語性が強化される
- 楽曲を「ボス＝モチーフ」で 1:1 設計すると、原作未プレイ層への入り口が音楽経由で確保できる
