## なぜ REW 主分類なのか

このタイトルでプレイヤーが第一に求めているのは「キャラ・装備・ペット（イマージェン）を育て、希少資産を蓄える」ことであり、動詞で言えば **集めて伸ばす** ことが主動機である。concept にある通り、5 職業のキャラ育成、装備強化、ペット収集、ギルド戦・領地戦という MMORPG の標準ループに、ブロックチェーン由来の Asprin / NKT という **ゲーム外に持ち出せる成果物** が積み上がる二重報酬構造を載せている。プレイヤーは「物語を読みに来た」のでも「対人プロセスそのものを楽しみに来た」のでもなく、**時間と課金を投じた成果が数値・資産として残ること** を目当てに継続している。

social_axis が「非同期」なのも REW 認定を補強する。ギルド戦・領地戦・サーバーランキングは確かに対他プレイヤーだが、これらは「自分の蓄積を比べる場」であって即時 PvP の腕前を競う場ではない。secondary に EXP が入るのは、戦闘アクションや領地戦の体験密度が一定確保されている点（ジブリ系アートと作り込まれた MMO 戦闘）から来ているが、これは主動機を補強するスパイスに留まる。

## 報酬ループの組成

二ノ国 Cross Worlds の収益エンジンは、伝統的な韓国系 MMO 経済の上に **P2E トークン層** を上乗せした多層構造になっている。

| 層 | 主な報酬 | 役割 |
|---|---|---|
| デイリー育成 | 経験値・装備素材・スタミナ消費 | 毎日のログイン理由を作る |
| キャラ・職業 | 5 職業の覚醒・スキル拡張 | 課金で時短可能な縦軸 |
| イマージェン | ペット収集・進化・親密度 | 「集める」ガチャ的横軸（IP との接点） |
| 装備強化 | 強化値・伝説装備の確率 | サーバー内ヒエラルキー形成 |
| ギルド・領地戦 | サーバー内ランキング、資源占有 | 重課金者の社会的可視化 |
| トークン経済 | Asprin（ソフトカレンシー）→ NKT | 蓄積をゲーム外に持ち出せる外部出口 |

このうち上 5 層は伝統的な韓国系 MMO がすでに完成させていた仕組みで、Cross Worlds はそこに **トークン層を「最上位の外部出口」として接続した** ことに発明がある。結果として、上位層プレイヤーには「強くなる」と「資産が増える」が同義となり、課金 → 育成 → ランキング → トークン獲得 → 再投資というループが閉じる。ジブリ系アートとイマージェンは、この硬派な経済構造に対する **入口の柔らかさ** と IP 認知を担っている。

## バズ要因 / 人気の根拠

- 日韓台先行ローンチ後 **11 日で全世界収益 1 億ドル突破**（Sensor Tower 集計）。同社調べでポケモン GO やリネージュ 2: レボリューションよりも到達ペースが速かった。
- 初期 11 日間収益のうち **日本市場が約 45%（約 4,560 万ドル）** を占有。Netmarble 系 MMO で日本比率がここまで高いのは異例で、二ノ国 IP の国内認知が効いた。
- ローンチ初年度で推定 5 億ドル超（Sensor Tower 2021-2022）と、モバイル MMORPG として上位常連クラスの規模感に到達した。
- 2021 年当時の **P2E ブームの代表例**として暗号資産メディア・ゲーム業界双方で取り上げられ、ゲーム外文脈（投資・トークン経済）でも話題化した。

## 企画への示唆

- 既存ジャンルの完成済み経済層の **最上位に「外部出口」を一段足す** と、課金者にとっての終着点が拡張され ARPU が伸びる（ただし規制リスクと表裏一体）。
- IP の **アート / 世界観の柔らかさ** と **経済設計の硬派さ** を意図的にミスマッチさせると、ライト層の入口とコア層の継続が両立する。
- ローンチを **日韓台同時** にすると、IP 認知が強い市場（ここでは日本）が初期売上の半分近くを叩き出し、グローバル指標を一気に押し上げられる。
- **「ポケモン GO より速い」のような既存ヒット作との到達スピード比較** を Sensor Tower 等の第三者集計で打ち出すと、ゲームメディア外（経済紙・暗号資産メディア）まで波及する。
- P2E をフックにする場合、**ゲーム性そのものは古典的な MMO でよい** ——新規ジャンル発明とトークン経済の同時提示は失敗確率を上げるため、片方は枯れた構造で固める。

## 比較で見るべきタイトル

- リネージュ 2: レボリューション（Netmarble、2017）：同社の韓国系 MMO 経済設計の原型。
- 黒い砂漠 MOBILE（Pearl Abyss、2018）：装備強化と狩り場占有を軸にした非同期 MMO 経済。
- Axie Infinity（Sky Mavis、2018）：P2E ブームの本家。経済設計が崩壊した先行事例として参照価値。
- 二ノ国 II レヴァナントキングダム（Level-5、2018）：同 IP の家庭用 RPG。IP の本来の文脈との距離を測る基準。
