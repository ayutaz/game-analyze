# ファイナルファンタジーVII リメイク

## なぜ NAR 主分類なのか

プレイヤーの主動機は「クラウドたちのミッドガル脱出を、現代の演出で**再び見届ける**」こと。戦闘の手応えは強いが、その手応えは「次のカットシーンに進むための報酬」として配置されている。

concept にある通り、本作は原作 FF7 の冒頭「ミッドガル脱出編」（原作では 6〜8 時間相当）を 1 本のフルプライス作品に**引き延ばし**、章単位の起承転結とキャラクター掘り下げを追加した分作リメイクである。target が「原作 FF7 プレイ経験のある 30〜40 代」を主軸に置く以上、購入動機の中核は「既に知っている物語を、現代の映像と声優演技で追体験する」点にある。これは典型的な NAR の購買動機構造。

secondary に EXP が入るのは、リアルタイム+ATB のハイブリッド戦闘が「ボタン連打のムービー消費」ではなく能動的な操作を要求するため。ただし戦闘システムは物語を進める駆動装置として配置され、戦闘自体を周回したくなる設計（高難度・対人・装備ガチャ）は薄い。

## 物語の駆動構造

分作リメイクという形式は、原作既プレイ層に対して「知っているはずの物語が、いつどこで分岐するか分からない」という新しい緊張を持ち込んだ。フィーラー（運命の番人）という新キャラを介して「原作と同じ結末に向かう力」と「原作から逸脱する力」を画面上に可視化したのが構造上の発明である。

| 駆動レイヤー | 仕掛け | プレイヤーへの効果 |
|---|---|---|
| 章立て | 全18章、各章=ミッドガル内の1エリア | 原作のダンジョン1つを章1つに拡張、ペース管理 |
| キャラクター掘り下げ | ジェシー/ビッグス/ウェッジに個別エピソード | 原作で死ぬモブの「人格化」、後半の喪失感を増幅 |
| 運命の改変装置 | フィーラー=原作の筋書きを守る存在 | 既プレイ層に「次作で何が変わるか」の不確実性を埋め込む |
| 戦闘演出 | リミット技・召喚獣のフルカットシーン化 | 戦闘の山場を映像的クライマックスとして消費させる |
| 探索動線 | 一本道+小ハブ構造 | 物語テンポを優先、寄り道は章クリア後の章選択で回収 |

## バズ要因 / 人気の根拠

popularity に挙げられた「原作 FF7 の世界的 IP 力と分作リメイクの話題性」を分解する。

- 原作 FF7 (1997) は PS1 時代に世界で 1000 万本超を売り、JRPG をグローバルに押し上げた基準作。その**リメイク発表自体**が 2015 年 E3 以降のニュース価値を持ち続けていた。
- 世界販売 7M+ / 国内初週 34 万本という数字は、PS4 後期かつ COVID-19 直撃期 (2020-04) のリリースとしては高水準。
- 「分作である」という構造そのものが、発表当初から賛否両論の話題を生み、SNS・YouTube の考察コンテンツを大量発生させた。フィーラー登場以降は「原作改変かどうか」が一次的な議論軸として残り続けた。
- PS5 強化版『INTERGRADE』(2021)、続編『REBIRTH』(2024) と分作ロードマップが続いたことで、IP のニュースサイクルが 5 年以上維持された。

## 企画への示唆

- **既知の物語を引き延ばすと、新キャラの死が効く**：原作のモブ (ジェシー等) に固有エピソードを与えると、原作既プレイ層は「知っていた喪失」を初めて感情として受け取り、リメイクの存在意義が立つ。
- **分作にすると話題の総量が増える**：1 本完結より、3 部作にして「次は何が変わるか」を未確定領域として残す方が、発表→発売→続報の各サイクルで継続的にバズを起こせる。
- **戦闘を「ATB+リアルタイム」のハイブリッドに置くと、JRPG とアクション両方の客を取れる**：コマンド世代には ATB 待機時間を、アクション世代にはリアルタイム回避を提供する 2 段構えで、target が「30〜40 代既プレイ層 + 新規アクション RPG 層」の両極をカバーできる。
- **原作改変の余地を画面内に出すと、考察コンテンツが SNS で勝手に走る**：フィーラーのような「メタ装置」は、ファンに「議論する権利」を渡し、自発的な拡散を生む。
- **章選択+クリア後寄り道の構造にすると、本編テンポと収集要素を両立できる**：物語の駆動を阻害せず、コンプ層には章選択経由でやり込みを与える二重構造。

## 比較で見るべきタイトル

- **バイオハザード RE:2 / RE:4** — 同じく PS1/PS2 期名作を現代エンジンでリメイクし、原作既プレイ層を主購入層に据えた事例。
- **ペルソナ 3 リロード** — 既存 JRPG IP の「物語再演」型リメイク。分作ではないが、target 構造が近い。
- **キングダムハーツ III** — 同じ Square Enix の長期 IP 物語消費型 ARPG。戦闘の手応えを物語推進に従属させる設計が共通。
- **ファイナルファンタジー XVI** — 同 IP の現行系列、ただし完全新作・1 本完結・アクション特化で、本作の「分作 + 既知改変」とは対照的な選択。比較すると「分作リメイク」固有の効果が浮かぶ。
