# Vainglory

## なぜ EXP 主分類なのか

プレイヤーの主動機は「タッチ画面で 3v3 の MOBA を競う」こと、つまり **対戦の体験そのもの** にある。育成やストーリー消費ではなく、レーン戦・ガンク・オブジェクト管理という MOBA 固有の局面処理を、スマホの限られた入力面積で実行することが楽しさの中核を成す。

concept にある「LoL / Dota 2 譲りの戦略性を独自エンジンで携帯端末に持ち込んだ元祖モバイル MOBA」、target にある「精密なスキルショットや連携を競いたい競技志向プレイヤー」という記述からも、ガチャ報酬や物語ではなく **マッチごとの勝敗と上達** が駆動軸であることが読み取れる。`social_axis: 対戦` は MOBA というジャンル定義そのものに直結しており、REW でも NAR でもなく EXP に置く以外の解はない。secondary は空配列で、ヒーロー収集要素はあるが副次動機としてはわざわざ持ち上げないという編集判断になっている。

## 体験の核 / 設計の発明

Vainglory の発明は「PC MOBA のメカニクスを 5v5 のまま削らず、3v3 シングルレーンに圧縮した」点にある。プレイ時間を 20 分前後に短縮しつつ、ラストヒット・ジャングル・オブジェクト・アイテムビルドという MOBA の構造要素を残し、片手プレイ前提のタップ操作で精密なスキルショットを成立させた。

| 設計要素 | PC MOBA（LoL/Dota2）の標準 | Vainglory の選択 |
|---|---|---|
| プレイヤー数 | 5v5 | 3v3（後に 5v5 も追加） |
| マップ | 三レーン＋ジャングル | 一レーン＋ジャングル |
| 1 試合時間 | 30–45 分 | 約 20 分 |
| 入力 | マウス＋ホットキー | タップ移動＋スキルアイコン |
| 描画 | デスクトップ GPU | EVIL Engine、Metal API 最適化 |
| クロスプレイ | なし | 後年 iOS/Android/PC を統合 |

対戦軸が成立する条件として、ヒーロー数と meta の流動性、マッチメイキングの精度、観戦・配信の容易さを担保した。Apple の Metal API デモに採用されたほどのグラフィック品質は、競技シーンの配信映像が「モバイルゲームに見えない」状態を作り、e スポーツ化のハードルを下げた。

## バズ要因 / 人気の根拠

- Apple が 2014 年 iPhone 6 発表会で Metal API のグラフィック性能デモに採用し、ハードウェアメーカーが「スマホで PC 級が動く」と公式に保証する形で露出した。
- 開発元 Super Evil Megacorp は Riot Games / Blizzard / Rockstar 出身者で構成され、処女作にもかかわらず「PC MOBA の DNA を持つチーム」として注目された。
- ピーク期で月間アクティブ 150 万人規模に達し、北米・アジア・欧州で公式 e スポーツ大会が開催された。
- 後発の Mobile Legends / Arena of Valor / Wild Rift と比較されつつ、「元祖モバイル MOBA」というポジショニングを長く保持した。

## 企画への示唆

- PC ジャンルをモバイルに移植するときは **プレイ時間を半減** させると、通勤・休憩の入力枠に収まり対戦が成立する（5v5 → 3v3、45 分 → 20 分）。
- タッチ操作の弱点である精密性を補うために **スキルのターゲット UI を専用 GUI で再設計** すると、PC 経験者の不満を吸収できる。
- ハードウェアメーカーの公式デモ枠を取りに行くと、初動の信用と露出をプロモ予算外で確保できる（Apple Metal デモ採用の構造）。
- 開発陣のクレジット（Riot / Blizzard 等の出身）を前面に出すと、競技志向ユーザーは β 段階から評価軸を持って参加してくれる。
- モバイルでも **クロスプレイ対応を後付け** で打てる設計にしておくと、サービス後期に PC プレイヤー層を流入させライフサイクルを伸ばせる。

## 比較で見るべきタイトル

- Arena of Valor（Tencent、2016）— 同じ「PC MOBA をモバイルに圧縮」を 5v5 のまま実装した後発。
- Mobile Legends: Bang Bang（Moonton、2016）— 東南アジアで主流を奪った 5v5 モバイル MOBA。
- League of Legends: Wild Rift（Riot、2020）— 本家による「モバイル最適化された LoL」。
- Pokémon UNITE（TiMi/The Pokémon Company、2021）— MOBA の構造を IP と組み合わせて家庭用＋モバイルに展開した例。
