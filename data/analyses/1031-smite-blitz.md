# SMITE Blitz — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「神々のカードを集めて編成し、戦力を伸ばす」こと、すなわち**集めて育てる**である。MOBA 原作の SMITE は対人操作スキルが体験の核だったが、Blitz はオートバトラー化することで操作プレイの比重を下げ、代わりに**60 柱以上の神々という IP プール**をガチャ式に展開する設計に組み替えている。concept にある「収集×自動戦闘」「神話キャラ IP をモバイル収益化レイヤーへ展開」という記述がそのまま分類根拠で、プレイヤーが日々ログインして消費する時間の大半は「ステージ周回→素材獲得→神の昇格・昇級」というメタゲームに置かれる。

`secondary: ["EXP"]` が示すとおり、フォーメーション編成・スキル発動順・PvP マッチアップ読みの面では体験型の余地も残るが、神を引かなければ編成が組めない点で REW が一段上流に立つ構造になっている。social_axis は未設定（PvP モードはあるが主要動線ではない）。

## 報酬ループの組成

オートバトラー型ガチャ RPG のテンプレを踏襲し、原作 IP を素材にして以下の層構造で時間消費を引き伸ばす。

| 層 | 主な仕組み | プレイヤーが追う指標 |
|---|---|---|
| 収集 | 60 柱以上の神を確率排出（神話モチーフ別に分類） | 未所持神の埋め、レアリティ昇格素材 |
| 育成 | レベル・スター（昇格）・装備・スキル強化の多軸 | 戦力スコア、編成シナジー |
| 周回 | ストーリー章・チャレンジステージで素材回収 | スタミナ、デイリー進捗 |
| 競争 | PvP モード（オートバトル）でランキング | シーズン報酬、編成メタの読み合い |

この四層は ヒーローウォーズ / サマナーズウォー / Empires & Puzzles 系の北米欧州向けミッドコア・ガチャ RPG と同型で、SMITE という MOBA IP を上に被せただけ、と言ってよい。短時間オート戦闘 → 報酬画面 → 編成いじり、というセッションが 1 サイクル数分で回るため、通勤・休憩のスキマ需要に合わせ込まれている。

## バズ要因 / 人気の根拠

popularity に記されたとおり、2019 年に iOS/Android で**ソフトローンチ**された段階の作品であり、グローバル正式展開・売上ランキング常連化には至っていない。注目軸は IP とジャンル選定にある。

- PC/コンソールで一定の地位を築いた MOBA「SMITE」のスピンオフという**既存 IP からの流入導線**
- 当時北米欧州で伸びていたオートバトラー／ヒーローコレクション RPG に**ジャンル合わせ**しに行った Hi-Rez 自身のモバイル布石
- 世界神話（ギリシャ・北欧・エジプト・日本・マヤ等）の神々という**横断的にキャラが立つ題材**で、ガチャ排出の物語性を確保しやすい
- ストーリー／チャレンジ／PvP の三本柱という**当時のミッドコア・モバイル RPG の定番フォーマット**を素直に踏襲

具体的な DL 数や売上は公開されておらず（`sales_world: unknown`）、評価は「MOBA ファンのスマホ向け受け皿」という定性レベルにとどまる。

## 企画への示唆

- **PC/コンソール IP をモバイルへ降ろすときはジャンルを置換する**と、操作スキル依存度を下げて課金者層を広げられる（MOBA → オートバトラー）
- **世界神話のような既存パブリックドメイン IP**を素材にすると、キャラデザインの差別化と物語的フックを同時に得られる
- ソフトローンチで**先に北米欧州ミッドコア層に合わせ込む**と、サマナーズウォー／ヒーローウォーズ系プレイヤーの既存習慣に乗せやすい
- 60 柱規模の**初期キャラプール**を用意してから運用に入ると、ガチャ排出と編成パズルの組合せ爆発でリテンションを確保しやすい
- オートバトラー化で**1 セッション数分**に圧縮すると、本編 MOBA の 30 分試合とは別のスキマ時間需要を取りに行ける

## 比較で見るべきタイトル

- ヒーローウォーズ（Nexters）— 同型のオートバトラー型ガチャ RPG、北米欧州ミッドコアの代表
- サマナーズウォー（Com2uS）— 世界神話的モンスター収集 × ターン制 RPG の先行例
- AFK アリーナ（Lilith Games）— 放置型 × ヒーローコレクションで Blitz と同層を狙った
- Mobile Legends: Bang Bang Adventure — モバイル MOBA IP をオートバトラーに展開した同型スピンオフ
- Marvel Strike Force — 大型 IP × 編成 RPG のグローバル成功例
