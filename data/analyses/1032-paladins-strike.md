# Paladins Strike (2018) — Hi-Rez Studios

## なぜ EXP 主分類なのか

プレイヤーの主動機は「短時間で対戦して撃ち合う／勝つ」こと。PC 版 Paladins のチャンピオンを使った 5v5 トップダウン MOBA という設計上、コアの楽しさは試合中のスキル運用・連携・撃ち合いそのものにあり、`social_axis: 対戦` がそのまま体験の中心線となる。

`concept` が「短時間マッチ・スキル主導のチーム戦闘をモバイルで再現」と明示している通り、本作はキャラ収集や物語消費を目的にしていない。`target` も「PC 版 Paladins ファン＋Vainglory / Mobile Legends 等のモバイル MOBA プレイヤー」で、PvP 体験を求める層に向けて設計されている。`secondary: REW` はチャンピオン解放やカードシステム（スキル拡張）など、対戦体験を補強する周辺ループとして機能していたが、ガチャ駆動の収集ゲームではない。

## 体験の核 / 設計の発明

PC 版 Paladins（一人称ヒーローシューター）の世界観・キャラを保ったまま、視点をトップダウンに変換した点が中核の発明。FPS のエイム操作をモバイルの 2 本指タッチに置換し、Vainglory/Mobile Legends 系の MOBA UI に寄せることで「PC IP × モバイル MOBA 文法」を橋渡しした。

| レイヤー | PC 版 Paladins | Paladins Strike |
|---|---|---|
| 視点 | 一人称 FPS | トップダウン MOBA |
| 操作 | マウス＋WASD | 仮想スティック＋スキルボタン |
| マッチ尺 | 15-20 分 | 4-6 分の短時間制 |
| ロール | タンク/DPS/サポート/フランク | 同構造を継承 |
| カード | デッキで自キャラを拡張 | モバイル向けに簡略化して継承 |
| 課金軸 | スキン中心 | チャンピオン解放＋スキン |

社会軸「対戦」の効果は明快で、ラダーやチーム戦の勝敗が即時のフィードバックループを構成する。一方で 5v5 の同期 PvP はマッチング人口に依存し、後述のサ終要因と直結した。

## バズ要因 / 人気の根拠

- 2017 年に豪州ソフトローンチ、2018 年 5 月にグローバル配信。Hi-Rez にとって初の本格スマホ向け対戦タイトルとして英語圏メディア（PocketGamer.biz、Android Police、mmos.com 等）が一斉に取り上げた。
- PC 版 Paladins のキャラ IP をそのまま流用し、既存ファンを初期母数として確保。`sales_world: 1M+ DL` 規模に到達。
- Vainglory 衰退・Mobile Legends 急伸という 2017–2018 年のモバイル MOBA 過渡期に投入され、ジャンル文脈で注目された。
- ただしコミュニティ規模を維持できず、2020〜2021 年にサービス終了。マッチング人口が閾値を下回ると 5v5 PvP は成立せず、対戦型 EXP の脆さがそのまま結末に現れた。

## 企画への示唆

- 親 IP のキャラを子タイトルにそのまま持ち込むと、初期 DAU の獲得コストは下がるが「親 IP の体験」と差別化されないとどちらも食い合う。視点・尺・操作のいずれかを変えて補完関係を設計する。
- FPS をモバイルで再現するなら、エイム操作の負担を軽減できる視点変換（トップダウン化／ロックオン化）が有効。ただし「銃の手応え」自体が IP の核なら、変換コストが体験毀損を上回る。
- 5v5 同期 PvP は KPI として「マッチング待機時間」がリテンションの天井を決める。ローンチ時 DAU の半減で待機が 30 秒→2 分になると、ユーザーは戻らない。
- モバイル MOBA は「短時間化（4-6 分）」と「操作簡略化」の 2 軸で差別化されるが、両軸とも先行作（Mobile Legends・Brawl Stars）と被ると参入余地が消える。第 3 軸（IP・ロール構造・カードシステム等）を必ず立てる。
- 親 IP のキャラ拡張ペース（PC 版で半年に 1 体）とモバイル運営の更新サイクル（月 1 体）は本来不整合。子タイトル独自のキャラを追加できる運営権限を契約段階で確保すべき。

## 比較で見るべきタイトル

- Vainglory（Super Evil Megacorp, 2014）— モバイル MOBA 第一世代、同じくサ終
- Mobile Legends: Bang Bang（Moonton, 2016）— 東南アジアで成功した同時代作
- Brawl Stars（Supercell, 2018）— 同年同ジャンル、3v3 と短時間化で成功
- Pokémon UNITE（TiMi/Pokémon, 2021）— PC/家庭用 IP をモバイル MOBA に転用した後発成功例
- Paladins（Hi-Rez, 2016）— 親 IP
