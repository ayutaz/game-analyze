## なぜ EXP 主分類なのか

プレイヤーの主動機は「Marvel ヒーローを自分の手で操作し、5v5 のレーン戦で相手チームに勝つ」こと——つまり**競う体験そのもの**にある。

concept にある通り、本作は League of Legends / Mobile Legends 系の 3 レーン MOBA フォーマットに Marvel IP のスキル演出を被せた構造で、社会軸は明確に「対戦」。target が示すように Mobile Legends / Wild Rift プレイヤー層と MCU ファンの重なる層を狙っており、勝敗とランクマッチを核に駆動するので NAR（物語追体験）でも REW（収集主導）でもなく EXP が主分類になる。

secondary に REW を入れているのは、ヒーロー解禁・スキン購入・シーズンランク報酬という MOBA 標準の収集要素が副次的に効いているため。ただしガチャ型のキャラ獲得ではなく「ロスター解禁＋スキン課金」型なので、主動機は対戦そのものに留まる。

## 体験の核 / 設計の発明

本作の設計は「既存 MOBA フォーマット × Marvel IP の能力翻訳」という二層構造で読み解ける。

| レイヤー | 採用したフォーマット | Marvel IP で差し替えた要素 |
|---|---|---|
| マップ構造 | 3 レーン + ジャングルの MOBA 標準 | レーン演出に Marvel 世界観のアセットを上書き |
| 役割設計 | タンク / ファイター / アサシン / メイジ / マークスマン / サポート | アイアンマン＝マークスマン、ヴェノム＝ファイター 等、原作能力を役割に翻訳 |
| スキルセット | 4 スキル + パッシブの MOBA 慣習 | ウェブ・スイング、リパルサー、シンビオート変身など原作技を MOBA スキルに再構成 |
| マッチ尺 | 10〜15 分の短時間 5v5 | モバイル MOBA の標準尺をそのまま継承 |

発明があるとすれば**「MCU の能力をいかに MOBA の役割表に当てはめるか」というキャラ設計の翻訳作業**で、ゲームシステム側に新規メカニクスは少ない。social_axis「対戦」は MOBA の宿命であり、5v5 のチーム連携が成立した瞬間に IP 愛着が勝利体験へ転化する設計。逆に言えば、対戦バランスが崩れると IP 愛着だけでは引き留められない構造でもある。

## バズ要因 / 人気の根拠

- **Marvel IP 初の本格モバイル MOBA** という立ち位置。Marvel 公式と NetEase の協業で、原作ファンが「推しヒーローを自分で動かして勝つ」体験に直結できる稀少な企画だった
- アイアンマン / スパイダーマン / ヴェノム など MCU 映画の主要キャラが操作可能で、映画公開タイミングのファン需要を受け止められるポジション
- 展開地域を**東南アジア・台湾香港・日韓豪 NZ** に絞り、Mobile Legends がすでに地場 MOBA として浸透している市場へ「IP で差別化した代替案」として投下
- 一方で 2024 年 6 月にサービス終了。Mobile Legends / Wild Rift の固いユーザー基盤を切り崩せず、IP 単独では MOBA の競争原理（meta 更新・eスポーツ・ローカル大会網）を維持しきれなかった事実が示唆される

## 企画への示唆

- **既存ジャンルに IP を被せるだけでは長期運営は成立しない**：ジャンル王者（ML / Wild Rift）がいる市場では、IP の引力で初動は取れても meta 更新と大会網が後追いだと淘汰される
- **MCU キャラを MOBA 役割に翻訳する設計表**は、IP 横断企画のテンプレとして有効：能力→役割→スキルの 3 段マッピングを最初に固めると IP とゲームメカニクスの矛盾が減る
- **地域絞り配信（SEA + 台港 + 日韓豪 NZ）**は、地場 MOBA 文化が成熟した市場での「IP 差し替え版」戦略として参考になる。ただし王者が強い市場ほど、地域大会・配信者エコシステムへの投資が必須
- **ロスター解禁＋スキン課金モデル**は対戦の公平性を保ちつつ収益化する手段として MOBA 標準。ガチャ型を選ばなかった点は、対戦コア層の信頼維持に効く
- **IP ライセンス取得と運営期間のミスマッチ**に注意：IP 契約料を回収する前にサ終すると、原作側の信頼にも影響する。事前に「最低運営年数 × 必要 MAU」の損益分岐を IP ホルダーと合意しておくべき

## 比較で見るべきタイトル

- **Mobile Legends: Bang Bang**（SEA で本作が挑んだジャンル王者、地場文化と eスポーツ網の厚みが参考軸）
- **League of Legends: Wild Rift**（PC LoL の IP 引力を持つ正統派モバイル MOBA、IP × 完成度の上限値）
- **Pokémon UNITE**（IP × MOBA の別解、レーン構造を簡略化して IP ファン層に最適化した対照例）
- **Marvel Snap**（同じ Marvel IP の別ジャンル展開、対戦カードゲームに翻訳した成功例として比較）
- **Marvel Rivals**（後発の Marvel × ヒーローシューター、PC/コンソール路線で同 IP がジャンル選定をどう変えたかの参照点）
