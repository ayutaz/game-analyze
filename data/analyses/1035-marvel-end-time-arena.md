# Marvel End Time Arena (2018) — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「Marvel ヒーローを操作してレーン戦と集団戦に勝つ」こと、つまり**競う**動詞に収束する。MOBA というジャンル定義上、勝敗を分けるのはチャンピオン性能・チーム編成・マップコントロールの読み合いであり、ストーリーや恒常的なコレクション資産ではない。`social_axis: 対戦` がそのまま中核体験を表しており、IP（Marvel）はチャンピオン外装として乗っているに過ぎない。

concept にある「自陣拠点を守りつつ敵基地破壊を目指す」は LoL / HotS 系の枠組みそのもので、target も「LoL / HotS から派生先を探していた競技志向プレイヤー」と明示されている。`secondary: []` で副次動機が登録されていない点も、本作が IP コラボ的な NAR / REW 要素（章立て・ガチャ・キャラ収集）を主軸に置かなかった——あるいは置けないまま終了した——ことを示唆する。

## 体験の核 / 設計の発明

中核は「Marvel IP × 韓国産 MOBA」というレイヤリングだが、設計的な発明は限定的で、むしろ **マップサイズの可変化** に独自性を寄せていた。3v3 / 5v5 / 7v7 と 3 種類のアリーナを並走させる構成は、当時主流だった LoL（5v5 一本）や Heroes of the Storm（5v5＋複数マップ）に対し、人数とテンポを切り替える選択肢を提示している。

| 設計要素 | Marvel End Time Arena | LoL (2009-) | Heroes of the Storm (2015-) |
|---|---|---|
| 標準モード | 3v3 / 5v5 / 7v7 の併走 | 5v5 (Summoner's Rift) | 5v5 (複数マップ) |
| チャンピオン素材 | Marvel IP（ヒーロー・ヴィラン混在） | オリジナル | Blizzard 既存 IP 横断 |
| 個別レベル | あり（LoL 系） | あり | チーム共有レベル |
| 配信地域 | 韓国限定 | 全世界 | 全世界 |
| サービス期間 | 2017 OBT — 2019/02 終了 | 継続中 | 継続中 |

7v7 モードは集団戦の比重を強制的に上げる装置で、IP ファンが「アベンジャーズらしい大規模乱戦」を求める動機と接続できる設計だった。一方でチャンピオンプールやバランス調整のコストは人数比で増え、Marvel 側のライセンス制約も重なるため、運営難度は LoL より高い構造を初日から抱えていた。

## バズ要因 / 人気の根拠

- Marvel IP を冠した本格 MOBA という訴求軸。スパイダーマン / アイアンマン / ハルク / ヴェノム / キャプテン・アメリカ等、MCU 全盛期（2018 年は Infinity War 公開年）の主要キャラを操作できるという点が初動の話題を作った。
- 開発・運営が Smilegate（Crossfire を擁する韓国大手）であり、PC オンライン運営の体力がある会社からのリリースとして注目された。
- 3v3 / 5v5 / 7v7 のマップサイズ可変は、LoL 一強への差別化として MOBA コミュニティで言及された。
- ただし韓国国外への正式展開がなく、`sales_world: unknown (service ended)` のとおり 2019 年 2 月 25 日にサービス終了。話題性が IP に依存し、競技シーンや配信エコシステムを韓国外で構築できなかった点が短命の主因として記録される。

## 企画への示唆

- **IP を乗せても主分類は変わらない**：Marvel という強 IP を載せても、勝敗ループが主動機なら EXP のままになる。IP は外装、ループは骨格、と分けて設計すると分類迷いが減る。
- **マップサイズの可変化は競合差別化の安価な手**：人数違いの並走モードはチャンピオンの追加よりライセンス・バランス負荷が小さく、IP コラボ MOBA の差別化点として再利用可能。
- **強 IP × 競技ジャンル × 単一地域配信は寿命を縮める**：競技ジャンルは配信文化と eスポーツ階層の蓄積で延命する。配信地域を絞ると、IP の世界的認知度を寿命に変換できない。
- **MOBA は副次動機（収集・物語）を設計に組み込まないと IP 寿命を引き伸ばせない**：本作は `secondary: []` のままで、IP ファンを「対戦継続」だけで繋ぎ止める構造だった。コレクション層 / シーズンストーリー層を載せていれば、競技層が薄くても周辺需要で延命できた可能性がある。
- **ライセンス IP の運営は撤退コストを事前に設計する**：IP 利用権の更新・地域制約は運営判断より先に走ることがある。サ終時のプレイヤー資産扱い（払い戻し / コミュニティ移管）を ToS とロードマップに先付けしておく。

## 比較で見るべきタイトル

- **Heroes of the Storm**（Blizzard, 2015）：自社 IP 横断 MOBA、チーム共有レベルでカジュアル寄り。IP MOBA の参照軸。
- **Marvel Super War**（NetEase, 2019）：Marvel IP × モバイル MOBA。本作のサ終後に出た「同じ問題設定の別解」。
- **Gigantic**（Motiga, 2017）：5v5 MOBA、サ終後に Gearbox がリバイバル。短命 MOBA の再起ケース。
- **Paragon**（Epic, 2016-2018）：3v3 / 5v5 を試した TPS 寄り MOBA。マップ可変思想の同時代事例。
- **Smite**（Hi-Rez, 2014-）：神話 IP × MOBA。IP レイヤリングで長期運営に成功した対照例。
