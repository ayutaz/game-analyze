# DC Battle Arena (2021, Electronic Soul / WB Games)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「DC ヒーロー／ヴィランを操作して、短時間の PvP で他プレイヤーに勝つ」こと。物語追体験ではなく、毎試合のリアルタイム操作と勝敗そのものが遊びの核にある。

concept にある通り本作は「IP × MOBA / バトルロイヤル」のハイブリッドで、1v1 DUEL／3v3v3／サバイバルの 3 モードはいずれも対人戦を前提に設計されている。target も Brawl Stars 系の短時間モバイル PvP 層を明示しており、social_axis「対戦」が主動軸であることは設計面から裏付けられる。したがって主分類は EXP（対戦）が妥当。

secondary に REW を置いているのは、モバイル F2P である以上キャラクター解放／スキン／ランク報酬が継続動機として効くため。ただしガチャ収集自体が遊びの中心ではなく、あくまで対戦の周辺ループとして機能している。

## 体験の核 / 設計の発明

中核メカニクスは「IP の象徴的なキャラクターを短時間 PvP に流し込む」構造。3v3v3 という 3 陣営同時対戦は伝統的な MOBA（5v5）でもバトロワ（100人ソロ）でもない中間サイズで、モバイルのセッション時間に合わせて圧縮されている。

| 要素 | 設計の意図 | 競合との差分 |
|---|---|---|
| 1v1 DUEL | ヒーロー対決の象徴性を最短で消費 | 格ゲー寄り、Brawl Stars にはない |
| 3v3v3 | チーム対戦と漁夫の利を両立 | MOBA より速く、バトロワより濃い |
| サバイバル | バトロワ要素の輸入 | MOBA タイトルでは稀少 |
| DC IP ライセンス | キャラ認知でチュートリアル省略 | オリジナル IP の MOBA より参入摩擦が低い |

social_axis「対戦」の効果は、IP 認知を「キャラ性能差を学ぶコスト」の軽減に転用している点に表れる。スーパーマンを使えば飛行、バットマンならガジェットといった既知の特徴をスキルに割り付けることで、初見プレイヤーにも操作スキーマが伝わりやすい。

## バズ要因 / 人気の根拠

- DC 公式ライセンス（WB Games パブリッシング）を取得した数少ないモバイル PvP 専用タイトルとして英語圏メディアが取り上げた
- スーパーマン／バットマン／ワンダーウーマン／ハーレイクインら看板キャラを操作可能にしたことで、DC ファン層と MOBA／バトロワ層の両方を集客対象にできた
- 3v3v3 というモード設計が「MOBA とバトルロイヤルの中間」として話題化し、TapTap 等のアジア向けストアでの注目を集めた
- 累計 5M+ DL に到達。ただし海外メディアでは「lazy cash grab」と批判される評価も存在し、IP 認知に頼ったローンチであることが話題と批判の両面で機能した

## 企画への示唆

- ライセンス IP のキャラ性能を「既知の象徴（飛行／ガジェット／怪力）」にマッピングすると、チュートリアル工数を削減できる
- 5v5 MOBA を 3v3v3 に縮退させると、モバイルのセッション時間に収まりつつ「漁夫」要素で意外性を演出できる
- 1v1 / チーム / サバイバルのモード三層を用意すると、対戦疲れに対する逃げ場を自タイトル内に確保できる
- IP ライセンスは集客の初速を作るが、「IP に乗っただけ」と評されるリスクも同時に背負うため、コアメカ独自性の打ち出しが必須
- 開発元（Barbarq 系）のカジュアル対戦ノウハウを別 IP に載せ替える「エンジン横展開」モデルは、IP ホルダー側の F2P 拡張戦略として再現性がある

## 比較で見るべきタイトル

- Brawl Stars（Supercell）— 短時間モバイル PvP の参照点
- Pokémon UNITE — 大手 IP × MOBA の成功例、5v5 維持
- MultiVersus（WB Games）— 同じ WB が手掛けた IP クロスオーバー対戦、PC／コンソール版
- Vainglory — モバイル MOBA の先行事例、長時間モード偏重で衰退
- Marvel Super War — 競合する米国 IP × モバイル MOBA
