# ペルソナ5 スクランブル ザ ファントム ストライカーズ

## なぜ NAR 主分類なのか

プレイヤーの主動機は「P5 本編クリア後の怪盗団の後日譚を読む」ことである。本作は無双系アクションを採用しているが、購入動機・継続動機の大部分は本編キャラクター（ジョーカー、モルガナ、双葉、ハル、明智…）の半年後を確認したいという物語欲求に支えられている。concept にある通り「無双系アクションに載せ替えた外伝」であり、戦闘テンポは EXP 寄りに刷新されているが、章立て・新キャラクターの掘り下げ・本編伏線の回収という構造が前面に出るため、主分類は NAR が妥当である。

target が「P5 本編クリア済みのファン」に絞り込まれている点も NAR 性を補強する。本編未プレイ層には物語の機微が刺さりにくく、新規プレイヤーが本作から入る設計にはなっていない。secondary に EXP が入っているのは、戦闘システムが従来のコマンド式から無双式アクションに切り替わり、戦う体験そのものにも独立した面白さを持たせているためで、副次的に「戦う」動機が走るハイブリッド構造になっている。

## 物語の駆動構造

本編との連続性を保ちつつ、外伝としての独立性を確保するために以下の層が並走する。

| 駆動要素 | 機能 |
|---|---|
| ロードムービー構造 | 夏休みに全国を巡る road trip 形式で、都市ごとに新ジェイル（異世界）と新王（敵）を配置 |
| 新キャラクター | 善吉（刑事）・ソフィア（AI）の 2 名を本編怪盗団に挿入し、関係性ドラマを再駆動 |
| 本編後の関係性 | クリア済みプレイヤーへの「あれからの怪盗団」提示。竜司・春・明智不在等の状態変化を物語る |
| ジェイル＝認知世界の再利用 | 本編のパレス／メメントス構造を無双マップに翻訳、各 boss が「歪んだ大人」テーマを継承 |
| 章末カットシーン | 各都市攻略の前後にイベントシーンを密に配置し、アクション疲れを物語報酬で回収 |

カレンダーシステムやコープ（社交）といった本編 NAR の主要装置は意図的に外され、「夏休みの旅」という時間軸に圧縮されている。これは時間管理 SLG の手触りより、物語の連続体験を優先した判断である。

## バズ要因 / 人気の根拠

popularity「P5 本編の正統続編として怪盗団の後日譚を提供」を分解すると以下の通り。

- **本編 IP の引力**：P5 / P5R が国際的に評価された後の正統続編という位置づけが、本編ファンの購入動機をそのまま接続した。
- **海外売上の偏り**：sales_jp 240K に対し sales_world 2M+ と海外比率が圧倒的に高く、無双ジャンルが欧米でも JRPG 系 IP との組み合わせで一定の市場を持つことを示した。
- **コーエーテクモ（オメガフォース）との協業**：ペルソナ独自のスタイリッシュ UI とオメガフォースの群衆処理アクションを結合する技術的な話題性が、発売前の注目度を押し上げた。
- **プラットフォーム展開**：PS4 / Switch / PC 同時展開でアクセス障壁を下げ、本編プレイヤーの所持ハードに合わせて選べる体制を取った。

## 企画への示唆

- **本編 NAR の続編をアクション化すると**、コマンド戦闘で疲れていた層に「同じ物語をもう一度」を提供できるが、本編未プレイ層を取り込めないトレードオフが発生する。
- **既存 IP × 異ジャンル開発スタジオの協業**を組むと、IP の物語資産と協業先のメカニクス資産が掛け算になり、単独続編より話題化しやすい。
- **時間管理 SLG を外し road trip 構造に置き換えると**、本編の名物システムを失う代わりに、外伝としての軽さと完結性が確保できる。
- **新キャラクターを本編パーティに挿入すると**、既存キャラの関係性を再起動する触媒になり、ファンが「もう一度この関係性を読める」と感じる。
- **海外比率が国内の 8 倍以上になる作品では**、ローカライズ・同時発売・対応プラットフォームを国内基準で設計すると機会損失になる。
