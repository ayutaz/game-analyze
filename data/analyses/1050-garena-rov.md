# ROV (Garena RoV / Arena of Valor SEA)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「片手で 5v5 の MOBA を 15 分で勝ち切る」という対人戦闘の体験そのものである。チャンピオン収集やバトルパスはあるが、PC MOBA を片手モバイル UI に圧縮し、SEA 圏で eスポーツ文化と接続したという positioning を踏まえれば、主軸はガチャや収集ではなく「ランクを上げる／ライバルに勝つ」という競技体験である。

`social_axis: 対戦` は典型的な 5v5 同期 PvP であり、ソロランク・フレンドとのパーティランク・ギルド戦・プロリーグ観戦が一体化している点で、RoV のプレイヤーは「対戦」を中心にゲームを記憶する。`secondary: REW` はチャンピオン解禁・スキンガチャ・シーズンランク報酬としてリテンションを支えるが、これらは試合を回し続けるための補助燃料であり、ガチャを引くこと自体が目的化する FGO 型 REW とは構造が異なる。

## 体験の核 / 設計の発明

PC MOBA（LoL / Dota 2）の中核体験を、SEA のミドル〜ローエンド Android 端末と片手プレイ環境に「翻訳」したのが設計の発明である。1 試合 15 分の短縮、仮想スティック + スキルボタンの UI、オートロック気味の補助、低帯域でも崩れない通信モデルが、本家『王者栄耀』からそのまま継承されている。

| 設計要素 | PC MOBA からの圧縮 | RoV / 王者栄耀の解 |
|---|---|---|
| 試合時間 | 30–45 分 | 12–18 分（双方向のラスト押し速い） |
| 入力 | マウス + 100 キー | 仮想スティック + 4 スキル + 召喚士技 |
| マップ | 3 レーン + 広大ジャングル | 3 レーン + コンパクト ジャングル（モバイル画面比率に最適化） |
| 課金 | スキン中心 | チャンピオン解禁 + スキン + バトルパス |
| 観戦 | Twitch / 専用クライアント | アプリ内観戦 + YouTube / Facebook 配信統合 |

SEA 版固有の貢献は、`RoV Pro League` を国別（タイ・ベトナム・台湾・フィリピン・インドネシア）に分割し、地上波・モール大会・大学リーグまで降ろした「eスポーツのローカル分散」である。プレイヤーは試合する／観る／応援するの 3 動詞を 1 アプリ内で行き来でき、対戦の社会性が増幅される。

## バズ要因 / 人気の根拠

- 中華本家『王者栄耀』が世界 MAU 4 億人超に到達したエンジン（TiMi 開発）を Garena 配信網で SEA にローカライズし、累計 1 億 DL を突破。
- タイ・ベトナム・台湾では国別 Pro League が地上波／YouTube 配信込みで定着し、トッププロが芸能人化、若年層の進路選択肢として e スポーツが可視化された。
- 片手 15 分 1 試合という設計が、SEA の通学・通勤・カフェ滞在の時間スロットに合致し、ミドルレンジ Android 端末で動く軽量さがインフラ的にも有利だった。
- DC / 火影忍者 / 鋼の錬金術師 / ワンパンマン等のクロスメディア IP コラボを各国で展開し、地域文化と接続したスキン売上で長期運営の燃料を確保。

## 企画への示唆

- PC で完成したジャンルを「片手 + ミドル端末 + 15 分」の制約に翻訳すると、新規市場で代名詞ポジションを取れる（RoV はモバイル MOBA の SEA 代名詞）。
- ローカル eスポーツリーグを国別に分散運営すると、対戦アプリが「観戦・応援・地元プロ」の文化基盤に化け、ガチャ依存度を下げてリテンションが伸びる。
- 中華本家（王者栄耀）を地域別パブリッシャーに任せる「コア開発 × 地域配信」分業モデルは、SEA・南米・MENA など Tencent 自身が直接届きにくい地域での標準解になりつつある。
- チャンピオン解禁 + スキンガチャ + バトルパスの三層課金は、対戦の競技性を壊さず（pay-to-win 印象を抑え）長期収益を取りやすい。
- 観戦 UI をアプリ内に統合すると、配信視聴者がそのまま新規プレイヤーに転換しやすく、外部プラットフォーム頼みの集客から脱却できる。

## 比較で見るべきタイトル

- 王者栄耀（Honor of Kings / 中国本家）— 同エンジン、中華本土最大の MOBA。
- Mobile Legends: Bang Bang（Moonton）— SEA で RoV の最大競合、よりカジュアル寄り UI。
- League of Legends: Wild Rift（Riot）— PC MOBA 王者のモバイル翻訳、後発で SEA に切り込んだ。
- ポケモンユナイト — 5v5 モバイル MOBA を IP × Switch / モバイル両対応で構成。
- Brawl Stars（Supercell）— モバイル特化 3v3 / 即時性の極端な圧縮で別解を提示。
