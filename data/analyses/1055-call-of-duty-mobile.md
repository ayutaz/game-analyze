# Call of Duty: Mobile

## なぜ EXP 主分類なのか

プレイヤーの主動機は「スマホで本格的な FPS を撃ち合う」こと、すなわち反射神経と立ち回りで他者と競う体験そのものにある。concept にある通り、据置 CoD の TDM / ドミネーションと 100 人 BR をタッチ操作に最適化して再現した F2P FPS であり、target も「コンソール CoD や PUBG Mobile / 荒野行動の経験者で、出先でも撃ち合いを求める層」と明示されている。動詞は「撃つ／勝つ」であり、ガチャやストーリー消費が主動機ではない。

social_axis は「対戦」。マッチ単位の PvP が体験の中核を成し、観戦・協力・非同期はいずれも従属的な要素にとどまる。secondary に REW を置いているのは、シーズンパスとガチャ風武器スキンによる収益エンジンが体験の継続を支えているためで、撃ち合いという EXP の上に課金リテンションを乗せた二層構造になっている。

## 体験の核 / 設計の発明

中核はモバイル UI で据置 FPS の操作感を成立させる「移植の翻訳」と、複数モードの並列提供によるセッション長の柔軟化である。

| 層 | 設計要素 | 効果 |
|---|---|---|
| 操作 | 仮想スティック + Aim Assist + ジャイロ / 外部コントローラ対応 | コンソール経験者の移行コストを下げ、ライト層は補助で参入できる |
| モード | TDM / ドミネ等 5-10 分のクラシックと 100 人 BR の並列 | 通勤の隙間と腰を据えたセッションを同一アプリ内に内包 |
| 進行 | シーズンパス（無料 + 有料トラック）と週次イベント | 継続ログインの動機を撃ち合い体験の外側に確保 |
| 収益 | 武器スキンガチャ + バトルパス + キャラスキン | EXP の上に REW 層を載せ、無課金プレイの体験品質は維持 |

前世代（PUBG Mobile / 荒野行動）との差分は、BR 単独ではなく「クラシック CoD のマルチプレイ」を共存させた点にある。これにより 1 マッチ 20-30 分の BR とは別に、5 分の TDM で復帰動機を作れる。

## バズ要因 / 人気の根拠

- リリース初月で 1.48 億 DL を記録し「史上最大のモバイルゲームローンチ」と報じられた（Activision IR 公式）
- 初年度収益 4.8 億ドル、累計売上 15 億ドル超、累計 DL 6.5 億超に到達
- 日本は米国に次ぐ第 2 位の課金市場として継続的にランクインし、コンソール CoD の知名度がモバイル流入を後押し
- 地域別パブリッシング（Activision / Garena / Tencent / VNG）の分業で、東南アジア・中華圏・日本の各市場に最適化した運営を実現

## 企画への示唆

- 据置 IP をモバイル移植する際、UI を「縮小」ではなく「翻訳」（Aim Assist / ジャイロ / コントローラ）すると、コア層と新規の両取りができる
- 長尺モード（BR）と短尺モード（TDM）を同一アプリに同居させると、ユーザーの可処分時間に応じた DAU を確保できる
- 体験型 PvP の上にシーズンパス + スキンガチャを載せると、勝敗とは別軸の継続動機を確保しつつ Pay-to-Win 批判を回避できる
- グローバル F2P を地域別パブリッシャ分業で出すと、決済 / コミュニティ / プロモーションを現地最適化でき、課金 ARPU を底上げできる
- 既存 IP のキャラ / 武器 / マップを「コラボ素材」としてシーズンで回すと、IP 資産の摩耗を抑えながら新規話題を作れる

## 比較で見るべきタイトル

- PUBG Mobile（同 BR、Tencent 系、地域分業モデルの先行例）
- 荒野行動（日本特化 BR、コラボ多用で日本 ARPU を伸ばした対照例）
- Apex Legends Mobile（同コンソール IP のモバイル展開、2023 サービス終了で対照軸）
- Fortnite（クロスプラットフォーム + スキン経済、モバイル単独ではない比較）
- フォートナイト / Warzone Mobile（同パブリッシャの BR、共食い回避設計の参考）
