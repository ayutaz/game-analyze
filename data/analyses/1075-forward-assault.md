# Forward Assault (2017 / Blayze Games)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「スマホ／ブラウザでも Counter-Strike 的な 5v5 タクティカル PvP を競う」こと、すなわち**戦って勝つ体験**そのものにある。武器購入経済とラウンド制をモバイルに移植したコンセプト、Counter-Strike / CS:GO の競技性を外に持ち出したい 10–30 代男性という target、`social_axis: 対戦` の三つが揃っており、報酬収集や物語追体験ではなく「同じスキルを持つ他者と撃ち合う緊張」を売りにした作品なので EXP が妥当である。

ガチャ的な強武器や育成によるパワーカーブは中核ではなく、武器購入経済はラウンド内の戦術選択を駆動するためのリソース管理として機能する。secondary が空であることもこの単焦点設計を裏付けており、ランクマッチによる「同格の他者と当たり続けるマッチング」が体験の最終層となっている。

## 体験の核 / 設計の発明

CS 系の「爆弾設置／解除」ルールを、タッチ操作と短時間プレイ前提に削ぎ落としたのが本作の発明点である。具体的にはマップサイズ、ラウンド秒数、武器購入 UI の簡略化、リコイル制御の再設計といった「PC で成立していたルールを携帯端末上で破綻させない」ための調整に寄っている。

| 設計レイヤ | PC 版 CS 系 | Forward Assault の選択 |
|---|---|---|
| ルール骨格 | 爆弾設置 / 解除・経済・ラウンド制 | そのまま継承 |
| 操作 | キーボード＋マウスの精密エイム | タッチ + ジャイロ、リコイル軽減気味 |
| 1 試合の長さ | 30 分超のことも | ラウンド数短縮で 10 分前後 |
| 入口 | Steam / 専用 PC | iOS / Android / Browser（Remix） |
| 競技層導線 | 公式大会・サードパーティ | ランクマッチ + クラン / スクリム |

social_axis「対戦」は、5v5 という人数・経済管理・索敵情報のメタが噛み合うことで「敵 5 人の購入傾向を読む」「味方とリソースを共有する」ミクロ協調を生む。ブラウザ版 Forward Assault Remix の存在は、インストール障壁を下げて対戦人口（=マッチング品質）を担保する装置として機能している。

## バズ要因 / 人気の根拠

- 「スマホで CS が遊べる」という一行で説明できるポジショニング。既存 IP の競技体験を別プラットフォームへ移植するパターンとして分かりやすい。
- 累計 10M+ DL。インディースタジオ（Blayze Games）の単一タイトルとしては十分大きく、サーバー人口が成立する規模を確保した。
- ブラウザ版 Forward Assault Remix の派生によって、アプリインストールを嫌う層・PC 学校端末層など追加導線を開拓。
- ランクマッチとクラン／スクリム機能で「フレンドと組んで上を目指す」という、コア対戦層が求める縦のモチベーションを実装している。

## 企画への示唆

- PC で確立した競技ルールを別プラットフォームへ移植する場合、ルールは触らず**入力と試合長だけを再設計**すると IP 連想を維持しながら新規層を獲れる。
- 5v5・ラウンド制・購入経済のような「マイクロ経済 × チーム戦」は、短時間で意思決定密度を出せるためモバイル / ブラウザでも体験が薄まりにくい。
- アプリ版とブラウザ版を併存させると、インストール障壁の異なる層を同じマッチングプールに合流させられ、対戦ゲームの寿命を伸ばせる。
- ガチャや強武器ではなく**ランクと装備購入のリソース管理**を縦軸に置くと、競技性ファンを長く保持しつつ pay-to-win 批判を避けられる。
- インディースタジオでも「既存メガ IP のサブセット体験を別環境で完結させる」一点突破であれば、10M DL 級のスケールに到達しうる。

## 比較で見るべきタイトル

- Critical Ops（モバイル向け CS クローン、同じ 5v5 爆弾設置軸）
- Standoff 2（モバイル CS 系、リコイル設計とスキン経済の比較対象）
- Counter-Strike 2 / CS:GO（移植元となるルール骨格と競技 UX）
- Valorant Mobile（同コンセプトの大手版、開発体制とローンチ規模の対比）
- PUBG Mobile（CS 系ではないが、PC 競技 FPS をモバイルへ移植した先行成功例）
