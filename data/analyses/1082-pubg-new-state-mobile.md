## なぜ EXP 主分類なのか

プレイヤーの主動機は「100 人の生き残り戦で最後の 1 人/1 部隊として撃ち勝つ」こと、つまり一回ごとに完結する対戦の操作体験そのものである。concept にある「弾道学を取り入れた撃ち合い」「スマホタッチ操作向けに最適化」という記述が示す通り、本作の中心は AIM とポジショニングの瞬間判断であり、`social_axis: 対戦` の通り 100 人の他プレイヤーとの相互作用が体験を駆動する。

target の「PUBG Mobile から派生作品を求める層」「ハイグラフィックなモバイルシューターを遊びたいコアゲーマー層」も、物語消費ではなく操作の手応えを求める層である。secondary に REW が入るのは、シーズンパス・銃器スキン・キャラ装備のガチャ要素がリテンションを補強しているためで、主動機ではないが収益と長期継続を支える副軸として機能している。

## 体験の核 / 設計の発明

| 設計要素 | PUBG Mobile（2018） | New State Mobile（2021） |
|---|---|---|
| 世界観 | 現代戦・近未来兵器無し | 2051 年・ドローン/EMP 手榴弾/装備カスタマイズ車両 |
| 銃器カスタマイズ | アタッチメント差し替え中心 | 試合中にカスタマイズキットで銃口/サイト/グリップを動的変更 |
| グラフィック | UE4・モバイル最適化軽量寄り | グローバルイルミネーション活用の高負荷ハイエンド向け |
| マッチ密度 | 8x8km 中心 | 8x8km の新マップ TROI を主軸に短時間化を意識 |

社会軸「対戦」の設計効果は二重で、(a) 1 試合 25〜30 分の決着で「もう 1 戦」のリプレイ動機が生じ、(b) スクワッド時のボイス連携が口コミ拡散と配信映えを生む。スマホ単体で重量級 FPS を回す技術的挑戦自体が本作の発明であり、PUBG IP の延命線として「現代戦 = PUBG Mobile / 近未来 = New State」のラインアップ分業を成立させた。

## バズ要因 / 人気の根拠

- 事前登録 5500 万人超は、Krafton が PUBG IP の正統続編としてグローバル同時展開した結果であり、ローンチ前から「PUBG Mobile を超える次世代版」というナラティブが形成されていた
- ローンチ後 1 か月で全世界 5000 万 DL を突破。短期での DL 爆発はバトロワジャンルがすでに市場に定着していたことの裏返しでもある
- 日本では渡辺直美を公式アンバサダーに起用し、コアゲーマー層外への認知拡張を狙った（FPS/バトロワ作品のローカルマーケティングとしては大胆な人選）
- 近未来 2051 年設定は実在地名に縛られず、ドローン/EMP/車両カスタマイズなど SF ガジェットを正当化する設計上のチケットとして機能した

## 企画への示唆

- 同一 IP で「現代戦版」と「近未来版」を並走させると、ユーザーを取り合わずプレイ動機（リアリズム vs ガジェット遊び）で住み分けが起きる
- バトロワに「試合中の動的カスタマイズ」を入れると、序盤の物資漁りに加えて中盤以降のビルド構築局面が増え、撃ち合いに至るまでの意思決定密度が上がる
- モバイル FPS でハイエンド端末を前提に組むと、ライト層を逃す代わりに配信映え・スクリーンショット拡散の上限が引き上がる（端末スペック分布の読みが収益を左右する）
- ローカライズ時に「コア層外のタレント」を公式アンバサダーに据えると、ジャンル未経験者の入口になり、初期 DL の天井が押し上がる
- 近未来年号（2051 年など固有値）を世界観の前提に置くと、後続シーズンで新ガジェット・新マップを追加する正当化コストが下がる

## 比較で見るべきタイトル

- PUBG Mobile（2018, Krafton）— 本作の前世代・同 IP 現代戦版
- Apex Legends Mobile（2022, EA/Respawn）— 近未来バトロワ × ヒーローシューターのハイブリッド
- Call of Duty: Mobile（2019, Activision/TiMi）— モバイル FPS のもう一方の巨頭、マルチモード戦略
- 荒野行動（2017, NetEase）— 日本市場での先行バトロワ、IP なしでローカル定着した対照例
- Fortnite Mobile（2018, Epic）— 建築要素ありの異種バトロワ、SF 世界観の長期運用例
