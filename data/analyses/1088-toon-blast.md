# Toon Blast / トゥーンブラスト 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「数千ステージを順番に踏破し、次のステージを開放する進捗報酬を集める」ことにある。同色ブロックをタップして消すだけの単純操作それ自体は数秒で飽きるはずだが、Peak Games はライフ制（失敗で 1 ライフ消費・時間回復）と数千レベルの線形進行を組み合わせ、「クリア → 次のステージ番号が増える → 詰まる → 課金 or 待機」という王道カジュアルループに置き換えている。`concept` が明示するとおり、提供価値はパズルの体験ではなく、ステージ番号という可視化された進捗の蓄積である。

`target` が示すライト層・非ゲーマー寄りユーザー、特に女性比率の高いプレイヤーにとって、Candy Crush 系と同様に「達成感を短時間で確実に得る」ことが利用動機になっており、`secondary: ["EXP"]` のタップマッチそのものの気持ちよさは入口装置に留まる。`social_axis` が null なのは、チーム要素（ギフト送付・週次ランキング）が補助的で、本作のコアが個人の進捗蓄積にあることを反映している。

## 報酬ループの組成

| 層 | 仕掛け | プレイヤーへの効果 |
|---|---|---|
| 基層 | 同色 2+ タップ消し、4/5/7+ で特殊ブロック生成 | 操作が単純で誰でも 10 秒で理解できる |
| 進行層 | 数千ステージの線形マップ・星アイコンによる達成可視化 | 「次のステージ番号」が報酬通貨化 |
| 摩擦層 | ライフ制（5 ライフ・30 分回復）と難所ステージ | 詰まり点で課金 or 離脱の二択を強制 |
| 課金層 | コイン購入・ブースター・追加手数・ライフ無限 | 詰まり解消の即時購入で ARPPU を押し上げ |
| 社会層 | チーム参加、ライフ・コイン送付、週次トーナメント | 離脱抑制と再訪フック、ただし対戦性は薄い |
| 演出層 | ピクサー風 3D カートゥーン（ネコ・クマ・オオカミ）の表情・効果音 | パズル成功体験を「気持ちいい映像」に変換 |

カジュアルパズルの定石である「短時間 → 詰まり → 課金」を、3D 演出と数千ステージというボリュームで延命している。Candy Crush Saga が 2D マッチ 3 で構築した同じ構造を、タップマッチ（同色隣接消し）に置き換えたのが Peak の発明である。

## バズ要因 / 人気の根拠

- 世界累計 150M+ DL 規模で、リリースから 9 年経った 2026 年時点でも週間推定売上 2 億 8000 万円規模を維持する長期運営型ヒット。
- 2020 年に Zynga が Peak Games を 18 億ドルで買収した主要因が本作と姉妹作 Toon Blast / Toy Blast の安定収益で、後の Take-Two による Zynga 買収（127 億ドル）の収益基盤の一部になった。
- ピクサー風 3D カートゥーン演出は、当時の 2D マッチ 3 主流 (Candy Crush / Homescapes) との視覚的差別化として機能し、非ゲーマー層に「親しみやすい」印象を与えた。
- TV CM・SNS 動画広告で「ステージ◯◯◯◯まで進めますか？」型のチャレンジ訴求を大量投下し、UA（ユーザー獲得）面でもカジュアルパズルカテゴリのベンチマーク的存在になった。

## 企画への示唆

- パズルの操作を最小単位（タップ 1 つ）まで削ぎ落とすと、ライト層のオンボーディングコストが消える代わりに、進捗の可視化（ステージ番号・星）が報酬の中核を担う必要が生まれる。
- ライフ制（失敗 → 時間回復）を入れると、課金転換点が「詰まったステージのリトライ」に集約され、設計者は難易度カーブの山谷を意図的に作るだけで ARPPU をコントロールできる。
- 既存ジャンルの王道ループをそのまま借り、演出（3D・キャラクター・効果音）だけ差し替える戦略は、UA 効率と差別化を両立しやすい。
- チーム要素は対戦性を持たせるより「ライフ送付・週次トーナメント」程度に留めた方が、ライト層を逃さず再訪フックとして機能する。
- 数千ステージの線形マップは「いつでも辞められるが、辞めると番号が増えない」という損失回避バイアスを利用しており、サブスク的な定着を生む。

## 比較で見るべきタイトル

- Candy Crush Saga（King）— 同じ「ライフ制 + 数千ステージ + マッチパズル」の元祖、2D マッチ 3
- Toy Blast（Peak Games）— 同社の姉妹作、Toon Blast の前身でタップマッチの原型
- Homescapes / Gardenscapes（Playrix）— マッチ 3 + メタゲーム（家のリフォーム）で進捗を二層化した派生型
- Royal Match（Dream Games）— 2021 年以降に世界トップに躍進した次世代マッチパズル、広告クリエイティブの設計が参考になる
