## なぜ REW 主分類なのか

プレイヤーの主動機は「料理素材をマージして合成し、レストラン再建のためのリソースを積み上げる」こと、つまり**収集と合成の進捗を貯めること**にある。タップごとに進む合成と、エネルギー消費 / 待機 / 課金で再開という構造が、典型的なソーシャル課金パズルの報酬ループそのものになっている。

`concept` にあるとおり Microfun は「決済 UI と広告 LTV を徹底的に磨き込んだ」運営型タイトルとしてこの作品を組んでおり、収益指標（DAU 540 万 / 月間最大 $70M+）から見ても、KPI 設計の中心はリテンションと ARPDAU の最大化＝REW 軸である。`target` の「20-40 代女性カジュアル層 / 米国売上の約 4 割」はマージ系課金主力層と一致する。

`secondary` の NAR は副次として効いており、ヒロイン Quinn を巡る昼ドラ的展開（離婚・裏切り・毒殺）が「次の章を読みたいから素材を貯める」動機の上塗りになっている。社交軸は基本ソロプレイで、`social_axis` は null のまま定義されているのもこの構造に即している。

## 報酬ループの組成

| 層 | 仕掛け | プレイヤー体験 |
|---|---|---|
| 短期ループ | エネルギー消費 → マージ → 素材完成 → ボード整理 | 数秒〜数十秒の即時フィードバック |
| 中期ループ | クエスト達成 → コイン / 経験値 → エリア改装 | 1〜数日でレストランが変化 |
| 長期ループ | エリア完成 → 章解放 → ストーリー進行（ゴシップ解明） | 数週間〜数ヶ月のリテンション |
| 課金層 | エネルギー / ジェム / 期間限定パック / イベント | 進行詰まりの解消売り |

マージ操作は短期報酬、レストラン改装は中期報酬、ゴシップの章解放が長期報酬で、3 階層が時定数を変えて並走する。広告とパック販売は「短期ループが詰まった瞬間」に差し込まれ、解消すると一気に複数素材が完成する設計でカタルシスを生む。NAR は最長階層の出口に置かれ、`secondary: ["NAR"]` の役割は「最後まで遊ぶ理由の供給」である。

## バズ要因 / 人気の根拠

- 2024 年後半に Merge Mansion / Travel Town を抜き、マージ系世界首位に到達。2025 年は累計収益 $677M+、単年 $650M+
- 1 日 $300 万を超える日があり、Gardenscapes を抜く売上 3 位級まで上昇
- DAU 約 540 万 / MAU 約 2800 万。米国が売上の約 4 割を占める
- 「離婚・裏切り・毒殺」の昼ドラ的ストーリーと、料理素材マージという軽量操作の組み合わせが SNS 上でショート動画化されやすい

## 企画への示唆

- マージ系の素材を「料理 / レストラン改装」というドメインに寄せると、女性カジュアル層の文脈接続が一気に良くなる
- 昼ドラ調のドロドロ群像劇を最長ループの出口に置くと、マージ操作という単調行動の動機が章単位で再生成される
- 短期 / 中期 / 長期で時定数の異なるループを 3 階層並走させると、課金訴求ポイントが自然に分散し ARPDAU が上がる
- 課金 UI と広告 LTV を「機能」ではなくゲーム設計の骨格として最初から組むと、後付け収益化より CPI 耐性が高くなる
- ヒロイン視点で「町ぐるみのゴシップを解いていく」フォーマットにすると、各章のクリフハンガーがそのまま継続インストールの理由になる

## 比較で見るべきタイトル

- Merge Mansion（Metacore） — マージ系の前世代代表、ミステリー系ストーリー
- Travel Town（Magmatic Games） — 同ジャンルのデザイン洗練度の参照
- Gardenscapes（Playrix） — 改装系ストーリー × カジュアル課金パズルの先行モデル
- Royal Match（Dream Games） — 同時期のカジュアル課金パズル収益首位、UA / LTV 設計の比較対象
