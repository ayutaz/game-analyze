# トラベル・タウン (Travel Town) 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「マージで段階的に上位アイテムを**集めて**街と住人クエストを片付ける」ことであり、報酬獲得が動詞の中心にある。concept にある通り、ゲームの基本動作は 2 つのアイテムをドラッグして合体させ、より高位のアイテムへ進化させていく「マージ 2」ループである。住人クエストの達成という外的目標と、アイテム進化ツリーの埋め戦という内的目標が二重に報酬を出し続ける構造で、ガチャや戦闘よりも「収集 → アンロック」というモバイル定番の REW 設計に近い。

target が「20-50 代女性のカジュアル層、短時間で達成感が欲しい層、Gardenscapes / Royal Match 等の街再建系に親しんだ層」と明確で、社交ではなくソロでの達成感が主軸であるため REW が妥当。`secondary: ["EXP"]` はマージ操作そのものに触り心地（タップ・ドラッグ・連鎖）の体験価値が乗っていることを示しており、ガチャ単体型ではなく操作快感を伴う REW という位置付け。`social_axis: null` の通り対人要素は実装の中心ではない。

## 報酬ループの組成

マージ 2 系の特徴は「エネルギー消費 → 素材出現 → マージ → クエスト納品」というループの各層に独立した報酬と詰まりが設計されている点にある。Travel Town はこのループに街再建のメタ報酬を重ねている。

| 層 | 入力 | 出力 | 詰まりと解消 |
|---|---|---|---|
| マージ操作層 | タップで原石を生成 | 同種 2 個 → 上位 1 個 | 盤面が埋まる → 売却 or マージで圧縮 |
| エネルギー層 | 時間回復 or 課金 | 原石生成回数 | 枯渇 → 待機 / 広告視聴 / 課金 |
| クエスト層 | 完成アイテム | コイン・経験値・住人解放 | 必要 tier が遠い → 上位アイテムへ進化 |
| 街再建層 | コイン蓄積 | 建物アンロック・景観変化 | 進行のご褒美として可視化 |
| イベント層 | 期間限定盤面 | 限定アイテム・装飾 | 月次で再導入されリテンション維持 |

55 人の住人クエストは「次に何を作るか」のキューを常に提示し、tier の遠いアイテムは複数段階のマージが必要なため自然と長時間プレイに誘導される。エネルギー制で 1 セッションが 5-10 分に区切られる設計が、可処分時間の短い対象層と噛み合っている。

## バズ要因 / 人気の根拠

- 世界累計 25M+ DL、2024 年に年間 1.17 億ドル超の売上を計上し、マージ 2 ジャンル世界トップクラスに到達。
- 元 Wooga 出身者の Magmatic Games が 2021 年にリリースし、2022 年に Coin Master の Moon Active が買収。マネタイズと UA に強い publisher 配下で急成長した。
- 美麗な等角ビジュアルと海辺の街という世界観が、Gardenscapes / Royal Match 等の街再建系に親しんだ層との互換性を持ち、ユーザー獲得広告のクリエイティブが効きやすい。
- マージ 2 自体は EverMerge / Merge Mansion 等の先行作で市場が温まっていたジャンルであり、後発ながら盤面体験とクエスト導線の磨き込みで王座を奪取した。

## 企画への示唆

- カジュアル層を狙うなら、メインループに「触って気持ちいい単一動作」(マージ) を据えると説明コストが下がり広告 CVR が上がる。
- クエスト発注者を 55 人規模で配置し、常に複数キューを提示すると「次にやることがない」状態を構造的に消せる。
- エネルギー制で 1 セッションを 5-10 分に切り、可処分時間の短いユーザー層に合わせると DAU の頻度を稼ぎやすい。
- 街再建というメタ進行を盤面の外に置くと、マージ単体では飽きるプレイヤーに「コインを貯める動機」を二重で与えられる。
- 後発ジャンル参入でも、UI のタッチ感・イベントの月次回転・住人キャラの個別性を磨けば、先発 IP を市場規模で追い抜ける（Merge Mansion を逆転した本作の事例）。

## 比較で見るべきタイトル

- Merge Mansion (Metacore, 2020) — マージ 2 のジャンル先行作。Travel Town に売上で逆転された比較対象。
- EverMerge (Big Fish Games, 2020) — 同じく初期のマージ 2、ファンタジー世界観。
- Gardenscapes / Homescapes (Playrix) — 「街再建メタ + カジュアルパズル」構造の源流。
- Royal Match (Dream Games) — target が重なるカジュアルパズルの近年王者。
- Coin Master (Moon Active) — 同じ publisher、エネルギー制と長期運営のマネタイズノウハウの参照先。
