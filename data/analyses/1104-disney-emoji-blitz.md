# ディズニー絵文字ブリッツ — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は **「ディズニー/ピクサー/スター・ウォーズの絵文字キャラを集める」** こと。マッチ3を解く行為そのものよりも、解いた結果として手に入る絵文字（コレクション枠）とそのアップグレードが反復動機の中心にある。

`concept` が示す通り、本作は「90 年分のディズニー IP を絵文字化して揃える」ことを設計の出発点に置いており、マッチ3はあくまで **収集の蛇口を回すための作業ループ** として機能している。`target` がディズニー IP に親しみのある女性層・ファミリー層を主対象としていることからも、ゲームの勝敗ではなく「推しキャラの絵文字が出るか」「自分のメッセンジャー上で使えるか」が継続の核になっていると判断できる。`social_axis` が `null` で対人要素を持たない典型的なシングルプレイヤー REW として運用されている。

副次の **EXP** は、マッチ3パズルそのもののテンポと「キャラ固有スキル」の発動気持ちよさを担う。盤面が爽快に弾けないと収集動機だけでは数百日継続しないため、EXP 要素はリテンションの基礎体力として効いている。

## 報酬ループの組成

| 層 | 要素 | プレイヤーへの効用 |
|---|---|---|
| 入手 | ボックスガチャ（絵文字本体） | 400+ 体から「推し」が来るかのスロット体験 |
| 育成 | 絵文字レベル / スキルランク | 重複絵文字を糧にスコア倍率を上げ、より高ステージへ |
| 収集 | アルバム / シリーズ別コレクション | 「ピクサー全揃え」「ヴィランズ全揃え」の達成欲 |
| 外部出力 | メッセージング用絵文字 keyboard | ゲーム外で使える「実用報酬」が課金正当化に効く |
| 期間限定 | 新作映画タイアップ / 季節イベント | ディズニーの公開スケジュールがそのまま新規 SKU 化 |

ループの特徴は **「外部出力レイヤー」がある** こと。一般的なマッチ3収集は集めた絵柄がゲーム内で完結するが、本作は集めた絵文字を iMessage 等のキーボードに書き出せる。これによってゲームを起動していない時間にも報酬の効用が及び、課金の心理障壁を下げている。Disney の作品公開カレンダーがそのままイベントカレンダーに直結しているため、コンテンツ供給を内製で抱え込まずに済む点も長寿運営の構造的優位。

## バズ要因 / 人気の根拠

- 世界累計 **50M+ DL** を 2016 年から積み上げ、マッチ3カテゴリで長期に渡り MAU を維持
- Disney/Pixar/Star Wars の **400 体以上**の絵文字キャラを横断的に扱える数少ない公式マッチ3として、IP の網羅性そのものが差別化要因
- 2019 年に開発元 Jam City が **Disney 内製スタジオごと買収**し複数年ライセンス契約を結んだことで、IP 利用の継続性とアップデート速度が保証された
- 新作ディズニー映画の公開タイミングに合わせた**期間限定キャラ追加**が、IP ファンの定期的な再ログインを駆動

## 企画への示唆

- **ゲーム外で使える報酬を仕込むと、未起動時間にも課金を正当化できる**（絵文字 keyboard のような実用出力）
- **IP 保有者のカレンダーをイベント供給に流用すると、運営側のコンテンツ製造コストが構造的に下がる**（新作映画 = 新キャラ）
- **収集ガチャの主役を「キャラ立ち絵」ではなく「絵文字（小サイズ）」にすると、SKU 量産コストが極小化される**
- **マッチ3の「キャラ固有スキル」は EXP を担保するだけでなく、重複入手の使い道（スキルアップ素材）として育成ループに直結させる**
- **IP の世界観横断（Pixar × Star Wars × Disney 本流）を 1 つのアルバムに統合すると、IP 単独では届かない総コレクション欲が起動する**
