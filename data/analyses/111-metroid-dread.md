# メトロイド ドレッド (Metroid Dread)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**追跡を振り切りながら迷宮を読み解く**」という運動と空間認知の体験そのものにある。物語（サムスとX寄生体の続編）や能力獲得による段階的アンロックも存在するが、それらは EMMI の追跡圏内をパリィ・スライディング・フリーエイムでくぐり抜ける**ハンドリング体験**を成立させるための舞台装置として機能している。

concept に明記される「EMMI に追跡される緊張感」と「アクション拡張」は、操作中の毎秒の判断密度を最大化する設計であり、目的は所有や物語消費ではなく**身体感覚としての到達**だ。target が「高難度2Dアクションと迷宮探索を好む層」に絞られていることも、報酬収集型（REW）や物語追体験型（NAR）ではなく、運動の熟達を求める EXP プレイヤーを想定していることを裏付ける。social_axis は「ソロ」で、他者軸の交流は持たず、自分とマップ・敵 AI のみで完結する純度の高い体験設計になっている。secondary は空で、副次動機を意図的に持たせていない。

## 体験の核 / 設計の発明

ドレッドの中核は、メトロイドヴァニアの定石「能力獲得 → 探索範囲拡大」に、**EMMI ゾーンによる時限的恐怖**を上から重ねた二層構造にある。前世代（『フュージョン』2002、『サムスリターンズ』2017）からの差分を分解する。

| 要素 | 旧2D本編 | ドレッドの拡張 |
|---|---|---|
| 移動 | ジャンプ・モーフボール中心 | フリースライド・フラッシュシフト追加で接地中も連続回避可能 |
| 防御 | ほぼ無し（被弾前提） | メレーカウンター（パリィ）が探索の基本動作に昇格 |
| 緊張源 | ボス戦・ヒットポイント管理 | EMMI ゾーンの「一撃死＋追跡」が常時の緊張源として常駐 |
| マップ | プレイヤーが手書きで覚える | 一方通行ゲートと色分けで誘導しつつ、ショートカット発見余地は維持 |
| 探索動機 | アイテム取得 | アイテム取得＋ EMMI 撃破後の安全圏拡大 |

「ソロ」軸の純度を保ちながら、AI（EMMI）が他者の代替として恐怖と緊張を供給する点が発明的で、対戦相手不在でもプレイヤーの心拍数を上げる装置になっている。

## バズ要因 / 人気の根拠

popularity の「19年ぶり2D本編、シリーズ最大ヒット」を分解すると次の通り。

- **19年の空白を埋める供給**：2D本編は『フュージョン』(2002) 以来で、休眠 IP の完全新作という事象自体が話題化のフックになった
- **シリーズ最大ヒット**：世界販売 3M+ / 国内 0.3M は 2D メトロイド史上最大規模で、3D の『プライム』系を含めても上位に位置する
- **MercurySteam の継続起用**：『サムスリターンズ』(3DS, 2017) で実績を積んだ外部スタジオに任せ、Nintendo EPD との二人三脚で「期待値設計済みの新作」として出荷
- **配信映え**：EMMI 追跡シーンと一撃死のリアクションが切り抜き・実況と相性が良く、コアゲーマー以外にも視覚的に伝わる緊張感を提供

## 企画への示唆

- **休眠 IP の復活は「形式の踏襲＋一個の新装置」で十分**に成立する → ドレッドはメトロイドヴァニアの基本構造を守り、EMMI ひとつを新装置として置いただけで「シリーズ最大ヒット」に届いた
- **AI を“他者軸の代替”として使う**と、ソロゲームでも緊張感を社会的体験に近づけられる → EMMI は対戦相手の心理的役割を肩代わりしている
- **能力アンロックに防御系（パリィ）を組み込む**と、習熟曲線がボス戦だけでなく道中にも分散し、密度が均される
- **一撃死ゾーンを「区画化」する**と、難易度を局所最大化しつつマップ全体の探索快適度を維持できる → EMMI ゾーンの外では従来通り探索を楽しめる
- **ターゲットを 20-40 代コアに絞る**と、難易度を妥協せずマーケティング・難度・UI 設計が一貫し、結果として広いライト層にも“硬派ブランド”として届く

## 比較で見るべきタイトル

- **メトロイド フュージョン** (2002) — 前作 2D 本編。SA-X による追跡という EMMI の原型を持つ
- **メトロイド サムスリターンズ** (3DS, 2017) — MercurySteam 初参加作。メレーカウンターの導入元
- **Hollow Knight** (2017) — 同ジャンル代表。広大なマップ × 高難度ボスのソロ EXP として比較対象
- **Dead Space** (2008) — 追跡型恐怖を 3D アクションに移植した先行例。EMMI の設計思想と通底
- **Returnal** (2021) — 同年発売のソロ EXP。死を中核体験に置く点でドレッドと並走
