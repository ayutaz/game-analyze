# Triple Match 3D 分析

## なぜ EXP 主分類なのか

このタイトルを支える主動機は「探す → 一致を見つける → 盤面が整っていく」という**視覚処理と整理整頓の手触り**である。プレイヤーは物語を読むためでもガチャを引くためでもなく、3D で積み重なったオブジェクトのなかから同じ形 3 個を脳内検索し、トレイへ運ぶ操作そのものを楽しむためにアプリを開く。動詞は「探す／揃える／片付ける」であり、報酬よりも操作の体感が先に立つ。

`social_axis: ソロ` はこのジャンルの構造的必然で、完全オフラインで他者の介入なく完結する。一方で `secondary: REW` は副次的に強く効いており、ステージクリア時のテーマアンロック、ブースター（タイル戻し・シャッフル・タイマー追加）課金、フルーツ／玩具／旅行小物といったコレクションテーマがリテンションを底支えしている。`target` に挙がる「20-50 代の女性層中心」「スキマ時間の脳トレ・整理整頓系を好むプレイヤー」は、操作の心地よさが主動機であることの傍証である。

## 体験の核 / 設計の発明

中核は **Triple Tile 系（三麻雀 / Match 3D）フォーマットの 3D ビュー化**にある。2D Match-3 が「隣接スワップ」の発明だったのに対し、Triple Tile 系は「離れた 3 個を 1 つのトレイに集める」発明で、視覚的な探索難度を意図的に上げた。本作はそこに 3D 視点を被せ、回転・遮蔽・奥行き手がかりを操作レイヤーに足している。

| 設計レイヤー | 具体要素 | プレイヤー側の効果 |
|---|---|---|
| 知覚 | 3D 配置・回転・遮蔽 | 「同じ 3 個」検索を空間認知タスク化 |
| 操作 | タップでトレイへ送る | 失敗ペナルティが軽く再試行が速い |
| 制約 | トレイ 7 枠・制限時間 | 詰みリスクで集中を強制 |
| 救済 | ブースター（戻す／シャッフル／時間） | 課金/広告で詰みを解消 |
| 装飾 | フルーツ・玩具・旅行テーマ | 飽きを遅らせるテーマローテーション |

前世代との差分は、Match-3D（2019, Loop Games）が確立した「3 個集めて消す」を**テーマパック型のステージ進行**と**高頻度ブースター販売**に最適化した点にある。social_axis ソロでありながら、操作テンポの良さで日次プレイを成立させている。

## バズ要因 / 人気の根拠

- App Store / Google Play で**累計 1000 万 DL 超**、月 90K DL ペースで月商 400 万ドル規模の高収益カジュアルパズルに成長。
- **累計 IAP 収益は 1.3 億ドル**に達し、Triple Tile / Triple Match ジャンルの代表作ポジションを獲得。
- レビュー数 33 万超・**平均 4.9** とストア評価が異常に高く、カジュアル層のオーガニック流入と広告 LTV を両立。
- パブリッシャー Miniclip（8 Ball Pool 等で知られるカジュアル老舗）の UA / クロスプロモ網に乗ったことで、Boombox Games という小規模スタジオでもグローバル展開が成立。

## 企画への示唆

- **既存ジャンルに知覚レイヤーを 1 段足すと別カテゴリ化できる**：2D Match-3 → Triple Tile → 3D Triple Match の系譜は、メカニクス本体ではなく「見え方」を変えて市場を再定義した好例。
- **ソロ × 短時間 × オフラインの三拍子を保つと女性 20-50 代に深く刺さる**：通信・対戦・ソーシャルを足さない判断が、対象層のスキマ時間とアプリ起動回数を確保する。
- **テーマパックをコンテンツ更新単位にすると、メカ追加なしで体感の新鮮さを供給できる**：フルーツ／玩具／旅行という装飾切替だけで「次のステージ」を演出可能。
- **詰みを通貨で解くブースター設計は ARPPU を持ち上げる**：シャッフル／タイマー追加など「直前の失敗を取り消す」アイテムは購買意欲の心理的タイミングが揃いやすい。
- **ストア評価 4.9 を維持できる難度カーブはオーガニック獲得を最大化する**：難しすぎず簡単すぎない調整に投資すると、UA コストを下げ LTV を引き上げる二重効果が出る。

## 比較で見るべきタイトル

- Match 3D / Tile Busters（Loop Games）— 3D Triple Tile ジャンルの源流。
- 三麻雀（Triple Match / Triple Tile 系全般）— 同フォーマットの中国・台湾発バリエーション。
- Royal Match（Dream Games）— 同じカジュアルパズルでも 2D Match-3 + メタゲームの極致、対比軸として有効。
- Gardenscapes / Homescapes（Playrix）— メタゲーム駆動のカジュアル代表、Triple Match 3D の「メタを持たないソロ完結」設計と好対照。
