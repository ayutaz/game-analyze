# クロックメーカー / Clockmaker — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「呪われた館・街を少しずつ修復し、コレクションとキャラクターのストーリーを集め切ること」であり、動詞は **集める／育てる**。マッチ3 そのものの操作快感も強いが、プレイヤーを翌日また起動させているのは盤面ではなく、修復進捗バー・コレクション完成度・新章の解放条件である。

concept にあるとおり Clockmaker は「ストーリー駆動型マッチ3」の系譜（Royal Match / Homescapes / Gardenscapes）に属し、target は 30–50 代の女性カジュアル層と長期パズラー。1 日数分のセッションで進む非同期プレイ前提で、課金は装飾・スタミナ・ブースター中心と REW 型の収益構造そのもの。social_axis は未設定（純粋なソロ非同期プレイ）で、他者との直接対戦は中核ではない。

secondary に NAR が入っているのは、ヴィクトリアン世界観のキャラ群像劇とミステリー仕立ての章立てがメタ進行の燃料として機能しているため。物語は主目的ではなく、報酬ループを回し続ける理由づけとして組み込まれている。

## 報酬ループの組成

| 層 | 仕掛け | リテンション寄与 |
|---|---|---|
| 短期（1 セッション） | マッチ3 のステージクリア → 星 / コイン獲得 | 5–10 分の即時快感、ステージ失敗時のスタミナ・ブースター課金導線 |
| 中期（数日） | コインを修復タスクに投下 → 部屋・建物が変化 | 「次の部屋が見たい」視覚的進捗 |
| 中期（週単位） | コレクション（時計・人形・小物）の収集枠埋め | ガチャ的な抜け枠補完欲求 |
| 長期（章/シーズン） | 新エリア開放、キャラ追加、ストーリー解禁 | ライブオプス更新で離脱を遅延 |
| 課金層 | 装飾スキン、スタミナ回復、レアブースター | ARPPU 引き上げ、課金者の自己表現欲求 |

特筆すべきは「マッチ3 を解くこと」と「館を直すこと」が別 KPI として並走している点。パズルだけだと飽きるユーザーをメタ進行が、メタ進行だけだと退屈なユーザーをパズル難度カーブがそれぞれ受け止め、片方が詰まってももう片方で進めた感を演出できる二重ループになっている。

## バズ要因 / 人気の根拠

- 累計 30M+ DL、IAP 累計約 2.9 億ドルという数字は、Royal Match / Homescapes に比べ規模は一段下だが「ストーリー付きマッチ3」の中堅ヒットとして長期に売れ続けていることを示す
- DAU 100 万超水準を維持しているのは、2019 年以降のライブオプス強化（シーズン・イベント・新章追加）が効いている結果
- ヴィクトリアン+時計仕掛けという美術トーンが Homescapes 系の北米郊外住宅 / Royal Match の中世王宮と差別化され、女性カジュアル層に独自ポジションを確立
- developer = publisher の Belka Games（ベラルーシ拠点）にとって主力タイトル化しており、2024 年の 20% 規模レイオフ後も継続運営される基幹資産

## 企画への示唆

- パズル本体とメタ進行を **別 KPI として並走**させると、片方が詰まってももう片方の進捗感で離脱を防げる
- 修復・装飾の **ビジュアル差分を「見せる」演出**（before/after の劇的変化）を入れると、コイン消費の体感価値が跳ね上がる
- マッチ3 のような既成ジャンルに参入する場合、**世界観の独自性**（ヴィクトリアン+時計仕掛け）で UA 広告クリエイティブの差別化が可能
- コレクション枠は **わざと 1 枠空ける設計**にしておくと、補完欲求がガチャ的に機能する
- ストーリーは主目的ではなく **メタ進行を回す理由づけ**として配置すると、NAR ユーザーと REW ユーザーの双方を取り込める

## 比較で見るべきタイトル

- Royal Match（Dream Games）— 同系マッチ3+メタ進行のグローバル最大手
- Homescapes / Gardenscapes（Playrix）— ストーリー駆動型マッチ3 の元祖的存在
- June's Journey（Wooga）— ヴィクトリアン世界観 + ヒドゥンオブジェクト+メタ進行
- Project Makeover（Magic Tavern）— マッチ3+変身ストーリーの派生形
