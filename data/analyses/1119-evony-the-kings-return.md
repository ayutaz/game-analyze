# Evony: The King's Return — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「資源を貯めて城と兵を**育てる**」こと、そしてその育成差を同盟戦で換金することにある。concept にある「都市建設×領主同盟戦」は一見 EXP（戦略の体験）に見えるが、戦闘自体はリアルタイムでの即時操作ではなく **建築タイマー・研究タイマー・兵舎キュー** の蓄積差で勝敗が決まる構造のため、行動の中核は「数値を伸ばす」側に寄っている。target にある「30-50 代の重課金志向ストラテジー好き」も、可処分所得を時短アイテム・VIP 課金・武将ガチャに投じて育成ループを早回しする層であり、典型的な REW プレイヤー像と一致する。

secondary に EXP が入るのは、同盟外交・サーバー間戦争・KvK（Kingdom vs Kingdom）といった **対人交渉と陣形駆け引き** の体験的厚みがリテンションを支えているため。育成 → 戦争 → 同盟内序列という三段ループのうち、上澄みの「戦争」だけが体験型として機能している。social_axis は本来「同盟（協力）＋サーバー間（対戦）」の複合だが、未設定のまま残されている。

## 報酬ループの組成

| 層 | 主要要素 | 課金接点 | リテンション機能 |
|---|---|---|---|
| 短期（分〜時） | 建築・研究・採集タイマー | 時短アイテム、VIP 加速 | ログイン頻度を吊り上げる |
| 中期（日〜週） | 武将ガチャ、装備強化、兵種解放 | 武将召喚パック、ジェムパック | デイリー / ウィークリー目標 |
| 長期（月〜年） | 同盟序列、KvK ランキング、サーバー覇権 | 月パス、シーズンパス、SVS 課金 | 社会的地位と離脱コスト |
| メタ（年単位） | 三国志 IP 武将追加、季節イベント、新サーバー開幕 | 新武将ピックアップ | 既存城主の再加熱 |

前世代の城ゲー（クラクラ、KOA など）から継承した「タイマー経済」に、三国志 11 ライセンスの武将収集と RTS 風の戦場ビュー、そして **広告クリエイティブで集めたカジュアル層を MMO に流し込むファネル** を組み合わせている点が組成上の特徴。

## バズ要因 / 人気の根拠

- 累計 3 億 DL 超、2023 年単年で $355M 超の売上（topgamevony 公表）と、長寿 RTS の中でも安定して上位を維持。
- 「ピンを抜くパズル」「お姫様を救え」式のミニゲーム風広告クリエイティブを 2018〜2024 年にわたり大量出稿し、本編とは無関係の UI で全世界の SNS に拡散。
- 日本ではトリンドル玲奈起用の TV CM を 2022 年に展開し（ファミ通報）、広告経由のカジュアル流入をコア課金層に転換する MMO 営業を強化。
- 三国志 11 の正式ライセンスにより、KOEI IP の認知を借りて中華圏・日本・欧米の三市場で武将収集の納得感を担保している。

## 企画への示唆

- **「広告クリエイティブと本編が違っても構わない」**と割り切ると、本編 MMO のターゲット外であるカジュアル層を CPI 低く獲得できる（ただし広告詐欺批判のリスクを織り込む）。
- タイマー経済の上に **同盟という社会的地位レイヤー** を被せると、課金停止＝序列降下になるため離脱コストが跳ね上がる。
- **既存 IP のライセンス（三国志 11）** を武将ガチャの絵柄に充てると、ガチャの正当性とコレクション動機が一段強くなる。
- KvK／サーバー間戦争を定期開催すると、**新サーバー開幕＝既存重課金層の引っ越し需要** が生まれ、ARPU が再加熱する。
- 30〜50 代の可処分所得が高い層を狙うなら、**操作スキルではなく投下時間と資金の差** で勝敗が決まるルールにする方が継続課金に向く。

## 比較で見るべきタイトル

- ライズ・オブ・キングダム（Lilith Games）— 同じ「都市建設 × 同盟戦」MMO の直接競合、文明選択でターゲット拡張。
- Lords Mobile（IGG）— 広告クリエイティブと MMO ファネルの先行例。
- State of Survival（FunPlus）— 同型タイマー経済をゾンビ題材で展開、IP 借用なしで成立させた例。
- 三國志 真戦（コーエーテクモ／Qookka）— 同じ三国志題材でも「シーズン制リセット」を入れた対極設計。
