# ポケットモンスター ブリリアントダイヤモンド・シャイニングパール

## なぜ REW 主分類なのか

このリメイクを買って遊ぶプレイヤーの主動機は「シンオウのポケモンをもう一度集め、育て、図鑑を埋める」ことに集約される。コンセプト文が明示するとおり本作は 2006 年 DS 版の「忠実リメイク」であり、ストーリー構造もマップも原作そのままに、現行機の対戦・交換インフラに接続する設計を取っている。つまり物語を新しく読ませるためではなく、収集と育成の体験を Switch 世代に移植することが商品としての存在理由になっている。

ターゲットが「DS 版を遊んだ 20-30 代の原作世代と、その子ども世代」に絞られていることも REW 寄りである根拠になる。原作世代にとって新規の物語的発見はほぼなく、買う動機は「ギラティナ・ディアルガ・パルキアを今のインフラで再収集する」「ホームに送って図鑑を完成させる」という収集ループの再起動にある。`secondary: NAR` は、シンオウ編のストーリーや銀河団のドラマがリメイクの体験を厚くしている副次効果として位置づけられ、ジムリーダー戦・チャンピオン戦を物語の節目として再消費させる役割を担っている。

## 報酬ループの組成

リメイクとしての本作は、原作の収集ループをほぼそのまま現行機の運用インフラに接続している。層構造は以下のように分解できる。

| 層 | 中身 | 駆動する動機 |
|---|---|---|
| 図鑑収集 | 全国図鑑、シンオウ図鑑、地下大洞窟の化石・隠しアイテム | 「埋める」充足感 |
| 育成 | 個体値・努力値・性格・特性、御三家厳選 | 対戦準備としての最適化 |
| 対戦・交換 | ランクマッチ、フレンド交換、Pokemon HOME 連携 | 外部ループとの接続 |
| メタ収集 | 色違い厳選、ポケッチ機能の解放、リボン | 上級者向け長期目標 |
| 物語節目 | ジムバッジ 8 個、四天王、シロナ戦、ギラティナ・アグノム3 体 | 育成・収集の到達感の節目化 |

特徴は「地下大洞窟」を独自拡張として置いたこと。原作の地下通路を立体化し、化石・希少ポケモン・統計上レアな色違い遭遇地として再設計することで、本編クリア後の収集ループに専用の場所を与えている。一方で、剣盾以降に標準化されつつあったオープン化・対戦バランスのモダン化はあえて見送られており、リメイクの保守度合いそのものが「原作の収集体験を再現すること」をプロダクト要件として優先した判断と読める。

## バズ要因 / 人気の根拠

JSON の `popularity` が示す「DS 版 DP の忠実リメイク、原作世代の懐古需要」を出発点に分解する。

- 世界累計 1500 万本以上、日本 270 万本以上という出荷規模は、ポケモン本編リメイクが既存ファン層だけで AAA タイトル相当の数字を作れることを示した
- 同年に発売された『Pokemon LEGENDS アルセウス』(2022 年 1 月) と連続でシンオウ地方を扱うラインアップだったため、IP 全体としての「シンオウ年」需要を吸い上げた
- 配信・SNS では「原作再現度の高さ」「ミニチュア等身」「地下大洞窟」が議論軸になり、賛否がそのまま露出になる構造を持った
- Pokemon HOME 連携によって、本作で捕獲した個体が剣盾・スカーレット/バイオレットの対戦環境にも流れる「補給線」としての役割が成立し、ガチ層にも購入動機が生まれた

## 企画への示唆

- 忠実リメイクを売る場合、ターゲットを「原作世代＋その子ども」に絞り込むと、新規層獲得を諦める代わりに既存ファンの再購入率が読みやすくなる
- 物語ではなく収集・育成のループをリメイクの主軸に置くと、シナリオ刷新のコストを抑えながら「現行機インフラ接続」だけで商品価値を作れる
- 原作要素を保守する一方で「地下大洞窟」のような独自拡張を 1 つだけ大きく入れると、賛否が割れても話題の中心を作れる
- シリーズ内の別作品（本作の場合 LEGENDS アルセウス）と発売時期を近接させると、IP 全体で「地方年」を演出でき、両作のクロス需要を取り込める
- 外部基盤（Pokemon HOME のような共通倉庫）と接続すれば、本編の対戦勢にも「補給線」としての購入動機を作れる

## 比較で見るべきタイトル

- ポケットモンスター ハートゴールド・ソウルシルバー (2009) — 同じく原作忠実リメイク路線、ポケウォーカー等の独自拡張で差別化
- ポケットモンスター LEGENDS アルセウス (2022) — 同じシンオウを扱いつつ、収集ループを再設計してオープン化した対照例
- ポケットモンスター スカーレット・バイオレット (2022) — 完全新作・オープンワールド化、本作の保守路線と対極
- ポケットモンスター Let's Go! ピカチュウ・イーブイ (2018) — 初代リメイクで GO 連携を入れた、別軸の「現行機インフラ接続」事例
