# Game of Thrones: Conquest

## なぜ REW 主分類なのか

プレイヤーの主動機は「七王国の領地を広げ、英雄・装備・ドラゴンを集めて鉄の玉座に最も近い同盟へ昇る」こと、つまり**集めて育てて積む**ことにある。Lords Mobile / Last War 系の領地建設＋同盟戦＋4X 拡張という骨格に、90 体超のドラマキャラ収集とドラゴン育成、英雄装備強化という RPG 的成長軸を重ねた構造で、リテンションを支えているのは戦闘体験そのものより資源・キャラ・装備が積み上がる進捗実感のほうである。

ターゲットの中核が「Lords Mobile / Rise of Kingdoms / Last War 等の 4X モバイル MMO を長期周回する 25–45 歳の重課金層」である点も REW を裏付ける。`social_axis` が null なのは、同盟戦（協力）／PvP（対戦）／個人内政（ソロ）の 3 軸が等価で同居し、特定の他者軸だけで定義できないためと読める。secondary に EXP が付くのは、深夜の同盟 PvP・タイル戦・KvK イベントなど、リアルタイム調整の体験そのものを楽しむ層が一定数いるためである。

## 報酬ループの組成

| 層 | 主要な報酬 | 課金接点 |
|---|---|---|
| 領地内政 | 建築 LV up・資源タイル占拠・研究ツリー解放 | 加速アイテム・建設キュー追加 |
| 英雄（キャラ）収集 | 90 体超のドラマキャラの解放と昇格 | 召喚石・断片パック |
| 装備鍛造 | 英雄装備の鍛造・強化・宝石ソケット | 素材バンドル・期間限定鍛造券 |
| ドラゴン育成 | 雛から成竜への成長・スキル付与 | ドラゴン育成資源・季節パス |
| 同盟戦 / KvK | 玉座占拠・サーバ統合戦の称号 / 個人ランキング | シールド・テレポート・バフ束 |

短期は「加速で建築キューを進める」、中期は「英雄断片を集めて昇格させ装備を鍛造する」、長期は「KvK で同盟と鉄の玉座を奪う」という 3 層が同居し、どの層でも課金が時短として効くため、月商 19M ドル級のピークを支える。HBO IP のキャラ収集が「ガチャの引く理由」を強化し、Lords Mobile 系の純粋資源 4X より動機の密度が高い。

## バズ要因 / 人気の根拠

- HBO ドラマ最終シーズン放送期（2019 年 4 月）に**月商 19M ドルで過去最高**を記録し、テレビ放送と DAU / 課金が直接連動した稀有な事例
- Sensor Tower 集計で**配信 1 年で 1.25 億ドル**、**2019 年 5 月時点で累計 2.14 億ドル**、ライフタイム $500M+ という IP × 4X の上位クラス売上
- Lords Mobile / Rise of Kingdoms 型の MMO ストラテジー枠で、4Gamer など国内媒体でも日本語版同時配信が紹介され、海外 IP × 日本同時ローカライズという当時珍しい体制を取った
- 90 体超のドラマキャラ収集とドラゴン育成という RPG 軸を、純粋資源 4X 系の上に乗せることで「IP ファン × 4X 重課金層」の二重需要を取りに行った設計

## 企画への示唆

- 連動するメディア展開（最終シーズン放送）に合わせて新キャラ / イベントをぶつけると、月商が同月内で 2–3 倍跳ねる
- 純資源 4X の上に「キャラ収集 + 装備鍛造 + ドラゴン育成」の 3 RPG 軸を重ねると、純資源系より課金接点が増え ARPPU が上がる
- 海外 IP ものでも日本語版を同時配信すると、後発ローカライズより日本 iOS 売上ランキング初動が安定する
- `social_axis` を 1 つに絞らず「同盟協力 / 対人 PvP / ソロ内政」の 3 軸を等価に置くと、課金しない内政勢も滞留しサーバ人口を支える
- IP のキャラ数（GoT は本編登場人物が極めて多い）が多い作品ほど、ガチャ召喚プールを長期で維持しやすく、リテンション曲線が緩やかになる

## 比較で見るべきタイトル

- Lords Mobile（IGG、純資源 4X の原型）
- Rise of Kingdoms / 万国覚醒（Lilith、文明 IP 抜きの英雄収集 4X）
- Last War: Survival Game（First Fun、近代戦 IP の 4X 派生）
- State of Survival（FunPlus、ゾンビ IP × 4X）
- MARVEL Strike Force / Star Wars: Galaxy of Heroes（海外 IP × ガチャ収集の対照例、4X ではなく RPG 軸）
