# Game of Thrones: Winter is Coming M — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「同盟（House）の一員として他プレイヤーと領地・鉄の玉座を奪い合う」こと、つまり**長期サーバ戦争を戦い続ける対戦体験**である。concept にある通り Lords Mobile / Rise of Kingdoms 系のテンプレートを継承しており、城の発展や英雄編成といった育成行為は単体で完結せず、最終的に King's Landing 争奪などサーバ単位の対人戦で消費される。動詞は「戦う／競う」であり、ロールプレイや物語消費ではない。

social_axis は「対戦」。ただし MOBA 的なリアルタイム対戦ではなく、**同盟チャット常駐型の非同期＋同期混合の領地戦**で、Discord / LINE グループに張り付くコアプレイヤーが中心に立つ構造になっている。target が 25–45 歳の重課金 SLG 経験層に絞られているのもこの社会性のコストゆえ。

secondary に REW が入るのは、Commander（ジョン・スノウ / デナーリス / ティリオン等の英雄）のガチャ的入手と装備強化が課金導線として強く効くため。ただし収集自体がゴールではなく、**対戦戦力を底上げするための資源**として REW が EXP に従属している、と読むのが妥当。

## 体験の核 / 設計の発明

中核メカニクスは「七王国マップ上の領地戦」「House＝同盟運営」「Commander 編成」の三層構造で、これを HBO 公式 IP がラップしている。Lords Mobile 型テンプレに対する本作の差分は、Commander スキルが原作キャラのナラティブと結び付いている点と、サーバごとに鉄の玉座を実際に獲るゴールが提示されている点にある。

| 層 | 機能 | 対戦における役割 |
|---|---|---|
| 領地 / 城 | 資源生産・兵舎・研究 | 兵力供給のスループット決定 |
| Commander | 原作キャラ＋スキル / 装備 | 部隊バフ・特殊能力で局地戦の質を決定 |
| House（同盟） | チャット・拠点・合同攻撃 | サーバ戦争のオーガナイズ単位、Discord 文化が外部化 |
| サーバ目標 | 鉄の玉座 / 領地ランキング | 数か月単位のシーズン勝敗と離脱タイミング |

「対戦」軸の効き方は、個別 PvP の勝敗よりも**同盟が崩れた瞬間に強プレイヤーが一斉に離脱する**サーバ寿命の設計に直結している。CTW の G123 経由でブラウザ起動でき、PC 多窓運用が事実上容認されているのもこの構造を前提にしている。

## バズ要因 / 人気の根拠

- HBO ライセンスを保有する Yoozoo Games の系譜作で、Game of Thrones IP のモバイル SLG 化として日本市場に登場した数少ない 4X タイトル。
- 日本展開は CTW の G123 が窓口で、ブラウザ HTML5 で配信され**インストール障壁を下げた**ことで X / Twitter 広告での到達量が大きかった。
- ジョン・スノウ／デナーリス／ティリオンら原作キャラを Commander として直接運用できる点が、原作ファンの初回課金トリガーとして機能した。
- 重課金型サーバ戦争の常で、上位は数十万円〜数百万円規模の課金者が同盟リーダーとして牽引し、彼らの戦況実況が SNS で副次的な広告になっている。

## 企画への示唆

- **既存テンプレ（Lords Mobile 型）に強 IP を被せると、新規ジャンル開発より広告 ROAS が安定する** — ゼロから戦争 SLG の文法を教える必要がなくなるため。
- **Commander にキャラ性を持たせると、ガチャを「収集」ではなく「戦力選定」に翻訳できる** — REW を EXP に従属させると重課金ユーザの満足度が安定する。
- **ブラウザ配信（G123）を併用すると iOS / Android の審査・課金率を回避しつつ多窓 PC ユーザを取り込める** — SLG とブラウザ配信は親和性が高い。
- **サーバ寿命を明示的にシーズン制で切ると、同盟崩壊による離脱を「次サーバへの移住」として吸収できる** — リテンション設計上、サーバ統合タイミングが鍵になる。
- **原作キャラのスキル名を世界観の固有名詞で揃えると、ローカライズ翻訳コストを下げつつ IP 忠実度を担保できる** — 翻訳が IP の質感を毀損するリスクを減らせる。

## 比較で見るべきタイトル

- Lords Mobile（IGG） — 本作のテンプレート元、無 IP で広告だけで世界トップ収益に到達した対照例。
- Rise of Kingdoms（Lilith） — Commander＝歴史上の人物に置き換えた同形式、IP の重みの差が比較しやすい。
- 信長の野望 出陣（コーエーテクモ） — 同じ「武将＝Commander」構造を位置情報 SLG に落とした和製比較対象。
- King of Avalon（FunPlus） — 中世ファンタジー IP なし版の同型、課金者構造の参照。
- Mobile Strike / Game of War（MZ） — 重課金 SLG の祖型、サーバ戦争設計の歴史的参照。
