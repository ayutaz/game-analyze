# アークナイツ：エンドフィールド — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「3D 化された Hypergryph IP のオペレーター（キャラクター）をガチャで引き、惑星タロⅡの開拓と戦闘で育成する」こと、すなわち **集める／育てる** にある。前作『アークナイツ』のタワーディフェンスを捨ててオープンワールドへ転身したのは、フィールド探索と建設で「キャラを連れ歩く動機」を強化し、ガチャ前提のリテンションループを 3D に移植するためと読める。

concept にある「シームレス 3D 探索＋4 人切替リアルタイム戦闘＋惑星開拓の RTS」は手触りとしては EXP 寄りだが、target が「原神／鳴潮系の 3D オープンワールドガチャ RPG を好む 20–40 代のコアゲーマー層」と明確にガチャ志向で切られているため、主動機はキャラ収集側に寄る。secondary に EXP（リアルタイム戦闘の操作快感）と NAR（前作から継続する重厚な SF 政治劇のシナリオ消費）を抱える典型的なハイブリッド HoYo 系ガチャである。

## 報酬ループの組成

| 層 | 内容 | リテンション効果 |
|---|---|---|
| ガチャ層 | 限定オペレーター + 武器スタンプ。前作 IP からの引き継ぎ強キャラ | 既存ファンの初動課金、PS5 同時配信でハイ ARPU 層を取り込み |
| 育成層 | スキル昇格・モジュール・装備強化。RTS パートで素材自動収集 | デイリー触圏を「自動化された工場巡回」に再設計 |
| 探索層 | オープンワールド宝箱・ボス・サブクエ | 原神型の週次／パッチ更新による新マップ報酬 |
| 開拓層 | エンドフィールド工業の基地建設・自動製造ライン | 放置でも素材が貯まる、ログイン頻度を下げずに離脱を抑制 |
| 物語層 | 前作世界観の延長章 + 新惑星タロⅡ編 | 既存ファンの IP コミット維持 |

ポイントは **「RTS 風の工場自動化」を素材ファーミングの代替に組み込んだ** こと。原神系の周回作業を「基地に任せて寝かす」設計に置き換え、コアゲーマーが嫌う反復労働を意図的に削っている。

## バズ要因 / 人気の根拠

- 発売 2 週間で全プラットフォーム合計 **$173M**、モバイル単独で初月 **$46M**、3 か月で約 **$96M** と、ローンチ後のスケールが原神級の規模で立ち上がった。
- **日本がモバイル課金シェア約 35%** で世界トップ。前作『アークナイツ』が日本市場で築いた IP 資産が、3D 続編にそのまま乗った形。
- **2026 年 1 月 22 日に PS5／PC／iOS／Android 全世界同時配信** という HoYo 型のマルチプラットフォーム展開で、モバイル専業ガチャの壁を越えハイエンド層を取り込んだ。
- 前作のタワーディフェンス → 3D オープンワールド + RTS 開拓という **ジャンル丸ごとのリブート** が話題化、原神／鳴潮の対抗馬として PV 段階から英語圏・中華圏で長期間ウォッチされていた。

## 企画への示唆

- **既存 IP のジャンル転換は、世界観とキャラを残してメカニクスだけ刷新すると、新規ユーザーと既存ファンを同時に取り込める**（ただし前作プレイ感の喪失リスクは別途ケアが要る）。
- **オープンワールドガチャに「基地建設＋自動製造」を載せる** と、周回ファーミングが工場放置に置換され、コアプレイヤーの摩耗を下げられる。
- **モバイル＋ PS5 同時配信で ARPU 上位層を取り込む** と、モバイル単独より高単価層が乗り、初動 KPI が一段跳ねる。
- **日本市場の課金シェアが世界トップになる IP** は、ローカライズ・声優・コラボ予算を日本側に厚く投下する判断が早期に必要。
- **3D 化のタイミングを「前作が成熟した 5〜6 年目」に置く** と、IP 残量が最大化された状態で続編 PR を打てる（前作の運営継続が前提）。

## 比較で見るべきタイトル

- 『原神』『鳴潮』 — 3D オープンワールド + ガチャの直接的競合、target も重なる
- 『崩壊：スターレイル』 — 同じく IP リブート系のターン制ガチャ、HoYo 比較軸
- 『アークナイツ』本編 — タワーディフェンス時代の前作、IP 継承と転換の差分検証
- 『ゼンレスゾーンゼロ』 — アクション + ガチャの新世代、リアルタイム戦闘設計の比較
