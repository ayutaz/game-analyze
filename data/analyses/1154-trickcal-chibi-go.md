# Trickcal: Chibi Go (Trickcal Re:VIVE) — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「もちもち SD 美少女を引いて、並べて、撫でる」こと。ガチャでキャラを集め、編成して眺める／触る一連のループが体験の中心にあり、戦闘そのものはオートで進む。

`concept` が示すように本作は「SD 美少女 × オートチェス × ガチャ収集」の三層構造で、戦闘判定はキャラ性能とフォーメーション順序にほぼ決まる。プレイヤーが能動的に触るのは「誰を引くか・誰を並べるか・誰のほっぺを引っ張るか」であり、操作スキルではなく所有と編成がコアドライバになる。`target` 記述にあるブルアカ / プリコネ / NIKKE と同じ「キャラ獲得→愛着醸成→課金で深掘り」のリテンション様式に属するため、主分類は REW が妥当。

`secondary: ["EXP"]` はオートチェス由来のフォーメーション組み替えの面白さを拾っている。配置・属性相性・スキル発動順という EXP 的レイヤーがあるからこそ「引いたキャラを使い切る楽しさ」が成立しており、純粋なコレクションではない点を補正している。

## 報酬ループの組成

| レイヤー | 提供する報酬 | リテンションへの効き方 |
|---|---|---|
| ガチャ排出 | 新規 SD キャラの所有 | 期間限定ピックアップで初回／復帰起点を作る |
| 育成・限界突破 | 既存キャラの戦力上限解放 | 重複排出に「無駄じゃない」を付与し沼の蓋を閉じる |
| もちもちタッチ演出 | ほっぺを引っ張る非機能的フィードバック | 育てたキャラへの愛着を強化し離脱率を下げる |
| 低価格課金パック | 月パス・小額バンドル | 1 回 ¥600〜¥1,500 帯で日常課金の心理障壁を下げる |
| オート編成バトル | ステージ突破・素材ドロップ | ガチャ排出と育成を回す資源供給ライン |

設計の要は「重複ガチャを限界突破素材として救済 → タッチ演出で愛着を上書き → 低価格パックで小さく払わせる」という、課金圧を抑える方向に最適化された REW ループになっている点。`popularity` の「無課金でも長く楽しめる」という評は、この三段のクッションの帰結。

## バズ要因 / 人気の根拠

- 2021 年初期版は失敗、2023 年「Re:VIVE」としてリブートし 2024 Korea Game Awards 最優秀賞を受賞 — 「再生プロダクト」のストーリー自体が話題化のフックになった
- 2025 年に Bilibili Game の配信で「Chibi Go」グローバル版が立ち上がり、中華圏配信ネットワークでアジア市場に拡張
- グローバルで 3M+ DL（公開時点）。日本国内売上は未公開だが、ブルアカ / プリコネ系のソフト課金層が SNS で SD イラストを拡散
- ほっぺを引っ張れる「もちもちタッチ」演出が短尺動画で切り抜かれやすく、TikTok / X でビジュアル単体が独立して回る素材になっている

## 企画への示唆

- 初期版失敗 → リブートで賞を取った例：**プロダクトを撤退ではなくリブランドで再投入すると、過去資産（IP・キャラ）はそのまま再収益化できる**
- ガチャ重複を「限界突破素材」に変換すると、被り体験が損失感ではなく前進感に再定義される
- 低価格課金パック中心の単価設計にすると、ARPPU は下がるが課金率（PU rate）が上がり、口コミ層の「優しい課金」評を獲得できる
- 機能と無関係なタッチ演出（ほっぺ引っ張り）を入れると、ゲーム体験ではなく「キャラを所有する満足度」が伸びてリテンションが上がる
- オートチェス × SD ガチャの組み合わせは、操作技能依存を切り離した上で「編成 EXP」を残すことで、ライトユーザー帯を取り込みつつ廃課金導線も両立できる
