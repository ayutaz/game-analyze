# インフィニティニキ (Infinity Nikki) — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は **「衣装を集める」** — オープンワールドの探索やパズルは衣装獲得とアンロックのための導線として機能している。

`concept` が明示するとおり、本作は「着せ替え × オープンワールド」を掲げる F2P タイトルであり、1000 着以上の衣装をガチャと収集で集めることが進行の中心軸にある。`target` も「ミラクルニキ系着せ替えゲームのファン」「コーデ共有を楽しむライト〜ミドルコア層」と、収集と自己表現を主動機として持つ層に明確に振っている。フィールドの飛行・グライド・釣り・パズル解除といったアクションは個別の衣装スキルに結びついており、行動の選択肢を増やすほどガチャ・素材集めの効率が上がるという報酬循環に組み込まれている。

`secondary: ["EXP"]` が示すのは、UE5 で描かれた広大な探索フィールドそのものを歩く体験価値で、これがミラクルニキ系の従来 2D 着せ替えと差別化される副次的な強みになっている。`social_axis: null` は、リアルタイムマルチや同盟戦のような同期的他者軸を中核に据えていない（コーデ共有やランキングは外周）ことを反映している。

## 報酬ループの組成

| レイヤー | 主な要素 | リテンション設計上の役割 |
|---|---|---|
| デイリー | スタミナ消費の素材集め、ログイン報酬、デイリー任務 | 毎日の習慣化を作る最下層ループ |
| ガチャ | 衣装単位ガチャ（複数パーツのセット集め）、ピックアップ更新 | 課金・期待値生成。1 完成形に複数パーツが必要なため擬似的にステップアップが効く |
| 探索 | オープンワールドの宝箱、パズル、コレクタブル、衣装スキルでアンロックされる地形 | 探索 → 素材 → 衣装合成のフィードを供給 |
| 衣装スキル | 飛行・グライド・釣り・浄化など能力解放 | 報酬（衣装）と進行（フィールドアクセス）を同一物に統合 |
| シーズン / コラボ | 期間限定衣装、SNS 投稿テーマ | 復帰トリガーと UGC（コーデ写真）の起点 |

衣装が「ガチャの戦利品」「フィールドアクションのキー」「SNS で見せるキャラ表現」の 3 役を兼ねるため、課金・進行・自己表現が単一通貨で接続される構造になっている。原神型の単体ガチャと異なり、1 体のキャラに複数衣装ピースが要求されるため擬似コンプ要求が強く、収集寄り設計に振り切れている。

## バズ要因 / 人気の根拠

- 全世界事前登録が **3000 万** を突破した時点で既にコア層への到達が確立しており、配信開始からわずか数日で **DL 1000 万** に到達した（`popularity` / `sales_world`）。
- モバイル単体でも **配信 4 日で約 400 万ドル**（Pocketgamer.biz）の初動を記録し、Papergames / Infold Games の 2024 年全体売上は約 **8.5 億ドル** 規模に達したと報じられている。
- 「ミラクルニキ」系の着せ替え IP を **オープンワールド + UE5 グラフィック** に再構築したことで、既存ファン層と原神／崩壊スターレイル等のオープンワールドガチャ層の両方を吸引した。
- ガチャ衣装のビジュアルと「ゼルダ BotW 風探索」の組合せが、衣装スクリーンショット中心の SNS 拡散と相性が良く、TikTok / X 上で短尺の着替え動画がバズの母体となった。

## 企画への示唆

- **報酬アイテムを進行のキーに同居させると**、課金と探索の動機が単一通貨に統合され、F2P プレイヤーの離脱率を下げられる（衣装＝ガチャ景品＋スキルキー）。
- **既存 2D IP を 3D オープンワールド化する場合**、世界の広さよりも「IP の主動機（ここでは着せ替え）が新しい地形・カメラ角度で映えるか」を最優先で設計すると、IP ファンの流入が読みやすい。
- **1 セットに複数パーツを要求するガチャ設計にすると**、単発キャラガチャより擬似的なコンプ欲求が強くなり、平均課金額の引き上げと再ピックアップの口実を同時に得られる。
- **SNS シェアを前提とする場合**、ゲーム内に撮影モード／ポーズ／背景の自由度を仕込むと、プレイヤー自身がクリエイティブ素材を量産し、広告費を肩代わりする。
- **UE5 ベースのモバイル配信を採用すると**、PC / 据置との同時展開でビジュアル訴求を最大化できる一方、端末要件と最適化コストが跳ねるため、対象端末セグメントの設定がローンチ初動の上限を決める。

## 比較で見るべきタイトル

- **ミラクルニキ / シャイニングニキ**（同 IP・前世代の 2D 着せ替えガチャ）
- **原神 / 崩壊：スターレイル**（オープンワールド × ガチャの先行例、UE/独自エンジンの収益モデル）
- **ゼルダの伝説 ブレス オブ ザ ワイルド**（探索とアクション能力の解放構造の参照元）
- **あつまれ どうぶつの森**（衣装・自己表現と SNS シェアの非同期報酬軸）
- **NIKKE: 勝利の女神**（キャラクター訴求 × 非同期ランキング型の F2P モバイル収益モデル比較）
