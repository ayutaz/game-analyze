# Code: To Jin Yong (代号:射雕) — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機を 1 文で言うなら「金庸の描いた江湖を、自分の足で歩き・自分の拳で戦う」体験そのものに置かれる。物語は射鵰英雄伝という既存 IP に強く依存し、報酬は AAA 規模のコスメ／装備が想定されるが、企画の中心は **Unreal Engine 5 で再構築された中華大陸を一人称的に踏破する** ことにある。社会軸が「ソロ」になっているのも、本作の体験設計が他者対戦や協力ではなく、個の旅と武術アクションの手応えに置かれていることを示している。

secondary に NAR と REW が並ぶのは妥当で、原作の章立てを追体験する物語消費（NAR）と、武術／装備／拠点の収集進行（REW）が EXP の周囲に厚く張り付く。ただし射鵰の物語は既に読者の頭の中にあるため、本作の付加価値は「読む」より「歩く・戦う」に立てざるを得ない。

## 体験の核 / 設計の発明

中核メカニクスは三層に分かれる。武術考証を入れた拳法・剣術モーションの **格闘体験**、フォトグラメトリで取得した華山などの実在地形を踏破する **景観体験**、そして射鵰英雄伝の章立てに沿った **江湖巡礼体験**。Genshin Impact 以降の中華系オープンワールドが「色彩豊かなファンタジー世界」で勝負してきたのに対し、本作は **実在地形 × 武侠考証** という現実寄りの軸でディファレンシエートを狙う。

| 層 | 担当する欲求 | 前世代との差分 |
|---|---|---|
| 武術アクション | 戦う手応え | モーション考証で「型」を再現、ガチャ系の派手な必殺技偏重から距離 |
| オープンワールド踏破 | 歩く・見る・登る | フォトグラメトリで実在山岳（華山等）を取り込み、ファンタジー地形ではなく実景に寄せる |
| 原作章立ての追体験 | 江湖を旅する | 既知の物語を空間化、読者の脳内地図を 3D 化する |

social_axis「ソロ」が効くのは、武侠小説の主人公体験が本来単独行であり、PvP / 協力レイドを差し込むと「江湖を一人で旅する」感が壊れるため。マルチが入るとしても拠点・観戦レベルに留めるのが構造的に整合する。

## バズ要因 / 人気の根拠

- 2022 年の発表トレーラーで、Tencent / LightSpeed が金庸 IP を Unreal Engine 5 でフル AAA 化することが明らかになり、中華圏のゲーム・武侠両コミュニティを横断して話題化
- 華山など実在地形のフォトグラメトリ素材が公開され、「ゲーム世界の景観」より「中国大陸の景観」を見せる広報設計に振り切っている
- 金庸の代表作『射鵰英雄伝』は中国・東アジア圏で複数世代にわたる読者基盤を持ち、ドラマ化・漫画化・既存ゲーム化の積み重ねで IP 認知が既に高い
- 開発元 LightSpeed Studios は PUBG Mobile / Arena Breakout 等を運営する Tencent 傘下大手で、AAA 規模の長期運営に必要な体力が担保されていると受け止められている

## 企画への示唆

- **「既存ファンタジーの彩度」より「実在地形のフォトリアル」に振ると差別化が立つ** — 同ジャンルが色彩 × アニメ調に寄っているとき、実景フォトグラメトリは認知コストの低い差別軸になる
- **古典 IP は「読者の脳内地図の 3D 化」として使える** — 章立てを追うだけでなく、読者が想像してきた地理（華山・桃花島）を歩ける体験そのものが価値になる
- **武術考証のような外部専門性を入れると、モーションがマーケティング素材になる** — 「型が違う」ことが切り抜き映像で伝わるため、SNS バズの単位が小さくて済む
- **ソロ主軸で AAA を組むときは、マルチを「拠点・観戦」までで抑える** — 江湖／西部劇／忍者など「個の旅」を芯にした世界観は PvP 導入で世界観が破綻しやすい
- **発表から発売までの長期スパンは、トレーラー単位でのコミュニティ更新運営とセット** — UE5 系 AAA は数年単位の開発期間が前提で、定期的な技術デモ公開が IP の熱量を維持する

## 比較で見るべきタイトル

- 原神 / 鳴潮（同じ中華系オープンワールド、ただしファンタジー寄り / ガチャ主軸）
- Black Myth: Wukong（中華 IP × 単一プレイヤー × UE5 アクションという最も近い参照点）
- Ghost of Tsushima（実在地理 × 一人称的な「個の旅」体験という構造類似）
- 太吾絵巻 / 下一站江湖（武侠 IP のシミュレーション側、章立てではなくシステムで江湖を表現）
- Assassin's Creed Shadows（実在地形のフォトグラメトリ × 武術考証という広報軸の類似）
