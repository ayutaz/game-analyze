## なぜ NAR 主分類なのか

プレイヤーの主動機は「猫として廃墟を歩き、滅びた都市と残されたロボットたちの物語を読み解く」こと、つまり**追体験する**ことにある。BlueTwelve Studio が提示するのは、人類が消えたサイバーパンク都市を野良猫の視点で踏査するという固定された筋書きであり、エンディングは一通り、章立ても線形に進む。`concept` が「サイバーパンクな廃墟都市を探索する三人称視点アドベンチャー」と置くとおり、戦闘やビルド構築よりも、世界観のレイヤーを 1 枚ずつ剥がしていく物語進行が中心に据えられている。

`target` の「短時間でクリアできる物語体験を求めるカジュアル層」は、ローグライト的なリプレイや MMO 的な反復消費ではなく**一本道の物語を一気に消費する層**を指す。social_axis が null なのも、他者との同期・非同期インタラクションが設計に組み込まれておらず、消費の単位が個人の鑑賞行為に閉じていることの反映である。

secondary の EXP は、猫の身体性を模した跳躍・擦り抜け・転倒等の挙動が「物語を読むための移動」を単調な作業に落とさないための副次的なエンジンとして効いている。探索の触感が物語消費の薪になっている構造。

## 物語の駆動構造

Stray の物語駆動は「都市の層構造」「相棒キャラクター」「猫であるという視点制約」の 3 つに分解できる。

| 駆動要素 | 仕掛け | プレイヤーが受ける効果 |
|---|---|---|
| 都市の垂直層 | スラム → ロストシティ → アンテナ → 中央/下水 と章ごとに別レイヤーへ降下 | 「下に行くほど真相に近づく」直感的な物語勾配 |
| 相棒 B-12 | プレイヤー以外で言語を持つドローン NPC を 1 体に絞り、記憶を断片収集 | 単独主人公の孤独と、解説役の両立 |
| 猫視点の制約 | UI を最小化、人語は字幕越し、操作は跳躍と鳴き声中心 | 「世界を理解しきれない」非全能感が物語の謎と整合 |
| 短尺設計 | クリア 6–8 時間、章ごとの目標が明確 | 配信・実況での消費単位とサイズが一致 |

選択肢分岐や複数エンディングは持たないが、ロボット住民との小規模クエスト群が「副次的に世界を読むテキスト」として配置されており、物語の本筋を太らせずに世界観の解像度を上げる役割を担う。

## バズ要因 / 人気の根拠

`popularity` の「猫主人公の珍しさと SNS 映え、配信向きの短尺で話題化」を分解すると次のようになる。

- 主人公が**人型でも擬人化動物でもなく実寸の猫**である事例が AAA/AA 帯にほぼ存在せず、ビジュアル一枚で識別可能なフックになった
- ワールドワイドで **7M+** の販売規模に到達し、Annapurna 配給作品としては突出した数字
- クリア 6–8 時間というサイズが、配信者にとって「1〜2 配信で完結できる長さ」として扱いやすく、リリース直後の同時配信本数を押し上げた
- 猫を飼っている視聴者・SNS ユーザーが**自宅の猫がモニターに反応する動画**を投稿する二次拡散が発生し、ゲーム外コンテンツが宣伝として機能した
- 2022 年末に PS Plus Extra カタログ入りしたことで、有償購入層と無償体験層の双方を確保している

## 企画への示唆

- **非人間主人公を採用すると UI を削減できる** → HUD を最小化することで「世界を理解しきれない」感覚を物語と整合させられる
- **クリア時間を 6–8 時間に絞ると配信単位と一致する** → 短尺は「内容が薄い」ではなく「配信流通性が高い」に転化する
- **言語を持たない主人公に解説役 NPC を 1 体だけ付ける** → 主人公の沈黙を保ったまま、プレイヤーへの情報供給チャネルを確保できる
- **垂直方向の層構造に章を割り当てる** → 「下に降りる」物理移動が、そのまま「真相に近づく」物語の勾配として読まれる
- **動物の挙動を忠実にシミュレートすると SNS 動画素材になる** → ゲームプレイそのものが二次拡散の燃料に変わり、広告費に依存しない認知獲得が生じる

## 比較で見るべきタイトル

- Journey (2012) — 言語を持たない主人公・短尺・線形構造の先行例。Stray の物語駆動の祖型
- INSIDE (2016) — 言語を持たない少年主人公で世界の異常を読み解く、垂直降下型の物語構造
- The Last Guardian (2016) — 非言語の動物パートナーと共に廃墟を踏査する、相棒 NPC 設計の参考例
- Little Nightmares (2017) — 非力な主人公が巨大な世界を踏査する設計、配信流通性の高い短尺アドベンチャー
- Goat Simulator (2014) — 動物視点ゲームの SNS 拡散事例として比較すると、Stray の物語性の差が浮かぶ
