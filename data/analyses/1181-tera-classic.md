## なぜ REW 主分類なのか

プレイヤーの主動機は「キャラと装備を集めて強くする」こと。PC版『TERA』のオープンフィールドと爽快アクションを忠実移植したアクション MMORPG だが、スマホ版で運営収益を支えていたのは戦闘体験そのものではなく、ガチャによる装備・スキル・マウントの収集と、ギルド単位での装備強化レースである。concept にある「ガチャによるキャラ・装備育成を軸に据える」、target にある「ハードコアな韓国系 MMORPG をスマホで遊びたい 20-40 代の重課金層」という指定は、いずれも報酬獲得を主動機とする層への設計であることを示している。

secondary に EXP が入るのは、UE4 で再現されたノンターゲティング寄りのアクション戦闘が確かに快感の核を持つため。ただしスマホ操作と片手 UI に最適化された結果、操作の上達余地は PC 版より縮小しており、アクションは「装備の強さを体感する場」として REW の従属変数になっている。social_axis は null だが、実態としてはギルド戦／攻城戦に基づく非同期競争に近い。

## 報酬ループの組成

| 層 | 内容 | 課金圧 |
|---|---|---|
| キャラ | 種族 × 職業（ヒューマン / エリーン / バラカ / アーマン × 戦士 / バーサーカー / ランサー / プリースト / ソーサラー …）の選択。リセマラ対象は装備寄り | 低 |
| 装備ガチャ | 武器・防具・アクセサリの ★レアリティ抽選。同種重ね合わせで限界突破 | 高（中核） |
| スキル / ルーン / 紋章 | 戦闘性能を底上げする補助層。装備と掛け算で効く | 中 |
| マウント / ペット / コスチューム | コレクション欲求と能力バフを兼ねる装飾層 | 中 |
| ギルド / 攻城戦 | ギルドダンジョン、領地戦、ランキング報酬。集団装備差が直結 | 高（社会的圧） |

リテンションはデイリー討伐・ギルド出席・週次レイドで「ログイン時間そのものが資源」になる構造。装備が陳腐化する更新サイクルで再ガチャを促し、ギルド戦という公開比較の場が「装備を更新しないと迷惑をかける」社会的圧として働く。これが韓国系 MMORPG の典型的なリテンション設計で、テラクラシックも忠実に踏襲している。

## バズ要因 / 人気の根拠

- PC版『TERA』(全世界 2,500 万 PL) の IP を踏襲し、原作プレイヤーへの想起喚起で初動を獲得
- UE4 で「PC 版の 20 年前」を舞台に据え、原作世界の前日譚として懐古層・新規層の両方に文脈を提供
- 韓国 2019 年 8 月配信を皮切りにアジア展開、日本は Gravity Game Arise が 2021 年 1 月にローカライズ配信
- 日本版は 2023 年 3 月にサービス終了。約 2 年間という運営期間は、IP 移植系スマホ MMORPG の中央値前後

## 企画への示唆

- PC IP をスマホに移植する際、原作の「象徴的体験」（TERA ならノンターゲティング戦闘）を 1 つだけ忠実再現すると、原作ファンの初動を確保できる
- 時系列を「原作の N 年前」に置くと、原作のネタバレを避けつつ既知キャラの若年版を出せて IP の射程を広げられる
- ギルド戦を装備強化レースの公開比較場に据えると、個人課金欲求を社会的圧に変換できる（ただし重課金者向けに偏る副作用あり）
- 装備ガチャ × 限界突破 × 定期的な装備上限解放を組み合わせると、初期キャラ選択の重要度を下げてリセマラ離脱を防げる
- 韓国系 MMORPG のリテンション設計（デイリー出席 + ギルド義務 + 週次レイド）をそのまま日本配信すると、日本のライト層離脱が早まりサ終を 2-3 年に短縮する傾向がある

## 比較で見るべきタイトル

- リネージュ2 レボリューション（Netmarble、2017）── ギルド戦・装備ガチャ・公開比較の典型例
- リネージュM（NCSOFT、2017）── IP 移植 × 重課金 MMO の長期運営モデル
- 黒い砂漠 MOBILE（Pearl Abyss、2019）── 同じく UE/独自エンジンで PC 系 MMO をモバイル移植
- ELYON / Bless Mobile 系 ── 韓国系装備ガチャ MMO の同世代群
- TERA（En Masse / Bluehole、PC 2011 / 2022 サ終）── 原作。ノンターゲティング戦闘の文脈を提供
