# キングスレイド / King's Raid 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機を 1 文で書けば「**推しの英雄を、ガチャの確率に振り回されずに最後まで育て切る**」である。英雄そのものは確定入手できるが、武器・専用装備・超越（Transcendence）など育成レイヤーに課金資源と時間が集中する設計で、入手の射倖性をやめた代わりに「育成の深さ」を商品にした REW である。

`concept` が示すとおり、本作は「キャラガチャ無し」をマーケティング上の核に据えた韓国産モバイル RPG であり、`target` も「ガチャに疲れた」「推し英雄を確実に育てたい」コアユーザーを名指ししている。この時点でゲーム外動機は **収集（英雄の自軍化）→ 育成（戦力曲線の押し上げ）→ ランキング・PvP・レイドでの戦力検証** という古典的なモバイル RPG のリテンションループに乗っており、報酬軸が主導していることが確認できる。`secondary: ["EXP"]` は、リアルタイム 3D の操作・スキル発動・回避といったアクション要素が「育てた英雄を動かす実感」として効いていることを示すが、これは育成成果のフィードバック装置であり、主動機を覆すものではない。`social_axis` は null だが、運用上はギルド・レイド・PvP の非同期協力／対戦が混在しており、明確な単一軸を立てづらいタイトルである。

## 報酬ループの組成

「英雄ガチャ無し」という看板を立てたうえで、課金資源がどのレイヤーに吸われるかを分解する。

| 層 | 内容 | 課金・時間の吸引点 |
|---|---|---|
| 英雄入手 | 確定入手・配布中心 | 入口の不確実性を排除（最大の差別化） |
| 装備 | 武器・防具・アクセサリのドロップ＋強化 | 周回／素材の吸引 |
| 専用装備（UW） | 各英雄固有の超強化武器 | 周回ループの中核 |
| 超越 | UW のさらに上のキャラ覚醒 | エンドコンテンツの戦力差 |
| コスチューム | スキル効果付きスキン | 課金キャラ強化＋見た目 |
| コンテンツ消費 | レイド／PvP／ギルド戦 | 育成済み英雄の検証場 |

入口（ガチャ）の不確実性を売らない代わりに、**英雄ごとに UW + 超越 + コスチュームを積み上げる縦の育成スタック**を売る構造である。1 体を完成させる総コストはガチャ型と比較して低いとは限らず、推しを 1 〜 2 体に絞るプレイヤーほど課金が読みやすく、結果として「課金額の見通しが立つ」感覚がコアファンの定着に寄与した。

## バズ要因 / 人気の根拠

- **「キャラガチャ無し」というメッセージの単純化**。ガチャ疲れが顕在化していた 2017–2018 年の市場に、否定形のスローガン 1 行で位置取りした。
- **配信規模**：全世界 150 か国で配信、累計 **1,600 万 DL 超**（2020 年時点）。韓国 Vespa の海外売上の大半を日本市場が支えた、と日本のゲームビジネス報道で繰り返し言及されている。
- **日本市場での着火**：2018 年の日本配信直後に App Store／Google Play 売上トップ 10 入り。韓国／グローバル運営 RPG に親しんだ重課金寄りファン層に浸透した。
- **IP の延命と再起動**：2025 年 3 月にサービス終了したが、Masangsoft が IP を取得し **2026 年再リリース予定**。サ終後の IP 売却・再起動という、近年のモバイル運営 IP に典型的なライフサイクルを辿っている。

## 企画への示唆

- **「やらないこと」を商品名の隣に置くと、レッドオーシャンで位置取りができる**：ガチャ全盛期に「ガチャ無し」を看板にしたことが、機能差ではなく価値観の差として刺さった。
- **入口の不確実性を捨てるなら、育成スタックを縦に厚くする**：UW ＋超越＋コスチュームのように **1 体あたりに積み上げられる課金レイヤーを 3 層以上**用意すると、推し集中型プレイヤーの ARPPU を確保できる。
- **アクション操作を「育成成果のフィードバック」として配置する**：自動戦闘で済むモバイル RPG にあえて操作介入を残すと、育成数値が画面の手応えに変換され、REW の主動機を EXP が補強する。
- **海外パブが日本市場で当てた場合、サ終後の IP 譲渡を前提にして資産設計する**：本作は IP が Masangsoft に渡り再リリースに繋がった。**キャラ設定・世界観・楽曲を運営会社から切り離せる状態**で運用すると、サ終即終焉ではなく次世代化の選択肢が残る。
- **「推し 1 〜 2 体に課金を集中できる」見通しを与えると、コア層のリテンションが伸びる**：ガチャ確率に依存しない設計は、課金額の予測可能性をプレイヤーに渡す行為であり、長期継続の動機になる。

## 比較で見るべきタイトル

- **エピックセブン / Epic Seven**（Smilegate、2018）：同じ韓国産・縦スクロール 2D 系の英雄収集 RPG。こちらはガチャ型で、King's Raid と対極の入口設計。
- **Brown Dust 2**（Neowiz、2023）：韓国産・キャラ育成深掘り型の後発。King's Raid の「育成スタックを縦に厚くする」アプローチを継承。
- **グランブルーファンタジー**（Cygames）：武器・召喚・上限解放など「育成レイヤーを縦に積む」設計の日本側の典型例。
- **Another Eden / アナザーエデン**（WFS）：「ガチャ疲れ層」を狙ったストーリー主導の RPG。King's Raid が REW で同じ層を取りに行ったのに対し、こちらは NAR で同じ層を狙った対照例。
