# ジェネシス：A New World (Genesis: A New World) 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「広大なファンタジー世界をパーティで踏破し、ボス・ダンジョン・対外戦を協力プレイで攻略する体験そのもの」にある。Krafton 傘下 Bluehole Studio の新規 IP であり、`concept` は「オンライン RPG ラインの一角を担う体験型 MMO 寄りの設計」、`target` は「MMORPG / オンライン RPG プレイヤー、特に Bluehole 既存作のファン」。Bluehole は元来 TERA を世に出したスタジオで、長期運営型 MMO の体験設計に強みを持つ系譜にある。

`social_axis` が「協力」になっているのは、MMORPG のセッション体験の中核がレイド／ダンジョン／フィールドでのパーティ＆ギルド協調にあるためで、PvP（対戦）よりも PvE 共闘が起点になっている前提を採用している。`secondary: ["REW"]` は装備ドロップ・育成・課金アイテムによる報酬層が必然的に積み上がる MMO の構造を反映しており、報酬は強い副軸だが、それを駆動するのは「ダンジョンに潜る体験」側にある。物語駆動 (NAR) ではなく世界に居続ける体験の総量が動機なので EXP が主分類になる。

## 体験の核 / 設計の発明

現時点で確定情報が乏しく、設計上の発明点は未確認である。ただし MMORPG というジャンル枠と Bluehole の系譜から、想定される設計要素を分解すると以下のレイヤに分かれる。

| レイヤ | 想定設計 | 体験への寄与 |
|---|---|---|
| 世界 | シームレスなオープンファンタジー | 「居続ける」動機の地ならし |
| パーティ | ロール分担型ダンジョン／レイド | 協力軸の中核ループ |
| 進行 | レベリング + エンドコンテンツ | リテンションの長尺化 |
| 経済 | 装備ドロップ + 取引 | REW 副軸の供給源 |
| 対外 | ギルド戦 / フィールド PvP（推定） | 協力軸の対象を「対人」に拡張 |

前世代（TERA、Black Desert、Lost Ark 等の韓国系 MMO）との差分は公表情報の段階では確認できない。協力軸の効き方としては、ソロでは到達できないコンテンツを意図的に閉じ、パーティ編成・ギルド帰属を強制することで「人と居る理由」を経済と進行の両側から作る MMO の定型に従う想定。

## バズ要因 / 人気の根拠

`popularity` は「Krafton 傘下 Bluehole Studio が手掛ける新作タイトルとして話題になったが、現時点で確かな出典に乏しく国内での認知は限定的」。これを分解する。

- Krafton 傘下という看板は PUBG の世界的成功（累計 12 億 DL 級のモバイル展開含む）の延長線で語られ、新作にメディア露出のベースラインを与える。
- Bluehole Studio は MMO `TERA` の開発元として知られ、MMORPG ジャンルファンの間で名前ベースの注目を得やすい。
- 国内（日本市場）での認知は限定的で、`sales_jp: —` / `sales_world: unknown` の通り、定量的なヒット指標は確認できない。
- 出典はスタジオ／パブリッシャの公式 IR・スタジオページに留まり、ローンチ規模・同時接続・課金ランクのいずれも一次ソースで裏付けられていない。

## 企画への示唆

- パブリッシャの看板 IP（PUBG 等）を持つ会社が新作を出すと、ジャンルが違っても「親会社の信用」だけで初期露出を獲得できる。逆に、看板を持たないスタジオの MMO 新作は同じプロモコストでは初期波が立たない。
- スタジオの過去作（TERA）をジャンル内で語り直すと、同ジャンル既存ユーザーへの想起コストがゼロに近づく。新規 IP でも「あのスタジオの新 MMO」と紐付けば、ファンダムが事前に形成される。
- MMORPG を協力軸で設計すると、リテンションは「コンテンツ追加速度」ではなく「人間関係の摩擦コスト」で決まる。ギルド離脱より転居の方が重い設計を組むと退会率が下がる。
- 体験型ジャンルで REW を副軸に置くと、装備ドロップを「協力の成果物」として配るので、ガチャ単体の摩擦を緩和できる。報酬は協力の副産物として渡す方が拒否感が低い。
- 一次ソースが揃わない段階で国内露出を仕掛けると、`popularity` 文が「話題になったが裏付けに乏しい」状態で固定される。マーケが先行しても、独立メディアの検証が後追いできる素材（KPI、開発者インタビュー）を同時に出さないと評価が定着しない。

## 比較で見るべきタイトル

- TERA（Bluehole / 2011）— 同スタジオの代表 MMO。協力軸＋アクション戦闘の前世代設計。
- Lost Ark（Smilegate）— 韓国系協力 MMO の現代型成功例。レイド設計とリテンション。
- Black Desert Online（Pearl Abyss）— オープン世界 MMO の経済・PvP 設計。
- Throne and Liberty（NCSOFT / Amazon）— 同時期の韓国系 MMO 新作。協力軸＋対外戦の現代的実装。
- Blue Protocol（Bandai Namco）— 同じく新規 IP MMO で、ローンチ後の認知獲得に苦戦した近接事例。
