# Seven Knights

## なぜ REW 主分類なのか

プレイヤーの主動機は **「500 体超の英雄を集めて、覚醒・転職・装備強化で仕上げる」** という収集と育成のループにある。concept にあるとおり、ガチャ召喚 → 育成 → 編成 → コンテンツ消費 → さらに上位の英雄を狙う、という韓国モバイル RPG のテンプレに沿った設計で、戦闘そのものはターン制コマンドという比較的浅いレイヤーに留めて、報酬ループの厚みで時間を奪う構造になっている。target が「廃課金層と、お気に入りキャラを愛でるライト層の両方をカバー」と書かれているのは、収集対象の幅（500 体超）が両端のプレイヤーを同じ経済圏に収めていることの裏返しである。

secondary に EXP が入っているのは PvP アリーナと協力レイドが用意されているためで、social_axis は非同期ランキングを軸にしつつ、対戦と協力を限定的に重ねる構成。ただし戦闘の操作スキルが勝敗を決める設計ではなく、編成されたユニットの強度＝収集と育成の成果が結果を決めるため、体験ではなく報酬が主動機側に来る。

## 報酬ループの組成

| 層 | 主要メカニクス | プレイヤー動機 |
|---|---|---|
| 収集 | 500 体超の英雄ガチャ召喚 | 図鑑埋め・推し集め |
| 育成 | 覚醒・転職・装備強化・スキル強化 | 強化曲線の踏破 |
| 編成 | パーティ構成・属性相性・スキル連携 | パズル的最適化 |
| 消費 | ストーリー / 探検 / PvP アリーナ / 協力レイド | 育成成果の検証 |
| 競争 | アリーナランキング・ギルドレイド | 上位報酬と承認 |

ガチャを単一の収集レイヤーで終わらせず、**覚醒（同名キャラの素材化）・転職（上位職展開）・装備（独立した強化軸）** で「同じキャラを引き続ける」「装備を別途回す」と二重三重に課金フックを置いているのが特徴。アリーナとレイドは育成成果の発表会として機能し、ランキングが廃課金層のリテンションを、図鑑とお気に入り編成がライト層のリテンションを担う。

## バズ要因 / 人気の根拠

- 2014 年 3 月韓国配信からシリーズ累計 1 億 DL を突破し、Netmarble の看板 IP に成長
- 日本では 2016 年配信開始から **10 日で 100 万 DL、約 1 か月で 150 万 DL** という当時の韓国産モバイル RPG として速いスタートダッシュ
- 500 体超のキャラクター数が、コラボ展開（多数の他 IP とのクロスオーバー）の受け皿として機能
- 2023 年に日本版サ終後、2025 年に正統リメイク『Seven Knights Re:BIRTH』が登場し、IP としての再起動が走った

## 企画への示唆

- **収集対象を 500 体規模まで広げると、廃課金とライトの両方を同じ経済圏に収められる** — 図鑑価値と編成最適化の両軸でリテンションが立つ
- **ガチャの上に覚醒・転職・装備の独立強化軸を重ねると、課金フックが直列ではなく並列で増える** — 同名キャラ重ね引きを「無駄」ではなく「素材化」と再定義する
- **ターン制コマンドという浅い戦闘層に育成深度を載せると、操作スキル差で離脱するプレイヤーを抑えられる** — 勝敗の根拠を編成と強化に寄せる
- **PvP アリーナのランキングと協力レイドを併設すると、社会軸が「非同期＋対戦＋協力」のハイブリッドになり、月次イベント設計の選択肢が増える**
- **サ終から数年後のリメイク投入は、累計 DL を持つ IP なら有効な再起動戦略になりうる** — Re:BIRTH のように「正統」を冠することで旧プレイヤーの帰還を促す
