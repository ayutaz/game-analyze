# ASTRA: Knights of Veda — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「ダークファンタジー世界観のキャラクターをガチャで引き、育成して編成を組む」こと、つまり**集める／育てる**にある。FLINT が 6 年以上開発した世界『プラニス』を舞台にしているが、配信モデルとマネタイズ動線は基本に忠実な縦軸ガチャ ARPG であり、ベルトスクロール戦闘と豪華フルボイス・ストーリーはガチャで引いたキャラを動かして消費するための「演出層」として機能している。

`target` に「原神／崩壊シリーズ系の縦軸ガチャ ARPG に飽きた層」と明示されている通り、競合の前提自体が REW 主軸のガチャ ARPG で、ASTRA はその文脈に対し「2D 美麗アート × 横スクロール戦闘」という差分で殴り込む立て付けになっている。`secondary` に EXP（回避とスキル連携が要求される手触り重視の戦闘）と NAR（豪華フルボイスのダークファンタジー物語）が両方入るのは、ガチャ ARPG では戦闘の手触りと物語演出がそのままキャラ排出への投資意欲を支える副動機になるためである。

## 報酬ループの組成

縦軸ガチャ ARPG として、ASTRA は「キャラガチャ → 戦闘で性能確認 → ストーリー消費 → 育成素材ファーム → 次ガチャ」というループを 2D ベルトスクロール戦闘の上に組んでいる。3D オープンワールド系の競合と異なり、移動・探索コストが小さく**戦闘とガチャ排出の距離が近い**のが構造的特徴となる。

| 層 | 実装 | 動機への寄与 |
|---|---|---|
| 排出層 | キャラガチャ（IP 単発） | コレクション欲求／推し排出 |
| 演出層 | 2D 美麗アート＋フルボイス・ストーリー | 引いた価値を「物語の中で見たい」 |
| 操作層 | 2D ベルトスクロール、回避＋スキル連携 | キャラ性能差を手触りで実感 |
| 育成層 | 強化素材ファーム／編成 | 周回モチベ＝次ガチャ準備 |
| 配信層 | PC／iOS／Android クロスプレイ | デバイスを跨いだ可処分時間の確保 |

ベルトスクロールという**画面が固定方向に流れる**形式は、3D アクションよりキャラのモーション・必殺技演出が中央に大写しになりやすく、ガチャで引いた個体の「絵としての価値」を提示しやすい。これは縦軸ガチャ ARPG の中では原神型（探索＋ARPG）より、アズールレーン／NIKKE 型（戦闘場面が固定構図でキャラ立ち絵を見せる）に近い演出設計と言える。

## バズ要因 / 人気の根拠

`popularity` の核は「HYBE IM 初の本格 RPG として注目、事前登録 1 週間で 100 万人突破」である。これを分解すると:

- **配信元の話題性**: HYBE IM は BTS 所属レーベル HYBE 傘下のゲーム部門で、K-POP IP 周辺からゲームに流入する関心層を初期母数として確保。
- **事前登録の速度**: 2024 年 3 月、サービスインに先立つ事前登録で約 1 週間で 100 万人突破。配信前の関心量として韓国・日本のモバイル／PC ガチャ ARPG では上位水準。
- **競合との差分提示**: 原神／崩壊系の縦軸ガチャ ARPG が 3D オープンワールドに集中していた市場に、2D ベルトスクロール × ダークファンタジーという別軸で参入。
- **マルチプラットフォーム**: PC（Steam）／iOS／Android のクロスプレイで初日から提供され、Steam ストアでもダークファンタジー 2D ARPG として可視化された。

## 企画への示唆

- **コアプラットフォームに対し「ジャンル軸の差分」で参入**すると、後発でも「○○系に飽きた層」という明確な訴求文を書ける（ここでは 2D ベルトスクロール × ダークファンタジーが原神型 3D ARPG への対立軸）。
- **配信元 IP の関心層を初期母数に流用**すると、ゲーム単体の純広告投下より事前登録の立ち上がりが速い（HYBE IM × HYBE レーベル経由の動線）。
- **ガチャ ARPG で 2D を選ぶ**と、3D 探索コストを切り捨てて「戦闘とガチャ排出の距離」を縮められ、引いたキャラを即座に演出層で見せられる。
- **ベルトスクロール／横スクロール構図**は、キャラ立ち絵とモーションが画面中央に大写しになるためガチャ排出個体の「絵としての価値」を強調できる。
- **長期開発（6 年以上）の世界観 IP**を最初から提示すると、ストーリー駆動の副動機（NAR）が REW 主軸を補強し、無課金プレイヤーの離脱を緩和できる。

## 比較で見るべきタイトル

- 原神 / 崩壊スターレイル（縦軸ガチャ ARPG の 3D オープンワールド側、ASTRA の競合前提）
- アズールレーン / NIKKE（2D／固定構図でキャラ立ち絵を中心に置くガチャ × 戦闘演出）
- ドラゴンファング Z / モンハンライダーズ系（2D／縦持ち寄りのアクション × ガチャ）
- ELYON / TL: Throne and Liberty（同じく韓国系スタジオの IP × ガチャ／成長型）
