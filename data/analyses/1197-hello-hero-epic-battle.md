# Hello Hero: Epic Battle 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「コミカルな 100 体超のヒーローをガチャで集め、5 体パーティに組んで育てる」こと——つまり**集める／育てる**にある。本作は片手・縦持ち・オート進行という UI 設計で戦闘の操作密度を意図的に下げており、プレイ時間の大半は「ヒーロー獲得 → 強化 → 編成最適化 → 周回」のループに吸収される。concept にある「5 体パーティ／100 体超のヒーロー収集」と target の「ヒーロー収集と PvP 両方を楽しみたいカジュアル〜ミドルコア層」は、いずれも収集動機を主、戦闘体験を従に置く REW 設計の典型である。

secondary に EXP が立っているのは、Adventure / Expedition / Conquest / World Boss / Battlefield と提供モードが多軸で、特に PvP 系（Battlefield・Conquest）と World Boss が「集めた手駒を試す場」として機能するため。social_axis は null（モード横断で対戦・協力・非同期が混在し、主軸を一本化できない）だが、運用上は非同期ランキング寄りに読める。

## 報酬ループの組成

| 層 | 仕組み | プレイヤー動機 |
|---|---|---|
| 入手 | ガチャによる 100 体超ヒーロー収集（3D モデル） | コレクション充足、推し獲得 |
| 育成 | レベル / 装備 / スキル強化 | 手持ち戦力の底上げ |
| 編成 | 5 体パーティ最適化、属性・役割の組合せ | 思考の余地、メタ追従 |
| 消費 | Adventure（PvE 周回）／ Expedition（自動派遣）／ World Boss（協力火力出し） | 育成成果の即時消費 |
| 競争 | Conquest / Battlefield など PvP・サーバ対抗 | ランキング・季節報酬 |

このうち Expedition（放置派遣）と Adventure オート周回が「片手・縦持ち」運用と直結しており、**接触頻度は高いが 1 セッションは短い**という、通勤・休憩時間に最適化された設計になっている。一方で PvP と World Boss が「育成成果を可視化する出口」として機能し、ガチャ → 育成 → ランキング登攀という縦の動機線を成立させる。

## バズ要因 / 人気の根拠

- 原作『Hello Hero』（前作）が**全世界 2,000 万 DL 超**の実績を持ち、IP 認知を継承した状態でローンチできた。
- リリース後 **49 か国でチャート入り**しており、コミカルなアートと多言語ローカライズで地域横断のリーチを取った。
- 日本では **2019 年 5 月にローカライズ配信**、縦持ち片手操作・**フルボイス対応**で参入し、4Gamer などの専門メディアで告知された。
- 全世界累計 **2M+ DL** という規模感は AAA モバイルには届かないが、Fincon（韓国系インディー〜中堅）規模としては 49 か国展開を支える基盤になっている。

## 企画への示唆

- **片手・縦持ち・オートを前提にすると、プレイ時間あたりの操作量が下がる代わりに 1 日あたりの起動回数が増える**。リテンション設計を「セッション長」ではなく「タッチポイント数」で組むべき。
- **ガチャ対象が 100 体超かつ 3D 化されていると、コレクション欲求と推し可視化（モデル鑑賞）が両立する**。2D カード型より制作コストは上がるが、配信・スクショ拡散の燃料になる。
- **PvE 周回（Adventure）／放置派遣（Expedition）／対人（Battlefield）／レイド（World Boss）の 4 出口を並列で持つ**と、育成成果の消費先が分散しプレイヤーの飽き耐性が上がる。
- **前作 IP の認知を 3D リメイクで継承する**設計は、新規 IP 立ち上げよりローンチ初動の摩擦を下げる。続編／リメイクは「アート刷新＋多言語先行」の組合せでリーチを稼げる。
- **49 か国同時運用を想定するなら、ローカライズは単なる翻訳ではなく UI（縦持ち）＋音声（フルボイス）まで地域別に積み増す**ことで、後発市場での話題化に直結する。

## 比較で見るべきタイトル

- Summoners War: Sky Arena（Com2uS）— 3D ヒーロー収集 RPG の先行格、PvP アリーナ依存度の比較対象。
- セブンナイツ（Netmarble）— 5 体パーティ × コミカル 3D × ガチャの構成が近い。
- Epic Seven（Smilegate）— 2D だが「ヒーロー収集 + 多モード（World Boss / アリーナ）」の出口設計が参照軸。
- 前作『Hello Hero』（Fincon, 2013）— IP 継承元、2D → 3D リメイクの差分研究に有用。
