# Heir of Light

## なぜ REW 主分類なのか

プレイヤーの主動機は **「5 人のマスターと 100 体超のサーヴァントをガチャで集め、属性・スキル・装備の組み合わせを最適化して PvP ランキングを登る」** ことに集約される。concept にある「ゴシック・ダークファンタジー世界観の収集型 RPG」「マスター + サーヴァントのコンボ構築」は、ストーリーや戦闘体験ではなく **集める／育てる／序列化する** という収集系報酬ループそのものを軸に置く設計だ。target も「Summoners War / Epic Seven 系の韓国産ガチャ RPG を好むコアモバイル層」「PvP ランキングとキャラ収集をやり込みたい層」と明示しており、ガチャ収集と PvP 序列が主要動機である点に揺れがない。

`social_axis` が未設定なのは、本作の主戦場であるアリーナ PvP が「対戦」（リアルタイム同期）ではなく **非同期ランキング戦** であり、かつ収集要素が主動機側に寄っているためカテゴリ境界に位置するからだ。secondary に **EXP** が入っているのは、オート戦闘の上に「特殊技手動発動でコンボを組み上げる」操作層が乗っており、純然たる放置ガチャより一段プレイヤースキルを要求する点を反映している。

## 報酬ループの組成

| 層 | 要素 | リテンション機能 |
|---|---|---|
| ガチャ | 100 体超のサーヴァント + 5 マスター、属性 5 種 | 排出ロングテール、限定実装で更新性確保 |
| 育成 | レベル / 覚醒 / 装備強化 / ルーン | 1 体あたりの投資時間を伸ばし周回動機を継続 |
| 編成 | マスター × サーヴァントのスキル連携、属性相性 | コレクションに「使える組み合わせ」の意味を付与 |
| 周回 | ストーリー / ダンジョン / 素材ステージ | スタミナ消費＝日次ログイン要因 |
| 序列 | アリーナ PvP（非同期）ランキング | 既存資産の優劣を可視化、ガチャ動機を再生産 |

中核は「ガチャで引いた素材を、装備・ルーン・覚醒で複利的に強化し、その強さを **非同期 PvP の数値ランキング** で他プレイヤーと比較する」というクラシック韓国産ガチャ RPG の構造。Summoners War が確立し Epic Seven が継承したこのループに、本作は **ゴシック / ダークファンタジーのアートディレクション** という差別化軸を載せている。

## バズ要因 / 人気の根拠

- グローバル累計 **3M+ DL**。GAMEVIL 配給で多言語同時配信、英語圏メディア（TouchArcade / Android Headlines）に 2018 年初頭にレビュー掲載され、英語コアモバイル層に到達した
- **Fire Emblem Heroes 系の縦持ち UI** を採用し、片手プレイ前提のガチャ RPG として通勤・空き時間に最適化
- 韓国産ガチャ RPG が **明るい王道ファンタジー（Summoners War / Seven Knights / Epic Seven）** に寄っていた市況下で、ゴシック・ダーク基調のビジュアルでニッチ層を取りに行ったポジショニング
- マスター + サーヴァントの二層編成という、シングルキャラ集めの一歩外側にある **「主従ペア構築」** という設計フックを提示

## 企画への示唆

- 既存ジャンルのレッドオーシャンに参入する際は、**アートディレクション軸（ゴシック / ダーク / 退廃）** で 1 軸だけずらすと、レビュー側に取り上げ角度を与えられる
- 縦持ち UI を採用すると、ゲームプレイ時間が「通勤・休憩の細切れ」枠に入り、セッション数 × 短時間という DAU 構造を作れる
- ガチャの単位を「単キャラ」から「**マスター + サーヴァント** のペア」にずらすと、組み合わせ爆発で同じ排出数でも体感の収集量を増やせる
- オート戦闘に「**特殊技だけ手動発動**」の浅い操作層を載せると、放置プレイ可能性を保ちつつ EXP（操作体験）を secondary に確保できる
- 非同期 PvP ランキングを終盤コンテンツに据えると、ガチャ投資の **数値的還元先** が常に存在し、引いたキャラに「使い道」が生まれる

## 比較で見るべきタイトル

- **Summoners War: Sky Arena**（Com2uS, 2014）— 韓国産非同期 PvP ガチャ RPG の原型
- **Epic Seven**（Smilegate, 2018）— 同年配信のアニメ調 2D ガチャ RPG、競合ポジション
- **Fire Emblem Heroes**（任天堂 / インテリジェントシステムズ, 2017）— 縦持ち UI とユニット収集の参照元
- **Seven Knights**（Netmarble, 2014）— GAMEVIL/Com2uS と並ぶ韓国モバイル RPG の系譜
