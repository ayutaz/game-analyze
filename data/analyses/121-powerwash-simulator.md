# PowerWash Simulator

## なぜ EXP 主分類なのか

プレイヤーの主動機は「汚れを落とす作業そのものを淡々と味わう」こと。物語の続きを読みたいわけでも、レアアイテムを集めたいわけでもなく、ノズルを当てた瞬間に塗装色が露出する反応そのものが報酬になっている。concept にある「高圧洗浄機で汚れを落とすことだけに特化した一点突破の作業シミュレーター」という設計は、ゲーム内通貨や装備強化を主動機にする REW とも、台詞やカットシーンで駆動する NAR とも明確に異なる。

social_axis は「協力」。target にある「Co-op で雑談しながら遊ぶフレンドプレイヤー層」が示す通り、最大 6 人の同時清掃は競争ではなく作業の分担と進行ペースの共有として機能している。物語要素（街の謎や顧客のメール）は薄く挿入されるが、あくまで作業の合間の調味料であり、secondary に何も入っていないのはこの一点突破設計をデータ上も裏付けている。

## 体験の核 / 設計の発明

中核メカニクスは「画面内の汚れ％を 100 → 0 に近づける」だけ。だがこの単純さの裏に、複数の小さな知覚的フィードバックが層構成されている。

| 層 | 設計要素 | プレイヤーへの効果 |
|---|---|---|
| 視覚 | 汚れが剥がれて下地色が現れる即時反応 | 「やった分だけ確実に進んだ」感覚 |
| 聴覚 | 水流音・対象素材ごとに変わる打音 | ASMR 的な没入、長時間プレイの摩耗低減 |
| 進捗 | 部位ごとの％表示・チェックリスト | 中断・再開が容易、寝落ちしても損が出ない |
| 拡張 | ノズル切替・薬剤・角度調整 | 単調化を防ぐ最小限のスキル要素 |
| 社会 | 最大 6 人 Co-op、ボイスチャット任意 | 作業 BGM 的な雑談プレイ |

前世代の作業系シミュ（House Flipper、Viscera Cleanup Detail 等）との差分は、目的を「掃除完了」ではなく「画面が綺麗になる過程の知覚」に置き直した点にある。タイマー・スコア・失敗条件をほぼ削ぎ落とし、退場のハードルを最小化した。

## バズ要因 / 人気の根拠

popularity にある「配信映え・ASMR 系で実況人気、寝落ちゲーとして話題」を分解する。

- 世界累計 700 万本超（2024 時点）に到達。地味なジャンルとしては異例の規模で、Square Enix Collective の publisher 案件としてもヒット作。
- 配信での「作業画面 + 雑談」フォーマットと相性が良く、Twitch/YouTube で長時間配信のジャンルとして定着。視聴体験が能動操作とほぼ同じ快楽を提供する珍しい構造。
- ファイナルファンタジー VII / SpongeBob / Tomb Raider 等の IP コラボ DLC が継続投入され、原作ファンを定期的に呼び戻すサプライズ要素として機能。
- Game Pass / PS Plus 等のサブスク配信で初回接触のハードルが下がり、口コミ経由で「寝る前に流すゲーム」として定着。

## 企画への示唆

- 目的をスコアや勝敗ではなく「知覚的フィードバックそのもの」に置くと、失敗概念のないリラックス用途が成立する。
- 進行が％・チェックリストで可視化されていると、プレイヤーは中断と再開のコストを意識せず、結果として長時間滞在する。
- 操作対象（汚れ）と発音体（水流音）を直結させると、画面を見ない配信視聴でも体験が成立し、二次的な集客チャネルになる。
- 任意ボイスチャットの Co-op は「協力ゲー」というより「共在ゲー」として設計でき、競争要素を入れない方がフレンド層の滞留率は伸びる。
- 主動機を一点に絞ったうえで IP コラボ DLC を継続投入すると、本体側の設計を変えずにロングテール売上を伸ばせる。
