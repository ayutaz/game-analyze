# ニューステート・モバイル / New State Mobile (2021)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「100 人のスマホシューターを相手に最後の 1 人として勝ち抜く」ことであり、撃ち合いと立ち回りの体験そのものが目的化している。`social_axis: 対戦` がそのまま動機そのものを指しており、報酬は次のマッチを有利にするための手段に過ぎない。

`concept` にあるとおり、本作は PUBG IP を 2051 年「Troi」マップに延長し、ドローン、EMP 手榴弾、車両カスタマイズなど未来兵器をルールに重ねた構造を取る。`target` も「スマホで本格的な FPS/TPS バトルロイヤルを遊びたい 10–30 代コアシューター層」と明示されており、ガチャや物語を消費しに来るプレイヤーではなく、操作技術を競技として楽しみに来るプレイヤーが想定読者である。よって主分類は EXP（対戦）。

`secondary: REW` は副次的に効いている。武器スキン、車両スキン、シーズンパス、サバイバーパスといった収集系メタゲームが用意され、対戦の合間にコレクション動機が回るが、これは「次に持ち込む武装の見栄え」を整えるための周辺装置であり、勝敗そのものを左右する報酬ではない。

## 体験の核 / 設計の発明

中核メカニクスは「100 人 → 1 人」の縮退ルールを維持しつつ、近未来ガジェットでスマホ特有の操作粒度に合わせた立ち回りを足したこと。原作 PUBG が「現代軍ミリタリー × PC・コンソール精度」だったのに対し、本作は「近未来 SF × タッチ操作前提」へとレイヤーをずらしている。

| 設計要素 | 既存 PUBG（2017）からの差分 | スマホ EXP としての効果 |
|---|---|---|
| 舞台 | 現代 → 2051 年「Troi」 | 未来兵器の導入に正当性を与え、IP 拡張の口実になる |
| ドローン | 無 → 偵察・投下要請 | スマホでも視界優位を取りやすく、エイム以外の勝ち筋を増やす |
| EMP 手榴弾 | 無 → 車両・電子機器を一時無力化 | 車両ラッシュ一辺倒の安全プレイを抑制 |
| 車両カスタマイズ | 固定 → パーツ装着 | 収集（REW）と立ち回り（EXP）を両端で接続 |
| 入力前提 | KB&M / コントローラ | タッチ + ジャイロ最適化 | 

社会軸「対戦」は 100 人スクワッド／デュオ／ソロのマッチング設計で実装されており、ボイスチャットや味方マーカーで野良編成でも体験が成立するよう調整されている。

## バズ要因 / 人気の根拠

`popularity` の「プレ登録 5500 万件超、配信約 1 か月で 5000 万 DL、日本では渡辺直美をアンバサダー起用」を分解する。

- **プレ登録 5500 万件超**: 既存 PUBG Mobile プレイヤー基盤を「次世代版」として一気に流し込むマーケティング設計が機能した。
- **配信約 1 か月で 5000 万 DL** (`sales_world: 50M+ DL`): モバイル BR としては立ち上がり最速級。後続 KPI（DAU・課金 ARPPU）に直結する初動を確保した。
- **渡辺直美アンバサダー**: 日本市場でコアシューター層の外側、ライトカジュアル層・女性層への到達を狙った起用。FPS が苦手とする「初心者の触る理由」を作る役割。
- **KRAFTON / PUBG Studios の自社開発**: 外部ライセンスではなく本家が出した「正史スピンオフ」であることが、コアファンの離脱抑止に効いた。

## 企画への示唆

- **既存 IP を別時代に飛ばすと、ルール改変の正当性が得られる**: 現代 → 近未来のジャンプは「なぜ EMP 手榴弾やドローンがあるのか」を説明不要にし、ゲームデザイン側の自由度を上げる。
- **スマホ BR は「視界優位」と「機動優位」のどちらかを足すと差別化が立つ**: New State はドローンで前者、車両カスタムで後者を補強した。
- **プレ登録 KPI を 8 桁台に乗せると配信初月の DL 山が安定する**: 5500 万プレ登録 → 5000 万 DL の転換率約 9 割は、IP 強度と事前訴求が噛み合った結果。
- **ローカルアンバサダーはコア層ではなくその外側に向ける**: 渡辺直美起用は「既存 FPS ファン向け」ではなく「FPS を遊ばない層に名前を届ける」役割で効いた。
- **競技性ゲームの REW レイヤーは勝敗に効かせない**: スキン・パスを実弾性能から切り離すことで、対戦 EXP の純度を保てる。

## 比較で見るべきタイトル

- **PUBG Mobile (2018)**: 同 IP の現代版モバイル、本作の比較対象であり競合自社作。
- **Call of Duty: Mobile (2019)**: 同じ「コンソール IP のスマホ BR / マルチ化」路線。
- **Apex Legends Mobile (2022)**: キャラクターアビリティ層を足した別解、本作との設計差を見るのに適する。
- **荒野行動 (2017)**: 日本市場での先行モバイル BR、ローカルプロモの蓄積で先行していた競合。
- **Fortnite (モバイル版)**: 建築という別軸の「機動優位」を足したケース、未来兵器ではなくクラフトで差別化した例。
