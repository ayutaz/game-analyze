# ラミー・サークル (RummyCircle) — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「13 枚ラミーの腕で実マネーを獲得する」こと。卓に座る理由が娯楽そのものではなく **賞金キャッシュアウト** に固定されている時点で、本作は遊技体験ではなく報酬獲得型プラットフォームとして設計されている。concept にある「24 時間いつでも現金トーナメント／キャッシュテーブル」「勝てば実マネーが引き出せる仕組みが主動機」という記述は、REW（報酬型）の典型的な定義（ゲーム内行動 → 確率的・腕前依存の報酬 → 次の参加動機）にそのまま重なる。

target が「カード勝負の腕を現金収入に変えたいスキルゲーマー層」と「家族で長年慣れ親しんだラミーを隙間時間に遊ぶカジュアル層」の二層構造になっており、前者は完全に賞金駆動、後者も「低額エントリーで小銭が増える」リテンションに支えられている。社会軸 (`social_axis`) が null なのは、相手は常に他人だが恒常的な「対戦コミュニティ」ではなく **その都度マッチングされる金銭卓の対面**だからで、ソシャゲ的な協力／非同期軸では捉えられない。

secondary に EXP が入るのは、ラミーそのものが捨て札読み・確率計算・ブラフを含むスキルゲームで、勝率が腕に比例する点。報酬を回す土台として「実際に上手いと勝てる」体験が必要で、これが運だけの賭博と差別化する命綱になっている。

## 報酬ループの組成

リアルマネー・ラミーの報酬ループは、ガチャ型 REW とは別系統の「腕→賞金→再投資」型である。階層を分解する。

| 層 | 仕掛け | プレイヤー動機 |
|---|---|---|
| エントリー層 | Free Practice テーブル / ウェルカムボーナス | ノーリスクで操作とルールを習得、登録 KYC への誘導 |
| キャッシュ層 | 低額 Cash Games (₹1〜) と Pool / Points / Deals 形式 | 小額で勝てる原体験を作り、入金行動を発火 |
| トーナメント層 | 24h 開催の現金トーナメント・週次大型大会 | 賞金プール上位入賞による「労働対価」型の報酬 |
| ロイヤルティ層 | レベル別キャッシュバック・リロードボーナス | 入金 → プレイ → 入金の循環を金銭で再強化 |
| 信頼層 | iTech Labs 認証 RNG・不正検出・KYC | 賭けた金が回収可能であるという前提を担保 |

ガチャ REW が「ハズレを引き続けて出費が積み上がる」設計なのに対し、本作は「上手ければ理論上はプラス収支」というスキル前提の建付け。ただし手数料（rake）が常にハウス側に流れるため、勝者上位以外は長期的にマイナスサムになる構造は変わらない。リテンションを支える本質は **賞金プールの大きさそのもの**で、登録 5,000 万人超・2023 年 50 億ゲーム以上というプレイ密度が、各卓の即時マッチング体験と高額トーナメント賞金を両立させている。

## バズ要因 / 人気の根拠

- 登録ユーザー **5,000 万人超**、2023 年だけで **50 億ゲーム以上** がプレイされたと自社発表。インドのオンライン・スキルゲーミング市場で実質的なデファクト
- 運営の Games24x7 は本作と同社のファンタジースポーツ My11Circle を中核に **2022 年にユニコーン入り**（評価額 10 億ドル超）
- インドで「ラミー」は家庭・祭事で何世代も遊ばれてきた国民的カードゲームで、新規ルール学習コストがほぼゼロ。ローカル文化資産をデジタル化した先行者利益が大きい
- 2017 年インド最高裁判決でラミーが「実質的スキルゲーム」と認定され、賭博規制の主要例外として運営が可能になった法的追い風

## 企画への示唆

- **既存の国民的アナログ遊戯 × 24h マッチング**を組むと、ルール教育コストを払わずに巨大ユーザー基盤を取りに行ける
- 報酬を「ガチャ確率」ではなく「腕前 × 賞金プール」に置くと、課金抵抗の異なる層（賭博嫌悪／投資思考）も取り込める一方、法規制リスクが主要な参入障壁になる
- KYC・RNG 認証・不正検出を**プロダクトの一機能ではなく信頼インフラ**として前面に出すと、リアルマネー領域の入金転換率が上がる
- フリープラクティス → 低額キャッシュ → トーナメントの **3 段ファネル**を明示すると、カジュアル層からハイローラー層への自然な引き上げが起きる
- 単一ゲーム専業（ラミー特化）でも、対象市場が大きく文化資産が固ければ MMO 級の同時接続を生み出せる
