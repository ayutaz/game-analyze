# リベンジ・オブ・ザ・ブレイブ (Revenge of the Brave) 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「集めて・育てて・所属ギルド内で序列を上げる」こと。Manya / Deva / Titans 三勢力のキャラ、ペット、騎乗獣、鳥（Avians）と育成対象が層状に並び、ターン制バトルはその育成成果を確認するための「測定装置」として機能する。動詞で言えば「戦う」ではなく「組成を磨く」。

concept には「ダンジョン襲撃・PvP・ボス戦・宝探し」と消費先が並ぶが、いずれも報酬を回収して育成キューに戻す経路として設計されており、target 文の「ターン制 MMORPG とギルド戦に時間を投じる 18-35 歳男性」という記述も、長時間の周回と課金で報酬密度を上げるプレイヤー像と整合する。secondary に EXP が入っているのは、League（ギルド）戦の協力プレイ自体に体験価値が成立しているため。ただし主動機はあくまでキャラ・ペット・装備のスタック構築であり、PvP / 協力戦は「育てたキャラを試す場」に従属している。

## 報酬ループの組成

| 層 | 収集対象 | 投下リソース | リテンション役割 |
|---|---|---|---|
| キャラ | Manya / Deva / Titans 各陣営の英雄 | ガチャ / イベント配布 | アカウントの基礎戦力、初期離脱防止 |
| ペット / 騎乗獣 / Avians | 補助ユニット 3 系統 | 餌・素材・進化アイテム | 中期の差別化、ニッチ収集動機 |
| 装備・スキル | ボス戦 / ダンジョン報酬 | スタミナ / 周回時間 | 日課ログイン理由 |
| League（ギルド） | メンバー枠・拠点強化 | 寄付・参加時間 | 長期離脱抑止、ソーシャル接着 |

ターン制という枠を選んだことが報酬設計上の核で、操作スキルの分散を抑え、戦力 = 育成投資の関数として読みやすくしている。アクション MMO 系より「課金額と編成知識でスコアが決まる」見通しが立つため、MENA 圏の入門 MMO 層が脱落しにくい。さらに 3 系統の補助ユニット（ペット・騎乗獣・Avians）を独立スタックで持たせ、メイン英雄のガチャに飽きた層が横方向に収集を拡張できる「サブ収集ライン」を確保している。

## バズ要因 / 人気の根拠

- 配信直後にアラブ圏 RPG カテゴリのチャート上位入りした、地域特化ヒット。
- Tamatem Games（ヨルダン拠点、MENA 向けローカライズで実績ある publisher）が初めて自社開発した本格 MMORPG という文脈で、地域メディアの注目を集めた。
- アラビア語フル翻訳・右から左（RTL）UI 対応で配信され、機械翻訳止まりの他作品と差別化された。
- MENA 圏のスマホ MMO は中華タイトルの再ローカライズが多い中、現地スタジオ製・現地読者向けに調整されたコンテンツとして話題化。

## 企画への示唆

- アラビア語フル翻訳と RTL UI を「最初から」設計に入れると、機械翻訳ベースの他社作品との差別化が単発のマーケ施策より持続する。
- ターン制 MMORPG はアクション MMO より戦力可視化が容易なので、入門層 / 重課金層の両方が同じ尺度を共有しやすい。新規流入を狙う地域では特に有効。
- メイン英雄ガチャに加え、ペット / 騎乗獣 / Avians のような「独立した収集ライン」を 2-3 本走らせると、メイン枠が飽和したユーザーの脱落を遅らせられる。
- League（ギルド）戦を中盤以降の主要消費先に据えると、ソーシャル接着で個別ユーザーの離脱コストを引き上げられる。
- 地域特化タイトルは「現地スタジオ製」というナラティブそのものが PR 素材になる。開発体制を見せるコンテンツマーケが効きやすい。

## 比較で見るべきタイトル

- Lords Mobile / Last War: Survival Game（MENA で強い課金力を持つストラテジー、地域別 ARPU の参考）
- リネージュ2 レボリューション / リネージュM（重課金 MMO の League / 攻城戦リテンション設計）
- AFK Arena（多系統キャラ収集と放置型育成の組成）
- ロマサガ リ・ユニバース（ターン制バトル × キャラ収集の RPG 設計、日本市場版）
