## なぜ NAR 主分類なのか

OMORI のプレイヤー主動機は「主人公 SUNNY が抑圧した記憶を **読み解くこと**」であり、戦闘や収集ではなく **物語の断片を回収していく動詞** が中心にある。concept にあるとおり「可愛いドット絵世界と現実パートが交錯し、徐々に主人公の抑圧された記憶が露呈する構造」が本作の骨格で、RPG ツクール製の戦闘システムは物語を進めるためのペース配分として機能している。target の「Undertale / Mother 系を好む考察・二次創作好き」という層も、戦闘の上手さではなく **テキストとビジュアル表現から意味を抽出する遊び方** を想定しており、これは NAR の典型読解動機に合致する。

副次の EXP は、HEADSPACE（夢世界）と REAL WORLD（現実）の往復に伴う **感情ステート（HAPPY / SAD / ANGRY / AFRAID）を使った戦闘ギミック**、および JRPG パロディ的な探索体験に現れる。物語駆動を阻害しない範囲で、ターン制 RPG としての手応えが副次的に張られている設計である。social_axis は null（シングル前提）。

## 物語の駆動構造

OMORI の物語は「**夢世界 = HEADSPACE / 現実 = REAL WORLD** の往復」と「**3 つのルート分岐（SUNNY / OMORI / 隠しエンド）**」の二軸で駆動される。プレイヤーは可愛い世界で友人と冒険するうち、徐々に「なぜ SUNNY は外に出られないのか」「親友 MARI に何が起きたのか」を断片で渡され、終盤で抑圧の正体が露呈する。

| 構造要素 | 役割 | 物語駆動への効き方 |
|---|---|---|
| HEADSPACE / REAL の二層 | 同じキャラの理想像と現実像を並置 | ギャップが進むほど不穏さが累積する |
| 感情ステート戦闘 | HAPPY/SAD/ANGRY の三すくみ | 戦闘テキストにキャラの心理が漏れる |
| 3 日構成の現実パート | カレンダー型ペーシング | 「外に出る／出ない」選択が分岐を決める |
| 隠し HIKIKOMORI ルート | 全期間引きこもり続行で到達 | 二週目以降の考察エンジン |
| ホラー演出の挿入 | 可愛さからの落差 | SNS スクショ拡散の最大の燃料 |

特筆すべきは、戦闘テキストや NPC 会話にまで **抑圧された記憶の伏線が散らされている** 点で、考察勢が二週目で再走する動機を構造として埋め込んでいる。

## バズ要因 / 人気の根拠

popularity の「考察・トラウマ系 SNS バズとファンアート文化」を分解すると：

- **可愛い見た目 × 重いテーマの落差** が Twitter/TikTok でスクショ単体で機能し、文脈を知らないユーザーの流入経路になっている
- **RPG ツクール製・小規模スタジオ OMOCAT 発** という出自が「インディーの当たり」物語として語られやすく、Steam で 1M+ を到達した PC 初発 → 2022 にコンソール／モバイル展開と段階的に話題を更新できた
- **二次創作の量産しやすさ**：キャラデザインが線が少なくファンアート化しやすく、HEADSPACE 側の世界観が pixiv / X の創作テーマとして反復された
- **考察ジャンルとしての成立**：隠しエンドや小ネタが多く、YouTube / ニコニコの解説動画が長期テールを作った

## 企画への示唆

- **可愛い表層と重いコアテーマを意図的に落差で配置すると、スクショ単体が SNS で機能する** → 物語の文脈を知らない人の流入装置になる
- **二層世界（理想 / 現実）の往復構造を採ると、同じキャラを 2 回設計する手間と引き換えに考察動機を作れる** → 二週目以降のリテンションを物語側で稼ぐ
- **戦闘システムを物語のペース配分装置として割り切ると、JRPG 文法を知らない層にも読み物として届く** → 戦闘難易度はテキスト摂取の邪魔をしない程度に抑える
- **隠しルートを「特定の日常選択を続けると到達」型にすると、考察コミュニティが攻略ではなく解釈で盛り上がる** → 動画／note 等の長期コンテンツ供給源になる
- **RPG ツクール／小規模ツールで作っても、テーマ密度とアートディレクションが揃えば 100 万本級に届く** という事例として企画通過の論拠に使える
