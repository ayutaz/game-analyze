# クラッシュバンディクー オン・ザ・ラン！ — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「右へ走り、左右スワイプとジャンプで障害物を捌き続ける反射体験」を回す部分にある。Subway Surfers 系のランナーをベースに Crash 原作の箱割り・果物取り・ボス戦演出を被せた構造で、コア体験は秒単位のスワイプ判断にある。

`concept` が「ステージ感をエンドレスランナー化」と明記する通り、原作 PS1 の縦軸走行（『Boulders』ステージ等）を抜き出し、F2P モバイル向けに最小単位へ圧縮した設計。`social_axis` は **非同期** で、チーム所属によるイベントランキングや素材プレゼント機能はあるが、リアルタイム対戦は持たない。あくまで「走る／高得点を出す／クラフトする」というソロ操作を、チーム文脈で持続させる構造である。

`secondary: ["REW"]` が示すように、武器（ガジェット）クラフト・スキン解放・島の解放がメタ層に重なる。ただしレースランナーである以上、報酬は「次の走りで使うため」のものであり、収集自体が一次動機にはなりにくい。

## 体験の核 / 設計の発明

| レイヤー | 仕組み | 効果 |
|---|---|---|
| 操作 | スワイプ・スピン・ジャンプ・スライディング | 1 セッション 30 秒〜2 分の高頻度反射 |
| ステージ | 島ごとに固定ボス（リッパー・ルー、スカルキャットほか）を配置 | エンドレスではなく「クリア可能な区切り」を持ち込む |
| クラフト | 走行で集めた素材 → ボス耐性に対応する武器・薬品を合成 | セッション間に育成タスクを置き、再起動動機を作る |
| チーム | 加入して素材リクエスト／プレゼント、共同イベント | リアルタイム同期なしで他者の存在を実感させる |
| F2P | 体力制ではなく素材待ち＋スキン課金 | 走行回数を絞らず、収益はメタ層に逃がす |

純粋なエンドレスではなく **「ランナー × ボスバトル × クラフト」** の三層構造で、Subway Surfers のセッションループに Clash Royale 的なメタ進行を接ぎ木した点が設計の眼目。非同期チーム制は同時接続を要求せず、モバイルの隙間時間に最適化されている。

## バズ要因 / 人気の根拠

- 配信 1 週間で **27M DL（世界 2 位）**、3 週間で **34M DL** を突破（Sensor Tower / 2021/04）。これは Candy Crush の King が持つストアフロント力と、Activision 側の Crash IP リブート（『N. Sane Trilogy』2017、『It's About Time』2020）で温まった市場が重なった結果。
- 原作 PS1 世代（30–40 代）の懐古層と、Subway Surfers で走り慣れたライト層のクロスターゲティングが奏功。
- 一方で IAP 収益は **$1.2M 止まり**。DL 効率と収益効率が大きく乖離した典型例として業界紙で取り上げられた。
- 収益性が伸び悩み、配信から約 23 ヶ月後の **2023/02 にサービス終了**。短期 DL 爆発 → マネタイズ不足 → クロージャという F2P ランナーの構造課題を可視化した事例。

## 企画への示唆

- **既存ジャンルに IP を載せる戦略**は DL 獲得には強いが、ジャンル本来の収益曲線（ランナーは ARPU が低い）を覆す設計を別途仕込まないと収益が伸びない。
- **エンドレスに「ボス＝終端」を入れる**と、達成感とクラフト動機が生まれ、メタループの起点が立つ。
- **非同期チーム制**を入れると、サーバ負荷を抑えたまま「他者の存在感」と継続率を確保できる（リアルタイム同期は不要）。
- **IP 懐古層と新規ライト層**を同時に取りに行く場合、操作レイヤーは新規ジャンル準拠・演出レイヤーは原作準拠で割る方法が機能する。
- **DL 数を KPI にしない**。F2P モバイルでは Day-30 ARPDAU と素材消費速度が生存指標であり、配信直後の DL ピークは収益保証ではない。

## 比較で見るべきタイトル

- Subway Surfers（同ジャンル原型、F2P ランナーの収益モデル基準）
- Sonic Dash（IP × ランナーの先行事例、長期運営に成功）
- マリオカート ツアー（IP × モバイルでの DL 爆発と収益課題、同時期の類例）
- Temple Run 2（クラフトなしの純ランナー、収益構造の対照群）
- Crash Bandicoot 4: It's About Time（同 IP の家庭用最新作、市場再加熱の背景）
