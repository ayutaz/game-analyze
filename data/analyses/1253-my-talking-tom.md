# マイ トーキング トム (My Talking Tom)

## なぜ REW 主分類なのか

プレイヤーの主動機は「**世話して、コインを貯めて、トムを着せ替える**」という収集・カスタマイズの累積である。喋るトムとの掛け合いそのものは入り口の体験だが、リテンションを支えているのはミニゲームで稼いだコインを家具・服・髪型に変換し続けるアンロックループであり、ここが REW として分類される根拠になる。

concept にある「餌・風呂・着替え・遊びで世話する非同期型バーチャルペット」は一見 EXP（ケア体験）に見えるが、target が未就学児〜小学生中心であることと噛み合わせると、子どもにとっての継続動機は「今日トムにどんな新しい服が買えたか」「部屋にどんな家具を増やせたか」という**所有物の増分**にあると読める。社会軸が未設定なのも、本作が他者交流より自己完結の育成箱に振り切っている設計を反映している。secondary に EXP が入っているのは、音声真似返しというインタラクティブ体験が入り口を強力に拡張するため、無視はできない補助動機として位置付けられる。

## 報酬ループの組成

| 層 | 内容 | サイクル |
|---|---|---|
| ケア層 | 餌・風呂・トイレ・就寝の世話 | 数時間ごと、HUDアイコンが赤くなる |
| ミニゲーム層 | タップ／スワイプ系の短尺ゲームでコイン獲得 | 1プレイ 1〜3分 |
| カスタマイズ層 | 服・髪型・家具・部屋テーマのアンロック | 数日〜数週間 |
| レベル層 | トムの成長段階（子猫→成猫）と新部屋解放 | 数週間〜数ヶ月 |
| マネタイズ層 | コインのIAPブースト、広告動画リワード、定期広告 | 任意 |

設計の鍵は**ケア層（必要）→ ミニゲーム層（労働）→ カスタマイズ層（消費）**の3層が綺麗に分業しているところ。ケア層は「やらないと罪悪感」、ミニゲーム層は「やると儲かる」、カスタマイズ層は「儲けると見た目が変わる」と動機の役割が重複せず、子どもでも因果が理解できる粒度に整理されている。

## バズ要因 / 人気の根拠

- 2013年11月リリースから**10日で135カ国のトップアプリ入り**、爆発的に初速を取った
- Android単独で**10億DL超**、Talking Tomフランチャイズ累計で**100億DL**という、モバイル史上でも稀な規模
- 「**自分の声をオウム返しで真似てくれる**」という即時面白さが、テキスト不要・年齢不問でバイラル動画素材としても拡散
- 子ども層を中心に据えつつ、広告動画リワードと低価格IAPで**ARPUは低くてもDAUの裾野で稼ぐ**マスマーケット型収益構造

## 企画への示唆

- **「世話の通知」をリテンションのコアに据える**と、能動的なプレイ予定がなくても起動理由が生まれる
- **入り口の体験を1動作で完結**（喋るとオウム返しする）させると、未就学児〜高齢層まで説明なしで触れる
- **コイン経済を1通貨に絞り**、ミニゲーム→着せ替えの単一変換にすると、子どもでも因果が見える
- **キャラクターIPを先に育て**、後続作（Tom 2、Friends、Outbreak など）でフランチャイズ展開すると、1作目のDAUを母集団として横展開できる
- **広告動画リワードを「コイン2倍」「ガチャ無料」に紐付ける**と、IAPに届かない層からも収益化できる

## 比較で見るべきタイトル

- **たまごっち** — 元祖バーチャルペット、世話の通知という発明
- **Neko Atsume / ねこあつめ** — 非同期×収集×放置の純度を上げた日本版
- **Pou** — 同時期のシンプル育成ペット、コイン×着せ替え経済の類型
- **どうぶつの森 ポケットキャンプ** — ケア×収集×着せ替えを大人向けに翻訳した IP 版
- **Hay Day / Township** — 「世話の通知 → 通貨 → カスタマイズ」3層構造を経営シムに転用した系譜
