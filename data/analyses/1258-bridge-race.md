# ブリッジレース (Bridge Race) — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「相手より先に頂上へ駆け上がる」=**競う**。色付きブロックを集める収集要素も橋を架ける建築要素もあるが、いずれも勝敗を決めるための手段であり、目的そのものではない。集めたブロックは橋として消費され、相手から奪い／奪われる対象になるため、収集体験（REW 的快感）は競争体験（EXP）に従属している。

`social_axis` が「非同期」なのは、対戦相手の多くが AI もしくはサーバ側で再生された他プレイヤーの行動ログであり、リアルタイム対戦ではなく**他者の存在感を借りた一人プレイ**として設計されているため。`target` が示すとおり通勤通学の数分セッションに最適化されており、マッチング待ちもチャットも持たない。`concept` の核である「ブロック収集 → 橋掛け → 相手から奪う」という三位一体ループは、収集（REW）・建築（創）・競争（EXP）を 60〜90 秒の 1 マッチに圧縮した EXP 体験そのものと言える。`secondary` は空配列だが、実態としては「収集の快感」が副次的に効いている。

## 体験の核 / 設計の発明

ハイパーカジュアルというジャンルは「1 つの操作で 1 つの快感」を旨とするが、本作はそこに**相互干渉**を導入したのが構造的発明である。同色ブロックしか取れず、自分の橋しか踏めないというルールが、収集と建築と妨害を同じアクションで成立させる。

| 層 | 要素 | プレイヤー認知 |
|---|---|---|
| 操作 | バーチャルスティック移動のみ | 1 秒で覚えられる |
| 収集 | フィールド上の同色ブロックを拾う | 「自分の山」が育つ快感 |
| 建築 | スティック移動だけで自動的に橋が伸びる | 操作コスト 0 で進捗が見える |
| 競争 | 他プレイヤーから橋ブロックを奪える | 一手で順位が反転する緊張 |
| 終局 | 頂上に最初に立った者の勝ち | 30〜60 秒で勝敗確定 |

前世代のハイパーカジュアル（`Stack Ball` / `Helix Jump` 等の単独進捗型）に対し、**他者の進捗を奪う**ことで広告 1 本挟む前に勝敗が二転三転する。非同期マッチは待ち時間を消し、AI 相手でも「奪う／奪われる」という EXP 感情を発生させる。これが Supersonic の case study で語られる「1 セッション平均が他カジュアルより長い」要因にもなっている。

## バズ要因 / 人気の根拠

`popularity` が示す「data.ai 集計で 2021・2022 年世界最多 DL ハイパーカジュアル」を支える要素を分解する。

- **累計 600M+ DL**：iOS/Google Play 双方で米国 1 位を獲得、ハイパーカジュアル史上でも上位の到達規模。
- **Supersonic (Unity) の配信網**：プロトタイプを発掘してから世界配信までを同社プラットフォームが代行する典型的成功例として Unity blog / pocketgamer.biz でケーススタディ化されている。
- **広告クリエイティブ適性**：「ブロックを奪われて崩れる橋」というショート動画 1 カットで遊び方が全部伝わる。TikTok/Reels 時代の獲得チャネルと噛み合った。
- **言語ゼロ設計**：UI に文章がほぼ無く、ローカライズコストを払わず新興国にも展開可能。Browser 版も配信され、低スペック層・低年齢層を取り込んだ。

## 企画への示唆

- **「収集 → 建築 → 競争」を 1 アクションに重ねる**と、操作習熟コストを増やさずに体験の層を増やせる。
- **AI を非同期対戦に偽装する**と、マッチング待ち 0 のまま「奪う／奪われる」EXP 感情を発生できる（リアル対戦に拘らない設計）。
- **同色制約を 1 つ入れる**だけで、フィールドを共有しながら個人の進捗を可視化でき、妨害と収集を同じアクションで両立させられる。
- **1 マッチ 60〜90 秒に収める**と、広告挿入と再挑戦のリズムを生活時間に埋め込みやすく、DAU/LTV が広告型と整合する。
- **言語要素を削る**と、グローバル配信時のローカライズ工数が消え、TikTok 等の動画広告で訴求できる範囲が一気に広がる。

## 比較で見るべきタイトル

- `Stack Ball` / `Helix Jump`（Voodoo）— 単独進捗型ハイパーカジュアルの前世代。相互干渉が無い。
- `Crowd City`（Voodoo, 2019）— 「他者を吸収して大きくなる」非同期競争の先行例。
- `Aquapark.io`（Voodoo, 2019）— 滑走 + 競争を 1 操作に圧縮したレース型の隣接事例。
- `Stumble Guys`（Kitka Games, 2021）— ハイパーカジュアルから「リアル対戦 + IP コラボ」へ進化した次世代型として比較すると、Bridge Race が留まった軽量側との対比が際立つ。
