# ARMORED CORE VI FIRES OF RUBICON

## なぜ EXP 主分類なのか

プレイヤーの主動機は「自分で組んだ機体で、構成を変えながら同じボスに何度も挑み、勝ち筋を発明する」ことにある。物語は惑星ルビコン3の資源争奪戦という骨格を持つが、ムービー消費のための時間配分は薄く、進行のたびに買い替えるパーツとアセンブル画面に滞在する時間のほうが長い。`concept` が示すとおり、本作はソウル系で再獲得した新規層に「組み立て＋反復攻略」というアクション体験を再提示するための作品で、`target` も「機体カスタマイズと反復攻略を好む層」と明示している。

`social_axis: ソロ` であり対人要素は対戦アリーナのみ。協力レイドや非同期ランキングはなく、体験は「自分の組み合わせで攻略を解く」一点に絞られている。`secondary` が空であることも、収集や物語追体験を主軸に格上げしない設計判断と一致する。

## 体験の核 / 設計の発明

中核は「アセンブル → 出撃 → 戦死 → アセンブル」のループで、ボス戦が"問題集"、機体構成が"解答用紙"として機能する。前世代 AC（V/VD、2013）からの差分は、アセンブル文法はそのままにソウル系で培ったボス演出を持ち込み、「決まった敵に対して構成で勝つ」フォーマットをパッケージ化した点にある。

| 設計要素 | 内容 | プレイヤーに発生する行動 |
|---|---|---|
| アセンブル | 脚部・コア・武器 4 スロット・内装パーツの組み合わせ | パーツ屋とテスト場での試行 |
| スタッガー | 衝撃蓄積でひるませ大ダメージ確定窓を作る | 衝撃武器と追撃武器の二段構成 |
| チェックポイント | ボス手前で機体構成変更が可能 | 死亡→組み直し→即再戦の高速ループ |
| アリーナ | ランク順 AI 戦で OS チップを獲得 | 強化解放のためのサブ攻略 |
| ミッション再挑戦 | クリア後にスコア・隠しパーツ目的で周回 | エンドコンテンツとしての構成研究 |

social_axis ソロながら、ボス前で詰まった構成を SNS に貼って助言を募るメタ層が発生し、配信・実況の「機体晒し」文化がコミュニティを駆動した。

## バズ要因 / 人気の根拠

- 「フロム待望の新作 AC、配信映え高難度ボス戦」（popularity）— エルデンリング（2022）でフロムに入った層が、初の "非ソウル系フロム新作" として可視化された。
- ワールドワイドで `2.8M+`。AC シリーズの過去作（V/VD は推定 50 万本台）から一桁伸びており、ソウル系経由の新規層獲得が数字に出ている。
- 序盤ボス「ヘリ」「バルテウス」が "詰みポイント" として SNS で集中的に話題化し、構成を晒し合う配信文化が形成された。
- 機体カラーリングと記章エディタの自由度が高く、版権機体（ガンダム、エヴァ、レイバー）の再現投稿が継続的に流通している。

## 企画への示唆

- ボス手前にアセンブル変更窓を置くと、難度を「練度」ではなく「準備」で解く文化が発生する。
- パーツ追加を段階的に開放すると、各章で新しい構成実験が起き、攻略の手触りが定期的にリセットされる。
- スタッガーのような「衝撃ゲージ」を導入すると、武器のロール（蓄積役／決定打役）が自然に分化し、構成の意味が言語化しやすくなる。
- ソロ作品でもエディタの自由度を上げると、SNS 上に二次創作的な「機体投稿」が並び、対人モードなしに横の繋がりが発生する。
- 既存 IP の名前を残したまま、上位ジャンル（この場合はソウル系ボス文法）の言語を取り込むと、旧来層と新規層の両取りが成立しうる。
