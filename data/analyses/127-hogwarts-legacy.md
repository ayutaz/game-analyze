# ホグワーツ・レガシー / Hogwarts Legacy

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**ハリポタ世界を歩き、呪文を撃ち、ホグワーツの廊下と禁じられた森を探索する**」という一人称の体験そのものにある。原作映画/小説で何度も観た場所を 1:1 で再現した空間に「自分のキャラクター」として入学する設計は、ストーリーを追う動機（NAR）や呪文・装備の収集（REW）よりも先に、空間に存在することの満足を主軸にしている。

`concept` が「原作世界を 1:1 で再現したオープンワールド ARPG」、`target` が「原作ファンの 20-40 代、シングルでじっくり世界観に浸りたい層」と書かれている通り、想定プレイヤーは "原作世界の住人体験" を買いに来る。`social_axis: ソロ` でマルチプレイを完全に排除し、自分のペースで世界と向き合わせる設計は EXP のソロ軸として一貫している。

secondary に NAR / REW が併記されているのは、メインクエスト（古代魔法を巡る陰謀劇）と魔法生物・部屋カスタム・装備ガチャ的要素が副次的動機としてしっかり機能しているため。ただし「物語を読みたいから買う」「装備を集めたいから買う」プレイヤーは少数派で、主軸はあくまで体験。

## 体験の核 / 設計の発明

中核メカニクスは「**IP の濃度 × オープンワールドの自由度**」のかけ算。これまで多くの版権ゲームが「IP の再現度は高いが回廊型」「自由度は高いが IP は表層」のどちらかに振れていたのに対し、本作は両方を AAA 予算で同時に最大化した。

| 要素 | 既存版権ゲームの典型 | ホグワーツ・レガシーの設計 |
|---|---|---|
| 空間 | 主要シーンの回廊・ステージ制 | ホグワーツ城+ホグズミード+周辺地域をシームレス接続 |
| キャラ | 原作主人公の追体験 | 自作キャラとして入学、組分け帽子で寮選択 |
| 戦闘 | IP に寄せた半棒立ち | 呪文の属性相性（炎/氷/閃光等）× 連鎖コンボ |
| 進行 | 章立て一直線 | メイン+寮固有+サブ+魔法生物+部屋設計を並列 |
| マルチ | 強引なオンライン要素 | 完全シングル、ソロで完結 |

前世代との差分でいえば、ハリポタ系ゲームは長く「映画ライセンス連動の低予算アクション」「LEGO Harry Potter のパロディ系」の二択だった。AAA オープンワールドとして全力で組まれたのは本作が事実上初で、IP ファンが 10 年以上待っていた "学校に通う" 体験を初めて成立させた。`social_axis: ソロ` は、原作の一人称的没入（手紙が届く、組分け帽子に話しかけられる、寮の談話室に帰る）を最大化するために必要な選択。

## バズ要因 / 人気の根拠

`popularity`「ハリポタ世界を歩ける夢、配信実況で大バズり」を分解する：

- **世界累計 30M 本超**。2023 年で最も売れた単一年タイトル級のスケールで、ライセンス IP ゲームとして異例の数字。
- **配信実況の組分け帽子フックが強烈**。チャンネル開始 5 分で寮が決まる導入は、配信者の人格表明・視聴者参加投票と相性が良く、初週に Twitch / YouTube のゲーム実況上位を占拠した。
- **20 年待ちの "学校生活" 需要**。原作ファンは映画/書籍/USJ までは満たされていたが「自分が入学する」体験だけは未提供で、潜在需要が一気に噴出した。
- **マルチプラットフォーム展開**（PS5/PS4/Xbox Series X/S/Switch/PC）でハリポタ世代の所有ハードを取りこぼさない設計。

## 企画への示唆

- **IP の "空間" を売る**: ファンが買いたいのはキャラやストーリーではなく "そこに居る" 体験。回廊型ではなくシームレス世界で出すと体験単価が跳ね上がる。
- **自作キャラ + 組分け儀式を冒頭に置く**: プレイヤーの自己投影と SNS 共有可能なフック（寮判定）を兼ねさせると、配信実況での初動が読みやすい。
- **マルチプレイを敢えて削る**: 強い世界観 IP では、ソロ完結のほうが世界への没入を毀損しない。マルチを足さない勇気が体験の質を守る。
- **AAA 予算 × 待たれた IP の組み合わせを狙う**: 「ファンが 20 年待った体験」を AAA 規模で初めて出すと、シリーズ単年最大記録を作れる。
- **副次動機を 3 層重ねる**: メインストーリー（NAR）+ 装備・呪文収集（REW）+ 探索（EXP）を並列で走らせると、層ごとに離脱タイミングを後ろにずらせる。

## 比較で見るべきタイトル

- **エルデンリング (#076)**: 同じ AAA オープンワールド × ソロ軸だが、IP は新規構築。"世界に居る" 動機の作り方が対照的。
- **ゼルダ BotW / TotK**: シームレス世界 × ソロ探索の系譜。ホグワーツ・レガシーが洋ゲー文脈で同じ仕事をした例。
- **Star Wars Jedi: Survivor**: 同じく既存 IP × AAA アクション ARPG。ただし回廊寄りで、空間設計の差が体験の射程を決めている。
- **ファイナルファンタジー XVI**: AAA × 大型 IP × シングル特化、しかし一本道で NAR 寄り。本作との差分は「歩ける空間か、見せられる空間か」。
