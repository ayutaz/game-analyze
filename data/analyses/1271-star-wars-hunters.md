# スター・ウォーズ：ハンターズ

## なぜ EXP 主分類なのか

プレイヤーの主動機は「Star Wars 世界の異種ハンターを操作して 4v4 で**競り合う**」ことにある。concept にある通り、ジェダイ／バウンティハンター／ウーキー／ドロイドなど役割の異なるキャラを切り替えて TPS アクションで撃ち合うリアルタイム対戦が中核で、social_axis は「対戦」、ジャンルも「アリーナバトル/4v4 TPS」と明示されている。物語や IP 消費はあくまで背景設定で、毎セッションごとの操作・連携体験そのものが商品である以上、NAR ではなく EXP に振るのが妥当となる。

secondary に REW が入るのは、F2P モバイル/Switch タイトルとして装備・コスメ・バトルパス的なメタ報酬を持っていたためで、対戦ループの定着を補助する役割にとどまる（主動機が「集める」になる Honkai: Star Rail 型ではない）。

## 体験の核 / 設計の発明

| レイヤ | 設計要素 | 競合との差分 |
|---|---|---|
| 操作 | モバイル/Switch でも握れる簡易 TPS（オートエイム寄り） | Apex / Overwatch のような PC/据置で重い操作系を捨て、片手プレイ可能域に圧縮 |
| キャラ | ジェダイ近接 / シューター / タンクの 3 ロール | Star Wars IP 文脈で「ライトセーバー vs ブラスター」を 1 試合内で成立させた |
| マップ | 短時間 4v4 アリーナ（数分単位） | バトルロイヤルの長尺ではなく MOBA/Hero Shooter のミニ版 |
| 配信 | Switch + iOS + Android クロスプレイ | コンソールとモバイルの母集団を 1 マッチに合流させる前提 |

「対戦」を core とした上で、Switch を含めることでモバイル単体の対戦シューター（CoD Mobile / PUBG Mobile）と差別化を狙った、というのが設計の発明点である。一方で発明はあくまで配信プラットフォーム面に寄っており、Overwatch 系のロール設計や Star Wars IP 表現自体に既存比の飛躍は薄い。

## バズ要因 / 人気の根拠

popularity を分解すると、商業成功というより「立ち上げと畳み方」が話題化したケースに分類される。

- Zynga × Lucasfilm Games の公式提携タイトルで、Star Wars IP × ヒーローアリーナ対戦という座組み自体に当初注目が集まった。
- 2021 年ソフトローンチからフルローンチまで **約 3 年延期** されており、待機期間に SNS 上で進捗を追われ続けた。
- 2024-06-04 に Switch / iOS / Android で正式リリース。Switch を含むモバイル F2P TPS という珍しい構成が話題化要因となった。
- 商業的に振るわず **2025-10-01 にサービス終了**が発表され、サ終ニュースとしても二度目の注目を集めた（automaton-media 等で報道）。

## 企画への示唆

- IP × 既存ジャンルテンプレ（Star Wars × Overwatch 型）を組むと話題化はしやすいが、ジャンル側の設計に飛躍がないと「IP 着せ替え」と評価されリテンションが伸びない。
- ソフトローンチ期間が長期化（3 年）すると、市場側の対戦シューターメタが変化してリリース時に古く見える。期間と市場変化速度のバランス設計が必要。
- Switch + モバイルのクロスプレイを採るなら、操作スキームの公平性（オートエイム強度・FPS 上限）を最初から固めないと、片側ユーザーが離脱する。
- F2P アリーナ対戦は **同時接続母集団の絶対数**でマッチング品質が決まるため、ローンチ直後の DL 速度がそのままサービス寿命に直結する（撃てない・待たされるが起こると一気に崩れる）。
- サ終発表自体が PR 機会になり得る一方、F2P で「いずれ畳まれる」前例として参照されると、後続の IP × F2P 案件で投資判断が慎重化する副作用がある。

## 比較で見るべきタイトル

- Overwatch 2（PC/据置のヒーローアリーナ対戦、ロール設計の原典）
- Apex Legends Mobile（モバイル展開した Hero Shooter、2023 サービス終了）
- ポケモンユナイト（Switch + モバイル クロスプレイ × IP × MOBA、対照的に成功）
- フォートナイト（クロスプレイ × IP コラボ × F2P アリーナの巨人）
- Marvel Rivals（IP × ヒーローアリーナの近年成功例）
