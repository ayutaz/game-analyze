# ハリー・ポッター：呪文と魔法のパズル — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「魔法カードを集めて、ホグワーツの世界を所有していく」点にある。マッチ3 自体は既知の枠組みで、Candy Crush / Royal Match を遊んでいる層にとってパズル盤面の体験そのものは目新しくない。新規価値はクリアの度に解錠される **杖・組分け帽子・呪文・魔法生物カード** など IP 資産の収集進行であり、ここが課金とリテンションを駆動する。

`concept` にある通り、本作は「Candy Crush 系の王道マッチ3」にハリポタ IP を被せた構造で、`target` も Candy Crush / Royal Match 系を遊ぶカジュアル層を想定している。つまり**パズル＝集めるためのチケット消化**、**カード＝集める対象**という非対称な比重で、主動機は完全に「集める／所有する」に寄っている。`secondary: ["EXP"]` は、盤面操作・連鎖の気持ちよさを副次的な動機として残しているという扱い。`social_axis: null` は、本作が他者軸（対戦／協力／非同期ランキング）をほぼ持たない純シングル収集である事実と整合する。

## 報酬ループの組成

カジュアル収集ゲームとして見たとき、報酬は 3 層に分かれる。

| 層 | 形式 | 周期 | 役割 |
|---|---|---|---|
| 短期 | ステージクリア・スター獲得 | 1〜5 分 | パズル本体のドーパミン、ライフ消費の対価 |
| 中期 | 杖／組分け／呪文／魔法生物カードの収集 | 数日〜数週 | IP 愛着の可視化、コレクション充足 |
| 長期 | プリベット通り→ホグワーツへ進むストーリー進行 | 数か月 | 「映画 1 作目を追体験している」という大目標 |

短期はマッチ3 既存型、中期がハリポタ IP 固有層、長期はストーリー追体験という映画 IP ならではの軸を入れている。ライフ制と購入アイテム（追加手数、ブースター、イベント参加券）は中期・長期へ進むためのチケットで、課金は**「カードを揃えたい／物語を進めたい」**という IP 愛着に紐付く形で発生する。Zynga 系の他作品（FarmVille、Words With Friends）と異なり PvP / 同期協力を入れず、**シングル収集 × IP 追体験**に絞ったのが構成上の選択。

## バズ要因 / 人気の根拠

- 原作映画のサウンドトラックとボイスをライセンスで取り込み、立ち上げ時点で「映画ファン向けの公式マッチ3」というポジションを確保
- 元 One Direction の Zayn Malik とのコラボ発表という、ゲームメディア外（音楽・芸能）への波及を仕込んだプロモーション
- リリース 1 周年時点で 2,000 以上のステージを擁するボリュームで、Candy Crush 系のコンテンツ消費速度に耐える運営
- パブリッシャー Zynga のカジュアルゲーム配信網（iOS／Android／Browser）に乗せ、非ゲーマー層・特に 30–50 代女性層を取り込んだ

## 企画への示唆

- **既知メカニクス × 強 IP** を組むと、メカニクス開発コストを抑えつつ「映画ファン → カジュアルゲーマー」へのチャネル変換が起きる
- 報酬層を **短期（パズル）／中期（IP カード）／長期（ストーリー追体験）** の 3 段にすると、課金の動機を IP 愛着側に逃がせる
- マッチ3 にライセンス音源・ボイスを乗せると、盤面の凡庸さを「世界に居る感覚」で塗り替えられる
- 他者軸を切り、純シングル収集に絞ると、対人嫌忌のカジュアル層・非ゲーマー層を取りこぼさない
- 芸能・音楽セレブとの単発コラボは、ゲーム外メディアへの初動露出を稼ぐが、本体ゲームとの整合より「話題量」が目的という割り切り

## 比較で見るべきタイトル

- **Candy Crush Saga**（マッチ3 王道、IP なし版の原点）
- **Royal Match**（マッチ3 × 城再建収集、近年のリテンション王者）
- **Disney Magic Kingdoms** / **Disney Emoji Blitz**（強 IP × カジュアルパズル の同系統）
- **ハリー・ポッター：ホグワーツ・ミステリー**（同 IP の RPG 寄り、収集動機の比重が違う対照例）
