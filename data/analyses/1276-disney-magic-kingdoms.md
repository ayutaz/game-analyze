# ディズニー マジックキングダム — 構造分析

## なぜ REW 主分類なのか

プレイヤーの主動機は **「ディズニー/ピクサーのキャラとアトラクションを 1 つでも多く解放し、自分の Magic Kingdom に並べる」** こと、つまり「集める」である。

concept にある通り、本作は F2P テーマパーク経営シムの形式を借りた **IP コレクションゲーム**だ。target が「重課金 PvP よりキャラ収集とパーク装飾の自己表現を楽しみたい層、特に女性層・親子層」と定義されている通り、勝敗や上達ではなく「ミッキー、エルサ、ウッディ、Stitch、ベイマックスなどがすべて自分のパークに揃っている」という所有体験が中心動機を成す。経営シムの体裁を取りつつ、内部で回っているのは時間ゲートとプレミア通貨でロックされた **キャラ／アトラクションのアンロックループ**であり、これは REW のテンプレートに完全に重なる。

secondary に EXP が入るのは、アトラクションを配置してパークの景観を自分で組み立てる「装飾＝自己表現」のサンドボックス的な楽しみが副次的に効いているため。social_axis は null だが、実態として SNS スクショ共有や友人とのパーク見せ合いが緩い非同期社交として機能している。

## 報酬ループの組成

| レイヤー | 報酬の単位 | 入手手段 | ゲート |
|---|---|---|---|
| ベース通貨 | コイン / マジック | アトラクションのタップ収益（時間経過） | リアルタイム待機 |
| プレミア通貨 | ジェム | 課金、レベルアップ、達成報酬 | 課金圧 |
| キャラ収集 | キャラ本体（ミッキー、エルサ、Buzz 等） | クエスト完了でアンロック → レベル上げ | 数時間〜数日のタスクチェーン |
| アトラクション収集 | 城、メリーゴーランド、ライド | 建設タイマー（数時間〜数日）+ 素材集め | 時間 + 通貨 |
| 期間限定 IP | 新作映画タイアップキャラ | 期間イベント完走報酬 | 開催期間 + 進行ペース |
| 装飾レイヤー | パークの自由配置 | 解放済みアトラクションを任意配置 | なし（自己表現） |

このゲームの巧妙さは、**収集対象が「キャラ」「アトラクション」「ストーリー（クエスト台詞）」の 3 軸に重ねられている**点にある。1 体のキャラを解放すると、そのキャラのクエスト系列が解放され、それを進めると映画の名場面を想起させる台詞と限定アトラクションが手に入る。つまり 1 ユニット解放するごとに **「キャラ枠」「ストーリー枠」「装飾枠」**の 3 つのコレクション欄が同時に進む構造で、達成感の単価が高い。

長期運営のリテンションは、ピクサー新作映画やディズニー周年に合わせた **期間限定イベント**で駆動される。劇場公開タイミングで対応キャラが入ってくるため、IP ファンは「映画を観たら DMK にログイン」という外部リズムを持ち込まれる設計になっている。

## バズ要因 / 人気の根拠

popularity の「公式 IP モバイル × 新作映画連動の長期運営」を分解すると以下：

- **ディズニー/ピクサーの公式ライセンス全方位**：ミッキー本流、ピクサー（トイ・ストーリー、モンスターズ・インク、ファインディング・ニモ）、プリンセス系、Frozen、ベイマックスまで取り扱い、IP 単独ファンを横断的に集客
- **累計実績**：全世界 100M+ DL、累計収益 $200M 規模。日本でも配信約半年（2017 年 4 月時点）で 200 万 DL を突破
- **新作公開連動の継続更新**：劇場新作の度に対応キャラ／アトラクションを実装する設計で、2016 年配信開始から 10 年近く運営継続中
- **Gameloft × Disney 公式座組**：非公式同人ではなく Disney 公認のため、IP の使い方が「正史」として認められる安心感が IP ファンの長期定着に効いている

## 企画への示唆

- **コレクション軸を 3 重ねにすると 1 アンロックの体感報酬が 3 倍になる** — DMK は「キャラ」「ストーリー進行」「装飾アセット」を 1 ユニットの解放で同時に進める。報酬を 1 軸に絞らず束ねよ。
- **IP の元体験を「集める対象」に翻訳すると IP ファンが課金理由を持てる** — DMK はテーマパークというメタ枠で「映画キャラを並べる行為」を正当化している。映画を観るだけのファンに、消費を超えた「所有」の場を作ると課金に繋がる。
- **時間ゲート × プレミア通貨スキップは F2P 経営シムの標準解** — 建設・クエスト・キャラレベルアップをすべてタイマー化し、ジェムで短縮させる構造は IP 装飾と相性が良い（戦闘がないので時間スキップが純粋な利便性として機能する）。
- **外部カレンダー（映画公開、周年）に運営リズムを同期させると、ゲーム単体の話題作りに依存しなくて済む** — 映画プロモーションが集客を兼ねる。IP ホルダーと組む際に絶対に取りに行くべき設計余地。
- **PvP を入れない選択が女性層・親子層への入口を広げる** — 競争軸を排し、装飾と収集の自己表現に絞ることで「ガチャ重課金 PvP に疲れた層」「キャラ目当てのライトユーザー」を取りに行ける。

## 比較で見るべきタイトル

- **The Simpsons: Tapped Out**（EA、2012）— Gameloft 以前の「IP × タップ式街づくり F2P」の原型。DMK は構造的にこの直系。
- **Disney Tsum Tsum**（LINE / Disney、2014）— 同じ Disney IP ×モバイル× 収集だが、こちらはパズルベース。IP 訴求軸が「キャラの絵柄」だけに絞られている対比例。
- **Animal Crossing: Pocket Camp**（任天堂、2017）— 装飾＋キャラ収集型モバイルの非ガチャ系比較対象。社交軸の入れ方が DMK より一段強い。
- **ハリー・ポッター：ホグワーツ・ミステリー**（Jam City、2018）— IP × クエスト解放 × 時間ゲートで DMK の構造をストーリー側に振った姉妹形。
- **ディズニー ツムツムランド**（コロプラ、2018）— 同 IP のパズル系で、装飾レイヤーがない分 DMK の「並べる楽しさ」の独自性が浮かび上がる対比。
