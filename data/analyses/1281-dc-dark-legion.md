# DC：ダーク・リージョン (DC: Dark Legion) 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「200 体超の DC キャラクターを引き当て、強化し、同盟内序列を上げる」ことであり、動詞で言えば **集める／育てる／競う** が中核に位置する。concept にある「4X ライク基盤＋ヒーロー編成」という構造は、State of Survival 系で蓄積された資源収集・建築・同盟戦のループに、DC コミックの IP を上書きしたもので、ガチャによるキャラクター獲得が最上位の報酬として機能する。

target が「DC ファン」と「SoS／ロードモバイル系プレイヤー」の二層に分かれている点も REW を裏付ける。前者は推しキャラを引く動機、後者は同盟戦の勝率に直結する戦力指標としてキャラクターを集めるため、いずれも報酬獲得が遊びの中心となる。secondary に EXP が立つのは、同盟戦・SvS（Server vs Server）といったリアルタイム対戦面が副次的な熱源になっているためで、報酬の使い道が PvP に流れ込む構造である。

## 報酬ループの組成

本作はガチャ・育成・領地拡張・同盟戦という 4 層が連動し、上位レイヤーが下位レイヤーの収集を駆動する典型的な 4X ガチャ MMO 構成を取る。

| 層 | 入手物 | 駆動するもの | 課金接点 |
|---|---|---|---|
| ガチャ層 | DC キャラクター（バットマン／ジョーカー他 200+） | 編成上限・属性カバー | 限定召喚・ピックアップ |
| 育成層 | レベル／覚醒素材／装備 | 戦力スコア | パック販売・スタミナ |
| 領地層 | 資源・建築 Lv・研究 | 同盟内戦力ランク | 加速アイテム |
| 同盟層 | KvK／SvS の勝敗・領土 | サーバ内序列・シーズン報酬 | 月パス・VIP |

リテンション設計のポイントは、(a) ガチャで得た IP キャラの満足度を「同盟内ランキング表示」と接続して見せ場を作る、(b) 領地・研究の数十時間スケールのタイマーを月パスで圧縮させる、(c) シーズン制 KvK で集めたキャラを「定期的に使い切らせる」点にある。SoS 系で確立された 90 日リテンションカーブの上に、DC IP の推し補正で離脱閾値を下げる構造である。

## バズ要因 / 人気の根拠

- DC 公式ライセンスのスマホ 4X ガチャという**未開拓ニッチ**を、SoS 系で実績ある FunPlus が獲りに行った座組。コミック・映画・アニメの既存ファンに「推しを編成して戦わせる」入り口を提供した。
- 配信 3 か月で **Android 単体で累計 10M ドル**（Sensor Tower）を突破。iOS / グローバル合算では 2025 年通年で **約 45.4M ドル**規模に達し、「2025 年で最も売れた IP ライセンスモバイル」のひとつに位置づけられた。
- 日本・韓国・香港・マカオ・台湾でアジア順次事前登録を展開し、英語圏先行→アジア追撃という FunPlus 定石の国際ローンチを踏襲。
- バットマン／ジョーカーといった**認知率が極めて高いキャラクター**を看板に据えたことで、ストラテジー文脈での非接触層（コミック／映画寄り）にもクロスリーチした。

## 企画への示唆

- **既存基盤＋強 IP の上書きは、ゼロから IP を立ち上げるより到達コストが低い**：FunPlus は State of Survival で 4X ガチャ MMO のテンプレを完成済み、その上に DC を被せることで開発リスクと運営ノウハウを同時に節約している。
- **「推し」と「戦力スコア」を同じ画面で接続すると課金動機が二重化する**：IP ファンの感情的動機と、4X プレイヤーの数値最適化動機を同じガチャに流し込める。
- **同盟戦のシーズン制が IP キャラの陳腐化を防ぐ**：定期的に編成を組み替える機会を強制することで、新キャラ追加への需要が常に発生する。
- **国際ローンチは英語圏→東アジアの順で「先行データを持って事前登録を打つ」と CPI が下がる**：本作の日本・韓国・台湾事前登録展開はこの定石を踏襲している。
- **IP ライセンス料を払うなら、IP 単独で売る作品ではなく既存ループに乗せる方が ROI が出やすい**：単独 IP ゲーは IP 寿命と心中するが、4X ガチャは IP を差し替えても基盤が残る。
