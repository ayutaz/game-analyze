# サウスパーク：フォン・デストロイヤー

## なぜ EXP 主分類なのか

プレイヤーの主動機は「3 分の対戦でレーンを押し合って勝つ」ことであり、デッキ構築やキャラ収集はその勝率を上げるための手段に従属している。Clash Royale 系のリアルタイム PvP として social_axis は **対戦** が中心軸で、原作 IP は世界観の被膜にとどまる。

concept にある「ごっこ遊びを題材にしたカードバトル」というメタ構造は、サウスパーク原作 S17–S21 で繰り返された「子供たちが裏庭で TCG ごっこをする」回（Black Friday 三部作、The Fractured But Whole）の延長にあり、プレイ体験そのものが原作のネタの再演になっている点が EXP らしさを補強する。secondary に **REW** が入るのは、5 テーマ（アドベンチャー / SF / 神秘 / ファンタジー / ヒーロー）のカード収集とレベルアップが長期リテンションを駆動するため。ただし課金構造より「次の 3 分の対戦をどう勝つか」が前面に出ている。

## 体験の核 / 設計の発明

| 要素 | Phone Destroyer の選択 | 効果 |
|---|---|---|
| マッチ長 | 3 分以内のリアルタイム | モバイル隙間時間と Trey/Matt の音声尺が両立 |
| 場の形 | 2 レーン押し合い（Clash Royale 型） | エネルギー管理 → 即時意思決定の往復 |
| デッキ縛り | 5 テーマでデッキカラー固定 | 原作のごっこ遊び設定をルールに翻訳 |
| 課金軸 | カードガチャ＋テーマ別レベル | REW 層が EXP 層に従属する形で同居 |
| 音声 | Parker/Stone 本人の英語 VO | IP の毒舌が「演出」ではなく「資産」になる |

発明はメカニクスではなく **「Clash Royale 型ループに既存 IP の脚本ネタを骨格として上書きした」翻訳**にある。レーンを押すクリプス／敵キャラが原作の Mysterion / Princess Kenny / Marshrooms 等で構成され、対戦中のヤジが原作声優の収録新規ボイスで返ってくる。ライセンスゲーが「ガワ替え」に終わる失敗を、対戦中の音響レイヤで回避している。

## バズ要因 / 人気の根拠

- Google Play で **500 万 DL 超**、IP モバイルゲームとしては Clash Royale 系の中で長期運営に成功した部類
- **Trey Parker / Matt Stone 本人**による新規収録ボイス、原作と同じ音響チームが対戦中の罵り合いまで担当
- 2017 年配信、Comedy Central と Ubisoft の販促連動で原作 S21 放映期と同期投入
- Ubisoft RedLynx（Trials シリーズの北欧スタジオ）が開発、モバイル PvP として 5 年超の運営

## 企画への示唆

- **既存ヒット型（Clash Royale 等）に強 IP の音響レイヤを被せる**と、ガワ替えではなく「対戦の演出資産」になる
- **原作内で「ごっこ遊び回」がある IP** は、プレイヤー操作と原作世界観のメタ整合が取れるためモバイル化適性が高い
- **3 分マッチ × 本人ボイス**の組み合わせは、IP ファンの「セリフを聞きたい」需要と非 IP ファンの PvP 需要を 1 つのループで満たす
- **デッキ色をストーリー上のテーマで縛る**と、収集軸（REW）が IP の脚本構造に乗るため離脱前の納得感が増す
- **MAU が小規模でも長期運営できる**のは、IP コラボが新規エピソード放映に同期しているため広告費なしで山が立つ

## 比較で見るべきタイトル

- Clash Royale（Supercell 2016）— ベースメカニクスの本家、レーン押し合いの完成形
- Marvel Snap（Second Dinner 2022）— IP × 3 分カード PvP の後継形、ロケーション要素で差別化
- South Park: The Fractured But Whole（Ubisoft SF 2017）— 同年・同 IP のコンソール RPG、ごっこ遊び設定は共通
- Family Guy: The Quest for Stuff（TinyCo 2014）— 米アニメ IP × 放映同期型モバイル運営の先行例
