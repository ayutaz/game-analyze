# スポンジ・ボブ：クラスティ・クックオフ

## なぜ REW 主分類なのか

プレイヤーの主動機は **「注文をさばき、稼ぎ、レシピ・装飾・キャラを集めて店を拡張し続ける」** こと。タイムマネジメント自体の操作快楽（EXP 的）も確かに存在するが、F2P クッキング系の構造上、ステージクリアで得るコインと素材を投下して **レシピ／調理器具／キャラ衣装／店舗装飾** の収集と段階的アンロックを進める「報酬獲得サイクル」がセッションの背骨になっている。`primary: REW`、`secondary: [EXP]` の配列は、ガチャ的収集を主、ステージ操作を従と位置付けた整理である。

`target` に書かれた「Cooking Fever / Diner Dash 系のカジュアル女性層」と「スポンジ・ボブ世代の 20-40 代」の二重ターゲット構造は、IP コラボキャラと店舗テーマを **収集対象** として常時投入できることを意味し、IP 由来のキャラ／メニュー収集がそのまま課金導線になる。`social_axis: null` なのは、ランキング/ギルド/同盟戦のような対他要素より、自分の店を育てるソロループに収益が集中しているため。secondary に EXP が付くのは、調理レーンの捌き＝指さばきの楽しさが滞在時間とリトライ意欲を支えているから。

## 報酬ループの組成

| 層 | 報酬の中身 | プレイヤーの動機 | 主な変換先 |
|---|---|---|---|
| ステージクリア | コイン・経験値・スター | 即時フィードバック | 調理器具のアップグレード |
| レストラン拡張 | 新メニュー解禁・新店舗 | 「次の店を開く」 | 客単価上昇＝周回効率 |
| キャラ収集 | スポンジ・ボブ / イカルド / パトリック等のスキン | IP 親和性 | キャラ専用ブースト |
| イベント／コラボ | 期間限定メニュー・装飾 | FOMO（取り逃し回避） | 課金パック購入 |
| 装飾カスタマイズ | 店舗テーマ・小物 | 自己表現・コレクション | 長期リテンション |

調理ステージは数十秒で 1 周回るが、その短サイクルの上に **レストラン拡張 → 新キャラ解禁 → 新装飾解禁** の中期ループが乗り、さらに上に IP コラボイベントの長期ループが乗る三層構造。短ループの密度がカジュアル女性層、中ループが収集欲、長ループが IP ファン心理をそれぞれ刺激する。

## バズ要因 / 人気の根拠

- 事前登録 **1,700 万人** を集めた段階で、ニコロデオン IP × F2P クッキングというニッチ重なりが既に強い需要を示していた。
- 2020 年ローンチ時に **iOS 97 カ国・Google Play 79 カ国** で Top10 入り。地域横断性が高く、英語圏以外の IP ファンも吸い込んだ。
- 2021 年 10 月時点で **全世界 5,000 万 DL** を突破し、2021 Mobile Games Awards で **Players' Choice** を受賞。プレイヤー投票型の評価で IP の動員力が裏付けられた。
- 2025 年に PS5 / Xbox Series / PS4 / Xbox One へ移植され、モバイル発タイムマネジメントとして家庭用コンソール展開まで延命できた数少ない事例。

## 企画への示唆

- **既存 F2P テンプレ × 強 IP** の掛け合わせは、新規メカニクス開発リスクを負わずに事前登録の天井を引き上げる。Cooking Fever 系の枠を借り、IP は装飾と収集対象として乗せる構成は再現性が高い。
- **収集対象を「キャラ」「メニュー」「装飾」の 3 系統に分離** すると、IP ファン／グルメ志向／インテリア志向の異なる動機をひとつのループに同居できる。
- **短ループ（数十秒）／中ループ（店舗拡張）／長ループ（IP イベント）の三層化** により、ライトプレイヤーの 1 セッション体験と重課金者の月次イベント参加を同じ KPI で計測しやすくなる。
- **ローンチ多国展開（80+ カ国 Top10）を狙うなら IP のグローバル認知度が起点**。ローカライズ前提のグローバル IP を選ぶことで、ロンチ前後のオーガニック獲得効率が跳ね上がる。
- **モバイル F2P → 数年後にコンソール買い切り移植** は、装飾とコラボの蓄積を「コンテンツ量」としてパッケージ化する出口戦略になる。

## 比較で見るべきタイトル

- **Cooking Fever**（Nordcurrent）— 同ジャンルの原型、IP なし版の対照群
- **Diner Dash Adventures**（Glu Mobile）— タイムマネジメント＋ストーリー進行型
- **Cooking Madness**（ZenLife）— 中華圏発の同ジャンル、収集要素強め
- **My Talking Tom**（Outfit7）— 強 IP × F2P カジュアルの先行成功例
- **Disney Magic Kingdoms**（Gameloft）— IP × 拡張型シミュレーションの構造参照
