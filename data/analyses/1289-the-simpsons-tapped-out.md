# シンプソンズ：タップド・アウト 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は **「原作キャラとギャグを次々アンロックして自分のスプリングフィールドに並べる」=集める**ことにある。戦闘や PvP は核ではなく、建物を建てる→住人を呼ぶ→新キャラ・新ギャグが解放される、というコレクションの連鎖がリテンションを駆動している。

concept が示す通り、本作はハードコア課金や勝敗を競う構造を避け、「コレクション + 待ち時間ループ」を中心に 13 年運営された IP モバイル F2P の典型である。target も「本格 PvP やガチャを避けつつ、好きな IP の世界観を箱庭で広げたいライト F2P プレイヤー」と明確で、報酬獲得が動機の主軸を成す。social_axis が null なのは、フレンド機能で他プレイヤーのスプリングフィールドを訪問できる程度の弱い非同期要素にとどまり、対戦／協力の中核ではないため。

secondary に EXP が入るのは、街を意のままに区画し動線を引いていく**箱庭づくりの操作体験**そのものが二次的な楽しさを供給しているからで、Animal Crossing 系の craft 動機が補助線になっている。

## 報酬ループの組成

シンプソンズ TO の報酬層は、IP コラボ前提の「期間限定イベント」に重心を置いて積層している。

| 層 | 内容 | リテンション効果 |
|---|---|---|
| 即時報酬 | 建物建設・タスク完遂で硬貨 / XP / ドーナツ獲得 | 1 セッション 5–10 分の満足を保証 |
| 中期報酬 | 新キャラ（住人）のアンロック | 「次に誰が来るか」の継続フック |
| 季節イベント | ハロウィン / クリスマス等の期間限定ストーリー＋限定建物 | 数週間スパンの目標 / 再ログイン誘導 |
| 長期報酬 | IP の世界観そのものを箱庭で完成させる | 数年単位の「収集完成」目標 |
| 課金経路 | プレミアム通貨「ドーナツ」で時短 / 限定建物即時購入 | ガチャに頼らない pay-to-skip 型 |

特筆すべきは**ガチャ層を持たず、ドーナツによる「待ち時間スキップ + 限定物販」だけで運営した**点。日本のソシャゲ的なレアリティ確率ループを採らず、IP コラボイベントの**カレンダー消化型**で継続率を稼いだ構造は、北米 F2P の代表的様式である。

## バズ要因 / 人気の根拠

popularity が「米国 iPhone 売上 Top 20 にほぼ常駐し続けた稀有な長寿タイトル」と要約する通り、バズは単発の話題というより**13 年連続稼働そのもの**が事件性を持っている。

- 累計売上 $323M+（AppMagic 推定、App Store + Google Play 合算）でモバイル F2P 長寿タイトル上位に位置
- 2012 年配信開始から **2025 年 1 月にサービス終了**、終焉のニュース自体が世界的に報じられバズ第二波を生んだ
- リリースから比較的早期に**ライフタイム収益 $100M 突破**（mcvuk 報道、2010 年代前半の IP F2P としては破格）
- 米国 iPhone 売上ランキング Top 20 への長期常駐実績で、「IP 強度 × ライトな F2P 設計 = 持続収益」のテンプレを業界に示した

## 企画への示唆

- **強 IP × 箱庭シミュは PvP / ガチャを抜いても長期黒字化できる** — ターゲットがファン層なら勝敗ではなく「世界観の完成」が一次動機になる
- **季節イベントを 13 年回し続けると累計が積み上がる** — 単発の派手さより、原作の年中行事に紐づくカレンダー設計の方が長寿に直結する
- **ガチャ非搭載でもプレミアム通貨で時短を売れば収益化できる** — 確率規制の風当たりを避けつつ、待ち時間ループの摩擦を直接マネタイズに変換できる
- **終了報道自体がマーケ資産になる** — 長期運営タイトルはサービス終了タイミングを設計すると、新規 IP F2P や同社の後継タイトルへの送客になる
- **コレクション完成欲は「カオス度の上限が高い IP」と相性が良い** — 数百のキャラ・ギャグを抱えるシットコムは、追加アセットを 10 年単位で延々と供給できる

## 比較で見るべきタイトル

- **FarmVille 2 / Hay Day** — 同じ「待ち時間ループ × 箱庭」前世代の北米／フィンランド代表
- **どうぶつの森 ポケットキャンプ** — 同型の IP × 非ガチャ箱庭、日本側のカウンターパート
- **Family Guy: The Quest for Stuff** — 同じ TinyCo 系シットコム IP × タップド・アウト直接フォロワー
- **SimCity BuildIt** — EA Mobile 同社による IP 抜きシティビルダーで、設計骨格の比較対象
