# ファミリーガイ：クエスト・フォー・スタッフ — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「Peter / Stewie / Brian ら原作キャラと、パロディが詰まった建物を**集めて街に並べる**」ことである。シティビルダーの体裁を取りつつ、実態は IP 収集 + 待ち時間消化型の運営ゲームであり、戦闘や物語選択のような能動的判断より、**キャラ・建物アンロックチェックリストの埋戦**が主動機を支配している。

`concept` は「ジャイアントチキンに破壊された街を再建する」枠組みで、復興の進行が**未取得 IP リストの開示**として機能する。`target` は北米コアファン層 + Simpsons: Tapped Out 系の長期課金カジュアル層であり、いずれも「自分の好きなキャラ／回が次のイベントで出るか」を回遊動機にしている層である。`secondary` の EXP は、街レイアウトの自由配置と建物タップ収益の見回りに残るが、副次的な手触りに留まる。`social_axis` は null で、他者と直接競うレイヤーは設計されていない。

## 報酬ループの組成

| 層 | 取得対象 | 駆動装置 | 課金接点 |
|---|---|---|---|
| 短期 | 建物のタップ収益（コイン）、素材 | 数分〜数時間の建設タイマー | タイマー短縮用クラム |
| 中期 | 限定キャラ・建物（イベント通貨で交換） | パロディ回連動の期間限定イベント | イベント通貨ガチャ／直接購入 |
| 長期 | 主要レギュラー＋脇役の全コンプ | キャラ別クエスト連鎖と街区拡張 | 復興進度ゲートの解放 |
| メタ | Simpsons: Tapped Out で学習した運営ノウハウ転用 | 1〜2 ヶ月周期のイベント連打 | 季節 / 映画パロディ便乗 |

中核は「建物 → コイン → 建物」の自走ループに、**期間限定の IP コラボ層**を被せて忘却前に回収する設計。原作 1 話分のネタを 1 イベントに圧縮し、ファンに「この回好きだったから取り逃したくない」と思わせる構造が、ライト課金とリテンションを同時に支える。

## バズ要因 / 人気の根拠

- 2014 年配信、2016 年時点で**累計収益 1 億ドル超**を到達（TinyCo 発表ベース）。
- **累計 30M+ DL** に達し、IP モバイルの中でもアニメ系単独タイトルとして比較的長寿命を維持。
- 先行作 **Simpsons: Tapped Out（EA / 2012）の成功パターンを Family Guy IP に横展開**したことで、ジャンル既習層をそのまま流し込めた。
- 原作脚本陣監修と思われる**書き下ろしパロディイベント**（ブラックジョーク、時事ネタ）が、原作ファンに「アプリ独占ストーリー」の価値を作った。

## 企画への示唆

- **コメディ IP × タップ収益のシティビルダー** は、戦闘設計コストを支払わずに長期運営ガチャ収益を作れるテンプレートとして再現性が高い。
- **「破壊された街の復興」フレーム**を採用すると、未取得キャラ／建物の空きスペースが「埋めたい欲」に直結し、コンプリート動機が地形そのもので視覚化される。
- **原作 1 話 = 1 イベント**の単位で限定キャラを切り出すと、ファンが「見逃せない回」感覚で課金接点に来る周期を作れる。
- 先発の同ジャンル ヒット作（ここでは Simpsons: Tapped Out）の**運営カレンダーと課金設計をコピーした上で IP だけ差し替える**戦略は、ジャンル既習層の獲得 CAC を大幅に下げる。
- `social_axis` を持たずとも、**期間限定イベントの締切**が擬似的なソーシャル圧（取り逃し回避）を提供できる。

## 比較で見るべきタイトル

- **The Simpsons: Tapped Out**（EA, 2012）— ジャンル原型かつ運営テンプレートの直系。
- **Futurama: Worlds of Tomorrow**（Jam City, 2017）— 同じ TinyCo 系列の IP × シティビルダー横展開。
- **Family Guy: Another Freakin' Mobile Game**（Jam City, 2017）— 同 IP のマッチ 3 派生で、IP × ジャンル組み合わせの別解。
- **South Park: Phone Destroyer**（RedLynx, 2017）— ブラックコメディ IP の別ジャンル（カード対戦）転用例。
