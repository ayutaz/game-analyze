# ルーニー・テューンズ：ワールド・オブ・メイヘム — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「バッグス・バニーやダフィー・ダックを **集めて編成する**」ことにある。戦闘自体はターン制のコマンド入力で操作介入は浅く、勝敗を決める変数はキャラのレア度・覚醒・装備で構成される。つまりプレイヤーが時間と課金を投じる先は「次のトゥーンを引く／育てる／覚醒させる」サイクルであり、REW（報酬獲得）が中核に据えられている。

concept にある「Toon Squad として集め、Toon Town を建設・カスタマイズする」という構造もこれを補強する。`70+ の人気キャラを収集` という規模感は、ガチャ・覚醒・装備の三層を回し続けるための在庫として設計されており、target に挙がる「ルーニー・テューンズ世代の 30–50 代」「ライト RPG ユーザー」のいずれにとってもメインの遊びは収集の達成感である。

secondary に EXP が入るのは、コマンドバトル中のドタバタ演出（アクメ社の罠、金床落下、追跡ギャグ）が「キャラを動かして見る」体験価値を補っているため。ただし操作の上達がスコアに直結する設計ではないので、あくまで主動機を支える副材として位置づける。

## 報酬ループの組成

IP × ガチャ × ライト戦闘の三位一体を、ループ別に分解する。

| 層 | 中身 | リテンションへの効き方 |
|---|---|---|
| 収集 | 70+ のトゥーン、レアリティ階層、衣装バリアント | IP ノスタルジーで「あのキャラが欲しい」が一次動機になる |
| 育成 | レベル・スター覚醒・装備強化 | 引いた後の到達点を伸ばし、ガチャの価値を毀損しない |
| 戦闘 | ターン制コマンド、PvE キャンペーン、PvP アリーナ | 育成成果の **見せ場** を提供。負けても次の編成に動機が湧く |
| 建設 | Toon Town のカスタマイズ | 戦闘の合間の **間延び時間** を埋め、復帰理由を作る |
| 演出 | ギャグアニメ調の必殺技、原典の効果音 | IP の親和性を可視化し、課金後の満足度を担保 |

特筆すべきは「Toon Town 建設」が独立収益ではなく、戦闘ループの **冷却期間** を埋める補助装置として機能していること。施設アンロックがトゥーン獲得と連動しているため、収集の前進が街の前進にも見え、ガチャ報酬の体感価値が二重に増す。

## バズ要因 / 人気の根拠

popularity の「配信初日に 100 万 DL 突破」を起点に、ヒットの構造を要素分解する。

- 配信初日で **1M+ DL** を達成。IP の知名度がインストール決定の摩擦をほぼゼロにした
- Scopely（Marvel Strike Force などガチャ RPG 量産元）× Aquiris（ブラジル拠点のモバイル特化）の組み合わせで、IP × 収集 RPG の **既知のテンプレ** に乗せた
- ワーナーの古典 IP は北米・欧州・中南米で広いノスタルジー層を持ち、target にある 30–50 代の可処分所得層に刺さりやすい
- カートゥーン IP のガチャ RPG は当時希少で、ディズニー系（Disney Heroes など）と並んで「**アニメ IP ガチャ**」というニッチを占めた

## 企画への示唆

- **誰もが知る IP × 既存テンプレ** に乗せると、初日 DL は IP 力で取れる（IP 単独で配信初日 100 万 DL ラインが見える）
- **収集枠を 70+ で開く** と、覚醒・装備の上位ループを設計しても在庫が枯渇しない
- **キャラ建設パート** を入れると、戦闘のスタミナ消費後の離脱を **街の更新** に振り替えられる
- **原典のギャグ演出** を必殺技に詰めると、戦闘自体の戦略性を浅く保ったままでも「見ていて飽きない」状態を作れる
- **ターン制 + 自動戦闘** はライト層の参入障壁を下げる一方、PvP アリーナを置けば中重課金層の継続消費先になる

## 比較で見るべきタイトル

- **Marvel Strike Force**（Scopely）— 同じスタジオの IP × ターン制ガチャ RPG。設計テンプレの源流
- **Disney Heroes: Battle Mode**（PerBlue）— カートゥーン IP × ターン制 RPG の同時代の競合
- **MARVEL Future Fight**（Netmarble）— IP × アクション RPG、対比的に「操作介入を深めた」事例
- **ディズニー ツムツム**（LINE/コロプラ）— 同じ「キャラ収集 × カジュアル」軸での日本側の到達点
