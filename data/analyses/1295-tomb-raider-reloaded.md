# トゥームレイダー：リローデッド 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「短時間で 1 ラン回し、毎回少しずつ強くなりながら遺跡を踏破する」** という体験ループであり、これは典型的なローグライク・トップダウンシューターの体験設計である。

`concept` に明示されている通り本作は「シリーズ初のローグライク採用作」で、トレジャールームを縦移動で踏破しながら武器・能力・パーマネントアップグレードを蓄積する構造を採る。`target` も Archero 等の "1 ラン数分のソロ・アクション" 愛好層を想定しており、ガチャ的な収集よりも **入力（操作）→ 出力（生死）** のフィードバックループが主軸になる。`social_axis` が「ソロ」である点も、対戦 / 協力的な社交要素ではなくプレイヤー個人の上達と運の引き直しが核であることを示している。

副次的に REW が効くのは、ローグライクである以上「パーマネント・アップグレード」「武器解放」「コスチューム蓄積」が継続動機の半分を担うため。ただし主はあくまでアクション体験で、収集は再走の燃料に位置づけられる。

## 体験の核 / 設計の発明

中核は **「IP の再ポジショニング × Archero 系のメカニクス輸入」** という二重の設計判断にある。シリーズ本編が PS4/PS5 のサバイバー三部作で重厚なシネマティック AAA を志向するなか、本作はモバイル縦持ち・1 ハンド寄りの軽量アクションへ意図的にスケールダウンし、ララの造形も三部作以前の "クラシック・ララ" に回帰させている。

| 要素 | 本作の選択 | ねらい |
|---|---|---|
| カメラ | トップダウン | モバイル操作との相性、視認範囲の確保 |
| プレイ単位 | 1 ラン数分のトレジャールーム踏破 | スキマ時間消費、Archero 系ユーザーの導線 |
| キャラ造形 | クラシック・ララ（90s シルエット） | シリーズ古参の郷愁喚起と差別化 |
| 進行設計 | ラン内ランダム強化 + ラン外パーマネント | ローグライクの "失敗が報酬になる" 構造 |
| 配信形態 | ストア通常版 + Netflix Games 版（広告/IAP なし） | 課金回避層の取り込み、ブランド露出の最大化 |

`social_axis` がソロであることで、ランキング / ギルドのような外部圧力ではなく "次のラン" への内的動機で回す設計になっている。

## バズ要因 / 人気の根拠

- 2023/2/14 のグローバル正式配信で、シリーズ本編が AAA に専念するなかでの **モバイル番外編** として国内外メディアに取り上げられた（4Gamer・Game Informer 等）
- **クラシック・ララ回帰** という意匠的な仕掛けが、シリーズの 25 周年以降の "原点回帰" 機運と合致し SNS で話題化
- **Netflix Games 同時配信**（広告 / IAP なし版）が独立した訴求を持ち、Netflix の "ゲーム枠拡張" 文脈でも報道された
- 「Archero × トゥームレイダー」というハイブリッド・ジャンルの言語化がレビュー軸を作りやすく、ジャンル比較記事の格好の素材になった

## 企画への示唆

- **AAA IP をモバイルへ持ち込むときはジャンルごと別物に振る** と、本編と食い合わない補完線が引ける（本作は AAA サバイバー本編に対するローグライク番外）
- **キャラ造形をシリーズ内の別世代から借用する** と、新規ユーザーには新鮮、古参には郷愁という二層訴求になる
- **Archero 系のラン構造を IP に乗せる** と、無名 IP では難しい初回 DL の壁を IP 認知で突破できる
- **ストア課金版とサブスク無課金版を併走させる** と、課金回避層もファネルに乗せられ、ブランド露出の総量が増える
- **social_axis をソロに割り切る** と、運営コストの高いランキング・同盟戦を持たずに済み、買い切り寄りの設計でも IP 露出商材として成立する

## 比較で見るべきタイトル

- Archero（Habby）— 本作が直接参照したトップダウン・ローグライク・シューターの祖型
- Tomb Raider: Definitive Survivor Trilogy — 同 IP の本編側で、AAA シネマティック路線との対比
- Hades — ローグライクの "ラン外パーマネント進行 + 物語" の代表例、本作の進行設計の親戚
- Vampire Survivors — モバイル・カジュアル層が "1 ラン数分のトップダウン" に慣れる素地を作った隣接ジャンル
- Lara Croft GO（2015、Square Enix Montréal）— 同 IP の過去モバイル番外（ターン制パズル）、IP のジャンル振り替え戦略の前例
