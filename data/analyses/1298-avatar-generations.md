## なぜ REW 主分類なのか

プレイヤーの主動機は「歴代アバター（アン／コラ／キョーシ／ロク）を含む原作キャラクターをガチャで集め、星上げ・装備強化でスクワッドを編成する」こと、すなわち **集める** である。Marvel Strike Force / SW: Galaxy of Heroes / RAID: Shadow Legends 系の Western スクワッド RPG の文法に Nickelodeon IP を載せた設計で、ターン制バトルは収集キャラクターの能力差分を体験させる「展示場」として機能する。

`concept` にある「キョーシ／コラ／ロクなど歴代アバター時代も横断して遊べる」という訴求は、原作の時系列を超えて IP の歴代主役を **同じ排出プール上に並べる** という Western gacha のコレクションロジックそのものであり、REW として読み解くのが素直である。secondary に EXP と NAR を置いているのは、(a) ターン制スクワッド戦闘の編成パズル（EXP）と、(b) 原作主要エピソードを章立てでなぞるキャンペーン（NAR）が下支えしている構造を示すため。両者は「集めたキャラを使う口実／集める動機を補強する素材」として REW に従属している。

## 報酬ループの組成

| レイヤー | 内容 | リテンション寄与 |
|---|---|---|
| 一次：キャラ収集 | アン／カタラ／ソッカ／トフ／スーキ／ズーコ＋歴代アバターをガチャ排出 | IP 横断ピックアップで「次の歴代アバター実装」を期待させる |
| 二次：星上げ・装備 | 同キャラ重複でランクアップ、装備強化で戦力底上げ | 引いた直後で終わらせず、断片収集を恒常タスク化 |
| 三次：キャンペーン進行 | 原作各章をステージ化、進行報酬としてキャラ片配布 | NAR を周回素材に変換、ファンの IP 消費欲をループに固定 |
| 四次：PvP/レイド類 | スクワッド編成の最適解探索（Western mobile gacha 定番） | 戦力インフレを通じて課金圧をかける |

前提として Western mobile gacha は「全キャラ最終解放可能だが、片集めに数百日かかる」設計が標準で、本作も Square Enix London Mobile が同じ枠組みを踏襲したと見られる。原作横断という IP 強度が一次レイヤーの推進力で、二次以降は MSF / SWGOH の写像である。

## バズ要因 / 人気の根拠

- Nickelodeon の看板アニメ『アバター：伝説の少年アン』および続編『コラの伝説』IP を使い、コラ／キョーシ／ロクなど **歴代アバターを横軸で並べた** ことが配信前トレーラーで話題化
- 配給は Square Enix（London Mobile スタジオ）、開発は Navigator Games という Western 体制で、和ゲー大手 × Western スクワッド RPG という座組自体がプレス露出を生んだ
- 一方で 2022/8 ソフトローンチ → 2023/1 グローバル配信 → **2023/12/2 サービス終了** と、配信わずか 11 ヶ月で撤退。Square Enix のモバイル事業縮小路線を象徴する事例として記憶された
- 日本国内では iOS/Android 配信ながらほぼ未認知で、`sales_jp` `sales_world` ともに非公表のまま閉鎖

## 企画への示唆

- **強力 IP × Western スクワッド RPG テンプレ** をそのまま重ねても、MSF/SWGOH/RAID の既存プレイヤー基盤を奪うのは難しい（同型ゲームは LTV を既存タイトルに先取りされている）
- 「歴代主役を横断ピックアップする」設計はファンの収集動機を最大化するが、**原作未消化のライト層には響かない** ため、IP 認知度に対する想定 ARPU を保守的に見積もる必要がある
- Square Enix London Mobile の撤退タイミングが示すのは、**和大手が Western mobile gacha を運営するときの固定費** が、IP ライセンス料込みでは回収困難だったという事実
- ソフトローンチから本配信までを長く取っても、**コア指標（D30, ARPDAU）が閾値に届かなければ閉鎖判断は 1 年以内** に下る、というモバイル運営のリアル
- ライセンス IP モバイル RPG を企画する場合は、Western 既存ガチャと並ぶのではなく **「原作にしかない儀式（アバター状態、ベンディング属性相性等）」をコアメカニクスに昇格** させて差別化軸を作る必要がある

## 比較で見るべきタイトル

- Marvel Strike Force（FoxNext / Scopely）— 同ジャンルの Western スクワッド RPG 王者
- Star Wars: Galaxy of Heroes（EA Capital Games）— IP × ターン制スクワッドの原型
- RAID: Shadow Legends（Plarium）— Western mobile gacha の運営テンプレ
- Marvel Future Fight（Netmarble）— アジア大手が Western IP を扱った先行例
- Hogwarts Mystery（Jam City）— 強 IP × モバイル運営の難しさを示す比較対象
