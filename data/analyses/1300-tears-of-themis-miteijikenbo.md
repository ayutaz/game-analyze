# 未定事件簿 / Tears of Themis 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機を 1 文で言うと「推しキャラの SSR カードを引き、衣装と専用ストーリーを集める」である。シナリオ自体は推理アドベンチャー仕立てだが、本作のリテンションを支える月次の駆動装置は **カードガチャ → 新衣装 → 限定ボイス／ミニ章開放** という収集ループで、これは REW（報酬型）の典型的構造をなぞる。

concept にある「カードガチャでキャラの新衣装・新ストーリーを集めるサ運営型」、target にある「推し活したい層」がそれを補強する。買い切りオトメゲーが「本編エンディング到達」をゴールに置くのに対し、本作は「推しの全カードコンプ＋親密度カンスト」が事実上の継続ゴールであり、消費の単位が章ではなく **キャラ別カードプール** になっている点が REW である根拠。secondary に NAR が入るのは、ガチャの引き先である SSR カードに本編とは別の短編シナリオが紐付き、物語消費が報酬の中核を成すためで、ガチャと物語の境界が曖昧な「シナリオガチャ」型として機能している。

## 報酬ループの組成

カードを引く動機を、層構造で分解すると以下の通り。

| 層 | 構成要素 | プレイヤーが得るもの |
|---|---|---|
| キャラ層 | 4 人の攻略対象（陸景和 / 左然 / 夏彦 / モーセス） | 「推し」を 1 人に固定して長期投資 |
| カード層 | SSR / SR / R のレアリティ × 月次新カード | 衣装・スチル・親密度バフ |
| シナリオ層 | カード固有の短編＋限定イベント章 | 本編とは別軸の「もしも」シナリオ消費 |
| ボイス層 | 親密度上昇でボイス・メッセージ解放 | 推し活の日常接触ポイント |
| 推理層 | 法廷パートのカードバトル（属性／親和） | ガチャ更新の機能的口実 |

ガチャは「新衣装プロモ → 期間限定イベント章 → ストーリー完結 → 復刻」というサイクルで回り、限定 SSR を逃すと当該シナリオが半年〜1 年読めない構造になっている。**読む権利＝引く権利** に変換することで、シナリオ評価が高いほどガチャ消費が伸びる正の循環が成立している。法廷推理パートは「カードを編成する理由」を機能面から提供しており、純粋なオトメ AVG ではなく **ガチャの動機付け装置として推理が乗っている** ことが本作の設計上の特徴。

## バズ要因 / 人気の根拠

- miHoYo（崩壊シリーズ・原神）が初めて手掛ける女性向け恋愛ガチャという話題性。同社の 3D 表現力とオトメ市場の組み合わせは事前段階で大きな期待を集めた。
- グローバル版（英語・日本語・韓国語等）の事前登録が 65 万を突破し、中国本土を超えた市場拡張に成功。
- 原神プレイヤー層からの流入が継続的に起き、miHoYo IP 横断のファン経済が機能している。
- SSR キャラ別のガチャイベントごとに SNS でカード絵・スチルが拡散し、月次の話題が途切れない運用設計。

## 企画への示唆

- ガチャの引き先に「衣装」だけでなく「短編シナリオ」を紐付けると、評価軸が確率から物語価値に移り、可処分時間と可処分金額の両方を奪える。
- 攻略対象を 4 人に絞ると「推し固定 → 推し関連カードのみ追う」プレイ最適化が起き、長期 LTV が安定する（多すぎると分散して脱落）。
- 本編と独立した「もしも IF シナリオ」をカード側に置くと、本編完結後もコンテンツを供給し続けられる。
- 既存 IP（原神等）と世界観を分離しつつ、開発元ブランドだけ共有すると、本体 IP のファンを送客しながら新規層を取り込める。
- 推理／戦闘などの機能的ゲームパートを乗せると、ガチャ更新に「機能的理由」を付与でき、純愛 AVG より運営寿命を伸ばせる。
