## なぜ NAR 主分類なのか

プレイヤーの主動機は **「読む」** — 5 人のキャラクターと 11 日間かけて積み上がる会話ログを追体験することにある。concept にある「実時間進行のチャットルームに参加し、11 日間かけて関係を築く」は、ノベルゲームの章立てを **カレンダー＋プッシュ通知** に置換した物語消費構造であり、入力されるテキストの大半は分岐選択肢ではなく「キャラ同士の雑談を読む」時間である。

target が示すように、SNS 共有込みで物語を消費する 10–30 代女性が中核で、操作スキルや戦略性は問われない。`secondary: ["REW"]` はハート（好感度）/ ハニーバット（時短アイテム）/ ルート解放 / CG/メール収集といったコレクションがリプレイ動機を底上げする副軸として効いている。社交軸は `null`（NAR のため）。

## 物語の駆動構造

メッセンジャー UI による「実時間で物語が進む」設計が、章立てと SNS 拡散性を同時に駆動している。

| 構造要素 | 仕掛け | プレイヤー行動への効果 |
|---|---|---|
| 章立て | 11 日間 × 24 時間のリアルタイム進行 | 「Day 5 朝のチャット」を逃すと欠席扱い、ログイン頻度が物語進度に直結 |
| 選択肢 | チャット中の選択 → 各キャラのハートゲージ | 5 ルート（Yoosung / Zen / Jaehee / Jumin / 707）+ Deep Story + V/Saeran で分岐 |
| キャラ関係性 | RFA というメッセグループに「新メンバー」として加入する自己投入 | プレイヤー自身が登場人物として会話に書き込む一人称没入 |
| カレンダー | 朝・昼・夕・深夜の固定タイムスロット + 不定期通話 | スマホ通知が日常生活に侵食、物語と生活の時間が同期する |
| 時短経路 | ハニーバットでバックログを開放 | 「リアタイで追えなかった会話」を救済し、社会人ユーザーを取りこぼさない |

物語消費を駆動しているのは **「キャラがプレイヤーの不在中も生活している」という錯覚** であり、これがログを読み返す動機を生む。

## バズ要因 / 人気の根拠

- マーケ予算ゼロで配信開始から **2.5M DL、累計 8.5M+ DL** に到達（韓国インディーの Cheritz が自社配信）
- チャット UI のスクリーンショットがそのまま **実在の LINE / KakaoTalk 会話のような見た目** で SNS に流通し、Tumblr / Twitter で fan art / 二次創作 / ミーム化が連鎖
- 深夜 3 時に「707 から電話が来た」など **物理時刻と物語時刻が一致する仕掛け** が体験談として共有された
- 韓流ドラマ的キャラ造形（財閥御曹司 Jumin、ハッカー 707 等）と乙女 / BL 兼用の関係性設計で、英語圏・日本・東南アジアにも越境

## 企画への示唆

- **UI を既存生活アプリに擬態させると物語が現実に侵食する** — 章立てをカレンダー / 通知に置換するだけで没入感が桁違いに上がる
- **「逃すと取り返せない」時間制約を入れるとログイン頻度が上がる** が、必ず時短アイテム（=課金フック）でセーフティを併設しないと社会人を逃す
- **スクショ映えする UI は最強のマーケ予算ゼロ施策** — フォント・吹き出し・タイムスタンプを実在 SNS に寄せると二次創作が勝手に広告化する
- **NAR 主分類でも REW を副軸に置くとリプレイ性が確保できる** — ルート分岐 + CG/メール収集の二層構造で 1 ルート完走→他ルート挑戦の動線を作る
- **プレイヤーを「グループチャットに後から入った新メンバー」として配置すると自己投入が滑らかに起こる** — 一人称の独白より既存コミュニティへの参入の方が自然な没入を生む

## 比較で見るべきタイトル

- **Lifeline (3 Minute Games, 2015)** — テキスト + 通知の実時間進行 NAR の祖型
- **A Normal Lost Phone (Accidental Queens, 2017)** — スマホ UI そのものをゲーム画面にした NAR
- **Florence (Mountains, 2018)** — モバイル特化のミニマル恋愛 NAR、章立てがメカニクス化
- **あんさんぶるスターズ!! (Happy Elements, 2020)** — グループチャット型コミュ + ガチャの NAR×REW ハイブリッド
- **Doki Doki Literature Club! (Team Salvato, 2017)** — UI 自体が物語仕掛けに変わる VN の系譜
