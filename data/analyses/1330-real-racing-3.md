# リアルレーシング3 (Real Racing 3) 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「実車を握って走るドライビング体験そのもの」** であり、車両収集や金策はその走行体験を更新するための副次的なループに置かれている。

concept にあるとおり、本作は実在サーキットと実車をフォトリアルに再現した「スマホ向けシム寄りレース」であり、Forza / GT 系の据置シミュレータをモバイルで成立させること自体が中核命題である。target は「通勤の合間に実車を運転したい 20–40 代の自動車・モータースポーツ好き」で、ガチャ的な収集動機より「F1 マシンをサーキットで走らせる手触り」を消費している層に焦点が当たっている。social_axis が **非同期** なのは後述する Time Shifted Multiplayer (TSM) の発明によるもので、サーバ常駐なしで「他人の走り」と競う体験を作り出している点が EXP を支える。

secondary に REW があるのは、車両アンロック・パーツアップグレード・F1 / NASCAR / Le Mans の期間限定イベント報酬といった育成系が長期運営を支える二次ループになっているためで、これらが無ければ走行体験のリプレイ動機が痩せる構造になっている。

## 体験の核 / 設計の発明

中核は「フォトリアル＋実車ライセンス＋非同期マルチ」を **基本無料モバイル** という制約下で成立させたパッケージングであり、特に TSM は据置レースゲームには存在しなかった発明である。

| 設計要素 | 内容 | 体験への効果 |
|---|---|---|
| 実車ライセンス | Ferrari / Porsche / Lamborghini / F1 / NASCAR 等 | 「あの車を運転している」感覚を担保 |
| フォトリアル描画 | 実在サーキットの 3D 再現 | 据置シムに近い没入感をモバイルで再現 |
| Time Shifted Multiplayer | 他プレイヤー走行データを AI ゴースト化 | 常時オンラインなしで「対人感」を提供 |
| 公式コラボイベント | F1 シーズン / Le Mans 24h 等を定期更新 | 実モータースポーツカレンダーに同期 |
| 段階的アンロック | 車両クラス → サーキット → イベント | 通勤プレイの 5 分単位で進捗が刻める |

前世代との差分は明確で、(a) Real Racing 1/2 では成立しなかった「常時オンラインなしの非同期 PvP」を TSM で解いたこと、(b) パッケージ売切りから F2P 長期運営へ移行したことの 2 点が、累計 5 億 DL 超という規模を生んだ設計上の分岐点である。

## バズ要因 / 人気の根拠

- **累計 5 億 DL 超**（popularity 記載）。2013 年配信開始から 13 年間の長期運営でモバイルレース最大規模のインストールベースを形成。
- **TSM が業界標準化**：以降のモバイルレース（Asphalt 系、CSR 系の一部、Need for Speed: No Limits 等）が非同期マルチを採用する流れを作り、「サーバ負荷を抑えつつ対人感を出す」設計テンプレートを提供した。
- **実車ライセンス網の広さ**：Ferrari / Porsche / Lamborghini / Aston Martin / Bugatti 等の高級スポーツカーと、F1 / NASCAR / Le Mans 公式の三大モータースポーツ枠を同一タイトルに集約した稀少なライセンス IP ポートフォリオを保有。
- **EA / Firemonkeys による継続運営**：パブリッシャーが EA、開発が豪 Firemonkeys Studios（旧 Iron Monkey + Firemint）で、買収後も 10 年超エイブにアップデートが続いている点が IP の信頼性を担保。

## 企画への示唆

- **オンライン PvP を AI ゴースト化すると**、サーバコスト・マッチング待ち・通信品質の 3 重課題を回避しつつ「他人と競う」体験を提供できる（TSM 方式）。
- **実モータースポーツのカレンダーに乗ると**、ゲーム外ニュース（F1 開幕 / Le Mans 開催）が自然に流入動線になり、広告費に依存せずに季節的なリテンションスパイクを作れる。
- **F2P モバイルでフォトリアル＋実車ライセンスを揃えると**、据置シムから降りてきたコア層と、カジュアル層の二層を同一タイトルで抱えられる（ただしライセンス費は重い）。
- **走行体験を主動機 (EXP) に置きつつ車両アンロックを副 (REW) で支えると**、ガチャ依存にせず「次に走らせたい車」を動機として育成ループを回せる。
- **据置 IP のモバイル移植ではなく、モバイル専用の長期運営タイトルを立てると**、配信開始時のグラフィック上限よりも「10 年運営を支える車両追加とイベント更新のパイプライン」を設計優先することが寿命を決める。

## 比較で見るべきタイトル

- **Asphalt 9: Legends** (Gameloft) — モバイルレースの対抗軸。アーケード寄り＋派手な演出で差別化。
- **CSR Racing 2** (NaturalMotion / Zynga) — ドラッグレース特化＋実車ライセンスの REW 寄せ事例。
- **Forza Motorsport / Gran Turismo 7** — 据置のフォトリアル・シム本流。Real Racing 3 が「モバイル側からどこまで詰めたか」の上限値を示す。
- **Need for Speed: No Limits** (Firemonkeys 同社) — 同じ開発元による F2P モバイルレースの姉妹線、アーケード寄りで対比可能。
