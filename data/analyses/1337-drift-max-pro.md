## なぜ EXP 主分類なのか

プレイヤーの主動機は「ハンドブレーキで車を滑らせ続ける操作の手触りを味わう」こと。報酬集めではなく、コーナーごとにスライドの角度を当て直し続ける運動的気持ちよさが軸にある。

concept が示すとおり、本作はレースゲームから「順位を競う」要素を意図的に削ぎ、「いかに長く、いかに高得点で滑るか」というスコア競技に振り切っている。コースは東京・NY・モスクワなどの夜景固定で、相手車両は実質的に「採点用のオブジェクト」に近い。target である頭文字 D / 湾岸ミッドナイト世代と Asphalt 卒業組のモバイルユーザは、ともに「ドリフトの操作そのものへの没入」を求める層であり、これは典型的な EXP（体験型）の動機構造。social_axis が「非同期」なのは、同期対戦ではなく世界ランキング / 友人スコア超え / SNS 動画拡散で他者と接続するためで、「観客がいないと続かないがリアルタイム対戦も不要」というモバイル特有の他者軸を踏襲している。

secondary に REW を置いているのは、車両ガレージ・パーツチューン・カラーリングのカスタム層が滑走体験を裏で支えているため。報酬は主目的ではなく「次のステージで気持ちよく滑るための装備」として機能する。

## 体験の核 / 設計の発明

中核メカニクスはきわめて単純で、「アクセル / ハンドブレーキ / ステアリング」の 3 入力だけでカウンターを当て続ける操作系。タッチ操作でも滑り出しが過敏にならないよう、グリップ→スライド遷移にイージングを噛ませている点がモバイル特化設計の肝。

| 設計レイヤ | 内容 | 効果 |
|---|---|---|
| 入力モデル | 加速 + ハンドブレーキ + 傾き / タッチ操舵の 3 入力 | 学習コスト 1 ステージで完結、片手プレイにも対応 |
| スコア構造 | ドリフト角度 × 持続時間 × 連結（コンボ） | 「同じコースを繰り返し走ってスコアを伸ばす」モチベを成立 |
| ステージ演出 | 東京・NY・モスクワ等の夜景固定背景 | カーカルチャー記号（湾岸 / 首都高 / 街灯反射）の共感装置 |
| カスタム層 | 実車風ライセンス車両 + パーツ + 塗装 / デカール | 滑走体験を「自分の車で気持ちよく滑る」体験へ変換 |
| 非同期他者軸 | リーダーボード + リプレイ / 動画書き出し | SNS 動画拡散がそのままマーケティングチャネル化 |

前世代の Real Racing / Asphalt はレースの「順位」を起点に車両報酬を回すのに対し、Drift Max Pro は「滑り続ける操作」自体をループ単位に据えた点が差分。CarX Drift Racing 系の物理志向に対しては、アーケード寄りの当てやすさで間口を広げている。

## バズ要因 / 人気の根拠

- 全世界 1.2 億 DL 超という配信規模で、ドリフト特化モバイルとしては最大級の母数を確保
- 直近 30 日でも約 580 万 DL を記録し、リリース 9 年目（2017→2026）でなおアクティブに新規流入が続いている長寿曲線
- 東京・NY・モスクワを舞台にした夜景コース + 実車風カスタム車両が、TikTok / YouTube Shorts のドリフト動画ジャンルと相性よく、ユーザ生成動画が継続的に拡散
- インディー publisher（Tiramisu の self-published）でありながら大手 Asphalt / Real Racing と同じ棚で生き残っており、ニッチ特化 × 長期運用の成功例として参照されやすい

## 企画への示唆

- レースから「順位」を抜き「単一動作の質」をスコア軸に置き換えると、ステージを使い回しても周回モチベが立ち上がる
- 3 入力以下に操作を絞ると、モバイルでアーケード文化（頭文字 D 等）の操作快感を翻訳できる
- 背景を実在都市の夜景に固定すると、ユーザ生成動画がジャンル記号（首都高 / 湾岸）に紐付き SNS 拡散コストが下がる
- カスタム / ガレージは「報酬目的」ではなく「次の体験を気持ちよくする道具」に位置付けると、EXP 主軸でも REW 層が無理なく機能する
- 非同期ランキング + リプレイ動画書き出しを実装すると、同期対戦のサーバ負荷なしに他者軸を成立させられる

## 比較で見るべきタイトル

- CarX Drift Racing / CarX Street（同ジャンル、物理志向の対極ポジション）
- Real Racing 3 / Asphalt 9（モバイルレースの王道、順位ベース REW 構造との対比）
- 頭文字 D Arcade Stage シリーズ（操作快感の源流、アーケード文化の参照点）
- Forza Horizon シリーズ（コンソール側のドリフトスコア設計、スコア軸の比較対象）
- need for speed: no limits（同じモバイル × カーカルチャー、ナラティブ／レース寄りの対照例）
