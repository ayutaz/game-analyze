# Forza Customs

## なぜ EXP 主分類なのか

プレイヤーの主動機は「マッチ3パズルを解いて廃車を自分好みに直していく作業そのものを味わう」ことにある。Forza 本編が運転体験を中核に置くのに対し、本作はステアリングもタイムアタックも持たず、レンチを振るう手触りとビフォーアフターの変化を体験させる。`social_axis: ソロ` であり、他者との対戦や同期協力は設計に含まれていない。クルマを愛でる時間を毎日数分ずつ積み上げるソロのクラフト体験が中心だ。

副次的に REW が効いているのは、修復のたびに新しい車両（クラシック / マッスル / 現代）がアンロックされ、カスタム塗装やパーツの選択肢が積み上がる収集レイヤーがあるため。とはいえガチャ排出や育成ステータスではなく、「直したクルマがガレージに増える」というショーケース型の報酬で、動機としてはあくまで修復作業を続けるためのドライバーに留まる。`target` が「レース操作は苦手だが車そのものは好きな 20–40 代カジュアル層」と定義されている通り、競争ではなくセラピー的なクラフト体験を提供する EXP/ソロ作品と読むのが妥当である。

## 体験の核 / 設計の発明

中核メカニクスは「マッチ3 → 通貨 → カスタムパーツ購入 → 車両見た目変化 → 次のステージ解禁」という Homescapes / Gardenscapes 型のメタゲーム構造を、Forza ライセンスのクルマに乗せた点にある。プレイヤーは盤面を消すたびに評判ポイントを稼ぎ、レストアの一手（ボンネット塗装、ホイール交換、シート張替えなど）を実行する。

| レイヤー | 入力 | 出力 | プレイヤー感情 |
|---|---|---|---|
| マッチ3パズル | 3 手以内に色を揃える | 評判 / コイン | 短時間の達成感 |
| レストア / カスタム選択 | パーツ・カラー指定 | 車両の見た目変化 | 「自分の趣味が反映された」感 |
| 車両アンロック | 章クリア | 次の廃車案件 | 次回起動の動機 |
| ガレージ | 完成車を並べる | コレクション陳列 | 振り返りの所有感 |

前世代との差分は、(a) Forza ブランドが初めてレース以外の核体験に踏み出した点、(b) Hutch Games が培ったライセンス車両モバイル運営ノウハウを Scapes 系メタ構造に流し込んだ点。social_axis ソロの効果として、対戦圧やリアルタイム同期がないぶん隙間時間に開きやすく、車種ロアやカスタム趣味の濃さで没入を作る方向に振っている。

## バズ要因 / 人気の根拠

- Forza シリーズ初の非レースタイトルとしての話題性（2023-11 ローンチ時に Forza ファンとパズルファン双方の注目を集めた）
- Xbox Game Studios パブリッシングと Turn 10 ライセンスというブランド信頼で、無名インディーパズルとは異なる初動到達を確保
- Hutch Games（Top Drives / F1 Clash 等）が持つ車ライセンスモバイルの運営実績による安心感
- 一方で `sales_world: —（2025-03-10 サービス終了）` の通り、運営収益が損益分岐に届かず約 16 か月で終了。話題先行で長期リテンションに繋がらなかった事例として記憶された

## 企画への示唆

- 既存 IP を異ジャンルに移植するなら「IP の何を残し何を捨てるか」を明示する → Forza は「車」を残し「運転」を捨てた、その判断の是非がそのまま KPI に出る
- マッチ3 メタゲームに IP を乗せるとアートと開発工数が跳ね上がる → Scapes 系の汎用キャラより車両モデリングのほうがアセット単価が高く、ARPU が同じなら回収難度が上がる
- ソロ × カスタム × ブランド愛 の三点だけで運営型モバイルを成立させるのは難しい → PvP / ギルド / 期間限定イベントの社会的圧力なしでは DAU 維持の足場が弱い
- 非レース層に届けたい意図は良いが、Forza 本編ファンと Homescapes 層は重なりが薄い → クロス IP 企画では「両方のファンが価値を感じる接点」を 1 機能で表現できるか検証する
- サ終までの期間（16 か月）を逆算すると、ライセンスモバイルは初年度の DL ではなく D30/D90 リテンションで判断すべき → ローンチ KPI の設計段階で「話題で稼ぐ DL」と「定着で稼ぐ収益」を別指標化する

## 比較で見るべきタイトル

- Homescapes / Gardenscapes（Playrix）— マッチ3 + メタ進行の原型、本作のメカニクス基盤
- Top Drives（Hutch Games）— 同 publisher の車ライセンスカードバトル、ライセンス調達ノウハウの源流
- Project CARS GO（2022 サ終）— 同じく車 IP × カジュアル化失敗事例
- Forza Horizon 5 — IP 本家、運転体験を核に置くオリジナル
- Hot Wheels Unleashed — 車 IP × 非シミュ系の別解、コンソール側のアプローチ
