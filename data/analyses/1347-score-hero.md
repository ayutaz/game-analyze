# Score! Hero (2015 / First Touch Games)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**スワイプで決定的瞬間の最適解を描く**」という操作体験そのもので、選手育成や報酬獲得は副次的なフックに留まる。試合をフルで操作する従来型サッカーではなく、**ゴール前・チャンスシーン 1 ピックアップだけを切り出してパス／シュート軌道をプレイヤーに線で描かせる**設計が中核にあり、この「線を引く快感」がリプレイ動機の大半を占める。social_axis は「ソロ」で、対人 PvP も同期マルチも持たず、純粋に自分とパズル化された 1 シーンの対話で完結する。

concept にある「試合まるごとではなく各シーンの最適解を引く構造で、隙間時間に 1 ステージずつ消化」という言い回しが体験の本質を端的に示しており、target も「片手プレイしたいライト層」「ガチサッカーよりカジュアル志向」と、操作感そのものへの嗜好で定義されている。secondary に REW が入るのは、選手レベル・髪型・装備などの育成カスタマイズと、ステージクリア時のスター獲得／リトライ消費 (Rewind) があるためで、これらは「もう一度トライしたい」を支える後付けレイヤーとして機能する。

## 体験の核 / 設計の発明

| 設計要素 | 内容 | 体験への寄与 |
|---|---|---|
| スワイプ軌道入力 | パス先・シュート角度を指でなぞって決定 | 「自分が描いた線でゴールが決まる」自己効力感 |
| シーン切り出し | 試合 90 分を 1 ステージ＝決定機 1〜5 本に圧縮 | 1 ステージ 30〜60 秒、隙間時間に消費可能 |
| Rewind 課金 | 失敗時に巻き戻し、同じ局面を再挑戦 | 詰まりポイントでの離脱抑止 + 課金導線 |
| シーズン進行 | キャリアモードでクラブ昇格・代表招集 | パズル消化に「ヒーロー成長譚」の薄い物語性を付与 |
| 3 ★ クリア | 完璧な軌道で★3、雑なクリアで★1 | やり込み層に再挑戦の理由を提供 |

前世代のモバイルサッカー (FIFA Mobile / ウイイレモバイル系) が**試合フル操作 + ガチャ編成**で AAA を再現していたのに対し、Score! Hero は **「サッカー = 11 人 90 分」を解体し、決定機だけを抽出してパズル化**した点が発明にあたる。Angry Birds や Cut the Rope と同じ「線を引いて物理を制御する」系譜にサッカー IP を載せた構造とも言える。

## バズ要因 / 人気の根拠

- **世界累計 3 億 DL 超**を達成し、英国 First Touch Games の Dream League Soccer と並ぶ二枚看板になった
- **「片手 + 縦持ち + 1 試合 1 分」**というスマホ最適化が徹底され、通勤・休憩シーンに刺さった
- スワイプで軌道を引く操作が**短尺動画映え**し、ファインゴールのクリップが SNS / YouTube で拡散
- 既存 IP に依存しない **オリジナル選手「Hero」を育てる**フォーマットがライセンス料を抑え、長期運営とグローバル横展開を可能にした

## 企画への示唆

- **競技の長尺を 1 シーンに圧縮**すると、隙間時間に消費される頻度が跳ね上がる（試合 90 分 → 決定機 30 秒）
- **入力をスワイプ軌道に置換**すると、ボタン操作より「自分が描いた」感が増し、成功時のシェア動機が生まれる
- **失敗 → Rewind (巻き戻し) の有料化**は、難所での離脱を防ぎつつ自然な課金導線になる（ライフ消費型より体験ロスが小さい）
- **オリジナル主人公を育てるキャリア構造**にすると IP ライセンス費を回避しつつ感情移入を確保できる（実名クラブ／代表は背景としてのみ使用）
- **★3 評価制**を入れると 1 ステージあたりの可処分時間を 1 周→ 3 周に伸ばせる

## 比較で見るべきタイトル

- **Dream League Soccer** (First Touch Games / 同社姉妹作、こちらは試合フル操作型)
- **FIFA Mobile / ウイイレモバイル** (試合フル操作 + ガチャ編成、AAA 再現型)
- **Angry Birds** (スワイプ軌道で物理を制御する系譜の元祖)
- **Golf Clash** (同じく決定機を線で引くカジュアルスポーツ、PvP 軸)
- **Football Manager Mobile** (試合の解体方向は同じだが、解体先がパズルでなく数表 / 監督視点)
