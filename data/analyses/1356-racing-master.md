# レーシングマスター 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「実車の挙動を 1/1000 秒削りながら他者と競う」こと、つまり**走らせること自体**が報酬になる構造である。concept にある「コンソール水準のグラフィックと挙動を 5G 時代のスマホに持ち込んだ本格 PvP レーシング」という設計思想は、車を見せるためではなく**走らせるため**のものとして組まれている。target も「Forza / グランツーリスモ / アスファルト経験者でモバイルで腰を据えてタイムアタック / PvP に取り組みたい層」と明示されており、社会軸が「対戦」に置かれている点と整合する。

副次的に REW が効くのは、ガチャ / 育成で入手した実車の性能差がそのままレース体験の選択肢を広げる**前提**を担っているためで、収集自体はゴールではなく「走らせたい挙動」を確保する手段として組み込まれている。

## 体験の核 / 設計の発明

中核メカニクスは Codemasters の F1 / DiRT 系シリアスシムをモバイル向けに再構築した物理挙動と、NetEase 側の運営・収益基盤の合成にある。前世代モバイルレース（アスファルト 9、Real Racing 3、CarX Street）との差分を分解する。

| 設計レイヤ | 前世代の標準 | レーシングマスターの選択 |
|---|---|---|
| 物理挙動 | アーケード寄り（ドリフト主体・グリップ簡略化） | Codemasters 由来のシム挙動、タイヤ温度・荷重移動を実装 |
| 実車表現 | モデルは精緻だがサウンドは合成 | 120 台超を**レーザースキャン**＋実車収録サウンド |
| 対戦設計 | 非同期ゴースト中心 | リアルタイム PvP を 5G/低遅延前提で常時稼働 |
| 収益化 | 燃料制 + IAP（Real Racing 3 型） | ガチャ/育成のモバイル RPG 型をレース体験に接続 |
| 操作系 | チルト or 簡易ボタン | 仮想ハンドル + アシスト段階化（コンソールパッドに近い） |

social_axis「対戦」は、Codemasters の物理を**他者の前で発揮させる**仕掛けとして機能する。タイムアタック単体だと REW 寄りの収集ゲームになりがちなところを、リアルタイム PvP を中軸に置くことで EXP に固定している。

## バズ要因 / 人気の根拠

- NetEase × Codemasters の**スタジオ間 IP コラボ**として 2021 年から開発が告知され、F1 / DiRT 系の物理ノウハウがモバイルに移植される事自体が業界的な話題になった
- 中国本国リリース時に **App Store 無料 1 位・売上 2 位**を獲得、グローバルで **5,000 万 DL** を突破
- 日本ローンチ前の**事前登録 100 万人超**、リリース後は Perfume とのコラボで音楽 / ファッション層にリーチを広げた
- BMW・ポルシェ・フェラーリ等の実車ライセンスを束で確保し、**実在サーキット + 実車サウンド**で「Forza / GT のモバイル版」というポジショニングを成立させた

## 企画への示唆

- **据置きスタジオの物理を借りる**と、モバイル単独開発では到達できないシム挙動が短期で得られる（Codemasters F1 系の継承が好例）
- **実車のレーザースキャン + サウンド収録**にコストを集中させると、車そのものを「ガチャの当たり」として成立させやすく EXP と REW の橋渡しになる
- **リアルタイム PvP を中軸に据える**と、収集ゲーム化を回避して social_axis を「対戦」に固定できる
- **コンソール経験者をターゲットにする**なら、仮想ハンドル + アシスト段階化など UI を据置きパッドに寄せる方が離脱率が下がる
- **音楽 IP（Perfume 等）とのコラボ**を国別ローンチに合わせると、レース外のオーディエンスに刺さる導線が作れる

## 比較で見るべきタイトル

- Real Racing 3（EA / 2013）— モバイル実車レースの前世代基準、非同期 TSM 設計
- アスファルト 9: Legends（Gameloft / 2018）— アーケード側の到達点、ドリフト + ナイトロ設計
- Forza Horizon 5（Microsoft / 2021）— 据置きオープンワールド側の参照系
- CarX Street（CarX Technologies / 2024）— 同時期のモバイル本格レース競合
- グランツーリスモ 7（PDI / 2022）— シム挙動とブランドコラボの最上位事例
