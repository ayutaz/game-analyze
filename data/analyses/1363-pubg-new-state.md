# PUBG: NEW STATE (NEW STATE MOBILE)

## なぜ EXP 主分類なのか

このタイトルでプレイヤーが手を動かす主動機は「100 人の他者と撃ち合って最後まで生き残る」という対戦体験そのものであり、報酬獲得や物語追体験ではない。`social_axis: 対戦` が示す通り、毎マッチがその場で消費される一回性の勝負として設計されている点が EXP の根拠になる。

`concept` にあるように本作は 2051 年の「TROI」を舞台に、ドローン／回収車両／グレネードランチャー等の新ガジェットと武器カスタマイズを核に据えており、原作 PUBG MOBILE との差別化はあくまで戦闘体験の手触りで取りに行っている。`target` も「コアバトロワ層」「モバイルで本格 TPS を求める層」と射撃体験寄り。一方 `secondary: ["REW"]` に表れる武器スキン／パスといった収益動線が、対戦体験の継続インセンティブとして二層目に挿入されている構造である。

## 体験の核 / 設計の発明

中核は「PUBG の TPS バトロワ文法をモバイル向けに圧縮しつつ、近未来ガジェットで戦術選択肢を増やす」ことにある。前世代である PUBG MOBILE との差分を分解すると下記。

| レイヤ | PUBG MOBILE (2018) | NEW STATE (2021) |
|---|---|---|
| 世界観 | 現代の架空島 | 2051 年 TROI（SF 寄り） |
| 武器 | 実銃ベース | 実銃＋カスタマイズパーツ |
| 戦術ガジェット | 車両・グレネード中心 | ドローン／回収車両／グレラン |
| 想定端末 | 当時のミドル機 | 2021 年以降のハイエンド前提 |
| ターゲット | 新規 BR 層 | PUBG MOBILE の飽和組 |

`social_axis: 対戦` の効果は、近未来ガジェットが「先読みと裏取り」の駆け引き層を増やすことで現れる。ドローン索敵と回収車両（=ダウン仲間の回収）が加わったことで、3〜4 人スクワッドの動線設計がより明示的なロールプレイ（索敵／盾／回収／火力）に分業しやすくなっている。

## バズ要因 / 人気の根拠

`popularity` の核は「ローンチ後 5 日で 2,000 万 DL、約 1 ヶ月で 4,000 万 DL、2021 年 12 月時点で 5,000 万 DL 超」というモバイル F2P 史でも上位の立ち上がり速度。分解すると下記。

- **既存 IP の流用**: PUBG ブランドが既に世界規模で確立しており、新作告知だけで予約 DL が積み上がる構造を初期需要に変換した
- **165 ヶ国以上で App Store/Google Play 首位**: 単一地域偏重ではなく、日本／韓国／米国／ドイツ／サウジアラビア等を含む同時多発的なグローバルローンチが奏功
- **タレント起用 PR**: 日本では公式アンバサダーに渡辺直美を起用し、コアゲーマー以外への露出を稼いだ
- **「PUBG MOBILE の続編 / 進化版」というポジショニング**: 既存 BR 飽和層への乗り換え訴求が明確で、新規獲得とリブランドを兼ねた

## 企画への示唆

- **強 IP の続編をモバイルで仕掛ける際、現行作と「世界観の時代」をずらすと、自社カニバリゼーションを和らげつつ続編ナラティブが立つ**
- **コアの操作文法（TPS バトロワ）は据え置き、ガジェット層だけ差し替えると、既存ファンの再教育コストを抑えながら新規性を演出できる**
- **マルチリージョン同時ローンチに合わせて各地域のローカルタレントを公式アンバサダーに据えると、コアゲーマー外への初動 PR が伸びる**
- **「5 日で 2,000 万 DL」級の初動を狙うなら、ハードコア向け差別化（武器カスタマイズ／タクティカル要素）と量的施策（165 ヶ国同時配信／予約 DL）を両建てで設計する**
- **EXP 主導でも secondary に REW（スキン／パス）を仕込むことで、対戦体験の継続課金導線が後から効いてくる**

## 比較で見るべきタイトル

- **PUBG MOBILE**: 直接の前世代。差分設計の出発点
- **Apex Legends Mobile**: 同時期の競合モバイル BR、ヒーロー制との対比
- **Call of Duty: Mobile**: モバイル TPS/FPS の収益化モデル比較
- **荒野行動 (Knives Out)**: 日本市場での PUBG 系派生の先行事例
