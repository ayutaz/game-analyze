# オクトパストラベラーII

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「8人それぞれの旅路を最後まで読み切る」** こと。コマンドバトルやブースト/ブレイクの戦闘体験は楽しいが、エンドコンテンツやランダム生成の遊びはなく、最終的に手が止まる理由は「次のキャラの章を見たい」「クロスストーリーを揃えたい」になる。concept にある「独立した物語 × クロスストーリーで群像劇を強化」という構造は、まさに物語消費を駆動するための装置である。

target が王道 JRPG 懐古層（FF / ロマサガ系の 30〜40 代男性中心）に振られていることも、NAR を支持する。彼らが課金で何を求めるかではなく、「最後まで読ませてくれるか」「キャラが立っているか」で評価する層であり、secondary に **EXP**（コマンド戦闘・ブレイク・JOB システム）が残るのは、物語を読み進めるための燃料として戦闘が機能している、という従属関係である。social_axis は null で、ソロ完結のシングル体験。

## 物語の駆動構造

8 主人公 × 各 4〜5 章という形式は、JRPG 史において **群像劇 / オムニバス** の系譜（ロマサガ・サガフロ・ライブアライブ）に属する。本作はその構造を「並列に読み進められる + 後から交差する」形に整理した。

| 駆動装置 | 仕掛け | 物語消費への効果 |
|---|---|---|
| 8 シナリオ並列 | 主人公選択画面から好きな順序で章開始 | プレイヤーが「次に読む話」を自分で選ぶ、長編疲れの回避 |
| クロスストーリー | 2 人の主人公が交差する短編サブイベント | キャラ同士の化学反応で群像劇に昇格 |
| 昼夜システム | 同じ町でも昼と夜で会話・依頼・NPC が変わる | 1 マップで読める物語量が倍、再訪動機 |
| フィールドコマンド | 「聞き出す」「買収」「決闘」等で NPC から情報を引く | 物語が能動的に解錠される手触り |
| ブレイク+ブースト戦闘 | 弱点を割って一気にダメージ | 章のクライマックス戦闘が「読み物としての山場」になる |

各章はトラベラー単位で読み切る短編であり、章間で別キャラに切り替えても文脈が壊れない設計になっている。これにより「100 時間級 JRPG を 8 つの中編に分解する」という、現代の可処分時間に合わせた再パッケージングが成立している。

## バズ要因 / 人気の根拠

popularity が示すのは「HD-2D の完成度と続編評価、王道 JRPG 回帰」。これを分解すると：

- **HD-2D の完成形**: 前作（2018）で実験された 2D ドット + 3D ライティング表現を踏襲し、海・船・昼夜光源で表現幅を広げた。同表現はライブアライブ R / トライアングルストラテジー / ドラクエ III HD-2D リメイクへ波及し、Square Enix のブランドエンジンとなっている。
- **続編としての評価上昇**: 1 作目で批判された「8 シナリオの交差が薄い」をクロスストーリーで解消、独立完結ながら集合体としての厚みを増した。Metacritic / レビュー集計で前作超えの評価。
- **世界 1M 本超**: 発売初週から Switch ハード牽引タイトルとして週販ランキング上位。Steam では発売直後から日本語コミュニティで攻略動画が拡散。
- **王道 JRPG 回帰の文脈**: 2023 年は FF16 / ゼノブレイド 3 DLC / ロマサガ 2 リメイク発表と JRPG 復活の年で、本作はその文脈で「ドット派の旗艦」として語られた。

## 企画への示唆

- **長編 RPG を中編 × N 本に分割すると、可処分時間が短い層が完走できる**。各章の長さを 5〜8 時間に抑え、章末で気持ちよく切れる構造を必ず作る。
- **並列シナリオはクロスイベントで化学反応させると群像劇に昇格する**。完全独立のままだとオムニバス、交差させて初めて 1 つの物語体験になる。
- **昼夜・天候など 1 軸の状態変化を入れると、マップ再訪の動機が単純に倍化する**。コンテンツ量を増やさず体感ボリュームだけ増やせる。
- **NPC 介入コマンド（聞き出す / 盗む / 決闘）は、世界の奥行きを「読み物として能動的に解錠する仕掛け」になる**。物語消費型の RPG でも、プレイヤーの手数で物語が出てくる体験を作れる。
- **絵作りはエンジンになる**。HD-2D を 1 本目で確立、2 本目で深化、リメイクで横展開、と再利用すれば IP を超えてスタジオの看板になる。

## 比較で見るべきタイトル

- **オクトパストラベラー (2018)**: 同じ 8 主人公構造の原型、クロスストーリー不在の比較対象
- **ライブアライブ HD-2D リメイク (2022)**: 同エンジン・同じ「短編オムニバス JRPG」型、最終章での統合という別解
- **トライアングルストラテジー (2022)**: HD-2D + 物語駆動 SRPG、選択肢分岐型の物語消費
- **ロマンシング サガ -ミンストレルソング- / サガフロ**: 並列主人公構造の系譜
- **クロノトリガー / FF VI**: 群像劇 JRPG の古典、懐古層の参照点
