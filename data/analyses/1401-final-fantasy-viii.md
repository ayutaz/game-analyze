# ファイナルファンタジーVIII

## なぜ NAR 主分類なのか

プレイヤーの主動機は「スコールとリノアの恋愛劇を最後まで見届ける／時間圧縮の謎を読み解く」ことであり、戦闘や育成より物語消費が前面に立つ。concept にある通り Garden という閉じた学園世界の青春群像と、魔女戦争という壮大な縦軸を、プリレンダ背景＋シネマティック CG＋現代風衣装で「映画的に観せる」設計に全振りされている。target でも「恋愛物語を主動機にする層・特に女性層」が想定されており、戦闘の達成感より物語進行のための装置として戦闘がある構造になっている。

secondary は EXP と REW。EXP はジャンクション／ドローという独自バトル設計の戦略立て、REW は G.F.（召喚獣）のアビリティ育成・カード収集（トリプルトライアド）が副次的に効く。ただしいずれも物語視聴のテンポを補強する役割で、主動機ではない。

## 物語の駆動構造

シネマティック演出と章立て、そして「学園生活 → 任務 → 宇宙 → 時間圧縮」という空間スケールの段階拡張で物語消費を駆動する。要素を分解すると：

| 駆動装置 | 構造的な役割 |
|---|---|
| プリレンダ背景＋リアル等身 | PS1 期 JRPG の「ドット＋デフォルメ」を破棄し、現代衣装の人物像で恋愛感情への没入を担保 |
| ディスク 4 枚組の段階的舞台拡張 | 学園内紛 → SeeD 任務 → 魔女戦争 → 時間圧縮と「世界の見え方」が章ごとに反転 |
| Faye Wong『Eyes On Me』を BGM 軸に据えた挿入歌演出 | ラブシーンに「主題歌」を充てる映画文法を JRPG に導入 |
| ラグナ編との並行進行 | プレイアブル過去編で世界の前史をプレイヤーに「読ませる」二重視点 |
| 召喚獣 G.F.（記憶喪失設定） | バトル装置をストーリーの伏線回収にまで接続する仕掛け |

## バズ要因 / 人気の根拠

- 日本国内 361 万本、世界 960 万本超を売り上げ、PS1 期 FF の中でも VII に続く規模を確保。
- Faye Wong 起用の主題歌『Eyes On Me』が「ゲーム主題歌が一般歌謡として消費される」初期事例となり、シリーズの外側にも届いた。
- プリレンダ＋リアル等身という路線変更が、FF VII から流入した新規ファンに「映画を遊ぶ」体験を提供。
- ジャンクション／ドローシステムは賛否両論で、賛否そのものが長期的な議論コンテンツとして残存し、20 年以上後のリマスター（2019）まで話題を維持。

## 企画への示唆

- 主題歌に外部アーティストを起用すると、ゲーム外メディア（音楽番組／カラオケ）に作品が漏れ出し、非ゲーマー層への到達経路ができる。
- プリレンダ背景＋現代風衣装にすると、ファンタジー RPG の文脈でも「自分と地続きの恋愛劇」として読まれ、女性層・非ゲーマー層を取り込める。
- バトルシステムを意図的に偏らせる（ドロー／ジャンクション）と賛否両論になり、賛否そのものが長期の話題資産になる。
- 並行する第二主人公編（ラグナ編）を挿入すると、本筋の前史を「読ませる時間」を強制でき、世界観の厚みをプレイ時間に変換できる。
- ミニゲーム（トリプルトライアド）を本編リソース獲得経路に接続すると、物語消費に疲れたプレイヤーの離脱を REW ループで吸収できる。

## 比較で見るべきタイトル

- ファイナルファンタジーVII（1997）— 同シリーズ前作、シネマティック JRPG の原点
- ファイナルファンタジーX（2001）— 主題歌＋恋愛劇路線の次世代展開
- クロノ・クロス（1999）— 同年・並行進行＋多視点による物語駆動の比較対象
- ゼノギアス（1998）— ディスク 2 枚組による章立てとスケール拡張の対照例
