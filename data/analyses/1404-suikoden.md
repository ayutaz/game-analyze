## なぜ NAR 主分類なのか

プレイヤーの主動機は「腐敗した帝国に対する革命の群像劇を読み進める」こと。中国四大奇書『水滸伝』を翻案した政治劇という骨子が前面にあり、108 星という大量の仲間にもそれぞれ背景の物語と帝国への立ち位置が割り当てられている。戦闘や城の発展は、章ごとに進む反乱軍の物語を駆動するための装置として機能している。

ただし本作は NAR の中でも純粋な一本道物語型ではなく、副次的に REW（108 星のコレクションとそれに紐づく城の段階的成長）と EXP（多人数パーティを編成する戦闘・一騎打ち・戦争）が強く効いている。「物語を読む」動機がメインでありつつ、コンプリート報酬（真エンディング条件）と編成体験が物語の進行に組み込まれることで、JRPG として総合的な厚みを持たせる構造になっている。

## 物語の駆動構造

物語消費を駆動している層は以下に分解できる。

| 層 | 駆動要素 | 効果 |
|---|---|---|
| 主筋 | 将軍の息子が父と袂を分かち反乱軍を率いる王道展開 | プレイヤーを「裏切り」「亡命」「決起」と章ごとに能動的に追わせる |
| 群像 | 108 星のキャラクター個別エピソード | 仲間加入のたびに小説的サブシナリオが挿入される |
| 拠点 | 本拠地の城が 1→4 段階で物理的に拡張 | 物語の進捗が可視化され、達成感が章末ごとに発生 |
| 分岐 | 108 星の取得数で結末が変化（真エンディング条件） | 1 周目に「読み残し」を意図的に残し、再読・周回を促進 |
| 翻案 | 中国古典『水滸伝』を東洋ファンタジーへ移植 | 当時 FF / DQ が西洋中世風だった JRPG 市場の中で差別化を獲得 |

章単位で「城に戻る → 新メンバー紹介 → 次の戦線へ」が反復する構造のため、プレイヤーは小説の章末めくりに近い読書リズムで物語を消費していく。

## バズ要因 / 人気の根拠

JSON の popularity に示された「政治劇＋108 星＋シリーズ累計 239 万本」を分解すると、

- 中国四大奇書『水滸伝』翻案という題材自体が、FF VII 以前の JRPG 市場で東洋路線として希少だった
- 108 星という具体的なコレクション目標数が、攻略本・口コミ・後続シリーズの軸として記憶されやすい数字になった
- 本拠地の城を段階的に成長させる「物語と紐づく拠点」設計が、後の JRPG・SRPG（FE 風花雪月の修道院、ペルソナ 5 の隠れ家など）と通じる原型を提示した
- シリーズは II・III・V を経てコナミ屈指の RPG ブランドへ成長し、累計 239 万本に到達。2023 年には『百英雄伝』として元開発者主導の精神的続編が国内外でクラウドファンディングを成功させた

国内 167K というローンチ初週水準の数字は同期の FF VI / DQ VI と比較すれば小さいが、コア層と続編成長が中長期で IP を支えたタイプの作品である。

## 企画への示唆

- 「具体的に数えられるコレクション目標数」（108 星）を物語条件に組み込むと、攻略情報の流通単位とエンディング分岐の動機を同時に獲得できる
- 拠点（城・本部・学園）を物語進行に同期させて段階的に育てると、章末ごとに「達成の可視化」が無料で得られる
- 真エンディング条件をコンプリート達成に紐づけると、1 周目に意図的な未読を残せて再周回・コミュニティ攻略が立ち上がる
- 大量キャラクターは「主筋」「群像」「拠点機能」の 3 階層で役割を分けると、量を抱えても物語の焦点がぼやけない
- 既存ジャンル（西洋中世 JRPG）が飽和した市場では、文学古典の翻案で「題材レベルの差別化」を取れる

## 比較で見るべきタイトル

- 『ファイアーエムブレム 風花雪月』 — 拠点（修道院）と多人数キャラクターを章間に挟む構造の後継例
- 『オクトパストラベラー』 — 群像 JRPG として 8 人主人公制を採用した別解
- 『百英雄伝』（2024） — 元開発者主導の精神的続編、108 星設計の現代的アップデート
- 『タクティクスオウガ』 — 同時期の政治劇 JRPG、分岐エンディング設計の比較対象
- 『ロマンシング サ・ガ』シリーズ — 仲間加入の自由度と拠点（フリーシナリオ）の別解
