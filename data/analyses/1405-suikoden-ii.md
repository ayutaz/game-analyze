# 幻想水滸伝II — 分析

## なぜ NAR 主分類なのか

プレイヤーの主動機を 1 文で言語化すれば「親友が敵将となった戦争の顛末を **読み届ける**」である。concept にある「都市国家同盟 対 ハイランド王国」のデュナン統一戦争は単なる戦闘の連続ではなく、主人公・ジョウイ・ナナミという三者の関係性が破断していく群像劇として配置されており、プレイヤーは戦闘システムを操作する以上に **誰が落ち、誰が裏切り、誰が生き残るのか** を見届けるために前進する。target でも「重厚な物語と政治劇を求める JRPG ファン」が筆頭に置かれており、設計の重心が物語側にあることが明示されている。

secondary には EXP と REW が並ぶ。EXP 側は一騎打ち / 戦争 / 通常戦闘の 3 層バトル、REW 側は 108 星集めと拠点（本拠地城）の段階的発展で、いずれも強力な副動機だが、108 星の収集自体が **エンディング分岐の条件**として物語に従属している点が、本作を REW ではなく NAR に位置付ける決定打になる。

## 物語の駆動構造

本作の物語消費は単線の章立てではなく、複数の進行軸が同時並行で巻き取られる構造で駆動する。主要レイヤーを分解する。

| レイヤー | 機能 | プレイヤー側の動機 |
|---|---|---|
| デュナン統一戦争（メイン） | 都市同盟が国家へと成長する政治史 | 「次の戦線で誰が動くか」を読む |
| 主人公／ジョウイ／ナナミ三角関係 | 親友が敵将となる関係破断のメロドラマ | 三者の再会場面を見届ける |
| 108 星集結 | 仲間勧誘＝サブクエスト群 | 全員集めて True End 条件を満たす |
| 本拠地城の段階発展 | 仲間が増えるたび拠点に施設・住人が増殖する可視化装置 | 自分の選択が世界を太らせている感覚 |
| 戦争パート（数十人 vs 数十人の象徴戦） | 政治状況をシステム面で再演 | ストーリー進行のテンポ管理 |

特筆すべきは **「108 星集結数」と「ナナミの生死／ジョウイとの最終局面」がエンディング分岐に直結する** 設計で、収集と物語が同じ KPI で接続されている点である。プレイヤーは「物語を最後まで読む」ために収集を強いられ、収集をやり切ったプレイヤーだけが True End の対話シーンを得る。secondary の REW 要素が物語駆動装置に組み込まれている典型例である。

## バズ要因 / 人気の根拠

popularity 文「国産 RPG 史に残る重厚な戦争群像劇として高く評価され、PS1 北米版の希少さからプレミア化、2025 年の HD リマスターで再注目を集めた」を分解すると以下のとおり。

- **戦争群像劇としての評価**：FF / DQ がパーティ単位の冒険譚を磨いていた 1998 年に、本作は国家・軍隊・政治を主題に据えた群像劇で差別化し、ストーリー RPG 史の参照点として語り継がれた。
- **PS1 北米版のプレミア化**：北米初回出荷が極端に少なく長らく中古市場で高額取引され、「遊びたくても遊べない名作」というラベルが結果的にブランド価値を維持した。
- **2025-03-06 の HD リマスター（I & II 同梱）**：PS4 / PS5 / Xbox One / Xbox Series X|S / Switch / PC のマルチで再販され、当時遊べなかった層と Z 世代の新規層に同時にリーチした（出典: Gematsu / Konami 公式）。
- **108 星設計の継承力**：以降の JRPG・ソシャゲの「仲間集めて拠点を育てる」構造の原型として参照され続け、語られるたびに本作の名前が再浮上する構造的バズ要因になっている。

## 企画への示唆

- **収集と物語の KPI を同一線上に置くと、収集が物語消費装置になる**：108 星の収集数が True End の解放条件になっているため、コレクションが「やり込み」ではなく「物語の続きを読む権利」に変換されている。
- **拠点を可視化装置として育成させると、プレイヤーの選択がワールドに刻まれる手応えが出る**：本拠地城に新住人が増えるたびに BGM・施設・会話が増殖する設計は、リテンションを「自分の世界が太っていく感覚」で確保する。
- **親友を敵将に置くと、戦闘のたびに物語が更新される**：戦争 RPG の各章クライマックスにジョウイを配置する構造は、メカニクスを変えずに物語的緊張だけを上昇させる安価な手段になる。
- **エンディング分岐を「行動の総和」で決めると、二周目の動機が自動生成される**：星集結数・ナナミ生存・特定イベントの達成度を組み合わせた分岐は、攻略情報の流通そのものをマーケティング資産化する。
- **長期空白後のリマスターは『遊べなかった層 × 新規層』の二重需要を取りに行ける**：北米版希少化で生まれた「伝説化」を、マルチプラットフォーム HD 化で一気に解凍する 2025 年型のリリース戦略。

## 比較で見るべきタイトル

- **幻想水滸伝 I**（1995 / Konami）— 親作品。リマスター同梱で対比される I の章立て構造と本作の差分を見ると、群像劇への進化が分かる。
- **タクティクスオウガ**（1995 / Quest）— 戦争＋分岐エンディング＋仲間収集の前例。物語駆動の戦記 RPG の参照系譜。
- **ファイアーエムブレム 聖戦の系譜**（1996 / 任天堂）— 親子二世代の戦記＋仲間死亡で物語が更新される構造。NAR×SRPG の同時代比較。
- **百英雄伝**（2024 / Rabbit & Bear）— 元 Konami 開発陣による精神的後継作。本作の 108 星設計が四半世紀後にどう翻案されたかの比較対象。
- **オクトパストラベラー II**（2023 / Square Enix）— 複数主人公視点で群像劇を成立させる現代版の解。本作が 1998 年に試した形式の現代回答。
