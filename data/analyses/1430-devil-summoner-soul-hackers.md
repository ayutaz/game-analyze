# 真・女神転生 デビルサマナー ソウルハッカーズ — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「サイバー都市の3Dダンジョンを潜って悪魔と交渉し、自分だけのパーティを編成しながら踏破する」体験そのものだ。ハッキング・悪魔会話・合体・属性相性という4つの相互作用ループを、1人称視点のダンジョン探索のなかで回し続ける構造が中核にあり、social_axis はオフラインの「ソロ」体験に閉じている。

concept にある通り本作は『真・女神転生』の交渉・合体システムを継承する派生作で、舞台をサイバーパンク都市『アマミ市』に移し替えたことで、悪魔召喚＝ネットワーク経由のプログラム呼び出しという独自フィクションが体験に乗る。target が示すメガテン／ペルソナ系コア層は、シナリオ消費以上に「会話で悪魔を口説き、合体で系譜を組む」遊びの反復に時間を投じる層であり、ここが EXP と判定する根拠になる。

secondary には REW（悪魔収集・合体による獲得圧）と NAR（サイバーパンク陰謀劇）が乗る。収集と物語は強力な副推進力だが、本作の場合は「ダンジョン探索→交渉→合体→再潜行」のループの中に組み込まれており、独立した報酬軸・物語軸として剥がれない。

## 体験の核 / 設計の発明

中核は「3つの不確実性」を同一ダンジョンに重ねた点にある。エンカウント先で何が出るか、会話で仲魔にできるか、合体で何が生まれるか——いずれもプレイヤーの即興判断を要求する。

| 設計レイヤ | 内容 | プレイヤー側の判断 |
|---|---|---|
| 探索 | 1人称視点3Dダンジョン、自動マッピング、トラップ | どこまで深追いするか |
| 戦闘 | 属性弱点・状態異常・銃と剣の併用 | 弱点を突くか防御を固めるか |
| 交渉 | 悪魔の性格に応じた4択会話 | 仲間にするかアイテムを取るか |
| 合体 | 種族×ランクで派生、スキル継承 | 系譜を組み直すか温存するか |
| ハッキング | 街・ダンジョンの一部ギミックを解除 | 物語・探索の進度を制御 |

前世代の旧『デビルサマナー』（サターン版1995）からの差分は、(a) 舞台を伝奇から近未来サイバーへ振り、(b) サブナビ NEMISSA とのバディ要素を増やし、(c) ハッキングという行為動詞をフィクションと UI 両方に乗せた点。social_axis がソロでも、悪魔という NPC との対話量が「他者」要素を擬似的に補完している。

## バズ要因 / 人気の根拠

- 1997年サターン版26万本＋PS版14万本＋3DS版10万本で**累計約50万本**と、当時のメガテン派生作としては堅実な数字を残した
- サイバーパンク都市『アマミ市』と攻殻機動隊的世界観が、90年代後半のサターン後期ユーザーに刺さりカルト的支持を形成
- 2012年3DSリマスターで**初の海外（北米・欧州）展開と全面ボイス化**が実現し、英語圏のメガテンファンに発見された
- 2022年『ソウルハッカーズ2』発売前後で原作の再評価が進み、旧作プレイヤーと2からの逆引き層が合流した

## 企画への示唆

- **舞台を時代設定ごとリブランドすると、同じシステムが別 IP に見える**：『真・女神転生』の交渉・合体エンジンをサイバー都市に置き換えるだけで、target 層が一段広がった
- **行為動詞をフィクションと UI 両面に乗せると世界観が締まる**：ハッキングを「物語上の職業」と「画面上の操作」の両方に実装することで、ゲーム外説明なしに没入が成立する
- **ソロ EXP でも NPC（悪魔）対話量で他者軸を擬似補完できる**：交渉の偶発性と性格テーブルが、人間プレイヤー不在でも社会的な手応えを生む
- **長期 IP は派生作の続編タイミングで原作を再露出させると拾い直しが効く**：2022年『2』前後の原作再評価が良い例
- **ハードを跨いだリマスターは「未訳市場の開拓」と「ボイス追加」を組むと再販価値が立つ**：3DS版がそれを実証した

## 比較で見るべきタイトル

- 真・女神転生 デビルサマナー（1995、サターン）— 直接の前作、伝奇→サイバーの差分が観察できる
- ペルソナ2 罪／罰（1999/2000）— 同時期 Atlus、悪魔交渉を継承しつつ NAR 寄りに振った姉妹系譜
- 世界樹の迷宮シリーズ（2007–）— Atlus による1人称3Dダンジョン探索の現代版、EXP×ソロの正統後継
- Wizardry シリーズ — 1人称ダンジョン RPG の祖、本作の探索設計の参照源
- ソウルハッカーズ2（2022）— 25年後のリブート、原作の再評価を促した実例
