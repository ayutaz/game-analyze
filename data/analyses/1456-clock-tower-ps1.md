# クロックタワー2 (PS1, 1996) 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「シザーマンから逃げる・隠れる・追跡を振り切る」というリアルタイムの緊張処理であり、戦闘も育成もない代わりに **追跡者から逃げ切る運動感覚そのものが報酬** になっている。social_axis は「ソロ」だが、孤独であることが恐怖の構造的前提として機能している（ヘルプを呼べない密室性）。

concept が示す通り、本作はポイント&クリックという静的 UI の上に「追跡されている時間」というリアルタイム軸を重ねた設計で、戦って勝つ達成ではなく**追跡からの離脱**という体験そのものを目的化している。target の「シネマティックなアドベンチャーを好む層」と「逃げるホラー」の組み合わせから、物語的没入（NAR）は secondary に位置し、マルチエンディングは追体験よりも**プレイ毎の挙動を変える可変要素**として機能する。

## 体験の核 / 設計の発明

「戦って排除する」バイオハザード型サバイバルホラーの隣に、「戦闘オプションを最初から削除する」という選択をした点が設計的な発明である。

| 設計要素 | 仕組み | プレイヤー側に生まれる効果 |
|---|---|---|
| ポイント&クリック UI | 移動・調査をクリックで指示する間接操作 | 操作レイテンシが本来の操作感を遅らせ「逃げ遅れる」恐怖を構造化 |
| シザーマンの突発出現 | スクリプト + ランダム要素で予期せぬ場面で襲撃 | 安全地帯がない緊張持続、定石化を阻む |
| パニックメーター（ジェニファーの精神状態） | プレイヤーキャラの状態が UI に表示され、悪化すると操作が破綻 | キャラ側の感情がメカニクス化、感情移入の機能化 |
| 隠れる / 振り切る成功判定 | 物陰・ベッド下・カーテン裏など環境物の利用 | マップ知識＝生存資源、観察行動への動機付け |
| マルチエンディング（複数分岐） | 訪問順・選択・生存者数で結末が変動 | 周回プレイのインセンティブ、可変性の演出 |

前作 SFC 版から継承した「逃走ホラー」の系譜は、後年の Haunting Ground、SIREN、Outlast、Amnesia 等「戦えないホラー」のテンプレートを形成する起点になっている。

## バズ要因 / 人気の根拠

- 国内 0.35M / 全世界 0.5M+ という、当時の PS1 アドベンチャー作品としては堅実な売上で、開発元ヒューマンが社員に 10 万円の特別ボーナスを支給するほどの成功と報じられた
- バイオハザード（1996 年 3 月）と同年・同プラットフォームで「戦わないホラー」というアンチテーゼを提示し、ジャンル内に明確な対抗軸を作った
- シザーマンというビジュアル明快な記号化された追跡者を中心に据え、ホラー雑誌・口コミでのキャラクター訴求が機能した
- ポイント&クリックという PC アドベンチャー文脈の操作系を家庭用に持ち込み、シネマティックな演出と相性の良い UI として再評価された

## 企画への示唆

- **戦闘オプションを削除すると、プレイヤーは観察と移動に動機を再配分する**：選択肢を減らすことで残った行動の意味が増す
- **追跡者を可視化された記号にすると、IP の波及力が出る**：シザーマン＝ハサミという 1 アイコンの強さは Slender、Mr.X、レメンテッド等の系譜を生む
- **キャラの精神状態をメカニクス化すると、感情移入が機能になる**：パニック演出を UI に組み込むと「感情移入＝攻略要素」に転化できる
- **間接操作 UI とリアルタイム脅威を混ぜると、操作遅延がそのまま恐怖になる**：UI の制約を弱点でなく演出装置として転用できる
- **マルチエンディングを「物語消費」でなく「挙動の可変性」として設計すると、EXP 中心作品でも周回動機が立つ**：分岐は物語専用の道具ではない

## 比較で見るべきタイトル

- バイオハザード (1996, PS1) — 同年・同プラットフォームの「戦うサバイバルホラー」、対極の設計
- SIREN (2003, PS2) — 「視界ジャック」で逃走ホラーを集団追跡者に拡張した系譜
- Haunting Ground (2005, PS2) — カプコンによるクロックタワー直系の「ヒロイン逃走ホラー」
- Outlast (2013, PC/PS4) — 一人称視点で「戦えないホラー」を再定義した現代的後継
- Amnesia: The Dark Descent (2010, PC) — 精神状態メカニクスを継承した洋ホラーの代表
