# Echo Night（エコーナイト）

## なぜ NAR 主分類なのか

プレイヤーの主動機は「漂う幽霊一人ひとりの未練を聞き取り、その物語を読み解く」ことにある。豪華客船オルフェウス号という閉鎖空間を舞台に、戦闘ではなく「光のスイッチを点ける」「彼らの願いを叶える」という非暴力的な所作を介して、断片化された短編幽霊譚を繋ぎ合わせていく。プレイヤーが残すのはハイスコアでもクリア時間でもなく、各幽霊に対して与えた成仏という結末そのものであり、これは NAR の中核条件（=物語消費が目的）に合致する。

secondary に EXP が入っているのは、King's Field 由来の一人称 3D 探索エンジンが「歩いて部屋を発見する」体験を成立させているためで、探索行為そのものはあくまで物語供給の経路として機能する。social_axis は null、完全ソロの内面的体験である。

## 物語の駆動構造

| 構造要素 | 仕掛け | 物語消費への効果 |
|---|---|---|
| 舞台 | 消息不明の豪華客船という閉鎖系 | 章ではなく「部屋」が物語の単位になり、探索順≒読書順となる |
| 視点 | 一人称、自分の手しか映らない | プレイヤー＝主人公の境界を曖昧にし、幽霊との一対一の聞き取りを成立させる |
| 解決動詞 | 「灯を点ける」「願いを叶える」 | 戦闘を排除し、対話と気づきだけが進行手段になる |
| エンディング | 救えた幽霊の数で分岐 | 全員救済が義務ではなく、見落とした未練がそのまま記憶に残る |

各幽霊エピソードは独立した短編構造を持ち、メインプロット（船の沈没原因）はそれらの集積として浮かび上がる。読書体験で言えば連作短編集に近い駆動方式である。

## バズ要因 / 人気の根拠

- FromSoftware の代表作群のうち、King's Field の一人称 3D 技術を「ホラー ADV」というジャンルに転用した数少ない試みとして、ゲーム史的な系譜整理の文脈で繰り返し言及される
- 戦闘・ゲームオーバー要素を極力排した「非暴力ホラー」という設計が、後年のウォーキングシム系の文脈で再評価された
- 宮崎英高が PSVR タイトル『Déraciné』の直接的インスピレーションとして本作を公言したことで、Bloodborne / Dark Souls 経由のフロムファンが遡って参照する作品となった
- 売上は国内外ともに公表値が乏しく（sales_jp / sales_world ともに未公開）、商業ヒットというより「カルト的代表作」「同窓的に語られる作品」として位置づけられている

## 企画への示唆

- 戦闘を捨てて「灯を点ける」程度の最小動詞だけ残すと、物語密度に対するノイズが激減し、短編集型 NAR が成立する
- 主人公の手しか映らない一人称視点は、語り手＝プレイヤーの距離をゼロにし、幽霊側の独白を主役にできる
- 一作品の中に独立短編を 10〜20 本散らし、全救済を任意目標にすると、未達 = 後悔という余韻が自然に発生する
- 既存の探索エンジン（自社の King's Field）を別ジャンルへ転用すると、技術的安全圏のまま「世界観だけ全く違う」派生作を低コストで作れる
- カルト作品はリリース時の売上より、20 年後に著名クリエイターが「自分の原点」と公言した瞬間に再点火する。長命化させたいなら、後継者にとって参照しやすい設計言語（非暴力 / 一人称 / 短編連作）で書いておく価値がある

## 比較で見るべきタイトル

- King's Field シリーズ（同じ一人称 3D エンジン、戦闘あり EXP 側）
- Déraciné（宮崎英高が本作を参照した PSVR 作品、NAR・非暴力）
- 街 〜運命の交差点〜 / かまいたちの夜（90 年代 ADV の短編連作型ナラティブ）
- Gone Home / Dear Esther（後年のウォーキングシム、非暴力 NAR の系譜）
