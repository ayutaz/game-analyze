# ステラーブレイド (Stellar Blade)

## なぜ EXP 主分類なのか

このゲームをプレイヤーが楽しんでいる動詞は「斬る／回避する／反撃する」である。崩壊後の地球を舞台にしたポストアポカリプス SF という物語フックは確かにあるが、毎ゲームセッションの主要な動機は「敵の攻撃パターンを読み、パリィとドッジで隙を作り、コンボに繋ぐ」という近接戦闘の体験そのものに集約される。concept にある通り「ソウルライク要素を含むスタイリッシュアクション」であり、target も「ニーアやベヨネッタ系のスタイリッシュ系を好む層」と明示されている。social_axis は「ソロ」で、他者との競合や協力は介在せず、敵 AI と自分の指先だけが噛み合うクローズドな体験になっている。

secondary に NAR を置いているのは、ストーリーミッションと崩壊後の地球を巡るロード式ハブ探索が「斬るだけのゲーム」を相対化し、報酬としての次章解放を駆動しているため。ただし物語の追体験は副であり、戦闘の手応えがなければ NAR 単独では成立しない構造になっている。

## 体験の核 / 設計の発明

中核メカニクスは「パリィ／パーフェクトドッジで主導権を奪う近接戦闘」と「ベータスキル・バーストスキルによるリソース管理」の二層構造である。前者は SEKIRO 系の読み合い、後者はベヨネッタ／ニーア系のスタイル系アクションを参照しており、ハイブリッド化が Shift Up の独自性になっている。

| 層 | 要素 | 前世代タイトルとの対比 |
|---|---|---|
| 基礎攻防 | パリィ・ドッジ・反撃 | SEKIRO（弾き）／ベヨネッタ（ウィッチタイム）の合成 |
| スキル層 | ベータ・バーストゲージ消費技 | NieR:Automata の Pod / チップに近いリソース技 |
| 探索層 | リニア戦闘ステージ + 半オープンのハブ「エイドス7」 | FF7 リメイク／ニーアのハブ＆ミッション式 |
| 装備層 | ナノスーツ・装備品・ジア（強化スロット） | スタイリッシュ系では珍しい「コスチューム＝コアコンテンツ」化 |

social_axis ソロの効果として、難易度をパリィ精度に振り切ることができ、「マルチで荒れる」要素を排して職人芸的な反復練習が成立している。半オープンの「エイドス7」が単調な戦闘の合間を稼ぐクッションとなり、撃破ボリュームの飽和を遅延させている。

## バズ要因 / 人気の根拠

JSON の popularity「主人公イヴの造形と PS5 独占で話題化、SNS 拡散」を分解すると、

- 主人公イヴのキャラクターデザインがプレイ動画・写真モードと相性が良く、X / YouTube Shorts でビジュアル単体で拡散しやすい設計になっていた
- PS5 独占（後に PC 展開）で発売初期は SIE のファーストパーティ的扱いを受け、PlayStation Showcase での露出を獲得した
- 体験版（デモ）が発売前に配信され、パリィの手触りを SNS で語る層が事前バズを形成した
- 世界販売 2M+ という規模は、韓国スタジオの単独 IP かつ非マルチ・非サービス型アクションとしては大きく、SIE のマーケティング配球と合致した

## 企画への示唆

- ソウルライクの「読み合い」とスタイリッシュ系の「魅せ技」を二層構造で同居させると、両ファン層を同時に取り込める
- 主人公のビジュアル設計を「写真モードで切り出して SNS に貼られること」を前提に作ると、プレイヤー自身が広告チャネルになる
- ファーストパーティ独占＋後発 PC 展開のウィンドウ戦略は、ローンチ時の話題性とロングテール売上を両立させる
- リニア戦闘ステージの合間に半オープンのハブを差し込むと、戦闘体験の単調化を遅延でき、NAR が副で効くようになる
- 発売前デモで「触感の良さ」を共有させる施策は、コア層の事前評価を発売後レビューより前に確立できる
