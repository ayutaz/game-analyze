# サイレント・ボンバー — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機を 1 文で言えば「設置→起爆→連鎖の最適解を、限られた持ち弾数の中で組み立て続ける」ことにある。撃つではなく**貼る**という入力に置換されたアクションが、本作の体験を一意に規定している。

concept にある通り、E-Unit ガントレットで爆弾を敵や床に貼り付け、任意のタイミングで一斉起爆する。敵を倒すことが目的ではなく、爆弾の**配置と起爆順序**そのものがミニパズル化されており、目的は戦艦 DANTE 内の 14 ミッション攻略という空間制圧。target が「コンボの最適化を楽しめるコア層」と規定されている通り、social_axis はソロで完結する。

secondary に NAR が入るのは、遺伝子操作された主人公 Jutah Fate の重い背景や、戦艦 DANTE 内部の SF 設定が体験を彩るためだが、物語は爆破ループを駆動する補助であって主動機ではない。

## 体験の核 / 設計の発明

中核は「撃つ → 当たる → 倒す」を「貼る → 溜める → 一気に起爆する」に置換した点。前世代の Run & Gun（メタルギア風潜入や 2D シューター）が連続入力で敵を削るのに対し、本作はリソース（爆弾数）と空間（貼り付け先）を**同時管理**させる。

| 要素 | 設計 | プレイヤーへの効果 |
|---|---|---|
| 入力 | 貼り付け + 起爆の 2 段階 | 撃つアクションが「準備 → 実行」に分割される |
| リソース | 同時保持できる爆弾数の上限 | 数を超えれば古い爆弾が消える、配置の選択が強制される |
| コンボ | 複数敵に貼って一斉起爆 | 単発を捨て、連鎖で稼ぐリスク選択が成立 |
| 空間 | トップビュー 3D・閉鎖空間 | 逃走線と起爆距離が常に意識される |
| 物語 | 14 ミッション + Jutah の出自 | 反復するループに段階的に文脈が乗る |

social_axis ソロの効果として、爆破コンボはランキングや観戦ではなく**自分のミッション内タイム/スコア**で完結する。他者比較ではなく、自プレイの作業最適化に没入させる構造。

## バズ要因 / 人気の根拠

popularity の「PS1 末期のカルト的人気作、後年アーカイブスで再評価」を出発点に分解する。

- 1999 年リリース当時、世界売上は約 7 万本にとどまり**商業的にはニッチ**。広い普及ではなく**限定的だが濃いコア層**を獲得した
- CyberConnect（後の CyberConnect2）の初期作で、独自メカニクス志向のスタジオ性が出ている点が後に評価材料となった
- 2006 年から PlayStation Network のゲームアーカイブスで DL 販売されたことで、原版を所有しなかった層への流通経路が再開
- 「銃ではなく爆弾」というワンアイデアの徹底度が SNS / レトロゲームレビュー文化で語り継がれる素材として機能した

## 企画への示唆

- **主動詞を 1 つ置換すると、ジャンルが再生成される** → 「撃つ」を「貼る」に置換するだけで、リソース管理とパズル性が後から自動的に乗ってくる
- **入力を 2 段階に分割すると、準備フェーズに戦略が生まれる** → 単発入力では消費されない「迷い」「待ち」がゲームプレイ要素になる
- **同時保持上限を低めに設定すると、配置選択が強制される** → 無限リソースは選択を消す、上限がプレイヤーに思考を要求する
- **コンボ報酬を空間構造に紐付ける** → 敵の配置とプレイヤーの逃走線が同じマップ上で交差するため、「どこに貼るか」が「どこへ逃げるか」と直結する
- **商業的にニッチでも、メカニクスの一貫性は長期的資産になる** → 7 万本という数字でも、アーカイブス再販と語り継ぎで 20 年以上カタログ価値を保つ

## 比較で見るべきタイトル

- **ボンバーマン**シリーズ — 「設置型爆弾」の系譜、ただし対戦軸でソロ最適化ではない
- **メタルギアソリッド**（PS1 末期） — 同時代の硬派 SF・潜入アクション、撃つを避ける設計
- **デビル メイ クライ** 1（2001） — 同社系譜的に近い「コンボ最適化アクション」の到達点
- **Bangai-O**（トレジャー） — トップビュー × 一斉発射の体験が近い、爆発演出の方向性
- **CyberConnect2 後期 .hack 系** — 同スタジオが「独自システム＋濃い SF/ファンタジー設定」を量産していく原型
