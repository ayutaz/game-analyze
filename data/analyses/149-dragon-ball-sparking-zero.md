# DRAGON BALL: Sparking! ZERO

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**原作の戦闘場面を自分の手で再演し、対戦相手とぶつける**」こと。物語を読むためではなく、空中戦・気弾・瞬間移動・界王拳といった原作固有の挙動を 3D 空間で動かす運動それ自体が中核に据えられている。

`concept` に明記されたとおり本作は『スパーキング!』シリーズ 17 年ぶりの新作で、182 体超のキャラ全員に固有モーションと変身段階が付与される。`target` の主層が「DB アニメ／原作世代の 30〜40 代男性」で海外比率 9 割という事実は、**物語そのものは既知**であり、プレイヤーは「悟空 vs フリーザを自分で再現すること」「フレンドと悟飯対セルを取り合うこと」を買っていることを示している。これは `social_axis = 対戦` と合致しており、報酬収集や物語消費ではなく **キャラ操作の体験** が一次動機であることを裏付ける。

secondary に NAR が入るのは「エピソードバトル」モード経由で原作章を追体験できる構造があるため。ただし本数の中核は対戦と自由バトルで、物語モードは「キャラを使う口実」として副次配置されている。

## 体験の核 / 設計の発明

スパーキング! 系の特異性は、原作再現の解像度と対戦ゲームとしての成立可否の **緊張関係** にある。本作の設計を分解する。

| レイヤー | 機能 | 体験への寄与 |
|---|---|---|
| キャラ数 | 182 体超（変身段階別カウント） | 「推しが必ずいる」状態を作りロスター閲覧自体が娯楽になる |
| 操作系 | 3D 全方位空中戦 + 高速移動 | 平面格ゲーでは表現できない「飛びながら打ち合う」原作絵作りの再現 |
| 必殺技演出 | カメラ寄り＋カットイン | 当てた／受けた瞬間を視覚的に「原作の絵」に変換する |
| エピソードバトル | 原作主要章の再現＋ IF 分岐 | 既知ストーリーを操作対象として消費する secondary NAR の受け皿 |
| カスタムバトル | ステージ・条件・台詞配置を編集 | UGC 動画化が前提のシェア導線、SNS バズの素材源 |

設計上の発明は「**対戦ゲームとしての公平性を一部捨てて、キャラ性能をキャラ性能のまま実装する**」点にある。悟空・ベジータ・フリーザ・ブロリーの強弱が原作通りに不均衡で、トーナメント格ゲー文法では破綻するが、`social_axis = 対戦` の対象が **競技プレイヤーではなく原作ファン同士** であるため、不均衡が「悟空でフリーザを倒すと熱い」という再現価値に転換される。

## バズ要因 / 人気の根拠

popularity 文「原作再現と全世代ファンの根強い需要、欧米先導のヒット」を事実で分解する。

- **発売 24 時間で 300 万本、累計 540 万本超**（`sales_world` 5.4M+）でドラゴンボールゲーム史上最速ペースを記録。バンナム IP モノとして近年最大級の初動。
- **海外比率約 9 割**（`target` 記載）。国内初週 PS5 約 6.4 万本に対し世界 540 万本という比率は、北米・南米・欧州の DB ファン層が主購買であることを示す。
- **17 年ぶりのナンバリング**という「待たされた期間」自体が話題化のフックになり、トレイラー公開ごとに参戦キャラ判明が SNS の継続話題を生む構造を作った。
- **配信・切り抜き親和性が高い**：182 キャラのドリームマッチ、カスタムバトル機能による「もしもの戦い」動画が YouTube／X で大量生産され、購入前ユーザーへの認知導線になった。

## 企画への示唆

- **休眠 IP の続編は「待たせた長さ」を資産化できる**：17 年ぶりという情報単独でトレイラーが拡散され、広告費の代替になる。
- **競技性より再現性を優先すると、対戦ゲームでも IP ファン市場が拓ける**：性能均衡を諦める意思決定は、対戦ジャンルでも事業判断としてあり得る。
- **キャラ数を「網羅」のレベルまで積むと、ロスター画面そのものが商品になる**：182 体は使い切られない前提で、「自分の推しが居る」確認体験を売っている。
- **エピソードバトルで原作章を「操作対象」に変換すると、NAR を secondary に格納できる**：ストーリーモードが対戦ゲーの動機を弱めないための分離設計。
- **カスタムバトルなど UGC 装置を初期搭載すると、配信切り抜き経由の二次認知が回る**：販売後 1 ヶ月のロングテールを動画文化で延ばす設計。
- **初動を海外に寄せる IP は、国内売上を成功指標から外して判断する**：日本初週 6.4 万本を「弱い」と読むと意思決定を誤る。

## 比較で見るべきタイトル

- **ドラゴンボール ゼノバース 2**（2016, Dimps）：同 IP の対戦アクションだが、アバター制＋オンライン Hub で「自分が修行する」体験寄り。Sparking と動機が対照的。
- **ジョジョの奇妙な冒険 オールスターバトル R**（2022, CC2）：同じく原作再現特化の対戦アクション、IP ファン市場向けの不均衡設計が近い。
- **ナルティメットストーム 4**（2016, CC2）：原作再現＋カットイン演出＋海外比率高めという構造が最も近い先行例。
- **ドラゴンボールZ Sparking! METEOR**（2007, スパイク）：直系の前作。17 年の空白期間の参照点。
- **鉄拳 8** / **ストリートファイター 6**（2023〜）：競技性側に振った対戦アクションの対照群として、設計思想の差分が見える。
