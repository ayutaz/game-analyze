# アインハンダー (1997)

## なぜ EXP 主分類なのか

このゲームでプレイヤーが楽しんでいる動詞は「奪う」「撃つ」「捌く」である。物語や収集ではなく、敵が落とすガンポッドを瞬時に判断して掴み取り、限られた弾数を使い切るまでに次の武装をどう確保するかという反射と取捨選択の連続が中核に置かれる。social_axis は「ソロ」で、対戦も協力も介在しない。スクウェアの作品としては珍しく、ストーリーはミッション間ブリーフィングと簡素な無線で背景を示すだけに留まり、プレイヤー自身の腕前と判断が主目的となる。

target で言及されているコナミ系 STG ファンが歓迎したのも、この体験設計が「物語を読む RPG 客寄せ」ではなく純粋にシューティング体験を組み立て直しているからである。secondary が空であることも、副次的な動機（ガチャ・育成・物語消費）に頼らない単機能エンタメであることを裏付ける。

## 体験の核 / 設計の発明

アインハンダーの設計上の発明は、ショット火力の管理をプレイヤーに委ねた点にある。自機は弱い基本ショットしか持たず、火力は完全に「敵から奪ったガンポッド」に依存する。武器ごとに弾数・連射・誘導・射角が異なり、敵編隊との相性で武装を持ち替えながら進む。

| 設計要素 | 前世代 STG（グラディウス / R-TYPE 等） | アインハンダー |
|---|---|---|
| 火力ソース | パワーアップアイテム拾得 | 敵機の武装を奪取 |
| 弾数 | 無制限 | 武器ごとに有限 |
| アーム | 固定オプション | マニピュレーターで前後付け替え |
| 描画 | 2D ドット | 3D ポリゴン 60fps |
| 演出カメラ | 横固定 | ボス戦で視点回り込み |

「奪う」という行為が「敵を倒すと火力が補給される」というポジティブフィードバックを生む一方、弾数有限が「弾を撃ち切ると最弱状態に戻る」というリスクを常時提示する。これが従来の「死ぬまでパワーアップが累積する」STG とは異なる、攻撃中も常に意思決定を強いるテンポを作っている。3D 描画によりボス戦で艦体が回転して別形態を露わにするような演出が成立し、横スクロールの平面性を視覚的にも更新した。

## バズ要因 / 人気の根拠

- popularity 文にある通り、RPG 専業と認識されていたスクウェアが STG を作るというギャップ自体が当時の話題を生んだ
- 「STG 氷河期」と呼ばれた 1997 年前後にアーケード移植や互換続編ではなく完全新規 IP を投じた稀有な事例で、ジャンル史的に取り上げられやすい
- 60fps の 3D 描画とオーケストラ調 BGM の組み合わせが、PS1 期の表現力ベンチマークとして繰り返し言及される
- 商業的なミリオン到達はしていない一方、PS1 屈指の STG として後年のレトロ再評価や HD リマスター待望論で名前が挙がり続けるカルト的延命構造を持つ

## 企画への示唆

- 既存ジャンルに参入するなら「拾う」を「奪う」に書き換えるなど、行為動詞を 1 個ひっくり返すと前世代との差分が一目で立つ
- 火力を有限化すると、攻撃局面そのものが資源管理ゲームになり、撃つ時間にも意思決定が宿る
- 描画技術（3D / 60fps）を「見せる」だけでなく、ボス変形やカメラ回り込みなど体験設計の自由度拡張に結びつけると、技術アピールが遊びに還元される
- 親会社の主力ジャンル（ここでは RPG）と異なる作品をリリースする際は、想定外客層（コア STG ファン）を獲りに行く硬派設計を貫いた方がコア層の信頼を得やすい
- 短期商業より長期カルト人気を狙うなら、独自メカニクス + 強烈なビジュアルアイデンティティ + 続編を作らない希少性、の三点セットが効く

## 比較で見るべきタイトル

- グラディウス / R-TYPE 系（横スクロール STG の前世代文法）
- レイディアントシルバーガン (1998, トレジャー)（同時期の 3D 描画 STG、武器切替が中核）
- Xexex (コナミ)（敵の武器を利用する系譜の参照点）
- グラディウス V (2004, トレジャー / コナミ)（後年の 3D 描画横 STG が到達した完成形）
- バンガイオー / シルフィード (メガ CD)（3D 描画 STG の系譜上の親戚）
