# ファイティング・フォース (Fighting Force, 1997)

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「2 人でストリートを進みながら、ザコをまとめて殴り倒す手応えを味わう」** こと。物語進行や収集要素は薄く、ステージ毎にギャングを掃討して次のセクションへ進む反復行為そのものが目的になっている。よって体験型（EXP）主分類が妥当。

`concept` にある「ベア・ナックル 3D 後継」「武器・オブジェクト破壊・2 人同時プレイ」、`target` にある「90 年代後半に 2D ベルトスクロール黄金期を遊んでいた層」という記述から、本作のコアは **アーケード／コンシューマで磨かれた "進む → 殴る → 武器を拾う → ボス → 進む" ループを 3D 空間で再構築すること** にある。social_axis が「協力」なのは、敵を二方向から挟む・ダウンを取って起き攻めを分担する・武器を受け渡す等、2 人プレイ前提の作戦が動線設計に織り込まれているため。`secondary` は空で、報酬蓄積・物語追体験の比重が薄いことを裏付けている。

## 体験の核 / 設計の発明

ベルトスクロールの 2D 公式を 3D に翻訳した過渡期作品として、設計要素を分解する。

| 設計要素 | 2D ベルトスクロール（参照: ベア・ナックル / ファイナルファイト） | Fighting Force の 3D 翻案 |
|---|---|---|
| 移動軸 | 横スクロール + 奥行きの 2.5 軸 | 3D ポリゴン空間で全方位移動 |
| 攻撃方向 | 自動ターゲット（左右のみ） | 周囲の敵に対する近接判定、囲まれ前提 |
| 武器 | 落ちている棒・刃物を拾って使い切る | 鉄パイプ・銃に加え、椅子・机・バイク等のオブジェクト破壊 |
| 進行構造 | ステージクリア型、固定ルート | 一部分岐ルート、4 キャラごとに展開差 |
| 協力プレイ | 2P 同画面、敵を分担 | 2P 同画面継続、3D 空間で挟み撃ち |

発明と呼べる差分は (a) **オブジェクト破壊を空間演出として組み込んだこと**、(b) **キャラクター 4 人 × 分岐ルートで反復プレイ動機を作ったこと**、(c) **2P 協力を 3D で維持した数少ない事例**であること。一方で 3D 化の代償としてカメラと当たり判定が不安定になり、2D 時代の「テンポよく敵を捌く」感覚は減退している。協力プレイは複数の敵に対する役割分担（ヘイト分割・武器の受け渡し・ダウン起こし）を発生させる素地を持つが、AI 挙動が単純なため戦術深度には限界がある。

## バズ要因 / 人気の根拠

- `popularity` 記載の通り、本作は当初 Core Design が **『ベア・ナックル』4 作目としてセガに提案 → 断られて独立タイトルに転用**された経緯がある。この「幻の Streets of Rage 4」という出自そのものがメディアの語り口を作った。
- 世界累計 **100 万本超** を記録。日本での売上は公開数字なし（"—"）で、欧米市場が主戦場だったことを示す。
- 1997 年は PS1 / N64 初期で 3D ベルトスクロールという企画自体が稀少で、**「2D の文脈を 3D に持ち込んだ過渡期サンプル」**として後年語られる存在になった。
- 開発元 Core Design は同年『Tomb Raider』を世に出したスタジオであり、その知名度がパブリッシング側の Eidos と相まって露出量を底上げした。

## 企画への示唆

- **既存ジャンルの「軸変換」をピボット案にする**: 横スクロール → 3D 全方位のような軸変換は、新規 IP よりも企画通過率を上げやすい（ただし操作感の再設計を伴う）。
- **キャラクター 4 人 × 分岐ルート**を入れると、シングル尺の短さを反復プレイで補える。ボリュームを増やさずに体感プレイ時間を伸ばすレバー。
- **「拾える / 壊せる」オブジェクトを背景に散らす**と、戦闘の空間が "ただの通路" から "舞台" に格上げされ、スクリーンショット映え・配信映えに直結する。
- **2P 協力を前提に動線を設計する**と、囲み・挟撃・武器受け渡しといった非言語コミュニケーションが自然発生し、社会軸＝協力の体験密度が上がる。
- **企画の出自ストーリー（"幻の○○ 4"）はマーケティング資産になる**。本来の経緯を伏せず公にすることでメディアの語り口を提供できる。

## 比較で見るべきタイトル

- ベア・ナックル III (1994) — 本作が後継を目指した直接の参照系。
- ファイナルファイト (1989) — 2D ベルトスクロールの原典。3D 化の差分を測る基準。
- ダイナマイト刑事 (1996, セガ) — ほぼ同時期に 3D ベルトスクロール翻案を試みた事例。
- Streets of Rage 4 (2020) — 結果的に四半世紀後に登場した「本物の続編」。3D 化を選ばず 2D に回帰した対比サンプル。
- The Warriors (2005, Rockstar) — オブジェクト破壊と協力プレイを 3D で煮詰めたベルトスクロール後継例。
