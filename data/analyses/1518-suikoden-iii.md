# 幻想水滸伝III / Suikoden III (Konami, 2002)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ゼクセン連邦とグラスランド氏族の戦争を、立場の違う 3 人の視点で読み解く」こと。戦闘そのものよりも、同じ事件が誰の目線で語られるかによって意味が反転していく群像劇の体験が中心軸にある。concept にある「ヒューゴ／クリス／ジョウイの 3 視点で最終的に 108 星に到達する」構造は、章を進めるごとに視点の差を埋めていくドキュメント体験であり、target である「幻水 I/II から続ける 20–40 代 JRPG ファン」「政治劇の重い物語を好む層」も物語消費を期待値の中心に置いている。social_axis が null なのは、本作がオフライン専用かつ他者との対戦／協力を主眼に置かないため、EXP の社会軸軸では分類されないこととも整合する。

secondary に REW が入るのは、108 星仲間収集と本拠地（城）育成という収集ループが物語進行と並走するため。ただし 108 星の収集自体が「真エンディング条件」として物語消費の駆動装置になっており、収集は物語完遂のための副次タスクに従属している。

## 物語の駆動構造

トリニティ・サイト・システム（TSS）は、章ごとに 3 主人公の誰でプレイするかをプレイヤーが選び、同じ時間軸の事件を別視点で追体験させる仕組み。視点切替が単なる演出ではなく、「相手陣営から見た自分の行動」を後から知らされる情報非対称の解消装置として機能する。

| 物語駆動装置 | 機能 | プレイヤーへの効果 |
|---|---|---|
| 3 主人公視点切替（TSS） | 同一事件を別陣営から再描写 | 善悪の単純化を解体、敵側の動機が事後的に明かされる |
| 章分割（Chapter 1–4 + Final） | 各章で 3 視点を一巡 → 視点が合流 | 「次章で別主人公の事情が分かる」継続フック |
| 108 星仲間収集 | 真エンディング条件 | 戦争の傍観者・市井の人物まで物語に巻き込む |
| 本拠地（城） | 仲間が定住し関係が更新 | 来歴の異なる仲間が同居することで群像感が物理化 |
| パーティ単位戦闘（3 人 1 組 ×2） | 6 人を「2 人 1 組」で操作 | 仲間同士の関係が戦闘表現に転写される |

PS2 移行で 3D 化したことにより、それまでドット絵で抽象化されていた民族・服装・地理の差が視覚的に立ち上がり、視点切替の情報量が増した。一方で表現コストが跳ね上がり、シリーズ続編の制作テンポが落ちる遠因にもなった。

## バズ要因 / 人気の根拠

- 「3 主人公視点で同じ戦争を描き直す」という構造的なギミックが、当時の JRPG では稀有で、群像劇 RPG というサブジャンルの代表例として語られ続けた
- シリーズ初の 3D 化と、I/II の世界観を時間軸で接続する設定（54 年後）が、既存ファンに「シリーズの正史を体験する」動機を提供した
- 北米では PAL 版が未発売だったことが逆にカルト化を促し、輸入プレイヤー・後年のレトロ評価で人気が再燃した
- シリーズ累計 239 万本のうち中核を担い、世界 50 万本超を売り上げた

## 企画への示唆

- 同じ事件を複数視点で再描写する構造を採ると、敵側の動機が後出しで開示でき、勧善懲悪を回避しやすい
- 視点切替を「章単位の必須切替」にすると、プレイヤーは特定主人公に過剰移入する前に別陣営の事情を知らされ、政治劇として読まれやすくなる
- 大規模仲間収集（108 星）を「真エンドの解錠条件」に紐付けると、戦闘外の市井キャラまで物語駆動に巻き込める
- 本拠地という「異なる出自の人物が同居する物理空間」を用意すると、群像感が UI 上で可視化される
- ナンバリングを跨いで時間軸を接続する（前作 54 年後など）と、既存 IP ファンの「正史補完」動機を励起できる

## 比較で見るべきタイトル

- 幻想水滸伝II（1998）— 同シリーズ前作。単一主人公・2D ドット時代の到達点として TSS との差分が見える
- ライブ・ア・ライブ（1994 / 2022 リメイク）— 章ごとに主人公が切り替わる群像 JRPG の先行例
- 十三機兵防衛圏（2019）— 13 人の視点でジグソーパズル状に同じ事件を再構成する現代の到達点
- Octopath Traveler（2018）— 8 主人公の章別構成で群像劇を再解釈した HD-2D 期の例
