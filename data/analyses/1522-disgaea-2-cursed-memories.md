# 魔界戦記ディスガイア2 (Disgaea 2: Cursed Memories) — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**レベル9999・転生・アイテム界という極端な数値レンジの中で、自軍を強くする手触りそのものを味わう**」ことにある。物語（呪われた青年アデルと暴君の娘ロザリンドのロードムービー）も、キャラクター育成の量（数百〜数千時間級）も両方厚いが、プレイヤーが画面に向かって繰り返す行為は「マップに潜って数値を伸ばす」であり、その手触りが報酬や物語より先に立つ。

`social_axis: ソロ` は同期マルチや非同期ランキングを持たないことを意味し、対戦・協力・観戦はすべて意図的に切り落とされている。深掘りの自由度を「他人との比較」で薄めず、自分の盤面の中だけで完結させる設計判断と読める。`secondary: [REW, NAR]` は、装備堀り＝報酬獲得ループ（REW）と、章ごとのコメディ寄り脚本（NAR）が体験を下支えしている構図を示す。動機は EXP、燃料は REW と NAR。

## 体験の核 / 設計の発明

中核メカニクスは「タイル合体（ジオエフェクト）／持ち上げ投げ／議会／転生／アイテム界」が積層した、**1 マップ内に閉じない深度**である。前作の発明を継承しつつ、転生で素質値を引き継いだまま Lv1 に戻し再育成するループを正面に据えた。

| 層 | 機能 | プレイヤーが得るもの |
|---|---|---|
| 戦闘 | ジオシンボル／持ち上げ投げ／合体技 | 1 ターンの操作で盤面が一変する瞬間 |
| 育成 | 転生・適性値・武具マスタリー | 「数千時間積んだ意味」が数値で可視化される |
| 装備 | アイテム界（装備品の内部ダンジョン） | 装備自体を成長対象に変える二重ループ |
| 統治 | 暗黒議会（賛成/反対を買収・粛清） | システムを「政治で曲げる」メタ層 |
| 物語 | 章間コメディシーン | やり込みの合間に提供される回復剤 |

ソロ軸でこれだけの層を積めるのは、**進行制限を時間以外で課さない**設計の帰結である。スタミナや日次リセットがないため、プレイヤーの可処分時間がそのまま体験の長さに変換される。

## バズ要因 / 人気の根拠

- Metacritic 84 点・IGN「Best PS2 Strategy Game 2006」受賞という、海外メディア評価でブランドを確立した。
- ワールドワイド 40 万本超という、PS2 後期の和製ニッチ SRPG としては手堅い数字を残し、NIS America の海外プレゼンスを押し上げた。
- 前作のやり込み深度（Lv9999・アイテム界）を温存し、ファン離反を起こさず系譜の信頼を維持した。
- 日本一ソフトウェアらしいギャグ・キャラ造形（ロザリンド、アクトレス、プリニー）が、SRPG の硬派な数字遊びとコメディ脚本という二層構造を成立させた。

## 企画への示唆

- **マルチを切り捨てるとソロ深度に投資できる** — 他者比較を捨てると、ランキング均衡・チート対策・サーバー運用の制約から解放され、内部ループの天井を青天井に上げられる。
- **「装備をダンジョンにする」と二重ループになる** — キャラ育成の上にアイテム育成を載せると、同じ戦闘マップが二度三度違う動機で再訪される。
- **数値の桁を一段上げると、それ自体が IP の語り口になる** — Lv9999 / ダメージ億単位は、ジャンル外の人にも一行で説明できるフックになる。
- **コメディ脚本はやり込みの疲労に効く回復剤** — 数百時間遊ぶ前提のゲームほど、章間の軽い物語が継続率に効く。
- **「政治で仕様を曲げる」サブシステムは長寿命化する** — 暗黒議会のように、プレイヤーがルール自体に介入できる層を 1 つ用意すると、攻略 Wiki と二次創作が育つ。

- 魔界戦記ディスガイア（2003 / PS2） — 前世代直系、転生・Lv9999 の原型。

- ファイナルファンタジータクティクス（1997） — SRPG の物語駆動側の極北。EXP/NAR バランスの対比。
- ラ・ピュセル 光の聖女伝説（2002 / 日本一ソフトウェア） — 同社の前々作、属性タイル系譜の起点。
