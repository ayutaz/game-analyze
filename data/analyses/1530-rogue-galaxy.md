# ローグギャラクシー — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「シームレスフィールドを走り回り、リアルタイム戦闘とサブシステムの密度を遊び尽くす」** こと。物語は王道スペースオペラだが、本作の時間消費の大半は戦闘・武器合成・ファクトリー・昆虫バトルといった「遊びの層」に吸われており、`concept` が明示する「遊びの密度で押し切る構成」が EXP 分類の根拠になっている。

`social_axis: ソロ` は PS2 世代の JRPG として標準的な選択で、PT メンバーを切り替えて操作する単独プレイ体験に閉じる。`target` が示す「ドラクエ VIII / ダーククロニクルを好むやり込み志向層」は、物語の続きより「武器を育て切る」「ファクトリーで部品を量産する」を駆動力にする層であり、これも EXP 寄りの動機構造を裏づける。

副次分類 `NAR` は宇宙海賊として惑星を巡る章立て、`REW` は武器合成と収集系サブゲームに対応する。両者は EXP の体験密度を支える「燃料」として効くが、主動機ではない。

## 体験の核 / 設計の発明

レベルファイブが 3 年以上をかけ「当時最大規模」で組み上げた本作の特徴は、**フィールド遷移を切らないシームレス構造** と、**戦闘の外側に置かれた複数のサブループ** が相互に資源を循環させる点にある。

| レイヤー | 中核メカニクス | プレイヤーが得るもの |
|---|---|---|
| 探索 | シームレスに繋がる惑星フィールド | ローディングで途切れない没入 |
| 戦闘 | リアルタイム ARPG + PT 切替 | 操作スキルと戦術選択の両立 |
| 武器育成 | 武器同士を合成し新武器を生成 | 「組み合わせ発見」の収集動機 |
| ファクトリー | パーツを組んでアイテム自動生産 | 経済的に資源を再投資する仕掛け |
| 昆虫バトル | 採集した昆虫で対戦するミニゲーム | 本筋から逸れた長時間プレイ余地 |

前世代の Level-5 タイトル（ダーククロニクル）からの差分は、**フィールドのシームレス化** と **サブシステムの並列多層化** に集約される。各サブループは資金・素材・経験値の出口を別経路で用意し、メインストーリーから一時的に逸脱しても「進捗が止まらない」状態を維持する設計。`social_axis: ソロ` の中で他者比較がない代わりに、内部の数値成長そのものが目的化する。

## バズ要因 / 人気の根拠

- 発売週に **日本で週間 1 位（23.7 万本）** を記録し、PS2 末期の大型 JRPG として初動は明確に出た。
- 累計は **日本約 27 万本（DC 版 2.9 万本含む）/ 全世界 60 万本超** に着地。SCE が期待した 100 万本ラインには届かず、商業期待値と実数の乖離が後年語られる材料になった。
- レベルファイブにとって **『レイトン教授』直前のターニング作品** として位置づけられ、自社パブリッシング戦略への転換契機の一つに数えられる。
- PS3 向け **ディレクターズカット版を欧州・北米・後に PS2 国内で展開** し、シームレス世界 + サブシステム多層構成というフォーマット自体は同社の後続作（イナズマイレブン、二ノ国）の設計思想に継承された。

## 企画への示唆

- **シームレス化はそれ単独で売れない / プレイヤー時間を伸ばすための土台に過ぎない** → 切れ目のないフィールドは「サブループを跨ぐ移動コスト」を下げる前提条件であり、上に乗せる遊びがないと体感されない。
- **サブシステムは資源の出口を別経路に分けると並列稼働する** → 戦闘 = 経験値、合成 = 武器、ファクトリー = 金、昆虫 = ミニゲーム報酬のように出口を分離すると、プレイヤーは「気分でループを切り替える」運用を始める。
- **初週 1 位 = 成功ではない** → 23.7 万本初週でも 100 万本目標未達なら社内的には未達評価になる。KPI を「初動」ではなく「累計 / LTV」で設計すると商業評価がブレない。
- **大規模長期開発は次作の方向転換コストを生む** → 3 年以上の大型 IP を期待値未達で着地させた場合、次作で IP リスクを下げた小規模高収益作（レイトン）に振る選択肢を準備しておく。
- **やり込み層向けに「本筋から逸脱できる出口」を 2 つ以上用意する** → 昆虫バトルのような完全に独立した遊びは、メイン進行の進度ばらつきを吸収するクッションとして働く。

## 比較で見るべきタイトル

- ダーククロニクル（Level-5、2002）— シームレス志向とサブシステム多層化の前身。
- ドラゴンクエスト VIII（Level-5、2004）— 同じ Level-5 の PS2 王道 JRPG。シームレス採用の社内文脈。
- 二ノ国 白き聖灰の女王（Level-5、2011）— ローグギャラクシーの「世界を巡る ARPG」設計を IP 強化で再実装した後続作。
- スターオーシャン Till the End of Time（tri-Ace、2003）— PS2 期スペースオペラ ARPG の比較点。
