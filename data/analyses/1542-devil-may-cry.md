# デビル メイ クライ (2001) — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「いかにカッコよく敵を倒すか」を自分の手で組み立てることにある。クリア可否ではなく、銃と剣・空中コンボ・回避を継ぎ目なく繋いで **スタイルランク (D→C→B→A→S)** を引き上げる行為そのものが報酬になっており、戦う動詞が主であって読む動詞ではない。

concept にある「クリア時間・被ダメ・スタイルでランク評価される高難度のソロ攻略型」という設計は、ダンテというキャラクターを操作する身体感覚と評価フィードバックの距離をきわめて近く結ぶ。target も「コンボや高ランク取得を楽しむ層」が中核に置かれており、物語没入よりも入力 → ランク表示までのループを反復することが楽しみの中心になる。social_axis は「ソロ」で、対戦も協力も介在せず、難易度設定 (Normal / Hard / Dante Must Die) を上げて自分自身と戦う構造。

secondary に NAR が立つのは、ダンテの出自・ムンドゥス討伐というゴシックホラー風の物語が世界観を支えているため。ただし物語の進行は章 (Mission) 単位の幕間にとどまり、戦闘ステージのフロー自体は物語消費から切り離されている。

## 体験の核 / 設計の発明

中核は **「行動の連結に対するメタ評価」** で、攻撃の種類・連続性・被ダメの少なさを 1 ステージ単位で採点し直す。前世代 (バイオハザード系の固定カメラ + 緊張型探索) からの差分が明確になる。

| 設計レイヤ | 内容 | 体験への効果 |
|---|---|---|
| 基本入力 | 剣 (Rebellion) / 銃 (Ebony & Ivory) / ジャンプ / 回避 / Devil Trigger | 同時押しではなく分業した武器系で「足を止めない」 |
| スタイルランク | 単発ヒットでは上がらず、連続性・多様性・被ダメで評価 | プレイヤー側が「見映え」を意識して入力を構築する |
| ミッション制 | 23 ミッションをランク付きで再走可能 | クリア後に S 取得を目指す second life が走る |
| 難易度層 | Normal / Hard / Dante Must Die | 同じステージを 3 種以上の解像度で味わわせる |
| 空中浮遊 | スティンガー → 空中コンボ → 銃で滞空継続 | 地上 2D アクションの語彙を 3D に移植 |

前世代の 3D アクションが「敵を倒す」までで評価を止めていたのに対し、本作は **倒し方** に二階層の評価を被せた。これが後年 Bayonetta / Devil May Cry 5 / メタルギアライジング / NieR:Automata 系の「スタイリッシュアクション」サブジャンルの祖型になる。

## バズ要因 / 人気の根拠

popularity の通り、神谷英樹監督・三上真司プロデュースで、当初『バイオハザード4』として開発されていた企画から派生したという出自が語り草となった。

- 銃と剣を同時に使う 3D アクションという新ジャンル提示で、ジャンル名「スタイリッシュアクション」自体が後発タイトルの参照点になった
- 国内 0.65M / 全世界 2.16M (PS2) で、PS2 初期 (2001) のサードパーティ新規 IP として高水準の数字を出した
- シリーズ累計 2700 万本超の礎となり、DMC2 / 3 / 4 / 5 と Reboot を経て四半世紀の長期 IP に成長
- ダンテのキャラクター造形 (赤コート・白髪・斜に構えた台詞) が以降のキャラクターアクション系主人公の参照テンプレート化

## 企画への示唆

- 倒したか否か (Boolean) ではなく **倒し方を採点する** メタレイヤを足すと、同一ステージの再走動機が生まれる
- 評価軸を 3 つ以上 (本作: 時間・被ダメ・スタイル) 並べると、プレイヤーが自分でビルドの優先順を決めるためリプレイの自由度が上がる
- 武器を片手ずつ別枠で持たせる (剣と銃) と、入力チャネルが分かれて「足を止めない戦闘」が成立する
- 難易度を 3 段以上用意し、同じ攻略動線を異なる解像度で味わわせると、クリア後の second life がそのままコミュニティ寿命になる
- 物語は世界観の輪郭づくりに留め、ループの主役を「入力 → 評価」に振り切ると、ソロ動詞でも観戦・実況文化が後から自然発生する

## 比較で見るべきタイトル

- Bayonetta (2009) — 同じ神谷英樹監督による正統進化形、評価メカニクスの拡張版
- Devil May Cry 3: Special Edition (2006) — スタイル切替を加えた本シリーズの完成形
- メタルギアライジング リベンジェンス (2013) — 切断ベクトルで採点軸を再発明したフォロワー
- NieR:Automata (2017) — スタイリッシュアクション × NAR ハイブリッドの代表例
- God Hand (2006) — クローバースタジオ系列の難度特化フォロワー
