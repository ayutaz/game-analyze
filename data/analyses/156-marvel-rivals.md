# マーベル・ライバルズ 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「マーベルヒーローの身体で 6v6 を競う」こと。読むでも集めるでもなく、戦う／競う体験そのものが核にある。`concept` が示す通り 36 体超のヒーローを操作する基本無料シューターであり、`social_axis: 対戦` が示すように勝敗のかかった同期マッチが時間消費の大半を占める。`target` はヒーローシューター経験者とマーベル映画ファンの中庸帯で、IP 親和性で軽度層を引き入れつつ MOBA / 競技層を留める二段構えになっている。

副次の `REW` は、シーズンバトルパスと無料／有料スキンの収益化レイヤーが、対戦のリテンションを下支えしているという意味合い。報酬は動機の中心ではなく、対戦動機の継続装置として機能している。

## 体験の核 / 設計の発明

| 層 | 設計要素 | OW2 系との差分 |
|---|---|---|
| 中核メカニクス | 6v6 ヒーローシューター、ロール解放（ロール強制なし） | OW2 が 5v5 ＋ロール固定化したのと逆方向、混雑感と即興性で勝負 |
| 差別化技 | チームアップ技（特定ヒーロー同士の組合せで発動） | 単キャラ性能ではなくペア／トライアド設計が編成発想を変える |
| マップ干渉 | 破壊可能地形 | 遮蔽物が試合中に消えるため、ポジション勝ちの賞味期限が短い |
| IP レバレッジ | スパイダーマンの三次元機動、ハルクの突進、ドクター・ストレンジのポータル | 「映画で知っている動き」を入力デバイスで再現させる学習負荷の低さ |

`social_axis: 対戦` の効果は二層ある。表層は MMR ベースのランクマッチで競技動機を維持すること、深層は「マーベル俳優の身体を借りて勝つ」という象徴消費を毎マッチに織り込むこと。後者は OW2 のオリジナル IP では再現しにくいため、コアファンとライト層を同じマッチに同居させる構造になっている。

## バズ要因 / 人気の根拠

- `popularity` が要約するように、OW2 のロール強制と 5v5 化に不満を持つ離脱層を、6v6 ＋ロール自由でほぼそのまま受け皿化した。
- `sales_world` ベースで 40M+ プレイヤー、初月 $136M。基本無料 IP コラボシューターとしてはローンチ直後の到達速度が突出している。
- マーベル MCU の世界的認知が、シューター未経験層の試遊コストをほぼゼロにした。アイアンマン／スパイダーマンを「とりあえず動かしたい」需要が無料配布と噛み合っている。
- 配信映え要素として、チームアップ技の発動演出と破壊可能地形のクライマックスがクリップ化されやすく、Twitch / YouTube の短尺動画文化と相性が良い。

## 企画への示唆

- 既存ジャンル王者が「軽量化」へ舵を切ったとき（5v5 化等）、逆張りで「重い旧仕様」を継承するとリーバー層を一網打尽にできる。
- 強力 IP × 既存ジャンルの組合せは、ジャンル経験者と IP ファンを同じマッチに同居させ、スキル格差を物語消費で隠蔽できる。
- ペア／トライアドの発動条件をスキルに紐付けると、「キャラ単体の強さ」ではなく「編成会話」が SNS の語り口になり、コミュニティ生成が早い。
- 破壊可能地形のように「試合中に環境が変わる」要素を入れると、ポジション最適化のメタが固まりにくく、配信クリップが量産される。
- 基本無料 ＋ スキン課金は対戦動機を阻害しない範囲で REW を副次化できる。報酬を中心に据えると競技層が剥落するので層を分ける。

## 比較で見るべきタイトル

- Overwatch 2（直接の競合、5v5 ＋ロール強制との対比）
- VALORANT（戦術系ヒーローシューター、アジア圏配信文化）
- Apex Legends（3 人 PvP のキャラ性能差別化、シーズンパス設計）
- Fortnite（IP コラボ × 基本無料 × クロスプラットフォーム）
- Paladins（旧世代 OW フォロワー、IP 力不在時の伸び悩み事例）
