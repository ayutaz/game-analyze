# バーンアウト ドミネーター

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ブーストを途切れさせず走り切り、相手をテイクダウンする」体験そのものである。物語も収集要素も主役ではなく、コーナーをインカット → 対向車線を逆走 → 追突寸前の Near Miss でブーストを焼き、その勢いでライバル車を壁にめり込ませる、という瞬間の運動連鎖が成立したかどうかにすべてが懸かっている。concept が示す通り、本作は Crash Mode を切り捨て「Burnout チェーン（ブースト消費中にさらにブーストを稼ぎ続ければ倍率が伸び続ける）」に特化した過渡期作品であり、target も「Crash Mode より純粋なレース・テイクダウン体験を求める層」と明示されている。

social_axis は「対戦」。シリーズ標準のオンライン Road Rage / Race に加え、PSP 版は ad-hoc によるローカル対戦が PSP アーケードレースの定番だった。secondary は空で、副次動機（収集・物語）は意図的に削がれている。シリーズの中でも最もミニマルな EXP 設計と言える。

## 体験の核 / 設計の発明

中核は **Burnout チェーン** と呼ばれる「ブーストを止めない」ことを報酬化するループ。Burnout 3 / Revenge から続く Boost ゲージ駆動を、Crash Mode 廃止と引き換えに過剰な純度で押し出した。

| 要素 | 設計 | プレイヤー側の効果 |
|---|---|---|
| Boost ゲージ | 危険運転（逆走 / Near Miss / Drift / Oncoming）で蓄積 | リスク行動が即報酬になる強化学習ループ |
| Burnout チェーン | ブースト消費中にゲージを満タンまで戻すと連鎖 | 「止めない」選択が常に最適行動になる |
| Dominator イベント | チェーン回数で評価される専用モード | ループそのものを目的化する競技ルール |
| テイクダウン | 接触で相手を破壊するとブーストとリトライ免除 | 攻撃が走行リソースに直結 |
| Crash Mode 廃止 | パズル的逸脱の削除 | アーケードレース純度の上昇 |

前世代 Burnout Revenge との差分は (a) Traffic Checking（同方向の一般車を弾き飛ばす）を継続しつつ、(b) Crash Junction を捨て、(c) PSP に最適化された短サーキット化に振った点。social_axis「対戦」の効果は、テイクダウンが「相手を 1 周分後退させる」直接干渉になっている点で、レース順位が走行スキルだけでなく接触ライン取りの読み合いに変質する。

## バズ要因 / 人気の根拠

popularity は「Criterion 不在で EA UK が引き継いだナンバリング。Crash Mode を廃しスピード一辺倒の Burnout 3/Revenge 直系として、PSP の代表的アーケードレースの 1 本に数えられた」。これを分解すると：

- Criterion Games が Burnout Paradise（オープンワールド転換）に専念する裏で、EA UK が「クラシック型 Burnout」最終作を出したという シリーズ史的なポジション。
- PS2 末期 / PSP 全盛期というハード端境にぶつけられ、PSP 携帯機ユーザーには「Burnout を持ち歩ける」最初で最後の据置直系体験になった。
- Crash Mode を切ったことで賛否が割れたが、Dominator イベントと Burnout チェーンに集約したことで「走り込み層」の支持を獲得。
- 売上数値は sales_jp / sales_world ともに「—」で非公開だが、シリーズ累計と PSP アーケードレースのレファレンスとして長期にわたり言及される作品となった。

## 企画への示唆

- 看板機能（Crash Mode）を**意図的に切り捨てる**と、残った核（ブーストループ）を競技ルール化する余地が生まれる。
- 「リソース消費中に同リソースを稼げば連鎖」という設計（Burnout チェーン）は、ブースト系・コンボ系・スコアアタック系すべてに移植可能な普遍メカニクスである。
- 接触＝資源化（テイクダウンで Boost 獲得）にすると、レース対戦は走行精度ゲームから**ライン読み合いゲーム**に変質する。
- 携帯機移植時はサーキットを短縮し 1 周ループを高速化すると、ブースト連鎖が短時間プレイと噛み合う。
- 開発スタジオを変えても**ゲージとイベント設計のレシピが残っていれば**シリーズ DNA は保てる。逆に言えば、シリーズの核は IP やスタジオでなくメカニクスの式に宿る。

## 比較で見るべきタイトル

- Burnout 3: Takedown / Burnout Revenge — 直系前世代、Crash Mode が現役だった時期
- Burnout Paradise — 同時期に Criterion 本体が作ったオープンワールド転換版
- Need for Speed: Most Wanted (2005) — 同 EA・同期のアーケードレース、警察追跡で別軸の干渉設計
- Ridge Racer (PSP) — PSP アーケードレースの並走参照、ドリフト主導
- Wipeout Pulse — PSP の高速感系、ブースト管理思想が近い
