# ラチェット&クランク (2002)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「奇抜な武器・ガジェットを切り替えながら 3D 空間を撃つ・跳ぶ・解く」体験そのものにある。物語は SF コメディとして機能するが、章を読み進めるための作品ではなく、各惑星のレベルデザインを攻略する手触りが中心に置かれている。`social_axis` がソロであることもこの判断を支える。対戦・協力の社会性ではなく、自分一人で多彩なステージを踏破する達成感が体験設計の中核にあるためだ。

ただし `secondary: REW` が示すように、武器・ガジェットの入手と段階的アンロックは強い副動機として効いている。プレイヤーは「次の惑星に行けば新しい武器が買える」「ボルト（通貨）を貯めれば派手な武器に持ち替えられる」という収集ループに乗せられ、3D アクションをハック&スラッシュ的なリプレイ体験へ拡張している。NAR 主軸ではなく EXP+REW の入れ子構造というのが、本作の分類上の特徴である。

## 体験の核 / 設計の発明

中核は「惑星ごとに違う武器を試す遊び場」というロールアウト構造だ。前世代のソニー系 3D プラットフォーマー（クラッシュ、ジャック&ダクスター）が一つの相棒能力で全ステージを通すのに対し、本作は武器ガジェットのスロットを増やすことで攻略バリエーションを線形的に増やす設計を取った。

| 設計要素 | 具体 | 体験への寄与 |
|---|---|---|
| 武器スロット拡張 | 物語進行で購入可能武器が増える | 同じ敵に複数の解法が成立 |
| ボルト経済 | 雑魚・箱から通貨が落ちる | 戦闘→収集→購入のサブループ |
| ガジェット鍵 | スイングショット等を入手して既出ステージに戻る | メトロイドヴァニア的逆戻り探索 |
| 相棒クランク | バックパック化で滑空・ジェット | プラットフォーマー側の縦軸を拡張 |
| 惑星単位ハブ | 銀河マップから自由選択 | 一つの惑星に「テーマ」を集中 |

社会軸はソロだが、武器のドタバタしたカートゥーン演出が観戦・実況とも親和し、後年シリーズの実況プレイ文脈にも繋がっている。

## バズ要因 / 人気の根拠

JSON の popularity が示す通り、本作の話題化は「西洋開発タイトルとして初めて PS2 本体の同梱パックに採用された」という流通イベントが起点だった。事実分解：

- 発売後 1 週間で世界 100 万本近い出荷を記録し、第 1 作からシリーズ化が確定した
- 日本では SCE が「アクションパック」として PS2 本体に同梱、当時西洋スタジオ製として異例の扱いを受けた
- 国内 26 万本 / 全世界 3M+ という、PS2 期 3D プラットフォーマーとして安定した数字を残した
- Insomniac Games の Spyro the Dragon (PS1) で培われた高速アクション設計を、武器多数化で再構成したことが評価の中心となった

## 企画への示唆

- 「相棒キャラ＝能力ガジェット」として実装すると、ストーリー上の絆と機構上の能力追加を一行で接続できる
- 武器ロスター制を導入すると、同じ敵に対する解法バリエーションが増え、プレイヤーごとの遊び方差が発生する
- ステージ内通貨を全ドロップに混ぜると、戦闘・破壊・探索のどれを選んでも前進感を返せる
- 惑星単位でテーマ（ジャングル、雪山、宇宙ステーション等）を切り替えると、3D アクションのマンネリ化を 5〜7 時間単位でリセットできる
- 西洋 IP を国内に出す際、ハード同梱パックという流通レイヤーで認知を獲得すると、ソフト単体露出より初動 PR コストを下げられる

## 比較で見るべきタイトル

- ジャック×ダクスター（2001, Naughty Dog）— 同時期 PS2 ソニー系 3D プラットフォーマー、相棒変身ギミック
- Spyro the Dragon（1998, Insomniac）— 同スタジオ前世代作、能力単一の純プラットフォーマー
- スーパーマリオ サンシャイン（2002）— ガジェット（ポンプ）で 3D アクションを拡張した同年作
- ヘイロー（2001, Bungie）— 武器スロット制を FPS で確立した同時期作
