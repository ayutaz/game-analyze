# マフィア (Mafia) — 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「禁酒法時代マフィアの一代記を追体験する＝読む／観る」** こと。タクシー運転手トミー・アンジェロがサリエリ・ファミリーに身を投じ、頭角を現し、やがて裏切られて破滅する一連の悲劇を、ミッション制で順番に通過していく構造になっている。concept にある通り中核は「映画的シナリオ」で、ゲームプレイ（運転・銃撃）はその物語を進行させるための演出装置として配置されている。

target も「『ゴッドファーザー』『ワンス・アポン・ア・タイム・イン・アメリカ』のような禁酒法時代マフィア物語を体験したい層」と明示しており、求められているのは TPS スキルではなく物語の重さ。そのため `social_axis` は EXP 軸ではなく未設定（NAR は他者軸を持たない）扱いで適切。一方 secondary に **EXP** を残したのは、12 平方マイルのオープン都市・時代考証された当時車両でのドライブ・銃撃戦の手触りも一定の比重を占めるため。物語駆動の枠組みの中で、走る・撃つ体験が章ごとの空気感を担保している。

## 物語の駆動構造

物語消費を駆動する仕掛けを分解する。

| 構造要素 | 内容 | 物語上の効果 |
|---|---|---|
| 主人公の一代記形式 | タクシー運転手 → ファミリー入り → 中堅 → 破滅 | 「上昇と転落」の古典的弧、観客の感情移入が安定 |
| ミッション制の章立て | 線形に並んだミッション単位で進行 | 映画のシーン単位編集に近い、密度を制御しやすい |
| 時代設定（1930年代禁酒法時代） | 架空都市 Lost Heaven、当時の車・服飾・音楽 | 既存の「マフィア映画的記号」を呼び出し、説明コストを削減 |
| 回想形式の語り | トミーが過去を警官に語る構造 | 結末の暗示で全章に不穏さを敷ける |
| オープン都市の存在 | 自由走行も可能だが進行は線形 | 「世界の実在感」を担保しつつ脱線を抑制 |

GTA III（2001）が同時期に「サンドボックスでの逸脱の自由」を売りにしていたのに対し、Mafia は **同じオープン都市技術を物語の密度のために使った** 点で系統が分岐している。

## バズ要因 / 人気の根拠

- 1930年代禁酒法時代を精密再現した時代考証——車両・建築・服飾・音楽が当時資料ベースで構築され、PC ゲーマーから「ゲームで初めて触れる本格マフィア映画」として評価された。
- GTA III と同年（2002 リリース）でありながら **逸脱の自由ではなく物語の重さで差別化**、「GTA とは一線を画す映画的マフィア物語」というポジションを獲得。
- ワールドワイド累計 **2M+ 本** を売り上げ、チェコ発（Illusion Softworks）の作品としては当時最大級のヒット。日本国内売上は記録上 "—" だが PC ゲーマー間でのカルト的評価は継続。
- 2020 年の **フルリメイク版 "Mafia: Definitive Edition"** によって新世代プレイヤーに再到達、原作の物語構造の強度が改めて確認された。

## 企画への示唆

- **同じオープン都市技術を「自由」ではなく「世界の実在感」に振ると**、サンドボックスでは到達できない映画的密度が生まれる。
- **既知の映画ジャンル記号（マフィア／西部劇／戦争映画）を採用すると**、ワールド構築の説明コストが大幅に下がり、プレイヤーの感情移入を初期化できる。
- **主人公の一代記（上昇 → 中堅 → 転落）を骨格に置くと**、ミッション単位の章立てが映画のシーン編集として機能し、線形でも飽きにくい。
- **回想形式の語り（結末を先に提示）を採用すると**、全章に通底する不穏な空気が無料で手に入り、終盤の悲劇への伏線処理が楽になる。
- **同時期の競合（GTA III）と同ジャンルでも、動機軸（NAR vs EXP）を明確にズラすと**、市場のカニバリゼーションを回避して別セグメントを獲得できる。

## 比較で見るべきタイトル

- **Grand Theft Auto III (2001)** — 同年・同オープン都市技術、EXP（逸脱の自由）軸に振った系統
- **L.A. Noire (2011)** — 同じ「都市犯罪×時代再現×線形物語」、Rockstar 系 NAR 寄りの後継系譜
- **Mafia II (2010) / Mafia III (2016)** — 同シリーズ、戦後／公民権運動期へと時代を移しながら同じ一代記構造を再利用
- **Red Dead Redemption (2010)** — オープン世界＋一代記＋時代記号（西部劇）の組み合わせを Rockstar が完成形に近づけた例
- **The Godfather: The Game (2006)** — 同題材（マフィア映画原作）を EXP（自由度）寄りに翻案した対照例
