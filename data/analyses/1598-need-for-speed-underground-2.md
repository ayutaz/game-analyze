# Need for Speed: Underground 2 (2004) 分析

## なぜ EXP 主分類なのか

主動機は「夜の街を流して、レースの合間に自分の車を作り変える」体感そのものを楽しむこと。`social_axis: ソロ` でオンライン対戦には依存せず、プレイヤーは AI と走る／カタログを眺める／パーツを当てる、というループを単独で延々と回す。

`concept` が示す通り、本作の発明は「違法ストリートレース＋深いビジュアル/メカチューン」を **オープンワールド「ベイビュー」を流して次のレースを探す** 構造に統合した点にある。レース→賞金→改造→次のレース、という流れの途中に「街を運転する時間」が物理的に挟まる設計は、勝つこと以上に「自分の車で街を流す」体験を主軸へ押し上げる。`target` にある JDM / USDM カルチャー層に対して、シミュレーターの精密さではなく「見た目を組む手触り」を提供している点も EXP 側に振れる根拠。

副次として `secondary: REW` が効くのは、ボディキット／ハイドロリクス／ガルウィングドア／デカール／ホイールといったカタログ収集と「DVD/雑誌カバーに載る」スコアリングがあるため。改造は完全に報酬ループでもあるが、それは「街を流す体験」のための装備調達として機能する。

## 体験の核 / 設計の発明

中核メカニクスは「街を流す時間」と「カタログを開く時間」の往復で、前作 (Underground 1) からの差分は明確に構造寄りに振れた。

| 層 | Underground (2003) | Underground 2 (2004) |
|---|---|---|
| ワールド | レース選択メニュー | オープンワールド「ベイビュー」5 区画 |
| 移動 | ファストトラベル前提 | 街を流してレースを発見 |
| カスタム | パフォーマンス＋外装中心 | ハイドロリクス／ガルウィングドア／SUV クラス追加 |
| カテゴリ | サーキット／ドラッグ／ドリフト | + Street X / アウトラン / URL |
| 文化的引用 | チューナーシーン全般 | 雑誌カバー (DUB 等) ／ MTV 系サントラ |

`social_axis: ソロ` の効果は「夜の街に他人が走っていない」ことが逆に審美性を担保する点にある。AI 交通とドライブの時間がプレイヤーの車をスクリーンセーバーのように見せ、停車してカメラを回す行為が成立する。ハイドロリクスやネオン管が「対戦相手に勝つための装備」ではなく「走っている自分を眺めるための装備」として意味を持つ構造は、対戦軸のレースゲームには成立しない。

## バズ要因 / 人気の根拠

`popularity` 文を分解すると以下：

- 前作 Underground (2003) の社会現象的ヒットを引き継ぎ、世界累計 **11M+** （JSON `sales_world`）に到達。シリーズ屈指の販売数。
- 舞台「ベイビュー」によるオープンワールド化が、当時のレースゲームの主流（メニュー選択型）から一歩抜け出した。
- ハイドロリクス／ガルウィングドア／SUV クラスなど **改造のネタの幅** を大きく広げ、雑誌カバー文化（DUB 誌等）を取り込んだ。
- リッキー・マーティン、スヌープ・ドッグら **MTV 圏の濃いサントラ** が、ワイルド・スピード初期作と同期してチューナー文化のサウンドトラックとして機能した。

## 企画への示唆

- レース／対戦ゲームに **「移動する時間」をオープンワールドで挟む** と、勝敗以外の動機（自分の機体を眺める／街を観光する）が立ち上がる。
- カスタム要素は **「対戦で効くパーツ」と「自己満足パーツ」を明確に分離** すると、ソロ軸の長期リテンションが伸びる（ハイドロリクスは戦闘性能に効かない）。
- IP コラボでなく **時代の音楽／雑誌／ファッション** をライセンスすると、文化的アイコン化のコストは下がり、世代記憶として残りやすい。
- カタログを **クラス × ブランド × 解禁条件** の三次元で並べると、収集はそれ自体が「次のレースを走る動機」に変換される。
- スコアリングを **数値ランクだけでなく「雑誌カバー掲載」「DVD 映え」など視覚的な称号** に置くと、SNS 以前の時代でもプレイヤー内発の自慢動機が生まれる（現代なら配信／スクショ用 UI に応用可）。
