# メタルギア ソリッド Δ: スネークイーター

## なぜ NAR 主分類なのか

プレイヤーの主動機は「2004 年版『MGS3』の冷戦スパイ劇とジャングル・サバイバルの脚本を、現代描画で**追体験する**こと」である。本作は concept にあるとおり「原作の脚本・声優・演出を保持したまま現代グラフィックで再構築したリメイク」であり、ゲームプレイの発明より物語の保存に主軸を置いている。target が「30-50 代の旧シリーズ経験層」と「小島監督期 MGS に思い入れのあるコア層」に明確化されている時点で、本作のドライバーは未知の体験ではなく既知の名作の再観賞である。

ただし `secondary: ["EXP"]` が示すとおり、肩越し操作・新規描画によりステルス／カモフラージュ／CQC といった体験面もモダン化されており、追体験のリッチさを支えている。物語が主、体験は物語を伝える媒体、という階層が成り立つ。

## 物語の駆動構造

「保存型リメイク」が物語消費をどう駆動するかを分解する。

| 層 | 原作（2004） | Δ（2025） | 機能 |
|---|---|---|---|
| 脚本 | 小島秀夫脚本 | 同一 | 旧来ファンの記憶補強 |
| 声優・演技 | 大塚明夫他 | 原音声を保持 | 「あの演技」を再生する装置 |
| 演出・カット割 | PS2 期のシネマティック | 原カット保持＋現代描画 | 名場面の高精細再演 |
| 操作系 | 真上俯瞰／TPS（Subsistence） | 肩越し TPS をデフォルト化 | 追体験の操作障壁を下げる |
| サバイバル系 | 食料・治療・カモフラージュ | 同設計を踏襲し UI 改修 | 冷戦ジャングルの世界観維持 |

物語駆動の核は「**変えない部分を明示すること**」である。原音声・原カット割を残すことで、プレイヤーは記憶の中の MGS3 と画面上の Δ を照合し続け、「あの場面が綺麗になった」体験が連続的に発生する。新規プレイヤー向けには映画的演出と冷戦スパイ譚という普遍的構造がそのまま機能する。

## バズ要因 / 人気の根拠

JSON の popularity「小島作品の名作再演、原音声・原演出を保持したリメイクで旧来ファンが復帰」を分解する。

- 世界売上 300 万本超（JSON 記載）と、リメイク／既存 IP の中では水準以上の初動を確保
- 小島秀夫退社後の Konami が放った最初の大型 MGS 案件として、関係者離散後にシリーズが続行できるかという業界的関心が事前期待を押し上げた
- 原音声・原演出を保持する「保存型」方針を発表段階から徹底し、リメイクで脚本・演出が改変されることへのファン側の警戒を初手で無効化した
- Virtuos が外部開発として参画した点が公開され、IP ホルダーと制作実務の分業モデルが旧シリーズファンに対しても受容された

## 企画への示唆

- 名作 IP の保存型リメイクでは、**脚本・演技・演出を変えない宣言を初手で出す**と、ファン側の改変不安が初期マーケのコストを下げる
- 体験層（操作系・UI）と物語層（脚本・カット）を**分離設計**し、後者を凍結／前者だけ現代化すると、新旧両層のプレイヤーを同時に拾える
- ターゲットを「当時のプレイ経験者」に明示すると、SNS で当時の思い出語りが二次拡散の燃料になる
- 創設者離脱後の IP でも、**当時の音声・演技資産を再利用**することで作品の連続性を視聴覚レベルで担保できる
- 外部スタジオ（Virtuos のような移植・リメイク特化）に開発委託する場合、保存対象の範囲を契約レベルで明文化することで仕様ブレを抑えられる
