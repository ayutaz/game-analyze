# サイレントヒル f 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「1960 年代の山間集落・戎ヶ丘で謎と異形に対峙し、息詰まる空気そのものを潜り抜けること」である。竜騎士 07 の脚本があるため NAR の側面は強いが、本作はビジュアルノベル的に「読む」のではなく、固定カメラ的緊張・限定リソースでの戦闘・徘徊する怪異の回避といった**ホラー特有の体験圧**を主軸に置く。concept が掲げる「美の中の恐怖」は彼岸花や着物のメタファーを介して**空間の歩き方を変えさせる**仕掛けであり、これは読み物ではなく体験の側に分類される。

social_axis は「ソロ」。誰にも助けてもらえない密室性こそがホラー成立条件であり、シリーズ伝統と同じく**他者の不在**が体験の核を支える。secondary の NAR は、戦闘・探索を駆動する燃料としての物語であり、エンディング分岐や象徴解釈が周回行動を呼ぶ位置づけ。

## 体験の核 / 設計の発明

| レイヤ | 旧シリーズ（2、3 など） | サイレントヒル f での差分 |
|---|---|---|
| 舞台 | 北米地方都市・霧の街 | 戦後日本の山村「戎ヶ丘」、和様式と植物モチーフ |
| 恐怖の質感 | 工業／鉄錆／血のメタファ | 花卉・苔・腐敗した有機物の美的腐食 |
| 戦闘 | 鈍器中心の防御的近接 | 近接武器の劣化・耐久と回避リソース管理 |
| 物語担当 | 内製ライター + Akira Yamaoka | 外部脚本：竜騎士 07（ひぐらし／うみねこ） |
| カメラ／視点 | 固定 + TPS | 完全 TPS、肩越し視点で和家屋の狭隘空間を強調 |

体験の発明点は二つ。第一に、**「美しいもの」を恐怖の語彙に変換**したこと（花、白塗り、着物）。これは血と錆を増やす方向だった近年の AAA ホラーへの反証で、視覚的に怖くないはずの素材が嫌悪閾値を超える瞬間を意図的に作る。第二に、ソロ社会軸の中で**主人公・雛子の身体性（10 代の女性、農村社会の規範）**を制約として運用し、走れない・反撃しにくい・声を出しにくいといった**社会的な行動制限**を恐怖の出力に変えている。

## バズ要因 / 人気の根拠

- popularity の通り、シリーズ最速ペースで 100 万本到達、ワールド累計 2M+ に達した。シリーズ復活作（Silent Hill 2 Remake）の余波で IP の関心が温まっていた直後のローンチ。
- 竜騎士 07 起用がアナウンス段階から日本の長文考察コミュニティを動員し、配信・YouTube の考察動画が周回プレイ需要を作った。
- 1960 年代の日本農村という舞台は海外ホラーファンには新鮮で、和ホラー潮流（呪詛、八番出口など）に乗って Steam の英語圏レビューでも話題化した。
- 国内初週 PS5 で 57,475 本という数字は近年のホラー単体としては高水準で、シリーズの中核プラットフォームが Sony 機に戻ったことを示す。

## 企画への示唆

- **「怖い物」ではなく「美しい物」を怖くするとブランド差別化が出る** — 既存ジャンルの語彙を増やす方向で IP リブートを設計すると、競合と並ばずに語られる。
- **外部脚本家を IP の解釈装置として使うとコミュニティが動員される** — 竜騎士 07 のように既に長文考察文化を持つ書き手を起用すると、初週の配信動画と二次創作が連動する。
- **舞台を「異国の田舎」に置くと海外で逆輸入的に新鮮さが出る** — 和風ホラーは日本向けではなく欧米市場向けの差別化として効く。
- **主人公の社会的制約（性別・年齢・身分）を操作制約に翻訳すると恐怖の出力が上がる** — UI 上のリソース制限よりも、物語上の制約を操作に縛りつける方がプレイヤーの体感に乗る。
- **シリーズリブートは「過去作の象徴」を温存しつつ「舞台を別文化に移植」する二段構えが効く** — メカニクスは継承、視覚は刷新で旧ファン離反を抑えながら新規を引く。

## 比較で見るべきタイトル

- Silent Hill 2 Remake（2024、Bloober Team）— 同 IP の保守的リメイク、対比すると f の「文化置換」戦略が際立つ
- バイオハザード ヴィレッジ（2021）— 村落＋異形の構図、ただし戦闘比重が高く EXP 側に振れている
- ひぐらしのなく頃に業／卒（2020、竜騎士 07 脚本） — 同脚本家の物語語法、ループと共同体規範のテーマが f に流入している
- 零 〜月蝕の仮面〜（2023 リマスター） — 和ホラーソロ、家屋と女性主人公の身体性という共通項
