# ファイナルファンタジー IV (1991) — 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「**セシルたち群像が裏切りと和解を経て世界を救う物語を読み進めること**」である。同じ ATB バトル＋固定ジョブのシステムを使っても、本作で記憶に残るのは戦闘の最適化ではなく、カイン裏切り→帰還、テラ・パロム&ポロムの自己犠牲、ゴルベーザとの兄弟関係といった一連の物語イベントであり、JRPG が「キャラの内面と関係性を章立てで切り出して見せるメディア」へ転換した節目として位置付けられる。

`concept` にある「世界を救う仲間たちの裏切りと和解」というテーマと、`target` の「ストーリー重視の JRPG ファン」「Pixel Remaster / DS リメイクで往年の名作を追体験したい層」という設計意図は、いずれも物語消費を中核に置く。`social_axis` が null なのは、本作がオンライン/対戦/協力どの社会軸も持たない**完全シングルプレイの物語追体験作品**だからで、NAR の純粋系として扱うのが妥当。`secondary: ["EXP"]` は、固定ジョブ × ATB × 5 人パーティ編成という戦術レイヤーが「物語を進めるための駆動装置」として副次的に効いていることを意味する。

## 物語の駆動構造

物語消費を駆動する要素を分解すると、複数の構造が積層している。

| 構造レイヤー | 本作での具体例 | プレイヤーへの効果 |
|---|---|---|
| 主人公の内面転換 | 暗黒騎士セシル → ミシディアでパラディンへ覚醒 | 戦闘システム（暗黒剣→聖剣）と物語が同じ瞬間に切り替わる |
| パーティ流動制 | 固定ジョブの 12 人以上が章ごとに離脱・合流・戦死・帰還 | ジョブ＝キャラ＝物語役割が一致し、別れの重みが戦闘編成に直結 |
| 章立てによる視点切替 | 月の地下世界 / バブイルの塔 / 巨人 など舞台が章で更新 | 「次の章でどこへ行くか」が物語の続きを読む動機になる |
| 関係性の三角構造 | セシル / カイン / ローザ、セシル / ゴルベーザの兄弟関係 | 戦闘パーティの中に内的葛藤が常駐し、台詞が刺さる |
| 自己犠牲モチーフの反復 | テラ、パロム&ポロム、シド、ヤン などが章末で「残る」 | 「次は誰が欠けるか」という張りつめた読み体験 |

ATB（アクティブタイムバトル）はシリーズ初採用だが、本作における ATB の本質は**戦術深化よりも「物語演出のテンポを止めない時間軸」を導入したこと**にあり、ボス戦のセリフ・形態変化・撤退といった物語イベントが時間制約の中で進む構造を生んだ。

## バズ要因 / 人気の根拠

- 1991 年 SFC 初期に**ATB をシリーズ初採用**し、ターン制と即応性のハイブリッドを JRPG の標準語彙にした。
- 国内 1.44M / 世界 1.78M+ という SFC 期 JRPG として大規模な売上を達成し、シリーズ売上ランキングでも上位常連。
- パロム&ポロムの石化、テラの「メテオ」発動など、**章末に置かれた自己犠牲シーン**が当時のプレイヤーの語り草となり、後年のリメイク / 移植のたびに語り直された。
- SFC 後、PS1 / GBA / DS（3D リメイク）/ PSP（Complete Collection）/ iOS / Android / Pixel Remaster と**30 年以上にわたって移植が継続**され、世代をまたいで再評価が起き続けている。

## 企画への示唆

- **戦闘システムの転換をキャラ覚醒シーンと同期させる**と、機械的なクラスチェンジが「物語の節目」として記憶される（暗黒騎士→パラディン）。
- **パーティ枠を固定ジョブで埋める**と、ジョブ＝キャラ＝物語役割が三位一体になり、離脱イベントの体験コストが跳ね上がる（汎用ジョブ制では出せない感情）。
- **章ごとに 1 人ずつ「残す」「失う」イベントを置く**と、プレイヤーは次章を開く前に「次は誰が欠けるのか」という張りを抱え続ける（リテンション設計）。
- **シングルプレイ完結作品でも、社会軸ではなく内的関係性の三角構造を厚くする**と、SNS 時代の二次創作・考察コミュニティが自走する（30 年経っても語られる理由）。
- **ATB のような中量級システム発明は、戦術深化目的より物語のテンポを担保する目的で導入する**と、後続シリーズの標準語彙に昇格しやすい。
