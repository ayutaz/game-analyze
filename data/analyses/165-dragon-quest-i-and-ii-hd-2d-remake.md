# ドラゴンクエストI＆II HD-2D版

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ロト三部作の通史を**読み通す**」こと。III HD-2D 版（2024）で勇者ロトの起源を見届けた読者が、その続編としての I・II をひと続きの物語として消化するために起動する作品で、戦闘やレベリングは物語消費を駆動するためのテンポ装置として配置されている。

concept にある「I と II を一本に統合し、ロト伝説の通史として遊ばせる構成」が示す通り、本作は単独の RPG ではなく **III → I → II の三部作の最終巻**として設計されている。target も「40-60代の原典世代の追体験」と「III HD-2D で入門した若年層」の二層に明確に分かれており、両者の共通動機は「ロトの物語を完結させること」である。social_axis が null なのは、本作がソロ前提ながら他者軸の話題そのものを売りにしていないためで、SNS で語られるのは堀井節のセリフや三部作の伏線回収といった**物語要素**に偏る。

副次的に効くのは secondary の REW、すなわち装備・呪文・職業まわりの古典的な収集／成長ループ。ただしこれは物語のページをめくる動機にはならず、章をまたぐ間の歩幅を整える役割に留まる。

## 物語の駆動構造

本作の物語消費を駆動する構造は、三部作という外枠と、原典 2 作の素朴な王道プロットを HD-2D 演出で**再叙述**する内枠の二層で成立している。

| 駆動レイヤー | 仕掛け | 効果 |
|---|---|---|
| 三部作の外枠 | III（ロトの起源）→ I（100 年後、勇者ロトの子孫）→ II（さらに 100 年後、ロト三兄妹）の時間軸 | III を遊んだ層を I・II へ連続購買に誘導 |
| HD-2D の演出 | ドット原典＋現代ライティング・BGM オーケストレーション・カットイン | 40-60 代の「記憶の中の DQ」を上書きせず増強 |
| ボリューム増強 | 原典の数時間級プロットに、サブクエスト・章間エピソード・通史的橋渡しを追加 | 現代基準のプレイ時間（数十時間）に再調整 |
| 二作統合 | I と II を一本のパッケージにし、クリア後に II へ繋ぐ | 「IIだけ未プレイ」「Iしか覚えていない」両層をカバー |
| 堀井節の温存 | テキスト・町人の一言・選択肢の言い回し | 原典の語り口を**物語ブランド**として固定化 |

特徴的なのは「原作の素朴さを HD-2D で**翻訳し直す**」という方針で、リブートでもリイマジネーションでもない。プレイヤーは新作を読むのではなく、**自分の中の DQ 体験を上書き保存する**ためにプレイする。

## バズ要因 / 人気の根拠

popularity の「DQ原点の HD-2D 化、III 版に続く三部作完結」を分解すると：

- **三部作プロジェクトの完結**：2024 年の DQIII HD-2D 版で立ち上げた Team Asano × HD-2D × ロト三部作の枠組みが本作で閉じる構造で、III 既プレイ層の購買が約束されたパイプライン型 IP 運用。
- **原典の年齢**：I（1986）／II（1987）は発売から約 40 年経過しており、原典世代（40-60 代）の追体験需要と、若年層の歴史的教養としての需要が並走する。
- **HD-2D ブランドの定着**：オクトラ／トライアングルストラテジー／ライブアライブ リメイク／DQIII HD-2D を経て、HD-2D = Team Asano の品質保証として認知が固まった状態でのリリース。
- **マルチプラットフォーム同発**：Switch / Switch 2 / PS5 / Xbox Series / PC の同時展開で、原典をプレイした各世代のメイン機をカバー。sales_jp / sales_world は未公開だが、シリーズ累計国内最大級 IP の最新タイトルとして話題性は確保された。

## 企画への示唆

- **「三部作」をプロジェクト単位で設計すると、2 作目以降の購買が前作の余韻で自走する**：単発リメイクの売上は IP 残存価値の関数だが、三部作宣言は購買約束を作る。
- **古典 IP のリメイクは「上書き」ではなく「翻訳」にすると原典世代を敵に回さない**：HD-2D は記憶のドット絵を否定せず増強する翻訳レイヤーとして機能している。
- **原典 2 作を 1 本に統合すると単体では成立しにくい短編 IP を商品化できる**：I・II は単体では現代基準のボリュームに届かないが、合本にすることで定価商品として成立。
- **時間軸が連続する旧作群は「通史パック」として再編集できる**：ロト三部作のように物語的に繋がる作品群を持つ IP は、リリース順ではなく**物語順**で再展開する選択肢を持つ。
- **既存 IP の HD-2D 系翻訳エンジンは横展開可能なプロダクトラインになる**：Team Asano が確立した HD-2D は単発タイトルではなく**スタジオの基幹技術**として、複数 IP・複数世代を横断する。

## 比較で見るべきタイトル

- **ドラゴンクエストIII HD-2D版（2024）**：本三部作の起点。本作の購買予約装置として機能した直接の前作。
- **ライブアライブ HD-2D 版（2022）**：Team Asano による「原典の素朴さを HD-2D で翻訳する」方針の先行事例。
- **聖剣伝説3 リメイク（2020）／FF7 リメイク**：同じく国内 RPG 古典の現代基準リメイクだが、3D フルリメイクで原典を**上書き**する方針。HD-2D の「翻訳」方針との対比に使える。
- **ロマンシング サガ2 リベンジオブザセブン（2024）**：同時期の国内 JRPG 古典 3D リメイク。原典世代の追体験需要を狙う点で並走。
