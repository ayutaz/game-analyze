# スターフォックス (1993)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「コックピット視点の高速ドッグファイトで弾幕と地形を撃ち抜き、ステージ最後まで操縦し切る」体験そのものにある。物語は独裁者アンドルフ討伐というアンチェイテッドな縦軸を持つが、それは舞台装置であって、消費の中心はあくまで「アーウィンを操る数十秒〜数分の操作快感」だ。`concept` が示すとおり本作の発明は「家庭用機で前方視点ポリゴン 3D ドッグファイトを成立させた」点にあり、これは物語消費でも収集動機でもなく **操作体験の地平を更新するもの**。`social_axis` は **ソロ** で、対戦・協力・観戦の社会軸を持たず、プレイヤーは編隊メンバー（ペッピー / ファルコ / スリッピー）との **NPC 会話による疑似協力感** だけで完結する。

`secondary` は空で、ガチャ・収集・物語追体験のような副ループは設計されていない。Easy / Normal / Hard の 3 ルート分岐は「再プレイ動機」を作るが、これは報酬収集ではなく **異なる体験を試させる EXP 設計の延長**と読むのが妥当。

## 体験の核 / 設計の発明

中核メカニクスは「レール上を強制スクロールする 3D 空間で、移動・照準・ロール回避を同時に処理する」三層操作。前世代の 2D シューターが「画面内の弾を避ける」だったのに対し、本作は **奥行き軸**を導入したことで照準と回避の難度を再定義した。

| 層 | 入力 | 担う体験 |
|---|---|---|
| 移動 | 十字キー（XY 平面） | 前方視点での回避と狙撃位置取り |
| 攻撃 | B（連射）/A（チャージ） | 単発精度 vs ロックオン |
| 防御 | L+R 連打（バレルロール） | 弾反射と一時無敵、ミーム化した動作 |
| 編隊 | NPC 救援要請 | 仲間機の救出による疑似協力感 |
| ルート | クリア時の Warp ゾーン | Easy/Normal/Hard 3 経路の再プレイ動線 |

設計の発明は **(a) スーパー FX チップ搭載で SNES に専用 GPU を載せる** というハード的アプローチ、**(b) レールシューターという「3D 空間の自由度を意図的に削る」フレーミング**で操作難度をコンシューマー層に下ろした点の 2 つ。完全自由飛行ではなく軌道を固定したことで、当時の CPU/ポリゴン性能でも演出と操作快感が両立した。

## バズ要因 / 人気の根拠

- スーパー FX チップ搭載カートリッジによる **家庭用機での世界初級リアルタイムポリゴン 3D**。Wolfenstein 3D や Virtua Racing と並ぶ「3D 表現の地平を切り拓いた歴史的タイトル」と評価され、メディア注目度が極めて高かった。
- 全世界 **約 400 万本**（`sales_world: 4M`）の販売実績。SNES サードパーティ含む 3D 系の中でも上位の商業成功。
- 「Do a barrel roll!」を筆頭にペッピーの **ボイス風サンプリング台詞**がインターネット・ミーム化し、リリースから 30 年を超えた現在もネットスラングとして残存。
- フォックス・マクラウド / アーウィン / アンドルフという **シリーズ IP の原典**を確立し、Nintendo の「擬人化動物 × 硬派 SF」というブランド軸を生んだ。続編（64 / Adventures / Assault / Zero）の母体。

## 企画への示唆

- **ハード制約をコンセプトに転写する** → 自由 3D が無理ならレールに固定する、というように「できないこと」をゲーム設計の制約として正面から飲み込むと、不完全さが演出に変わる。
- **専用チップ／ミドルウェアを「タイトル専用に作る」** → 汎用ハードでは届かない表現を 1 本のために用意すると、競合との差分が物理的に再現不可能になる。Argonaut との共同開発は外部 R&D 取り込みの好例。
- **入力 3 系統を 1 ボタン×コンビネーションに圧縮する** → L+R 同時押しで回避、A 長押しでロックオンのように「複合入力で新挙動を出す」設計はコントローラ拡張をせずに表現幅を増やせる。
- **NPC 編隊で「ソロなのに孤独でない」体験を作る** → social_axis がソロでもチャットボイスで疑似マルチ感を出せる。運営型でもプレイヤー認知に効く。
- **3 ルート分岐でリプレイ動機を生む** → 報酬を盛らず「別の景色を見せる」だけで再プレイは設計できる。
- **キャッチフレーズになる短い台詞を 1 つ仕込む** → "Do a barrel roll!" のような **動作とセットで覚えられる音声**は 30 年単位で IP の認知を維持する装置になる。

## 比較で見るべきタイトル

- **スターフォックス 64 (1997)** — 同シリーズ続編。振動パック対応、マルチエンディング深化で本作の構造を拡張。
- **パンツァードラグーン (1995, SS)** — 同時期のレールシューター。背景設定で物語性を上乗せした NAR 寄りの兄弟分。
- **アフターバーナー II (1987, AC)** — レール式 3D 戦闘機シューターの原型。アーケード由来の体感シューターと比較すると、本作の家庭用化アプローチが見える。
- **Wolfenstein 3D (1992, PC)** — ほぼ同期の「家庭環境での 3D 表現」。FPS 側からの 3D 普及と対比で読むと面白い。
- **Virtua Racing (1992, AC)** — Sega の業務用ポリゴン 3D。アーケード vs コンシューマーの 3D 競争の文脈で並べると本作の位置が立体的に見える。
