# エフゼロ (F-Zero) 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「Mode 7 がもたらす擬似 3D の高速感を、コーナーごとのライン取りとブースト管理で削り込む」こと。物語ドライブもキャラ収集もなく、純粋に走行体験そのものが報酬になる構造である。

concept にある「15 コース 4 マシンというシンプルな構成」「タイム短縮の駆け引きを軸にした 1 人用レース体験」が示すとおり、本作はコンテンツ量で押し切るタイトルではなく、限られた走行素材を反復して身体化する設計に振り切られている。target も「タイムアタックや高難度コーナリングを突き詰めるソロ志向のプレイヤー向け」と明示され、social_axis が「ソロ」であることと整合する。SNES ローンチ期で対戦プレイは画面分割やリンク通信に依存するが、F-Zero は対戦モードを持たず、CPU マシンとの群れの中で「自己ベスト更新」をループさせる作りに徹している。

secondary が空であることも、ガチャ報酬や物語消費の要素が意図的に排除されていることを裏付ける。

## 体験の核 / 設計の発明

中核は (a) Mode 7 による地面の回転・拡縮で実現された「視点が前方に走り抜ける」感覚と、(b) ホバー機特有の挙動 — グリップではなくサイドアタックと壁反射で曲がる物理 — の組み合わせである。前世代のレースゲーム（スプライト並べ替えの疑似 3D）に対し、地面そのものが連続的に変形して見える点が当時の差分にあたる。

| 設計要素 | F-Zero での実装 | プレイ体験への寄与 |
|---|---|---|
| 速度表現 | Mode 7 による路面回転拡縮 | 「次世代ハード」を視覚的に体感させる |
| 車両物理 | ホバー＝低グリップ + 壁反射 + サイドアタック | グリップ走行とは別系統のライン取り技術が要求される |
| コース構成 | 15 コース / 3 リーグ / 難度段階制 | 同一コースの反復で身体記憶を積む構造 |
| 失敗ペナルティ | パワーメーター制 + リトライ制限 | 「無限コンティニュー」を許さず緊張感を維持 |
| マルチプレイ | 非対応（ソロ専用） | 競争対象を「自己ベスト」と「CPU 集団」に限定 |

social_axis が「ソロ」であることは、後年のシリーズ作（X / GX / 99）が対戦・観戦・非同期競争へ拡張していくのと対照的に、初代は「タイム」という抽象指標を媒介に他者と非同期に並ぶ構造に閉じている。

## バズ要因 / 人気の根拠

- SNES（スーパーファミコン）本体ローンチタイトルとして 1990 年に登場し、ハード黎明期に「Mode 7 = 新世代の表現力」を象徴する代表作になった。
- 全世界売上は約 2.85M 本（日本国内は個別公開なし）。ハード普及初期のソフトとして長期にわたり売れ続けた。
- キャプテン・ファルコンの初出作品であり、後年のスマブラ参戦によってキャラクター単独でも IP 拡張の起点になった。
- 同時期の擬似 3D レース（アウトラン系の体感レース）と異なり、家庭用据置機で同等以上の速度感を提供したことが、レースジャンル全体の技術期待値を引き上げた。

## 企画への示唆

- ハードの新機能（Mode 7）を「ジャンルの表現上限を上げる用途」に紐づけると、ローンチタイトルとして長期参照される。
- 対戦・物語・収集を意図的に削ぎ、走行 1 軸に絞ると、タイムアタック文化が自走しコンテンツ寿命が伸びる。
- 物理を「グリップ」ではなく「壁反射＋サイドアタック」に置き換えると、既存ジャンル経験者にも再学習コストが発生し、習熟曲線そのものが報酬になる。
- パワーメーター＋リトライ制限のような失敗ペナルティを残すと、コース反復が「練習」ではなく「挑戦」として駆動される。
- 主人公キャラ（キャプテン・ファルコン）を 1 体だけ立てて意匠化しておくと、本編がソロ主体でもクロスオーバー先で IP が再活性化する余地が残る。

## 比較で見るべきタイトル

- F-Zero X (1998, N64) — 4 人対戦と 30 台同時走行で初代の「ソロ」軸を覆した続編。
- F-Zero GX (2003, GC) — セガAM2 開発によりタイムアタック文化を最大化した到達点。
- Wipeout (1995, PS) — ホバー機レースの並走系譜、テクノ／映像演出で別方向に進化。
- マリオカート (1992, SNES) — 同じ Mode 7 / 同じ任天堂レースだが、対戦＋アイテムでソロ軸から離脱。
- F-ZERO 99 (2023, Switch) — 初代コースを 99 人非同期バトロワに転用、social_axis を「対戦」に再定義した派生。
