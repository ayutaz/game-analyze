# レンダリングレンジャー R2

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ハードの限界で殴り合う高密度な弾幕と地形を、自分の指で捌き切る」ことである。物語要素は薄く、収集や育成のメタ進行もほぼない。9 ステージを一定の難度カーブで通し切る、純粋なアクション・シューティング体験が主軸であり、`social_axis: ソロ` のとおりオフライン単独の腕試しに収束する。

`concept` にあるとおり本作は Turrican の作者 Manfred Trenz が pre-rendered グラフィックを SNES に持ち込んだ「技術デモ的アクション＋高速シューティングのハイブリッド」で、`target` も「Turrican / コントラ系の硬派な体験」を求める層と明示されている。報酬は得点でもガチャでもなく「次のステージへ進めた」事実そのもの。これは典型的な体験駆動型である。`secondary` は空で、副次動機は意図的に削ぎ落とされている。

## 体験の核 / 設計の発明

横スクロールアクションと縦/横スクロールシューティングを 1 本のキャンペーンに同居させたのが構造上の発明である。ステージごとに視点と移動軸が切り替わり、同じ「Trenz 的反射神経の設計言語」のまま別フォーマットを行き来させる。

| 要素 | 設計 | 体験への効き |
|---|---|---|
| ステージ構成 | 9 ステージ、アクションと STG が混在 | 単調化を防ぎ、同一スキルセットを別フォーマットで再使用 |
| グラフィック | pre-rendered スプライト（DKC 路線） | SNES 後期の「次世代感」を演出し技術デモの説得力を出す |
| 操作系 | Turrican 直系の多方向射撃と武器切替 | 弾を撃ち分ける思考時間が削られ、反射の純度が上がる |
| 難度曲線 | 高難度・即死多めの硬派調整 | 進行そのものが報酬となり、リテンションを腕前に依存させる |
| 配信形態 | 1995 年に日本でのみ約 1 万本生産 | 希少性そのものが後年のブランドを構築 |

`social_axis: ソロ` は「他者と比較する仕掛けを持たない」ことを意味し、リプレイ動機はスコアではなく「クリアできた／できなかった」という二値の自己評価に寄せられている。

## バズ要因 / 人気の根拠

`popularity` の「日本国内のみ約 1 万本生産」「Manfred Trenz の技術デモ級演出」「2022 年の Ziggurat / Limited Run による復刻発表」を分解する。

- 1995 年に日本国内のみ約 1 万本という限定生産で発売され、流通量の希少さ自体がプレミアソフト相場を押し上げた
- 開発者 Manfred Trenz が Turrican シリーズの作者という来歴で、欧州 Amiga／C64 系の硬派ファンが固定層として残り続けた
- pre-rendered グラフィックや多重スクロールなど SNES 末期の演出が、後年のレトロ研究系メディアで継続的に取り上げられた
- 2022 年に Ziggurat / Limited Run が Switch・PS5・PS4・PC への復刻を発表し、Nintendo Life や 4Gamer など海外・国内メディアが同時に報じてニュース化した

## 企画への示唆

- 「単一カテゴリの中で 2 フォーマットを切り替える」と、同一スキルセットを使い回しながら倦怠を抑えられる（横アク → 縦 STG など）
- 「物語・収集・育成を削って腕前以外の進行軸をゼロにする」と、ターゲットは狭まるが復刻時に強いコア顧客が残る
- 「作家性のあるクリエイターを前面に出す」と、出荷数が少なくとも長期で語り続けられる資産になる（Trenz / Turrican 系譜）
- 「技術的に攻めた表現（pre-rendered など）をハード末期に投入する」と、後年のレトロ文脈で再評価の入口になる
- 「初版の生産数を抑える」と希少性プレミアが立ち、四半世紀後の復刻発表自体がメディア露出機会に転化する

## 比較で見るべきタイトル

- Turrican シリーズ（Manfred Trenz 直系、同じ操作系の前世代）
- 魂斗羅シリーズ（同 social_axis ソロの硬派横アクション・シューティング）
- ドンキーコング国（pre-rendered グラフィック技術の同時代比較）
- メタルスラッグシリーズ（横アク＋射撃の高密度演出の参照点）
- Axiom Verge / Blasphemous（現代のソロ硬派 2D アクションの後継系譜）
