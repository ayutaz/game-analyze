## なぜ EXP 主分類なのか

プレイヤーの主動機は「16bit ハードを限界まで叩いた 2D 射撃の手触りを、短い区切りで撃ち抜き続ける」ことにある。物語上の動機（侵略者撃退）は最低限のフレーミングに留まり、収集要素もスコアもメインドライブにはならず、`social_axis: ソロ` の通り対戦・協力チャネルも持たない。つまり、面ごとに用意された敵配置・地形ギミック・乗り物セクションを処理する「操作体験そのもの」が遊ぶ理由になっており、EXP の典型である。

concept にある「Turrican シリーズの広大な探索性をいったん削ぎ、コナミ『魂斗羅』寄りのテンポの良い横スクロール射撃に振り切った続編」「バイオニックコマンドー風のグラップリングアームと多彩なビークルステージ」という設計意図も、探索メトロイドヴァニア型 EXP ではなく**ステージクリア型ラン&ガン EXP**であることを裏付ける。target に挙がる「Turrican / Contra / Metal Slug 系の硬派 2D シューターを好む層」もこの読みと整合する。secondary は空で、副次動機は明示的に切り落とされている。

## 体験の核 / 設計の発明

前作までの Turrican は「広大なマップ＋制限時間内の探索」を売りにした欧州型メトロイドヴァニアだったが、本作はその構造を畳んで `ラン&ガン / 横スクロールアクション` に振り切った。差分を要素分解する。

| 要素 | 前世代 Turrican / 同期作 | Super Turrican 2 の改変 |
|---|---|---|
| マップ構造 | 多層分岐の大型ステージ | 短尺・一方向のステージ群 |
| 主武装 | マルチディレクション射撃＋ロープビーム | 武装簡素化＋グラップリングアームで地形干渉 |
| ステージ多様性 | 徒歩中心 | ダンバギー / 水中ジェット / 落下シーン等のビークル面 |
| 演出 | Amiga 由来 BGM とパララックス | Dolby Pro-Logic サラウンド + Mode 7 拡縮・回転 |
| プレイ動機 | 探索＆隠し収集 | 区切り良いステージ消化と操作精度 |

設計の核は **「Factor 5 の技術ショーケース化」と「魂斗羅的テンポへの統合」** の二つで、ハード末期 (1995) に SNES の Mode 7 とサラウンド出力を演出資源として束ね、プレイヤーには移動・射撃・グラップリングの三系統だけを毎面少しずつ違う文脈で要求する構造になっている。

## バズ要因 / 人気の根拠

popularity の一文「Factor 5 の SNES 後期表現力を象徴する一本」を出発点に分解する。

- **技術ショーケースとしての評価**：Dolby Pro-Logic サラウンドと Mode 7 演出が、SNES 末期 (1995) というタイミングで「ハードがまだここまで動く」ことを示し、海外レトロ層の語り草になった。
- **ジャンルハイブリッドの新鮮さ**：グラップリングアーム（バイオニックコマンドー由来）とダンバギー / 水中ジェット等のビークル面を 1 本に詰めた結果、純ラン&ガンより画面の絵替わりが多く、レビュー時の話題化に寄与した。
- **メディア定着**：IGN の Top 100 SNES Games で 93 位にランクインし、欧米メディアの SNES 名作リスト系記事で繰り返し言及される定番になった。
- **長期的な再流通**：2022 年にアフターマーケットカートリッジで再販され、エミュレーション・ROM 流通だけでなく実カートリッジ需要が残っていることを可視化した。`sales_world: unknown` で当時の出荷規模は公開資料に乏しいが、再販が成立する程度のロングテール需要が観測される。

## 企画への示唆

- **シリーズ刷新時にコア体験を一本化すると、技術ショーケースを乗せやすくなる**：探索を削って射撃テンポに寄せたことで、Mode 7／サラウンド／ビークル演出の予算を 1 系統に集中できた。「広く深く」を諦めると見せ場が立つ。
- **競合の文法を一部移植するとレビュー軸を増やせる**：魂斗羅のテンポ＋バイオニックコマンドーのグラップリングを取り込んだことで、純 Turrican ファン以外の比較対象を呼び込めた。系譜の異なる文法を 1 つだけ借りる設計は、メディア記事の切り口を増やす。
- **ハード末期はカタログより演出特性で差別化する**：1995 年の SNES 市場で本作は新規 IP 訴求より「サラウンド対応」「Mode 7 演出」を前面化した。ハード末期に出すソフトは、機能対応自体を商品価値に格上げできる。
- **ステージ単位の絵替わりは記憶可視性を高める**：徒歩面と乗り物面を交互に挟むことで「ダンバギー面のあるゲーム」「水中ジェット面のあるゲーム」と語られる素材を増やせる。SNS 時代の現代企画でも、面ごとに別ゲーム化する設計は配信・クリップ映え要件と相性が良い。
- **絶版後の再販を前提にした IP 管理は資産化につながる**：2022 年カートリッジ再販のように、20 年以上経って物理メディア需要が顕在化したケースは、IP ホルダーがマスターと商標を保全していたから成立した。長寿命設計の前提条件として「権利を散逸させない」ことの価値を示す。

## 比較で見るべきタイトル

- 魂斗羅スピリッツ / Contra III (1992, SNES)：同 social_axis ソロのラン&ガンで、テンポ設計の直接の参照元。
- Turrican II: The Final Fight (1991, Amiga)：本作が削った「広大探索」型 Turrican の代表作。前世代として対比する。
- Metal Slug (1996, Neo Geo)：1 年後のアーケード作で、ビークルと徒歩を交互に挟む構成の到達点。
- Rocket Knight Adventures (1993, MD)：グラップリング系移動と高密度ステージ演出の同期作。
- Mega Man X3 (1995, SNES)：同年・同ハードのアクション 2D で、SNES 末期の演出競争を比較できる。
