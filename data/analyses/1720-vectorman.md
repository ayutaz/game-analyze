# ベクターマン (1995, BlueSky Software / Sega)

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「滑らかに動く 23 パーツの主人公を、変形を絡めながら駆け抜ける」** という運動体験そのものにある。物語は「退役した清掃ロボがオービタル・ボットの陰謀から地球を救う」と一行で語り終えられる規模で、章間カットや会話シーンによる物語消費は最小限。`social_axis: ソロ` が示すとおりオフライン 1 人プレイ前提で、ランキング・対戦・収集要素も持たない 90 年代型の純粋な腕前ゲーである。

`concept` にある「ジェット・球体・砲弾など多彩な変形アクション」と `target` の「技術デモ的なグラフィックと爽快な高速アクションを好む 2D アクション・プラットフォーマー愛好者」の組み合わせは、動かす快感をフックにした EXP の典型像と一致する。`secondary` が空なのは、収集・物語・育成といった副動機を意図的に薄め、運動と射撃の純度を保った設計判断と読める。

## 体験の核 / 設計の発明

中核メカニクスは BlueSky 独自の **「Vector Piece Animation」** と、それを活かした **モーフ変形ジャンプ＆シュート** の二層である。`Donkey Kong Country` が SGI レンダリングのスプライト差し替えで「立体に見せる」のに対し、ベクターマンは 23 個の球体パーツを関節として動かすので、走り・ジャンプ・撃ち分けが個別パーツの回転で表現され、60fps の滑らかさが見た目とレスポンスの両方に効く。

| レイヤー | 設計要素 | プレイヤー側の効果 |
|---|---|---|
| グラフィック | プリレンダ 3DCG を 23 パーツに分解 | 1 キャラの動きで Mega Drive の処理限界を体感させる「技術デモ」フック |
| 移動 | 走り＋多段ジャンプ＋無重力区間 | プラットフォーマーとしての縦軸が増え、ステージ密度が上がる |
| 攻撃 | ボディからの 360° ショット | 上下方向の脅威に対応でき、地形＋敵配置の自由度が広がる |
| 変形 | ジェット／球体／砲弾／ドリル等 | ステージ内ミニゲーム化と移動手段の交代でテンポを刻む |

`social_axis: ソロ` だからこそ、ステージ毎に変形をギミックに紐付け、アケード的な短サイクルで「次の変形を見たい」動機を引っ張れる。Mega Drive 末期に DKC へカウンターを当てるための、ハード性能の見せ場づくりが設計の起点になっている。

## バズ要因 / 人気の根拠

- `popularity` が述べるとおり、SNES 側 `Donkey Kong Country` の SGI レンダ表現に対する **Mega Drive 陣営からの技術カウンター** という文脈で語られた
- 23 パーツの関節アニメと 60fps の表示が当時の専門誌・北米 EGM 系レビューで評価され、**欧米で批評・商業ともに成功**（`popularity`）
- 国内未発売だったため日本での原体験は薄いが、後年 **Wii バーチャルコンソール（vc.sega.jp）／Sonic's Ultimate Genesis Collection／Steam Genesis Classics** など複数の再録ルートで継続的に触れられている
- 続編 `Vectorman 2`（1996）まで素早く投入された一方で、サターン以降の本格 3D 時代に主役を譲り、シリーズはレトロ枠として固定された

## 企画への示唆

- **キャラを「1 枚絵 + 数枚の差し替え」ではなく「複数パーツの関節」として組むと、同じ枚数の素材でも 1 桁多い動きの語彙が得られる**
- **競合機の上位グラフィックに対しては「同じ土俵」ではなく「自機種でしか出せない見え方」を見せる方が、ハードの晩年期にコアファンの支持を集めやすい**
- **変形＝移動モードの切り替えをステージギミックと 1:1 で紐付けると、横スクロール一辺倒の単調さを 1 ステージ単位で破れる**
- **ソロ前提のアクションでも「次の変形・次のギミック」というニンジン構造を埋め込めば、ストーリーや収集に頼らずリプレイ動機を作れる**
- **国内未発売でも、後年の再録プラットフォーム（バーチャルコンソール／クラシックス集）に乗せておくと、20 年単位でファン層が継続再生産される**

## 比較で見るべきタイトル

- `Donkey Kong Country`（1994, Rare / Nintendo, SNES）— プリレンダ 3DCG をスプライト化した同時代の競合表現
- `Earthworm Jim`（1994, Shiny Entertainment）— Mega Drive / SNES 両対応、キャラのコミカルな多段アニメで個性を立てた同世代プラットフォーマー
- `Ristar`（1995, Sega）— 同じく Mega Drive 末期、Sega 内製で「伸びる腕」というギミックで個性を作った EXP ソロ作
- `Rocket Knight Adventures`（1993, Konami）— 変形・ブースト移動を軸にした横アクションの先行参照
- `Vectorman 2`（1996, BlueSky Software / Sega）— 同シリーズ続編、変形バリエーションをさらに増やした直接比較対象
