# ベクターマン2 (Vectorman 2) 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「走って撃つ 2D アクションを、Genesis の限界に挑むビジュアル下で操作する」体験にある。物語の追体験でも、収集や育成でもなく、ステージを駆け抜けるラン&ガンの手応えそのものが目的化している。

concept で示される「3DCG 風スプライト × 走って撃つ 2D アクション」は、操作の快適さと前作からの延長で評価される設計であり、target も「Mega Drive を最後まで遊んでいたコアユーザー」「2D アクション・シューティング好き」に絞り込まれている。social_axis は「ソロ」で、対戦・協力・観戦のいずれもメインに据えられていない。プレイヤーは Vectorman の変身ギミックとステージ突破の操作を一人で味わい、その達成感が報酬になる。したがって EXP（体験型）が妥当で、NAR や REW の動機は構造的に弱い。secondary は空で、副次的に効くサブシステム（収集や物語）は明確には設計されていない。

## 体験の核 / 設計の発明

中核は「プリレンダリング 3DCG をスプライト化して 2D 平面で動かす」という Mega Drive 末期の技術的アプローチ。これによりハード性能の天井を擬似的に押し上げ、当時の同世代 2D アクションと差別化された見え方を獲得している。

| 要素 | 内容 | 体験への効果 |
|---|---|---|
| ビジュアル | 3DCG プリレンダ → 2D スプライト変換 | 同世代 2D 作品との視覚的差別化 |
| 操作系 | 走る・ジャンプ・撃つを基本に変身ギミック追加 | 単調になりがちなラン&ガンに局所的変化を持ち込む |
| ステージ構造 | 前作の延長線上で多様化 | 探索＋射撃の混成リズム |
| 投入時期 | Genesis ハード寿命の末期（1996） | 技術デモ的価値と「最後の名作」文脈 |
| social_axis | ソロ | スコアアタック・周回前提、対戦設計なし |

前作との差分は、ビジュアル路線の継承を前提に、変身ギミックやステージ設計の幅を広げた点に集約される。社交軸を「ソロ」に固定したまま、操作のバリエーションで持続時間を確保しに行く構図といえる。

## バズ要因 / 人気の根拠

- Mega Drive 末期というタイミングで投入され、「Genesis 最終世代の代表作の一つ」として後年語られた
- 前作（1995）のビジュアル衝撃を継承したため、続編としての期待値が事前に形成されていた
- プリレンダ 3DCG をスプライト化する手法が、当時の 16bit ハード上で目を引く視覚効果として機能した
- 売上規模（sales_jp / sales_world）は公開数値が乏しく、評価は商業的成功よりも「技術的到達点」「シリーズ文脈」での参照として残った

## 企画への示唆

- ハード末期に技術デモ的タイトルを投入すると、商業数字より「シリーズ文脈」「世代の象徴」として長期に記憶される
- プリレンダ 3DCG → 2D スプライト変換のような「上位技術を下位ハードに落とす」見せ方を採ると、同世代との視覚差別化が安価に作れる
- 続編はビジュアル路線を踏襲しつつ操作バリエーション（変身ギミック等）を増やすと、新規性と既視感のバランスを取りやすい
- social_axis を「ソロ」に固定する場合、周回・スコア・ステージ多様化のいずれかで反復価値を担保しないと寿命が短くなる
- ハード寿命末期にコア層へ向けた投入を行うと、新規獲得は薄いがレトロ市場での再評価で資産化される

## 比較で見るべきタイトル

- Vectorman (1995) — 前作。ビジュアル路線と操作の起点
- Earthworm Jim 2 (1995) — 同時期の Genesis 2D アクション、奇抜な演出での差別化
- Donkey Kong Country 2 (1995, SNES) — プリレンダ 3DCG スプライト化の代表例、競合ハード側の到達点
- Comix Zone (1995) — Sega 自社の Genesis 末期 2D 表現特化作
- Gunstar Heroes (1993) — Genesis ラン&ガンの系譜上にある参照点
