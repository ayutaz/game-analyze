# エイリアンソルジャー (1995 / Treasure / Sega)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「Treasure が組んだ高密度のボス戦パターンを、自分の指で解読・捌き切る」こと、つまり**戦って捌く体験そのもの**にある。物語面でも独自設定（エプシロン・イーグルの政治劇）は用意されているが、テキストはステージ間に短く挟まれるのみで、消費の中心は明確に戦闘側にある。

`concept` が言うとおり本作は「ボス戦こそが本編」と割り切った設計で、雑魚戦は移動と弾抜きの調整、本番は計 25 体のボスを連続で処理する濃縮型ラン＆ガンになっている。`target` が示すメガドラ後期のヘビー層・Treasure ファン・高難度 2D アクション愛好者という訴求は、いずれも報酬で釣る／物語で引き込むタイプではなく**操作練度を上げる過程を快とする層**である。`social_axis: ソロ` も、ボスごとに最適行動を発見していく一人称的な学習サイクルが核であることと整合する。`secondary` は空で、副次動機を持たないストイックな EXP 型と読み取ってよい。

## 体験の核 / 設計の発明

中核は「短ステージ＋ボスラッシュ」というリズム発明と、4 種類の武器スロット・通常テレポート・ゼロテレポート・パリィ気味のカウンタ動作などを束ねた**操作レイヤの厚さ**である。同時期の Treasure 作品『ガンスターヒーローズ』が複数武器の組合せで遊ばせたのに対し、本作はそれをボス戦特化に振り直し、武器切替・位置取り・ホバリングを毎瞬選ばせる構造に詰めている。

| 設計要素 | 体験への効き方 |
|---|---|
| 25 体連続ボスラッシュ | 短時間で密度が出る／学習対象が「ボス＝パターン」に純化 |
| 4 武器スロット切替 | ボスごとに有効武器を持ち替える事前設計＝戦略層 |
| 通常テレポート | 被弾回避の最終手段、しかしコスト管理を要求 |
| ゼロテレポート（接触ダメージ） | 攻撃と回避を兼ねる、Treasure らしい多義アクション |
| ステージの短さ | リトライ摩擦が低く、ボス研究サイクルが速い |
| 「FOR MEGADRIVERS CUSTOM」 | 難度を客観化せず、ターゲット層に向けた“宣言”として機能 |

social_axis がソロでも体験密度が落ちないのは、対戦相手の役割を**ボス AI の手数とパターン量**が肩代わりしているためで、後の Boss Rush ジャンルが踏襲する方法論の原型になっている。

## バズ要因 / 人気の根拠

- メガドライブ末期（1995）に投入された Treasure 作品として、ハード愛好家のあいだで「最後にして最高難度の弾」という位置付けで語られ続けている。
- パッケージ裏の「FOR MEGADRIVERS CUSTOM」という煽り文句が、難度の言い訳ではなくユーザー層宣言として機能し、コア層の所有欲を刺激した。
- Wii バーチャルコンソール、メガドライブミニ 2、Nintendo Switch Online（メガドライブ）への収録で再頒布経路が確保され、世代を跨いだ追体験が断続的に発生している。
- 出荷本数自体は小規模で `sales_jp: —` / `sales_world: unknown` ながら、Treasure の代表作群（ガンスターヒーローズ／罪と罰／斑鳩）の系譜を語る文脈で必ず参照される“批評上の指標タイトル”として残り続けている。

## 企画への示唆

- **「本編」だと思っている要素を捨てると密度が出る**：道中をデフレさせボスに資源を全振りすると、短時間で覚える価値のある体験になる。
- **難度を謝らず、ターゲット宣言で囲い込む**：「FOR ◯◯ CUSTOM」式のコピーは、フィットしない層を最初に弾くことでコア層のロイヤルティを高める。
- **多義アクションは“攻撃＝回避”の重ね掛けで作る**：ゼロテレポートのように 1 ボタンに複数意味を持たせるとプレイヤーの判断密度が跳ね上がる。
- **ボスごとに最適解を分散させると武器スロットが生きる**：装備選択が事前戦略になり、リプレイ時の発見が増える。
- **再頒布経路（VC／ミニ機／サブスク）を長期で確保できる作品は、初動売上が小さくても評価資産が積み上がる**：移植可能なデータ規模で作っておくこと自体が長期 IP 戦略になる。

## 比較で見るべきタイトル

- ガンスターヒーローズ（Treasure / 1993, Mega Drive）— 多武器組合せの原型、本作の前提となる Treasure 文法。
- 罪と罰 〜地球の継承者〜（Treasure / 2000, N64）— ボス戦密度とカメラ演出の発展形。
- Cuphead（Studio MDHR / 2017）— ボスラッシュを中心動機に置く現代的後継。
- Furi（The Game Bakers / 2016）— 1 対 1 ボス連戦の純化、social_axis ソロ・EXP の同型例。
- 斑鳩（Treasure / 2001）— 武器切替の代わりに属性切替で“多義アクション”を更新した同系譜作。
