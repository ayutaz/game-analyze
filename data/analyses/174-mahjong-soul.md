# 雀魂 -じゃんたま- / Mahjong Soul

## なぜ EXP 主分類なのか

主動機は「他人と段位を競って麻雀を打つ」こと。プレイヤーが画面の前に座る理由は、四人で卓を囲み雀豪・雀聖・魂天と段位を上げていく対戦体験そのものにある。

concept にある「萌え系キャラクター・ボイス・スキンと本格的なルール・段位戦を融合」という設計は、麻雀という極めて成熟したルールセット（日本リーグルール準拠）を base に、Yostar が美少女 IP 運営で培った長期運営ノウハウを被せる構造になっている。target が示すとおり「段位戦で腕を競う層」と「キャラ・スキン収集層」が同じ卓で同居するが、卓上での勝敗ロジックは麻雀のままなので、競技性は希釈されない。

`secondary: ["REW"]` は、キャラ・スキン・立直棒などのガチャ／収集レイヤーがリテンションを支えていることを意味する。ただし REW 要素は対戦結果に直接干渉せず（pay-to-win ではない）、主動機はあくまで「打つ」側にある。

## 体験の核 / 設計の発明

伝統的な麻雀ゲームと雀魂の差分は、「対戦の物理（4 人卓・段位・牌譜）」と「F2P モバイルの運営文法（ガチャ・コラボ・ボイス）」を**同一クライアント内で接続した**点にある。前世代の「ネット麻雀（天鳳・MJ）」は競技性に振り切り、家庭用麻雀ソフトは IP コラボに留まっていた。雀魂はその間を埋めた。

| レイヤー | 担当する機能 | 既存麻雀ゲームとの差分 |
|---|---|---|
| ルール層 | 段位戦（銅／銀／金／玉／王座／魂天）、四麻・三麻 | 天鳳の段位設計を踏襲、ただし UI が現代的 |
| 演出層 | 役満ボイス、キャラ別立直演出、卓上スキン | 萌え系 IP を入れたことで「打つたびに見たい絵」が増える |
| 収集層 | キャラガチャ、スキン、立直棒、家具 | 麻雀ゲームに本格的な F2P 運営文法を持ち込んだ |
| 接続層 | クロスプラットフォーム（iOS / Android / PC） | 同一アカウントで通勤中スマホ → 帰宅後 PC が成立 |

social_axis=対戦 の効果として、「自分の段位ポイント」と「相手の振る舞い」が常に観測対象になるため、配信映え（後述）と相性がよい。

## バズ要因 / 人気の根拠

- popularity が示す通り、Vtuber／配信者の対局配信が拡散装置になった。にじさんじ・ホロライブ系を中心に「麻雀部」的なグループ対局企画が継続的に組まれ、視聴者が「同じ卓を体験するため」にダウンロードする導線が確立した。
- M リーグ（プロ麻雀リーグ）放映に合わせて雀魂内でコラボ・キャンペーンが展開され、競技麻雀ファンと F2P ガチャ層が同じ卓に流入。日本国内の麻雀番組視聴文化と直結している。
- 登録 1000 万超（全世界）。日本は publisher Yostar の主戦場で、`咲-Saki-` `アカギ` 『MJ』『天鳳』など麻雀コンテンツに親和性の高いユーザー基盤がそのままスライドした。
- 基本無料 + 課金は外見要素中心という設計上、競技勢が「ガチャゲーに金を払う層」と対戦することへの忌避感が薄く、両層が共存できた。

## 企画への示唆

- 成熟したルールゲーム（麻雀・将棋・チェス・ポーカー）に「F2P 運営文法」を被せると、ルール設計コストを払わずに長期運営できる。
- 競技性に干渉しない課金（スキン・ボイス・演出）に絞ると、競技勢の流出リスクを抑えながら収集勢を取り込める。pay-to-win を避けることが両層共存の前提条件になる。
- 配信映えする「演出層」（役満ボイス・キャラ立直演出）を分離して実装すると、ゲーム内の同じ局面が毎回違う絵になり、配信視聴者の「自分も打ちたい」モチベを引き出せる。
- IP コラボ（咲、アカギ等）を運営イベント化すると、対戦ゲームでありながらガチャゲーと同じカレンダー型運営に乗せられる。
- クロスプラットフォーム（スマホ／PC 同一アカウント）は、競技勢が PC、ライト層がスマホという二層が同じ卓に座るための必須条件として機能する。
