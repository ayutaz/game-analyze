# Fate/stay night (2004 / TYPE-MOON)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「聖杯戦争という伝奇世界をテキストで読み切る」こと、つまり**読む**である。ゲームメカニクスは選択肢式 ADV のシンプルな分岐に限定され、戦闘パートはコマンドではなくテキスト演出として処理される。「面白い」と語るプレイヤーが言及する対象は、ほぼ常にシナリオ・キャラ造形・台詞回しであり、操作スキルや収集進捗ではない。

concept にある「Fate / UBW / HF の 3 ルートで世界観と主人公の倫理観が大きく変質する多層構造」が示す通り、本作の中核価値は**ルートを跨いで再構築される物語体験**にある。target も「物語消費・キャラクター愛着重視のプレイヤー」と明示されており、伝奇 ADV というジャンル定義と合わせて NAR 主分類が素直に成立する。secondary が空なのは、ガチャや育成といった REW 要素、対人や非対称マルチといった EXP 要素が一切実装されていないためで、純粋な物語消費型として分類が単純化される。

## 物語の駆動構造

伝奇ビジュアルノベルとしての駆動装置を分解する。

| 構造要素 | 具体内容 | 物語消費を駆動する効果 |
|---|---|---|
| 3 ルート並走 | Fate (セイバー) → UBW (凛) → HF (桜) の固定読了順 | 同じ世界観を 3 つの倫理観で再走査する「読み直し」体験 |
| ヒロイン別解釈 | 主人公・士郎の正義観がルート毎に問い直される | キャラ攻略ではなく**世界観の多重露光**として機能 |
| バッドエンド群 | TIPS による補足解説付きの多数の死亡分岐 | 失敗もテキストの一部として消費資産化 |
| Réalta Nua 再構成 | 18 禁要素除去・新規 BGM/CG 追加で家庭用展開 | 1 シナリオを 20 年・5 プラットフォームに延伸 |

Fate→UBW→HF の固定順は、後ルートが前ルートの結末を裏面から読み解く設計になっており、選択肢の自由度より**読了順の制約**が物語駆動の中核を担う。これは同時期の Key 作品（Kanon、CLANNAD）が並列ヒロインルートを採るのと対照的で、シナリオの順序自体が体験設計の一部となっている。

## バズ要因 / 人気の根拠

popularity の「TYPE-MOON 奈須きのこの伝奇ビジュアルノベル代表作」を出発点に分解する。

- **20 年継続の IP プラットフォーム化**: 2004 PC 初版 → 2007 PS2 Réalta Nua → 2024 REMASTERED と展開し、累計 750K+ 規模に到達（PC 初版 ~400K + 家庭用累計）
- **派生展開によるリーチ拡張**: Fate/Zero 小説・アニメ、UBW / HF 劇場版アニメ、Fate/Grand Order モバイルが原作 VN への参照需要を継続生成
- **REMASTERED の数字的裏付け**: 2024 年 12 月時点で REMASTERED が 100K 本超を記録（Gematsu 報）、20 年前の VN として異例の再立ち上げ成功
- **3 ヒロインルートの語り口の多様性**: 王道騎士道譚 (Fate) / 自己実現劇 (UBW) / 倫理崩壊サスペンス (HF) と読了体験が大きく異なり、ルート毎に独立した語り対象を生んだ

## 企画への示唆

- **読了順を固定すると世界観の多重露光が成立する**: 自由分岐より制約付き順序が、後ルートで前ルートを裏読みする快楽を生む（→ シナリオ密度の体感が増す）
- **ルートを別ヒロインではなく別倫理観に紐付ける**: 「攻略対象が変わる」ではなく「主人公の信念が試される」設計にすると、各ルートが独立した思想実験として機能
- **失敗分岐に TIPS を載せると消費資産になる**: バッドエンドを単なる game over でなく世界観補足の場にすると、プレイヤーは「全 BE 回収」を目的化する
- **18 禁版とコンシューマ版を分けて 20 年延伸する**: 初版で濃いコア層を獲得し、再構成版で家庭用・モバイル・REMASTERED と段階的にマス化する展開モデル
- **派生 IP が原作 VN への需要を生む循環**: アニメ・小説・スマホゲーが原作参照を駆動するなら、原作はメディア展開を前提とした「設定の深さ」を初期から仕込むべき

## 比較で見るべきタイトル

- **月姫 (TYPE-MOON, 2000)** — 同作者・同社の前作 VN、伝奇 ADV としての設計原型
- **Fate/Grand Order (2015)** — 同 IP の REW 派生、VN 物語消費がガチャ収集に変換された事例
- **CLANNAD (Key, 2004)** — 同年発売の並列ヒロインルート型 VN、駆動構造の対照例
- **シュタインズ・ゲート (5pb./Nitroplus, 2009)** — 単一主軸 + 分岐収束型の伝奇 ADV、ルート設計の別解
- **428 〜封鎖された渋谷で〜 (Sega/Chunsoft, 2008)** — 群像劇 + 強制読了順の構造的近縁作
