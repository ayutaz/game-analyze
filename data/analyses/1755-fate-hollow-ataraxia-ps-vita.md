# Fate/hollow ataraxia PS Vita版 — 分析

## なぜ NAR 主分類なのか

このタイトルにおけるプレイヤーの主動機は「本編後の続きを読む／キャラクターたちの“その後”を追体験する」ことに集約される。戦闘や報酬獲得は副次で、テキストとボイスを読み進めること自体が消費活動の中心になる。

concept にあるように、本作は『Fate/stay night』本編から約半年後の冬木市を舞台に「繰り返される 4 日間」を描く伝奇ヴィジュアルノベル。target も「原作 PC 版・PSP 版を遊んだシリーズ既存ファン」「ufotable アニメから入った 20-40 代のノベルゲーマー」と、明確に物語消費層に絞り込まれている。プレイヤーが対峙するのは選択肢分岐の最適化ではなく、本編で死亡した／別ルートに分かれたサーヴァントたちが「日常」を共有する光景そのものであり、これは NAR の中でも「キャラクター関係性の事後譚」を駆動力にする副ジャンルだ。

secondary の EXP は、同梱ミニゲーム『カプセルさーばんと』（簡易タワーディフェンス）と、Vita のタッチ操作で行うミニコンテンツ要素を指す。読み物の合間に挟まる軽い体験レイヤーで、本編進行のリズム調整役にとどまる。

## 物語の駆動構造

ファンディスク型ノベルが「読ませ続ける」ために用いる構造を分解する。

| 構造レイヤー | 本作での実装 | 駆動効果 |
|---|---|---|
| 時間軸の縛り | 「繰り返される 4 日間」というループ設定 | 短いスパンで複数キャラの日常を回せる／本編の重さを回避 |
| キャラ供給 | 本編で退場したサーヴァントが全員“居る” | 死亡ルートを通過したファンへの補填として強い動機 |
| 解放トリガー | 日付・場所・キャラ条件のフラグ収集 | プレイヤーに「全部読む」目標を与え、線形読書から探索読書へ |
| 周辺コンテンツ | カプセルさーばんと、グロッサリ、CG モード | テキスト消費の合間に置く小さな到達感 |
| 移植固有の付加 | ufotable 制作の新規 OP、フルボイス、全年齢化 | 既プレイ層に「もう一度買う理由」を提供 |

ループ世界の中で「謎」が伏線として走り、シリアスパートが日常の裏で進行する二層構造になっているため、ギャグ・日常・本筋の三種テキストを同じセッション内で混在させても破綻しない。これは本編ルート分岐型のシナリオ設計よりも、後発のファンディスク／側面譚に向いた骨格と言える。

## バズ要因 / 人気の根拠

- 発売初週で約 **53,979 本**（JSON sales_jp）を売り上げ、2014 年の Vita 向けノベルゲームとしては上位帯に着地。PSP 版・PC 原典を持つ既存ファンが二重・三重に購入した構図。
- ufotable 制作の新規オープニング映像が公開時から SNS で拡散し、アニメ『Fate/Zero』『stay night [UBW]』の視聴層を Vita 移植側に流し込んだ。
- 初回特典に Nendoroid ぷち（グッドスマイルカンパニーのフィギュア）が付属し、フィギュア収集層 × ノベル層のクロス購入を誘発。
- 同梱ミニゲーム『カプセルさーばんと』が「本編とは別腹で遊べる」副コンテンツとして、ファンディスクの“お得感”を可視化した。
- 角川ゲームスが PS Vita のノベル流通網（限定版 BOX、特約店特典、メディアミックス）に乗せたことで、TYPE-MOON 直販ではリーチしない一般ゲームショップ流通に届いた。

## 企画への示唆

- **本編で“失われたキャラ”を全員生かす舞台を別建てで用意すると、分岐ルートで取りこぼされたファンの未消化感を回収できる。** 続編より前日譚／側面譚／ループものの方が、本編プレイヤー全員に均等に刺さる。
- **「繰り返される短期間」を物語舞台にすると、シナリオ供給コストが下がる。** 同じ場所・同じ NPC を使い回しながら別シーンを足せるため、ファンディスクのテキスト量当たり制作費が低い。
- **副ジャンル（ミニゲーム）を同梱すると、ノベル本体の長尺消費に“達成感の挿入点”ができる。** リテンションが純テキストだけより安定する。
- **既プレイ層に再販する際は、フォーマット差分（フルボイス／全年齢／新規 OP／新規 CG）を 3 点以上束ねる。** 単一の差分では二重買いは起きにくい。
- **アニメ化された IP のゲーム移植は、アニメ制作スタジオの新作映像（OP）を販促資産として再利用すると CV が伸びる。** 別チャネルで形成されたファン層を本作に流す導線になる。
