# Fate/hollow ataraxia Remastered

## なぜ NAR 主分類なのか

プレイヤーの主動機は「『Fate/stay night』本編の後日譚を**読む**」ことに尽きる。本作はジャンルとしては伝奇ビジュアルノベルであり、戦闘や育成のシステムレイヤーは原作 PC 版（2005）の枠組みを継承した既読スキップ＋選択肢分岐のテキスト体験で、攻略によって獲得する「報酬」は新シナリオの解放と CG・用語辞典の埋め込みに限定される。concept にある「約束の四日間を再体験する」というフレーズが象徴的で、ループする日常を巡り直しながら本編で死んだキャラクターを含む全サーヴァントの後日譚を読み進めること自体がゲームの全てに近い。

target は (a) 原作既読の old Fate ファン、(b) FGO から入って原典に遡りたい層、(c) 公式英訳を初めて読める海外読者の三層で、いずれも「物語の続きを読みたい」が動機。social_axis は null で対人要素はなく、純粋なソロ読書体験として設計されている。secondary が空なのは、収集要素（CG / イベント絵）が物語消費に従属する補助項目に過ぎないため。

## 物語の駆動構造

『hollow ataraxia』はシリーズ正典ではなく、本編三ルート（Fate / UBW / Heaven's Feel）の後に置かれた**ファンディスク的アフターストーリー**である。テーマは「閉じた四日間のループ」で、原作の死者を含む全キャラクターが日常に再集結する構造を持つ。Remaster 版で再構築された駆動構造を要素分解すると以下のようになる。

| 構造要素 | 中身 | 物語消費への寄与 |
|---|---|---|
| ループ日常パート | 「約束の四日間」が反復し、選択肢で各キャラの日常エピソードに分岐 | 短編 70+ 本を「ばら撒き」、20 周年で再訪する満足感を担保 |
| タイガー道場 / おまけシナリオ | 既存 FD 同様のコメディ短編 | シリアス本編の緊張を緩めるリズム調整 |
| メインプロット（バゼット / アヴェンジャー線） | ループの謎を解く縦軸 | 全短編を読了させる「読破動機」 |
| HD リマスター + 英・簡中翻訳 | 解像度引き上げ + 公式ローカライズ | 既読層の「もう一度読む」理由と海外新規層の参入導線 |
| 武内崇描き下ろしメインビジュアル / FGO 連携 | 20 周年記念のメタ装飾 | 物語外で「読む決心」を後押しする発火点 |

縦軸（プロット）と横軸（キャラ別短編）が二重に走り、横軸の消化率がプレイヤーの体感ボリュームを膨張させる典型的なファンディスク設計を、20 年越しに翻訳・解像度・配信プラットフォームの三点でアップデートした構造になっている。

## バズ要因 / 人気の根拠

- **Fate/stay night 20 周年記念タイトル**として 2025 年に発表され、シリーズの節目イベントの中核としてポジショニングされた。
- **2014 年 PS Vita 版以来初**となる『hollow ataraxia』のコンソール再販で、Switch / Steam（Steam App ID 2396990）という現行主要プラットフォームに初到達。
- **英語・簡体字に公式対応した初の hollow ataraxia** という訴求が海外 Fate コミュニティに刺さり、Gematsu・4Gamer・Famitsu など国内外メディアが同期して報じた。
- **武内崇による描き下ろしメインビジュアル + FGO 連携キャンペーン**で SNS 上の発火点を作り、原作未読の FGO ユーザー層への遡及流入を狙う構造。

## 企画への示唆

- **20 周年などのアニバーサリーに「未翻訳の傍流作品」を初公式翻訳で出すと**、既存ファンの再購入 + 海外新規層の初購入を二重に獲得できる。長寿 IP のリマスター戦略として極めて費用対効果が高い。
- **本編より外伝・FD を先に英訳すると**、本編未読層に「先に原典を読みたい」という遡及需要を生み、後続のリマスター原作の販売アシストになる。
- **既存 IP の現役モバイル版（FGO）と連携キャンペーンを敷くと**、現役ガチャ層を「20 年前の原典」へ誘導でき、IP の世代継承装置として機能する。
- **VN リマスターでは解像度より「現行ハードで遊べる」「現代言語で読める」の方が販売トリガー**になる。Vita で止まっていた作品を Switch + Steam に再着地させる動線は、テキスト中心ジャンルの標準解。
- **新規イラスト 1 枚 + 開発元の象徴的クリエイター名（武内崇）を前面に出すと**、メディア露出時のサムネイル占有率が上がり、SNS 上の「絵で語れるニュース」になる。

## 比較で見るべきタイトル

- **月姫 -A piece of blue glass moon-（2021）** — TYPE-MOON 旧作の Switch / PS4 リメイクで、20 年越し再構築の比較対象。
- **CLANNAD / リトルバスターズ! コンソール版** — Key 作の VN を Switch / Steam に展開し海外流入を取り込んだ事例。
- **STEINS;GATE ELITE（2018）** — フルアニメ化リマスターで「既存ファン再購入 + 新規」を狙った同世代 VN リマスター。
