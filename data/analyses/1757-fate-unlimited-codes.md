# Fate/unlimited codes 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「Fate のサーヴァント同士を 3D アリーナで自分の手で戦わせる」** という対戦体験そのものである。原作『Fate/stay night』が PC ノベルで提示してきた「英霊バトルロワイヤル」を、操作可能な格闘ゲームに置換したのが本作のコア提案であり、購入動機はストーリー再消費ではなく "セイバー vs ギルガメッシュを自分の入力で実現する" 体験に寄っている。

`social_axis: 対戦` が示すとおり、設計の重心はアーケード筐体での対人戦と PSP 版のアドホック通信対戦に置かれている。`concept` にある「もしあの英霊同士が戦ったら」の実演は、原作世界で実際にはほとんど直接対決しなかった組み合わせ（例：ライダー vs アーチャー）を プレイヤー入力で生成する装置で、CPU 戦シナリオ・対人戦のどちらでも成立する。`secondary: NAR` はキャラクターストーリーモードや原作再現の宝具演出で物語愛を補強するレイヤーとして機能しているが、購買と継続の主動機ではない。

## 体験の核 / 設計の発明

3D アリーナ格闘 × 原作再現 × クロスオーバー IP という三層構造を、開発・販売・原作の分業で実装したのが特徴である。

| レイヤー | 担当 | 設計上の役割 |
|---|---|---|
| 格闘システム | Eighting | 3D 軸移動アリーナ、宝具ゲージ、原作再現必殺技 |
| 企画・原作監修 | Cavia / TYPE-MOON | キャラ性能を原作スペックに寄せる「英霊バランス」 |
| 販売・アーケード流通 | Capcom | 格ゲー筐体ネットワークと家庭用販路 |
| プラットフォーム展開 | Arcade → PS2 → PSP | 筐体対戦 → 家庭用据置 → 携帯機通信対戦 |

前世代の Fate 派生ゲーム（同人格闘 Melty Blood は型月別作品、Fate/tiger colosseum はパロディ路線）と比較すると、本作は **「本編キャラを真面目な対戦格闘で動かす初の公式作」** である点が差分になる。3D アリーナ採用は Capcom 流の 2D 格闘文法ではなく Eighting が『機動戦士ガンダム』系で培った 3D 軸ずらしの応用で、宝具ゲージとカットイン演出を「ここぞの一発」に集約する設計が原作の必殺技演出と接続している。

## バズ要因 / 人気の根拠

- アーケード稼働時、TYPE-MOON 原作の人気ピーク期と重なり、店舗対戦に Fate ファン層と既存格ゲー層の双方が流入したことが店内対戦の盛り上がりを牽引した
- PSP 版はアドホック通信対戦に対応し、家庭用ファン層が「キャラ愛で触る → 通信で対戦に流れる」動線を作った（`sales_jp`: PSP 版初週 3.4 万本）
- Cavia 企画 / Eighting 開発 / Capcom 販売という、IP・開発・流通を分けたクロスオーバー座組み自体が当時の格ゲー業界で珍しく、話題化に寄与した
- 原作の宝具やセリフをそのまま技演出に乗せる「再現度」がファン層の SNS・動画文化（ニコニコ動画期）で拡散素材として機能した

## 企画への示唆

- **ノベル / アニメ原作を対戦格闘に翻訳するなら、3D アリーナ + 宝具ゲージで「ここぞの一発」を演出に集約**すると、原作の必殺技文化と格ゲーのリソース管理が同じ UI に乗る
- **原作監修・開発・販売を分業**すると、IP ホルダーが演出と性能監修に集中でき、開発スタジオは格闘システムに集中できる（本作の Cavia / Eighting / Capcom 構成）
- **アーケード → 据置 → 携帯機の順次移植**は、対戦コミュニティを店舗から家庭、家庭から外出先へ段階的に拡張できる流路設計
- **クロスオーバー型タイトルは "原作で実現しなかった組み合わせ" を売る**と、ストーリーを既に消費したファンに新規購買理由を提供できる
- **PSP のアドホック通信対戦**のような「同じ場所にいる仲間と無料で対戦」装置は、原作ファンが格ゲー入門する障壁を下げ、コミュニティ形成のシード機能を持つ

## 比較で見るべきタイトル

- **Melty Blood Act Cadenza / Actress Again**（フランスパン × TYPE-MOON、2D 格闘で同じ「型月キャラを対戦に落とす」課題に別解を出した同人発の系譜）
- **機動戦士ガンダム VS. シリーズ**（Eighting / Capcom の 3D 軸ずらしアリーナ格闘の系譜、IP キャラを 3D 空間で動かす設計の同根）
- **JoJo の奇妙な冒険 未来への遺産 / All-Star Battle**（少年漫画キャラの宝具的必殺技を格闘に翻訳する近接事例）
- **Dissidia Final Fantasy**（Square Enix、PSP、IP オールスター × 3D アリーナ、同時期の競合提案）
