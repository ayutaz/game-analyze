# Fate/tiger colosseum UPPER (2008, PSP)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「Fate サーヴァントの SD キャラを操作して、相手と殴り合う」こと。すなわち戦う体験そのものが目的であり、social_axis は対戦に置かれる。3D アクション格闘という枠組み上、勝敗・キャラ操作の手応え・組み合わせの引き出しが遊びの中心で、原作のシリアスな聖杯戦争を**ギャグタッチに崩した上で対戦土俵に乗せる**という設計判断が、EXP として成立させる前提になっている。

ただし、本作の購買動機は純粋な格闘ゲーマーではなく Fate ファンであり、target にも「原作・アニメファン」と明記される。掛け合いボイス・SD 化表現・パロディシナリオは強い副次レイヤーで、secondary に NAR が入るのはそのためだ。「対戦の上に張られた、原作キャラの再演を読む薄い物語層」が共存している構造である。

## 体験の核 / 設計の発明

| レイヤー | 仕掛け | 効果 |
|---|---|---|
| 操作 | 3D 対戦アクション（簡易コマンド + 必殺技） | 原作未プレイの Fate ファンでも触れる難易度に圧縮 |
| キャラ表現 | サーヴァント / マスターの SD 化 | 原作のシリアス性を脱臭し、ギャグ世界に同居させる名分を作る |
| ロスター | UPPER で追加キャラ + hollow ataraxia / Fate/Zero 系を統合 | 前作不参戦キャラへの不満を吸収し、続編購入動機を作る |
| 物語層 | 藤村大河（タイガー）が主催する非公式トーナメント | 原作の聖杯戦争と衝突しない並行世界として、何でもありの対戦を正当化 |
| 対戦軸 | ローカル対人 + ストーリー / 対 CPU | PSP の携帯機文脈で、自宅対人と一人プレイの両立 |

設計の発明は単一メカニクスではなく、「**重い IP をギャグ枠に逃がしてから対戦土俵に持ち込む**」という枠組み変換そのものにある。Fate 本編のシナリオ重力を温存したまま格闘ゲームを作ることはできないが、タイガー道場ノリの非公式大会という外枠を用意することで、世界観矛盾を回避しつつ全キャラ参戦を実現している。

## バズ要因 / 人気の根拠

- 前作『Fate/tiger colosseum』(2007) が TYPE-MOON 公認の SD 化総出演という稀少な座組で話題化し、UPPER はその続編として追加キャラ・新ストーリーで Fate コミュニティが再点火した
- 配信が PSP 期に当たり、ローカルアドホック対戦という当時のソーシャル動線にハマった
- hollow ataraxia / Fate/Zero 由来キャラの参戦が、原作横断の「夢の対戦カード」需要に応えた
- 売上数値は非公開（sales_jp / sales_world ともに「—」）で、商業的ヒットというより**ファンサービス作品として記憶に残っているタイプ**

## 企画への示唆

- 重厚なシリアス IP を対戦・ミニゲーム枠に展開するとき、**SD 化 + ギャグ外枠**を被せると本編との世界観衝突を吸収できる
- 「公式・正史」ではなく「非公式トーナメント」「並行世界」「夢オチ」のメタ装置を据えると、原作で死亡 / 退場済みのキャラも全員ロスター化できる
- 続編で「前作未参戦キャラ + 新作 IP 由来キャラ」を一気に投入すると、前作購入層・新規 IP ファンの両方に買い替え動機を作れる
- 携帯機格闘は**ローカル対人前提のロスター充実度**が口コミ駆動因子になる（自宅で 1 台あれば集まる文脈）
- IP コラボ格闘の主動機は「キャラを動かす権利」であり、対戦バランス調整より**掛け合いボイス・勝利演出の物量**が満足度を左右する

## 比較で見るべきタイトル

- 『Fate/unlimited codes』(Capcom, 2008 AC / PS2 / PSP) — 同年・同 IP・SD ではない本格 2D 格闘という対照軸
- 『ジャンプスーパースターズ / アルティメットスターズ』(任天堂 DS) — 重 IP × SD 化 × パーティ格闘という同型の枠組み
- 『電撃文庫 FIGHTING CLIMAX』(SEGA, 2014) — ライトノベル横断のオールスター格闘という後継例
- 『大乱闘スマッシュブラザーズ』シリーズ — 「シリアス IP を対戦土俵に乗せる」設計の総本山
- 『Fate/EXTELLA』(Marvelous, 2016) — 同 IP の後年アクション展開、こちらは無双系で SD ではない本格路線
