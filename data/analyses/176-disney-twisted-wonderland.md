## なぜ REW 主分類なのか

プレイヤーの主動機は「ディズニーヴィランズをモチーフにした男性キャラを集めて、SSR を引き、育てて愛でる」こと。リズムバトルと章立てシナリオはあくまで配布カードとイベント SSR を消費する場であり、課金圧の核は**ガチャによるキャラ獲得**にある。

JSON の concept にある「枢やな（原案）×ディズニー×アニプレックス運営の女性向けハイブリッド」は、IP 二層（ディズニー原作キャラ＋日本側オリジナル寮生）でガチャ供給を持続させる構造を意味する。target が「ディズニー／乙女／推し活 SNS 文化に親しむ 10–30 代女性」と明示されている通り、推しキャラの新衣装 SSR を引く動機がそのまま課金行動になる。secondary に NAR が入るのは、章立てメインストーリーとイベント書き下ろしが「次のガチャを引く理由」を生産し続けるためで、物語そのものの完結より**キャラ供給を維持する燃料**として機能している。

## 報酬ループの組成

| 層 | 報酬の中身 | 駆動するもの |
|---|---|---|
| ガチャ層 | 衣装替え SSR（学園服／式典服／カード／オバブロ衣装） | 推しキャラの「新しい姿」需要 |
| 収集層 | 7 寮×複数キャラ × 衣装バリエーション | コンプ欲・寮単位の推し活 |
| 育成層 | 魔法レベル・覚醒・バディボーナス | SSR の戦闘性能化 |
| 物語層 | メインストーリー章 / イベント書き下ろし / ボイス | ガチャ実装の動機づけ・SNS 共有素材 |
| 行動層 | リズムバトル・授業（スタミナ消費） | 日次ログイン・素材回収 |

リテンションは「メインストーリー新章 → 新キャラ登場 → ピックアップガチャ」のサイクルで回り、加えて季節イベント（ハロウィン／クリスマス／バレンタイン／周年）で衣装 SSR を投下する二重構造。授業（スタミナ消費）とリズムバトルは素材を絞る関門であり、課金により完全に飛ばすより「育成途中」を維持する設計になっている。

## バズ要因 / 人気の根拠

- **ディズニーヴィランズ × 男性キャラ運営**という、それまで国内女性向けモバイルが触れていなかった IP 領域を独占的に押さえた
- 国内 1000 万 DL 超（公式公表）、Sensor Tower 推定で月商 $3M+ クラスを継続している国内女性向け運営の上位常連
- 枢やなのキャラデザインと寮ごとの濃い世界観設定が、SNS 二次創作（イラスト／SS／コスプレ）の供給を継続させ、ゲーム外で話題が再生産される
- アニメ化（2024 配信開始）／グッズ／カフェ／舞台等のメディアミックス展開が広告費を肩代わりし、新規流入の動線を維持

## 企画への示唆

- **強 IP × オリジナル設定**で組むと、原作改変の制約を回避しつつ「公式が用意した二次創作的フォーマット」が成立する（→ファン創作の燃料になる）
- ガチャ供給を「衣装替え軸」に置くと、キャラ総数を増やさずに SSR を発行し続けられる（→開発負荷を抑えて運営寿命を伸ばせる）
- 寮／部活／クラス等の**グルーピング構造**を入れると、推しの単位が複数（個人推し／寮推し）に増え、課金トリガーが多層化する
- 物語章の更新と新規ガチャを連動させると、ストーリー消費そのものが課金導線になる（NAR を REW の燃料に転化）
- SNS 二次創作を許容するガイドラインを早期に整備すると、広告費を払わずに新規流入を恒常化できる
