## なぜ NAR 主分類なのか

プレイヤーの主動機は「電脳聖杯戦争という並行世界線で、自分のサーヴァントと7週間を過ごし、別解の Fate を読む」ことである。`concept` が示す「Fate/stay night の並行世界」「週ごとにダンジョン→週末に一対一バトル」という構造は、見かけはダンジョン RPG だが、週次の予習と一騎打ち、敗北したマスターの退場ドラマ、サーヴァントごとに分岐するルートという**ノベルゲームの読み口を RPG の進行に擬態させた**設計だ。

`target` にある「TYPE-MOON ファン」「後年の FGO・EXTELLA からのルーツ探訪」需要が示すとおり、本作の消費は戦闘上達よりも「ネロ / 無銘 / キャスター三ルートの読み比べ」に重心がある。`secondary: EXP` が立っているのは、週末バトルが「相手のスキル使用パターンを観察→属性じゃんけんを読み切る」推理型 1on1 として機能し、物語演出と地続きの体験として効いているためで、戦闘ループ単独で完結する作りではない。

## 物語の駆動構造

週 = 章という外形カレンダーが、ノベル / 探索 / 戦闘 / 敗者退場の四拍子を強制的に回す。

| 構造要素 | 役割 | プレイヤー体験への効果 |
|---|---|---|
| 7 週カレンダー | 章立てを RPG 進行に擬態 | 「次の対戦相手は誰か」が章ヒキになる |
| 予選 → 本戦の入れ子 | サーヴァント淘汰の縮図 | 倒した相手のサイドストーリーが負債として残る |
| サーヴァント 3 ルート | ネロ / 無銘 / キャスターで台詞・展開分岐 | 周回前提、「もう一周読む」動機を制度化 |
| 1on1 ターン制バトル | 攻撃 / 防御 / 妨害の三すくみ＋スキル | 相手の手筋を読む「推理ノベル」化 |
| 電脳世界という前提 | 死＝データ消去で再起不能 | 敗者の台詞が世界観で擁護される |

並行世界という設定が「stay night 本編との矛盾」を構造的に消すので、ファンディスクではなく**正史と等価な別ルート**として読める。これが NAR としての読みごたえを担保している。

## バズ要因 / 人気の根拠

- TYPE-MOON 初の本格コンシューマー RPG という肩書きで発売週は週間 2 位・初週 7.2 万本（`popularity` より）。当時の Fate IP がノベル / アニメ中心だったところに、家庭用ハードでオリジナルストーリーを供給した稀少性が効いた。
- ネロ・クラウディウス、無銘（後の EMIYA Alter 系譜）、キャスター（玉藻の前）など**後年 Fate/Grand Order で人気上位に入るサーヴァント群の初出**を抱えており、FGO 以降のファン層が遡及的に掘り起こす流れが続く。
- 続編 Fate/EXTRA CCC、アクション化の Fate/EXTELLA、FGO の Moon Cell イベント、Last Encore のアニメ化と、**派生作品のハブ作**として再評価され続けている。
- 2025 年に Fate/EXTRA Record（リメイク）が発表されたことで、PSP 原作の再評価需要が再点火しており、「ルーツ探訪」需要は現在進行形で更新中。

## 企画への示唆

- ノベルの章立てを「曜日 / 週」というカレンダーに翻訳すると、選択肢ではなく**進行リズムそのものが物語を駆動**できる（ペルソナ系列と同じ発見）。
- 一騎打ちバトルを「相手のスキル予測」に絞り込むと、戦闘が物語演出と地続きの**推理パートに変質**する（戦闘の難度ではなく観察の難度で勝負できる）。
- 本編のスピンオフを並行世界として設計すると、**正史との矛盾コストをゼロにしつつ独自キャラを正式追加**できる。FGO の Moon Cell 取り込みはこの設計の配当である。
- 「死＝データ消去」という世界観ルールを設定に組み込むと、**敗者の退場演出が世界観の整合性で正当化**され、サブキャラの一回性が物語の重みに直結する。
- サーヴァント 3 択を「ルート分岐」と兼用すると、**周回プレイの動機を制度化**でき、ボリューム 1 本で 3 本分のリプレイ価値を確保できる。

## 比較で見るべきタイトル

- ペルソナ 3 / 4 — カレンダー進行 × RPG × ノベル消費の先行事例
- Fate/stay night（原作 VN）— 並行世界として本作が分岐元にしている本流
- Fate/Grand Order — 本作のサーヴァント群を引き継いで巨大化したモバイル別解
- 世界樹の迷宮 — PSP 同世代の一人称ダンジョン RPG として戦闘設計の参照軸
- Fate/EXTRA CCC、Fate/EXTELLA — 同シリーズ内での物語と体験の比重シフト事例
