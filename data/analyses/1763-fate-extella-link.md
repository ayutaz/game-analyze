# Fate/EXTELLA LINK 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**推しサーヴァントを操作して群れを薙ぎ払う／他プレイヤーと撃ち合う**」という身体動作の体験にある。Fate という IP に紐づくため物語消費の側面は強いが、ゲーム時間の大半は無双系一騎当千の戦闘モーションを反復することに費やされる。concept にある「ハイスピード一騎当千アクション」と social_axis「対戦」（4vs4 オンライン）が示す通り、本作の差別化軸はシナリオではなく「サーヴァント乱舞の手触り＋対人戦」である。

secondary に NAR が入るのは、target にある「Fate シリーズ、特に EXTRA / EXTELLA 系世界観のファン層」が章立てのストーリーモードを目当てに購入する比率が無視できないため。前作『Fate/EXTELLA』はストーリー単独構成だったが、本作はそれを保持したまま **対戦モード（操作の体験軸）を上乗せ** している。物語は維持コスト、対戦が新規誘引というレイヤ構成と読める。

## 体験の核 / 設計の発明

中核メカニクスを分解すると以下のように層が見える。

| レイヤ | 担当する楽しさ | 設計上の役割 |
|---|---|---|
| サーヴァント操作 | 「推し」を直接動かす身体感覚 | IP ファンの NAR 動機を EXP 動線に変換 |
| ハイスピード乱舞 | 雑兵を薙ぎ払うカタルシス | 無双系の定型ループ |
| 参戦数の増加 | 自分の推しが居る／居ないの可視化 | 前作からの差分セールス材料 |
| 4vs4 オンライン対戦 | 他プレイヤーとの腕比べ | リピート再生数の延命装置 |

前作との差分は二点に集約される。第一にプレイアブル英霊の大幅増、第二に対戦モードの追加。これにより「物語をクリアしたら終わり」だった単発消費ゲームを、social_axis「対戦」をフックにした **継続プレイ可能なオンラインタイトル** に組み替えている。Fate IP のキャラクター数を物量で見せ、それを対戦の駒として再利用するという、IP 資産の二重活用構造になっている。

## バズ要因 / 人気の根拠

popularity に挙がる「テラリンク」愛称と Fate ファンの注目は、以下の事実で裏付けられる。

- 初週 PS4+Vita で **117K+** を売り、PS4 国内チャート 1 位を獲得（pushsquare 報）
- 全世界累計は **300K+** とコアファン向け規模に着地。AAA 規模ではなく **IP ファンダム特化** のスケール
- 発売当初は PS4 / PS Vita のみだったが、後に **Switch / Steam / iOS / Android** までプラットフォーム拡張。Fate ファンの所有デバイスを順次拾い上げる長期戦略
- 参戦サーヴァント大幅増と 4vs4 対戦という二つの差分が **Fate ファンの SNS 上で「自推しが居るか」議論** を継続的に発生させた

## 企画への示唆

- **続編で差分を作るときは「物量増 + 新モード」を二段で重ねる**と、前作プレイヤーの再購入動機（推しの追加）と新規動機（対戦）を別個に作れる
- **IP ファンのキャラクター愛を「操作したい」欲求に変換する**と、NAR 消費層を EXP の長期プレイ層に転化できる
- **物語モードを土台に対戦モードを上乗せする二層構成**は、シングル消費型タイトルの寿命延長に有効
- **初動 PS4/Vita → 段階的 Switch/PC/モバイル展開**でファンダムの所有デバイス差を取りこぼさず、長期売上を回収できる
- **「○○vs○○」の対戦人数設計**は、IP の登場キャラ数とパーティ編成の見せ場が両立する数字（本作は 4vs4）に調整すると、IP の物量を活かせる

## 比較で見るべきタイトル

- 真・三國無双シリーズ — 同ジャンル（無双系一騎当千）の原典、対戦モードの参考系譜
- ゼルダ無双 / FE 無双 — IP×無双の先例、キャラクター愛の操作変換モデル
- Fate/EXTELLA（前作） — 直接の前世代、シングル特化型の対照
- Fate/Samurai Remnant — Fate IP のアクション系譜、シングル NAR 寄りの比較対象
