# Fate/Grand Order Arcade

## なぜ EXP 主分類なのか

このタイトルの主動機は「FGO のサーヴァントを自分の手で操作して 3 対 3 で **戦う**」ことにある。本編 FGO がコマンド選択型の RPG であるのに対し、Arcade 版は同じ IP を **操作型 3D アクション** に翻訳し直しており、プレイヤーが筐体に座って得る価値の中心はバトル体験そのものだ。social_axis が「対戦」であり、店内 PvP・全国マッチングを通じて他プレイヤーと直接ぶつかる構造が前提になっている。

target が「対戦アクションとカード収集の両方を楽しみたいコアプレイヤー」と二層に分けて書かれている通り、副次的に REW（カード排出・限定サーヴァント収集）が強力に効いている。secondary に REW が入っているのはこのため。ただし、カードを集めるだけなら本編ガチャで十分であり、Arcade に通う動機を最終的に支えているのは「IP キャラを操作して勝つ」体験側だと判断して EXP 主分類とした。

## 体験の核 / 設計の発明

中核メカニクスは、(1) 物理カード排出によるサーヴァント獲得、(2) IC カードで紐づくマイサーヴァントの育成、(3) 3 対 3 のチーム対戦アクション、の三層を 1 筐体に詰めた点にある。コマンド RPG の FGO とは別レイヤーで「操作の上手さ」と「サーヴァント編成」の双方を競わせる設計になっている。

| レイヤー | 体験要素 | 前世代との差分 |
|---|---|---|
| 筐体ハード | 3D アクション + 物理カードディスペンス | 「艦これアーケード」(SEGA, 2016) の系譜を FGO IP で延長 |
| 操作系 | サーヴァントごとの固有モーションと宝具演出 | 本編のコマンド戦闘を実時間アクションに翻訳 |
| 所有 | 物理カード = 自分のサーヴァント | デジタルガチャと違い「現物」として残る |
| 対戦 | 3 vs 3 のチームバトル | ソロ周回中心の本編に対し対戦軸を新設 |
| 拡張 | 男性ジャック・ド・モレー、カーマ等 AC 限定サーヴァント | IP の世界観そのものを Arcade 側で拡張 |

social_axis「対戦」が成立しているのは、ゲームセンターという物理空間が同卓・観戦・常連コミュニティを生み、ランキング戦やイベント大会が継続的に運用されていたためだ。

## バズ要因 / 人気の根拠

popularity に記載の「初月 1000 万枚以上のカード排出 / 約 9 億円超のカード売上」は、アーケード経済としては突出した数字。これを起点に分解する。

- 2018 年 7 月稼働開始から初月で **物理カード 10M 枚以上**を排出、初月カード売上 **約 9M USD 超**。
- TYPE-MOON × SEGA というブランド同士の組み合わせで、稼働前から FGO ファンとアーケード勢の両方の注目を集めた。
- **男性ジャック・ド・モレー**や**カーマ（ミステイク）**といった本編に存在しないオリジナルサーヴァントを生み、Arcade 限定設定が IP 全体の話題に逆流した。
- 2026 年 3 月 30 日に **約 7 年 8 ヵ月**の稼働を終えることが発表され、FGO 関連プロジェクトとしても長寿アーケードとしても惜別の文脈で改めて言及された。

## 企画への示唆

- 既存 IP をジャンル変換して別筐体に載せると、原作と並走しても食い合わずに**新規体験層と収益チャネル**を作れる（本編 RPG → Arcade 対戦アクション）。
- カードディスペンス型は「ガチャの当たり」を**物理現物**として残す装置であり、デジタル完結のソシャゲでは作れない所有感と来店動機を同時に得られる。
- 派生メディアで**本編に存在しないキャラ**を出すと、IP 全体の議論を喚起しつつ「ここでしか触れない」希少性を担保できる（男性ジャックの例）。
- 3 対 3 対戦 + 観戦可能な物理筐体は、ゲームセンターを**コミュニティ拠点化**しやすく、運用期間を伸ばす（7 年 8 ヵ月）。
- アーケード経済の指標は DAU よりも**筐体回転と消耗品売上**で見ると、ソシャゲと別の収益モデルとして評価できる。

## 比較で見るべきタイトル

- 艦これアーケード（SEGA, 2016）— カードディスペンス + IP 移植の直接的な先行モデル
- WCCF / ガンダム エクストリームバーサス系 — 3 対 3 / チーム対戦アーケードの設計参照
- Fate/EXTELLA（本編 IP の家庭用アクション化）— 同じ「FGO/Fate を操作型に翻訳」する別解
- 三国志大戦シリーズ — 物理カード × アーケード長期運営の経済モデル
