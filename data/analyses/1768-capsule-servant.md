# カプセルさーばんと 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**SD 化したサーヴァントを盤面に並べて相手陣地を崩す**」という対戦型タワーディフェンスの手番ごとの判断プレイ。Fate の世界観を素材にしているが、消費の中心は「物語を追体験する」でも「ガチャで強キャラを引く」でもなく、**配置・召喚タイミング・キャラ相性の試行錯誤**にある。

`social_axis: 対戦` が示すとおり、士郎ルート／凛ルートに分岐する難易度設計と 50 体超のさーばんとのプール構成により、対 CPU 戦であってもプレイヤーは「相手 AI の召喚パターン → 自陣のレーン振り分け」という対戦的な意思決定を回し続ける。この体験ループが買い切り 490 円で完結することが本作の価値の中核なので、主分類は EXP が妥当。

secondary に REW を置いたのは、50 体超のサーヴァントを揃えてデッキを組み替えるコレクション動機が副次的に走るため。ただしガチャや課金で集める設計ではなく、**プレイ進行で開放される収集**なので、報酬ループは主動機を補強する位置づけに留まる。

## 体験の核 / 設計の発明

中核メカニクスは「Fate IP × タワーディフェンス × SD キャラ」の三層構造。**Fate/hollow ataraxia (PS Vita 2014) 収録のミニゲームを独立アプリ化**するという、IP 内のサブコンテンツをスタンドアロン商品に昇格させる手法そのものが設計の発明点。

| レイヤー | 内容 | 体験への寄与 |
|---|---|---|
| IP レイヤー | サーヴァント 50 体超を SD 化 | キャラ愛着で対戦の摩擦を緩和 |
| ゲームメカニクス | 対戦型タワーディフェンス（レーン制召喚） | リアルタイム判断 → 対戦感の中核 |
| ルート分岐 | 士郎ルート / 凛ルート | 同じ盤面を異なる視点で再走させる |
| 価格 / 課金 | 期間限定 490 円・App 内課金なし | FGO の対極にある「完結する Fate」 |

`social_axis: 対戦` の効果は、ガチャ運営型ではなく**毎試合の盤面設計に意思決定を寄せる**ことで、SD キャラのお祭り感と戦術プレイを両立させた点に出ている。

## バズ要因 / 人気の根拠

- **Fate/stay night 15 周年記念プロジェクト**の一環として 2019 年に配信され、IP 周年枠の話題性を確保した
- **Fate/hollow ataraxia (2014) のミニゲームをスタンドアロン化**という再パッケージ手法が、原典ファンの「あれを単体で遊べる」期待を直撃した
- **期間限定 490 円・App 内課金なし**の買い切り設計が、FGO の高額ガチャ運営との対比で「潔い価格設定」として SNS 上で話題化
- 開発が **FGO と同じ DELiGHTWORKS**、配信元が **TYPE-MOON** 自社という体制で、IP コントロールと公式感が担保された

## 企画への示唆

- **既存大作内のミニゲームをスタンドアロン化すると、開発コストを抑えつつ周年記念の話題枠を作れる**（hollow ataraxia ミニゲーム → 独立アプリ）
- **運営型ガチャ IP の周年に「買い切り・課金なし」のサテライト作品を投げると、本編の課金疲れに対するカウンター需要を拾える**（FGO 並走期の Capsule Servant）
- **メインストーリーを描かない IP 派生作は SD 化＋お祭りジャンル（TD・パーティ等）に寄せると、世界観の整合性負債を負わずにキャラ消費を提供できる**
- **ルート分岐を難易度差として扱うと、同じ盤面を 2 周させる動機をシナリオコストゼロで生み出せる**（士郎ルート / 凛ルート）
- **490 円という「映画 1 本以下」価格帯は、IP ファン向け記念グッズとしての心理的ハードルを劇的に下げる**（買い切りモバイルのレンジ設計）

## 比較で見るべきタイトル

- **Fate/Grand Order**（同 IP・運営ガチャの本流。Capsule Servant の対極設計）
- **Fate/Samurai Remnant**（同 IP・買い切り単体作品。SD ではなく本編格の派生）
- **千年戦争アイギス**（対戦型に近い TD・スマホ／PC のキャラ召喚型）
- **コトダマン**（IP×SD×買い切り感のパーティ的モバイル）
