# MELTY BLOOD 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「**空中ダッシュとシールド/ヒートで構築された高速 2D 格闘の手応えを、対人で確かめる**」ことにある。`social_axis: 対戦` が示すとおり、本作の中核体験は CPU 戦ではなく対人戦のフレーム単位の駆け引きで、`concept` でも「ギルティギア／カプエス級のテンポ」と TYPE-MOON IP の両立が明言されている。

ただし `secondary: ["NAR"]` は無視できない。月姫キャラを使う以上、原作ファンが「アルクェイドや志貴を操作する物語的快楽」を入口に来ることは設計に折り込まれており、ストーリーモードがナラティブ補完装置として機能している。それでも主動詞が「読む」ではなく「**戦う／競る**」である以上、分類は EXP が妥当である。NAR は来場導線と離脱抑制のレイヤーとして効いている。

## 体験の核 / 設計の発明

中核メカニクスを 3 層に分解すると、本作の独自性は「同人サークルが商業格ゲーの語彙を圧縮した」点に集約される。

| レイヤー | 採用要素 | 前世代との差分 |
|---|---|---|
| 機動 | 空中ダッシュ・空中ガード | カプコン VS 系の機動性を 1 ボタン操作で展開 |
| 防御 | シールド（ジャストガード相当） | ガード自体を能動的なリソース化、読み合いを攻防両側へ |
| 攻撃 | ヒートゲージ／MAX 化 | スパキャン的な大ダメージ機会を恒常的に内蔵 |
| IP 接続 | 月姫キャラの再演出 | 既存 IP の 3D モデル不要、2D ドット表現で同人スケールに最適化 |

`social_axis: 対戦` の効きは、コミックマーケットという「ローカル対戦コミュニティが面で発生する場」と相性が良かった点に出る。原作読了済みの濃いプレイヤー同士が即売会後にロビーで集まり、対戦コミュニティが自走した。

## バズ要因 / 人気の根拠

- 2002 年コミックマーケット 63 で頒布された**同人格闘ゲーム**であり、頒布チャネル自体が「TYPE-MOON 文脈の濃いユーザー」に直結していた
- 『月姫』スピンオフという**既存 IP からの引力**で、原作ファンが格ゲー未経験でも入口に立った
- 同人版からアーケード版 (Re-ACT / Act Cadenza)、PS2 版、後年の TYPE LUMINA まで**プラットフォーム階段**を駆け上り、商業格ゲー史にトレースされる存在になった
- 渡辺製作所が後に商業スタジオ **French-Bread** として『アンダーナイト』を継ぐ礎となり、開発系譜が同人→商業の成功事例として参照される

## 企画への示唆

- **既存 IP の二次創作レイヤーで新ジャンルに侵入する**と、原作ファンを格ゲー未経験のままチュートリアル動機に変換できる
- **頒布チャネルとコミュニティを一致させる**（即売会＝対戦コミュニティ）と、ローンチ直後の対戦人口が自走しやすい
- **同人スケールで商業の語彙を圧縮採用する**（空中ダッシュ＋シールド＋ヒート）と、リソース制約下でも「触ったときの差別化」が立つ
- **ストーリーモードを副軸として併走させる**と、対戦離脱者にもキャラ消費の出口が残り、プレイ時間が伸びる
- **同人→アーケード→家庭用→商業リブートの階段**を初期から想定すると、各段階の権利・開発リソース移譲がスムーズになる
