# MELTY BLOOD Act Cadenza 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「読み合いで対人戦に勝つ」ことであり、原作『月姫』の世界観は入り口に過ぎない。`social_axis: 対戦` の通り、ゲーセン筐体での対戦と、PS2 移植以降のオフライン対人が体験の中心に据えられている。

`concept` にあるように本作はシリーズ初のアーケード稼働作で、コンボ・バーストを軸にした高速の駆け引きをループさせる設計になっている。`target` も TYPE-MOON ファンと「2D 格闘ゲームのアーケード／対戦コミュニティ層」を併記しており、版元の想定上もすでに**対戦コミュニティ向けの製品**として位置づけられている。`secondary: NAR` は、月姫世界のキャラ・固有技・スタイル別表現が継続購買と参戦キャラ議論を支えている、というキャラクター駆動の上乗せ要素を示す副軸である。

## 体験の核 / 設計の発明

中核は「同人格ゲー由来のキャラ表現」と「アーケード基板で要求される対戦バランス」を接続したことにある。前作『Re·ACT』までで蓄積されたシステム（シールド／ヒート／ブラッドヒート、3 スタイル）を、SEGA NAOMI 基板で稼働させるためにテンポと判定を再調整し、Ver.A → Ver.B のバージョンアップで対戦環境を継続的に書き換えた。

| 要素 | 前世代（Re·ACT 等同人版） | Act Cadenza での更新 |
|---|---|---|
| 稼働環境 | PC（同人頒布） | アーケード（NAOMI）＋ PS2 |
| バランス調整 | 同人サイクル | Ver.A / Ver.B の業務用アップデート |
| スタイル | クレセント／ハーフ／フル | 整理し直された 3 スタイル＋追加キャラ |
| 主軸メカニクス | コンボ・シールド | コンボ・バーストを核に高速読み合い |
| プレイ場 | LAN・身内対戦 | ゲーセン対戦台＋家庭用ロビー |

`social_axis: 対戦` が効くのは、コンボ／バースト／シールドの三層がいずれも**「相手の択を読み返す」入力に直結している**ためで、観戦も「次に何が通るか」を理解できる視認性に支えられる。

## バズ要因 / 人気の根拠

`popularity` の「同人格ゲーがアーケードへ進出した象徴」という文脈を分解すると、以下のような事実層が積み上がっている。

- TYPE-MOON『月姫』という既に二次創作の厚い IP を、French-Bread が同人格ゲーとして実装していた前史がある（『MELTY BLOOD』『Re·ACT』）。
- Ecole が商業パブリッシャとして加わり、SEGA NAOMI 基板での業務用稼働を実現。**同人 → アーケード**の経路自体が当時の事件として語られた。
- 2005 年に PS2 へ移植され、家庭用の対戦コミュニティと、月姫ファン層の双方を取り込んだ。
- Ver.A／Ver.B と業務用アップデートを重ね、後継『Actress Again』『Type Lumina』へと続くシリーズの土台になった。

`sales_jp` / `sales_world` は「—」で公式数値が出ていないが、シリーズが 2021 年の『MELTY BLOOD: TYPE LUMINA』まで継続している事実が、本作で確立された対戦コミュニティの厚みを裏付けている。

## 企画への示唆

- **同人発 IP を業務機に載せると、コミュニティ厚と新規露出を同時に得られる** ― 既存ファンの熱量が稼働率を担保し、ゲーセン側の通行人にリーチが広がる。
- **バージョン分割を最初から計画に入れると、対戦バランスを「商品の継続性」に変換できる** ― Ver.A → Ver.B の更新が話題と再来店を生む。
- **原作 IP の固有設定をシステム名に落とすと、世界観と操作が同じ語彙で語られる** ― 「シールド」「ブラッドヒート」のように、設定語＝メカニクス語にすると学習が早い。
- **同人版で完成度を上げてから商業化すると、初版から競技耐性のあるゲーム性で出せる** ― プロト期間を別レーベルで吸収する設計手法として読める。
- **EXP 主分類 × NAR 副軸の格ゲーは、キャラ追加が継続購買の主な口実になる** ― ロスター更新のロードマップが収益とコミュニティの両方を回す。
