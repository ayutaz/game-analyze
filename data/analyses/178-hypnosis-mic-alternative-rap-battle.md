# ヒプノシスマイク -Alternative Rap Battle-

## なぜ REW 主分類なのか

プレイヤーの主動機は「推しキャラを引き、育て、楽曲とともに所有する」こと。原作 CD・声優ライブ・アニメで先に好きになった男性キャラクターを、ガチャを通じて自分のチームに取り込み、衣装違い・楽曲違いのバリエーションでコレクションを完成させていく行為が中心にある。

concept にある通り本作は「楽曲消費とキャラ育成ガチャを橋渡しする位置づけ」のリズム / 育成ゲームであり、target も「キャラクター推し活と楽曲視聴を起点に流入」する 10-30 代女性ファン層と明示されている。リズムゲームの操作体験は楽曲を「体験する」ためのインターフェイスというより、推しのカードを盤面に乗せて演出を見るための儀式に近く、報酬獲得（カード入手・限定衣装・楽曲開放）こそが継続の駆動力になっている。social_axis が null なのも、同期 / 非同期の他者軸より「推し × 楽曲ガチャ」の縦軸が支配的だからである。

secondary に NAR が立つのは、原作ラップバトル CD / アニメで積み上げられた 6 ディビジョン × 3 人 = 18 人の関係性とストーリーが、カードに紐づくサブシナリオやコラボイベントとして消費されるため。物語は外部 IP から持ち込まれ、ゲーム内では「カードを引く理由」を補強する役割を担う。

## 報酬ループの組成

ガチャ・楽曲・キャラ関係性が三層に積み重なって 1 回のセッションを構成する。

| 層 | 中身 | 主動詞 |
|---|---|---|
| 入口 | 新規 SSR カード / 限定衣装 / コラボ楽曲ガチャ | 引く |
| 中核 | リズムゲームで楽曲をプレイ、スコア達成で育成素材 / ストーリー解放 | 叩く・聴く |
| 外周 | 原作 CD・声優ライブ・アニメ連動の期間限定イベント | 推す・参加する |

入口のガチャは原作 CD のリリースサイクル（新曲 / 新ディビジョン参戦）と同期しており、外部 IP のニュースがそのまま課金イベントに変換される設計。中核のリズムパートはスコアアタックよりも楽曲を回数消費させる動機付け（イベント pt 集め）に寄っていて、外周のライブ / アニメ / 漫画と相互参照することで「ゲーム外で発生した推し事象」が次回ガチャの動機を作るループになる。原作 IP が新譜を出すたびにゲーム側でガチャ・楽曲・ストーリーが解禁され、リテンションを外部供給に依存できるのが運営上の強みである。

## バズ要因 / 人気の根拠

popularity は「ラップ×イケメン男性キャラの IP ブーム」と要約されているが、これは次の要素の合成として読める。

- 国内 500 万 DL 以上。声優ユニット × ラップという従来の女性向け IP に無かった音楽ジャンルの切り口
- 6 ディビジョン × 3 人 = 18 人の MC を地域 / 性格で差別化し、ファンが「推しディビジョン」単位で集団化できる構造
- 原作が CD ベースで楽曲が先行し、ライブイベント / アニメ / 漫画 / 舞台と多方向にメディアミックス展開、ゲームはその合流点
- アイディアファクトリー / イデア・レコードという女性向け乙女系の運営知見を持つ座組

## 企画への示唆

- 外部 IP の更新サイクル（CD / ライブ / アニメ）にガチャを同期させると、ゲーム内コンテンツ生産コストを外部に逃がしつつ課金トリガーが定期供給される
- キャラクター集団を「3 人 × 6 集団」のような小単位 × 集合体で設計すると、推しの粒度が個人単位と集団単位の両方で語れて二次創作が伸びる
- 音楽ジャンル（ラップ）を IP の独自軸にすると、既存の女性向け作品との差別化が音響的に成立し、楽曲単体でも商品化できる
- リズムゲームをコア体験ではなく「楽曲を聴く / カードを眺める儀式」と割り切ると、難易度より演出と回数消化の設計に資源を集中できる
- secondary NAR をゲーム内で全部書こうとせず原作 CD / アニメに委ねる構成にすると、ゲーム側のシナリオ制作負荷が抑えられる
