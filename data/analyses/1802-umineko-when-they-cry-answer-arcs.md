## なぜ NAR 主分類なのか

プレイヤーの主動機は「読む」、より厳密には**前半 Question Arcs で提示された謎に対する解答を、戦人と並走して受け取る**ことである。本作はサウンドノベルというフォーマットを採るが、選択肢分岐は実装されず、テキストを最後まで読み切ること以外に進行ルートは存在しない。よって動詞は「戦う」でも「集める」でもなく、「読み解く」になる。

concept にある「推理と反証のメタゲーム構造を物語そのものに織り込み、読者＝戦人として真実を選び取る体験」が示す通り、ミステリのロジックは UI（推理ノートや証拠リスト）ではなく地の文と魔女ベアトリーチェとの対話劇に埋め込まれており、操作で介入する余地がない。target の「長文サウンドノベルを腰を据えて読める層」「同人作品としての文脈を楽しむ層」も読書動機の純度の高さを裏付けている。social_axis が null（EXP ではない）こと、secondary が空であることも、副次的な収集・対戦要素を意図的に排した設計を示す。

## 物語の駆動構造

Episode 5-8 という章立てと、メタ層（黄金の魔女と戦人の盤上対局）／物理層（六軒島の連続殺人）の二層構造が、読者の解釈作業を絶えず駆動する。

| 構造要素 | 機能 | 効果 |
|---|---|---|
| Episode 5-8 の 4 部構成 | 1 話あたり 30-50 時間の長尺で 1 つの「赤字／青字ルール」を検証 | 読者は仮説を 4 サイクル更新できる |
| 赤字 / 青字 / 金字 | 真偽の確定したテキスト記号 | 命題論理として推理を共有可能にする |
| ベアトリーチェ vs 戦人の盤上劇 | 物理層の事件をメタ層で「ゲーム」として再演 | 読者を陪審員ポジションに置く |
| EP8 「最後のお茶会」 | 解答を明示せず読者に委ねる | コミュニティ議論を恒久燃料化 |

選択肢を持たない代わりに、**真実を選び取る作業を読者の脳内に外注**しているのが構造的発明である。前半 Question Arcs（2007-2008）が提示した謎を 2009 年に一気に回収する時間設計も、議論を「解答前」「解答後」で明確に分断し、コミュニティの参加体験を時間軸で組成している。

## バズ要因 / 人気の根拠

popularity が示す「コミケ76 で頒布開始 → ミステリ／同人界隈の長期議論 → 2017 年 MangaGamer 英語版でカルト的人気獲得」を分解すると：

- **コミケ76（2009年8月）頒布**という同人流通起点が、初期から考察コミュニティ駆動の文化を形成
- 前半 Question Arcs から数えて 2007-2009 の 2 年間、未解決状態で議論が蓄積していたため、解答編リリース時の話題集中度が高い
- 赤字／青字ルールという**形式化された推論プロトコル**が、考察ブログ・2ch・掲示板で共通言語化し、ファン間の二次推理を成立させた
- 2017 年 MangaGamer による英語公式化で、Steam / GOG 経由の海外ビジュアルノベル受容層に波及し、シリーズとしてカルト的地位を維持
- ひぐらしのなく頃に既読層が継続流入する**シリーズ資産**を持つ

## 企画への示唆

- **選択肢を消して読書動機を純化する**と、考察コミュニティが UI の代わりになる
- **真偽記号（赤字／青字）を導入する**と、ファン間の二次推理が共通プロトコルで走り、議論寿命が延びる
- **Question 編と Answer 編を時期分割する**と、未解決期間そのものが宣伝期間として機能する
- **メタ層と物理層の二重構造にする**と、同じ事件を複数視点で再演でき、長尺化しても情報密度が落ちない
- **解答を完全に開示しない設計**にすると、コミュニティ議論が恒久燃料化し IP 寿命が伸びる
- **同人流通起点を維持する**と、初期コアファンが伝道者になり、後年の商業化／英訳ローカライズへ橋渡しできる
