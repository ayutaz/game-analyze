## なぜ NAR 主分類なのか

このタイトルでプレイヤーが行う中心動作は「**長大な幻想ミステリのテクストを読み、解と反証のレイヤーを往復しながら推理する**」ことである。家庭用機でうみねこ全 8 エピソードに「翼」「羽」「咲」の追加章を重ね、新規スプライト・CG・OP・部分ボイスを上乗せした構成は、すべて「読みやすさと体験密度の更新」に振られており、操作系のメカニクスや報酬ループは持たない。

JSON の `concept` が「読むことそのものが推理ゲーム」と言い切っており、`target` も原作 PC / PS3 版のファンとビジュアルノベル愛好層、ミステリ読者層を主たる受け手として明示している。`social_axis` が null なのは、本作が **完全ソロの読書体験**であり、対戦・協力・観戦・非同期のいずれの軸にも該当しないことを意味する。`secondary` も空で、ガチャ・コレクション・対戦性などの副次動機が一切混じらないという編集判断が反映されている。だからこそ NAR 主分類が妥当であり、副次的にも他カテゴリへの引きはない。

## 物語の駆動構造

うみねこの物語消費を駆動する仕掛けは、**「ミステリと反ミステリの二重読解」を読者にメタレベルで強要する**ことに集約される。本リマスターはその構造を温存したまま、家庭用機での通読体験を以下の層で再編している。

| 構造レイヤー | 役割 | 本リマスターでの拡張 |
|---|---|---|
| 本編 Ep1〜4（Question Arc） | 連続殺人と魔女の幻想を提示 | 新規スプライト／CG／部分ボイスで再演出 |
| 本編 Ep5〜8（Answer Arc） | 「赤き真実 / 青き真実」で推理対決を駆動 | 同上、章間 OP の追加 |
| 追加 TIPS「翼」「羽」「咲」 | サブシナリオで世界観を補完 | 一本にまとめてオールインワン化 |
| 特典『黄金夢想曲†CROSS』DL コード | 同 IP の格闘ゲーム外伝 | 同梱でファン体験を拡張 |

章立ては単線の time-line ではなく、**同じ事件を異なる視点で繰り返す「カケラ」構造**を採り、選択肢分岐ではなく「読者がどこまで疑い、何を真実と認めるか」という認知作業そのものが進行軸になる。キャラクター関係性（右代宮家の血縁争いと魔女ベアトリーチェの遊戯）が固定された駒として配置され、プレイヤーは盤面を読むメタ探偵として置かれる。

## バズ要因 / 人気の根拠

- 07th Expansion『うみねこのなく頃に』エピソード 1〜8 に「翼」「羽」「咲」を加え、**家庭用機で初めて全編をオールインワンで遊べる版**として 2021 年 1 月 28 日に発売された
- 新規スプライト・CG・OP・ボイス追加という**演出面の総入れ替え**が、原作 PC 版から十年以上待ったコア層に強い再購入動機を提供した
- 特典として格闘ゲーム『黄金夢想曲†CROSS』DL コードを同梱した点が、長年のファンの間で「うみねこ関連を一本で総括できる」**コレクター訴求**として話題化した
- Switch / PS4 マルチプラットフォーム展開で、PC や PS3 を手放した旧ファンの再アクセス障壁を一段下げた

## 企画への示唆

- **長期作品を「オールインワン版」として再パッケージする** → 散逸していたファンを一点に再集合させ、旧 IP の再評価サイクルを起こせる
- **新規スプライト＋部分ボイス＋章間 OP** を更新ポイントに据える → 内容は同じでも「もう一度通読する理由」が成立する
- **同 IP の外伝コンテンツ（本作では格闘ゲーム）を特典 DL コードとして同梱** → 既購入層にも再購入インセンティブが生まれ、IP の派生群を一度に整理できる
- **選択肢分岐ではなく「読者の認知作業そのもの」を進行軸に据える** → 物語消費を能動化でき、攻略情報の検索行動・考察コミュニティ生成を誘発する
- **家庭用機（Switch/PS4）マルチプラットフォームで配信** → 旧 PC / 旧コンソール世代のファンの再アクセス障壁を下げ、ロングテール販売が成立する
