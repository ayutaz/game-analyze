# ROBOTICS;NOTES DaSH

## なぜ NAR 主分類なのか

プレイヤーの主動機は「種子島の元ロボ部メンバーの後日談を **読む**」ことに尽きる。サウンドノベル形式で選択肢分岐を持つが、戦闘・収集・育成といったゲームメカニクスは介在せず、物語を読み進めて分岐回収することがプレイ体験そのものになる。

concept にある通り、本作は『ROBOTICS;NOTES Elite』からの地続きの後日談として設計されており、target も「20-40 代の VN/ADV 愛好層」「本編経験者の続編希望層」に限定される。前作のキャラクターと感情の積み増しを前提に書かれるため、ゲームプレイの巧拙よりも「物語消費の連続性」がプレイヤーの主動機となる。EXP（体験／対人遊び）にも REW（収集・育成ループ）にも該当しない、純粋な NAR である。social_axis が null なのもこの構造を反映している。

secondary は空。クロスオーバー要素（Steins;Gate のダル登場）はファンサービスとして機能するが、副カテゴリを成立させるほどの独立メカニクスではない。

## 物語の駆動構造

科学アドベンチャー（科アド）シリーズの「結節点」として駆動するため、章立てよりも **シリーズ世界観の参照密度** で物語を引っぱる設計になっている。

| 駆動装置 | 機能 |
|---|---|
| 後日談という時間軸 | 本編の感情の残債を回収させ、続編購入動機を作る |
| Steins;Gate クロスオーバー | 別シリーズファンの流入導線 + 既存ファンのご褒美 |
| 選択肢分岐型サウンドノベル | 主要ルートに加え分岐回収のリプレイ動機を作る |
| 種子島ローカル舞台の継承 | 前作の場の記憶を再活性化、ノスタルジア駆動 |
| 5pb. 既存ボイスキャスト | 続編としての「本物感」を担保 |

科アドシリーズ全体が「単体作品の物語消費」よりも「シリーズ年表への参加」を価値にしているため、DaSH は単独タイトルとしての物語強度よりも、シリーズ内の橋渡し役として読まれることが想定されている。

## バズ要因 / 人気の根拠

- 『ROBOTICS;NOTES』の正統続編かつ科学アドベンチャーシリーズ最新作という、シリーズ年表上の位置づけそのものが事前話題を作った
- Steins;Gate のダルが登場するクロスオーバー要素が、両シリーズの SNS コミュニティで言及材料になった
- 2020 年に Spike Chunsoft が英語版を PC / PS4 / Switch でリリースし、海外 VN コミュニティに展開（『ELITE』との Double Pack 形式でも提供）
- 初週売上は PS4 約 4,360 本 / Switch 約 2,834 本（Famitsu）と、シリーズファン向け中堅 VN らしい規模感に収まる

## 企画への示唆

- シリーズ世界観に「結節点」キャラ（ここではダル）を一人配置すると、既存 IP 間のファンを相互送客できる
- ナンバリング本編ではなく「後日談」フォーマットを取ると、本編の感情負債を回収しながら IP 寿命を伸ばせる
- VN の海外展開は本編単体ではなく Double Pack 形式で出すと、未プレイ海外層への参入障壁を下げられる
- 中核ファン層（20-40 代 ADV 愛好層）に絞ると、初週 1 万本前後でも商業的に成立する規模設計ができる
- マルチプラットフォーム（PS4 / Switch / PC）同時展開は、VN ファン層の購買プラットフォームの分散に対応する標準解になっている

## 比較で見るべきタイトル

- STEINS;GATE ELITE — 同シリーズの「リマスター + 全分岐統合」型の後日談アプローチ
- CHAOS;CHILD — 科学アドベンチャーシリーズ内での同時代作品、舞台と主題の対比
- ROBOTICS;NOTES ELITE — 本作の前提となる本編リマスター、Double Pack の片割れ
- AI: ソムニウム ファイル — Spike Chunsoft 系 ADV の同時期作品、推理 + 物語駆動の対比軸
