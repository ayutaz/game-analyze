# アイドルマスター シンデレラガールズ スターライトステージ 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「担当アイドルの SSR を引き当て、特訓して MV を解禁する」という収集・育成サイクルにある。動詞でいえば**集める→育てる→鑑賞する**であり、リズムゲーム部分は SSR 確保後の「素材を回す道具」として機能している点で、REW（報酬型）に収まる。

concept にある通り、本作は 2D ブラウザの『モバマス（アイドルマスター シンデレラガールズ）』の後継として 3D フルモーション MV を実装した。前作で培われた**担当アイドル固定型のキャラ愛**がそのまま課金導線に乗っており、target が示す「20–30 代男性 P」の中核層はガチャ→特訓→親愛度→MV 解禁の階段を上るために回す。social_axis が null なのは、本作のソーシャル要素（ルーム訪問、協力イベント、ランキング）が中核ループの装飾に留まり、対戦・協力・非同期のいずれかひとつに強く帰属しないためで、設計の重心が一人の収集行動に置かれていることの裏返しでもある。

secondary に EXP が入るのは、譜面難度 MASTER+／SSS+ を完走する音ゲー上級者層が無視できないボリュームを占めるためだ。担当 SSR 編成のスコア最適化と高難度フルコンボ挑戦が、収集モチベが落ちた後の継続装置として効いている。

## 報酬ループの組成

階層化された報酬を一覧化すると次のようになる。

| 層 | 入手手段 | 中核プレイヤー動機 | リテンション役割 |
|---|---|---|---|
| ガチャ（SSR / SR） | スタージュエル（課金 + デイリー配布） | 担当アイドルの SSR 確保 | 月次「シンデレラフェス」「リミテッド」で再燃 |
| 特訓（覚醒） | 同名カード重ね / 専用素材 | レアリティを最大化した立ち絵 | 復刻イベント走行の動機付け |
| 親愛度 | リズムゲームプレイ + コミュ解放 | キャラ別エピソード（コミュ）の閲覧 | 「育てきった担当」を作る最終目標 |
| MV 解禁 | 楽曲取得 + 衣装入手 | 3D フルモーション MV の鑑賞 | コレクションの可視化、SNS 拡散素材 |
| イベント / ランキング | 期間限定走行 | 限定 SR の確保とポイント報酬 | 月次サイクルの強制装置 |

この階層は「ガチャを引く→楽曲を回す→コミュを読む→MV を見る」と進むにつれて**プレイヤーが触れる情報の密度**が上がる設計になっている。SSR を当てた瞬間より、特訓を終えて MV をフルで見る瞬間に最大の感情ピークが置かれており、課金→消費の時間差が長いのが特徴だ。リズムゲーム本体は素材回収の口実として頻度を担保しつつ、難度カーブが極端に長い（DEBUT〜MASTER+ で 6 段階）ため secondary の EXP 層を別枠で抱え込める。

## バズ要因 / 人気の根拠

- 国内累計 DL は 2,500 万超（公式公表）に達し、日本のアイドル系モバイルゲームの代表格としてシリーズ三部作（ミリシタ／シャニマス）にフォーマットを継承した。
- アイドル 190 人超という極端な大世帯設定により、担当固定型のファンを薄く広く取り込めた。アイマス本家から流入した層と、シンデレラガールズ独自層の二層を持つ。
- 3D フルモーション MV を 2015 年時点のスマホで実装した点が技術的なフックとなり、リリース当初の SNS 拡散と YouTube 上の MV 再生数を押し上げた。
- ライブイベント・声優ライブ・CD リリースとアプリ内楽曲解禁が連動しており、**メディアミックスのハブ**として機能してきた（毎年の幕張ライブが代表例）。

## 企画への示唆

- 既存 IP の 2D 資産を 3D MV で再実装すると、同じキャラクター愛が**より長い視聴時間と SNS 共有素材**に変換できる。
- ガチャの最終報酬を「カードのレアリティ」ではなく「**特訓後の MV と専用衣装**」に置くと、課金→消費の時間差が伸びてユーザーが離脱しにくくなる。
- 担当アイドル固定型の IP では、キャラ数を 100 人超に積み増すと**全員推しに勝てなくても誰かの担当になれる**確率が上がり、薄く広い課金が成立する。
- 音ゲー難度を 5〜6 段階に分けて MASTER+ 帯を別レイヤーとして用意すると、収集モチベが落ちた廃課金層の継続装置になる。
- 月次イベント＋復刻 SR＋限定 SSR の三層ローテーションを固定化すると、復帰タイミングを月単位で読み込めるためマーケティング計画が立てやすい。

## 比較で見るべきタイトル

- アイドルマスター ミリオンライブ！シアターデイズ（2017）— 同じ 3D MV + リズム + ガチャの後継、765AS 中心の世界観。
- アイドルマスター シャイニーカラーズ（2018）— モバマス系譜だがリズム要素を排して育成 ADV に振り切った対極の設計。
- アイドルマスター シンデレラガールズ（モバマス、2011）— 2D ブラウザ前世代、本作の IP・担当固定文化の源流。
- ラブライブ！スクールアイドルフェスティバル（2013）— 同時期のアイドル × リズム × ガチャの代表、3D MV を持たない 2D ノーツ寄り。
- バンドリ！ ガールズバンドパーティ！（2017）— アイドル文脈をバンドに置換した類型、カバー曲戦略で別軸の集客を実現。
