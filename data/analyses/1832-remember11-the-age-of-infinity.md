# Remember11 -the age of infinity-

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「二つの極限状況を行き来しながら世界の真相を読み解く」** こと。雪山遭難編（冬樹）と精神病院編（心としと）を交互に進めながら、なぜ二人の意識が時空を越えて入れ替わるのかを思考実験的に追跡するノベルである。

JSON の concept にある「infinity シリーズ完結作」「人格交換する極限状況」、target にある「ハッピーエンドより思考実験と謎解きを求める読者層」が示す通り、本作の体験価値は分岐選択肢で得るゲーム的快感ではなく、**断片化されたテキストを脳内で再構築する読解作業**そのものに宿る。social_axis が null なのも、他者と共有して遊ぶ設計が一切ないシングル完結のノベル体験だからである。secondary が空であることも、本作が REW 的収集要素や EXP 的操作介入を排した「純粋な読み物」として設計された事実を反映している。

## 物語の駆動構造

本作の駆動装置は「二場面の同時進行」「人格交換」「未解明のまま閉じる結末」という三層の仕掛けが噛み合った構造にある。

| 構造要素 | 機能 | 読者への効果 |
|---|---|---|
| 二場面パラレル進行 | 雪山×精神病院を時間軸ごとに切り替え | 情報の非対称性を作り、片方の場面で得た断片がもう片方の読みを更新する |
| 人格交換ルール | 一定条件で冬樹↔心としの意識が入れ替わる | プレイヤー視点と登場人物視点の二重化、信頼できない語り手の問題が常時発生 |
| 未解明のラスト | 真相を明示せず物語が閉じる | 読了後の二次的考察コミュニティ／攻略 wiki／note 記事を恒久的に駆動 |
| infinity シリーズ連結 | Never7 / Ever17 と世界観・テーマで接続 | 既読層に「3 作通読で初めて見える構造」というメタ層を提供 |

選択肢はあるが、複数 END を網羅させて達成感を与える設計ではなく、**読者を「考えさせるための余白」を確保するためのルート分岐**として機能している。

## バズ要因 / 人気の根拠

- infinity シリーズ第 3 作という枠で、Ever17 で確立されたファンベースをそのまま継承して立ち上がった
- 雪山と精神病院という対極の閉鎖空間を交互に行き来する構成自体が、当時の ADV としては珍しい二場面パラレル方式だった
- 物語の核心が未解明のまま閉じる「未完の大作」と呼ばれる作風が、考察コンテンツ・実況・note 記事を 20 年以上にわたって生み続けている
- PS2 初出（2004）以降、PC・PSP・iOS・Android と複数世代の機種にわたって移植され続け、世代をまたいだ再評価の入口が用意されてきた

## 企画への示唆

- **謎を全部解かずに閉じる**と、考察コンテンツが半永久的に二次拡散の燃料になる（攻略 wiki / 解釈動画 / note 記事）
- **二場面パラレル進行**を採用すると、情報の非対称性そのものが推進力になり、章ごとの引きが自動的に強くなる
- **「信頼できない語り手」**をシステムレベル（人格交換）で常時発生させると、文章の表層と本意が乖離し続け、再読価値が跳ね上がる
- **シリーズ連結のメタ構造**（前作既読で見える層）を仕込むと、新規参入のハードルと既読層の優越感を同時に設計できる
- **選択肢を「達成感の道具」ではなく「思考の余白」として使う**と、ADV であってもパズル的体験に寄せられる
