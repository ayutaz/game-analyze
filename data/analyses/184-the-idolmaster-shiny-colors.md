# アイドルマスター シャイニーカラーズ

## なぜ REW 主分類なのか

プレイヤーの主動機は「担当アイドルのSSRを引き、プロデュースで育て、ライブ報酬と限定シナリオを集める」こと。ガチャ＋プロデュース育成ループが収益と継続の両輪を担うため、主分類は REW が妥当である。

concept にあるとおり、シャニマスは enza ブラウザ基盤で動く軽量プラットフォームでありながら、アイマス系列の中でも特に「アイドル個別シナリオの読み物性」が高く評価されてきた。target である 20–30 代男性プロデューサー層は、ガチャでの引き当て・WING/G.R.A.D.等の育成完走・恒常&限定の P-SSR コンプリートを継続動機とする中課金/重課金層であり、報酬獲得が動機の中心に置かれている。一方で secondary に NAR が立っているのはイベント・コミュ・W.I.N.G 編シナリオの読み物としての強度が異常に高く、ガチャで引いた P-SSR を読むこと自体が報酬になっているからで、REW と NAR が補完しあう構造になっている。social_axis は未設定だが、実質ソロプロデュース型で他プレイヤーとの直接インタラクションは弱い。

## 報酬ループの組成

シャニマスのリテンションは複数層のループが重なって成り立っている。

| 層 | 内容 | 主動機 |
|---|---|---|
| ガチャ層 | 恒常 P-SSR / 期間限定 P-SSR（フェス限・限定衣装）/ サポートSSR | コレクション・担当の限定衣装獲得 |
| 育成層 | WING / G.R.A.D / ノクチル等のプロデュースシナリオを 1 周クリア | スコア最大化・True End 解放 |
| シナリオ層 | アイドル個別コミュ・ユニットコミュ・イベントコミュ・感謝祭 | キャラクター理解・物語消費 |
| イベント層 | ファン感謝祭・GRAD・ノクチル特殊シナリオなど期間制イベント | ランキング報酬・記念衣装 |
| 課金導線 | フェス限ガチャ・ステップアップ・天井（200 連） | 推し当て確率を保証 |

前世代のデレステ／ミリシタが「3D ライブ＋音ゲー」で体験に振っていたのに対し、シャニマスは音ゲー部分を削り、代わりにテキスト分量と育成シナリオ分岐の密度に投資した。enza ブラウザという低スペック前提が「テキスト読み物＋ターン制プロデュース」という設計選択を強制した結果、シリーズの中で最もシナリオ重心のブランドが生まれた。

## バズ要因 / 人気の根拠

popularity の「ブラウザ展開とシナリオ評価でアイマス新世代支持」を分解する。

- enza というブラウザ基盤での運用により、Mobage 経由でアプリインストール不要のリーチを確保。アイマス IP の参入障壁を下げた。
- W.I.N.G. 編・G.R.A.D. 編・ノクチル特殊シナリオなどの「読み物としての強度」が SNS で繰り返し共有され、アイマス内で「シャニマスはシナリオ」というブランドポジションを確立。
- ユニット単位（イルミネ／アンティーカ／放クラ／アルストロメリア／ストレイライト／ノクチル／シーズ／コメティック）でファンダムが分散し、長期運営の中で常に「新ユニット実装」のニュースを供給。
- 2023 年に音ゲー要素を加えた家庭用版『シャイニーカラーズ Song for Prism』が展開され、ブランドが mobile→cross-platform に拡張された。

## 企画への示唆

- ガチャの当たり（P-SSR）に「使ったら即読める数万字シナリオ」を紐付けると、引いた瞬間の満足度がスキルや絵だけのガチャより跳ね上がる。
- 音ゲーやリッチ 3D を競合が握っている市場では、ブラウザ＋テキスト密度という逆方向の設計選択が独自ポジションを作る。
- アイドル系運営において、ユニット単位で世界観・楽曲・物語を分離しておくと、長期運営中の「新ユニット追加」が継続的なニュースバリューを生む。
- プロデュース（育成）に True End 分岐を仕込むと、ガチャの 1 枚が「読み物 1 本」に変換され、ガチャ→消費→再課金のループが短くなる。
- IP 上の派生（家庭用音ゲー、ライブイベント、CD）と本体（mobile）を役割分担させることで、本体側はテキストに専念できる。
