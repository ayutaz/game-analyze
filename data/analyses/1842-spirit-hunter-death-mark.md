# 流行り神 -警視庁怪異事件ファイル- / Spirit Hunter: Death Mark

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「東京 H 市の都市伝説と怪異の正体を、一章ずつ読み解く」** ことにある。concept にある通り、死印を受けた生存者を巡る章立て構成は「次の章で何が暴かれるか」を読みたい欲求でページをめくらせる設計で、戦闘・育成・コレクションといった体験ループはほぼ存在しない。target に挙がる零シリーズ / SIREN 経験者層は和風ホラー固有の「土地に染みついた因縁」を読み解く快楽を期待しており、テキスト主体のホラーノベルとして消費される。

secondary に EXP が入っているのは、各章ボスとの対峙で「正しいアイテム × 正しい使用順 × 正しい判断」を選び損ねると即死エンドに分岐する詰将棋的な操作層があるためで、純然たる読み物 (NAR) よりも操作判断による緊張がワンポイント挟まる。ただしこの判断パートはあくまで章末イベントに局所化されており、ループ全体は「読み物 → 探索 → 怪異対峙の判断 → 次章」という物語駆動順序に従う。

## 物語の駆動構造

| 構造要素 | 設計 | 物語消費を駆動する効果 |
|---|---|---|
| 章立て | 怪異 1 体＝1 章のオムニバス、各章で固有の生存者がパートナー化 | 短い完結単位で「次の都市伝説」への期待を毎章リセット |
| パートナー選択 | 章開始時に同行者を 1 名選び、ストーリー分岐と使えるスキルが変動 | 「もう一人連れて行けばよかった」二週目動機を生む |
| 救う / 見捨てる分岐 | 怪異対峙でパートナーの生死が分岐、ED 数に直結 | END 回収のための周回テキスト消費を正当化 |
| 即死エンド | 選択ミス・アイテム誤用で章途中に BAD END | 「正解の物語」を追体験する快楽（弟切草系統） |
| 警視庁怪異事件ファイル形式 | 各章の終わりに事件報告書としてまとめ | プレイヤーを「読者」ではなく「捜査官」として接地 |

オムニバス＋同行者選択＋分岐エンドという構成は伝統的なホラーノベルのフォーマットを踏襲しつつ、章ボス戦に「アイテム選択ミス＝即死」という古典 AVG の緊張を持ち込み、読書テンポに山場を作っている。

## バズ要因 / 人気の根拠

- 実況配信向きの「即死分岐＋怖い画面静止画」が口コミ拡散の温床となり、PS Vita 末期作にもかかわらず初週国内 8 位に入る初動を作った
- PS Vita → Switch / PS4 / Xbox One / PC と段階的にマルチ展開し、ホラー需要のある北米市場へ Aksys 経由でローカライズ展開、英語圏レビューで「最良のホラーノベルの一つ」評価を獲得
- 和風怪異・都市伝説・実話怪談という題材は配信での「説明不要の恐怖体験」になりやすく、Vtuber / 配信者の単発実況素材として定期的に再消費される
- 単発の話題作で終わらず NG (続編) / 真流行り神シリーズへ系列化し、Experience Inc. の主力ホラーレーベルとして 300K+ 規模を維持

## 企画への示唆

- **オムニバス＋同行者選択を組み合わせると周回動機が二乗化する** — 章単位の完結性 × パートナー差分が END 回収の動機を生み、読了後の再起動率が上がる
- **章末ボスを「詰将棋」にすると、読み物テンポに山場ができる** — 通常パートをテキスト中心にしても、章末に判断ミス＝即死の操作層を 1 つ挟むだけで配信映え・緊張持続が両立する
- **実況配信時代のホラーは「画面の怖さ」より「決断の怖さ」が拡散する** — ジャンプスケアより「救うか見捨てるか」を視聴者にコメントさせる設計の方がクリップになる
- **警察 / 捜査官という枠を被せると、怪異 / オカルト題材が大人向けに昇格する** — 「警視庁怪異事件ファイル」副題ひとつで、ジュブナイル怪談から成人ホラー ADV へ受け手を移せる
- **末期ハード作でもマルチ展開＋海外パブリッシャー (Aksys 等) を組めば寿命が伸びる** — Vita 単発で終わらせず PC / Switch まで引くことで、配信文化に拾われる時間軸が広がる

## 比較で見るべきタイトル

- **零 〜月蝕の仮面〜 / SIREN** — 和風ホラー × 操作型恐怖の前世代。Death Mark はそこから「操作」を抜き「判断」に置換した位置にある
- **かまいたちの夜 / 弟切草** — 章末即死＋END 回収という古典サウンドノベル文法の直系継承
- **NG / 真流行り神** — 同 Experience レーベルの後継作。シリーズ化による IP 経済の参考例
- **Spirit Hunter: NG (2018)** — 同一エンジン / 同一設計で題材だけ差し替えた続編、IP 横展開の手本
- **クロスチャンネル / 沙耶の唄** — ノベル × ホラー × 周回前提分岐の系譜上の比較対象
