# DEATH MARK II 流行り神 — 分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「**心霊スポットで起きた怪異の正体を読み解き、誰が生き残るかを引き当てる**」こと。すなわち戦闘の手応えや収集ではなく、章ごとに用意された怪談を消費して結末を見届けることが中心にある。

JSON の concept にある通り、本作は Experience が積み上げてきた『Death Mark』『NG』からの直接的続編で、「死印を解く」というシリーズ共通の物語装置を H 中学校という新ステージで再起動する構造を持つ。target も「和製ホラー・実話怪談・心霊スポット系の物語消費が好きな層」と明示されており、ジャンルも「ホラーアドベンチャー/ビジュアルノベル」。怪異調査というアクション部分はあるが、その答え合わせと生存ルートの読み解きこそが満足の源泉になっているため NAR が妥当。secondary に EXP が入っているのは、霊障の解除手順をアイテム選択でトライ&エラーする「謎解きパズル」的な手触りが副次的に効いているため。読む体験を主軸に、選んで試す体験が後ろから支えている。

## 物語の駆動構造

シリーズの定型をなぞりつつ、章単位の独立した怪談集 + 全体を貫く H 中学校の因縁という二層構造で物語消費を駆動する。

| レイヤー | 駆動要素 | プレイヤー側の動機 |
|---|---|---|
| 章単位 | 1 怪異 = 1 短編怪談 | 「次の章の怪異が何か」を見たい |
| 同行者 | 章ごとに連れていく仲間が変わる | 各キャラの背景・生死フラグを回収したい |
| 霊障解除 | アイテム選択ミスで同行者死亡 | 正解ルートを引き当てたい / 全員生存を狙いたい |
| エンディング分岐 | 章末選択と最終章で結末が分岐 | 真エンドの全貌を知りたい |

短編怪談を 1 章ごとに片付けていく「実話怪談アンソロジー」の読書感と、同行者の生死が選択で動く「ノベルゲー的緊張」を重ねている点が、シリーズ通底の設計。前作までの「呪殺される側」というモチーフを保ちつつ、舞台を学校に絞ったことで都市伝説／学校怪談という和ホラーの王道タームを取り込んでいる。

## バズ要因 / 人気の根拠

- Spirit Hunter シリーズ第 3 作という積み上げ。1 作目『Death Mark（死印）』が和ホラー ADV のニッチで定着し、続編『NG』を経て固定客が育っている文脈に乗っている。
- 舞台が「H 中学校」という学校怪談アーキタイプ。きさらぎ駅／花子さん／旧校舎系の都市伝説ラインに親和性が高く、配信・実況映えするネタを内蔵。
- Aksys Games による英語ローカライズで欧米ホラー ADV ファン層にも同時配信。和製ホラーノベルを継続して翻訳出版する Aksys のパイプに乗せられた点が国際露出を底上げした。
- マルチプラットフォーム（Switch / PS4 / PS5 / PC）で配信され、Steam ストアにも掲載。サイレントヒル系の系譜を求める PC ホラー層への入口が確保されている。

## 企画への示唆

- **長期シリーズの「型」を持つと続編が読書のように消費される** — 1 作目で「死印 + 章別怪異 + 同行者の生死選択」というフォーマットを確立しておくと、3 作目でも既存客が説明なしで走れる。
- **舞台を 1 場所に絞ると怪談アンソロジーの密度が上がる** — シリーズ前作の街単位から「H 中学校」1 校に閉じることで、章ごとの怪異が同じ屋根の下で連鎖する厚みを得る。
- **同行者の生死選択は読書動機を駆動する最小装置** — キャラ消失リスクが章ごとにある構造は、「次章で誰が消えるか確かめたい」だけでプレイ継続が成り立つ。
- **和ホラー IP は専門ローカライザーとの長期契約で海外露出が伸びる** — Aksys のような特化型 publisher を抱え込むと、シリーズ単位で欧米市場の常連枠を確保できる。
- **怪異解除の「正解ルート探し」は EXP を NAR に従属させる良い装置** — 戦闘で勝つのではなく「正しい順序で読み解く」ことを EXP 層に置くと、物語のテンポを壊さず緊張感を継続できる。

## 比較で見るべきタイトル

- 『死印』『NG（流行り神）』 — シリーズ直系の前作。型の継承度合いを測る基準。
- 『弟切草』『かまいたちの夜』 — 和ホラー / サスペンスノベル ADV のルーツ。章別怪異と分岐生存の祖型。
- 『街』『428 〜封鎖された渋谷で〜』 — チャプター単位で複数キャラの生死を切り替えるノベル構造の延長線。
- 『夕鬼』『パラノマサイト』 — 心霊スポット / 都市伝説をモチーフにした近年の和製ホラー ADV。
