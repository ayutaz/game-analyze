# リバース：1999 — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「20世紀の各年代を旅する女性キャラクター（アルカナイスト）を集め、育てる」ことにある。戦闘やストーリーは存在するが、ユーザーが課金とログイン時間を投じる先はガチャと育成リソース確保であり、報酬ループが行動を駆動している。

concept にある「アール・デコ調ビジュアル＋英語ボイス」と target の「HoYoverse 以外のガチャを探す女性・サブカル層」を合わせると、本作のポジションは明確で、**ガチャを引く動機を「強さ」ではなく「雰囲気とキャラクター世界観の所有」に置き換えた REW** である。social_axis が null なのは、ギルド対戦や同盟戦のような同期的他者要素を持たず、ソロで黙々と引いて読んで育てる構造のため、対戦・協力のいずれにも回収しにくいことを反映している。

secondary に NAR が入るのは、各キャラクターのストーリーと年代別メインクエストが「次のガチャを引く動機」として強く効いているからで、報酬ループの上に物語消費が乗る二層構造になっている。

## 報酬ループの組成

| 層 | 内容 | プレイヤー動機 |
|---|---|---|
| ガチャ層 | 6 ☆ アルカナイスト・専用心相（武器相当）の限定ピックアップ | キャラクター世界観の所有 |
| 育成層 | レベル / 共鳴 / 洞察（凸相当）/ 心相強化 | 戦闘ステージとローグライト挑戦の通過 |
| シナリオ層 | 年代別メインクエスト＋キャラ個別ストーリー＋イベント | 次の限定キャラへの感情移入 |
| 演出層 | ライブ 2D 立ち絵 / 英語ボイス / アール・デコ UI | SNS スクリーンショット / 二次創作 |

リテンション設計は HoYoverse 系（原神／崩壊：スターレイル）の文法を踏襲しつつ、**演出層の差別化で「強さ」軸の比較から逃げている**のが特徴。戦闘もコマンド＋カード合成で軽量に抑えられ、長時間の周回より「ガチャを引いて読んで眺める」短セッションに最適化されている。

## バズ要因 / 人気の根拠

popularity「20世紀レトロ美術と英語音声、女性向け雰囲気ガチャの新潮流」を分解すると：

- **アール・デコ／レトロ美術**：1920s-1990s の各年代を意匠化した UI とキャラデザインで、SNS でのスクリーンショット拡散性が高い
- **英語ボイス基本**：中国産ガチャとしては異例の「英語が一次音声」設計で、欧米と日本のサブカル層に同時に刺さった
- **推定累計収益 $300M+**：HoYoverse 系の独占状態だった「女性向け中華ガチャ」枠に新規参入を成立させた
- **キャラ造形の女性比率と神秘主義設定**：オカルト・アート・洋画趣味のサブカル層への狙い撃ち。target に書かれた層を明示的に取りに行った設計

## 企画への示唆

- **「強さ」軸を捨て「雰囲気」軸で差別化すると、後発でも先行ガチャ大手と直接競合せずに済む**（リバース：1999 が原神とプレイヤーを食い合わずに伸びた構造）
- **音声を英語一次にすると、中国産であることを後景化でき、欧米／日本／中華圏を同時に拾える**（ローカライズコストは増えるがリーチが広がる）
- **時代設定を 10 年単位で章立てすると、衣装・小道具・音楽の差分でアートチームの設計言語が量産できる**（毎年代ごとに新キャラ群を投入する根拠になる）
- **戦闘を軽量カードバトルに留めると、ガチャと物語に滞在時間を割り当てられる**（アクション性を高めるとガチャ動機が薄まる罠を回避）
- **女性キャラ＋オカルト＋アート趣味の交点を明示ターゲットにすると、広い「女性向け」より深い熱量が立つ**（target を絞った設計が二次創作とコラボ動員を生む）

## 比較で見るべきタイトル

- **原神 / 崩壊：スターレイル**（HoYoverse、雰囲気ガチャの本流。比較対象として常に意識される）
- **アークナイツ**（Hypergryph、中国産・女性キャラ比率高・サブカル層、戦略タワーディフェンス側に振った別解）
- **アズールレーン**（Yostar、中国産先行世代の「キャラ所有型ガチャ」、軍艦擬人化＋日本声優の組み合わせ）
- **ブルーアーカイブ**（Nexon、雰囲気＋ストーリー駆動型ガチャの近隣事例、学園×軍事の意匠で差別化）
