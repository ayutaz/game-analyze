# Summer Pockets REFLECTION BLUE

## なぜ NAR 主分類なのか

プレイヤーの主動機は「鳥白島というクローズドな夏休みの中で、ヒロインごとの別離と再会の物語を **読む**」ことに集約される。concept にある通り無印に新ヒロイン 3 名と新ルートを追加した完全版で、ゲームメカニクスとしての挑戦要素は薄く、章を進めて選択肢で分岐するという Key 流の伝統的な恋愛 AVG 構造に収まっている。`social_axis` が null なのも、本質的に他者と関わる体験ではなく、ひとりで読み終えて泣くソロ読了体験であることの裏返しである。

target が「Kanon / AIR / CLANNAD / リトバス世代の Key ファン」と明示されている点も NAR を補強する。彼らが期待しているのは戦闘の手応えでも収集の達成感でもなく、章末で来る「離別の構造」とその余韻だからだ。secondary は空で、副次動機は明示されていない。

## 物語の駆動構造

Summer Pockets は Key 作品の標準フォーマットを踏襲しつつ、REFLECTION BLUE で「拡張版」としての構造を獲得している。物語消費を駆動する仕掛けを分解する。

| レイヤー | 構造 | 駆動要素 |
|---|---|---|
| 季節の枠 | 夏休み（限られた日数） | 終わりが確定している切なさ |
| 島という閉鎖空間 | 鳥白島 = 外界から遮断 | 登場人物の固定と濃密化 |
| ヒロイン別ルート | 個別章＋共通章 | 各ヒロインに独立した別離の意味 |
| 真ルート | 個別ルート踏破後に解禁 | 周回プレイの動機付け |
| 拡張版（RB）の追加分 | 新ヒロイン 3 名と新シナリオ | 既プレイヤーの再購入動機 |

「夏休みの終わり」というカレンダーが先にあり、その有限性に向かってヒロインごとの別離が積み上がる設計は Key が CLANNAD 以降磨いてきた手筋で、REFLECTION BLUE はそれを再パッケージしている。

## バズ要因 / 人気の根拠

- popularity にある通り、PC 2020 → Switch 2021 → PS4 2022 と段階的にプラットフォームを広げ、家庭用ハードでのビジュアルノベル受容層を取り込んだ。
- 2025 年 6 月に Steam で英語版が世界配信され、海外の Key / 泣きゲー再評価文脈と接続したことで「泣きゲー復権の象徴」として再注目された（出典: Anime News Network 2025-06-16）。
- 世界累計 300K+ 規模はビジュアルノベル単体としては大きく、Key ブランドの IP 寿命が 20 年以上にわたって機能していることを示している。
- 既プレイヤー向けの「拡張ヒロイン追加」というモデルは、無印購入者の二度買いと、未プレイ層の最終版選択という二重需要を同時に取れた。

## 企画への示唆

- **季節 × 閉鎖空間 × 別離の三点セット**を物語の骨格に置くと、章末ごとの感情ピークが自然に作れる。鳥白島 = 夏休みのように、終わりが確定したカレンダーは強い駆動装置になる。
- **個別ルート → 真ルート解禁**の二層構造を採ると、周回プレイの動機が外発的に発生し、平均プレイ時間とユーザー語り量が伸びる。
- **完全版／追加ヒロイン版**を 2〜3 年遅れで出すと、初版購入者の二度買いと新規ファンの最終版需要を同時に取れる。ただし「初版を買い損ねた」感を残さないため、新規シナリオ量を明示しておく必要がある。
- **段階的プラットフォーム展開**（PC → 据置 → 携帯 → モバイル → Steam 海外配信）を 5 年計画で組むと、IP の摩耗を避けつつ各セグメントに最大露出できる。
- **海外配信は IP ジャンルの再評価サイクルに合わせる**と効果が出る（2025 年の Key / 泣きゲー再評価波に Steam 英語版を当てたタイミング設計）。

## 比較で見るべきタイトル

- CLANNAD（Key、2004）— 季節と別離の原型、本作の文体上の祖。
- リトルバスターズ！（Key、2007）— 閉鎖グループ × 別離構造の前例。
- AIR（Key、2000）— 夏 × 短い時間 × 別離の原型。
- 素晴らしき日々（KeroQ、2010）— 同時代の章立て型シリアス VN との対比。
- 9-nine- シリーズ（ぱれっと）— 拡張版商法と日常 → シリアスの構造比較対象。
