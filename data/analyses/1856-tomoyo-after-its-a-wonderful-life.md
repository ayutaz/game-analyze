# 智代アフター 〜It's a Wonderful Life〜 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ヒロイン 1 人との同棲生活が辿る顛末を最後まで読み届ける」こと。選択肢で攻略対象を分岐させる Key 本流のマルチヒロイン構造を捨て、智代へ視点を固定した一本道に近い設計で「読み物」としての密度を最大化している。

concept にあるとおり本作は『CLANNAD』本編・坂上智代ルートの後日談スピンオフであり、テーマは「家族・記憶・余命」。target に置かれた "シリアスで重い物語と読み物としての完成度を求める層" がそのまま示すように、ゲームメカニクスの再発明や育成・収集要素ではなく、章を順に追ううちに登場人物の関係性が変容していく時間体験そのものが価値の核となる。社会性は基本的に存在せず（social_axis は null）、副次要素も secondary 空欄で純粋な NAR として整理できる。

## 物語の駆動構造

外伝という前提を活かし、本編で築かれた智代像を出発点に「日常 → 同居 → 家族の再構築 → 喪失」と段階的に温度を上げる構造を取る。読者は前作で得たキャラ理解を資本として持ち込み、初期コストを払わずに重い後半に到達できる。

| 駆動レイヤー | 仕組み | 物語消費を後押しする効果 |
|---|---|---|
| 前提共有 | 『CLANNAD』本編ファンを前提に智代の人物像を継承 | プロローグの感情導入を短縮できる |
| 視点固定 | ヒロインを智代 1 人に絞り選択肢由来の分岐を抑制 | 一本道の濃度で物語に集中させる |
| 同居生活パート | 朝・夜の生活描写、家族（鷹文 / 朋華）との関係 | 中盤の幸福と後半の喪失の落差を作る |
| テーマ階層 | 恋愛 → 家族 → 記憶 → 余命と段階的にレイヤーを重ねる | 後半の急展開を必然として受容させる |
| 外伝補完 | 本編で語られなかった智代側の内面を後付け補強 | 既存ファンに "二度目の読解" を促す |

選択肢は分岐より「読み進める意思の確認」として機能し、結果として小説的な線形性とアドベンチャーゲーム特有の身体的没入を両立させている。

## バズ要因 / 人気の根拠

- Key 作品としては異例の「ヒロイン 1 人固定 + 後日談」というフォーマット。マルチルート期待を裏切り、純愛ノベルとして再評価された
- 『CLANNAD』本編の高い知名度が前提需要を生み、外伝として導線が短い。本編ファンの取りこぼし回収先として機能した
- PC 発売（2005）後、PS2 / PSP / Xbox 360 / PS3 / Android / Switch と長期にわたり移植が続き、世代をまたいで新規読者を獲得し続けた
- 2020 年に Switch 版 CS Edition、Steam 英語版が公開され国内外で長期再評価のフェーズに入っている

## 企画への示唆

- 既存 IP の外伝を作るとき、ヒロインを 1 人に固定するとマルチルートでは扱えない重いテーマ（余命・家族・記憶）を中央に置ける
- 「本編の選択肢の先」を商品化すると、ファンは初期コストを払わず重い物語に没入でき、ARPU の高い層に最短で届く
- ビジュアルノベルでも段階的にテーマレイヤーを積み増す構造（恋愛 → 家族 → 喪失）を取ると、章ごとの離脱を抑制しつつクライマックスへ誘導できる
- 長寿シリーズは「ハード乗り換えのたびに新規流入」を狙える移植戦略を組むと、初動売上ではなく累積評価で評価軸を立てられる
- ノベル系で社会軸（PvP / 協力）を持たない作品でも、母 IP のファンコミュニティが SNS で考察・感想を回す前提があれば話題持続力は確保できる

## 比較で見るべきタイトル

- CLANNAD（Key、2004）— 本編。本作の前提となるマルチルート構造との対比
- planetarian 〜ちいさなほしのゆめ〜（Key、2004）— 同じく Key の短編一本道。ヒロイン固定 NAR の比較例
- 智代アフター以前の Key 作品（Kanon / AIR）— マルチヒロインの泣きゲー設計と本作の単一ヒロイン設計の差分
- ぬきたし系後日談やシリーズ FD（ファンディスク）作品全般 — 本編ファン向け補完商品としての設計比較
