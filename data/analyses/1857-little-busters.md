# リトルバスターズ！ 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「仲間との青春群像を最後まで読み切る」** こと、つまり「読む」である。Key の鍵泣きゲー系譜（Kanon / AIR / CLANNAD）に連なる長尺ビジュアルノベルであり、ヒロイン個別ルートを経て大きな「真相」へ収束する構造をプレイヤーは時間をかけて辿る。`concept` に明記された「仲間との青春の追体験」「最終的に大きな物語の真相へ導く構成」は、体験そのもの（戦闘・収集）ではなく **物語消費の駆動** が主動機であることを示している。

`target` も「腰を据えて読みたい層」と物語消費志向を明示している。野球などミニゲームを内蔵し能動的な日常パートを置いてはいるが、それは物語のリズムを作る装置であって、プレイの目的そのものではない。よって主分類は NAR、`secondary` は空、`social_axis` は null（マルチプレイ要素なし）が妥当である。

## 物語の駆動構造

「日常 → 個別ルート → 真相ルート」という Key 系の三段構成を、本作は「仲間 = リトルバスターズ団」という横の関係性で組み直している。日常パートで仲間との関係を編んでから、ヒロイン個別ルートでそれを縦に深掘りし、最後にメインプロットの謎へ収束する。

| 構造要素 | 役割 | 物語消費を駆動する仕掛け |
|---|---|---|
| 日常パート（ミニゲーム・選択肢） | 仲間とのリズム形成 | 野球・バトルランキング等で能動感を維持し、ノベルの「読まされ疲れ」を緩和 |
| ヒロイン個別ルート | 縦の感情曲線 | 各ヒロインのトラウマ・別れを段階開示し涙腺を稼働 |
| リフレイン／真相ルート | 全ルートの種明かし | 個別ルートで張った伏線を一括回収、世界観の前提を反転 |
| 仲間というレイヤー | 横の関係性 | 鍵作品で薄かった同性友情を太い柱に据え、ラスト到達の動機を強化 |

選択肢は分岐量より「日々の過ごし方の選択」に重きが置かれており、選んだ仲間と過ごす積み重ねが個別ルート開放条件になる。`concept` にある「ミニゲームと選択肢で日常パートを能動的に遊ばせる」設計は、長尺ノベルの読者離脱を防ぐためのリテンション装置として読める。

## バズ要因 / 人気の根拠

- Key 第 6 作として PC 版が **10 万本超** のヒット。鍵泣きゲー系譜の蓄積したファンベースがそのまま購買層となった。
- 鍵作品で薄かった **「男女混成の仲間もの・青春群像」** を真正面に据えた点が新規層獲得につながり、口コミで広がった。
- 野球・バトルランキング等の **ミニゲーム内蔵** が「ノベルなのに動かせる」話題性を生み、配信・実況とも相性が良かった。
- 2012–2013 年の **アニメ化**、PS2 / PSP / PS Vita / PS3 / Switch（2020）への **多機種展開** で接触機会が継続更新され、初出から十数年後も新規層が流入し続けた。

## 企画への示唆

- **長尺ノベルにミニゲームを内蔵すると、読者離脱を緩和できる**。ただし主動機を「読む」に留めるなら、ミニゲームは進行の必須経路ではなく日常パートの装飾に留めること。
- **「個別ルート → 真相ルート」の二層化により、感情曲線と謎解きを別レイヤーで設計できる**。各ルートで張った伏線を最終ルートで一括回収する型は再現可能。
- **同性友情を太い柱に据えると、恋愛 ADV の購買層を拡張できる**。鍵作品が縦軸（ヒロイン × 主人公）中心だったところに横軸（仲間）を足すことで新規層を取った。
- **多機種展開を 10 年スパンで継続すると、原作からのリーチが累積的に伸びる**。PC → 家庭用 → 携帯機 → Switch という移植階段は、IP の寿命延長として参考になる。
- **「日々の過ごし方の選択」をルート開放条件にすると、選択肢の重さが薄くても積み重ねの実感が生まれる**。明示的分岐に頼らないルート設計の一例。

## 比較で見るべきタイトル

- **CLANNAD**（Key, 2004）— 同じ Key 系譜の前世代作。家族 × 個別ルート → 大きな物語の構造を共有
- **Kanon / AIR**（Key, 1999 / 2000）— 鍵泣きゲーの原型。本作との「仲間レイヤー」の有無の差分が見える
- **ToHeart2**（Leaf, 2004）— 学園恋愛 ADV における群像型の系譜
- **シュタインズ・ゲート**（5pb., 2009）— 個別ルート → 真相ルートの二層構造の別解
