## なぜ NAR 主分類なのか

プレイヤーの主動機は「青春群像劇を読み、その裏に隠された“この世界の秘密”を解き明かす」こと。コア体験は戦闘でも収集でもなく、長時間テキストを読み続けて感情を蓄積し、最終ルートで世界観の伏線を回収する読書行為に収斂する。

concept にある「野球をやろうと誓った幼なじみの青春」と「現実の裏に隠された“この世界の秘密”」という二層構造は、Key の Kanon / AIR / CLANNAD と同じく **個別ヒロインルート → True End で世界の謎が回収される** 物語駆動型 ADV の典型であり、target も「Kanon/AIR/CLANNAD の流れを汲む Key の泣きゲーファン」と物語消費層に明示している。social_axis が null なのは、ビジュアルノベルがソロ読書専用で他プレイヤーとの関わりを設計に持たない（EXP の社会軸の対象外）ことを示す。secondary は空で、報酬や育成の副動機を持たない純粋な物語型タイトル。

## 物語の駆動構造

Ecstasy 版は原作（2007）に **3 ヒロインルート追加 + 18 禁シーン全実装** という拡張モデルで、章立て・ルート分岐・感情の山を以下のように分解できる。

| 構造要素 | 設計 | 駆動効果 |
|---|---|---|
| 共通ルート | リトルバスターズ！の野球チーム結成 / 部活動 | 群像劇として読者を世界に着地させる |
| 個別ルート（原作 4 + Ecstasy 追加 3） | ヒロイン別の短編完結譚 | 「泣き」エピソードを反復で蓄積 |
| Refrain（真ルート） | 共通の伏線を回収し世界の正体を開示 | 個別ルートの感情が真相認識に転化 |
| 18 禁シーン追加 | Ecstasy で全ヒロインに実装 | PC 成人向け市場への商品再パッケージ |
| 野球パート（ミニゲーム） | テキストの合間にカード/野球システム | ADV の単調さを破る間奏として機能 |

この「個別ルートで感情を積む → 真ルートで世界の構造を開示する」二段構えは Key 作品共通の駆動装置で、Ecstasy は **追加ルート 3 本ぶん読書時間を延長**しつつ 18 禁要素で旧購買層を再動員する商品設計になっている。

## バズ要因 / 人気の根拠

- 発売直後の Getchu / メディアクリエイトの PC ゲーム売上ランキングで 1 位を獲得し、Key ブランドの当時の集客力を示した
- 累計 10 万本超は PC 18 禁ノベル市場では上位クラスの規模
- 原作（2007）の高評価が下地にあり、Ecstasy は「追加ルート + 18 禁」のダブル訴求で原作既プレイ層を再購入させた拡張版商法の成功例
- PS2 / PSP / PS Vita への家庭用移植、TV アニメ化、漫画展開などメディアミックスが続き、Kanon / AIR / CLANNAD の系譜に連なる Key 代表作として長期的に認知が維持されている

## 企画への示唆

- **個別ルート × 真ルートの二段構造**を採用すると、短編完結の感情体験と長編伏線回収の達成感を一本のパッケージで両立できる
- **完全版（追加ルート + 成人向けシーン）を 1〜2 年遅らせて出す**と、原作既プレイ層の再購入と新規層の取り込みを同時に狙える拡張モデルになる
- **テキスト主体の長編 ADV にミニゲーム（本作なら野球パート）を挿入する**と、読書の単調さが緩和されリテンションが伸びる
- **「日常群像劇 + 世界の秘密」の二層プロット**を組むと、序盤の低刺激な日常パートを終盤の真相開示で意味付け直すことができ、長時間プレイの正当化につながる
- **キャラクター単位のルート分割**は、ヒロイン別の二次創作・グッズ・アニメ単話エピソードに分解しやすく、メディアミックス展開のモジュール化に有利

## 比較で見るべきタイトル

- CLANNAD（Key, 2004）— 同じ「日常群像 + 真ルートで世界開示」構造の前世代代表作
- AIR（Key, 2000）— Key 三部作の起点、個別ルート × 真ルートの原型
- ONE 〜輝く季節へ〜（Tactics, 1998）— Key スタッフの前身作、「えいえん」概念の原型
- WHITE ALBUM2（Leaf, 2010〜2011）— 同時期の長編恋愛 ADV、章分割と感情駆動の別解
- Steins;Gate（5pb./Nitroplus, 2009）— 同じ「日常 → 世界の秘密」二層構造の SF 寄り解
