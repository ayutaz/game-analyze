# クドわふたー / Kud Wafter (2010)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「能美クドリャフカという 1 人のヒロインの物語を読み切る」こと、その一点に集約される。Key 第 8 作・『リトルバスターズ! エクスタシー』の派生ヒロインを単独主役へ昇格させた IF スピンオフであり、構造上ルートはクドリャフカ周辺キャラに限定され、攻略の幅ではなく「クドという 1 人の少女を見届ける」濃度に振り切られている。

`concept` に明記された「IF 設定のスピンオフ恋愛 ADV」、`target` の「特にクドリャフカ推しのキャラクター愛好家」という記述からも、本作はゲーム的選択や育成ではなく、特定キャラへの感情移入を商品価値の中心に置いていることが分かる。`social_axis` が null である点も、他者との関わりではなくテキスト単体で消費される NAR 型作品であることを裏付ける。`secondary` は空で、報酬ループや体験ループの副流もない、純粋なノベル消費型である。

## 物語の駆動構造

リトバス本編の世界線を借りつつ、クドリャフカに視点を固定して「ヒロイン 1 人 × ファンディスク的尺」で物語を編む構造を採る。攻略本体の選択肢は最小限にとどめ、テキスト密度と CG・音楽の演出に比重を寄せた、Key 系王道のノベル設計である。

| レイヤ | 本作での実装 | 物語駆動への寄与 |
|---|---|---|
| 視点 | クドリャフカ単独ヒロイン IF | 母艦作のサブを「主役」に昇格し既存ファンの未消化欲求を回収 |
| ルート構造 | 中核ヒロイン + 周辺サブヒロイン少数 | 攻略の広さより 1 人の深掘りを優先 |
| テキスト密度 | Key 標準の長文ノベル尺 | 読了体験そのものを商品化 |
| 演出資産 | 樋上いたるキャラ + Key サウンド | 既存ファンの音楽 / 絵柄期待値に直結 |
| 移植リレー | PC → PSP → Vita → Android → Switch (Converted Edition, 2023) | 13 年越しの読者再獲得 |

母艦作の世界観・キャラデザ・サウンドラインを継承するため、新規ユーザが「物語に入る初期コスト」をリトバス側が肩代わりしている点が、駆動効率の本質である。

## バズ要因 / 人気の根拠

- スピンオフでありながら 2017 年にクラウドファンディング 7,800 万円超を集め、劇場版アニメ化が実現した点が最大の話題。ファンが資金面で「次の媒体展開」を直接駆動した稀有な事例
- 母艦『リトバス』が Key 屈指の長寿 IP であり、能美クドリャフカが本編内でも人気投票上位常連だったため、単独主役化への需要が事前に可視化されていた
- 2010 年 PC 版以降、PSP / Vita / Android / Switch（2023 年 Converted Edition）と 13 年以上にわたる移植リレーが続いており、世代ごとに新規読者を獲得し続けている
- Key 作品としては『リトバス』本編・EX に次ぐ「派生でここまで延命した」唯一クラスの事例で、ノベルゲーマー界隈での参照頻度が高い

## 企画への示唆

- 母艦タイトルで人気投票上位のサブキャラを単独主役に抜き出すと、新規 IP 立ち上げよりも初期コストを大幅に圧縮できる
- ファンディスクをクラウドファンディングと組み合わせると、次媒体（アニメ化等）展開の事前需要検証と資金調達を同時に達成できる
- ノベルゲームは PC → 携帯機 → スマホ → 据置の順で 10 年スパンの移植リレーを設計しておくと、1 本のテキスト資産で世代ごとに新規読者を再獲得できる
- 単独ヒロイン構成は攻略幅を捨てる代わりに「キャラ濃度」を売れるため、推し文化と相性が良く、グッズ・楽曲・コラボ展開の起点になりやすい
- IF 設定にすることで本編の正史と矛盾せずに新規物語を量産でき、シリーズ世界観の整合性リスクを抑えられる
