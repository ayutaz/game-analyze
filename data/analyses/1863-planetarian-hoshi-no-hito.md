# planetarian 〜星の人〜 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「読む」── 屑屋とロボット少女ゆめみの邂逅、そしてその後屑屋が辿る一生の旅路を、テキストとして追体験することにある。VN という形式上、選択肢分岐や戦闘・育成は最小化され、画面に流れるテキストと立ち絵・BGM を読み進める一本道のシナリオ消費が体験の中核を占める。

`concept` が示す通り本作は短編 VN『planetarian』本編に対する後日譚（屑屋が「星を語る老人」になるまで）であり、`target` も「短時間で読み切れる物語消費を求める VN 読者」と明示されている。`social_axis` が null なのは、本作にマルチプレイ／非同期／観戦の社会的レイヤが存在しないことの裏返しで、純粋な単線型 NAR であることを示すデータ的なシグナルでもある。`secondary` も空で、副次動機を設けないシンプルな物語特化型として整理されている。

## 物語の駆動構造

本作は本編『planetarian』が築いた「閉鎖された廃墟プラネタリウム × 壊れたロボット × いつか、きっと──」という未完の約束を、時間軸を数十年伸ばして回収する構造を取る。短編 VN 単体ではなく、本編 → 雪圏球（ゆきまりも）→ 星の人 という連作読み物のラストピースとして機能している点が固有の駆動装置になっている。

| 構造要素 | 本作での実装 | 物語駆動への寄与 |
|---|---|---|
| 時間軸 | 屑屋の老年期から本編を回想 | 既プレイヤーに「答え合わせ」の動機 |
| 視点 | 子どもたちに「星を語る」枠物語 | 本編テーマ（伝承・継承）の自己言及化 |
| 連作との関係 | 本編 / 雪圏球 / 星のドームより / 星の人 を束ねる総集編的位置 | 単独でも、通読でも成立する二層消費 |
| メディアミックス | 2016 年劇場アニメ『planetarian 〜星の人〜』と同期 | アニメ → 原作 VN への逆流入口 |
| プラットフォーム | PC HD 版 / iOS / Android | 短時間読了型として通勤読書の受け皿 |

カレンダーや選択肢で物語を分岐させるのではなく、「本編の余韻を時間で延長する」ことで物語消費を駆動するのが構造的特徴である。

## バズ要因 / 人気の根拠

- 2016 年の劇場アニメ『planetarian 〜星の人〜』公開と同期して HD エディションとサイドストーリーが再注目され、アニメ → 原作 VN の逆流入経路が成立した。
- Key / 麻枝准ブランドの泣きゲー文脈（Kanon・AIR・CLANNAD・Rewrite）に連なる作品として、長期にわたりコミュニティ内で語り継がれている。
- 本編『planetarian』が低価格・短時間で読了できる入門 VN として広く流通しており、その続編である本作は既プレイヤーの「答え合わせ」需要を確実に取り込めた。
- PC（Steam HD 版）／iOS／Android の三系統で配信されており、デバイスを問わず短時間読書として消費可能。売上数値は非公開だが、Key 系 VN の長期残存型販売モデルに沿った息の長い流通を続けている。

## 企画への示唆

- 本編の「未回収の約束」を時間軸後方で回収する後日譚を別商品化すると、既存ファンの再購買と新規アニメ視聴者の遡及購入を同時に取れる。
- 短編 VN を連作で束ねる構成にすると、単独購読でも通読でも成立する二層の消費経路が生まれ、ライフタイム売上を伸ばせる。
- アニメ化と原作 HD 版のリリースを同期させると、メディアミックスがクロスプロモーションとして機能し、両媒体の認知が相互強化される。
- 選択肢・分岐を持たない一本道 VN でも、ブランド（Key / 麻枝准）と前作の余韻という外部資産を持っていれば、純粋な物語消費だけで商品が成立する。
- モバイル配信を併設すると、短時間で読み切れる VN は通勤・隙間時間消費の文脈に乗り、PC 単独配信より到達層が広がる。
