## なぜ NAR 主分類なのか

プレイヤーの主動機は「終末前夜の小さな日常と別れを最後まで読み届ける」ことである。本作は選択肢のないキネティックノベルであり、ゲームメカニクスとしての分岐や攻略要素を一切持たない。プレイヤーに残された行為はテキストを送り、立ち絵と背景の差分を眺め、BGM と声を聞き続けることだけで、ここに自由意志を介在させない設計は「物語の純粋消費」を最上位の体験として据えた選択である。

concept にある通り、本作は『ちいさなほしのゆめ』本編の前日譚として「ゆめみがまだ客を待っていた頃」を切り出した構造を持つ。target が Key 作品（AIR / CLANNAD / Kanon）の読者と原作 planetarian の既読層に明示的に向けられている点も、感情の解像度が事前知識込みで最大化される NAR 型の典型である。secondary が空、social_axis が null であることは「他者との関わり」「副次的な動詞」が一切設計されていないことの裏返しで、NAR の中でも極北のシナリオ中心型に位置する。

## 物語の駆動構造

選択肢を消したキネティックノベルの駆動力は、章立て・回想構造・既知の結末への接続にある。

| 駆動要素 | 本作での実装 | 効果 |
|---|---|---|
| 時系列の置き換え | 本編より前、廃墟化前の街と稼働中のプラネタリウム | 既知の崩壊像から逆算した「失われゆく日常」の感情強度 |
| キャラクター固定 | ロボット「ほしのゆめみ」一人称軸 | 視点の絞り込みで内省と機械の感情処理を可視化 |
| 分岐の不在 | 一本道キネティック | 読み手の責任を消し、消費だけに集中させる |
| 媒体の連動 | OVA クラウドファンディング（支援 7,798 万円）と並走 | アニメ視聴後にゲームで補完、または逆方向の体験が成立 |
| 多言語化 | 2024 年 Steam/Switch 版で英・仏・中翻訳 | 国外の Key 読者層を 3 年遅れで取り込み |

本編の結末を知っている読者にとっては「今読んでいる平穏が後に何になるか」がすべての場面に二重露光として乗る。前日譚という形式は、新規シナリオを書きながら旧作の読後感を再賦活する装置として機能している。

## バズ要因 / 人気の根拠

- 17 年ぶりの planetarian 系列新作という稀少性。原作（2004 年）と本作（2021 年）のブランクが、既存ファンの「もう新作はない」という諦観を裏切る形でニュース価値を生んだ
- 2024 年の Steam / Switch 同時展開で英・仏・中翻訳が付き、国外の Key 系 VN コミュニティに再点火
- 並走した OVA 版クラウドファンディングが支援総額 7,798 万円でサクセス、媒体横断のキャンペーンが相互参照された
- 開発元 Key と Visual Arts の長期 IP メンテナンス力。AIR / CLANNAD / Kanon の読者が地続きで反応する「既知の作家圏」が下支えしている

## 企画への示唆

- 前日譚を作ると、既存ファンの読後感をゼロコストで再起動できる。新作の正当性は「本編をもう一度読み返したくなるか」で測れる
- 選択肢を完全に削ると、シナリオ品質への評価が逃げ場なく一極集中する。書き手の力量が試される代わりに、議論は「何が起きたか」ではなく「どう書かれたか」に集約される
- アニメ／ゲーム／クラウドファンディングを時間差で並走させると、媒体ごとに別ニュースが立ち、合計の話題持続時間が直列より延びる
- 多言語版を本国版から 2〜3 年遅らせて出すと、本国の話題が一度沈静化した後の「再発見」枠で海外メディアに拾われる
- 一人称ロボット視点のように「視点キャラクターの認知が制限されている」設定は、読者側に補完を強要し、感情移入の解像度を意図的に高める
