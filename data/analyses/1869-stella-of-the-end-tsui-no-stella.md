## なぜ NAR 主分類なのか

プレイヤーの主動機は「読む」── 運び屋ジュードと人間になりたいアンドロイド・フィリアの旅路を、選択肢のない一本道で **読み進める** ことに収束する。

本作は Key が `Planetarian` の延長として打ち出した「キネティックノベル」名義の SF 短編で、分岐ルートも好感度パラメータも存在しない。プレイヤーが介入できるのはページ送りのみで、提示される文章と演出を順番に消費する以外の遊び方が用意されていない。`concept` に明記された通り「読ませる SF 短編」という設計意図そのものが NAR の定義（物語を見ること／追体験することに特化）と一致するため、`primary: NAR` 以外に置きようがない。`target` も Key 作品ファン・田中ロミオの SF 読者・泣きゲー / キネティックノベル経験者という「テキストを読む文化」を共有した層が前提になっており、`social_axis: null`（他者との関わり軸は不在）も「一人で読む」体験の純度を担保している。`secondary` が空であるのも、副次的な収集・対戦・育成といった REW/EXP 要素が意図的に削ぎ落とされているからである。

## 物語の駆動構造

「読ませる」キネティックノベルは、分岐や好感度に頼らず別軸の駆動力で読者をページ送りさせる必要がある。本作の駆動構造は以下のように分解できる。

| 駆動レイヤー | 本作の実装 | 効果 |
|---|---|---|
| 世界設定 | 地球が人類の世界ではなくなった遠未来、運び屋という旅路の職能 | 移動＝物語進行が自然に一致 |
| キャラ二項関係 | 人間ジュード × 人間になりたいアンドロイド・フィリア | 「人間とは何か」を問う対話構造に直結 |
| 作家性 | 田中ロミオ脚本 ×「Planetarian」系譜の Key SF | 既存ファンに読解コードを共有させる |
| 演出 | SWAV 音声 / キネティック演出（選択肢なし） | 映画的なペース管理を作家側が完全制御 |
| 尺 | 短編（CLANNAD のような長編ではない） | 一気読みでカタルシスを設計しやすい |

選択肢を捨てる代わりに、**ロードムービー型の旅路** と **異種二人組による哲学的問答** をフレームに採用し、読者の集中力を分散させない構造になっている。

## バズ要因 / 人気の根拠

`popularity` に記された「Key×田中ロミオ×SWAV のタッグ」と多言語展開を出発点に分解する。

- **作家三者の組み合わせ**：Key（CLANNAD / AIR / Planetarian）× 田中ロミオ（CROSS†CHANNEL / AURA）× SWAV（Key 系音楽ブランド）という、ノベルゲーム文脈で固有名詞として通る三者を集約した企画体制。
- **Planetarian の系譜**：選択肢なしのキネティック SF 短編という形式そのものが「Planetarian の正統後継」として読まれ、長編 Key 作品とは別の文脈で語られた。
- **マルチプラットフォーム展開**：PC 版から Switch / iOS / Android、Steam では英語・簡体字を含む多言語版へ広げ、国内 PC アダルトゲーム文化圏の外（英語圏・中華圏の VN コミュニティ）にも到達。
- **Visual Arts 配給**：Key 単独ではなく Visual Arts ブランドでの流通により、家庭用・スマホへの移植が標準化された。

## 企画への示唆

- **分岐を捨てると作家のペース制御が完全になる** → 短編尺で映画的カタルシスを設計したいなら選択肢を排除する選択は合理的。
- **異種二人組のロードムービーを採用すると、世界観説明と哲学的問答を同じシーンで両立できる** → 移動が情報開示の自然な口実になる。
- **作家三者（脚本・ブランド・音楽）を固有名詞として束ねると、コアファンの初動を確保できる** → 名前で売れる体制は短編ほど効く。
- **PC 発のキネティックノベルでも、Switch / スマホ / Steam 多言語を初期計画に入れると VN 文化圏の海外読者へ届く** → 翻訳前提でテキスト量を短編に絞る判断と整合する。
- **既存 IP の「系譜」を明示的に名乗ると、新規 IP でも文脈消費が始まる** → 「Planetarian の系譜」表記が読解の予熱として機能する。
