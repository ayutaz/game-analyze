# プリマドール 〜無銘天嶺〜 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「アニメ本編で語られなかった箒星の過去を**読む**」ことに集約される。本作は選択肢を持たないキネティックノベル形式で、ゲームメカニクスとしての判断・育成・収集要素を意図的に削ぎ落としている。`concept` が明示する通り、Key／ビジュアルアーツのメディアミックス『プリマドール』の外伝エピソードを線形に消費させる読み物特化型 VN であり、プレイヤーは物語の受け手として座り続ける。

`target` が「アニメ／キネティックノベル第1作で『プリマドール』に触れた既存ファン」と限定されている点も NAR 寄りを補強する。新規層に対する間口（チュートリアル・キャラ紹介・ゲーム的成長曲線）を提供せず、本編世界観を既に内面化したプレイヤーに**補完情報**を渡す構造になっている。`social_axis` が null なのは EXP 軸の他者関係性が成立する余地がそもそも存在しないためで、副次動機を示す `secondary` も空配列のままである。報酬ループや収集も持たないため、NAR 単独カテゴリに収まる。

## 物語の駆動構造

キネティックノベルというフォーマットは選択肢を捨てる代わりに、別軸の駆動装置を載せて読了動機を維持している。本作の場合、駆動構造は「メディアミックス本編との接続」と「視聴覚演出のフル投入」の二層に分解できる。

| 駆動要素 | 本作での実装 | 物語消費への効果 |
|---|---|---|
| 選択肢 | なし（キネティックノベル） | 分岐管理コストを 0 にし、執筆リソースを単一線の濃度に集中 |
| 章立て | 箒星単独視点の外伝1本 | アニメ本編で削られた過去パートをピンポイント補完 |
| 接続点 | アニメ本編キャラ・世界観・地名 | 既存ファンの脳内設定と結合し、追加情報の重みが増幅 |
| キャスト | アニメ版フルボイス継続 | 声の継続性が「同じ作品の続き」として認知される |
| 音楽 | 書き下ろし楽曲 | Key 作品の伝統である楽曲駆動の感情ピークを再現 |
| プラットフォーム | PC 単独 | パッケージ／DL いずれも既存 VN ユーザー導線に絞り込み |

第1弾 KN から続く「アニメ本編の余白を埋める短編 VN」という商品設計は、本編 1 クールでは描けない過去・心情・サブキャラを切り出して提示する補完財として機能する。プレイヤーの動機は新規発見ではなく**穴埋めと答え合わせ**に近い。

## バズ要因 / 人気の根拠

- `popularity` 記述の通り、2024 年 5 月 31 日発売の **Key 原作メディアミックス『プリマドール』キネティックノベル第2弾**として、シリーズの継続発売実績そのものが既存ファンへの告知装置になっている
- アニメ本編で描かれなかった**箒星の過去を補完**するという外伝的位置付けが、本編視聴済み層の「もっと読みたい」需要にピンポイントで応える
- 開発が Key、販売が Visual Arts という Key 作品系譜の信頼ブランドが、未プレイ層に対する品質保証として機能
- 第1弾 KN との連作構造により、シリーズ単位での収集動機が二次的に発生（ただし本作内に収集要素はない）

## 企画への示唆

- **アニメ→VN への外伝供給は本編で削った場面に絞ると刺さる** — 本編リメイクではなく余白補完にすると、既存ファンの脳内設定と衝突せず追加価値が立つ
- **キネティックノベル形式は選択肢コストを楽曲・声・絵に再投資できる** — 分岐管理を捨てる代わりに視聴覚演出の濃度を上げるトレードが成立する
- **アニメキャストのフルボイス継続はメディアミックス連続性の最重要シグナル** — 声が変わると同一世界観として認知されにくく、補完財としての価値が落ちる
- **ターゲットを既存ファンに絞り込めば新規導線設計を省略できる** — マーケティング・チュートリアル・初心者配慮を捨てる代わりに、深度と単価を確保する選択
- **シリーズ番号（第1弾／第2弾）を付けると次作への期待が前作のうちに発生する** — 単発タイトルでは生まれない「次はどのキャラの外伝か」という待ち動機を作れる

## 比較で見るべきタイトル

- **CLANNAD / リトルバスターズ!**（Key／NAR／ソロ） — 楽曲駆動と長尺物語消費という Key 作品の原型
- **Summer Pockets**（Key／NAR） — Key のキネティックノベル系譜と本編 VN の中間形
- **Fate/stay night Réalta Nua**（Type-Moon／NAR） — アニメ展開後にフォーマットを変えて再供給する VN メディアミックス事例
- **ヘブンバーンズレッド**（Key 麻枝准／NAR+REW） — 同じ Key 系譜が運営型に振った対極事例、線形 VN とのリソース配分差が比較対象
