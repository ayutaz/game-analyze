# ヒビキの魔法 (Hibiki no Mahou) — 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「麻枝准的な喪失と再生の物語を読み切る」ことに尽きる。戦闘も収集要素もなく、紅茶しか上手く淹れられない弟子・ヒビキと魔法使いシロツキの師弟関係を、出会いから別れまで一本の線で追体験するためにプレイされる。

concept にある「喪失と再生」は CLANNAD / AIR / Angel Beats! と地続きのモチーフで、target も「Key/麻枝准作品のファン層、泣きゲー愛好者」と明示されている。social_axis が null なのも、他者との同期的なやり取り（対戦・協力・観戦）が一切設計に組み込まれていないからで、純然たる単独物語消費体験として閉じている点が NAR を裏付ける。secondary が空なのは、副次動機（収集・育成・対戦）を意図的に排除した「物語純度を優先した設計」であることの表れでもある。

## 物語の駆動構造

原作漫画『ヒビキのマホウ』（いづみ礼）を下敷きに、Key/麻枝准的な構成へ翻案している。物語を駆動するのは「未熟な弟子の成長カーブ」と「師との別れの不可避性」という 2 軸で、両者を交差させて読者の感情を一方向にドライブする泣きゲー定石を踏襲する。

| 駆動要素 | 役割 | プレイヤーへの効果 |
|---|---|---|
| 弟子ヒビキの未熟さ（紅茶しか淹れられない） | 共感入口・成長余地の提示 | 序盤の親近感、後半の達成感の落差 |
| 師シロツキの存在 | メンター + 別れの対象 | 別離フラグの埋め込み |
| ファンタジー世界の魔法ルール | 日常と非日常の往復 | 序盤の世界観吸収 → 後半の代償構造 |
| 麻枝准的モチーフ（喪失と再生） | 終盤の感情ピーク設計 | カタルシスの方向付け |

選択肢分岐や攻略順による複数ルートを大々的に押し出すタイプではなく、原作付きの「読み切り型」短編に近い形式で、章立ては師弟の関係深化 → 危機 → 別れ → 残された者の再生という古典構成を踏む。

## バズ要因 / 人気の根拠

- popularity 記述どおり、麻枝准・いづみ礼という原作チームの組み合わせが Key/Visual Arts ファンの間で局所的に話題化した
- Key ブランド本体の主力（CLANNAD、AIR、Angel Beats!、Summer Pockets 等）の文脈で語られ、世界売上 30K+ という規模からも「広く一般化はしていないが Key 系コミュニティで認知された」位置づけ
- sales_jp が「—」で公開資料が乏しいことそのものが、商業 IP というより**ファンサービス的な小品**として流通したことを示唆する
- 出典が Wikipedia / Tropedia / Jun Maeda 個人ページ中心で、Famitsu などの主要メディアレビューが不在 ＝ 一般メディア露出より口コミ駆動

## 企画への示唆

- 既存 IP（原作漫画）にブランド作家（麻枝准）を交差させると、両方のファンダムに小規模だが確実な需要が生まれる
- 物語純度を上げたいなら secondary を空にする選択肢を持つ ＝ 副次動機を足すたびに「読む動機」は薄まる
- social_axis を null に振り切ると同期的バズは起きないが、ファンダム内での長期的な語りの対象になりやすい
- 「紅茶しか淹れられない」のような**主人公の極端な不器用さ**を 1 行で表現できると、序盤共感と後半成長の落差を最大化できる
- 商業規模 30K+ クラスの作品は出典提示の薄さが宿命なので、企画段階で「公開数値が出にくい前提のマーケ設計」を持つ

## 比較で見るべきタイトル

- CLANNAD（Key / 2004） — 麻枝准的「喪失と再生」原型、家族関係と日常から非日常への跳躍
- AIR（Key / 2000） — 別れの不可避性を中核に据えた短編構成の参照点
- Angel Beats!（Key / 2010-） — 麻枝准が原作・脚本に深く関わった近年作、世界観共有のヒント
- Summer Pockets（Key / 2018） — 同時期 Key 作品、夏・少女・別れの再演
- planetarian 〜ちいさなほしのゆめ〜（Key / 2004） — 短編フォーマットで泣きゲー骨格を凝縮した先例
