# planetarian HD — 詳細分析

## なぜ NAR 主分類なのか

このタイトルでプレイヤーがしていることは「読む」に尽きる。選択肢もマルチエンディングもなく、廃墟のプラネタリウムを舞台にジャンカーと接客ロボット「ほしのゆめみ」の数日間を 4–5 時間で読み切るキネティックノベルである。`concept` が明示する通り分岐は存在せず、プレイヤー側の意思決定で物語が変質する余地はゼロに設計されている。

`target` が示すのは Key の Kanon / AIR / CLANNAD 系叙情譚を求める読者層、および「短時間で完結する SF 短編を一気読みしたい」層で、いずれも能動的なゲーム体験ではなく物語消費が主動機である。`social_axis` が `null` なのも、ソロで読み切る前提で他者軸の設計余地がないからだ。`secondary` も空で、報酬や体験の副次性すら持たない、純粋な NAR タイトルとして位置づけている。

## 物語の駆動構造

planetarian の駆動原理は「閉じた舞台 × 限定された時間 × 一本道」の三層で成立している。崩壊後の世界という SF 外枠を背景に、廃墟のプラネタリウム 1 室と数日間という極小スケールに物語を圧縮し、その密度で読者を引きつける。

| 層 | 装置 | 効果 |
|---|---|---|
| 外枠 | ポストアポカリプス SF | 「いつ終わるか分からない世界」というメタ時計を常駐させ、別れの予感を初手から付与 |
| 舞台 | 廃プラネタリウム 1 室 | 移動・探索を捨て、二人称関係性に焦点を集中 |
| 時間 | 数日間 + 一本道 | プレイヤーに迂回経路を許さず、結末への落差を制御 |
| キャラ構造 | 無口な軍人ジャンカー × 過剰に明るい接客ロボット | 感情曲線を一方向に積み上げる対比配置 |
| 文体 | キネティック（選択肢なし） | 解釈の責任を読者ではなく作家側に固定 |

選択肢を捨てることで、Kanon / CLANNAD のような攻略順最適化ゲームとは違い、「全員が同じ結末を読む」前提で泣きの強度を最大化できる構造になっている。HD 化はあくまで立ち絵・CG 解像度のリファインで、構造そのものは 2004 年の原作版から不変。

## バズ要因 / 人気の根拠

- `popularity` が示す通り、Key の中でも SF 短編という異色枠で、Kanon / AIR / CLANNAD の長編泣きゲー系列とは別系統のファンを獲得した。
- 2016 年に HD 版で Steam に展開し、後年 Switch 版も追加。`sales_world` で 200K+ と短編 VN としては安定した規模を継続。
- 原作は 2004 年の同人 / 廉価ノベル、その後 OVA → 劇場アニメ → Web アニメ → HD ノベル → Switch 移植と、20 年超にわたって複数メディアで延命している長寿コンテンツ。
- 4–5 時間で完結する短さが「VN の入り口」として SNS でも勧めやすく、海外 VN 入門タイトルとしての引用頻度が高い。

## 企画への示唆

- 選択肢を捨てると、物語の責任を作家側に集中できる → 「読者全員が同じ結末を読む」を前提にできれば、泣きの強度を最大化しやすい。
- 舞台を 1 室・登場人物を 2 人に圧縮すると、移動・探索・サブキャラ管理のコストを 0 にでき、感情曲線の制御に予算を全振りできる。
- 4–5 時間という尺は「SNS で勧めやすい」「実況で 1–2 配信で完結する」境界に置くと拡散効率が上がる → 30 時間級と差別化できる。
- ポストアポカリプスの世界観を「外枠の時計」として使うと、別れの予感を初手から無料で付与できる → 終盤の落差設計が安く済む。
- HD リマスター展開は、立ち絵・CG・解像度のリファインだけで Steam / Switch という新規プラットフォーム層を取りに行ける → 構造を弄らないリマスターの教科書。
- 旧作 IP は、メディアミックス（OVA・劇場アニメ・Web アニメ）を 10–20 年単位で重ねていくと、それぞれの世代の入り口になり長期売上を生む。

## 比較で見るべきタイトル

- CLANNAD / AIR / Kanon — 同 Key 系の長編分岐型泣きゲー。planetarian は「分岐を捨てた短編版」と位置づけられる。
- ATRI -My Dear Moments- — 同系統のポストアポカリプス × 少女ロボット叙情譚、より新世代の作例。
- Steins;Gate Elite — キネティックに近い一本道展開で SF 短中編を読ませる構造。
- ひぐらしのなく頃に — 選択肢なしの一本道で多巻構成、同じく VN ジャンルの「読み物」極北。
