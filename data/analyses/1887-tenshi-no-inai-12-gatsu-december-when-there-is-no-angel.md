# 天使のいない12月 / Tenshi no Inai 12-gatsu — 分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「軽さを装う主人公の独白を読み、関係が壊れていく過程を見届ける」こと、すなわち**読む**である。Leaf 系 ADV としてはセーブ＆選択肢の負荷が低く、ゲーム的成長要素・収集要素はほぼ存在しない。`concept` にある通り、信条として掲げた「軽く・薄く・小さく」が性的関係の発生によって徐々に破綻していく**冬の群像劇**を、テキストの密度で読ませる作品である。

`target` は Leaf / Key 系のシリアスな ADV を読み込む 20–30 代と明示されており、依存・嫉妬・倦怠といった重い人間ドラマを直視する読者を狙う。一人で黙々と読み進める単独消費型で、`social_axis` は null（NAR なので体験の社会軸は不要）。`secondary` も空で、ハイブリッド要素を持たず**純粋な物語消費**に振り切られている点が分類の決め手になる。

## 物語の駆動構造

本作の駆動装置は「軽さの信条 → 関係の発生 → 信条の崩壊」という 3 拍構造を、**ADV ウィンドウと NVL（ビジュアルノベル）形式の演出切替え**で読み分けさせる点にある。会話シーンは ADV ウィンドウで距離を取らせ、独白は NVL で全画面の地の文に切り替えて読者を主人公の内側に閉じ込める。同じテキストでも器を変えることで「対外的な軽さ」と「内側の重さ」のギャップを構造化している。

| レイヤー | 形式 | 駆動する感情 |
|---|---|---|
| 会話シーン | ADV ウィンドウ | 軽口、社会的演技、距離 |
| 独白 | NVL 全画面 | 倦怠、嫉妬、自己嫌悪 |
| 性的シーン | CG イベント | 信条が壊れる転換点 |
| 冬の時系列 | 12月の章進行 | 出口のないカウントダウン |

「12月」というタイトル自体が**期日付きの密室**として機能し、季節の閉塞感とクリスマス＝祝祭の不在が、関係修復の不可能性を背景音として鳴らし続ける。

## バズ要因 / 人気の根拠

- ErogameScape で **79/100** の高評価が長期維持され、Leaf 内の異色作・鬱ゲー文脈の参照点として語り継がれている。
- 発売当時 PUSH! 誌など成人向け誌で特集が組まれ、「軽さを信条にした主人公が壊れる」筆致がコミュニティで話題化した。
- Leaf ブランドの中で『To Heart』系の王道ラブコメから外れた**ブランド内コントラスト**として、後年のレビュー言及で繰り返し取り上げられる。
- 売上は公表されておらず（`sales_jp: —` / `sales_world: unknown`）、口コミと評点で長期残存しているタイプの作品である。

## 企画への示唆

- **同一テキストでも器（ADV/NVL）を切替えると、距離感と没入度を演出で分離できる**：会話＝距離、独白＝侵入というレイヤリングは ADV 一般に応用可能。
- **「軽く生きる」という信条をプレイヤーに刷り込んでから壊す**と、崩壊の落差そのものが物語の推進力になる：価値観の提示 → 反証イベントの順序設計が効く。
- **季節と期日（12月）を物語の壁として置く**と、解決の猶予を可視化でき、出口のなさを演出できる：カレンダー進行は RPG だけのものではない。
- **明示的な「悪役」を置かず、性的関係そのものを破壊装置にする**と、加害／被害の二分法を崩した中間域のドラマを書ける。
- **公式売上ゼロでも 20 年語られる作品が成立する**：評点コミュニティ（ErogameScape 等）への露出と、ブランド内コントラストの設計で長期想起を作れる。

## 比較で見るべきタイトル

- 『SWAN SONG』（Le.Chocolat, 2005）— 同じく鬱ゲー文脈で語られる重量級 NVL。
- 『最果てのイマ』（ザウス, 2005）— 関係性と自我の崩壊を扱う NVL。
- 『シルバー』（ねこねこソフト, 2000）— 静かに壊れていく Leaf/Aquaplus 周辺の系譜。
- 『White Album』（Leaf, 1998）— 同社の重い恋愛三角形 ADV の前世代参照。
- 『CROSS†CHANNEL』（FlyingShine, 2003）— 同年発売、閉塞した関係性を反復構造で描く NVL。
