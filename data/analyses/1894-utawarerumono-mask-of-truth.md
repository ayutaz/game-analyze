# うたわれるもの 二人の白皇 — 三部作完結編としての物語消費装置

## なぜ NAR 主分類なのか

このタイトルでプレイヤーが投じる時間の主動機は、**「二作分積み上げた伏線が回収されるところを読む」** である。前作『偽りの仮面』のラストで宙吊りにされた政変・正体・戦争の各プロットを、ハク視点の長編シナリオとして読み進めることが体験の中心であり、SRPG パートはその進行を区切るペーシング装置に近い。

JSON の concept が示すように本作はあくまで「ファンタジー戦記 AVG の完結編」であり、target も「前作を遊んだプレイヤーが事実上の前提」と前作読了を要求している。ヘックス制 SRPG は戦況描写を読者に体感させる演出として効くが、戦闘単体の発明性で売る作りではない。secondary に `EXP` を置いたのは、その SRPG パート（BREAK / CRITICAL / 共鳴連携など）が NAR 一辺倒では成立しない章間メリハリを担っているためで、`social_axis: null` であることも一人で物語を消費する設計と整合している。

## 物語の駆動構造

三部作完結編としての構造は、「**回収すべき問い**」と「**回収する装置**」の二層に分解できる。

| 層 | 要素 | 本作での実装 |
|---|---|---|
| 問いの設定 | 散りゆく者 / 偽りの仮面で残された伏線 | ハクの正体、ヤマトの政変、トゥスクルとの戦争、ウィツァルネミテアの存在 |
| 駆動装置 | 章＝戦記の局面 | 戦闘パートが「次章への扉」として配置され、各章クリアで世界情勢が一段進む |
| 感情の梃子 | ヒロイン群像と仲間の喪失 | クオン、ルルティエ、ネコネ、アンジュらの私的物語が戦記の進行に重ねて回収される |
| 区切り | SRPG ヘックス戦闘 | 戦記の山場ごとに挿入され、長尺シナリオの可読性を担保 |
| 終端 | エンディング | 二部作分の問いを一括処理する長尺ラスト、ボリュームを終章に集中 |

選択肢分岐はほぼ無く、**一本道の戦記を「章＝戦闘」のリズムで読ませる**ことで、20-40 時間級の長尺を破綻させずに配信し切る構成になっている。

## バズ要因 / 人気の根拠

- 三部作完結編という配信フォーマット自体が、前作『偽りの仮面』(2015) を遊んだ層を確定読者として抱えた状態で出る前提だった
- popularity 文の通り、二作分の伏線回収を一気に行う構成が「シナリオ評価が極めて高い」評価軸を生み、AQUAPLUS 代表作としてシリーズの参照点になった
- AVG + SRPG という「読む時間と遊ぶ時間が交互に来る」設計が、純 AVG の読了脱落と純 SRPG の物語希薄化の両方を避ける形でレビュー評価を底上げした
- PS4 / PS3 / PS Vita / PC とマルチプラットフォーム展開し、後に Steam 版も出たことでファン層の入口を狭めずに長期販売へつないだ

## 企画への示唆

- **「前作の宙吊りを意図的に作って次作で回収する」**設計に振り切ると、続編のマーケティングコストを既存読者に依存できる代わりに、単体作の新規参入難度は跳ね上がる
- **長尺シナリオに別ジャンルのミニゲームを章区切りとして挟む**と、ノベル単体では脱落する 30 時間級のテキスト消費を完走させやすくなる
- **戦闘システムを物語ペーシングの装置と割り切る**と、戦闘の発明性で勝負しなくてもジャンル混成として評価される（逆に戦闘を主役に置くなら別物の設計判断になる）
- **ヒロイン群像を「戦記の駒」として配置**すると、私的感情の回収と戦況の前進を同じシーンで二重に処理でき、終盤の感情密度が上がる
- **三部作配信の最終巻に終章ボリュームを集中させる**と「ここまで読んだ甲斐」の感覚を演出できるが、未完結のまま打ち切られるリスクと表裏である

## 比較で見るべきタイトル

- うたわれるもの 偽りの仮面 (2015) — 直接の前作。問いを残して終わる第 2 巻側の設計
- ファイアーエムブレム 風花雪月 (2019) — AVG + SRPG ハイブリッドの後発例、章立てとキャラ群像の組み方
- 戦場のヴァルキュリア (2008) — 戦記 SRPG として物語と戦闘の比率設計を比較する参照点
- Fate/stay night (2004) — 一本道長尺ノベルがファンダムの参照点になる過程として比較可能
