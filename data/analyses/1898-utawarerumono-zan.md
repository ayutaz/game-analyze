# うたわれるもの斬

## なぜ EXP 主分類なのか

プレイヤーの主動機は「既知のキャラを自分の手で動かして雑兵をなぎ倒す爽快感を味わう」ことであり、本作は読む（NAR）よりも振り回す（EXP）に重心がある。

『うたわれるもの』本編は ADV + SRPG の二段重ねで、戦闘はターン制グリッド上の指揮プレイだった。本作は同じ世界観・同じシナリオ範囲を 3D アクションに作り替え、ハクオロ / クオン / オシュトル等を直接スティック操作で動かせるようにした、いわゆる無双系ハック＆スラッシュである。concept にあるとおり「ストーリー追体験と爽快アクションの両立」が設計目的であり、シナリオ自体は本編の縮約・ダイジェストに留まる。social_axis は「ソロ」で、オンライン PvP も非同期ランキングもなく、各章ステージを 1 人で順番に進める純シングル構造。

secondary が NAR なのは、ターゲットの大半が原作 ADV プレイ済みでキャラ愛から入る既存ファンであり、戦闘の合間に挟まる会話パートと本編シナリオ進行が購入動機の半分を支えているため。物語は新規には書き下ろされていない（本編の追体験）ので主分類には届かない。

## 体験の核 / 設計の発明

中核は「無双テンプレート × IP キャラのモーション固有化」。無双系のフォーマットを借りつつ、各キャラを原作の戦闘スタイル（弓・双剣・薙刀・霊術 等）で差別化した。

| 設計レイヤ | 内容 | 既存無双系との差分 |
|---|---|---|
| マップ規模 | 章単位の中規模戦場、目標撃破式 | 真三國無双より狭く、章クリア型 |
| 操作キャラ | シリーズ主要キャラ十数体を切替 | 各キャラに固有モーションと術技 |
| 進行軸 | 本編 2 作のシナリオを章で消化 | オリジナル時系列ではなく追体験型 |
| 育成軸 | スキル習得 + 武具強化の浅い RPG 層 | 派手な装備厳選ループは持たない |
| 難度 | 原作未プレイ層にも開く軽め調整 | コア無双プレイヤーには物足りない可能性 |

social_axis「ソロ」の効果として、PvP バランス調整の制約から解放されたぶん、キャラ個別の派手な必殺演出と原作再現に振り切れる構造になっている。

## バズ要因 / 人気の根拠

- 「うたわれ初の無双系」という IP × ジャンルの組み替え自体が話題化フックになり、シリーズファンに初動を作った
- PS4 初週 約 4.5 万本、累計 約 6 万本（sales_jp 60K+）と、IP スピンオフとしては想定内のレンジで着地
- 開発が Tamsoft（閃乱カグラ系の 3D アクションで実績ある中規模アクションスタジオ）で、AQUAPLUS の IP 側と分業し IP オーナーがアクション専業に外注する構図
- 2025 年に Steam 移植が発表され、北米でも NIS America 経由で英語版が継続展開された——日本市場で消化された後に海外ロングテール需要を拾う典型動線

## 企画への示唆

- 既存 IP を別ジャンルに移植すると、本編プレイ済み層の「キャラを別の動詞で動かしたい」需要が即座に立ち上がる
- ADV/SRPG など読み物寄り IP を 3D アクション化する際は、シナリオを新規書き下ろさず「追体験」に絞ると開発コストと既存ファン満足のバランスが取りやすい
- 無双テンプレートはキャラ数 × 固有モーションが工数の主軸になるため、IP 側のキャラクター層が厚いほど ROI が高い
- アクション開発を専業スタジオ（Tamsoft 等）に外注し、IP オーナー（AQUAPLUS）はシナリオ監修に専念する分業は、中規模 IP のスピンオフで再現性が高い
- 国内消化後 5〜6 年で Steam 移植を打つと、海外ファン層と価格訴求でセカンドピークを作れる
