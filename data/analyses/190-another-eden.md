# アナザーエデン

## なぜ NAR 主分類なのか

プレイヤーの主動機は「クロノ・トリガー系譜のシナリオを **読み進める**」ことにある。本作は加藤正人（クロノ・トリガー / クロノ・クロス脚本）と光田康典（同音楽）がメインスタッフとして名を連ねる「時を超えるソロRPG」であり、concept が明示する通り **スタミナ制と PvP を排した「シングルプレイ前提のスマホ RPG」** という設計選択がそのまま分類根拠になる。`social_axis` を null としているのは、フレンド機能・協力戦・ランキングといった他者軸の中核メカニクスが存在しないためで、プレイヤーは他人の存在をほぼ意識せず時の狭間を旅する。

target が「スーファミ期 JRPG 原体験のある 30〜40 代」と限定されている通り、本作の物語消費は懐古層に対する **シナリオの長さ・章立て・世界観の濃度** で駆動されている。secondary に REW が入るのはガチャと育成が存在するため（キャラ＝報酬という構造）だが、ガチャを引く動機は「ステータスではなく加藤シナリオで描かれるキャラ章を読みたいから」であり、報酬は物語アクセス権として機能している。

## 物語の駆動構造

シングル RPG なのに 9 年運営できているのは、物語消費を継続させる複層構造を持つためである。

| レイヤー | 役割 | プレイヤー動機 |
|---|---|---|
| メインストーリー（古代編〜未来編） | 加藤正人の本筋シナリオ。完結章を順次追加 | 「次の章はいつ来るか」を待つ |
| 外伝（IF / 個別キャラ章） | サブキャラのバックストーリーを長編で深掘り | 推しキャラの章解放を引く動機 |
| シンフォニー / 真伝 | 高難度後日譚。既存キャラ再解釈 | 物語の再読体験 |
| 光田康典楽曲 | 章ごとに新曲。CD 化 | 音楽から章を記憶する |

クロノ・トリガー的な「過去・現在・未来を行き来する」時間構造を中核に据え、章追加のたびに既存マップへの再訪線を引くため、後発章ほど旧章への参照密度が上がる。スタミナがないため一気読みが可能で、**1 章＝1 本の長編 JRPG** に近いボリュームを 8〜12 時間で消化させる。

## バズ要因 / 人気の根拠

popularity「加藤正人脚本 × 光田康典音楽、クロノ系譜の物語」が出発点。

- **スタッフ訴求**: 加藤・光田の名前そのものが宣伝になり、クロノ・トリガー / ゼノギアスのファン層を直接取り込んだ。
- **国内 1,000 万 DL（2021）**: スタミナ・PvP 排除のシングル RPG としては異例の規模。
- **シングル特化という逆張りポジション**: 2017 年当時のスマホ RPG が PvP・ギルド戦・ログボ前提だった中で、「ソーシャルを嫌う JRPG 層」をすくい上げた数少ない設計。
- **長期運営**: 2017 年配信から 9 年、メインストーリーが章追加で継続し、PC（Steam）・Switch にも展開されている。

## 企画への示唆

- **スタミナと PvP を捨てると、ソーシャル疲弊層の受け皿になる**: 残るレイヤーが少ないぶん、シナリオと音楽の純度で勝負する設計に追い込まれる。
- **作家名・作曲家名をフロントに置くと、IP がない新規タイトルでも訴求軸が立つ**: クロノ系譜という「精神的続編」ポジショニングは IP ライセンス費なしで懐古市場を取れる。
- **章追加型の長編 JRPG をスマホで運営すると、1 章のボリュームが課金の言い訳になる**: ガチャを「キャラ章を読む権利」として再定義できる。
- **時間軸を物語構造に内蔵すると、後発章で旧マップに再訪線を引ける**: マップ再利用が懐古ではなく伏線回収として機能する。
- **ソロ前提なら `social_axis` を null にすることが設計上の宣言になる**: 「他人と比べさせない」ことそのものが target 層への約束になる。

## 比較で見るべきタイトル

- **クロノ・トリガー / クロノ・クロス** — 直接の精神的源流。時間軸構造と加藤・光田スタッフの参照点。
- **オクトパストラベラー** — スーファミ期 JRPG 懐古層を狙う家庭用側の同世代作品。
- **ロマンシング サガ リ・ユニバース** — ガチャ＋シングル RPG という近接ポジションだが、PvP・ランキングを併用する点で対照的。
- **崩壊：スターレイル** — 同じ「ソロ JRPG ×ガチャ」だが、PvE 高難度コンテンツとイベント駆動で運営し、章追加よりキャラ追加で回す逆設計。
