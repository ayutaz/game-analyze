# 沙耶の唄 — 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「不可逆に堕ちていく主人公の認知と恋愛を、最後まで見届けるために読む」ことに集約される。ゲームメカニクスは選択肢を数回挟む程度のノベル形式で、戦闘も育成もマップ探索も持たない。`concept` が明示する通り、本作の体験価値は「脳手術後遺症で世界がグロテスクに歪んで見える医学生」と「彼にだけ少女として映る異形」という二者の認識ズレを、固定したテキスト・固定したCGで追体験する点にある。

`target` が「成人向けノベル愛好層／虚淵玄ファン／ラヴクラフト・伊藤潤二的サイコホラー志向の20-40代コアゲーマー」と明示的に読者像で記述されていることも、本作が「遊ぶ対象」ではなく「読む対象」として消費されている裏付けになる。`social_axis` が null なのは、対戦・協力・観戦・非同期いずれの他者軸も持たない単独消費のノベルだからで、これは NAR タイトルとして典型的な構成である。`secondary` は空で、副次動機を持たない純粋ノベルとして分類している。

## 物語の駆動構造

短編ノベル（プレイ時間 5〜8 時間）であることを逆手に取り、虚淵玄は「主人公の認知が読者の認知を侵食する」という一点に物語装置を集中させている。章立てより認知の段階分けが主軸になる。

| 駆動装置 | 機能 | 効果 |
|---|---|---|
| 主観視点の固定 | 匂坂郁紀の見る「肉塊化した世界」を地の文で描写 | プレイヤーが沙耶=美少女と認識せざるを得ない |
| 沙耶の謎の小出し | 何者かを終盤まで明かさない | 倒錯した愛が成立してから正体が反転 |
| 3 エンディング分岐 | 終盤の選択肢で破滅の形だけが変わる | バッドエンド回避ではなく「どの破滅か」の選択 |
| 短尺×高密度 | 5-8 時間で完結 | 一気読み圧が強く、口コミが起きやすい |
| 全年齢化されない設計 | 性描写・グロ描写が物語装置として不可分 | 移植・全年齢化が困難で原典の権威性が保たれる |

選択肢の役割が「ハッピーエンド到達のための正解探し」ではなく「どのエンディングを引き受けるかの宣告」になっているのが、本作の物語駆動の最大の発明である。

## バズ要因 / 人気の根拠

- 発売当時（2003年）から虚淵玄脚本のラヴクラフト的サイコホラーADVとして熱狂的カルト人気を獲得し、ニトロプラスのブランド形成に直結した
- 2011年『魔法少女まどか☆マギカ』ヒットで虚淵玄が一般層に認知され、過去作として遡及的に再評価が進み、新作1本分相当の追加売上を記録した
- Steam / GOG での英語版配信（JAST USA）が海外コアファン層で話題化し、英語圏 Lovecraftian / Eroge VN 文脈でも継続的に言及される作品になっている
- 短尺（5-8 時間）×高密度の構造がレビュー・配信・考察記事と相性が良く、口コミが断続的に再燃する長期テールを持つ

## 企画への示唆

- 主観視点と認知歪曲を地の文で固定すると、プレイヤーは作者の意図した「誤読」を強制的に受け入れる
- 短尺ノベルで終盤に分岐を集中させると、「正解探し」ではなく「どの結末を引き受けるか」の倫理的選択になる
- 脚本家個人のブランド（虚淵玄）を前面に出すと、後年その作家が一般化した瞬間に過去作が遡及的に再評価される長期収益化が起きる
- グロ／性描写を物語装置と不可分に設計すると、全年齢移植が困難になる代わりに原典の権威性と希少性が保たれる
- 海外ストア（Steam/GOG）への英語ローカライズは、コア向けノベルでも 10 年遅れで国外コミュニティを獲得しうる
