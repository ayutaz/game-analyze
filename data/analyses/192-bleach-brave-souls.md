# BLEACH Brave Souls

## なぜ REW 主分類なのか

プレイヤーの主動機は「原作キャラの衣装違いを集め、育て、編成する」ことにある。3Dアクションは爽快だが、ユーザーが10年遊び続ける駆動力は新キャラ・新衣装の追加とその所有欲・コレクション欲である点で、本作はREWに収まる。

concept にあるとおり、本作は「BLEACH IPを3Dアクション+ガチャで運用する長寿スマホRPG」であり、原作各篇のキャラを衣装違いで継続実装し続ける設計が中心軸になっている。target が示す「原作BLEACHのファン層（30代男性中心）」は、ゲーム体験そのものよりも「推しキャラの新フォルムが実装されたから引く」という IP コレクション動機で動く層だ。social_axis が null（事実上ソロ）であることも、本作の楽しみがランキングや対人ではなく「自分の手元のキャラを揃える」ところにあることを裏付ける。

secondary に EXP が入っているのは、片手で操作できる3Dアクションそのものに一定の手触りがあり、ガチャで引いたキャラを使う「動かす楽しさ」が報酬獲得後の二次的な満足を担保しているためだ。

## 報酬ループの組成

| 層 | 中身 | 期待される効果 |
|---|---|---|
| 引く | 原作篇別・衣装別の追加実装、復刻、★6召喚 | 推しキャラの新フォルム解禁による瞬間最大風速 |
| 育てる | キャラLv、技ソウル、リンクスロット、超覚醒 | ガチャで終わらせず、引いた後も触らせ続ける |
| 使う | 3Dアクションのストーリー／高難度／曜日／PvE討伐 | 育てた強さを毎日確認する場を提供 |
| 周年・IP | 千年血戦篇連動、コラボ衣装、無料10連 | 離脱層を年単位で呼び戻す節目を作る |

ガチャ→育成→使う、の3層が独立せず、新キャラ実装が即「次の高難度」に接続している点が特徴。アクション側で多少不器用でも、適切なキャラを揃えれば突破できる設計のため、課金圧が滑らかに維持される。

## バズ要因 / 人気の根拠

popularity が示す「BLEACH原作の長期IP活用、千年血戦篇アニメ復活で再加熱」が出発点。

- 累計 90M DL+ の規模に達し、海外比率が日本国内モバイルRPGの平均より高い（北米・中南米・東南アジアでBLEACH原作の根強い人気を取り込んだ）
- 2022年に始まった TVアニメ『BLEACH 千年血戦篇』の放送によって、原作完結後10年以上経過していたIPに新規／復帰層が流入した
- KLabの長期運営ノウハウ（コードギアス・うたプリ系列を含むIPゲーム運用）で、毎年の周年・コラボ・無料配布のリズムが安定している
- 3D アクションを「片手・短時間」で消費できる設計のため、原作ファンが復帰してもアクションゲーム経験を要求されない

## 企画への示唆

- 原作の長期連載 IP を扱うなら「篇 × 衣装」の二軸でキャラ実装枠を多重化すると、同一キャラでも数年単位で新規ガチャ価値を作り続けられる
- アニメ／原作の続編が確定している IP は、続編放送タイミングに合わせた周年設計を仕込むと、休眠ユーザーを呼び戻す確度が上がる
- ガチャ × アクションを両立させる場合、アクションの腕前で詰むのではなく「正しいキャラを揃えれば抜ける」高難度にすると、課金導線が壊れない
- ソロ（social_axis null）に振り切ったうえで「推しを所有して動かす」設計に集中すると、対人運営のコストを払わずに長期 IP ゲームを維持できる
- 海外比率が高い IP は、ローカライズと配信地域の継続拡張を初期から織り込むと、国内モバイル収益の単一依存から逃げられる
