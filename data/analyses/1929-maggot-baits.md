# Maggot Baits 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「魔窟と化した旧カジョウ市で展開される暗黒劇を**最後まで読み切ること**」にある。Clock Up が標榜する「クルエルノワール ADV」というジャンル呼称自体が、ノワール小説の語りと哲学・宗教モチーフを成人向け描写に縫い付けた**物語消費型**の体裁であることを宣言している。

`concept` にある「不死の魔女と契約者・ヤクザ・難民が蠢く暗黒都市」「哲学・宗教を扱う重厚なシナリオ」、`target` にある「Clock Up / euphoria 系列のファン」という指定は、いずれもメカニクスやランダム性ではなく**長尺のテキストと固有の世界観**を消費目的とする層を想定したもので、ガチャ等の REW、操作の体験 EXP には収まらない。`social_axis` が null なのも、シングルプレイ完結のビジュアルノベルとして他者との関与軸を持たないためで、NAR ソロ消費型 ADV として整合的に位置付くデータ構造となっている。`secondary` は空欄で、副次動機が立たないほど物語駆動に振り切れた設計だと示唆される。

## 物語の駆動構造

nukige としては異例の長尺・高密度で、専用イラスト約 2,500 枚という規模感が物語駆動の中心装置となる。テキストを「読ませる」ための章立てと、エログロ描写を「見せる」ための差分枚数が、互いをエンジンとして相互参照する構造になっている。

| 駆動レイヤ | 仕掛け | 効果 |
|---|---|---|
| 世界設定 | 旧カジョウ市＝魔窟・難民・ヤクザ・契約者の階層化 | プレイヤーが地理と派閥を学習することで読み進める動機を作る |
| キャラクター | 不死の魔女と契約者の主従関係 | 力関係と倫理の反転を章ごとに揺さぶれる |
| 描写密度 | 約 2,500 枚の専用 CG | テキストと差分が同期し、シーン単位の体感時間を引き延ばす |
| 思想層 | 哲学・宗教モチーフを下敷きにした台詞 | エログロを「ただのフェチ」ではなく作品論として語れる余地 |
| 強度設計 | クルエルノワールという独自ジャンル提示 | 既存 nukige の文脈から切り離し、長期評価対象にする |

選択肢分岐よりも**一本道の没入度**で押すタイプの構造で、章を跨ぐごとに倫理水準を下げていくことで読者を引き返せない状態に置く設計と読める。

## バズ要因 / 人気の根拠

`popularity` の Moe Game Award 2015「フェチ作品賞」受賞および MangaGamer 英語版による海外波及を起点に、以下のレイヤで認知が積み上がった。

- Moe Game Award 2015「フェチ作品賞」受賞により、成人向け ADV 業界内で「単なる nukige ではない」評価軸を獲得した
- 約 2,500 枚という専用イラスト量が、ボリュームと密度の両面で同価格帯の作品から際立った
- Clock Up は『euphoria』など先行するダーク系成人向け ADV で固定ファン層を確立しており、ブランド連続性が初動を支えた
- MangaGamer による公式英語ローカライズが、英語圏 VN コミュニティ（fuwanovel 等のレビュー文化圏）で長期的なカルト評価を形成した

`sales_jp` / `sales_world` はいずれも非公開で、成人向け PC ソフトの一般的な傾向通り定量データは取得困難。定性的には「Clock Up カタログの代表格」「英訳された日本成人 VN の中で話題化した一本」と語られる。

## 企画への示唆

- **ジャンル名を自前で命名すると**、既存タグの平均評価から切り離され、長期のカルト評価軸を作りやすい（例：クルエルノワール ADV という宣言）
- **専用 CG 枚数を 4 桁オーダーに積むと**、テキストとビジュアルの同期密度が上がり、価格帯上限と長尺レビューの両方を正当化できる
- **倫理水準を章ごとに段階的に下げると**、読者は「ここで止めると損」というサンクコストに支配され離脱率が下がる
- **哲学・宗教モチーフを下敷きに敷くと**、エログロ描写が作品論として語れるようになり、二次的なレビュー文化が発生する
- **海外向け英訳パートナー（MangaGamer 等）と組むと**、国内売上が頭打ちでも英語圏 VN コミュニティで二次寿命が伸びる

## 比較で見るべきタイトル

- 『euphoria』（Clock Up）— 同系統ダーク ADV、ブランド連続性の前世代
- 『沙耶の唄』（Nitroplus）— 哲学・宗教モチーフを成人向け短編に圧縮した先行例
- 『鬼畜王ランス』（アリスソフト）— ダーク世界観 × 成人向け描写の長尺カルト系譜
- 『Saya no Uta』英語版（JAST USA）— 日本成人 VN の海外波及モデル
- 『装甲悪鬼村正』（Nitroplus）— クルエル + 思想層という近接構造
