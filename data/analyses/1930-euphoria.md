## なぜ NAR 主分類なのか

プレイヤーの主動機は「閉鎖空間に押し込まれた極限状況の物語を、選択肢で分岐させながら読み切る」ことに尽きる。`concept` が示すとおり、白い密室で進行する「アンロックゲーム」という強制ルールが物語装置の中核で、メカニクスはほぼ選択分岐のみ。育成・収集・対戦のような行動報酬は存在せず、プレイヤーが画面に向かう理由は次の章で何が起こるかを目撃するためである。

`target` の「Clock Up ブランドや過激系シナリオを追うコア VN ファン」という記述も、購買動機が「ヒロインを集める」ではなく「このライターが書く倫理的に重い物語を読みに行く」側であることを示す。`social_axis` が null で `secondary` が空であるのは、対人軸・副動機軸のいずれも実質ゼロというこのジャンルの典型的な姿で、NAR を主軸に据える判断と整合する。アダルト要素はあるが、それ自体が報酬ループを成すのではなく、シナリオの暴力性・倫理的重さを増幅させる演出材として物語に従属している。

## 物語の駆動構造

| 構造要素 | 本作での実装 | 物語駆動への寄与 |
|---|---|---|
| 舞台設定 | 完全密室「ホワイトルーム」と「アンロックゲーム」のルール | 退路を断ち、ヒロイン全員を物語の俎上から外せない状況を作る |
| ルート分岐 | 攻略順により真相情報の開示順が変動、複数バッドを経由して真ルートへ | 1 周完走では世界観が閉じない設計で周回前提の読書動線 |
| 章設計 | ヒロイン別シナリオ＋包括する真相パート | キャラ単位の悲劇とメタ謎が二層で並走 |
| 演出材としての過激描写 | 鬼畜・凌辱シーンが心理崩壊や信頼形成の引き金になる | 倫理的負荷を高め選択肢の重みを増幅 |

複数の Bad / True ルートを経由しなければ謎が閉じない「謎解き型ダーク VN」のフォーマットを、Clock Up ブランドの過激シナリオで書ききった点が本作の構造的特徴である。

## バズ要因 / 人気の根拠

- `popularity` が示すように、鬼畜系アダルト VN の代表格としてカルト的支持を確立し、ジャンル内で参照点になった。
- 2014 年に HD リマスター版が出ており、初出 2011 年から 3 年で再パッケージされるだけの継続需要があったことが裏付けられる。
- 2015 年に MangaGamer が英語版をリリース。Steam の規約外で配信される鬼畜系 VN として海外コミュニティに到達し、日本語圏外でも認知が広がった。
- OVA 全 6 話のアニメ化に至っており、原作 IP として二次媒体展開する規模に達した稀少な鬼畜系 VN である。

## 企画への示唆

- 物語の舞台を完全密室＋強制ルールに固定すると、登場人物の退場・離脱を物理的に封じられ、全員を同じ俎上に並べたまま心理描写を深掘りできる。
- アダルト・暴力描写を「報酬」ではなく「心理破壊と再構築のトリガ」として配置すると、過激度がシナリオ密度に変換される（単発描写の積み上げではなく物語駆動になる）。
- True ルート到達に複数 Bad 経由を必須化すると、プレイヤーは「読み残し」を能動的に潰しに行き、周回そのものが物語消費の自然な単位になる。
- ターゲットを「ジャンル全体のライト層」ではなく「特定ブランド／ライターを追うコア層」に絞り込むと、過激方向の表現許容度が上がり、結果として差別化された IP になりやすい。
- 国内ニッチ作品でも、英語ローカライズ・HD リマスター・OVA という 3 つの再展開経路を持てば、初出後 5 年以上の長尾収益と海外認知を獲得できる。

## 比較で見るべきタイトル

- 沙耶の唄（ニトロプラス）— 短編密室型ダーク VN として構造が近い
- 鬼畜王ランス（アリスソフト）— 過激シナリオの先行ベンチマーク
- Saya no Uta / Subahibi 系の海外ローカライズ流通モデル
- Doki Doki Literature Club!（同人 / 国際）— 物語装置として「閉鎖空間＋メタ的不安」を共有する後発例
