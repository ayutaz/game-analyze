# 殻ノ少女 (Kara no Shoujo) — Innocent Grey, 2008

## なぜ NAR 主分類なのか

このタイトルの主動機は**「猟奇事件の真相を読み解く」**こと、つまり物語消費に集約される。プレイヤー入力はビジュアルノベルの選択肢にほぼ限定され、システム的な操作性や報酬ループは前面に出ない。concept が示す通り「鬱々とした世界観」「複数 ED」「読ませる一作」という設計意図そのものが物語駆動である。

社会軸が `null` でソロ前提、target が「重い物語と陰鬱な雰囲気に耐性のあるコアな VN プレイヤー」と限定的であることも、本作が「他者と語り合う」より「ひとりで読み切る」体験設計であることを裏付ける。secondary が空なのは、推理パートでの簡易な選択行為（手帳の単語選択など）も最終的には物語分岐のスイッチでしかなく、それ自体が独立した遊びにならないため。1956 年戦後東京というシリアスな時代設定とミステリ骨格を持ったまま、選択肢が誤れば即バッドエンドに堕ちる構造は、推理 ADV というよりも「読み手を物語の中に閉じ込める」NAR の振る舞いに近い。

## 物語の駆動構造

戦後東京・大場女学院を舞台にした猟奇事件を、探偵・時坂玲人が追う章立てフォーマット。物語消費を駆動する仕掛けを分解すると、複数のレイヤーが重なっている。

| レイヤー | 仕掛け | 効果 |
|---|---|---|
| 時代設定 | 1956 年（戦争の影が残る東京） | 風俗・倫理観が現代と異なるため、暴力描写が「説明可能」になる |
| 探偵フォーマット | 関係者ヒアリング → 手帳整理 → 推理 | 読者を「観客」でなく「捜査主体」として物語に縛る |
| ヒロイン群 | 大場女学院の少女たち（被害者候補） | 攻略 ≒ 死亡フラグ反転、感情移入が恐怖に直結 |
| 複数 ED 構造 | 選択ミスでヒロインが惨殺される BAD ED 群 + True | 「読み手の選択が殺した」罪悪感をエンディングが背負わせる |
| 須田彩加の絵柄 | 透き通った線画と猟奇描写のギャップ | 美少女ゲーム文脈と猟奇ミステリの接続点を提供 |

この組み合わせは、当時の美少女ゲーム市場で主流だった「学園恋愛 + 泣き」とは別軸に立つ。プレイヤーは「正解の選択肢を引く」ことでヒロインを生存させる責任を負わされ、章ごとに「読まされた」量と「失わせた」量が累積するため、物語投資量が高くなるほど離脱コストが上がる。

## バズ要因 / 人気の根拠

- popularity に示される通り、戦後東京を舞台にした猟奇殺人ミステリーADV として 2008 年当時の VN レビュアー層で高い評価を獲得し、Innocent Grey の出世作として位置付けられた。
- 続編『殻ノ少女 罪人ノ系譜』（2010）、『虚ノ少女』（2014）と三部作展開し、IP として 10 年以上更新が続いた。
- 2011 年に MangaGamer から英語版がリリースされ、海外コア VN コミュニティへ越境（ADV の海外輸出例として頻繁に言及される）。
- 2023 年に HD リマスター『The Shell Part I: Inferno』が Shiravune から配信され、Steam を含むモダン PC 環境で再流通している。

## 企画への示唆

- **時代設定をひとつ過去にずらすと、現代では描けない暴力／倫理を「資料的説得力」で持ち込める**。1956 年という設定が猟奇描写の言い訳ではなく物語の必然になっている。
- **探偵フォーマットを採用すると、プレイヤー入力が少なくても「自分で物語を進めた」感覚が作れる**。手帳・聞き込み・推理という形式的手続きが、選択肢主導の VN を能動的に錯覚させる。
- **ヒロイン攻略 = 生存確認に置き換えると、ギャルゲ文法そのものを恐怖装置に転換できる**。攻略が報酬ではなく救済になることで、感情移入と緊張感が同時に立つ。
- **BAD ED を「読まないと True に辿り着けない」構造にすると、グロ描写の消費が物語の必須経路になる**。鬱描写の正当化と分岐ボリュームの水増しを両立させる古典的技法。
- **シリーズ三部作 + リマスターの長期 IP 化を最初から想定すると、初作はあえて未解決要素を残せる**。Innocent Grey は初作で世界観の骨格だけ提示し、続編で因果を回収している。
