# 天ノ少女 (Kara no Shoujo: The Last Episode) — 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「三部作で散らかった謎を読み切る」** こと。戦闘もミニゲームも収集要素も存在せず、昭和33年の探偵・時坂玲示の視点で猟奇事件と少女たちの死を **テキストで追体験する** ことに体験がほぼ全振りされている。

JSON の concept にある「三部作の完結編で、シリーズ全体の謎を回収する物語消費に特化した一作」が分類根拠そのもの。target も「重厚な大人向けミステリー・猟奇 ADV を好む層」「Innocent Grey シリーズの既存ファン」と、すでに前 2 作の伏線を抱えた状態で続きを読みに来る読者を想定しており、ゲームメカニクスではなく **続きを読むこと自体** が動機になっている。`social_axis` が null である点も、他者との関わりを設計に持たない純粋なソロ読書体験であることと整合する。

`secondary` は空。報酬要素（CG コンプ、エンディング分岐回収）は副次的にはあるが、それらは「物語の全貌を把握する」という主動機に従属している。

## 物語の駆動構造

シリーズ三部作の完結編という設定が、本作の物語駆動を大きく規定している。新規プレイヤーには敷居が高い代わりに、既プレイヤーには「前 2 作で開いたままだった伏線が閉じる」というリテンション効果を提供する。

| 構造要素 | 本作での実装 | 駆動する読書動機 |
|---|---|---|
| 時代設定 | 昭和33年 (1958) 1月、終戦から13年の東京 | ノワール／探偵モノの空気感 |
| 視点キャラ | 探偵・時坂玲示（シリーズ通しの主人公） | 三部作を通読してきた読者の継続接続 |
| 開幕シーン | 「ある少女の葬儀に立ち会う」 | 死から逆算で謎を解く構造の予告 |
| 章構造 | ミステリーADV 章立て + 推理パート | 読者を能動的に犯人推理へ巻き込む |
| シリーズ位置 | 『殻ノ少女』『虚ノ少女』に続く第 3 作 | 既存ファンの「結末を見届けたい」動機 |
| 表現規制 | 18 禁・猟奇描写あり | コンシューマ移植不可な題材を扱う自由度 |

『殻ノ少女』(2008)、『虚ノ少女』(2014) と 6 年〜12 年スパンで続いた三部作の終着点というプロジェクトそのものが、物語消費型 VN の **長期回収モデル** として機能している。

## バズ要因 / 人気の根拠

popularity 欄に記された「2020 年 Moe Game Award 年間 26 位入賞、シナリオ賞金賞受賞」が出発点。

- **シナリオ賞金賞** — 業界内評価軸（Moe Game Award）でテキスト品質を公的に裏付け、ジャンル内オピニオンリーダーへの訴求材料となった
- **三部作の完結** — 『殻ノ少女』からの 12 年越しの結末という時間的稀少性そのものがニュース価値
- **MangaGamer 経由の英語圏展開** — Innocent Grey 作品は早期から海外 18 禁 VN ローカライズ流通網に乗っており、国内 18 禁 ADV としては珍しく英語圏レビューで言及される
- **猟奇ミステリーという狭く深いニッチ** — 数量バズではなく、ジャンルを牽引する固定客層（殻ノ少女ファン、Innocent Grey ファン）のなかでの存在感が強い

`sales_jp` / `sales_world` が共に非公開なのは 18 禁 PC ゲーム市場の一般特性で、数字ではなく **批評賞 + シリーズ完結というナラティブ** で語られる作品である。

## 企画への示唆

- **三部作の最終章を別 SKU として独立販売すると**、過去作未プレイ層を切り捨てる代わりに、既存ファンの「結末への投資意欲」を最大化できる
- **賞レース（Moe Game Award 等）で部門賞を取りに行く構造をシナリオ段階から仕込むと**、売上数字を公開しないニッチ市場でも信頼指標を獲得できる
- **昭和ノワール × 探偵 × 猟奇という時代設定の三層を固定すると**、続編ごとに同じ世界観の再利用コストを抑えつつ、シリーズの個性として認識される
- **コンシューマ移植不可能な表現を意図的に選ぶと**、PC 18 禁市場の規制裁量を競争優位に転化でき、IP の希少性が保たれる
- **完結まで 12 年かけても完走するという信用を積むと**、次シリーズの初動が読者の「最後まで出してくれるブランド」という事前評価で底上げされる

## 比較で見るべきタイトル

- **殻ノ少女 (2008) / 虚ノ少女 (2014)** — 同シリーズ前 2 作。三部作プロジェクトの起点と中間点として必読
- **CARTAGRA -契約の閲覧- (2005)** — Innocent Grey 第 1 作。猟奇ミステリー VN のスタジオ原点
- **シークエンス・パラディウム (2016) / FLOWERS シリーズ** — 同じ Innocent Grey 系列で異なる題材を扱う比較対象
- **428 〜封鎖された渋谷で〜 / Ever17** — 物語駆動・章構造・複数視点のミステリー ADV として構造比較
- **Steins;Gate 0** — シリーズ完結編という位置づけで、既存ファンの伏線回収需要をどう設計するかの参照点
