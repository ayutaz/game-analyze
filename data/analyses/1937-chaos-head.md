# ChäoS;HEAd — 分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「渋谷で連続する猟奇事件と引きこもり主人公・西條拓巳の認知崩壊を読み解くこと」であり、ゲームプレイは妄想トリガーによる選択肢分岐に絞り込まれている。concept にある「妄想と現実が混濁していく一人称サイコサスペンス」「妄想トリガーで主人公の認知歪みそのものを操作する」という構造は、戦闘・収集・育成といった反復報酬ループを持たず、テキストとシナリオ進行を主軸に据えた純然たるビジュアルノベルの設計である。

target も「重厚な伝奇・サイコホラー系シナリオを好む 18+ ノベルゲーマー層」「STEINS;GATE 等科学 ADV シリーズのルーツを辿りたい既存ファン層」と明示されており、選択動機は物語消費に直結する。social_axis が null なのも、他プレイヤーとの関わりではなくテキスト体験単体で完結することを反映している。secondary が空である通り、副次動機としてのコレクションや成長要素は意図的に切り落とされている。

## 物語の駆動構造

科学 ADV 共通文法の原型がここで成立している。分岐は通常の選択肢ではなく「妄想トリガー」という独自 UI を介し、ポジ／ネガ／非選択の三択で主人公の認知そのものを偏向させる。プレイヤー入力が世界の事実ではなく「主人公がどう世界を歪めるか」を制御する点が、後続作（STEINS;GATE のフォーントリガー、ROBOTICS;NOTES のタブレットトリガー、CHAOS;CHILD のデリュージョントリガー）に継承されたシステム的祖先である。

| 駆動装置 | 機能 | 物語上の効果 |
|---|---|---|
| ポジ妄想トリガー | 主人公の願望投影を本文に挿入 | 自己肯定的な現実逃避と一時的高揚 |
| ネガ妄想トリガー | 被害妄想・恐怖を本文に挿入 | 不安増幅と次章への伏線生成 |
| 非選択 | 妄想なしで進行 | 真相ルートへの収束条件 |
| ルート分岐 | ヒロイン別 / 真相ルート | 認知の歪みを「客観的事実」として再評価 |
| 渋谷限定の舞台 | 実在地理を固有名詞で密に配置 | 都市怪談的リアリティと閉鎖性の両立 |

## バズ要因 / 人気の根拠

popularity が示す通り「科学 ADV シリーズ第 1 作」というシリーズ起点性が長期的な存在感を担保している。

- 2008 年 PC 版以降 Xbox 360 / PSP / PS3 / Vita / Switch / iOS / Android / Steam と 8 プラットフォームに移植され、世代をまたいで触れられる窓口が維持されている
- 後発の STEINS;GATE がメガヒット化したことで「ルーツ作品」として遡及的に参照され、周年特集・アニメ化・コミカライズで継続的に再露出している
- 2022 年に英語版 CHAOS;HEAD NOAH が Spike Chunsoft から発売され、海外科学 ADV ファン層への正規供給ルートが確立した
- 開発体制が Nitroplus と 5pb. の共同で、虚淵玄を起点とするニトロプラス系シナリオ作家文化と 5pb. のコンソール展開力が交差した最初期事例として語られる

## 企画への示唆

- 選択肢を「世界への入力」ではなく「主人公の認知への入力」に再定義すると、プレイヤー操作とテーマ（精神崩壊・現実認識）が一致して没入が深まる
- シリーズ第 1 作で UI 装置を発明しておくと、後続作が同じ語彙で差分だけ提示できるため IP として継承コストが下がる
- 実在の都市（渋谷）を固有名詞密度高く描くと、低コストで「実在感のあるサイコホラー」を成立させられる
- 一人称引きこもり主人公を選ぶと、外界情報の信頼性を恣意的に操作でき、不安定な語り手構造を低リソースで実装できる
- 移植プラットフォームを世代ごとに更新し続けると、後続作のヒット時に「ルーツ作品」として再評価される受け皿を残せる
