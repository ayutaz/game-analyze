# 星空のメモリア -Wish upon a shooting star-

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ヒロイン視点で語られる過去と願いの記憶を読み解く」こと。テキストを進めて分岐を辿ること自体が目的であり、戦闘・収集・対戦のような外部報酬は存在しない。

concept にある「星にまつわる願いと記憶をモチーフに、ヒロインたちとの交流と過去の回想を幻想的なファンタジー仕立てで描く」は、典型的な NAR の駆動原理である**因果の遡行**（現在の交流 → 過去の記憶 → 願いの解消）に依拠している。target が「学園もの・天文・幼馴染・ファンタジー要素のあるしっとり系シナリオを好む層」と明示的に物語消費者を指していること、social_axis が null（他者プレイヤーとの相互作用が設計上存在しない）であることからも、NAR 単独で支配的と判断できる。

secondary は空。アダルト要素は購買動機の一部にはなるが、本作の再販履歴（FD・コンプリート版・HD・英語版）が示す通り、長期評価を支えているのはシナリオライター中弘のテキスト体験であり、副次カテゴリで処理する性質のものではない。

## 物語の駆動構造

学園恋愛 AVN の標準形式（共通ルート → ヒロイン個別ルート → エピローグ）に、「星 = 願い = 記憶」という三位一体のモチーフを通底させて、章単位でテーマの反復を起こす設計。

| 構造要素 | 本作での実装 | 物語消費を駆動する効果 |
|---|---|---|
| 舞台 | 高校の天文部 | 季節カレンダー × 観測イベントで時間進行を可視化 |
| モチーフ | 流星・星座・夜空 | ルートごとに異なる「願い」を象徴的に再解釈させる |
| ヒロイン構成 | 幼馴染 + 学園 + ファンタジー寄りキャラ | プレイヤー層の典型嗜好を並列カバーし、周回動機を発生 |
| ルート構造 | 共通 → 個別 → 真ルート想定 | 個別で示された伏線を真で回収する遡行型読解 |
| 演出 | 天体観測シーンを核にした幻想美術 | 「夜空を見上げる」反復シーンで情緒の鍵を内製 |

ファンタジー要素の混入（記憶・願いの超常的扱い）が、通常の学園恋愛 AVN では到達できない**喪失と再会**のトーンを許容しており、ここがシナリオ評価の核になっている。

## バズ要因 / 人気の根拠

popularity に示された「中堅エロゲメーカー FAVORITE の代表作 / 中弘評価 / 2ch ベストエロゲ 2009 で 5 位 / 長期再販」を分解する。

- 2009 年リリース時点で 2ch ベストエロゲランキング 5 位という同人・商業横断のテキスト評価指標で上位入り。当時の上位はテキスト評価で決まる傾向が強く、シナリオ寄り評価層の支持を得た。
- ファンディスク → コンプリート版 → HD リマスター → 英語版（Sekai Project / Steam 2017）と複数フォーマットで再販されており、約 8 年スパンで継続収益化されている。
- 英語版 Steam 配信は日本 AVN の海外市場展開の典型例で、Sekai Project のラインナップ価値を支えるカタログのひとつとして機能。
- 開発・販売が同一の FAVORITE による self-published 体制で、中堅ブランドが「代表作 1 本を長期で回し続ける」典型パターン。新作の話題ではなく**シナリオ資産の長期再活用**が人気の継続要因。

## 企画への示唆

- **モチーフを 1 つに絞り三位一体化する**と、章をまたいでも世界観の統一感が崩れず、ルート間の温度差を吸収できる（本作の「星・願い・記憶」）。
- **学園カレンダー × 観測イベント**を時間進行装置として使うと、章立てを明示せずとも物語の進度をプレイヤーが体感で把握できる。
- **幼馴染 + ファンタジー寄りキャラを並列配置**すると、嗜好の異なる層を 1 本で取り込みつつ、ルートごとに異なるトーンを正当化できる。
- **再販ラダー（FD → コンプリ → HD → 翻訳版）を初期から設計**すると、中堅規模ブランドでも 8 年単位の長期収益化が可能になる。シナリオ資産は新作よりリパッケージで回した方が ROI が高い局面がある。
- **海外配信パートナー（Sekai Project 型）との早期接続**を見越したテキスト・ CG 構造にしておくと、後年の英語化コストを抑えられる。

## 比較で見るべきタイトル

- ef - a fairy tale of the two.（minori, 2006）— 同時代の中堅ブランド代表作、モチーフ統一型シナリオ。
- 群青の空を越えて（light, 2005）— 中弘以前の「テキスト評価で長期再販」モデルの先行例。
- リトルバスターズ！（Key, 2007）— 学園 × ファンタジー要素の混入と真ルート回収構造の比較対象。
- 9-nine-（ぱれっと, 2017〜）— ファンタジー学園 AVN の後発系列。再販・HD・コンプリ展開の現代版モデル。
