# 紅い瞳に映るセカイ — 分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「西鎌真琴ルートの後日譚を読み切る」こと、すなわち**未完だった一人のヒロインのその後を物語として閉じる**ことに尽きる。本作はゲームメカニクスを刷新するための作品ではなく、本編『いろとりどりのセカイ』および外伝『いろとりどりのヒカリ』を経たプレイヤーに対し、シリーズ世界観と「紅い瞳」をめぐるシリアス線を完結させるために置かれたアフターファンディスクである。

concept にあるとおり前半は嵐山学生寮の日常ラブコメ、後半はシリーズ恒例のシリアスへ落ちる二段構成で、選択肢による分岐よりも一本道のシナリオ消費が前面に来る。target が明示するように新規参入を一切想定せず「真紅ルートを愛する既存ファン」だけを対象に切り出した時点で、読書体験＝主動機・収集や育成は不在という NAR 的純度が確定する。`social_axis` が null・`secondary` が空配列で書かれているのも、他者との関わりも副次的動機も設計上組み込まれていないことの裏返しである。

## 物語の駆動構造

アフターファンディスクという形態は「本編で語り切れなかった残余」を回収する装置であり、本作の駆動構造は前作群の負債を返済する形で組まれている。

| 構造要素 | 本作での実装 | プレイヤーへの作用 |
|---|---|---|
| 前提知識 | 本編+外伝を既読である前提 | 序盤からキャラ関係性の説明を省略でき、密度を上げられる |
| 章立て | 日常ラブコメ前半 → シリアス後半の二層 | テンションの落差で「読み続ける動機」を維持 |
| 主役ヒロイン | 西鎌真琴 1 名に絞り込み | 真琴ルート未練ファンに完結感を提供 |
| 限定版同梱 | 「真紅といっしょ」豪華限定版（真紅ぬいぐるみ） | 物語消費の外側に物理的所有欲を接続 |
| 主題歌 | eufonius「glowing world ～かがやきの、せかいへ～」 | OP 視聴が単独で SNS 上の話題接点になる |

ファンディスクとして「本編→外伝→本作」と段階的に世界観を継ぎ足す構造は、シリーズ全体を一つの長尺ノベルとして消費させる動線になっている。

## バズ要因 / 人気の根拠

`popularity` の出発点は「シリーズファンの間で限定版とOPが話題になった」という一文だが、その背景を分解すると以下の層に整理できる。

- FAVORITE 看板シリーズ『いろとりどりのセカイ』の正統続編枠であり、本編・外伝で積み上げたシリーズ資産がそのまま購買母数になっている
- 「真紅といっしょ」豪華限定版がぬいぐるみ同梱という物販性で、SNS の開封・所有報告が広がりやすかった
- eufonius による OP「glowing world ～かがやきの、せかいへ～」がアーティスト経由の流入を生み、ノベルゲーム外の音楽リスナーにも作品名が届いた
- アフターファンディスクという立て付け上、新規バズより**シリーズ追跡層の濃いコミュニティ内会話**で持続的に言及される性格を持つ

## 企画への示唆

- シリーズの未消化ルートを「単一ヒロイン特化のアフターディスク」として切り出すと、本編で散らした感情をピンポイントで回収でき既存ファンの満足度を最大化できる
- 限定版にキャラクター固有の物理グッズ（ぬいぐるみ等）を同梱すると、物語消費とコレクション欲が別経路で重なり SNS 露出が増える
- 前半ラブコメ／後半シリアスの二段構成にすると、ファンディスクという軽量フォーマットでも読み切る駆動力を確保しやすい
- 既存アーティスト（本作なら eufonius）に主題歌を委ねると、作品コミュニティ外への接点が一本増え、レビュー外の話題流入を生む
- 新規参入を**意図的に切り捨てて**シリーズ既読者に絞ると、説明コストを削れて密度の高い物語提供に資源を集中できる

## 比較で見るべきタイトル

- いろとりどりのセカイ / いろとりどりのヒカリ（FAVORITE） — 本作の前提となる本編＋外伝。シリーズ累積構造の参照点
- ましろ色シンフォニー *sakura* / ましろ色シンフォニー HD（ぱれっと系） — ヒロイン特化アフターの先行事例
- 星空のメモリア -Eternal Heart-（FAVORITE） — 同社のヒロイン特化アフターファンディスク。比較で本作の二段構成が見える
- ヨスガノソラ -Haruka no Souzora-（Sphere） — 単一ヒロイン拡張型 FD の代表例
