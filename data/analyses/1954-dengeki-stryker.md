# 電激ストライカー (2011 / OVERDRIVE)

## なぜ NAR 主分類なのか

このタイトルでプレイヤーがしている主動機は「少年漫画的ヒーローものを最初から最後まで読み切る」ことで、選択肢で攻略順を変える程度の介入しかない一本道アダルトビジュアルノベルである。concept にある通り、漫画の主人公「電激ストライカー」に変身して幼なじみを守るという特撮王道展開を、文章・立ち絵・スチル・主題歌・戦闘アニメで連続的に消費させる構造で、ゲームメカニクスは選択肢分岐とフラグ管理に限られる。

target も「OVERDRIVE 過去作（キラ☆キラ / DEARDROPS）のファン」「特撮・少年漫画的なヒーローもの直球展開を文章とアニメ演出で味わいたい男性アダルト VN ユーザー層」と明示されており、求められているのは攻略の手応えではなく物語の純度。`social_axis: null` で他者との関わりはなく、`secondary` も空なので「物語消費 100%」という単一動機の純度が高い NAR である。

## 物語の駆動構造

OVERDRIVE 5 周年記念作として、ヒーローものという定型ジャンルを長尺ノベルの密度で本気で書き切る構成を取っている。駆動要素を分解する：

| 層 | 仕掛け | 物語推進への寄与 |
|---|---|---|
| ジャンル選択 | 特撮・少年漫画ヒーローもの直球 | 読者の様式知識を借用し、説明コストを節約 |
| 主人公動機 | 漫画の主人公に「変身」して幼なじみを守る | 自己投影と憧れを二重化 |
| 戦闘演出 | SILVER LINK. 制作・渡辺信一郎監督のアニメパート | 文章で書けない「決め技」の高揚を映像で補完 |
| 章立て | 漫画的な章・回構成＋ボスバトル | 読書ペースに「次回への引き」を強制供給 |
| 増補版 | 『超電激ストライカー』(2014) で追加シナリオ | 既読層に再読動機、新規層に決定版を提示 |

選択肢は攻略順とエンド分岐を司る最小限の介入装置で、プレイヤーの仕事は「読み続けるか中断するか」の決定だけに近い。代わりに OP / ED 楽曲、戦闘アニメ、変身バンクといった「動く山場」を等間隔で配置し、ノベル単体だと単調になりやすいヒーローものをアニメ的リズムで持続させる設計になっている。

## バズ要因 / 人気の根拠

- OVERDRIVE 5 周年記念作という社内的なフラッグシップ位置付けで、戦闘パートのアニメーション制作に **SILVER LINK.** と **渡辺信一郎監督** という TV アニメ第一線スタッフを起用したことが企画段階から話題化
- アダルト VN としては珍しい「特撮ヒーローもの直球」というジャンル選択で、キラ☆キラ（バンド）／DEARDROPS（バンド）と続いた OVERDRIVE の「サブカルジャンル × 熱血」シリーズの一本として継続的に語られる
- 2012 年に **MangaGamer** が英訳版を配信、海外ヒーローもの・特撮ファンに到達。`tvtropes` にエントリが立つレベルで英語圏のコア層が言及
- 2014 年に増補版『超電激ストライカー』が **Steam** に進出（全年齢版）。アダルト VN のヒーローもの直球作品が Steam で買えるという稀少性が、英語圏の VN レビュアー界隈で再注目された

## 企画への示唆

- ジャンルの様式美を借用すると、説明コストが下がり物語の「決め」に予算を集中できる：特撮・ヒーローものの定型を本気で踏襲したことで、読者が初見でも文脈を補完できる
- ノベル単体で単調になりやすい長尺ジャンルには、外部スタジオによる映像パートを「決め回」に挿入すると山場のリズムが保てる：アニメ会社起用は単なる豪華さでなく読書持続装置として機能する
- 周年記念作を「実験ジャンル × 既存ファン」で組むと、ブランド資産を毀損せずに新規流入経路を作れる：OVERDRIVE 既存ファンに向けつつヒーローもの新規層を取りに行く設計
- 数年後の「増補版＋全年齢化＋海外配信」のリパッケージは、原作が完結ジャンル（一本道ヒーローもの）であるほど効きやすい：長期 IP より「決定版を売り直す」モデルが適合
- 海外パートナー（MangaGamer 等）に翻訳権を渡す前提で、ジャンルが海外の既知文脈（superhero / tokusatsu）に翻訳しやすい題材を選ぶと、二次市場が自然に立ち上がる

## 比較で見るべきタイトル

- キラ☆キラ (OVERDRIVE 2007) — 同社の「サブカルジャンル × 熱血」シリーズの起点、バンドもの
- DEARDROPS (OVERDRIVE 2010) — 同シリーズのバンドもの第二弾、本作の直前作
- BALDR SKY (戯画 2009) — 同時代の「アダルト VN × アクション映像」志向作
- 鬼哭街 (Nitroplus 2002) — 王道ジャンル（武侠）をアダルト VN で本気で書く系譜の先行例
