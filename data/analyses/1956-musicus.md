# MUSICUS!

## なぜ NAR 主分類なのか

プレイヤーの主動機は「対馬馨というロックバンドマンの人生を読み切る」こと。バンドを組み、社会と折り合いをつけながら音を鳴らし続ける主人公の選択を、3 ヒロイン + Bad End を含む 4 ルートで追体験する構造であり、ガチャもランキングも対戦もない一本道に近い物語消費が中心になる。

concept にある「『音楽とは何か』を問う」というテーゼと、target に示された「OVERDRIVE 旧作（『キラ☆キラ』『DEARDROPS』）ファン + バンド題材の重めの物語を求める層」は、いずれも「読み物として完成された物語を浴びたい」という需要に向いている。social_axis が null なのも一貫しており、他者との相互作用ではなく作家性とテキスト量が体験の核を成す。secondary が空である点も、音楽 = 体験型（EXP）として遊ばせる仕組み（リズムゲーム化、自由演奏）が意図的に避けられている=テキストで「音」を語り切るという編集判断を示す。

## 物語の駆動構造

MUSICUS! の駆動装置は「ロックという題材」と「VN 形式の章 / ルート分岐」の組合せに集約される。下表は構造要素ごとの役割を分解したもの。

| 要素 | 内容 | 物語駆動への効果 |
|---|---|---|
| 主人公 = バンドマン視点 | 対馬馨はバンド「対馬馨と仲間たち」を組み演奏活動を続ける | 音楽 = 青春 = 挫折の三題噺を一人称で内面化させる |
| 3 ヒロイン + Bad End ルート | メイン 3 + 帰結としての Bad End を含む 4 ルート構成 | 「どの女と / どの生き方を選ぶか」が音楽家としての岐路と重なる |
| OVERDRIVE 最終作という枠 | ブランド最後の VN として制作・告知 | メタな「最後のロック」感が物語のテーマと共振 |
| 重テキスト + 楽曲 | ロックンロール ADV を標榜、劇中バンド楽曲を伴う | 静止画 + 文章で「音を鳴らす」読書体験を構築 |

カレンダーや自由行動のシミュレーション層を持たず、選択肢で分岐する古典的 VN の作りに留めた点は重要で、「音楽家の人生」を体験ゲーム化せずあくまで読み物として閉じる設計判断になっている。

## バズ要因 / 人気の根拠

- 2018 年 7 月の CAMPFIRE クラウドファンディングが日本記録級の速度で目標達成し、リリース前から VN コア層の話題を集めた。
- 『キラ☆キラ』『DEARDROPS』を手掛けた OVERDRIVE の最終作という「ブランド締めくくり」文脈が、旧作ファンの購買と SNS 言及を集中させた。
- 2021 年に MangaGamer 経由で英訳版が Steam / 18+ 版同時配信され、海外 VN コミュニティでも高評価レビューを獲得（売上数値は非公開）。
- PC リリース後、Entergram から PS4 / Switch のコンシューマ移植版が出ており、エロゲ → CS 展開の典型ルートで読者層を拡張した。

## 企画への示唆

- ブランドの最終作を明示的に「最後」と告知すると、過去作ファンの非合理な購買行動（応援消費）を引き出せる。
- 音楽題材を VN で扱う際、リズムゲーム化を避けてテキストで「音を語る」方向に振り切ると、ライティングの濃度で勝負できコア読者の評価が安定する。
- クラウドファンディングを「先行販売」ではなく「ブランド継続宣言」として設計すると、達成スピード自体が広告素材になる。
- 一本道に近い NAR タイトルでも、Bad End を含む 4 ルート程度の分岐を残しておくと、語り直しと考察コンテンツの量産余地を作れる。
- 18+ PC 版 → CS 版 → 海外 Steam 版という 3 段リリースを年単位で分散させると、各タイミングで新規話題化を起こせる。

## 比較で見るべきタイトル

- 『キラ☆キラ』『DEARDROPS』（OVERDRIVE）— 同ブランド・バンド題材の直系前世代作
- 『White Album2』（Leaf / Aquaplus）— 音楽 × 三角関係 × 重い分岐の代表作
- 『Steins;Gate』『CLANNAD』『沙耶の唄』— 「PC 18+ 発 → 全年齢移植 → 海外 Steam」の VN ロングテール流通モデル
