# 赤い糸 (2004 / Success)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「羽藤桂と複数のヒロインの関係性が辿る分岐結末を、選択を重ねながら読み解くこと」である。戦闘システムやスコアではなく、テキストと選択肢で駆動される伝奇ホラーアドベンチャー／ビジュアルノベルというジャンル骨格そのものが、本作を NAR に位置付ける根拠である。

concept にある通り「赤い糸の伝承」を軸に、選択次第で物語と結末が大きく分岐する設計で、target も「PS2 世代のビジュアルノベル／伝奇ホラー愛好層、百合作品ファン」と物語消費層に最適化されている。social_axis が null なのも、プレイ自体は単独で進めるがオンラインの他者要素を持たない、典型的なソロ完結型 ADV だからである。secondary は空で、ホラー演出や百合的関係性は NAR を補強する装飾として機能するが、独立した主動機にはならない。

## 物語の駆動構造

伝奇ホラー × 百合 ADV の構造を分解すると以下のようになる。

| レイヤー | 仕掛け | 物語駆動への寄与 |
|---|---|---|
| 主題 | 「赤い糸」＝運命的な繋がりの伝承 | 関係性の必然・宿命をテーマ化し、選択の重みを底上げ |
| 主人公 | 羽藤桂（女性主人公） | 同性ヒロインとの関係描写が自然に成立する視点を確保 |
| 分岐 | 各章での選択肢が複数ルート＆エンディングへ収束 | 周回プレイで補完される構造、TRUE END 探索の動機 |
| ヒロイン群 | 複数の女性キャラクターと個別ルート | 「誰の糸を辿るか」というキャラ単位の読了体験 |
| 演出 | 伝奇ホラー（怪異・神話・血脈） | 百合の情緒的読み味とホラーの緊張を同居させる二重トラック |

選択肢→分岐→エンド回収のサイクルは美少女ゲーム／VN の標準的フォーマットだが、「百合ルート」を明示的に組み込んだ点がこの世代の PS2 SuperLite 2000 帯では希少であり、後続の『蒼い雪』との二部作的な位置づけにも繋がっている。

## バズ要因 / 人気の根拠

- PS2 SuperLite 2000 シリーズという廉価帯から登場し、低単価で手に取りやすかったことが初期普及を後押し
- 当時の家庭用ハードで百合的結末を明示した分岐 ADV は数少なく、ニッチ層に深く刺さりカルト的支持を獲得
- 姉妹作『蒼い雪』（同じく Success）との世界観連結により、シリーズとして語られ続けるアーカイブ性
- 2023 年に Strictly Limited Games から Switch / PC 向け HD リマスター（英語対応・『蒼い雪』同梱）が海外発売され、20 年越しに新規層と海外 VN コミュニティから再評価
- 売上数値（sales_jp / sales_world）は非公開だが、HD リマスター企画化に耐える程度の長期需要が裏付けとして存在

## 企画への示唆

- 廉価帯 IP を「20 年後にリマスター可能な資産」として残すには、世界観・キャラ造形に固有性を持たせると、ジャンル全体の再評価期に拾われる
- マイノリティ向け関係性描写（本作の場合は百合）を主分岐として明示すると、ニッチだが熱量の高いファンベースが長期に渡って言及し続け、カルト的人気の燃料になる
- 姉妹作との世界観共有（『赤い糸』×『蒼い雪』）は、単体タイトルではなく「シリーズ」として記憶される確率を上げ、合本商品化の余地を生む
- 海外 VN 市場が成熟した時期に英語版を投入すると、原作発売時には存在しなかった顧客層を後追いで取り込める（Limited Run 系パブリッシャーとの提携が有効）
- 伝奇＋ホラー＋恋愛など複数ジャンルを縦に重ねると、各層の入口を増やしつつ TRUE END で統合的に回収する設計が機能する

## 比較で見るべきタイトル

- 蒼い雪（Success, 2006）— 姉妹作。世界観継承と二部作戦略の参照例
- ひぐらしのなく頃に（07th Expansion）— 伝奇ホラー × 同人／低価格帯から長期 IP 化した先行例
- FLOWERS シリーズ（Innocent Grey）— 百合をメインテーマに据えた分岐 VN の系譜
- 蠅声の王／STEINS;GATE 系（5pb. 他）— 伝奇・SF ホラー寄りの分岐 ADV、選択構造の比較対象
