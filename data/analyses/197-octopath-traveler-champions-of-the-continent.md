# オクトパストラベラー 大陸の覇者

## なぜ REW 主分類なのか

プレイヤーの主動機は「8 人ではなく数十人のトラベラーを **召喚で集め、編成して章クエストを進める**」ことにある。本編が「8 人の旅人を選び終わるまで」を商品にしていたのに対し、本作はキャラ獲得そのものが商品で、シナリオはガチャを回すための強い動機付け装置として機能する。concept にある「8 人の旅人に代えてキャラ召喚で物語を集める」という設計反転が、主分類を NAR ではなく REW に落とす根拠になっている。

target の「通勤・空き時間に消費する層」も REW を補強する。短時間プレイ前提のモバイル環境では、長尺の物語消費よりも「ガチャ → 育成 → 章クリア → 次のガチャ」のループが収益と継続を支える。ただし secondary に NAR が入る通り、章ごとのテキスト分量はモバイル RPG としては多めで、キャラ獲得をシナリオ閲覧の鍵として機能させている点が同ジャンル他作との差分になる。social_axis は未設定で、対人要素より単独周回が中心。

## 報酬ループの組成

ガチャと章クエストが二重の収集対象になっており、片方が片方を引き出す構造になっている。

| 層 | 内容 | プレイヤー動機 |
|---|---|---|
| キャラ召喚 | トラベラー（職業 × レアリティ）を引く | 編成の選択肢を増やす |
| 武器召喚 | キャラ専用ではない武器ガチャ | ダメージ天井の押し上げ |
| 章クエスト | キャラごとに用意された個別シナリオ | 「引いた」キャラの背景を読む |
| 育成（覚醒・職アビ） | 素材周回でレア度・スキルを伸ばす | 編成完成までの中期目標 |
| 大陸の覇者ストーリー | 三国志モチーフの本筋章立て | ガチャを回す長期動機 |

本編譲りの「キャラ ×8 章」フォーマットをガチャ筐体に流し込んだことで、**獲得したキャラに固有章が付いてくる** という珍しい構造になった。一般的なガチャ RPG ではキャラの絆ストーリーは付録だが、本作はキャラ章そのものが本編相当のシナリオ密度を持ち、引いた直後の満足度を高めると同時に「次のキャラ章を読みたい」という回し直し動機を生む。HD-2D の画面映えがガチャ演出の価値を底上げしているのも収益面で効いている。

## バズ要因 / 人気の根拠

- 配信 1 週で 500 万 DL（国内中心）を達成。HD-2D を期待していた SFC〜PS 世代の JRPG 層がそのまま流れ込んだ。
- 本編「オクトパストラベラー」（2018 / Switch）の前日譚として位置付けられ、原作の世界観 IP がそのまま転用可能だった。
- HD-2D 表現を縦持ちスマホ画面に成立させた最初期の作品で、ビジュアル文脈での話題性が大きかった。
- コマンド RPG という非アクション系の戦闘設計で、通勤シーンと相性がよく長期運営に乗りやすかった。

## 企画への示唆

- IP の中核ビジュアル（HD-2D）をモバイルに移植すると、グラフィック単体で初動の獲得チャネルになる。
- 本編が「8 章 / 8 主人公」の章立て構造を持っていれば、ガチャ筐体に流し込んだとき **キャラ章 = 引いた直後の体験報酬** に変換できる。
- コマンド戦闘 + テキスト主体にすると、片手・短時間プレイの通勤層を捕まえやすく、ARPG 系より老齢ユーザーの離脱が抑えられる。
- ナンバリングではなく「前日譚」「if」「外伝」を立てると、本編の物語と矛盾しないままキャラ追加余地を確保できる。
- ガチャ対象を「キャラ」だけでなく「武器」「職業」と層分割すると、無償石の使い道を分散させて課金導線を増やせる。
