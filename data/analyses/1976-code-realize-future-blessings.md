# Code:Realize 〜祝福の未来〜 (2016)

## なぜ NAR 主分類なのか

このタイトルでプレイヤーがしている主動機は「本編で結ばれた相手のその後を読み続けること」である。動詞は明確に **読む** であって、戦う／集めるではない。

`concept` にある通り、本作は『創世の姫君』のファンディスクであり、独立した新作 RPG ではなく **後日談（after story）と if ルートの集合体**である。主人公カルディアと攻略対象（アルセーヌ・ルパン、インピー・バルビケイン、ヴィクター・フランケンシュタイン、サン・ジェルマン、エイブラハム・ヴァン・ヘルシング）との恋愛関係を、本編完結後の時間軸でさらに掘り下げるという構造上、ゲームプレイ要素はテキスト読了と選択肢分岐に限定される。`target` で「本編をクリアした女性オトメユーザー」と明示されているとおり、設計上 **既に物語に資産投資した読者** を対象にしており、プレイ動機の重心は完全に物語の延長消費にある。

`secondary` が空、`social_axis` が null である点も NAR 純度の高さを裏付ける。マルチプレイ要素や報酬収集ループは設計に組み込まれておらず、評価軸は「キャラクターアークの満足度」だけに絞られている。

## 物語の駆動構造

ファンディスクという形式は、本編とは違うリテンション設計を要求する。新規読者は基本的に想定されないため、章立てや関係性のグラデーションが本編と二重構造になる。

| 構造レイヤー | 本作での実装 | 物語消費への効果 |
|---|---|---|
| アフターストーリー | 本編で結ばれた 5 ルート + 追加キャラの後日談 | 「ハッピーエンドの先」を読む需要を回収 |
| if ルート | 本編で選ばなかった分岐の補完 | プレイヤーの未練（取りこぼし）を後付けで救済 |
| 世界観 | スチームパンク・ロンドン継続 | 美術／音楽資産の再利用で開発効率と没入感を両立 |
| 攻略対象 | 歴史・文学上の人物（ルパン、フランケンシュタイン等）の翻案 | キャラ単体の知名度が物語外で機能、初見の心理的ハードルを下げる |

ファンディスクの章立ては「本編エンディング → エピローグ → 短編集」という時間軸三層構造をとり、長編 RPG 的な達成感ではなく **短編集を読み進める読書体験** に近い。選択肢は感情の調整ノブとして機能し、ストーリーフロー自体を分岐させるよりキャラクター視点の濃淡を変える役割が大きい。

## バズ要因 / 人気の根拠

`popularity` が示すのは「本編 IP の熱量を後発作品が継承した」という事実である。これを分解すると：

- 本編『創世の姫君』が国内オトメイトファン層で確立した固定読者を持ち、その読者が **そのままファンディスクの初動需要** になった
- Aksys Games による英語ローカライズで海外オトメ市場に到達、北米・欧州の女性向け ADV 受容層に届いた
- 初出は PS Vita だが PS4・Switch へ多段移植され、ハード世代交代の度に **新規プレイヤー流入の窓** が再開された
- 攻略対象が「ルパン」「フランケンシュタイン」など世界文学・大衆文化の固有名詞を冠しており、ジャンル外の人にも口頭で説明しやすい

## 企画への示唆

- IP の熱量が一定以上あれば、続編より **ファンディスクで「結末の先」を売る** ほうが ROI が高い場合がある（資産再利用率が高く、コアファンの財布開度も高い）
- if ルートを後付けで実装すると、本編プレイ中に「選ばなかった後悔」を抱えた読者を二度目の購買に変換できる
- スチームパンク × 歴史人物翻案のように **既存固有名詞をキャラ名に流用** すると、ジャンル未経験者への説明コストを下げられる
- 携帯機初出 → 据置 / Switch 移植のラダー設計は、女性向けジャンルでも世代越え再販の有効ルートになる
- secondary を意図的に空にして「読む」体験だけに資源集中する設計は、レビュー軸を一本化でき口コミが拡散しやすい
