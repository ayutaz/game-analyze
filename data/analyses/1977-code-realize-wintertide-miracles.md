# Code:Realize 〜白銀の奇跡〜 詳細分析

## なぜ NAR 主分類なのか

プレイヤーの主動機は「本編で結ばれた相手の "その後" を読み続ける」こと、つまり消費する動詞は **読む** である。ゲームメカニクスではなくシナリオ追加と差分立ち絵で成立しているファンディスクで、戦闘も探索も育成も存在しない。

concept にある通り本作は『Code:Realize 〜創世の姫君〜』のアフターストーリー集であり、target も「前作クリア済みの 20–40 代女性プレイヤー」に明示的に絞り込まれている。乙女ゲームのファンディスクという商品形態自体が「特定キャラとの恋愛物語の延長視聴券」として設計されており、プレイヤーが買う理由は新規メカニクスではなく **既読関係性の継続描写** である。そのため NAR が妥当で、social_axis は null（純シングル読み体験）になる。

secondary は空。本編がもともと「読み物 + キャラ収集動機（5 人攻略コンプ）」を持つが、ファンディスクは攻略コンプ済み層が前提なので REW 動機は剥がれ落ち、純粋な物語消費レイヤーだけが残る構造になっている。

## 物語の駆動構造

ファンディスクは新規 IP よりも「どの粒度で物語を切るか」のパッケージング設計が肝になる。本作の構造を分解すると次のようになる。

| レイヤー | 役割 | プレイヤー側の駆動 |
|---|---|---|
| アフターストーリー | 本編エンディング後の各攻略対象との日常 | 「結ばれた後に何があったか」を確認したい欲求 |
| 新規シナリオ（白銀の奇跡本編） | クリスマス〜冬を軸にした新作短編 | 既存キャラ群の再集結を読む欲求 |
| サブエピソード / ミニ話 | ギャグ・サブキャラスポット | キャラ単位の補完欲求 |
| BGM / CG / 立ち絵差分 | 鑑賞メニュー | コンプ動機（残るのはここだけ） |

時間軸が「冬・クリスマス」に固定されているため、5 ルート分のアフターを別個に並べても季節感で世界観の一体性が担保される。本編で完結した恋愛関係を扱うので **選択肢による分岐は最小限**、ほぼ一本道のリーディングモードに寄せている。これは「分岐を読ませる本編」と「結末を確認させる FD」という乙女ゲームの伝統的役割分担に忠実な設計である。

## バズ要因 / 人気の根拠

- 前作『Code:Realize 〜創世の姫君〜』(2014) が乙女ゲーム市場で長期人気を維持していたため、ファンディスク需要が累積していた
- スチームパンク・ロンドンというビジュアル世界観が乙女ゲームとしては差別化が利き、海外ローカライズで Idea Factory International を通じて英語圏乙女ゲーマー層にも到達した
- 初出 PS4 / PS Vita (2017 国内) からの **Switch 移植 (2018)** によって、Vita 末期離脱層と Switch 新規層の二段階で取り込みに成功
- ファンディスクとしては珍しく Switch / PS4 / Vita のマルチプラットフォーム展開で、所持ハードの分散にも対応した

## 企画への示唆

- **本編で「攻略コンプ」動機を完結させたら、FD では報酬軸を捨てて純物語に振り切る** → REW を剥がして NAR に純化することで、想定ターゲット（本編クリア層）の読書時間を最大化できる
- **FD のシナリオを季節イベントに紐付ける** → 各キャラ別ルートを横断する世界観統一装置として、季節（本作なら冬・クリスマス）が機能する
- **本編クリア証明を前提にした商品設計** → 説明・チュートリアル・関係導入を全省略でき、シナリオ純度を上げられる代わりに新規参入障壁は意図的に高くする
- **マルチプラットフォーム後追い移植で生存延長** → Vita 末期タイトルでも Switch 移植で 1〜2 年単位で寿命を延ばせる（乙女ゲーム市場の Switch 移行パターン）
- **海外ローカライズを諦めない** → 日本国内では小規模でも、Idea Factory International のような専門 publisher を経由すれば英語圏乙女ゲーマー層という別市場が立つ

## 比較で見るべきタイトル

- 『AMNESIA LATER』『AMNESIA CROWD』(Idea Factory) — 同社系列の代表的乙女ゲーム FD、章立て構造の参照点
- 『うたの☆プリンスさまっ♪』シリーズ FD 群 — アイドル文脈での「本編クリア後のキャラ別アフター」設計
- 『Collar×Malice -Unlimited-』 — Otomate の同系列 FD、本編完結後の追加物語パッケージング
- 『Steins;Gate 比翼恋理のだーりん』 — 男性向けだが「本編後アフター集」という FD 構造の他ジャンル比較対象
