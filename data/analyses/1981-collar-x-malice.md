# Collar×Malice

## なぜ NAR 主分類なのか

プレイヤーの主動機は「24 時間タイマー付きの事件簿を、5 人の元警察官との関係性込みで読み解く」こと。動詞は **読む／推理する／関係を選ぶ**であり、戦闘や育成ループはない。

concept にある「新宿封鎖」「毒入り首輪」「24 時間以内の事件解決」は、ロマンスを駆動するための制約装置である。乙女ゲームの伝統的フォーマット（攻略対象を選ぶ → 個別ルート → エンディング分岐）の上に、サスペンス／ミステリーの章立てを乗せた構造で、プレイヤーは「会話選択肢で関係性を選ぶ」と「証拠を読み解く」を同時に行う。target が逆転裁判やダンガンロンパとの併読層を明示している通り、推理 ADV の構文を恋愛 ADV に輸入したハイブリッドであり、報酬軸（収集・育成）も体験軸（マルチプレイ・反射神経）も持たない以上、**物語消費が単独動機**として残る。secondary は空配列で、副次的な動機は設計上意図的に削ぎ落とされている。

## 物語の駆動構造

サスペンスと恋愛の二重螺旋を、章 × ルートのマトリクスで管理している。

| 駆動レイヤー | 仕掛け | プレイヤー側の動作 |
|---|---|---|
| 共通章 | 新宿封鎖／X-Day／毒入りチョーカーの時間制限 | 事件の輪郭を把握、攻略対象を選択 |
| 個別ルート | 5 人の元警察官それぞれの背景と X-Day への関与 | キャラ視点で事件の別解釈を読む |
| 選択肢分岐 | 関係性ゲージ＋推理選択肢 | 恋愛 END / バッド END / 真相 END の振り分け |
| 真相ルート解放 | 全ルート踏破後に開く meta 章 | 5 視点の差分から真犯人を再構成 |
| 拡張プロダクト | Unlimited（FD）／劇場アニメ Deep Cover | 完結後のアフターと前日譚で IP を延長 |

特徴は「攻略対象 = 事件の容疑者／関係者」であること。誰を選ぶかが恋愛の選好だけでなく事件のどの面を読むかの選択になり、ルート間で証言や因果が補完される。複数周回前提の設計で、1 周目のロマンスが 2 周目の伏線回収に変わる。

## バズ要因 / 人気の根拠

popularity 文を分解する。

- 発売週に Media Create で 7,271 本、ファミ通で 9,406 本。乙女ゲーム単一ブランド・PS Vita 専売としては高水準で、ニッチジャンル内では話題化に成功した。
- 「新宿封鎖」「毒入り首輪」という社会派サスペンス寄りのフックが、乙女ゲーマー以外（ミステリー / サスペンス ADV 層）の流入を生んだ。
- Switch 移植で母数を Vita 末期から救出し、PC 版で海外（Aksys 配給）にも届けた。3 プラットフォーム展開で長期可動カタログ化。
- ファンディスク Unlimited、劇場アニメ Deep Cover と派生コンテンツが続き、単発タイトルではなく IP として運用された。Otomate の主力枠の一つに定着。

## 企画への示唆

- **攻略対象を事件の関係者にする**と、恋愛選択がそのまま推理視点の切替になり、周回プレイの動機を恋愛と謎解きで二重化できる。
- **時間制限（24 時間以内）を物語の前提に置く**と、選択肢の重さが自動的に上がり、シナリオ密度を圧縮できる。
- **共通章で社会派の重い設定（封鎖／毒物／組織犯罪）を提示**し、個別章でロマンスに着地させると、乙女ジャンル外から検索流入を取れる。
- **真相 END を全ルート踏破で解放**するメタ構造を入れると、1 本のシナリオ単価でプレイ時間 5–10 倍を担保できる。
- **FD（Unlimited 型）と劇場アニメ化を出口に設計**しておくと、本編完結後に IP 寿命を 5–10 年延ばせる。Otomate / Aksys の標準モデル。
