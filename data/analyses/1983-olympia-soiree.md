# オランピアソワレ (Olympia Soiree)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「色で階層化された世界の物語を読み解く」ことにある。攻略対象との恋愛は媒介であって、目的は「太陽を取り戻す巫女・オランピア」の運命を最後まで追体験することだ。

concept にある「色によって階級が分かれる天空島」「太陽を取り戻す儀式」という設定、そして target が示す「倫理的にダークなテーマを許容できる成人向け乙女ゲームファン」という客層を踏まえると、本作の駆動力はメカニクスでも収集要素でもなく、シリアスな世界観テキストそのものである。social_axis が null なのも、ソロ消費のビジュアルノベルに他者軸を当てる余地がほぼないことを反映している。secondary が空なのは、収集 / 戦闘 / 育成のような副動機を持たない純粋ノベル設計であることを示す。

## 物語の駆動構造

オトメイト系乙女ビジュアルノベルとして典型的な「メインルート × ルート分岐 × FD 級サイドストーリー」を本編内で完結させているのが特徴。

| 構造レイヤー | 機能 | 本作での実装 |
|---|---|---|
| 世界設定 | 倫理的緊張の供給源 | 色による階級制度・「無色」差別という社会構造 |
| 主人公の役割 | 物語駆動の必然性 | 太陽を取り戻す儀式を担う巫女という不可避ミッション |
| 攻略対象 | 色 = テーマの擬人化 | 各色領主が世界観の異なる側面を体現 |
| サイドストーリー | 世界観の補完 | 30 本のサブエピソードで多視点的に階級制度を掘る |
| ESRB Mature | 表現の許容範囲拡張 | 暴力 / 性 / 差別描写を物語の必要要素として組み込み可 |

「攻略対象 = 色 = テーマ」という三項一致により、ルート選択がそのまま「どの社会的論点を読むか」の選択になる。サイドストーリー 30 本は、メインで描き切れない階級・差別の側面を補完するための文献的補強として機能している。

## バズ要因 / 人気の根拠

- イラストレーター Satoi 氏の色彩設計が「色で世界を分ける」というコンセプトと完全に同期し、ビジュアル単体で SNS 拡散の起点となった。
- ESRB Mature 指定という乙女ゲームでは珍しいレーティングを取り、欧米ローカライズ (Aksys Games) 経由で重厚系乙女ゲーマー層に届いた。
- Metacritic 等で海外乙女ゲームレビューサイトから高評価を獲得し、英語圏のコア乙女コミュニティで言及が継続した。
- Switch 独占で 2020 年発売、その後 2024 年に 4Gamer 等で改めて取り上げられ、ロングテールで再評価が続くタイプの作品となった。

## 企画への示唆

- 攻略対象を「単なる恋愛相手」ではなく「世界設定のテーマを擬人化した存在」として設計すると、ルート選択がそのまま社会的論点の選択になり、消費体験が知的に厚くなる。
- レーティングを意図的に Mature まで引き上げると、表現可能な倫理的論点（差別・性・暴力）が広がり、ターゲット年齢を 20-40 代成人女性層に明確化できる。
- メイン本編にサイドストーリー 30 本級を内包すると、別売 FD を待たずに「世界観マニア」需要を一作で吸収できる（買い切りビジュアルノベルの強み最大化）。
- イラストレーターのアートディレクションをコンセプト（色・階級・儀式）と一体化させると、ビジュアル単体で SNS 拡散時のコンセプト伝達が成立する。
- 北米ローカライズパブリッシャー (Aksys) と組むと、国内ニッチ層向けの重厚系乙女作品でも英語圏のコアコミュニティに届き、ロングテール評価を得られる。
