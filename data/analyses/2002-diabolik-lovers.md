# DIABOLIK LOVERS

## なぜ NAR 主分類なのか

プレイヤーの主動機は「吸血鬼 6 兄弟に噛まれ・支配される一人称の物語を読み進める」ことであり、システムは選択肢付きビジュアルノベルに極限まで寄せている。concept にある通り、各キャラルートが Dark / Maniac / Ecstasy の 3 層構造で展開する設計は、戦闘や育成といった体験ループではなく「同じキャラの異なる支配段階を読み直す」ための物語パッキングである。プレイヤーは小森ユイを通じて 6 人 ×3 層 =18 単位の関係性ドラマを消費しに来ており、これは典型的な NAR の動機（読む／追体験する）に該当する。

target が「ダークで支配的な男性キャラに惹かれる乙女ゲーム／BLCD ファン層」と明示されている点も NAR を強化する。Rejet 系シチュエーション CD のヘビーリスナーと地続きであることから、コンテンツは「キャラ別シチュエーションのアンソロジー」として消費される。secondary は空で、収集や対戦の動機は副次的にも設定されていない。

## 物語の駆動構造

Dark / Maniac / Ecstasy という 3 層ルート構造は、単なる「グッド／バッドエンド分岐」ではなく、同一キャラとの関係深度を段階化することで再読の必然性を作っている。下に駆動要素を分解する。

| 構造要素 | 内容 | 物語消費への効果 |
|---|---|---|
| キャラ枠数 | 逆巻家 6 兄弟（後続作で 6 兄弟＋ムカミ家が追加） | 6 × 3 = 18 ルート分の縦深を一作で確保 |
| 階層構造 | Dark → Maniac → Ecstasy の 3 段階 | 同キャラを 3 回読み直す動機を内蔵 |
| 主人公の立ち位置 | 牧師の娘 × 噛まれる側 | ドS × ドM の固定役割で読み手の入り方を一意化 |
| 出自ミステリ | ユイ自身が吸血鬼との因縁を持つ伏線 | キャラ別ルートを縦軸の謎で束ねる |
| ボイス比率 | フルボイス・シチュエーション CD 文脈 | 「耳元で噛まれる」聴覚消費が物語の駆動装置に |

噛む／血を奪うという行為がキャラ別の儀式として演出され、ASMR 的な聴覚消費と物語消費が分離できない設計になっている。シチュエーション CD 文化が PSP / PS Vita のヘッドホン前提プレイと噛み合った例。

## バズ要因 / 人気の根拠

popularity に「ドM 向け吸血鬼ボーイズに『噛まれる』という S な展開で乙女ゲーム界に衝撃を与えたヒット作」とある。これを事実層で分解する。

- 2012 年 PSP リリース後、Rejet 系シチュエーション CD で先行構築されたファンベースを直接 IP に変換できた（同社のドラマ CD 文脈との地続き性）
- 「噛まれる」というドS × ドM の役割固定が、当時主流だった「優しいイケメンに守られる」乙女ゲームに対する明確な差異化軸として機能した
- 2013 年テレビアニメ化、続編 More,Blood、Lunatic Parade、Dark Fate、Lost Eden、Chaos Lineage と PSP / Vita / Switch をまたいでシリーズ化、舞台化・CD ドラマ・キャラソン CD 等の多面展開で長期 IP に発展
- ヒロイン名義の「キャラソン CD」「シチュエーション CD」が物販単独で回り、本編プレイ前にキャラを認知させる導線として機能

## 企画への示唆

- ルートを「分岐」ではなく「層」として設計すると、同キャラを 3 回読む動機を作れる（Dark / Maniac / Ecstasy の階層化が再消費を生む）
- 既存ファンベースを別メディアで温めてからゲーム化すると、初動の集客コストを大幅に下げられる（Rejet シチュエーション CD → ゲーム化の順序）
- 「主人公の役割固定」を明示すると、合わない層は離れるが合う層の没入が深くなる（万人向けを捨てると IP が長期化する）
- 携帯機 × ヘッドホン前提のフルボイスは、聴覚を物語駆動装置に組み込める（噛み音・吐息の演出が PSP / Vita 世代と噛み合った）
- 「タブー寄りの欲望」を明示的に肯定する商品設計は、レビュー賛否は割れても継続購入率が高くなる（潜在ニーズの可視化が忠誠度を生む）
