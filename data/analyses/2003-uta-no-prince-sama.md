# うたの☆プリンスさまっ♪

## なぜ NAR 主分類なのか

プレイヤーの主動機は「6 人のアイドル候補生それぞれの物語を読み、推しのデビューまでを追体験する」ことに集約される。concept にあるとおりゲーム本体は早乙女学園を舞台にした恋愛 ADV で、作曲ミニゲームとリズム試験は「物語の節目を演出する儀式」として機能しており、ミニゲーム自体の腕前を競うことが目的化していない。target が「10〜30 代女性、推しキャラを深く追体験したい層」と定義されていることからも、物語消費＋推し選びが主動機であることが分かる。

secondary に REW が入るのは、ルート分岐ごとの CG・楽曲・スチル収集、後年のシリーズ作（Repeat / Debut / All Star 等）での累積収集要素が、リピート購入を駆動する副次レイヤーとして効いているため。社会軸はソロ完結型の読み物であり `social_axis` は null。

## 物語の駆動構造

PSP 期の女性向け ADV では珍しく、テキスト分岐に「作曲ミニゲーム」「リズム演奏試験」を挟むことで、章間に強い儀式性を作っている。物語の駆動要素を分解すると次のようになる。

| レイヤー | 役割 | プレイヤー行動 |
|---|---|---|
| 共通ルート（学園パート） | キャラ 6 人の関係性を提示する導入 | テキストを読み、選択肢を選ぶ |
| 個別ルート | 推し 1 名の背景・葛藤・恋愛の深掘り | キャラ別の選択肢で好感度を積む |
| 作曲ミニゲーム | 「主人公が作曲家志望」の世界観を担保 | 歌詞ピース選択で曲を仕上げる |
| リズム演奏試験 | 章末イベントとして物語にリズムを与える | タイミング入力でデビュー試験突破 |
| EDテーマ／挿入歌 | キャラ専用楽曲を物語の余韻に貼り付け | 視聴（後の CD / ライブ導線） |

「楽曲＝そのキャラの物語の総括」という対応関係を作ったことが構造上の肝で、ゲーム本編のエンディング曲が後にアニメ OP / ライブセトリ / CD シングルとして再利用され、物語体験を商品レイヤーへ橋渡しする。

## バズ要因 / 人気の根拠

popularity の「PSP 発の女性向け恋愛 ADV＋リズムが大ヒットし、アニメ・劇場版・ライブと連動して一大 IP に発展」を分解すると、

- 主人公（女性）×アイドル候補生 6 人という、当時の女性向け ADV の標準枠に「音楽デビュー」という共通目標を載せ、物語の終点を明確にした
- A-1 Pictures 制作のアニメ（2011〜）、劇場版（2019〜）、ライブイベント連動で、ゲーム外でキャラと楽曲を継続消費できる動線を整備した
- 楽曲を歌うキャラ役声優が実際にライブで歌唱する「2.5 次元」導線を、後のアイドル系コンテンツに先行して確立した
- シリーズ累計で複数の続編・移植・スマホ展開（Shining Live 等）が出るほどの長寿 IP となり、女性向けアイドル IP の典型例として参照され続けている

## 企画への示唆

- 主人公の役割を「ファン」ではなく「共犯者（作曲家）」に置くと、推しの成功＝主人公の成功になり、感情移入が深くなる
- 章末にミニゲームを挟むと、テキスト一辺倒の女性向け ADV にリズムができ、可処分時間の使い方として息継ぎが生まれる
- キャラ別エンディング曲を 1 曲ずつ用意すると、その曲が後年のライブ／ CD ／配信での再収益化資産になる
- ゲーム → アニメ → ライブ →CD → 続編、という消費階段を最初から設計しておくと、原作売上が小さくても IP として伸ばせる
- 「アイドル候補生」設定はキャラの成長余地を内蔵しているため、続編・スマホ版で「デビュー後の世界」を描く拡張が自然に効く

## 比較で見るべきタイトル

- 金色のコルダ（コーエー、2003）— 女性向け＋音楽演奏という先行モデル
- ときめきメモリアル Girl's Side（コナミ、2002）— 学園恋愛 ADV の女性向け定型を確立
- アイドルマスター SP（バンナム、2009）— アイドル育成＋リズムの男性向け対照例
- あんさんぶるスターズ！（Happy Elements、2015）— うたプリ後のスマホ世代女性向けアイドル IP
- 薄桜鬼（アイディアファクトリー、2008）— 女性向け ADV のアニメ・舞台展開先行例
