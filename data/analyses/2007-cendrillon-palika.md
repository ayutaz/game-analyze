# Cendrillon palikA

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ガラスに侵食されていく街と 7 人の男性キャラクターの結末を読み切る」ことであり、戦闘・収集・対戦のループは存在しない。動詞は「読む」と「選ぶ」の 2 つに収斂する。

concept にある通り、本作は童話シンデレラを下敷きにしつつ「透京（とうけい）」というガラスに呪われた街で結末分岐を駆動する。好感度に加えて選択肢「審判ノ刻」と隠しパラメータ「透刻度」が条件分岐に効くため、プレイヤーは攻略対象との関係性ではなく **世界そのものの結末** に介入していく構造になる。target も「ダークファンタジー寄りの世界観と童話モチーフを好む 10-30 代女性」と物語消費前提に絞られており、social_axis が null（他者との同期的・非同期的関わりがゲーム内に存在しない）であることも NAR を後押しする。secondary は空、つまり報酬ループも体験設計も主従関係上の従にすら回らない、純粋な一本道型物語商品である。

## 物語の駆動構造

| 駆動要素 | 仕様 | 物語上の効果 |
|---|---|---|
| 攻略対象 7 人 | シンデレラ童話の人物配役を再解釈 | 1 周クリアでは世界観の片面しか見えず、複数ルート前提 |
| 「審判ノ刻」選択肢 | 通常の好感度選択肢とは別レイヤー | プレイヤーに「誰を救うか」より重い倫理判断を強いる |
| 「透刻度」パラメータ | 街がガラスに侵食される進度を数値化 | 恋愛感情とは独立に Bad End / True End を決める世界軸 |
| ガラスの靴 | シンデレラモチーフのキーアイテム | 主人公の能動性を担保する装置（受動的ヒロイン回避） |

恋愛 ADV としての好感度ループに、世界滅亡進度という第二軸を上乗せした **二軸分岐型** が骨格で、Otomate 作品群の中でも「恋愛より世界線」寄りに寄せられている。

## バズ要因 / 人気の根拠

- 2018 年 Switch 版発売時、童話シンデレラ × ダークファンタジーという題材が乙女ゲームファンダム内で世界観評価を獲得し、4Gamer など専門メディアで世界観・ビジュアル紹介中心の記事が組まれた。
- 「透京」というガラスの街、攻略対象の整合的な配役、楽曲・スチルなど **ビジュアル＋音楽＋設定のパッケージ完成度** が SNS 上の口コミに乗りやすかった。
- 2023 年に Idea Factory がモバイル向けに展開する「オトメイトセレクション」アプリでスマホ版が配信開始され、Switch を持たない新規層に対して再リーチが起きた（旧作の二次流通効果）。
- Otomate ブランドの既プレイヤー層に対して「シリーズの新規 IP として安心して触れる」という参入障壁の低さがあり、外伝・ファンディスクなしの単発作品ながら継続的に名前が上がるロングテールを獲得している。

## 企画への示唆

- **二軸分岐を導入すると、恋愛 ADV のリプレイ動機がキャラ数ではなく世界軸の網羅に転化する**：好感度だけだと「7 人クリアで終わり」だが、透刻度のような世界進度を足すと「同じキャラで複数結末を見る」需要が生まれる。
- **童話の再解釈は世界観コストを劇的に下げる**：シンデレラ / ガラスの靴 / 7 人という記号は説明不要で、節約した文字数を独自設定（透京・審判ノ刻）に振れる。
- **モチーフをモノ（ガラス）に固定すると美術が一貫する**：ガラスというマテリアルを街・呪い・パラメータすべてに通底させることで、スチル / UI / 楽曲のディレクションが揃い、ビジュアル評価が上がる。
- **発売 5 年後にモバイル再配信を仕込むと、旧作 IP は二度バズる**：オトメイトセレクション型の旧作モバイル化は、開発コストほぼゼロで認知再点火できる施策として再現性が高い。
- **能動性を担保する小道具を主人公に持たせると受動ヒロイン批判を回避できる**：本作のガラスの靴は、主人公を「救われる側」から「呪いに介入する側」へ反転させる装置として機能している。

## 比較で見るべきタイトル

- **Code:Realize 〜創世の姫君〜**（Otomate / 2014）：ダークファンタジー世界観 × Otomate という同系譜。
- **薄桜鬼**（Otomate / 2008）：世界軸（時代の終わり）が恋愛より重い、Otomate の二軸分岐型の先行例。
- **Collar×Malice**（Otomate / 2016）：プレイヤーの選択が事件解決の世界線を動かす、サスペンス寄りの NAR。
- **House in Fata Morgana**（Novectacle / 2012）：童話・寓話モチーフをダークに再解釈した日本産ノベルの代表例（乙女ジャンル外比較）。
