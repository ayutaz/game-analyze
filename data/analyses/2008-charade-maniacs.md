# CharadeManiacs / Charade Maniacs

## なぜ NAR 主分類なのか

プレイヤーの主動機は「異世界アルカディアで進行する群像ミステリーを読み解く」こと。恋愛シミュレーションの皮を被りつつ、実際の駆動力は世界観の謎と裏切り者探しというサスペンス構造にある。concept にある「ドラマ（嘘）を演じながら裏切り者を探す」という設定上、プレイヤーは攻略対象との恋愛感情よりも「誰が本当のことを言っているのか」を追跡する読解者として座席に着く。

target が「ライトな恋愛より、伏線回収と世界観の謎を味わいたいプレイヤー」と明示されている通り、本作は乙女アドベンチャーのジャンル文脈の中でも NAR 寄りに振り切った設計。social_axis は null（=ソロ消費）で、選択肢分岐による物語消費がコア体験そのもの。secondary は空で、収集／対人要素を意図的に排した一本道型 NAR と判断できる。

## 物語の駆動構造

10 人の被拐者・SF ミステリー・乙女アドベンチャーという 3 つの異質な系を、フローチャートと調査パートで束ねている。物語消費を駆動するのは「シナリオ通りに演じるべきか／逸脱すべきか」という分岐ルールで、選択が単なる好感度ではなく真相到達条件に直結する点が特徴。

| 構造要素 | 仕掛け | 物語消費への効果 |
|---|---|---|
| 舞台「アルカディア」 | 二つの月の異世界、10 人監禁、与えられた台本 | 世界観の謎自体が縦糸の question になる |
| 共通ルート | 全攻略対象が並走する群像劇 | 個別 ED より先に「世界の真相」を提示する余地が生まれる |
| 調査パート | 選択肢＋情報収集で疑念を更新 | 受動的な読書を能動的な推理に変換 |
| フローチャート | 既読分岐を俯瞰、未踏ルートを誘導 | 複数周回前提の伏線回収を成立させる |
| キャスト | 斉藤壮馬・鈴村健一・浪川大輔ほか | 演技で「嘘をついている瞬間」が音声から推察できる |

恋愛 ED は到達条件であって主目的ではなく、世界の真相にアクセスするためのキーとして機能している。

## バズ要因 / 人気の根拠

- SF ミステリー × 乙女という異色ジャンル接合で、シナリオ重視層に「Otomate の中でも読み応えのある作品」というポジションを獲得した
- 斉藤壮馬・鈴村健一・浪川大輔ら売れ筋男性声優の起用で、声優ファン経由のリーチが効いた
- 2018 PS Vita 発売 → 2021 Switch 移植 → 2023 英語版（Idea Factory International / Aksys Games）と、約 5 年かけて配信プラットフォームを段階的に拡張
- 舞台「CharadeManiacs 再演」が成立しており、原作 IP がメディアミックスに耐える物語密度を持つことが市場に証明された

## 企画への示唆

- 乙女ジャンルに SF やサスペンスを接合すると、恋愛飽和層に新規スロットを作れる
- 「台本通りに演じるか逸脱するか」を選択肢の意味づけに使うと、好感度メーターを真相到達条件に昇格させられる
- 群像劇＋共通ルートの厚みを先に積むと、個別 ED が「真相のピース」として機能し周回モチベが持続する
- Vita → Switch → 英語版の三段移植は、ニッチ国内 IP をグローバル乙女市場に届ける標準ルートとして再現性がある
- 舞台化を視野に入れる場合、固定空間（密室・閉鎖世界）と固定人数（10 人前後）の設定は脚色コストを下げる
