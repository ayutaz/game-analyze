# パラダイス・キラー / Paradise Killer

## なぜ NAR 主分類なのか

プレイヤーの主動機は「読む」――関係者全員の長尺対話と物証テキストを読み込み、自分の中で物語を再構成して裁判で結論を提示することにある。concept にある通り「物的証拠を欠いても起訴できる主観的真実」が核であり、ゲームは正解を返さない代わりに、読者＝プレイヤーが選び取った筋書きそのものを物語の結末として確定させる。

ただし本作は典型的なビジュアルノベルではない。target が示すように『ディスコ・エリジウム』『13 機兵防衛圏』の系譜に位置し、ヴェイパーウェーブの島を一人称で歩き回る `secondary: EXP` の探索層が読みを下支えする。歩き回って手がかりを拾う行為は楽しいが、それ自体が目的ではなく「読むべきテキストを開く鍵」として機能している。だから主分類は NAR、社会軸は null（単独完結のソロ物語消費）でよい。

## 物語の駆動構造

物語消費を駆動しているのは、複数容疑者・複数仮説を並行管理させる「捜査ノート（Starlight）」と、裁判で確定するまで真実が宙吊りのままという構造的引き伸ばしである。

| 構造要素 | 役割 | 物語駆動の効果 |
|---|---|---|
| Starlight（捜査ノート） | 発言・物証・仮説の自動整理 | プレイヤーが自分の推理を可視化し、矛盾を発見できる |
| 容疑者 9 名の同時聴取 | 並行ルート探索 | どの順番で誰を詰めるかでテキスト体験が変わる |
| 仮想通貨 Blood Crystals | 探索報酬 → ファストトラベル/情報購入 | 読みのテンポを自分で買える |
| 裁判フェーズ | プレイヤーが告発対象と動機を確定 | 「読了」が能動的儀式になる |
| 主観的真実 | 物証なしでも起訴可 | 解釈の余地を残し二周目動機を作る |

章立てが固定化された VN と違い、空間と時間（島の踏破順）を自由化することで「同じ事件を異なる順序で読む」体験を生んでいる点が、純粋なテキストアドベンチャーとの差分である。

## バズ要因 / 人気の根拠

popularity の「ヴェイパーウェーブ美術＋自由裁判」が出発点。これを分解すると：

- ヴェイパーウェーブ / シティポップ / Y2K を一人称オープンワールドに持ち込んだ視覚言語が、Switch 版配信後に SNS スクリーンショット拡散の燃料になった
- 「誰を犯人にしてもゲームは進行する」という Kaizen Game Works の設計判断が、英語圏の批評家レビューで構造そのものを論じる対象になりやすかった
- BGM（Barry "Epoch" Topping）がサウンドトラック単体で Bandcamp 評価を獲得し、ゲーム外チャネルから流入経路を作った
- Fellow Traveller（『In Other Waters』『Citizen Sleeper』のナラティブ系インディー publisher）配下で、テキスト密度型ファン層にリーチしやすい流通網に乗った

具体的な売上は公開されていない（sales_world: unknown）が、Switch 版を契機に日本のインディーレビュー媒体（4Gamer / Famitsu / Leoful）で個別記事化され、カルト層を獲得した経緯が裏付けられる。

## 企画への示唆

- **正解を返さないと、考察コミュニティが代わりに正解を作る** → 主観的真実モデルは UGC とレビュー記事を誘発する
- **テキスト密度型ゲームに一人称オープンワールドを被せると、空間が「目次」になる** → どこから読むかを地理で表現できる
- **美術様式を一つの流派に振り切ると、ジャンル外の文化圏から流入が起きる** → ヴェイパーウェーブ層のような非ゲーマー読者を呼べる
- **小規模スタジオが Fellow Traveller 系のナラティブ専門 publisher を経由すると、批評家レビュー枠が確保しやすい** → 流通先選びはトーン設計の一部
- **「証拠を集める」と「告発する」を別フェーズに分けると、プレイヤーの能動性が儀式化される** → クライマックスをメカニクスで作れる
