# シャーロック・ホームズ チャプター・ワン

## なぜ NAR 主分類なのか

プレイヤーの主動機は「若きホームズが母の死の真相に辿り着く前日譚を、自分の手で読み解く」ことにある。事件解決は手段であり、終着点はホームズという人物の自己形成譚を見届けることに置かれている。

concept で示される「ワトソンと出会う前の若きシャーロックが地中海の島コルドナへ戻る」設定は、原作キャラクターの空白期を物語的に埋めるための装置であり、target も Disco Elysium や L.A. Noire 等「読み物として推理する」コア層と明示されている。オープンワールド構造は採用されているが、これは島を踏破することが目的ではなく、メイン事件と母のサブプロットを物語的に縫合する舞台として機能する。よって NAR が妥当である。

secondary に EXP が入るのは、観察・推理ボード・変装・尋問・現場再構築といったメカニクスがプレイヤー操作として残っているため。物語進行が「自分の推理で犯人を断定する」決断を経由する点で、体験型の意思決定が物語消費に組み込まれている構造になっている。

## 物語の駆動構造

中核は「個人的動機（母の真相）」を背骨に、島の事件を章として連ねる前日譚カレンダー型ADVである。プレイヤーが推理ボード上で証拠を結線し、最終的に「断定する」決断を下すまでセーブ前にやり直せない仕様が、物語消費を強制的にプレイヤー判断へ接続する。

| 構造要素 | 役割 | NAR への寄与 |
|---|---|---|
| メイン: 母コルドナ帰還 | 縦軸の感情線 | ホームズ像の自己形成を背骨に据える |
| サブ: 島で起きる事件群 | 横軸の章立て | オープンワールドを物語ユニットに分割 |
| 推理ボード | 証拠 → 結論の結線 UI | プレイヤー側の解釈責任を可視化 |
| 「断定」決断 | 犯人を自分で指名 | 物語結末をプレイヤー判断で確定 |
| 変装・尋問 | NPC からの情報引き出し | 章内テンポを EXP 的に支える |
| ジョン（ワトソン以前の相棒） | 内面の対話相手 | モノローグを会話劇に変換 |

## バズ要因 / 人気の根拠

popularity の核は「発売 3 週間で Frogwares 史上最速ペースの売上、シリーズ歴代最速ヒット」という記録にある。これを分解すると：

- Frogwares 自社販売・自社開発体制でのリリースで、Crimes & Punishments / The Devil's Daughter 等を経た 20 年超のシャーロック・ホームズ ADV シリーズ蓄積の上に立つ続編枠だった
- 「若きホームズ前日譚 + 地中海の島オープンワールド」という未踏の組み合わせが、原作・映像 IP ファンと推理 ADV コアの両方を刺激した
- 母の死というパーソナルな縦軸が、事件もの ADV にしては珍しい感情訴求として国内外メディアで取り上げられた
- ロシア・ウクライナ拠点開発スタジオの自社パブリッシングという背景もメディアの言及点として機能した

## 企画への示唆

- 既存 IP のキャラクター空白期（前日譚 / 学生時代 / 出会う前）を題材に取ると、原作ファンと新規層の両方に営業面が立つ
- 「事件 = 章ユニット」「個人的動機 = 縦軸」の二層構造に分解すると、オープンワールドでも物語の中心を見失わせずに済む
- 推理ゲームで「正解判定をシステム側で即時返さない」設計にすると、プレイヤーの解釈責任が増し、物語への没入が深くなる
- 推理ボード等の可視化 UI を導入すると、プレイヤーが自分の推論過程を物語の一部として体験できる
- スタジオの過去作系譜（同 IP の旧作）を継続することで、レビュー初動と既存ファン基盤の動員速度が確保できる

## 比較で見るべきタイトル

- Disco Elysium（NAR / ソロ）— 推理を物語的選択肢へ完全に翻訳した極北
- L.A. Noire（NAR / ソロ）— 尋問・現場再構築を中心に据えた推理 ADV の前世代
- Return of the Obra Dinn（NAR / ソロ）— 「断定」型推理の純粋形
- Sherlock Holmes: Crimes & Punishments（NAR / ソロ）— 同 Frogwares 直系の前世代作
- Pentiment（NAR / ソロ）— 章立てとキャラクター関係性で前日譚的厚みを作った同時代作
