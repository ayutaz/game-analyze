# 極限脱出ADV 善人シボウデス

## なぜ NAR 主分類なのか

プレイヤーの主動機は **「分岐した世界線を行き来して真相を読み解く」** ことにある。9 人が「アンビデックス・ゲーム」で裏切り／協力を迫られる SF サスペンスを、タイムリープ＋多重分岐＋メタ構造で全ルート読破させる設計であり、消費されるのは脱出パズルの達成感ではなく **複数ルートを束ねた巨大な物語の地図** である。

concept にあるとおり「全ルート読破して真相に辿り着く」構造は、フローチャートそのものが読書体験のインターフェイスになっており、社会軸を持たない単独プレイの読み物として完結する（social_axis = null）。target も 999 / Ever17 / Steins;Gate 系の **ループもの・メタ構造 AVN の読み手** に明確に向けられている。

secondary に EXP が入るのは脱出パズル部分が「読む」体験を一時停止させる能動操作として機能するため。ただしパズル難度の評価より「ルート分岐の鍵を回収した感覚」が中心で、あくまで物語装置の側面が強い。

## 物語の駆動構造

| 駆動装置 | 効果 |
|---|---|
| アンビデックス・ゲーム（裏切る／協力する 2 択） | 1 ラウンドごとに 9 人の関係が再編、選択が即ルート分岐になる |
| フローチャート式の全ルート俯瞰 UI | プレイヤーがメタ視点で「未読の枝」を直接管理、再読の摩擦を消す |
| ロック（封印された選択肢） | 別ルートで得た情報がないと進めない箇所を作り、横断読みを強制 |
| タイムリープ／意識遷移という SF 設定 | キャラクター自身が他ルートの記憶を持つことを正当化、メタ構造を物語内に内包 |
| 9 人 × AB 各々の動機 | 同じシーンが視点と前提情報で別物に化ける |

前作『999』が DS の上下 2 画面を物語装置に組み込んでいたのに対し、本作は **フローチャート UI を物語そのものに昇格** させ、「分岐の網目を編む」読書体験へ更新した。脱出パズル部屋は分岐の合間に置かれ、読書のリズムを刻む拍子木として機能する。

## バズ要因 / 人気の根拠

popularity の「日本では振るわず、海外評価で再評価された」を分解すると：

- **海外メディアの高評価が先行**：英語圏のレビューで複数ルート構造とメタ展開が突出評価され、カルト的支持を形成。
- **日本市場では商業的に伸びず、続編『刻のジレンマ』が一時凍結**：パッケージ AVN の縮小期と重なり、国内売上は公開されないレベルに留まった。
- **後年バンドル『The Nonary Games』で前作 999 とセット再販**：PC / PS4 / Xbox One への展開で実況・配信文化と接続し、再評価の波が起きた。
- **「全員が嘘をついている可能性がある」前提**：考察コミュニティ向きの素材として配信・ブログ・Wiki で長期消費された。

## 企画への示唆

- **フローチャートを UI ではなく物語装置に格上げすると**、「再読」が苦痛ではなく能動的な探索行為に変わる。
- **ルート間ロックを設計に組み込むと**、横断読みが強制され「全ルート読破＝真相」の納得感が生まれる。
- **メタ構造を SF 設定で正当化すると**、キャラクターが他ルートの知識を持っても破綻せず、伏線回収の自由度が跳ね上がる。
- **パッケージ AVN が国内で振るわなくても**、後年のバンドル再販＋多プラットフォーム展開＋配信文化への露出で再評価できる余地がある。
- **9 人 × 2 択の同盟動学**を物語の中心に置くと、関係性の組み替えが章ごとの推進力になる。
