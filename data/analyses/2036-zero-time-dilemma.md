# ZERO ESCAPE 刻のジレンマ

## なぜ NAR 主分類なのか

プレイヤーの主動機は「**9人が殺し合う密室サスペンスの真相を読み解く**」こと。三部作の伏線を回収し、複数タイムラインの矛盾を辻褄合わせる行為が中心で、脱出パズルや決断は物語消費を駆動する触媒として配置されている。

concept にあるとおり「断片的なフラグメントを時系列横断で繋ぎ合わせる構成」が骨格で、プレイヤーは章を順に追うのではなく、バラバラに提示されるエピソードを self-assemble して因果関係を構築する。target に挙がる前 2 作「999」「VLR」既プレイ層が中心という事実も、本作の本体が **シリーズ累積した謎の決算**であることを示している。secondary に EXP が入るのは、各エピソードに残る脱出パズル（密室の機械仕掛けを解く部分）が体験として残っているため。ただしパズルそのものは進行ゲートで、難度より物語進行の調速装置として働いている。

## 物語の駆動構造

本作の駆動エンジンは「**フラグメント＋ディシジョン・ゲーム＋三チーム並走**」の三層構造で、章立ての権限をプレイヤーに渡すことで生まれる。

| 構造要素 | 仕組み | プレイヤー体験への効果 |
|---|---|---|
| フラグメント方式 | 全 90+ の短編エピソードがタイムライン上に散在、任意順で解放 | 「次に何を見るか」の選択そのものがメタ推理になる |
| ディシジョン・ゲーム | 6人死ねば脱出という設定下で、致死選択を強要する分岐点 | 倫理的決断がそのままルート分岐に直結し、傍観できない |
| Cチーム / Qチーム / Dチーム | 3 グループが別棟で並走、相互に情報の死角を持つ | 同時刻の出来事を別視点で再構成、矛盾自体が手掛かり |
| フルボイス＋3Dムービー化 | 前作までのスチル＋テキストから映像演出に転換 | 暴力描写と心理ショックの強度が上がり、CERO Z で出荷 |
| シリーズ完結のメタ構造 | 999/VLR の登場人物が再登場、Radical-6 など世界線が回収 | 既プレイ層への報酬と新規層への閾値が同時に立つ |

ループは「フラグメント開放 → 局所決断 → 他チームの矛盾発覚 → 既存フラグメント再解釈」というサイクルで、線形 ADV の章進行を**非線形のジグソーパズル化**している。

## バズ要因 / 人気の根拠

- 「999」「善人シボウデス」と続いた Zero Escape 三部作の**完結編**という位置づけで、6 年越しの伏線回収が読まれる構造的需要を抱えていた
- フラグメント方式と 3 チーム並走が公開時に新規性として語られ、コアファンの間で「どの順番で開放すべきか」「どのチームが本筋か」という考察コンテンツが多く生まれた
- 3DS / PS Vita / Steam / PS4 とマルチプラットフォーム同時展開し、北米 Aksys Games が並行ローカライズしたため**海外カルト人気**を獲得
- ジャンルが「アドベンチャー / ビジュアルノベル / 脱出」で、ローカライズ ADV を継続的に消化する海外 VN 愛好家層が固定客として残った

## 企画への示唆

- **章順をプレイヤーに渡すと推理装置になる**：時系列を崩した断片提示は、提示順そのものをメタゲーム化できる
- **完結編というラベルは購入動機を兼ねる**：3 作目を「最終回」と宣言することで既存読者の購入を確定させ、未プレイ層には敷居を上げる代わりに pre-order を稼げる
- **倫理的決断をルート分岐に直結すると傍観者をなくせる**：選択肢の重さが UI のクリック行為と切り離されないとき、プレイヤーは物語に責任を持つ
- **複数視点並走は矛盾そのものを手掛かりにできる**：3 チームの情報差を意図的に作り、「相手の知らないこと」を読者だけが知っている状態がサスペンスを駆動する
- **映像演出への切り替えはシリーズ末期に効く**：スチル＋テキストで蓄積したファンに対し、最終作だけ 3D ムービー化する投資は CERO Z 級の衝撃と引き換えに記憶に残せる
