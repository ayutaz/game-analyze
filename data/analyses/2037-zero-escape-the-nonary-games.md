# ZERO ESCAPE The Nonary Games

## なぜ NAR 主分類なのか

プレイヤーの主動機は「9 人が密室から脱出するシナリオの分岐を読み切り、フロー図を埋めて真相に到達する」ことにある。脱出パズルは挟まるが、パズルそのものを反復遊技する設計ではなく、各部屋を 1 度クリアすればフラグとして消費されてしまう一方通行型である。プレイヤーが時間を投下するのは、分岐選択肢と複数ルートを横断しないと結末が見えない物語構造の解読作業であり、ここに NAR の主動が立つ。

concept にある「フロー図」は本作の特徴的な UI で、ノベル系作品が暗黙にしていた「どこから読み直すと別ルートに入れるか」を可視化した。target にある「打越鋼太郎・打越脚本系」愛好層という記述が示す通り、消費されているのは作家の物語構造そのものである。secondary に EXP が立つのは、脱出パズル部分が単なる演出ではなく実プレイ時間の数割を占めるためで、ノベルに「手を動かす緊張」を継ぎ足す副次機能を担っている。

## 物語の駆動構造

複数ルート分岐 + フロー図 + 時系列パズルという 3 層が、プレイヤーに「読み返し」を強制する駆動構造になっている。

| 層 | 機能 | プレイヤーに引き起こす行動 |
|---|---|---|
| 分岐選択肢 | 9 人をどう組み合わせるか / どのドアを開けるか | 1 周目で必ずバッドエンドを踏ませる |
| フロー図 | 既読・未読ルートをツリーで可視化 | 任意ポイントから再開、全埋めの達成感を喚起 |
| 時系列パズル | 9時間9人9の扉では作中時系列に仕掛けがある | 全ルート読了後にメタ層で再解釈を要求 |
| 脱出パズル | 各部屋で物理的な制限時間下の謎解き | ノベルの受動性を能動的緊張に切り替える |

『9時間9人9の扉』『善人シボウデス』の 2 本が合本収録されている合本構造そのものも、シリーズの世界観継続性をフロー図的に俯瞰させる外側の枠になっている。

## バズ要因 / 人気の根拠

- DS 版『9時間9人9の扉』(2009) と 3DS/Vita 版『善人シボウデス』(2012) が国内では小〜中規模ヒットに留まりつつ海外で熱量を獲得、本合本(2017)が「シリーズ再評価の起点」として popularity で位置づけられている。
- Steam / PS4 / PS Vita / Xbox One / Xbox Series X|S と長期にわたってプラットフォームを継ぎ足し続け、2017 年発売作品としてはロングセラーの SKU 展開になっている。
- 打越鋼太郎脚本という作家性の系譜（後の AI: ソムニウム ファイル系列に接続）を辿るアーカイブ的価値で、後発タイトルのファン層がさかのぼって買う構造が出来ている。
- Aksys Games による北米ローカライズが先行し、海外コミュニティで考察ブログ・配信文化が育っていたため、合本リマスター時点で既に潜在需要が形成済みだった。

## 企画への示唆

- 既読・未読をツリーで可視化する UI を実装すると、ノベル系の「読み返し」が達成感のあるコレクション行為に変質する。
- 1 周目でバッドエンドを必ず踏ませる分岐設計にすると、プレイヤーは自発的に複数ルートを開拓し、平均プレイ時間が押し上がる。
- ノベル本編に物理パズル（脱出/制限時間）を挟み込むと、受動的読解に能動的緊張が継ぎ足され、EXP を secondary に呼び込める。
- 旧世代機（DS/3DS）のシリーズを HD 合本でリマスター収録すると、新規層の「シリーズ全体への入口」になり、後続新作の地盤になる。
- 海外コミュニティで考察文化が育った IP は、リマスター時の宣伝コストを大幅に抑えられる（潜在需要が事前に可視化されているため）。
