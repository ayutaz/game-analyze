# √Film ルートフィルム

## なぜ NAR 主分類なのか

プレイヤーの主動機は「島根県を舞台にした連続殺人ミステリーを **読み解く**」ことにある。実写背景＋ノベル形式という構造上、操作の大半はテキストを読み進め、選択肢を選び、章の進行に従って真相が開示されることに費やされる。concept にある「島根ミステリードラマプロジェクト」を巡る連続殺人と、新人映画監督 / 若手女優のダブル主人公構成は、いずれも物語消費を駆動するための装置で、プレイヤーは「証拠を蓄える」のではなく「劇中ドラマの脚本を追う」感覚で進む。target が逆転裁判 / ダンガンロンパ系ノベルファンに置かれている点も、本作が NAR レイヤを主軸に設計されていることを示している。

中核には「マックスモード」尋問システムがあり、これは EXP（パズル的体験）寄りの要素を含むが、本作では矛盾突きの成否が**章の物語進行とエンディング分岐に直結する設計**になっており、メカニクスは物語を回すためのギアとして従属している。social_axis が null（実質ソロ）なのも、ノベル単体としての完結性を優先したためで、secondary が空であることは「ミステリードラマを読む」一点に絞った設計判断を裏付けている。

## 物語の駆動構造

ダブル主人公制と実写背景という 2 つの仕掛けで物語消費を駆動している。前作『√Letter』の精神的続編として、舞台を島根県に固定し、ロケーションそのものを物語の登場人物に格上げしている点が構造上のポイントである。

| 駆動要素 | 役割 | プレイヤーへの効果 |
|---|---|---|
| ダブル主人公（監督 / 女優） | 章ごとに視点切替 | 同じ事件を別レイヤで再解釈させる |
| 実写背景（島根ロケ） | 場所の固有名詞化 | 聖地巡礼動機と没入の二重化 |
| マックスモード尋問 | 矛盾突きで章を進める | 受動読書を能動推理に変換 |
| 「島根ミステリードラマプロジェクト」入れ子構造 | 劇中ドラマと現実の重ね合わせ | メタ構造でラストの説得力を担保 |
| 連続殺人の章割 | 1 章 1 被害者の縦軸 | 次章クリフハンガーで離脱を抑える |

選択肢分岐ではなく **章単位の矛盾突きが進行ロック**になっているため、プレイヤーは「読み終える」のではなく「解き終える」感覚を持つ。これがビジュアルノベル単体としての完走率を押し上げている。

## バズ要因 / 人気の根拠

- 前作『√Letter ルートレター』が問題作扱いを受けたあとの「精神的続編」として位置付けられ、**前作との対比**そのものが話題のフックとして機能した。
- ファミ通クロスレビューで 8.0 を獲得し、Steam では「やや好評」レンジに着地。前作からの**評価リカバリ**が口コミ上のストーリーになった。
- 島根県を実写背景で再現するというロケ手法が、ノベルファン層と聖地巡礼層の**二重マーケットに刺さった**。
- Switch / PS4 / PC のマルチプラットフォーム同時展開と、海外パブリッシャー PQube の関与により、国内ノベルとしては比較的広い**英語圏リーチ**を確保した。

## 企画への示唆

- **前作の失敗を物語的に引き受ける**と、続編が単独タイトル以上の文脈を持つ（評価リカバリそのものがバズ要素になる）。
- **舞台を実在ロケに固定し実写背景化**すると、ノベル本体のリーチに観光 / 聖地巡礼層が乗ってくる。
- **ダブル主人公で同じ事件を別レイヤから見せる**と、章後半の伏線回収が「読む快感」から「再解釈する快感」にスイッチする。
- **矛盾突きを章進行のロックに据える**と、ノベルの離脱率を能動的操作で下げられる（純読書より完走率が上がる）。
- **小規模 IP でもマルチプラットフォーム＋海外パブリッシャー提携**で展開すると、国内ノベルが英語圏ノベル市場の在庫に紛れ込み、長期に薄く売れる構造を作れる。
