# 祝姫 -祀- (Iwaihime: Matsuri)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「竜騎士07が書く呪いと祟りの物語を読み解く」ことにある。煤田町という閉鎖的な土地、日本人形を抱える黒髪十重、武術道場の家に生まれた煤原進という配役自体が、ひぐらし／うみねこと同じく「異物として町に入った主人公が連続怪事の構造を観測する」フォーマットを踏襲しており、ゲームメカニクスではなくテキストとフラグ分岐で進行する純然たるビジュアルノベルとして設計されている。

concept にある「現実と悪夢が交錯する運命に飲み込まれていく」という構造は、選択肢による分岐より、章ごとに視点と時間軸を切り替えて真相に近づかせる竜騎士07固有のレイヤード・ナラティブを念頭に置いている。target も「20-40 代のノベルゲーマー、サイコ／和風ホラー系ファン」と明示されており、報酬獲得や体験そのものではなく**物語の謎を解き、後味の苦さを味わうこと**が来訪動機。social_axis が null である点も、これが単機・読書的な体験であることを裏付ける。secondary も空で、ハイブリッド要素を持たない NAR 純粋型。

## 物語の駆動構造

| 要素 | 本作での実装 | 物語駆動への寄与 |
|---|---|---|
| 章立て | 既存章 + PS 版追加章「結姫編」 | 真相の追加レイヤーを移植版で開示 |
| 視点 | 主人公・煤原進中心、章により切替 | ヒロインごとの「呪い」の相を多角化 |
| 時間軸 | 現実と悪夢の交錯 | プレイヤーに「どちらが本物か」を保留させ続ける |
| ヒロイン | 黒髪十重（日本人形）ほか複数 | 各人物が固有の祟り/秘密を抱える分岐核 |
| トポス | 煤田町＋武術道場の家系 | 土地神／血族の因縁を伏線化 |

PC 版（DMM.com 2016）の段階で完結していた物語に対し、PS4/Vita 版「-祀-」が「結姫編」を加えた**完全版として再パッケージング**された点が構造上の特徴。竜騎士07作品では「真相編／解答編」を別パッケージで出すことで読者を二段階で囲い込む手法が一貫しており、本作もそのリリース戦略に乗っている。

## バズ要因 / 人気の根拠

- **ひぐらし／うみねこブランドの引力**: 竜騎士07の名義で出る和風ホラーノベル、というだけで既存読者層が反応する。新規 IP でも事前期待が形成されやすい。
- **完全版モデルでの再販**: PC オリジナル（2016）と PS 移植「-祀-」（2017）で**新章を後出し**することで、コアファンの二度買いを誘発し、移植時に再びメディア露出を取った。
- **日本一ソフトウェアの流通力**: パッケージ販売は同社が担い、ニッチ層向けノベルを店頭・通販に乗せる経路を確保。海外は MangaGamer が英語版を扱い、海外コアファンにも到達。
- **和風ホラーというジャンル文脈**: 日本人形・道場・閉鎖集落といったモチーフは、ひぐらし以降のサイコホラー需要と接続しやすく、配信・実況での画面映えも担保した。

## 企画への示唆

- **作家ブランドを軸に新 IP を立てる**と、ジャンル単独で売るより事前期待が桁違いに大きくなる（ただし作家性への忠実さが要求される）。
- **オリジナル版を 1〜2 年後に新章付きで再パッケージ**すると、コアファンの二重購入と新規層への再露出を同時に取れる。
- **章ごとの視点切替＋時間軸の交錯**を採用すると、選択肢分岐に頼らずにプレイヤーに「真相の保留」を強制でき、テキスト量で物語密度を確保できる。
- **モチーフのドメスティック化（和風・閉鎖集落・血族）**は海外ローカライズ時に「Japanese horror」として独自カテゴリ訴求でき、ニッチ海外パブリッシャー経由で確実に売れる。
- **武術道場の家系という主人公属性**のように、ホラーの被害者ではなく「対処能力を持つが解明はできない」配役にすると、読者の没入と無力感を両立させやすい。

## 比較で見るべきタイトル

- **ひぐらしのなく頃に / うみねこのなく頃に** — 同作者の構造的祖型。閉鎖集落＋連続怪事＋多視点。
- **Higanbana no Saku Yoru ni / Rose Guns Days** — 竜騎士07の中規模 IP、章別パッケージ販売のモデル。
- **沙耶の唄 / 殻ノ少女** — 和風／日本特有のサイコホラー VN、海外ローカライズ経路の比較対象。
- **流行り神 / Death Mark** — 和風オカルト ADV のコンソール流通モデル、日本一ソフトウェア系の比較。
