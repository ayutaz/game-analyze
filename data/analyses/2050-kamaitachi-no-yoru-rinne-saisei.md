# かまいたちの夜 輪廻彩声

## なぜ NAR 主分類なのか

プレイヤーの主動機は「雪山ペンションで起きる連続殺人の真相を読み進めて知ること」であり、操作対象は思考と選択肢のみ。サウンドノベルというジャンル自体がテキストと BGM による物語消費に最適化されており、`social_axis` が null なのは想定通り（基本ソロでの黙読体験）。`concept` にある「読み進めて選択する」「新規アートで令和の表現で再構築」も、新規キャラ作画・新キャストといった物語の輪郭を強化する更新であって、ゲームメカニクス側の発明ではない。`target` の「文字と BGM による没入型ホラー」も読書体験を志向する層を明示しており、NAR 以外の主分類は当てはまらない。`secondary` は空で、収集や対戦の副動機はほぼ無視できる。

## 物語の駆動構造

原作 1994 年版を基幹に、追加分岐と書き下ろしシナリオを重ねる「リメイク + 増築」型。読者を引っ張るレイヤーが多層化している。

| レイヤー | 機能 | 本作での実装 |
|---|---|---|
| 主軸シナリオ | 殺人ミステリーを解く本筋 | SFC 版原作シナリオ（ペンションのオーナー殺害から派生する分岐） |
| 追加シナリオ | 既プレイヤーの再来店動機 | 竜騎士07による書き下ろし新章 |
| 表現更新 | 新規層への入口 | シルエット表現を廃し有葉の新規イラスト＋新キャスト |
| 分岐の網 | 反復プレイの動機 | 選択肢から派生する複数エンディング・ピンクのしおりによる踏破管理 |

シルエット表現の廃止は感情論ではなく「キャラクター個体識別の解像度を上げ、犯人特定の手掛かりとして顔の差異を機能させる」設計判断とも読める。竜騎士07の追加章は『ひぐらしのなく頃に』ファンを取り込みつつ、SFC 版だけでは閉じていた世界線を増やしてリプレイ価値を補強する役割。

## バズ要因 / 人気の根拠

- 1994 年 SFC オリジナル版がサウンドノベルというジャンル自体を立ち上げた歴史的タイトルで、シリーズ累計 100 万本超とされる長期ブランド資産を持つ
- リメイクに『ひぐらしのなく頃に』竜騎士07を起用し、原作世代（30-50 代）と「ひぐらし」経由の新本格 / 鳴き声系ホラー読者層を同時に呼び込む二段構え
- 美少女ゲーム / ライトノベル系で広く知られる有葉を新規絵師に据え、シルエット時代を知らない層にも視覚的な入口を用意
- PS Vita / PC (Steam) / Switch / PS4 のマルチ展開で、携帯機・据置・PC のいずれでも「読む」体験に最適化された環境を選べる

## 企画への示唆

- 古典 IP をリメイクする場合、原作シナリオを温存しつつ「同ジャンルで別 IP の人気作家」を増築側に起用すると、両 IP のファンベースを足し算できる
- ビジュアル表現の様式（シルエット等）を更新すると、新規層獲得と同時に推理ゲームならゲームメカニクス側（手掛かりの識別性）も改善できる
- 操作技能を要求しない物語ジャンルでは、Vita / Switch / PC マルチ展開が「通勤読書」「就寝前読書」など読書文脈と直接競合し、可処分時間を奪える
- 分岐エンディング × 既読管理 UI（ピンクのしおり等）を残すと、リメイクでも反復プレイが報酬化し、SNS でのエンディング数比較が自然発生する
- 「読書層」をターゲットに据えるなら、target を「アクション操作が苦手で BGM による没入を求める層」と明文化することで、レビュー・実況の評価軸も期待値とずれにくい
