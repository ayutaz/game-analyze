# 東京クロノス・ゴーストハンター / Tokyo Twilight Ghost Hunters

## なぜ NAR 主分類なのか

プレイヤーの主動機は「現代東京を舞台にしたオカルト怪異譚を**読み解く**」ことにある。依頼者の証言、現場調査、登場人物との会話、霊との因縁 — これらをビジュアルノベル形式で消費していくサイクルが本作の太い背骨であり、戦術パートは「読み終えた章を締める儀式」として機能している。

concept にある通り、構造は ADV パート（調査・会話）と TRPG パート（戦闘）の交互ループだが、各エピソードの導入・依頼解決・後日談はすべて文章で語られる。target も「国産 ADV / 伝奇オカルトもの（女神転生・東京魔人學園 系）を好む層」と明示されており、戦闘システム単独で人を呼べる作品ではなく、伝奇 ADV の系譜として企画されたことが読み取れる。social_axis は null（シングルプレイ完結）。

secondary に EXP が入っているのは、後述する戦術パートが「読むだけ」では完結しない**変則的なメカニクス体験**を要求するためで、純粋な NAR タイトル（Steins;Gate 等）と一線を画す副軸として効いている。

## 物語の駆動構造

伝奇 ADV の駆動装置を分解すると次のようになる。

| 構造要素 | 本作での実装 | 物語推進への寄与 |
|---|---|---|
| 章立て | 依頼ごとのエピソード集積型 | 短編連作で離脱コストを下げる |
| 場所と時代 | 現代東京の高校＋繁華街 | 都市伝説のリアリティ補強 |
| 主人公の立場 | 学生兼「幽霊退治業」 | 日常⇔非日常の往復を正当化 |
| 選択肢 UI | 五感（見る／聞く／嗅ぐ等）と感情のマトリクスから選ぶ独自系 | 選択そのものをロールプレイ化 |
| 戦闘の役割 | 章のクライマックスでの霊との対峙 | 読書のリズムにアクセントを置く |

特に選択肢を「感情 × 五感」のグリッドから選ばせる UI は、テキスト ADV における「Yes / No 二択」を脱構築する設計で、伝奇ものの曖昧な人間関係に幅を与えている。

## バズ要因 / 人気の根拠

popularity に書かれた「ニッチな題材 × 独特の戦術システムが海外レビューで話題」を分解する。

- 現代東京×学園オカルト×幽霊退治という題材の組み合わせが、女神転生 / 東京魔人學園 系統の系譜上にあり、海外伝奇 ADV ファンに刺さった
- グリッド戦闘が「同時行動・予測読み」型で、ターン制とリアルタイムの中間という珍しい設計が話題化
- 北米 Aksys、欧州 NIS America の二社からローカライズされ、PS3 / PS Vita 両機で展開されたためコア層に届いた
- 売上の具体数値は非公開だが、Steam 版『Daybreak Special Gigs』が後発でリリースされ一定の継続需要があった

## 企画への示唆

- 伝奇 ADV にニッチな戦闘システムを同居させると、レビュー文化圏で「珍しい設計」として記事化されやすい
- 選択肢 UI を独自グリッド（五感×感情等）にすると、二択 ADV の物量勝負から差別化できる
- 章立てを「依頼ごとの短編連作」にすると、伝奇ものでも離脱コストを下げられ、長尺プレイの心理障壁が小さくなる
- 国内 ADV ファン層と海外コア ADV 層は重なるため、Aksys / NIS America 等の小規模ローカライザーと組めば北米欧州での話題化チャネルが確保できる
- ジャンルを「ビジュアルノベル / タクティカル RPG」と二枚看板で打ち出すと、どちらか単体では手を出さなかった層に二重露出できる
