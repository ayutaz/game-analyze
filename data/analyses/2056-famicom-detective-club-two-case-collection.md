# ファミコン探偵倶楽部 消えた後継者・うしろに立つ少女

## なぜ NAR 主分類なのか

プレイヤーの主動機は「昭和的湿度のミステリーを読み進める」こと。ディスクシステム期の 2 本を 1 本に束ね、フルボイスとアニメ演出を載せた本作は、コマンド選択式 ADV というジャンルの定義通り「テキストを読み、選択肢を試し、進展を回収する」体験が遊びの中心になる。concept が「視覚・聴覚演出を全面刷新」と明言する通り、リメイクで投資された予算は **物語消費の解像度** に集中しており、戦闘や成長といった他軸はそもそも設計されていない。

target も「1980 年代後半のオリジナル経験者」「MAGES. の Steins;Gate 系 ADV ファン」「フルボイスで気軽に名作探偵モノを追体験したい層」と三層すべてが物語追体験動機で揃っている。social_axis が null なのは他者との関与軸が成立しないソロ ADV だからで、secondary が空なのは「収集や育成の副動機もない、純粋なテキスト読破型」であることを示している。

## 物語の駆動構造

リメイク 2 本パックの構造を分解すると、駆動装置は「コマンド総当たりの抵抗感を、演出の濃度で補償する」設計に集約される。

| 駆動レイヤ | 原作（1988/89） | リメイク（2021） |
|---|---|---|
| テキスト供給 | 静止画＋効果音＋BGM | フルボイス＋立ち絵アニメ＋差分多数 |
| 探索インタフェース | コマンド総当たり | コマンド総当たり（基本維持） |
| シナリオ尺 | 各 5–8 時間 | 演出延伸で各 8–12 時間 |
| 章構造 | ディスク両面の物理区切り | チャプターセレクト＋進捗保持 |
| キャラクター関係性 | テキスト主導 | 声優演技＋表情差分で前景化 |

ポイントは **コマンド総当たりという 80 年代的 UI を温存したまま** 演出層だけを更新した点にある。「もう一度聞く」「考える」を繰り返すリズムが昭和の湿度を維持し、その上にフルボイスが乗ることで「読まされている」のではなく「事情聴取に立ち会っている」感覚が立ち上がる。2 本同梱という形態は、`消えた後継者` の血縁ミステリーと `うしろに立つ少女` のホラー寄りプロットを地続きに体験させる装置として機能している。

## バズ要因 / 人気の根拠

popularity の「5 万本の倍以上売れている」発言を起点に分解する。

- Collector's Edition 初週で 12,269 本（物理のみ）。DL 版を含めた累計は 10 万本超と推定され、コマンド選択式 ADV としては国内で例外的な水準。
- 開発元 MAGES. の Steins;Gate 系 ADV ファン層と、任天堂 IP のレトロゲーマー層という **二つの非重複ファンダム** を同時に動員したことが話題化の核になった。
- 4Gamer など専門メディアでの取材露出に加え、志倉千代丸の SNS 発信が「想定を上回った」というナラティブを増幅させた。
- 2021 年は Switch のテキスト ADV 市場が拡張期にあり（同年前後に逆転裁判 123 復活、AI:ソムニウム ファイル続編などが並走）、ジャンル全体の追い風を受けた。

## 企画への示唆

- **UI を温存し演出層だけ更新すると**、原作経験者の郷愁を壊さずに新規層を入れられる（コマンド総当たりを廃止しなかった判断）。
- **2 本同梱パッケージにすると**、単体ではリスクが高い旧 IP リメイクで「両方遊んで欲しい」というメッセージを 1 つの購入決定に圧縮できる。
- **眠っている自社 IP を外部スタジオに開発委託すると**、社内リソースを割かずに IP の延命と再評価が同時に走る（任天堂×MAGES. の構造）。
- **フルボイス化の投資対象を主要キャラに集中させると**、テキスト ADV の「読み」のリズムを壊さずに体験密度を上げられる。
- **発売後に開発者本人が販売実績を発信すると**、ニッチジャンルでも「想定超え」のナラティブが二次的なバズを生む。

## 比較で見るべきタイトル

- `逆転裁判 123 成歩堂セレクション` — 旧 IP 複数本同梱＋演出更新という同型のリメイク戦略。
- `AI: ソムニウム ファイル` — 現行 Switch ADV の演出基準。本作のフルボイス化が向かった先。
- `かまいたちの夜 輪廻彩声` — 昭和〜平成 ADV の UI をどこまで現代化するかという別解。
- `ダンガンロンパ 1・2 Reload` — 旧作 2 本同梱で新規参入を促す PSV 期の先行事例。
