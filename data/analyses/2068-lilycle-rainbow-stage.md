# Lilycle Rainbow Stage!!!

## なぜ NAR 主分類なのか

プレイヤーの主動機は「15 人の女子高生のあいだに編まれた 7 組の百合カップルを、関係性が固まっていく時系列で**読む**」ことに尽きる。ゲームメカニクス（戦闘・収集・育成）はなく、JSON の concept が示す通り「カレンダー型のイベント選択式」で各カップルの場面が解放されていく構造であり、選択は分岐ではなく**閲覧順の組み替え**として機能する。target も「キャラ造形と関係性の機微を時間をかけて味わいたい層」と明示されており、プレイ動機が読書体験そのものに収束する点で NAR の典型である。

social_axis が null なのは EXP 系の他者軸（対戦/協力/非対称）を持たないためで、secondary も空。副次的な動機としては「みわべさくらの絵柄」「豪華声優陣（木村珠莉/MAO/内田真礼）」によるキャラ消費が走るが、いずれも物語駆動の補強要素であり主分類を揺さぶる強度はない。

## 物語の駆動構造

並行する 7 組のカップル線をカレンダー UI でスケジューリングし、メイン三角関係を骨子として群像を畳み込む。読者は「どのカップルから読むか」を時間軸に沿って選び、関係性の重なりを自分で編集する。

| 構造要素 | 役割 | 効果 |
|---|---|---|
| カレンダー型イベント解放 | 章立ての代替 | プレイヤーが読み順を編成できる |
| 7 組の百合カップル | 並列サブプロット | 一周あたりの関係性密度を高める |
| メイン三角関係 | 物語の重心 | 群像が拡散しないアンカー |
| 15 人のオールスター構成 | キャストの広さ | 推しカップルが見つかりやすい |
| みわべさくらの絵柄 + フルボイス | 演出層 | 関係性の機微を視聴覚で補強 |

群像 VN にありがちな「誰の話を追っているか分からなくなる」問題を、三角関係の中心線とカレンダー UI で抑え、各カップルの解像度を保ったまま並走させている。

## バズ要因 / 人気の根拠

popularity に書かれた「オールスター構成 × 豪華声優陣 × 英語圏 Steam 91% Positive」を起点に分解すると以下になる。

- **15 人 × 7 組という物量感**：単一カップル特化型が多い百合 VN ジャンルにおいて、群像オールスターを正面から組んだ希少な構成。
- **声優キャスト**：木村珠莉 / MAO / 内田真礼ら知名度の高い面々を投入し、声優ファン層を百合 VN 文脈に橋渡し。
- **海外展開のヒット**：2016 年和製インディー VN が MangaGamer 経由で英語化され、Steam で 91% Positive を獲得。日本国内よりも海外百合コミュニティでのリファレンス化が大きい。
- **みわべさくらの絵柄**：百合・きらら系列で支持の厚い作家性を、声付き VN に乗せたことで「読む百合」と「眺める百合」を両立。

## 企画への示唆

- **群像百合は「中心線」を一本通すと拡散を防げる** — オールスター構成でもメイン三角関係を骨子に置くと、各サブプロットが装飾として機能する。
- **カレンダー UI を章立ての代替に使うと、群像の並列読みが成立する** — 分岐ではなく解放順をプレイヤーに委ねる設計は、群像物語と相性が良い。
- **インディー百合 VN は英語圏ローカライズで二次寿命が伸びる** — MangaGamer / JAST 経由の英語版が、原語版より長くコミュニティ参照される事例。
- **絵柄ブランド × 声優ブランドを掛け合わせると、ジャンル外の客を引き込める** — みわべさくらの百合ファン層と、メジャー声優の声優ファン層が交差点を作る。
- **secondary を空にできる NAR は強い** — 報酬や育成を持ち込まず、関係性の読書体験だけで完結させる潔さが、ジャンル代表作としての引用されやすさにつながる。

## 比較で見るべきタイトル

- **FLOWERS シリーズ**（春・夏・秋・冬）：百合 VN の四季構成・章立て型、より文学寄り。
- **その花びらにくちづけを**：単一カップル深掘り型、Lilycle の群像とは対極。
- **Kindred Spirits on the Roof**：海外で広く受容されたもう一つの百合 VN、群像 × 学園という近接ポジション。
- **アマガミ**：カレンダー / スケジュール型 ADV のメカニクス祖型、攻略対象並列構造の参照点。
