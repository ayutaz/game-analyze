# コクーン（Cocoon, 2023）

## なぜ EXP 主分類なのか

プレイヤーの主動機は「世界を運んで世界に入る、その入れ子操作のアハを体験する」こと。物語テキストはほぼ無く、収集要素やレベルアップも存在しないため、駆動するのはひたすらメカニクスの発見と操作体験そのものである。concept にある「オーブの中に世界が入れ子状に格納された幾何学的なパズル」は、報酬でも物語でもなく **触ったときに何が起きるかを理解する過程** が遊びの本体であることを示している。

social_axis は「ソロ」。target が示すとおり、無言・幾何学・アハ体験型パズル（INSIDE / The Witness / Monument Valley）の系譜にあり、他者との関わりよりも自分の脳内で世界モデルを組み立てる体験を重視する。secondary が空なのも、収集／物語／競争などの副次的動機を意図的にそぎ落とした構造であることと整合している。

## 体験の核 / 設計の発明

中核メカニクスは「球を背負う」という一手で世界の入れ子関係そのものを操作対象にしたこと。前世代のパズルアドベンチャーが「マップ内でギミックを解く」のに対し、コクーンは **マップそのものをアイテム化** することで、解くべき空間と道具の境界を消した。

| 設計要素 | コクーンの実装 | 前世代との差分 |
|---|---|---|
| 道具 | 世界を内包したオーブ | 通常はマップ内の鍵・スイッチ |
| 移動 | 異なる次元世界を背負って行き来 | マップ間遷移は一方向／ロード |
| ヒント | UI なし・テキストなし | 矢印・ログ・チュートリアル |
| 難度 | 確実に解ける曲線設計 | 詰み・総当たり許容 |
| 入力数 | 実質 1 ボタン（インタラクト） | 複合キー前提 |

入力が 1 ボタンに切り詰められているため、プレイヤーの認知負荷はすべて「世界の構造をどう読むか」に集中する。social_axis がソロであることは、この没入を他者の声で割らないための設計判断と読める。

## バズ要因 / 人気の根拠

popularity に挙がるとおり、Jeppe Carlsen 初監督作という作家性と、賞レースでの連続受賞が拡散の起点になった。

- 『LIMBO』『INSIDE』のリードゲームプレイデザイナー Jeppe Carlsen の独立後初監督作という来歴自体がプレス話題化。Playdead 系インディーの「次の一手」として注視された。
- TGA 2023 ベストデビューインディー、D.I.C.E. 2024 最優秀インディー受賞。授賞ノミネートがダイレクトに Steam ウィッシュリストを押し上げた。
- Steam 所有 200K–500K、全機種で 50 万本超推定。Annapurna 配給によるマルチプラットフォーム（PC / Switch / PS5 / PS4 / Xbox / iOS）展開でリーチを最大化。
- 配信・実況との相性：プレイ時間 5–7 時間、ネタバレ耐性が低い代わりに「初見の驚き」が映え、ストリーマー初見配信で短期的に視聴が集中するタイプ。

## 企画への示唆

- **マップを道具化すると、世界モデルそのものが解の対象になる** — 鍵やアイテムではなく空間を持ち運べる設計にすれば、「解く対象」と「解く手段」の境界が消える。
- **入力を 1 ボタンに絞ると、認知の全帯域をパズルに割ける** — UI とキーマップの学習コストをゼロに近づけることが、メカニクス理解の解像度を上げる。
- **言語ゼロ・テキストゼロで設計するとローカライズ費用が消える** — 無言設計はアート選好であると同時に、グローバル同時配信を低コスト化する経営判断でもある。
- **「詰まらない」難度曲線を引くと、ライト層がアートゲームに踏み込める** — The Witness 系の「総当たり許容」より、コクーンの「確実に解ける」設計のほうが裾野が広い。
- **作家来歴を一行で説明できると、初動の PR が走る** — Carlsen / Playdead という導線は、レビュー媒体に書いてもらう前提のフックとして機能した。

## 比較で見るべきタイトル

- INSIDE（Playdead, 2016） — Carlsen 前作。無言・横スクロール・暗喩の系譜。
- The Witness（Thekla, 2016） — 無言・幾何学・パズル単一動機。難度設計は対極（総当たり許容）。
- Monument Valley（ustwo, 2014） — 幾何学的錯視を道具化した先行作。
- Tunic（Andrew Shouldice, 2022） — 無言・記号体系の解読を遊びの核に据えた近作。
- Patrick's Parabox（2022） — 「箱の中に箱」の入れ子をパズル化した同時期の論理パズル。
