# ランス10 / Rance X: Showdown

## なぜ NAR 主分類なのか

プレイヤーの主動機は「29 年間追ってきたシリーズの伏線と関係性の総決算を**読み届ける**こと」である。`concept` が「シリーズ全伏線を回収する集大成」、`target` が「物語の結末を見届けたい既存ユーザー」と明示している通り、本作の購入動機は新規メカニクスの体験ではなく、長期 IP のクロージャーに置かれている。200 人超のプレイアブル / 登場キャラと巨大スクリプトという物量は、戦闘ゲームの素材としては過剰で、物語消費装置として読むと整合する。

副次的に `EXP`（デッキ構築型カードバトル + 国盗り SLG という新規ループ）と `REW`（旧作キャラの収集・編成・解放）が効いているが、これらは**結末まで歩を進めるためのペース調整装置**であり、メカニクス単体で本作を語ることはできない。`social_axis` が null なのも、終局を一人で見届けるソロ消費型 NAR としての性格を裏付けている。

## 物語の駆動構造

| レイヤー | 仕掛け | 機能 |
|---|---|---|
| マクロ | 魔人軍 vs 人類連合軍の Great War | 旧作で散らした勢力/種族を一枚マップに集約し「全部出てくる」期待を作る |
| 中位 | 国盗り SLG 進行 + イベント解放 | 章立てを地理で代替し、攻略ルートが選好で分岐 |
| ミクロ | デッキ構築型カードバトル | キャラ単位の参戦理由をデッキ枠で可視化し、編成 = キャスティング化 |
| メタ | 200 人超の登場キャラ + 過去作呼び戻し | 既存ファンの記憶資産を換金、未読部分は「次に読む」フックに転化 |

物語の駆動は「**広い盤面に旧キャラを段階解放し、編成枠の埋まりがそのまま読了率の可視化になる**」点に集約される。カードバトル単体は珍しくないが、IP の集大成という文脈に乗せると「カード = キャラ = 既読フラグ」の三位一体になり、戦闘画面が物語進捗ダッシュボードを兼ねる。

## バズ要因 / 人気の根拠

- 29 年（1989 年『Rance -光をもとめて-』〜2018 年）続いた長期シリーズの**完結宣言**そのものがイベント性を持ち、発売前後にファンコミュニティが沸騰した。
- プレイアブル含む登場キャラ 200 人超・巨大スクリプト量で「ゲーム史最大級」と評され、容量・物量がそのまま語られ広告化した。
- 国内アダルト PC ゲームというニッチ市場でありながら、海外英語圏ファン翻訳コミュニティ（alicesoft wiki 等）が並走し、IP の越境寿命を延ばしてきた累積効果。
- 売上数値は非公開（`sales_jp: "—"` / `sales_world: "unknown"`）だが、AliceSoft 自社販売 + 同人 / 専門店流通という特殊チャネル特性により、定量よりも**完結というイベント密度**で語られた。

## 企画への示唆

- **長期シリーズの最終作はメカニクスを 1 段足すと、物語消費に進行可視性が乗る**：単純な ADV にせず国盗り + デッキ構築を被せたことで、既読量が UI 上で見える化された。
- **キャラ数 = スクリプト量を物量で誇示する**と、コアファン向けの広告が自走する：「200 人」「ゲーム史最大級」という形容詞は、メディアではなくファンが運ぶ。
- **編成枠を読了フラグに転用する**と、戦闘準備画面が物語ダッシュボードを兼ねて滞在時間が伸びる。
- **新規動線を切り捨てて既存ユーザー特化に振る**判断は、IP 寿命の最終段では合理的：`target` を「シリーズを長年追ってきた 30-50 代」に絞り込み、新規向けチュートリアルに資源を割かない選択。
- **完結という物語イベントそのものを商品化する**と、定量売上が非公開でも「29 年の集大成」というナラティブが PR コストを代替する。

## 比較で見るべきタイトル

- **戦国ランス（2006 / AliceSoft）**：本作の国盗り SLG 部分のプロトタイプ。完結作で再採用された理由が読める。
- **ペルソナ 5 the Royal**：長期 IP の決定版を「総量で押す」構造の参照系。NAR 主分類の隣接事例。
- **Fate/stay night → Fate/hollow ataraxia**：シリーズの後日譚 / 集大成として、既存ファン特化で物量を出す設計の比較対象。
- **Disgaea シリーズの周年作**：キャラ総出演で過去資産を換金する手法の隣接事例。
- **うみねこのなく頃に 散**：長編 VN の完結章という意味で、`target` 設計と物量誇示の構造が近い。
