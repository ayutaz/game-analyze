# 鬼畜王ランス (1996, AliceSoft)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「大陸マップを制圧していく領土征服 SLG を回す」体験そのものであり、戦う／攻めるの動詞が主動詞になる。concept にある通り、レアザス王ランスが大陸全土を陣営ごとに蹂躙する territory conquest 型ストラテジーが核で、ターンごとの軍備・侵攻・外交ループが本作のプレイ時間の大半を占める。target が「戦国ランス / 大悪司 / 大番長系の領土征服 SLG を遊ぶ層」と並列されていることからも、本作の主軸が戦略シミュレーションの体験側にあることが裏付けられる。

ただし secondary に NAR が入っているのは、各国の女性キャラ個別エピソードや、レアザス陣営の家臣関係・分岐エンディングといった物語要素が、征服マップの上に乗っている二層構造だからである。social_axis は「ソロ」で、AI 陣営との単独プレイ。

## 体験の核 / 設計の発明

中核は「マップ全体を国家単位で塗り替える戦略レイヤー」と「個別キャラクターの加入・イベント分岐」を同居させた点にある。後の戦国ランス・大悪司・大番長へと継承される AliceSoft 領土制圧 SLG の原典がここで成立した。

| 層 | 役割 | 後継への継承 |
|---|---|---|
| 領土マップ | 国家ごとのターン制侵攻、軍団編成、外交 | 戦国ランス（国別 → 大名別へ拡張） |
| 個別キャラ | 加入条件付き仲間、専用イベント、ルート分岐 | 大悪司の組員管理、大番長の派閥制 |
| ランス本人 | 全陣営の中心、女性キャラ攻略を兼ねる主人公 | シリーズ通底のキャラ設計 |
| ED 分岐 | 制圧範囲・条件達成で複数 ED に分岐 | 戦国ランス 5 ルート構造の祖型 |

Windows 95 / SYSTEM 35 への対応も設計の節目で、PC-98 期のランス 1〜6 から Windows 期シリーズへの橋渡しになった。

## バズ要因 / 人気の根拠

- ランスシリーズ 7 作目で、戦国ランス・ランス VI 以降に通底する領土征服システムの原典として後年にも参照されている
- 2006 年に AliceSoft 自ら「配布フリー宣言」を出してフリーウェア化、商業作品としては異例の自由配布で長期流通の母体になった
- 2014 年に Arunaru による英訳パッチが公開され、海外の和製エロゲコミュニティ（ULMF / Fuwanovel 系）でカルト的に拡散
- archive.org / kichikuou.github.io 等で配布ミラー・攻略 wiki が継続維持され、リリース後 30 年経っても新規プレイヤーの導線が残っている

## 企画への示唆

- 「マップ戦略 SLG × 個別キャラ攻略」を一作で重ねると、戦略リプレイ性とキャラ消費の両方をリテンションに使える
- ナンバリングシリーズの中盤に「次世代システムの原典」を据えると、後継作の販売文脈で過去作が継続的に再評価される
- 商業作品の途中フリーウェア化は、IP 露出と新規流入を長尺で稼ぐ手段になりうる（ただし続編販売とのカニバリは設計が必要）
- 主人公（ランス）を「全陣営の中心」に置く設計は、女性キャラ多数とのイベント分岐を単一動線にまとめるので、コンテンツ量に対して構造をシンプルに保てる
- 領土征服 SLG の ED 分岐を「制圧範囲 + 条件達成」で組むと、周回プレイの目的が自然に分散する

## 比較で見るべきタイトル

- 戦国ランス（2006, AliceSoft）— 領土征服システムの正統後継、本作の発明を大名制で再構築
- 大悪司 / 大番長（AliceSoft）— 同系領土 SLG、舞台を現代ヤクザ・学園に置換
- ランス 10（2018, AliceSoft）— シリーズ完結作、本作の構造的子孫としての到達点
- Sengoku Rance fan community — 本作の英訳パッチ文化と地続きの海外受容圏
