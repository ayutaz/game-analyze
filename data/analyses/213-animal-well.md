## なぜ EXP 主分類なのか

プレイヤーの主動機は「観察し、試し、解く」体験そのものにある。`Animal Well` は戦闘や成長曲線でリテンションを設計するタイプではなく、薄暗い地下迷宮の各画面に仕込まれた挙動・記号・パターンに気づくこと自体を快楽に変換するゲームで、報酬は新しいガジェットではなく「自分の認識の更新」だ。`concept` に明記されている通り、ヨーヨーやフリスビーなど **戦闘ではない 8 個前後の道具** をどう組み合わせ、どこで挙動を再観察するかというループが中核を占める。

`social_axis` は「ソロ」だが、`target` が示すように **外部コミュニティでの謎共有** が裏で強く効いている。1 周目のメインクリアで終わらず、卵 64 個・最深部の暗号・QR コード・ARG 級レイヤーまで続く構造は、個人プレイの内側に閉じた体験でありながら、Discord や YouTube での集合的攻略を二次的に駆動する。secondary に NAR が入っているのは、明示的な物語テキストではなく **環境と記号でほのめかされる神話** をプレイヤー側が再構成する余地を残しているためで、これは体験設計の補助線として機能している。

## 体験の核 / 設計の発明

| レイヤー | 解かれる対象 | 必要な道具 |
|---|---|---|
| 1. 表層メトロイドヴァニア | マップ踏破・8 個の道具入手・エンドクレジット | 基本ガジェットのみ |
| 2. 卵 64 個収集 | 各画面に隠された挙動・タイミング | 道具の応用組合せ |
| 3. 最深部 / 暗号層 | 文字体系・天体記号・色順序 | 観察ノート＋外部共有 |
| 4. ARG 層 | Discord 配布の物理シール・電話番号 | コミュニティ集合知 |

前世代の `Fez` や `La-Mulana` が「ゲーム内の言語そのものを学習させる」発明をしたのに対し、`Animal Well` は **観察対象を環境挙動（動物 AI・水位・光源・残像）に分散** させ、ガジェットを攻撃ではなく観察補助具として再定義した。Steam Deck 含む全プラットフォームに収まる **33MB のビルドサイズ** も設計判断の現れで、リソースを増やすのではなく挙動の再利用密度で深さを稼いでいる。

## バズ要因 / 人気の根拠

- YouTuber **videogamedunkey** が設立した新興パブリッシャー **Bigmode の第 1 弾** という発売前の文脈付与で、インディー単独作にしては異例の事前注目を獲得した
- **33MB** という極端な軽量さと、その中に詰まった多層構造のギャップが SNS で話題化しやすかった（スクリーンショット 1 枚で語れる「軽さ」が見出しになる）
- ローンチ後すぐに卵 64 個 → 最深部 → ARG レイヤーへと **コミュニティの集合的攻略** が連鎖し、配信・解説動画・Discord での暗号解読が並行発生した
- **2025 DICE Awards Outstanding Game Direction** 受賞で批評的評価が決定し、`sales_world` は 100 万本+ に到達

## 企画への示唆

- **戦闘を外しても物理挙動が密ならメトロイドヴァニアは成立する** → 攻撃ボタンを観察用ガジェットに置き換えると、リプレイ価値はマップではなく挙動側に積み上がる
- **ビルドサイズを意図的に小さく作ると、それ自体が広報資産になる** → 「33MB」のような単一数字は SNS 共有の摩擦をゼロにする
- **1 周クリアの奥に隠しレイヤーを 2 段重ねると、ソロゲームに集合的攻略の社交性が後付けされる** → ARG 級の暗号は実プレイ時間以上の話題寿命を生む
- **パブリッシャー側の文脈（誰が出したか）を発売前に語らせる** → 個人開発者単独より、語り部を持つレーベルから出す方が初動コミュニティが立ち上がりやすい
- **道具を 8 個前後に絞り、各道具の用途数を増やす** → アイテム数で深さを稼ぐより、組み合わせの再解釈で深さを稼ぐ方が個人開発でスケールする

## 比較で見るべきタイトル

- `Fez` (2012) — 2D 画面に隠された言語体系をプレイヤーに学習させる前世代の到達点
- `La-Mulana` / `La-Mulana 2` — 古代遺跡の石碑解読を中核に据えた日本側の系譜
- `Tunic` (2022) — マニュアル風の取扱説明書ページ自体を解読対象にした近作
- `Outer Wilds` (2019) — 戦闘ゼロで観察と仮説検証のみで進むソロ探索の代表例
- `Voices of the Void` / `Inscryption` — 表層クリア後に別レイヤーが立ち上がる多層構造の比較対象
