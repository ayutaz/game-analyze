## なぜ NAR 主分類なのか

プレイヤーの主動機は **「テキストを読んで小惑星帯の漂流者たちの物語を選び取る」** ことに集約される。concept にある「デジタル意識体スリーパーとして乗組員を集めながら星系を渡る」も、target にある「Disco Elysium / Norco など『読むRPG』を好む層」も、いずれも **物語消費を中心に置いた設計** を示している。ダイスロールやリソース管理（コンディション、ストレス、契約期限）は副次的な張力装置で、ロール結果そのものを楽しむというより、結果が分岐させる文章を読むためのトリガーとして配置されている。

副次の EXP は、前作から刷新された「ダイス＋ドリフター（乗組員）運用」の手触り。サイコロを誰の枠に割り当てるかという TTRPG 的な選択の体験は確かに面白いが、それは物語を分岐させる手段であり、戦闘や探索そのものの面白さで完結する EXP 主分類のゲームとは骨格が違う。social_axis は未設定だが、実態としては完全シングルのソロ体験である。

## 物語の駆動構造

続編で物語消費を駆動するために、前作（拠点 Erlin's Eye の定点滞在型）から **「契約（Contract）を渡り歩く移動型」** へ構造を入れ替えている。要素を分解すると以下のようになる。

| 構造要素 | 機能 | 物語上の効果 |
|---|---|---|
| ドリフター（乗組員） | 各キャラがバックストーリーと固有スキルを持つ | 旅の同伴者という関係性が章ごとに深まる |
| 契約 / コントラクト | 期限付きミッションで星系内を移動 | カレンダーと地理が同時に進む、読み切り短編連鎖 |
| ダイスロール | 行動成否を確率的に決定 | 失敗テキストが固有に書き分けられ「読むため」に振る |
| コンディション / ストレス | 体調と精神の二軸メーター | ダイスの質を劣化させ、選択に重さを与える |
| ハンター（追跡者） | スリーパーを追う組織的脅威 | 「逃亡 SF」というジャンルトーンを通奏低音化 |

つまり「読む → 振る → 結果を読む → 体調が削れる → 次の契約地へ」というループが章立てに重ねられ、テキスト量と意思決定が交互に押し寄せる構造になっている。

## バズ要因 / 人気の根拠

- 前作 Citizen Sleeper（2022）でサイバーパンク叙情系インディーRPG として GOTY 候補レベルの評価を獲得しており、続編は **既存ファンベース＋評論家の期待値** を背負ってのリリースだった。
- Game Pass / PS Plus 同時配信により、購入摩擦ゼロでメインストリーム層に届いた。PC 単独だったインディー文芸 RPG にとってのアクセシビリティ閾値が一段下がった。
- プレイヤー数は 2026 年 1 月時点で **250 万人+**（PC/コンソール、Game Pass / PS Plus 経由含む）に到達。
- ダイス + リソース管理という TTRPG 系メカニクスの可視性が高く、配信・実況での「振った瞬間に解説が入る」ナラティブ実況と相性が良い。

## 企画への示唆

- **続編で構造を「定点 → 移動」に切り替えると、同じ世界観のまま全く別の物語密度を作れる。** 前作で出し切ったロケーションの再利用に頼らず、章ごとに新しい客演キャラを連れてこられる。
- **ダイスロールに「失敗専用テキスト」を書くと、失敗が罰ではなくコンテンツになる。** 成功時テキストだけ書く設計だと失敗は損だが、両方書けば「読むために振る」動機が立つ。
- **期限付き契約 + 体調メーターを重ねると、自然に「諦める判断」が選択肢に入る。** プレイヤーに撤退を選ばせると、その判断自体が物語になる。
- **同伴 NPC を「ガチャ的に増やす」のではなく「契約ごとに偶発的に出会う」設計にすると、コレクション圧から自由な物語が書ける。** ドリフターは集めるものではなく、出会うものとして配置されている。
- **サブスク同時配信は「読むゲーム」の最大の追い風になる。** 文芸寄りインディーは購入意思決定の摩擦が高く、Game Pass / PS Plus がそこを溶かす。

## 比較で見るべきタイトル

- **Citizen Sleeper（2022）** — 前作。定点 vs 移動の構造差分が明瞭に読める。
- **Disco Elysium（2019）** — テキスト量とスキルロールを核に据えたナラティブ RPG の参照点。
- **Norco（2022）** — 南部 SF テキスト RPG。叙情とディストピアの距離感が近い。
- **Mask of the Rose（2023）** — Failbetter のカレンダー駆動ナラティブ。期限と関係性のメカニクスが対照的。
- **Sunless Sea（2015）** — 「契約を渡り歩くテキスト海洋 RPG」の祖型。
