# ナインソールズ (Nine Sols)

## なぜ EXP 主分類なのか

このゲームのプレイヤー主動機は「Sekiro 直系のパリィ拍子を、2D で身体に刻み込みながら戦う」ことに集約される。動詞は **戦う／弾く**。`concept` が掲げる「Taopunk」世界観も没入の核ではあるが、プレイ時間の大半は中ボス・ボスのモーションを読み、パリィの間合いを身体化するループに費やされる。`target` が Hollow Knight / Sekiro / Katana Zero を並べているのは、世界観目当てではなく **高難度アクションを技術として攻略する層** を主読者に据えていることを示しており、`social_axis: ソロ` と矛盾しない（パリィ拍子の習得は他者観戦ではなく一人で繰り返すしかない）。

副次の `NAR` は、道教神話 × SF の重層プロットと九つの「Sols」をめぐるキャラクター造形が体験の意味を裏打ちしているという意味で効いている。物語が動機の主柱ならボス前の語りを飛ばす設計にはしない――本作はパリィを通れない限り物語も進まない点で、構造的に EXP 優位である。

## 体験の核 / 設計の発明

中核は **「Sekiro パリィ × メトロイドヴァニア」** の接合。Sekiro の体幹／弾きをそのまま 2D の左右距離管理に落とし、回避は最小限、パリィ／チャージ反撃を主軸に据えている。前世代（Hollow Knight、Salt and Sanctuary）が「回避＋連打」だったのに対し、本作はリズムゲーム的な拍子を強要する。

| 設計要素 | 前世代 2D メトロイドヴァニア | Nine Sols の差分 |
|---|---|---|
| 防御 | ローリング回避が主、ガード補助 | パリィ／弾き返しが必須、回避は補助 |
| ボス比重 | 道中 7：ボス 3 程度 | ボス／中ボスの比重が体感 5：5 まで上昇 |
| 世界観 | 西洋ダーク／和風どちらかに寄る | 道教 × サイバーパンク（Taopunk）の独自合成 |
| アート | ピクセル or ベクター | フル手描きアニメーション（Red Candle の同人 / VN 系譜） |
| 物語接続 | ロア散逸型（読解任意） | ボス前後のカットシーンで関係性を強制提示 |

`social_axis: ソロ` の効果は、難度設計が「他者参考」を許容しないこと――敵モーションの個体差が大きく、攻略動画を見ても自分の指でパリィ拍子を身体化しなければ突破できない構造になっている。

## バズ要因 / 人気の根拠

- Steam で「圧倒的に好評」を維持しつつ、2025 年 5 月時点で全プラットフォーム累計 **80 万本超** に到達。アジア発インディーの中では Red Candle Games（『返校』『還願』）のブランド再利用効果が大きい。
- Sekiro ファンが「2D で同じ拍子を味わえる作品」を渇望していた市場真空を直撃し、配信／実況コミュニティで「ボス突破クリップ」が独立コンテンツとして拡散した。
- 台湾発・道教モチーフ × SF という **欧米にも東アジアにもオリエンタルに映る合成** が、欧米メディアでの紹介角度（"Taopunk"）を提供し、レビューでの引用句として機能した。
- 手描きアニメーションと VN 系譜のキャラクター演出が、アクション勢だけでなく物語勢の二次創作・ファンアートを引き寄せ、コア層外への染み出しを生んだ。

## 企画への示唆

- **既存高難度作品の操作言語を「次元変換」して移植すると、ジャンル真空を狙える**：Sekiro の拍子を 2D に変換した本作のように、3D で確立された操作感を 2D／逆方向へ写すと、既存ファンの飢えを直接満たせる。
- **アートワークの作家性を VN／同人系譜から借りると、アクション層の外に拡散する**：パリィの一拍子だけでは届かない物語層・SNS 層を、手描きアニメと顔つきキャラで拾える。
- **ローカル神話 × 普遍的 SF の合成は、海外メディアに「紹介語」を与える**：「Taopunk」のように一語の造語があると、レビュー本文に引用されやすく、見出し化されやすい。
- **`social_axis: ソロ` でも、ボス突破クリップは観戦コンテンツになる**：マルチを実装せずとも、難度そのものを配信向け素材に転化する設計（チェックポイント直後リトライ、ボス UI の即読み取り性）を意識すると拡散が伸びる。
- **過去作の評判（Red Candle の『返校』『還願』）をジャンル横断で資本化できる**：ホラー AVG から 2D アクションへ完全にジャンル転換しても、開発者ブランドが移植されるなら成立する。

## 比較で見るべきタイトル

- **Sekiro: Shadows Die Twice**（パリィ拍子のオリジン、3D／本作はその 2D 翻訳）
- **Hollow Knight**（2D メトロイドヴァニアの前世代基準、回避中心 → 本作はパリィ中心への転換）
- **ENDER MAGNOLIA / ENDER LILIES**（同時期アジア発 2D アクション、物語比重は本作より NAR 寄り）
- **Katana ZERO**（高速・即死系 2D アクション、`target` に列挙されたリファレンス）
- **Blasphemous シリーズ**（ローカル宗教モチーフ × 2D ソウルライク、本作 Taopunk の対照例）
