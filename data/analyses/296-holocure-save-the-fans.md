# HoloCure - Save the Fans!

## なぜ EXP 主分類なのか

プレイヤーの主動機は「推しキャラを操作してオートアタックの弾幕を組み上げ、20分間生き延びる体験そのもの」である。`Vampire Survivors` 型ローグライトの中核である「短時間で完結する build-craft 体験」が遊びの本体であり、ホロメン 47 人という固有スキルセットの差分がランごとの体験を駆動している。

`concept` が示す通り本作は VTuber 二次創作の皮を被った操作系ローグライトであり、`target` も「短時間ローグライトを好むカジュアル〜コアゲーマー」を含む。`social_axis: ソロ` の通りリアルタイム協力やランキング戦は中核ではなく、ラン単位で武器シナジーを発見する一人称の体験設計に振り切っている。

`secondary: REW` が効くのは、ラン外メタプログレッションの部分である。ホロコイン（ラン報酬）でガチャを引きキャラを集め、武器・アイテムの恒常アップグレードを買う構造が、ラン体験の外側に「集める動機」を一層敷いている。ただしこれは継続のためのフックであり、主動機は依然としてランそのものにある。

## 体験の核 / 設計の発明

`Vampire Survivors` (2021) のオート射撃 + レベルアップ毎の武器選択というフォーマットを引用しながら、二次創作という制約を逆手に取った差分を持ち込んでいる。

| 設計要素 | Vampire Survivors の原型 | HoloCure の差分 |
|---|---|---|
| 操作キャラ | 匿名の戦士 | ホロメン 47 人（実在 VTuber） |
| 武器の意味 | 抽象的なシナジー | 配信ネタ・歌・公式グッズの参照 |
| BGM | オリジナル | ホロメン楽曲のチップチューンリミックス |
| ラン外進行 | 永続パッシブ購入 | ガチャ + コイン + ステージアンロック |
| 価格モデル | 有料（数百円） | 完全無料（Cover 公式ライセンス） |

差分の核は「武器・スキル・楽曲のすべてが配信文脈の引用になっている」点にある。ぺこらの「Yagoo Box」、マリンの「海賊砲」、フブキの「タロット」など、内輪ネタを武器に変換することでビルドの発見そのものがファン的な"気づきの快感"と二重化される。`social_axis: ソロ` でありながら、ビルド結果のスクリーンショットを SNS や Twitch クリップで共有する非同期的な社交が外部に発生する構造を持っている。

## バズ要因 / 人気の根拠

- 2022 年 itch.io 公開時に **ホロリス・VTuber 層が一斉拡散**。Cover 公式ライセンス済みファンゲームという「許諾を得た二次創作」のクリーンさが、配信者本人による実況可能性を担保した。
- 2023 年 Steam 無料リリース時に **全世界 F2P ランキング上位入り**。価格 0 円の参入障壁の低さと、`Vampire Survivors` の流行に乗ったタイミングが噛み合った。
- Steam 同接ピーク約 45,000、レビュー 26,000+ の **「圧倒的に好評」99%**。無料タイトルとしては突出したレビュー密度で、ファンベースの濃さを示す。
- **47 人のホロメン操作化** + 楽曲チップチューンリミックスという密度。1 キャラあたりの作り込みがガワ替えではなく専用スキルセット・専用ボイスネタに踏み込んでいる。

## 企画への示唆

- **既存ヒットメカニクスを IP の文脈で再翻訳**すると、機構の発明コストを払わずに体験の意味だけを書き換えられる（Vampire Survivors 型 → 推し操作型）。
- **公式ライセンスを取った二次創作**にすると、配信者本人の実況可能性を解放でき、IP 側にも宣伝になる Win-Win 構造ができる。
- **完全無料化 + 個人開発スケール**にすると、F2P ランキングと SNS 拡散が同時発火し、課金設計を持たずとも到達範囲が広がる。
- **キャラ固有スキル = ファンが知っている文脈**にすると、ビルド発見の快感と「ネタが分かる快感」が二重に重なり、内輪知識を持つ層ほど深く遊べる構造になる。
- **ガチャ + コインのラン外ループ**を薄く敷くと、ラン中心のローグライトでもリテンションが伸びる（主動機を侵食しない程度の REW 副要素）。

## 比較で見るべきタイトル

- `Vampire Survivors` (poncle, 2021) — 直接の祖型。武器シナジー → ビルド発見の純度を比較する基準。
- `20 Minutes Till Dawn` (flanne, 2022) — 同型ローグライトサバイバーの兄弟作。手動エイム差分。
- `Brotato` (Blobfish, 2022) — クラス選択 + ショップ要素を加えた変種。HoloCure のメタ進行と比較可能。
- `アイドルマスター シンデレラガールズ スターライトステージ` 等の VTuber/アイドル IP × ゲーム化事例 — ガワ替えに留まる事例との対比軸。
