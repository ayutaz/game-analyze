## なぜ NAR 主分類なのか

プレイヤーの主動機は「閉鎖空間で起きる連続殺人の真相を、章単位で読み解く」ことである。ジャンルは Mystery Visual Novel / Adventure、コアの行動は会話を読む・矛盾を突く・証言を整理することであり、戦う／集める／競うはいずれも副次的位置にとどまる。

学級裁判という対話パートは「コトダマを撃つ」アクション風 UI を持つが、これは推理過程を**読み手の能動性に変換する演出**であり、戦闘的な手応えのために存在するわけではない。プレイヤーが翌章へ進む動機は「次に誰が死に、誰が黒幕か」を読むためであって、メカニクス習熟ではない。social_axis は空欄だが、構造的にはソロ完結型の物語消費で、SNS 上の考察・実況視聴という二次的な共有層が外側に乗る。secondary は空配列だが、自由行動パートでのキャラ好感度上げ（収集要素）と裁判パートのミニゲーム（体験要素）が薄く重なる構造である。

## 物語の駆動構造

章ごとに「日常編 → 非日常編（事件発生）→ 学級裁判編 → 処刑」という固定リズムを敷き、プレイヤーの読書ペースを章単位でリセットする。

| 構造要素 | 役割 | プレイヤー側の効果 |
|---|---|---|
| 閉鎖空間（希望ヶ峰学園） | 容疑者プールを 15 名に固定 | 誰が次に死ぬか／黒幕かを予想可能にする |
| 「コロシアイ学園生活」ルール | 殺人＝卒業条件という前提 | 倫理的緊張と推理動機を同時供給 |
| 学級裁判 | 章クライマックスを儀式化 | 読了感を章ごとに確定させる |
| モノクマ | 進行管理 NPC 兼脅威 | テンポ管理とトーン（黒コメディ）の保証 |
| 自由行動 | キャラ深掘り＋スキル収集 | 死者への愛着を能動的に積ませる |
| 学園の謎（メタ層） | 連続殺人を超える上位の謎 | 章を跨いだ通読動機を提供 |

この設計の要点は、章末で必ず一人を「黒（犯人）」として処刑することで、**プレイヤーが直前に好感度を上げたキャラを自分の手で送り出す**痛みをループ化している点にある。自由行動の収集が物語消費の感情エンジンとして機能している。

## バズ要因 / 人気の根拠

popularity の「2010 PSP killer that defined modern Japanese mystery VNs」を起点に分解すると、

- 原作 PSP 版（2010）が現代日本ミステリ ADV のフォーマット（閉鎖空間 × デスゲーム × 学級裁判）を確立し、後続の Zero Escape 系・AI: ソムニウム系の系譜に影響を残した
- Steam 版で 1M–2M owners・約 35,000 レビューで 96% positive と、海外 VN 市場で稀な install base を獲得
- シリーズ累計は世界 5M+ で、続編（2 / V3 / Ultra Despair Girls）とアニメ展開で IP を多媒体化
- モノクマというキャラクターデザインと「絶望」というキーワードが SNS・実況・コスプレで強い記号性を持ち、本編未プレイ層にも到達

## 企画への示唆

- 章末に必ず「強制的な別れ」を置くと、その章で積んだ好感度がそのまま読了感の重力に変わる
- メカニクスをミニゲーム化（コトダマ／論破）すると、推理 ADV の読者でも能動性を錯覚できテンポ感が出る
- 閉鎖空間 × デスゲームの組み合わせは容疑者数を固定化し、プレイヤーが推理に参加するハードルを下げる
- マスコット NPC に進行管理を委ねると、開発側のテンポ介入が物語内行為として正当化できる
- IP の記号（モノクマ的アイコン）を 1 個用意すると、本編未プレイ層への到達コストが下がる
