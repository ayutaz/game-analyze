# マーク・オブ・ザ・ニンジャ リマスター 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「2D 横スクロール画面の中で、視線・音・影を読み切って“消える”体験を組み立てる」ことにある。動詞で言えば「忍ぶ」。Klei Entertainment は 3D ステルスの定石（視界錐・聴覚円・状況に応じた索敵 AI）をすべて 2D 平面に翻訳し、視界が光円錐として可視化される、音は波紋として可視化される、敵の警戒状態は頭上アイコンで可視化される、という「情報の完全可視化」を中核に据えた。social_axis が「ソロ」なのは設計の必然で、画面の情報密度が一人の判断速度に最適化されており、協力や対戦を入れる余地がない。

secondary の NAR は「裏切られた忍者一族と憑依する刺青」を軸にした短編劇画的物語、REW は各ステージのスコア／実績／パスシステムによるリプレイ動機として副次的に作用するが、いずれも EXP の中核ループを補強する装飾レイヤーに留まる。

## 体験の核 / 設計の発明

中核メカニクスは「視界 / 音 / 影」を 2D 画面で完全に観測可能にしたうえで、プレイヤーに **逃げる / 無力化する / 殺さず通過する** の選択肢を提示する点にある。同じステージを「ノーキル」「ノーアラート」「タイムアタック」で複数回走らせる多義性を、シングル前提のままで生んでいる。

| 設計要素 | 具体実装 | プレイヤーへの効果 |
|---|---|---|
| 視界 | 敵の視界錐が光円錐として明示 | 「見える / 見えない」が確定情報になり、運要素を排除 |
| 音 | 走行・破壊・遺体落下が波紋として表示 | 音の伝播が読み切れるため計画ステルスが成立 |
| 影 | 暗所では完全に消える二値ロジック | スポット移動の起点・終点が読みやすい |
| 装備セット | 開幕で武器・道具・スーツを 1 組ずつ選択 | 同じ面を「殺傷型 / 不殺型」で別ゲーム化 |
| スコア | ノーキル・ノーアラート等の独立評価 | ソロでもリプレイ動機が立ち上がる |

リマスターでは原作（2012）の 1080p / 60fps 化と DLC 全部入りに留め、メカニクスを触らない判断を取っている。「2D ステルスとして既に完成しており、足し算より露出回復のほうが価値が高い」という判断の表れである。

## バズ要因 / 人気の根拠

- ジャンル定義作として「2D ステルスのゴールドスタンダード」と評され、後続作（Aragami、Ghost of a Tale、The Messenger の隠密パート等）が引用する参照点になっている
- Steam 評価が「Overwhelmingly Positive」、オリジナル＋リマスター累計で 100 万本超規模に到達
- Klei が後の Don't Starve / Oxygen Not Included で築いた「情報を完全可視化する UI」の原点として開発者コミュニティで再評価されている
- 2018 年のリマスターで PS4 / Xbox One / Switch に展開され、特に Switch の携帯機適性とステルスの「一手考える」テンポが噛み合った

## 企画への示唆

- 3D で確立されたジャンル文法を 2D に翻訳すると、情報密度を上げられて運要素を排除できる
- 視界・音・影をすべて UI レベルで可視化すると、プレイヤーが「失敗の理由」を即座に学習でき、難度を下げずに挫折率だけ下げられる
- 同一ステージに対し独立した複数スコア軸（殺傷 / 不殺 / 速度 / 完全潜入）を置くと、ソロ専用でもリプレイ動機が成立する
- 開幕装備セレクトで「同じ面を別ゲームに変える」と、コンテンツを増やさずプレイ時間を伸ばせる
- リマスター時にメカニクスを触らず解像度・フレーム・DLC 統合だけ行うと、原作の評価資産を毀損せず新規層に届けられる

## 比較で見るべきタイトル

- Hitman 3 / Hitman World of Assassination（3D ステルスの同コンセプト、リプレイ前提のスコア設計）
- Aragami / Aragami 2（後続の忍者ステルス、影をリソース化）
- Gunpoint / Heat Signature（同じ Klei 系譜ではないが 2D 情報可視化系ステルスの隣接作）
- 天誅 シリーズ（和製忍者ステルスの源流、3D 時代の参照点）
- Volume（Mike Bithell、2D 俯瞰の Metal Gear オマージュ）
