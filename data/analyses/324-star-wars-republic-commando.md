# スター・ウォーズ リパブリック コマンドー — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「4 人クローン特殊部隊の隊長として、戦況を読み AI 部下に的確な命令を出して戦う」ことにある。スター・ウォーズ世界の物語装飾は厚いが、本作の遊びの中心は分隊指揮 FPS としての戦闘体験そのものであり、ストーリー進行はミッションを繋ぐ枠組みに留まる。

`social_axis` は「ソロ」。マルチプレイは PC 版に限定的に存在したものの、本作の体験設計はあくまで AI 部下 3 名（Scorch / Fixer / Sev）を抱える単一プレイヤーキャンペーンとしてチューニングされている。`secondary` の NAR はクローン部隊の人格描写と Geonosis → Kashyyyk → RAS Prosecutor の章立てが副次的に効いているため、REW はステルス装備・武器アンロックの育成寄与による。いずれも主動機を補強する役割で、戦闘指揮の手触りが薄ければ成立しない。

## 体験の核 / 設計の発明

ソロ FPS でありながら「分隊長視点」を採用したことが本作の中心的発明である。プレイヤー自身も射撃するが、戦闘の質を決めるのは 3 名の部下への命令出しの精度で、これにより孤独感のないソロ体験が成立する。

| 設計要素 | 内容 | 体験への効果 |
|---|---|---|
| コンテキスト依存コマンド | 照準カーソルを地形に当てて 1 ボタン発令 | UI を画面から消し、指揮の即応性を維持 |
| 4 つの基本命令 | Search & Destroy / Secure / Sniper / Demolition | 戦況読解 → 配置の判断ループが生まれる |
| AI 部下の自律行動 | 倒れても蘇生可能、命令なしでも応戦 | 微細操作からの解放、戦術判断に集中 |
| 一人称ヘルメット HUD | バイザー越しの視界・血痕・スピーカー音声 | 個の兵士ではなく「装備の中の人」に没入 |
| Geonosis / Kashyyyk / RAS の三幕 | 屋外戦 → 高地ジャングル → 漂流艦内 | 命令の使い分けを各幕で再学習させる構造 |

前世代の Star Wars FPS（Dark Forces / Jedi Knight 系）が「フォース／ライトセイバー個人プレイ」を中心に据えたのに対し、本作は「ジェダイなしでクローンだけを操る」軸を選び、シリーズ内での FPS 設計の差分を明示している。

## バズ要因 / 人気の根拠

- Steam 上で 9,900 件超のレビューに対し 95% Positive、20 年経過後も Star Wars FPS 推奨リスト上位に挙げられるカルト的支持
- Aspyr が 2021 年に Switch / PS4 へ再リリースし、コンソール世代を跨いで現役流通している
- Steam 推定オーナー 50 万〜100 万、廃盤を経ても二桁年スパンで言及され続けるロングテール
- クローン部隊個々の人格付け（Delta Squad）はその後の Star Wars: The Clone Wars アニメや拡張 EU 設定へも影響し、IP 側のキャラクターとして再利用されている

## 企画への示唆

- AI 部下への命令を「照準当てて 1 ボタン」に圧縮すると、操作量を増やさずに戦術ゲームを成立させられる
- ヘルメット HUD（バイザー曇り・血痕・無線ノイズ）を画面に持ち込むと、世界観演出と UI 機能を同居させ没入を二重化できる
- IP の主役（ジェダイ）から意図的に外したサブクラス（クローン）を主人公に据えると、本編とは違う遊びの軸を確保でき、シリーズ内ポジションを明確化できる
- AI 部下に蘇生 / 自律応戦 / セリフ掛け合いを持たせると、ソロ体験でも「孤独な FPS」にならず、長時間プレイの感情負荷を下げられる
- カルト人気作はパブリッシャー横断の再販（Aspyr 型）で 15 年スパンの収益化が成立しうる、という事例として扱える

## 比較で見るべきタイトル

- Rainbow Six 3 / Ghost Recon（同時期の分隊指揮 FPS、命令系の祖型）
- SWAT 4（2005、AI 部下指揮 + 一人称没入の同年作）
- Halo: Combat Evolved（一人称ヘルメット HUD と SF 兵士視点の先行例）
- Star Wars: Battlefront II (2005)（同じクローン題材で大規模戦闘を主軸にした対照例）
- Aliens: Colonial Marines / Brothers in Arms 系（小隊長ソロ FPS の系譜）
