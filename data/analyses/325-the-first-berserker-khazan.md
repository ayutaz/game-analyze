# The First Berserker: Khazan

## なぜ EXP 主分類なのか

プレイヤーの主動機は「重量級の剣戟と回避タイミングを身体に刻み込み、ボス戦を 1 回ずつ攻略していく」こと。物語（Dungeon Fighter Online＝アラド戦記の前日譚として「裏切られた将軍カザンの復讐」を描く）よりも、ソロで反復するアクションの達成感が中核に置かれており、`social_axis: ソロ` の通り他者との関わりは積極的には設計されていない。

`secondary: REW` が示すのは、Nioh / Wo Long 系列に近いハック&スラッシュ的な装備ドロップとビルド構築のレイヤーが、ソウルライクの反復プレイを下支えしている点。武器固有スキルツリーや遺物の組み合わせで「次の周回はこのビルドで」と動機付けるが、これはあくまでアクション体験を増幅する補助で、主動機ではない。

## 体験の核 / 設計の発明

セルシェード調のビジュアルと、フロムソフトウェア系の硬派なボス戦設計を組み合わせ、Neople が Dungeon Fighter Online で培った「2D 格闘的なスキル感触」を 3D ソウルライクに転写したのが本作の発明点。

| 設計要素 | 内容 | プレイヤーへの効果 |
|---|---|---|
| 武器種別ビルド | 双剣 / 大剣 / 槍の 3 系統 + 固有スキルツリー | 1 周クリア後に別ビルドで再走する動機 |
| ステージ制構造 | オープンワールドではなくミッション選択型 | ボス攻略の単位が小さく、再挑戦コストが低い |
| 防御の三層化 | 回避 / パリィ / ガード崩しの使い分け | ソウルライク経験者にも新しい手触り |
| 難易度オプション | 後日パッチでイージーモード追加 | コア層と新規層を分けて拾う |

DFO 由来の「アクションのキレ」がソウルライクの「死んで学習するループ」と接続することで、Sekiro 系の緊張感に近づきつつ、よりカジュアルにビルドを切り替えられる中間ポジションを取った。

## バズ要因 / 人気の根拠

JSON の popularity に「2025 年の予想外のソウルライク批評的ダーリン」とある通り、事前期待が低かった韓国スタジオ作品が口コミで伸びた構造。

- Steam の同時接続レビューが launch 時点で「圧倒的に好評」、推定 100 万本超を数ヶ月で達成
- Dungeon Fighter Online（韓国・中国で巨大な MAU を持つ MMORPG）の IP 拡張として、既存ファンのベース需要を確保
- Neople / Nexon が AAA スタジオではない（MMO パブリッシャー）という意外性がレビュアー受けした
- 同時期に乱立したソウルライクの中で、セルシェード調の独自ビジュアルが配信映えして差別化に成功

## 企画への示唆

- **既存 IP の前日譚をシングル AAA で出すと、MMO の長期 MAU から新規プラットフォームへの橋渡しになる**（DFO ファンが PC/PS5/Xbox の据置市場に流入）
- **ソウルライクにビルド構築層を厚く重ねると、難易度の壁を「装備で乗り越える」逃げ道が生まれ離脱率が下がる**
- **ミッション選択型のステージ制を採用すると、開発コストを抑えつつボス単位のリプレイ性を確保できる**（オープンワールド ARPG より省コスト）
- **セルシェード調を選ぶと、フォトリアル ARPG の物量競争を回避してビジュアル IP として記憶に残りやすい**
- **後日パッチでイージーモードを追加すると、コア層の launch 評価を確保しながら新規層を後追いで拾える**（難易度設計の時間差リリース）

## 比較で見るべきタイトル

- Nioh 2（Team Ninja、ハック&スラッシュ × ソウルライクの先行例）
- Wo Long: Fallen Dynasty（同上、ステージ制 + 装備ドロップの構造が近い）
- Lies of P（Round8 Studio、韓国製ソウルライクの先行成功例）
- Sekiro: Shadows Die Twice（防御の多層化と剣戟タイミングの源流）
- Stellar Blade（同 2024–2025 期の韓国製シングル AAA、IP/ビジュアル戦略の比較対象）
