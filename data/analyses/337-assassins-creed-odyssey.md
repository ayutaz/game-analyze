# アサシン クリード オデッセイ 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「古代ギリシャという広大な地中海世界を、傭兵として駆け回り、戦い・探索し続ける」ことにある。Ubisoft Quebec が 2018 年に提示したのは、シリーズの伝統だったステルス暗殺中心の構造を解体し、肉弾戦・能力ツリー・装備厳選・海戦・拠点制圧を全部入りにしたオープンワールド ARPG であり、`social_axis: ソロ` の一人遊びとして設計されている。プレイヤーが画面に向かう時間の大半は「次の島へ渡る」「未踏のエリアに行く」「敵兵団を制圧する」という体験の連鎖であり、ストーリーや報酬はその周回を駆動するためのフックとして機能する。

ただし secondary に `NAR` と `REW` が同居しているのは妥当で、本作は AC シリーズで初めて選択肢付きダイアログとマルチエンディングを導入し、装備のレアリティ・エングレービング・伝説武器収集という ARPG 的報酬ループも併設している。物語と収集は EXP を「やめ時を失わせる」ための従属レイヤーである。

## 体験の核 / 設計の発明

前作 Origins で導入された RPG レベル制を拡張し、`戦闘 / 探索 / 物語` の三層を同時並行で進められる構造に再編した点が中核の発明である。

| レイヤー | 体験のドライバ | 前世代との差分 |
|---|---|---|
| 戦闘 | 3 ツリー（ハンター / ウォリアー / アサシン）のアビリティ切替、敵レベルスケーリング | Origins の単線ツリーから役割分岐へ |
| 探索 | エーゲ海全域のシームレス海戦＋上陸、傭兵 NPC 動的追跡 | 砂漠単一バイオームから多島海＋海洋移動へ |
| 物語 | 主人公性別選択、対話分岐、複数結末 | シリーズで初の RPG 的選択導入 |
| 報酬 | 伝説装備セット、エングレービング合成、傭兵階級ランキング | 直線的装備強化から ARPG 的ビルド構築へ |

`social_axis: ソロ` ながら、傭兵システムが「自分の悪事に応じて他 NPC が刺客として襲ってくる」非同期な対人感覚を作り出しており、シングルプレイの中に擬似的な競合関係を埋め込んでいる。

## バズ要因 / 人気の根拠

- `~10M+ units` と推定され、Valhalla 登場までは **Ubisoft の AC シリーズで最も収益性の高い作品** だった、と公表されている
- 「古代ギリシャ × オープンワールド ARPG」というテーマ的新規性が、Witcher 3 以後の RPG 市場で強い差別化要因になった
- 主人公の性別選択（カサンドラ / アレクシオス）と対話分岐は、AC シリーズの長年のファン層を二分する論点となりコミュニティの議論を駆動した
- DLC 「最初の刃の遺産」「アトランティスの運命」と無料アップデート「Discovery Tour: Ancient Greece」によりロングテール売上を確保した

## 企画への示唆

- **シリーズの中核メカニクスを 1 本だけ捨てる**と、長期 IP の刷新コストを抑えつつ新規層を取り込める（本作はステルス暗殺の必須性を捨てた）
- **オープンワールドに「動的な脅威 NPC」を流す**と、ソロ体験に擬似的な対人緊張を持ち込める（傭兵システム）
- **対話分岐とマルチエンディングを後付けで導入する**と、既存 IP の「物語消費型」副次動機を一気に立ち上げられる
- **戦闘ツリーを役割分岐型 3 系統にする**と、同じ素材で複数周回の動機を作れる
- **無料の教育モード（Discovery Tour）を別線で提供する**と、ゲーマー以外への露出と長期 PR 効果を兼ねられる

## 比較で見るべきタイトル

- アサシン クリード オリジンズ（2017、Ubisoft Montreal）— RPG 化の最初の一歩
- The Witcher 3: Wild Hunt（2015、CD Projekt RED）— 選択肢付きオープンワールド ARPG の参照点
- Horizon Zero Dawn（2017、Guerrilla）— 同世代のソロ向け広域探索 ARPG
- アサシン クリード ヴァルハラ（2020、Ubisoft Montreal）— Odyssey の構造を北欧で再演した直系の後続
