## なぜ EXP 主分類なのか

プレイヤーの主動機は「ソウルライクの戦闘テンポを、アニメ調キャラクターと相棒同行で味わう」こと、つまり**戦う**動詞である。Dark Souls の死にゲーループ（探索→死→学習→踏破）が骨格にあり、物語やキャラクリエイトは戦闘文脈を彩る装置として機能している。

`social_axis: ソロ` は実情と一致する。本作は AI バディ（同行 NPC）または他プレイヤーとのオンライン協力に対応するが、コア体験はソロ探索＋AI 相棒であり、PvP もパーティ前提のレイドもない。`concept` 不在ながらジャンル表記の "Anime soulslike action RPG" が本質を言い切っている：Dark Souls のメカニクスを **God Eater 系のアニメ表現**に乗せ替えた派生形である。

`secondary: REW` は副次的に効く。武器・ブラッドコード（クラス）・ギフト（スキル）の収集とビルド最適化が長時間プレイの動機を補強しており、アクションの上達曲線と並走するもう一本の柱になっている。ただし主動機が「戦闘の踏破」である以上、ガチャ的収集が中心の REW 主分類のタイトル群とは構造が異なる。

## 体験の核 / 設計の発明

Code Vein の発明は「ソウルライクの孤独感を AI バディで意図的に**緩和**した」点にある。From Software 系の難度設計を残しつつ、常時同行する NPC が被弾を引き受け、回復ギフトを撒くことで、ジャンル耐性のないプレイヤーにも踏破可能性を開いた。

| 設計要素 | Code Vein の選択 | 標準的なソウルライクとの差 |
|---|---|---|
| 同行者 | AI バディ常時 1 体 + オンライン CO-OP | ソロが原則、CO-OP は召喚制 |
| クラス | ブラッドコードで戦闘中も切替可 | 育成後固定が一般的 |
| スキル | ギフト = 装備品扱いで付け替え | スキルツリーで一方向 |
| 回復 | リジェネレーション（瓶）+ AI のヒール | 瓶のみ、回数制限が厳格 |
| キャラ表現 | アニメ調＋詳細キャラクリ | ダークファンタジー固定 |

ブラッドコード切替によるビルド流動性は本作固有で、「同じセーブで戦士／魔法／支援を試行錯誤できる」設計は装備による役割固定が一般的なジャンル内で珍しい。AI バディの存在は同時に難度設計のジレンマも生み、エンドコンテンツでは協力前提のバランスに偏る局面がある。

## バズ要因 / 人気の根拠

- `sales_world` は **出荷 3M+（2023 時点）、Steam 推定オーナー 2–5M レンジ**。バンナム製アクション RPG として商業的に成立した規模。
- "God Eater meets Dark Souls" のキャッチが SNS で機能。**God Eater シリーズの作画スタッフが流用された**ためビジュアルアイデンティティが既存ファン層に届きやすかった。
- ソウルライクが PC で大衆化した 2017–2019 年の波に乗り、Steam 配信を主軸にしたことで `target` の海外コアアクション層を取り込んだ。
- 詳細キャラクリエイトとアニメ調表現は VTuber／配信文化と相性が良く、配信映え素材として中規模の継続露出を得た。

## 企画への示唆

- **ジャンル耐性の低い層を AI 同行で取り込むと**、ニッチジャンルの裾野が広がる（難度を下げず、踏破可能性だけ底上げできる）。
- **ビジュアルアイデンティティを既存 IP の作画資産から借用すると**、新規 IP でも初動の認知コストが下がる（Code Vein は God Eater 系の絵で参入障壁を圧縮した）。
- **クラスを装備品化すると**、1 周のリプレイ価値が伸びる（ブラッドコード方式は「別キャラを作り直さなくていい」体験を生む）。
- **ソウルライクに詳細キャラクリを足すと**、配信・SNS 露出が増える（同じゲームでも視覚的にユニークな自キャラが拡散される）。
- **AI バディに依存させすぎると**、エンドコンテンツのバランス調整が難しくなる（ソロ前提と CO-OP 前提のどちらに最適化するかを最初に決める必要がある）。

## 比較で見るべきタイトル

- **Dark Souls III** — 直接の参照元。死にゲー骨格の比較で Code Vein の AI バディの効果が浮かぶ。
- **God Eater 3** — 同社のアニメ調アクション RPG。作画資産と世界観継承の度合いを測る基準。
- **Lies of P** — 後発の AAA ソウルライク。アニメ調ではない方向に進んだ並行解。
- **Stranger of Paradise: Final Fantasy Origin** — 和製 AAA がソウルライクに参入した別解、ジョブシステムが Code Vein のブラッドコードに近い。
- **Wo Long: Fallen Dynasty** — 同じく和製ソウルライクで、こちらは士気システムで難度緩和を試みた対照例。
