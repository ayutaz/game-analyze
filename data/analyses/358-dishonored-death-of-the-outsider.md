## なぜ EXP 主分類なのか

このタイトルでプレイヤーが何度も立ち上げる動機は「同じ任務を別解で潜る」体験そのものにある。マップに撒かれた経路、暗殺対象、非殺傷ルート、能力の組み合わせを毎回違う配合で試す行為が中核で、動詞でいえば「忍ぶ・読む・切り抜ける」。social_axis がソロなのは、敵 AI と地形パズルに対する単独行動の連続だからで、対戦も協力も介在しない。

secondary に NAR と REW が入るのは妥当である。Billie Lurk を主人公にした Dishonored サーガの幕引きという物語的機能（NAR）と、契約ボード形式のサイドジョブ報酬・骨片改造による能力ビルド（REW）が、ステルス体験のリプレイ動機を厚くしている。ただし物語は一本道で再生視聴に耐える章立てではなく、報酬もガチャ的ランダム性は持たない。主動機はあくまで「自分の手で潜入する手触り」であり、EXP に分類するのが妥当。

## 体験の核 / 設計の発明

本作の発明は、Corvo / Emily の Outsider 由来マーク制を捨て、Billie 固有の 3 能力に絞ったことにある。マナ自動回復化により「能力をケチる」緊張が消え、代わりに「いつどう連打するか」のリズムが前面に出る。

| 設計要素 | 前作 (Dishonored 2) | 本作 (DotO) |
|---|---|---|
| 能力ツリー | アウトサイダーの印 + 大量の枝分かれ | Billie 固有 3 能力に絞り、枝なし |
| マナ管理 | エリクサー消費型、節約が前提 | 自動回復、連打前提のテンポ |
| 進行構造 | 主線ミッション + 高低差マップ | 主線 + 契約ボード（任意暗殺・盗難） |
| 死生判定 | カオスシステム（殺害数で結末分岐） | カオス廃止、結末は固定 |
| 物語の主体 | 王族復権の大義 | 元暗殺者 Billie の終止符 |

カオスを外し、契約ボードを足した結果、「殺してもいい / 殺さなくてもいい」が倫理ではなく純粋な技術選択に変質している。EXP としての解像度を上げ、NAR の重みを下げた割り切りといえる。

## バズ要因 / 人気の根拠

- Standalone 形態で本編未プレイ層にも入り口を提供し、Dishonored アークの「閉じ方」として扱われた。
- SteamSpy 推定 1〜2M owners という規模感は本編 2 より小ぶりだが、Immersive sim 文脈では十分に語られる帯域に収まる。
- Arkane Lyon の Immersive sim 系譜（Dark Messiah → Dishonored → Prey）の到達点として批評文脈で頻繁に参照される。
- Billie Lurk という脇役主人公への切り替えにより、シリーズの「視点を変える続編」モデル（Wolfenstein: TNC → TOB に近い）として議論された。

## 企画への示唆

- スタンドアロン・エピローグ形式で出すと、本編未プレイ層と既存ファン両方に同時にリーチできる（価格と長さを本編より下げるのが前提）。
- 能力ツリーを「無くす」と、ビルド多様性は減るが代わりに各能力の手触り検証に開発リソースを集中できる。
- マナを自動回復にすると、能力使用は「資源管理」から「リズム」に変質し、初心者のフラストレーションが下がる。
- カオスシステムのような倫理スコアを外すと、ステルスは道徳劇ではなく純粋なパズル / 体験になり、再走の心理的コストが下がる。
- 契約ボードのようなサイドジョブ列挙 UI を入れると、メインミッション中の寄り道が「面倒な探索」ではなく「リスト消化の達成感」に置換される。

## 比較で見るべきタイトル

- Dishonored 2（前作、能力豊富 / カオスあり / ソロ EXP）
- Prey (2017)（同 Arkane、Immersive sim、ソロ EXP の純度比較）
- Hitman 3（契約ボード形式の暗殺サンドボックス、ソロ EXP）
- Deathloop（同 Arkane、能力 3 軸化と時間ループの実験）
- Thief: Deadly Shadows（ステルス系譜の起源側、ソロ EXP）
