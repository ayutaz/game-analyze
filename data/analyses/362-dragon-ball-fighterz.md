# ドラゴンボール ファイターズ

## なぜ REW 主分類なのか

プレイヤーの主動機は「好きなキャラを 3 人組ませて、原作再現の必殺技と派手な演出を引き出す」こと――つまり**集めた駒を盤面で輝かせる**ことに寄っている。本作は格闘ゲームの皮を被っているが、入口の大半は「悟空 / ベジータ / フリーザ / セル / ブロリー…と続く IP アセットを自分のチームに組み込む」消費であり、ガチャこそ無いものの DLC キャラの継続追加によって**コレクション欲求が長期リテンションを牽引**してきた。`concept` 欄は空白だが、Arc System Works が原作カットインを各キャラに数十パターン仕込み、特定の組み合わせ（悟空 vs フリーザ等）で専用演出が走る設計は、「機能としての強さ」より「**この組み合わせを再生したい**」という収集動機に直結している。

副次の EXP（対戦の腕前を磨く体験）は確かに強い柱で、EVO 等のトーナメントシーンも形成された。ただし `sales_world` の `10M+ worldwide` という規模は競技勢だけでは到達しない数字で、**圧倒的多数は「ドラゴンボールのキャラで遊びたい」層**が押し上げている。よって主分類は REW、副次に EXP を置く構成が妥当となる。

## 報酬ループの組成

3 対 3 のアサルトチーム制と、シーズンごとの DLC キャラ追加が、収集ループの主軸を構成している。

| 層 | 報酬コンテンツ | 駆動する欲求 |
|---|---|---|
| キャラ取得 | 初期 24 + DLC で 40 超に拡張 | 推しキャラの実装待ち |
| チーム編成 | 3 人組のシナジー / アシスト選択 | 「自分の三体」を作る快感 |
| ドラマチック演出 | 特定マッチアップで原作再現カットイン | コレクションの「再生」報酬 |
| 称号・ロビーアバター | オンラインロビーで可視化 | 自己提示・承認 |
| 競技スコア | ランクマッチ / ワールドツアー | 上位帯への到達欲 |

ガチャ的な確率設計は無いが、**「DLC でキャラが増える → 新しいチーム編成を試す → 既存キャラの評価も変わる」というメタ更新型のループ**が、F2P モバイルゲームのシーズンパスに近い役割を果たしている。`secondary: EXP` が効くのは最上層だけで、中間層は「キャラが揃ったから遊ぶ」動機が支配的。

## バズ要因 / 人気の根拠

`popularity` の "Arc Sys/Bandai's biggest fighting-game hit; defined modern anime fighter design on Steam" を起点に分解する。

- **世界累計 10M 本超**（`sales_world`）。Arc System Works 製格ゲーの中で歴代最大規模で、ギルティギア / ブレイブルー単体の数倍の到達。
- **Steam で 91% positive / 26K レビュー**。コア層からの評価が高止まりし、長期間ディスカウントの度に新規が入る構造。
- **2.5D 表現の確立**。3D モデルをセルシェード＋手描き調エフェクトで処理する手法を Steam の anime fighter ジャンルに広め、後発（GBVS、DNF Duel 等）の参照点になった。
- **EVO 等での競技採用**と原作ファン層の二重に強い IP（ドラゴンボール）が重なり、観戦コンテンツとしても拡散した。

## 企画への示唆

- **既存 IP の「組み合わせ演出」を仕込むと、ゲームプレイ自体がガチャ的な再生報酬になる** ― 確率や課金が無くても、コレクション動機を駆動できる。
- **3 人タッグ構造を採用すると、キャラ実装 1 体あたりの編成バリエーションが組合せ的に増える** ― 単体追加コンテンツの体感価値が水増しされる。
- **コアの競技性とライト層の IP 消費は両立できる** ― 操作の入口を簡単（オートコンボ）にしておけば、上級者の伸びしろは別レイヤーで担保できる。
- **DLC キャラの追加サイクルをシーズン化すると、買い切り型でも F2P 的なリテンション曲線を作れる** ― 「次は誰が来るか」という期待が再起動を生む。
- **発色とエフェクトの過剰さは、配信映え＝マーケ予算の削減につながる** ― 観戦コンテンツとして拡散する設計を最初から織り込むと、後年の競技シーン化が安価に進む。
