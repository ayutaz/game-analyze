# ドラゴンボールZ KAKAROT — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「原作の名場面を一つひとつ回収し、悟空たちを最強状態まで育て切る」こと——つまり**集める／育てる**である。CyberConnect2 がアニメ ARPG として構築した本作は、Z 編全体（サイヤ人〜魔人ブウ）を「サブクエスト・修行・コミュニティボード・料理・Z オーブ・装備魂玉」という多層の収集系で覆い、原作追体験そのものを「報酬の獲得行為」へ変換している。`social_axis` が空欄なのも妥当で、フローはほぼ完全な単独周回型である。

副次の EXP は格闘ゲームならぬ「フィールド飛行 + ロックオン格闘」のアクション部分が担っている。ボス戦の演出と回避・気弾・界王拳といった操作面の手応えは確かに快感だが、それ単体ではゲームは持たない。**敵を倒すと Z オーブと経験値とドロップが落ちる**という設計が示す通り、戦闘は報酬抽出ループの駆動装置として位置づけられている。

## 報酬ループの組成

KAKAROT は報酬を「IP 由来 × メカニクス由来」で重ね合わせ、原作読了体験そのものを進行 KPI に落とし込んでいる。

| レイヤー | 内容 | リテンション役割 |
|---|---|---|
| メインストーリー | Z 編全章のアニメ準拠リプレイ | IP ファンの離脱抑止（次章への牽引） |
| サブストーリー | チチの家事、ヤジロベー、孫娘パンら脇役クエスト | 周回プレイ動機、アニメ未収録小ネタ補完 |
| コミュニティボード | キャラ間絆を「ソウルエンブレム」配置で強化 | 育成パズル、思考時間の延長 |
| 修行・Z オーブ | フィールド散策で集めて技をアップグレード | 探索リワード、空中飛行の意味づけ |
| 料理・釣り・採集 | 一時バフ + 図鑑コンプ | セッション間の小目的、原作世界観の質感 |
| DLC（Trunks、Bardock、超編、神と神、F 復活） | 追加章 + 新ボス + 新装備 | LTV 延長、原作外伝・劇場版の取り込み |

「ストーリー追体験 → 戦闘 → ドロップ → ボード配置 → 強化 → 次章」のループは、ガチャを使わない買い切り型でありながら、運営型モバイルゲームの**収集 → 編成 → 戦闘**サイクルとほぼ同形である。

## バズ要因 / 人気の根拠

- ローンチから約 6 年で**累計 10M 本超**（2026 年 1 月時点、クロスプラットフォーム）と、CyberConnect2 系 IP ARPG として最上位の成功スケール
- **Steam レビュー約 43K 件 / 93% 好評**——コア層の評価が安定しており、セール時の長期売れ筋として機能
- 原作 Z 編を「ゲームで丸ごとプレイできる初の本格 ARPG」というポジションが、Sparking! 系格ゲー・Xenoverse 系オンライン格闘とは別の購買動機を生んだ
- DLC（Trunks 編、超編、神と神、復活の F、バーダック）と新規無料アップデートを継続投入し、シリーズ累計需要を切らさずスナップショット販売を伸ばし続けた

## 企画への示唆

- **強 IP × 原作追体験 × ARPG ループ**を組むと、ガチャに頼らず買い切りでも 10M 級が成立する（IP 側のテキスト資産がコンテンツ量問題を解決する）
- 戦闘を「報酬抽出装置」と定義し直すと、アクション部分の単調さがリテンション設計上の弱点にならない（ドロップ品で意味づけし直せる）
- 「サブクエスト = 原作の脇役を主役にする補完話」と定義すると、ファン心理を満たしながら本編尺を引き延ばせる
- コミュニティボード型の**配置パズル育成**は、装備強化より思考時間を稼げ、セッション当たりの可処分時間を伸ばす
- DLC を「劇場版・外伝の取り込み」と位置づけると、本編の世界観を壊さず LTV を 2〜3 倍に延ばせる
