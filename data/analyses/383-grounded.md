# グラウンデッド / Grounded

## なぜ EXP 主分類なのか

プレイヤーの主動機は「縮小した自分の身体で、見慣れたはずの裏庭を未知の生態系として踏破する」という空間体験そのものである。Obsidian Entertainment が手掛けた本作は Honey-I-Shrunk-the-Kids 的な縮小設定をサバイバルジャンルに落とし込み、虫の鳴き声、雨、夜の静寂までもがスケールのコントラストで脅威へ変換される。`concept` は明示されていないが、`genre` の "Backyard miniature survival" と `social_axis: ソロ` が示すのは、まずシングルプレイで「歩く・観察する・恐怖する」体験を成立させる設計だ。

`secondary: REW` は素材収集と装備ツリー、BURG.L クエストによる収集・育成ループとして効いている。アリの卵、グモのキバ、芝刈り機の刃片などを集めてベース建築と武器を強化する報酬構造はあるが、それ自体が目的化しているわけではなく、より深い庭・より危険な虫へ到達するための「探索の足腰」として機能している。だから主分類は REW ではなく EXP となる。

## 体験の核 / 設計の発明

中核メカニクスは「スケール変換による既知の再認識」である。プレイヤーは自分が小さくなったのではなく、世界の解像度が上がったかのように設計されており、芝の一本一本、露の一滴、コガネムシの体節までゲームプレイの地形となる。

| レイヤー | 何を再構築したか | 体験への効果 |
|---|---|---|
| スケール | 人間サイズ → 数 mm | 既知のオブジェクト（ジュース缶、レゴ）がランドマーク化 |
| 生態系 | 通常の動物相 → 節足動物相 | 虫が恐怖対象になり虫嫌い対応の "arachnophobia safe mode" が必要に |
| サバイバル文法 | 木の伐採・採掘 → 草の刈り取り・キノコ採取 | 既存ジャンル文法を裏返して新規性を確保 |
| 4 人 Co-op | ソロ完結＋協力可 | `social_axis` はソロ基準だが、友人を呼ぶと探索の役割分担が発生 |

前世代の Subnautica が「水中＋未知の惑星」で達成したサバイバルの再発明を、Grounded は「身近すぎる裏庭」という逆方向の素材で達成している点が設計上の発明である。

## バズ要因 / 人気の根拠

- 2023 年時点で **20M+ プレイヤー**到達。Xbox Game Pass の first-party サバイバルとして最大級の成功例として `popularity` に記録されている。
- Game Preview（Early Access 相当）で 2020 年から段階的に拡張し、1.0 リリース時には完成度とアップデート履歴の双方で評価を獲得。
- クロスプレイ対応（PC / Xbox / 後に PS5・Switch）により、Game Pass 経由の流入と PS / 任天堂ユーザーが同じセッションで遊べる構造を作った。
- 「虫が苦手な人向けモード」という UI 設計上の配慮が広く言及され、配信での話題化と参入障壁の低下を同時に達成した。

## 企画への示唆

- **既知の空間をスケール変換すると、ジャンル文法を再利用したまま新規性が出る**：森や宇宙ではなく裏庭を選んだ判断が、低コストで「未踏感」を生んでいる。
- **生理的反応を持つ題材（虫・高所・暗所）にはアクセシビリティモードを最初から実装する**と、配信文化での拡散と参入障壁低下を両立できる。
- **Game Preview → 1.0 の長期運営型サバイバル**にすると、無料配布や Game Pass 経由の試遊が「未完成だから様子見」を「無料で参加できる開発」に転換できる。
- **ソロ完結を担保しつつ 4 人 Co-op を被せる**と、`social_axis: ソロ` のままコミュニティ動画が生まれ、マーケ予算に依存しない口コミが回る。
- **収集・建築は EXP の従属レイヤーに留める**：報酬ループを主従の主にすると探索の緊張が崩れる。Grounded は素材を「次の庭へ行くための装備」に紐づけている。

## 比較で見るべきタイトル

- Subnautica（未知環境を「水中」で再構築したサバイバルの原型）
- Valheim（北欧神話×ソロ可協力サバイバル、コミュニティ駆動）
- Don't Starve / Don't Starve Together（題材切替×アート様式の差別化）
- Raft（小スケール題材を選んだサバイバルの先行例）
- The Forest / Sons of the Forest（恐怖を主軸に置いたサバイバル比較対象）
