# ジャストコーズ3 (Just Cause 3) 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「壊す・飛ぶ・繋ぐ」を組み合わせて自分なりのアクションを連鎖させること、つまり**遊び場を物理的に弄ぶ体験**そのものにある。Avalanche Studios が用意した架空地中海島嶼 Medici は、独裁政権の解放という物語の外装をまといつつ、本質はグラップリングフック・ウィングスーツ・無限パラシュート・破壊可能オブジェクトを掛け合わせる**サンドボックスの遊具**として設計されている。social_axis が「ソロ」であるのも、誰かと競うのではなく自分の即興アクションを自分で評価するループが中心だからだ。

secondary に NAR と REW が付いているのは、解放戦争の縦軸ストーリーと、地域解放％・ギア MOD アンロック・チャレンジメダルといった進捗報酬が副次的にプレイ時間を支えているため。ただし主動機はあくまで「次にどんな破壊コンボを試すか」であり、物語や収集は遊び場へ戻る口実として機能する。

## 体験の核 / 設計の発明

中核は **「常時 3 つの移動／攻撃手段が同時に走る」** ことだ。前作 (Just Cause 2, 2010) はグラップル＋パラシュートの 2 層構造で「上空からの強襲」が定型化していたが、3 ではウィングスーツを加えた 3 層化と、グラップルテザーの**複数本同時接続と再引き寄せ**で「物理ギャグの作者になる」設計に踏み込んだ。

| 設計要素 | 前世代 (JC2) | JC3 での更新 | 体験への効果 |
|---|---|---|---|
| 移動レイヤ | グラップル＋パラシュート | ＋ ウィングスーツ（無燃料）| 着地せず巡航、移動＝アクション化 |
| テザー | 1 本ずつ | 複数本同時＋再引き寄せ可 | 物体同士の連結で物理パズル生成 |
| 破壊対象 | 拠点ごと固定 | SAM／燃料タンク／統治モニュメントが「赤いオブジェクト」として明示 | 破壊ターゲットの可視化＝目的が即座に立ち上がる |
| 解放％ | 個別 % | 州単位の解放ゲージ＋市民蜂起 | プレイ後の世界が視覚的に変わる |

social_axis は「ソロ」だが、即興アクションが**動画化しやすい**ため SNS / YouTube 経由で擬似的な観戦軸を獲得する構造になっている点も見逃せない。

## バズ要因 / 人気の根拠

JSON の popularity は「Wingsuit-and-grappling sandbox destruction — defining mid-2010s playground」。これを分解すると次の要素に還元できる。

- **ウィングスーツ導入による移動の遊具化**：ファストトラベルを使わず自走することそのものが快感になる設計。
- **テザーの拡張による物理ギャグ**：兵士と神像を繋ぐ／戦闘機に岩を縛り付けるといった即興コンボが**配信者の素材**として消費された。
- **SteamSpy バンドで 5〜10M owners**（sales_world）規模。Square Enix の PC AAA としては Tomb Raider 系に次ぐクラス。
- **発売直後の最適化問題**（PS4 のフレームレート低下）が話題化し、逆に「PC 版で遊ぶサンドボックス」というポジショニングを強化した。

## 企画への示唆

- **移動手段を 3 層化すると、移動それ自体がアクションになる**：徒歩→車→ファストトラベルではなく、徒歩→グラップル→ウィングスーツのように**減速ゼロで切替可能な 3 層**にすると、移動時間が遊びに転化する。
- **破壊対象を色（赤）で明示すると、目的設定コストが消える**：オープンワールドで「次に何をすればよいか」を UI ではなく**ワールド側のオブジェクト色**で示すと、プレイヤーは常に行動候補を持てる。
- **物理オブジェクト同士を結ぶ仕組みを 1 つ足すと、創発コンボが指数的に増える**：単体メカニクスではなく**メカニクス間の組み合わせ可能性**を増やすことが、ソロ EXP のリプレイ性を生む。
- **解放％を州単位で可視化すると、サンドボックスにも進捗の物語が乗る**：純粋な遊び場に「世界が変わる」レイヤを薄く重ねると、NAR / REW 動機のプレイヤーも巻き込める。
- **ソロ作品でも「動画映え」を設計に組み込むと、配信が観戦軸を代替する**：social_axis がソロでも、SNS で共有される瞬間を作る構造（スローモーション、即時リプレイ等）を仕込めば、観戦コミュニティが自然形成される。

## 比較で見るべきタイトル

- **Just Cause 2 (2010)** — 前世代。2 層移動と固定破壊ターゲットの設計を比較する基準点。
- **Just Cause 4 (2018)** — 続編。テザー MOD のさらなる自由化と天候システム導入による設計変化。
- **Saints Row IV (2013)** — 同時期の「物理ギャグ系オープンワールド」。スーパーパワーでの移動遊具化を別解で実装。
- **Red Faction: Guerrilla (2009)** — 構造物破壊を物理で表現した先行例。破壊の質感が対照的。
- **Mercenaries 2 (2008)** — 破壊可能ワールドの先祖。JC シリーズが何を継承し、何を捨てたかが見える。
