# シャドウラン ドラゴンフォール (Director's Cut)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「サイバーパンク × ファンタジーのベルリンで、ランナー（傭兵）チームを組み、ハック・魔法・銃撃のどれで切り抜けるかを毎ミッション選び直す」体験そのものにある。`concept` が示すアイソメトリック CRPG は、ターン制戦闘と非戦闘解決（ハッキング・交渉・隠密）を 1 つの任務内で切り替えられるよう設計されており、戦って勝つことよりも「どう切り抜けたか」を反芻させる構造になっている。`social_axis` がソロなのは、共闘 NPC を含むパーティ運用を 1 人のプレイヤーの判断で回す設計のためで、対人要素を持たない代わりに「自分のランナー像」をビルドで物語る余地を最大化している。

副次の NAR は、Berlin Flux State という独自設定とミッション間のハブ「キーズ」での会話・関係性が、戦闘体験を物語に紐づける接着剤として機能している点に表れる。物語自体を読みに行くのではなく、体験の意味づけを物語が補強する役割を負う。

## 体験の核 / 設計の発明

このリバイバル三部作の中で Dragonfall が「高点」と評される理由は、前作 Returns の硬直したミッション直列構造を解体し、ハブ＋分岐型のキャンペーンへ書き換えた点に集約される。Director's Cut で投入されたチームメイトの個別ミッションとアビリティ刷新が、戦闘以外の解決軸を実装レベルで担保した。

| 設計レイヤ | 内容 | 効果 |
|---|---|---|
| クラスレス成長 | カルマで全スキルへ自由配分 | 「ストリートサムライ＋デッカー」のような越境ビルドが可能 |
| ハブ拠点キーズ | 任務間にチーム会話・装備・依頼選択 | プレイヤーがランナー集団の一員という連続感を獲得 |
| マトリクス + 戦闘並走 | 1 名がデッキ潜行中も現実側で戦闘継続 | 役割分担が時間軸でも発生 |
| Run 単位の解決経路 | ハッキング / 交渉 / 戦闘 / 隠密が並立 | 同じミッションが別ビルドで別体験になる |

Returns との差分は「ミッション選択順がプレイヤー裁量」「同行 NPC が固有人格を持つ」「敵 AI が改修された」の 3 点で、Director's Cut でこれらが完成形に到達した。

## バズ要因 / 人気の根拠

`popularity` が示す「Shadowrun CRPG リバイバルの高点」という評価は、以下の要素の合成で形成された。

- Steam owners が約 50 万本規模に達し、レビューが Overwhelmingly Positive で安定している
- 2012 年の Kickstarter で 185 万ドル超を集めた本家プロジェクトの 2 作目で、コミュニティ期待値が初動を支えた
- Director's Cut が既存ユーザーへ無償アップデートで配布され、レビューの母集団とスコアを同時に底上げした
- Berlin Flux State という TRPG 原作の人気設定を CRPG として初めて本格描写し、原作ファン層を取り込んだ

## 企画への示唆

- ハブ拠点に「装備・会話・次任務選択」を集約すると、ミッション間の物語的接着が自然に発生する
- 戦闘・ハック・交渉を同一ミッション内で並立させると、1 つのコンテンツが複数ビルドで再消費される
- クラスレス + カルマ自由配分は「越境ビルド」を生み、プレイヤーごとに固有のランナー像が物語化される
- 前作の不満点（線形構造・固定パーティ）を Director's Cut として再パッケージし無償配布する手は、新規よりレビュー基盤の補強に効く
- 設定の希少性（サイバーパンク × ファンタジーのベルリン）が、ジャンル飽和市場での識別子として機能する

## 比較で見るべきタイトル

- Shadowrun Returns (2013) — 同シリーズ前作、構造的反省点を読む
- Shadowrun: Hong Kong (2015) — 同フォーマットの後継、Dragonfall の設計が継続したか検証
- Pillars of Eternity (2015) — 同時期 Kickstarter 発の Isometric CRPG リバイバル
- Wasteland 2 (2014) — クラスレス成長と倫理選択肢を共有する近接ジャンル
