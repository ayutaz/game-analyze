# Slay the Spire

## なぜ EXP 主分類なのか

このゲームの主動機は「**毎回違うデッキを組み上げて塔を登り切る**」というラン単位の体験であり、プレイヤーは「集める」ためにカードを取るのではなく、その場の盤面と相性を読み解いて「**戦う**」ためにカードを取る。social_axis が「ソロ」であることが象徴的で、対戦相手や協力者を必要とせず、AI が出してくる敵パーティとカードプールという乱数だけで、毎回新しい戦術的体験が立ち上がる。

secondary に REW が入っているのは、Ascension 解禁・キャラクター解禁・Relic コレクション・実績などのメタ進行が長期動機として効いているため。ただしこれらは「ラン中の戦闘で何を組むか」という EXP の核を**補強する燃料**として配置されており、主従関係は明確に EXP が上位にある。

## 体験の核 / 設計の発明

中核は「**カード／レリック／ポーション／敵パターン**という 4 種類の乱数を、ノードマップ上の経路選択で部分的にコントロールできる」点にある。プレイヤーは引きを嘆くのではなく、引きに合わせてデッキの設計図を書き換え続ける。

| 設計要素 | 機能 | 体験への効果 |
|---|---|---|
| カード選択（戦闘後 3 択 1）| デッキを膨らませる主動線 | 「取らない」も有効解になる緊張感 |
| Relic（パッシブ効果）| デッキ全体に乗算的に効く | 同じカード構成でも別ゲーム化 |
| マップ分岐（？/$/焚火/エリート）| 乱数の選択的取得 | ルート設計＝メタ戦略 |
| Ascension 20 段 | 難度ハードコア化 | 上級者向け再周回動機 |

前世代の TCG (Hearthstone / MtG Arena) との差分は、**デッキ構築と対戦が一体化したシングル体験**にまとめ、対人マッチング・資産課金・メタ環境を消したこと。Dream Quest (2014) のアイデアを発展させ、UI・カードバランス・Relic 設計を商業水準まで磨き上げた。social_axis 「ソロ」は意図的な選択であり、対人を排したからこそ「**敗北は完全に自分の判断ミス**」という閉じた因果系が成立する。

## バズ要因 / 人気の根拠

- 全プラットフォーム累計 500 万本以上、Steam 推定所有者 500 万〜1000 万本という、インディーとしては突出した到達距離
- ジャンル名「ロードライク・デッキビルダー」を事実上発明し、Monster Train / Inscryption / Across the Obelisk / Wildfrost / Slay the Spire 2 等の後続ジャンルを牽引
- Twitch / YouTube で「Ascension 20 Heart kill」「Watcher infinite」などの解析配信が定着し、**観戦コンテンツとしての寿命**を獲得
- Early Access 期から有志による Mod / Tier list / 統計サイト（spirelogs 等）が立ち上がり、コミュニティ駆動の研究が継続

## 企画への示唆

- **乱数を「嘆くもの」ではなく「設計の材料」にすると、毎ランが新作になる** — 4 種類以上の乱数軸を直交配置し、相互作用で組み合わせ爆発を起こす
- **対人を切り捨てる代わりに難度を 20 段刻みで提供すると、上級者のリプレイ動機を MMO 級に伸ばせる** — Ascension 構造はサ終リスクなしの長寿命設計
- **「取らない」が有効解になる選択肢を出すと、プレイヤーの主体性が跳ね上がる** — カード追加 = デッキ希釈というデメリットを明示する
- **ジャンル名を作れるレベルの簡潔なコア（カード + Relic + マップ）に絞ると、フォロワーが勝手にジャンルを広げてくれる** — 後発の成功が原典の権威を補強する
- **Mod API と統計の公開を早期に許容すると、コミュニティが永続的なメタ研究機構として機能する**

## 比較で見るべきタイトル

- Monster Train（2020）— レーン構造と二重デッキで差別化したフォロワー
- Inscryption（2021）— デッキビルダーをメタ演出・ホラーで再解釈
- Balatro（2024）— ポーカー役に乱数構築を載せ、コア発明を別ドメインに移植
- Hades（2020）— ロードライク × 物語進行で「死ぬほど進む」を別解化
- Darkest Dungeon（2016）— ソロ × 構築 × ハードコア難度の前世代代表
