# ヴァルヘイム アシュランズ

## なぜ EXP 主分類なのか

プレイヤーの主動機は「友人と少人数の常設サーバーで、ヴァイキング的な未知の大陸を切り拓きながら基地と装備を更新する」体験そのものにある。Ashlands は base Valheim の終盤バイオームを丸ごと追加する無料大型アップデートで、`social_axis=協力` が示すとおり、新規ボス・新規金属・新規敵 AI を 2〜10 人の常設ワールドに投入することで「再度集まる理由」を作る設計になっている。コンテンツ消費ではなく、火山地帯の制圧という共同プロジェクトを回す過程が中核なので EXP が妥当。

`secondary=REW` は、新金属 Flametal や新装備ツリーが「次に何を掘る／作る」という収集・育成のループを供給している側面を指す。ただし装備強化はあくまで火山地帯に滞在し続けるための手段であり、装備獲得それ自体を目的化させる回し方はしない。

## 体験の核 / 設計の発明

Ashlands は base Valheim の前提（手作業の建築 / 段階解放されるバイオーム / 死亡時アイテムドロップ）を保ったまま、終盤側に「敵対的すぎる土地」を 1 層足したもの。前世代との差分を分解すると以下になる。

| 要素 | base Valheim までの設計 | Ashlands での変化 |
|---|---|---|
| 地形 | 上陸後に拠点を張れる | 上陸自体が戦闘イベント（沿岸砲台 / 常時湧き） |
| 進行 | バイオーム→ボス→次解放 | 既存ボス済み前提で「火山要塞攻略」型に転換 |
| 建築 | 木・石中心 | 黒石・グラウスタン等の高難度素材で要塞化 |
| 協力 | 採集分担で並行作業可 | 沿岸保持と内陸侵攻のロール分業が必要 |

social_axis「協力」の効きは、上陸後に常時敵が湧くため**誰か 1 人が拠点防衛を担う必然性**が生まれる点に表れる。Iron Gate は「ソロでも遊べるが協力で本領」というラインを base から維持しており、Ashlands はその設計を最も尖らせた層である。

## バズ要因 / 人気の根拠

- base Valheim は早期アクセス開始（2021）から累計 2000 万本超を売っており、Ashlands はその巨大な休眠アカウントを呼び戻す「late-EA renaissance（後期 EA の再起動）」として配信時に同接が急上昇した
- 無料大型アップデートという課金構造のため、過去購入者全員がそのまま追加コンテンツの対象になる（拡張販売型より復帰摩擦が低い）
- 既存サーバーを残したまま追加バイオームに行ける設計なので、フレンドグループ単位の再集合が起こりやすい
- Coffee Stain 配下の Satisfactory / Deep Rock Galactic と同様、「少人数協力サバイバル」ジャンルの定番として Steam・配信プラットフォームでの可視性が高い

## 企画への示唆

- 終盤コンテンツを**有料 DLC ではなく無料アップデート**で出すと、休眠アカウントの復帰モーメンタムが課金売上より大きな PR 効果を生む
- 既存ワールドを破壊せず新規大陸として追加する設計にすると、**フレンドグループの常設サーバー資産が継続価値**として機能する
- 上陸イベント自体を戦闘化すると、ソロ周回ではなく**ロール分業による協力プレイ**が自然発生する
- 最終バイオームのキー資源（Flametal）を既存装備ツリーの延長に置くと、コンテンツ追加が**新しい収集ループの追加**として二重に効く
- 早期アクセス期間中に「late-EA renaissance」を狙ったマイルストーン更新を仕込んでおくと、正式リリース前にもう一度メディア露出が取れる

## 比較で見るべきタイトル

- Deep Rock Galactic（同 Coffee Stain 系、少人数協力サバイバル、シーズン更新型）
- Satisfactory（同 Coffee Stain、長期 EA 運営、無料大型更新で復帰促進）
- Terraria（無料大型更新による late-EA 的な復帰モーメンタムの先行事例）
- 7 Days to Die（協力サバイバルの長期 EA、終盤コンテンツ追加の参考系）
- Enshrouded / Nightingale（直近の協力サバイバル新興、Valheim 比較対象）
