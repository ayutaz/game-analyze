# ARC Raiders（アーク・レイダーズ）

## なぜ EXP 主分類なのか

プレイヤーの主動機は「機械生命体 ARC が支配する地表に降りて、限られた装備で遺物を持ち帰り、他のレイダーと撃ち合うか・避けるか・組むかを毎ラウンド判断する」体験そのものにある。物語のクライマックスや恒久報酬の積み上げではなく、出撃ごとに発生する緊張・交渉・撤退の連続が中核で、`social_axis: 対戦` も「PvP の撃ち合い」より「他者の存在によって毎セッションの意思決定が変わる」体験設計を指す。

Embark Studios が Battlefield 系の元 DICE スタッフを中心に立ち上げたスタジオであり、PvPvE 抽出シューター（extraction shooter）というジャンル自体がプレイヤーの即興判断と心拍数の維持に最適化されているため、NAR（物語追体験）でも REW（恒常的なガチャ・育成）でもなく EXP に置く方が動機を素直に説明できる。`secondary` が空なのは、収集（戦利品）や育成（クラフト）といった層が存在しつつも、それらは「次の出撃をもう一度面白くするための燃料」として機能しており、独立した主動機にまで成長していないという編集判断である。

## 体験の核 / 設計の発明

抽出シューターというジャンルは Escape from Tarkov が 2017 年に提示し、その後 Hunt: Showdown、The Cycle: Frontier、Marauders などが追随した文脈にあるが、ARC Raiders は「PvE 敵対者（ARC ロボット）」を一次的な脅威として強く前面に出し、PvP は二次的に発生する構造を採る。これにより従来の extraction が抱えていた「初心者狩り問題」を緩和し、Tarkov 系のハードコア層と Apex Legends 系のカジュアル層の中間市場を取りに行った。

| 設計要素 | ARC Raiders の選択 | 旧世代 extraction との差分 |
|---|---|---|
| 視点 | サードパーソン | Tarkov / Marauders は FPS。遮蔽物越しの索敵で恐怖と判断の幅が広がる |
| 主敵 | ARC 機械（PvE） | Tarkov は Scav、Hunt は Boss。PvE が常に「中央の重力」として機能 |
| 抽出フロー | 地下拠点 → 地上 raid → 抽出ポイント | 共通だが、拠点側の生活感（クラフト・隣人）に時間を割く |
| マッチ人数 | 比較的小規模（数十名規模） | Tarkov 同等。Battle Royale より密度低く、遭遇は意図的 |
| 死亡時ペナルティ | 持ち込んだ装備喪失 | extraction の共通言語をそのまま維持 |
| 社会軸の重さ | 対戦（敵対 PvP）が中心、協力も可 | PvP / 協力 / 単独のどれを選ぶかが毎セッションの戦略になる |

中核の発明は「PvE を恐怖の中央装置に据えたまま、PvP を選択可能にした」点である。Tarkov が「他プレイヤーが最大の脅威」だったのに対し、ARC Raiders は「機械が脅威、他プレイヤーは交渉対象」というレイヤー反転を行い、結果として撃ち合わずに撤退するプレイ（pacifist runs）にも遊びの正当性を与えている。

## バズ要因 / 人気の根拠

- Nexon の自社配給作品として過去最大のグローバルローンチを記録し、発売 2 週間で 400 万本、2026 年 2 月時点で 1,400 万本まで積み上げた
- Steam 同時接続ピーク 70 万人は extraction shooter ジャンルでは突出した数字で、Tarkov が独占してきた市場の中間層を一気に獲得した
- 配信文化との相性が高く、撃ち合いを回避して交渉や逃走を選ぶ「非戦闘 run」がクリップ素材として拡散しやすい設計になっている
- Embark Studios のグラフィック技術（machine-learning ベースのアニメーションや音響）が新興スタジオの技術ショーケースとして評価され、Nexon の西側展開戦略の象徴になった

## 企画への示唆

- ジャンルの中央装置（脅威）を PvP から PvE に置き換えると、初心者と熟練者の体験差が縮み、市場の中間層を取り込める
- 撤退・回避・交渉を「正規の勝利条件」として認知させると、配信向けクリップの種類が増え自然な広告になる
- サードパーソン視点を選ぶと、遮蔽物越しの索敵と緊張表現が成立し、FPS よりも「観察 → 判断」のテンポが視聴者に伝わりやすい
- 装備喪失リスクを残しつつ拠点側にクラフトと生活感を置くと、ラウンド間のクールダウンが「次の出撃の計画フェーズ」に変換できる
- ジャンルの先行作（Tarkov）が抱える参入障壁（操作・UI・治安）を一つずつ穏当化すると、ジャンルの「2 番目の標準形」を取りに行ける

## 比較で見るべきタイトル

- Escape from Tarkov（抽出シューターの原型、PvP 中央型）
- Hunt: Showdown（PvE ボス + PvP の二層構造、サードパーソン寄りカメラの先行例）
- The Finals（同 Embark Studios、社会軸：対戦、技術基盤を共有）
- Marauders / The Cycle: Frontier（抽出シューターの近接競合、市場縮小事例として）
- Helldivers 2（PvE 中央装置 + 協力プレイで 2024 年に extraction 隣接市場を取った事例）
