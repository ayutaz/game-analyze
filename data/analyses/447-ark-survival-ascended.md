## なぜ EXP 主分類なのか

このタイトルでプレイヤーが繰り返している動詞は「探索する／恐竜を手懐ける／拠点を建てる」であり、物語を読むためでも、ガチャを引くためでもない。動機の核は、未知の島で生存資源を確保し、敵対生物を制圧して飼い慣らし、それを足場に次の生物圏へ踏み込む――という体験そのものにある。

`social_axis: ソロ` は逆説的だ。ARK シリーズは公式 PvP / PvE サーバーや非公式サーバーで強いコミュニティ性を持つが、Ascended は UE5 リマスターという性質上、シングル＋小規模プライベートサーバーでの遊び方が主軸になり、「自分の島・自分のテイム履歴」を積み上げるソロ体験の比重が高い。`secondary: REW` が示すのは、テイムした恐竜の系統・カラー・突然変異を集める／繁殖させる収集要素が、体験の中に強く絡みついている点である。テイム自体は EXP のプロセスだが、その成果物は REW として残る二層構造になっている。

## 体験の核 / 設計の発明

ARK の中核は「危険な生物を時間と資源を投入して味方化する」テイミングと、そこから派生する**運搬・戦闘・建築の生態系**である。Ascended は新規ゲームプレイの発明ではなく、UE5 への載せ替えによってこの生態系全体を再点火することを狙った再構築版だ。

| 設計レイヤ | 内容 | Ascended での差分 |
|---|---|---|
| サバイバル基盤 | 空腹・体温・酸素・病気 | UE5 ライティングで昼夜サイクルの脅威が再描画 |
| テイミング | 気絶 → 餌付け → 個体育成 | 全 DLC マップを順次バンドル提供 |
| 建築 / 拠点 | スナップ式パーツ＋電力／配管 | Nanite による大規模建築の描画安定化 |
| 繁殖 / 突然変異 | 世代を重ねて色・ステを選別 | 既存メカニクスを継承（収集動機の支柱） |
| サーバー文化 | 公式 PvP / PvE、非公式 MOD | クロスプレイ前提の再構築 |

発明された新メカニクスは少ないが、「同じゲームを世代を跨いで作り直し、コミュニティの再集合場所を強制的に作る」という運営上の打ち手が、Ascended の最大の設計判断と言える。

## バズ要因 / 人気の根拠

- `popularity` が示すとおり、UE5 リマスターに全拡張をバンドルした構成と「有料リメイク」論争自体が話題化の燃料になった。
- `sales_world: Steam owners 1-2M` の規模感は、AAA 級ではないものの、サバイバル系としては十分大きい。`sales_jp` 空欄が示すように、日本での独立した市場プレゼンスは大きくない。
- ARK 2 の延期を背景に、コミュニティの「次の集合場所」として Ascended が事実上の橋渡しを担った。
- MOD・非公式サーバー文化が UE5 環境に移植され、配信・SNS での建築／繁殖コンテンツが継続的に流通している。

## 企画への示唆

- 新メカニクスを発明しなくても、**描画エンジンを更新して同じ体験を作り直す**と、コミュニティの離脱層を呼び戻す効果が出る。
- テイム → 繁殖 → 突然変異の三段ループを敷くと、EXP の行為が REW の収集物に転化し、長期プレイヤーの離脱コストが上がる。
- 全拡張をバンドルする課金設計は、新規参入の摩擦を下げる反面、既存ユーザーの「二重購入」感情を必ず生む。リマスターは**価格と互換性のコミュニケーションが本体機能と同じ重さ**を持つ。
- ソロでも遊べるが PvP/PvE サーバーで化けるという二層構造は、配信向け／コミュニティ向けの両方の入口を確保する。
- 次世代タイトル（ARK 2）の遅延リスクに対し、現行 IP を載せ替えた「橋渡し版」を出す手法は、長期 IP 運営の保険として参考になる。
