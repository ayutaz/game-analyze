# ATOM RPG: Post-Apocalyptic Indie Game (2018) 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ソ連崩壊後の荒野を、自分のビルドと選択で歩き回って生き残る」こと。すなわち**探索する／戦う／組み立てる**の体験そのものが目的であり、ストーリーは舞台装置として消費される。`social_axis: ソロ` はクラシック CRPG の系譜そのままで、マルチプレイ要素を一切持たない単独探索ゲームである。

`concept` / `target` フィールドは未記入だが、`genre: Soviet post-apoc isometric CRPG` と `popularity` が示すように、本作は Fallout 1/2 の SPECIAL 系ステータス・ターン制 AP 戦闘・分岐会話・スキルチェック・カルマ風評価を、東欧開発スタジオが「ソ連版」として再構築したオマージュ作品である。プレイヤーが楽しんでいるのは物語の結末ではなく、**ステータス振りとパーク選択が会話と戦闘の解像度を変える瞬間**にある。`secondary: NAR` が立っているのは、ソ連崩壊後のディストピア叙事と無数のサイドクエスト文章量が体験を支えているためで、主動機ではなく体験の厚みを担保する役回りに留まる。

## 体験の核 / 設計の発明

中核メカニクスは **Fallout 1/2 を忠実に再構築した古典 CRPG フォーマット**を、Bethesda 以降の Fallout が捨てた要素ごと持ち帰った点にある。

| レイヤ | 設計 | プレイヤーへの効果 |
|---|---|---|
| キャラ作成 | SPECIAL 風 7 ステータス + パーク + スキルポイント | ビルド分岐がそのまま会話・解錠・修理・戦闘の成否に直結 |
| 戦闘 | 6 角ヘックス + AP 制ターンベース + 部位狙撃 | リアルタイム化された Fallout 3/4 とは別解像度の戦術選択 |
| 探索 | アイソメトリック俯瞰の固定マップ + Wasteland 風ランダムエンカウント | 「歩いていたら何かが起きる」古い乱数体験を意図的に温存 |
| 会話 | スキル / ステータス要件付きの分岐選択肢 | 同じ NPC が知力 6 と魅力 8 で別の結末を返す |
| 経済 | クラフト・修理・空腹・睡眠・放射線 | サバイバル要素が戦闘ループに圧をかける |

「ソ連崩壊後」というレイヤーは見た目だけのスキンではなく、KGB 残党／コルホーズ農村／犯罪組織といった東欧固有の派閥構造をクエスト設計に持ち込んでおり、Fallout シリーズが描かなかった文化的余白を埋めている。

## バズ要因 / 人気の根拠

`popularity` の「ロシア発 Fallout オマージュがニッチジャンルでグローバルヒットに化けた」を分解すると：

- Steam 所有者 50 万〜100 万本規模、レビューは Very Positive を長期維持し、`Soviet post-apoc isometric CRPG` という極小ジャンルで明確な代表作になった
- AAA 化した Fallout 3/4/76 に不満を持つ「古典派 Fallout ファン」の受け皿として機能し、Wasteland 2 / Underrail / Encased と並ぶリバイバル CRPG 群の一角を占めた
- ロシア語圏スタジオ AtomTeam が自前パブリッシングで PC・Mac・Linux・モバイル・据置 3 機種まで横展開し、ニッチ作品としては異例の 8 プラットフォーム配備に達した
- 続編 `ATOM RPG: Trudograd`（2021）が同じエンジン・同じ設計思想で追加コンテンツ的に出たことで、シリーズ IP としての継続性を獲得した

## 企画への示唆

- **「死んだジャンルを忠実に蘇生する」と、ジャンルそのものを独占できる** — Fallout 1/2 の文法を Bethesda が捨てた瞬間、その文法を欲する数十万人の市場が空席になる。最大化より独占を狙う設計
- **スキンを文化的に差し替えると、オマージュは派生作品になる** — 米国核戦争後を「ソ連崩壊後」に置換するだけで、派閥・建築・服装・ジョーク全てに固有性が生まれ、単なるクローン批判を回避できる
- **ステータスを会話とスキルチェック両方に効かせる**と、ビルドが「戦闘の強さ」ではなく「世界の見え方の解像度」になり、ソロ体験の周回価値が跳ね上がる
- **小規模スタジオでもプラットフォーム展開 8 機種は可能** — アイソメトリック 2D + ターン制戦闘は移植コストが低く、モバイル・Switch まで配備して長期テールを稼げる
- **続編を「フルプライス DLC」設計で出す**と（Trudograd 型）、エンジン再利用で開発コストを抑えつつ IP 継続を演出できる

## 比較で見るべきタイトル

- Fallout 1 / 2（1997–1998、Interplay）— 直接の参照元、SPECIAL とヘックス戦闘の原典
- Wasteland 2 / 3（2014 / 2020、inXile）— 同時期の古典 CRPG リバイバルの旗手
- Underrail（2015、Stygian Software）— よりハードコアな等尺 CRPG、ビルド分岐の極北
- Encased（2021、Dark Crystal Games）— ソ連系開発による別の Fallout オマージュ、直接の競合作
- ATOM RPG: Trudograd（2021、AtomTeam）— 同シリーズ続編、エンジン再利用と都市集中設計の対比
