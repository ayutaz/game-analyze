# 黒い砂漠 (Black Desert) — 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は **「装備を磨き、キャラを盛り、序列を上げる」** — つまり数値とビジュアルの双方を伸ばし続けることにある。Pearl Abyss が 2014 (韓国) / 2016 (北米欧州) / 2017 (Steam) と地域展開してきたサンドボックス MMORPG であり、戦闘アクションの瞬間体験 (EXP) は確かに強烈だが、それは「強化済みキャラを動かして気持ちよくなる」報酬の出口であって、入口ではない。プレイヤーが日々ログインする理由は、エンチャント / Lifeskill / マーケットボード / 馬の繁殖 / ペット強化 / 衣装課金といった**多層に重なった育成ループ**にある。

`secondary: ["EXP"]` がそれを補強する。コンボベースのノンターゲット戦闘と圧倒的なキャラクリエイターが「報酬を可視化するための演出装置」として機能しており、装備の見栄え・モーションの差・スクリーンショット文化が報酬欲求を強める方向に効いている。`social_axis` が空欄なのは PvE ソロ進行・大規模ノードウォー PvP・ギルド協力・マーケットを介した非同期経済が並列に走るためで、単一の軸に押し込めにくい構造になっている。

## 報酬ループの組成

| 層 | 主な要素 | プレイヤーが追う指標 |
|---|---|---|
| 戦闘装備 | Boss 装備 / 強化 (PEN まで) / 結晶 | AP / DP の整数値 |
| Lifeskill | 採集 / 釣り / 貿易 / 料理 / 錬金 | 各スキル Mastery レベル |
| 経済 | Central Market / ノード投資 / 労働者 | 銀貨保有量・時給 |
| 見た目 | キャラクリ / 衣装課金 / 馬の毛色 | スクショ映え・SNS 露出 |
| 序列 | ノードウォー / アリーナ / Guild Boss | サーバー内ランキング |

PEN (PEN+15 など) への強化は確率破壊を伴うため、ガチャ的なドキドキを「自己責任の素材投入」に翻訳している。Lifeskill は戦闘から独立して走らせられるため、戦闘がだれてきたプレイヤーをそのまま釣り船・農場・貿易に逃がし、解約させずに別レーンの育成へ流し込む。衣装課金 (Pearl 衣装) は性能ではなく見た目とユーティリティに振っており、装備強化の結果を「見せる」場として経済の出口になっている。

## バズ要因 / 人気の根拠

`Defining Korean sandbox MMORPG on Steam; benchmark for combat/character creation in the genre.` を起点に分解する。

- 累計 **5500 万人超の登録プレイヤー**、フランチャイズ収益 **€20 億超**、Steam 版だけで西欧 / 北米で **120 万本超** 販売されており、F2P 移行前の有料パッケージ時代から長期に渡って収益が積み上がっている。
- ノンターゲット戦闘 + コンボ入力という、当時の MMORPG では珍しい操作感が「動画映え」を生み、配信文化と相性が良かった。
- キャラクリエイターはジャンル内でも参照点として扱われ、本体を買わずキャラクリだけ DL する層が一定数存在するほどの仕上がり。
- ノードウォー / 攻城戦という大規模 PvP がギルド単位のニュース化を生み、韓国・台湾・東欧サーバーで上位ギルドの抗争がそのまま広報になる構造。

## 企画への示唆

- **戦闘以外の独立ループを並列で走らせる**と、戦闘飽和層をそのままサービス内に留めて LTV を伸ばせる (Lifeskill が好例)。
- **強化を「ガチャ」ではなく「素材投入の確率破壊」に翻訳**すると、ユーザーは課金の意思決定を自分で握っていると感じやすく、心理的抵抗が下がる。
- **キャラクリ単体で配布**すれば、本体購入前の試遊・SNS 拡散・コスプレ文化への接続が無料で得られる。
- **見た目課金を性能課金から分離**すると、Pay-to-Win 批判を回避しつつ ARPPU の高い層を抱えられる。
- **大規模 PvP の結果を「サーバー史」として可視化**すると、ギルド単位の物語が外部メディア化されやすく、運営の広報コストが下がる。

## 比較で見るべきタイトル

- ArcheAge (XL Games, 2013) — サンドボックス MMORPG の系譜・経済重視という意味で直接の前世代。
- TERA (Bluehole, 2011) — ノンターゲット戦闘 MMORPG の韓国系前駆。
- Lost Ark (Smilegate, 2018) — 韓国 MMORPG の Steam 大ヒット組、強化ガチャ構造の参照点。
- Final Fantasy XIV — 戦闘 + Lifeskill (クラフター / ギャザラー) 並列構造の別解。
- New World (Amazon, 2021) — 西側サンドボックス MMORPG のローンチ後失速例として対照。
