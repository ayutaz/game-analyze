# Blasphemous 2 — 武器スワップで再設計された巡礼

## なぜ EXP 主分類なのか

プレイヤーの主動機は「3 種の武器を瞬時に切り替えながら、ゴシック宗教画めいたマップを踏破する**戦う／探る**」体験にある。前作 (2019) がピクセル 2D メトロイドヴァニアの「重い一撃 + パリィ」を軸に据えていたのに対し、本作は **メイス系 (Veredicto) / 双剣系 (Sarmiento & Centella) / ロングソード系 (Ruego Al Alba)** という質感の異なる 3 武器を最初期から獲得させ、組み合わせで道を開かせる構造を採用した。social_axis が「ソロ」である通り、対戦・協力要素はなく、自己最適化の閉じたループだけで完結している。

secondary に REW が入るのは、ロザリオビーズ / 遺骨 / 心臓 / ステータスツリーなどビルド要素が層状に積まれているため。ただしこれらは戦闘・探索を解像度高くするためのモジュレータであり、装備を集めること自体が目的化する設計ではない。あくまで「武器を振る快感」が動機の中心にある。

## 体験の核 / 設計の発明

| レイヤー | 前作との差分 | プレイヤー行動への効果 |
|---|---|---|
| 武器 | 1 種類固定 → 3 種類同時所持・即時スワップ | コンボ中の中断キャンセル、属性に応じた地形破壊 |
| 探索 | 単一ルート → 武器ごとに開く地形ギミック | 3 周目までの「再訪読み替え」が前提化 |
| プラットフォーミング | 控えめ → 二段ジャンプ + 鏡瞬間移動 + 壁ピン | アクション操作の比重が増し、Hollow Knight 寄りに接近 |
| アップグレード | 線形 | ステータスツリー + パッシブビーズの 2 軸 | ビルド差分が体感操作に直結 |

「武器をスワップする」操作が単なるダメージ調整ではなく、**移動・解錠・コンボ継続のすべてを兼ねる**点が設計の核。1 つのボタンが攻撃 / 探索 / パズルの 3 軸を背負うことで、プレイヤーの瞬時の判断量を増やしている。

## バズ要因 / 人気の根拠

- popularity は「武器スワップに振り切った続編。批評筋から高評価」と要約され、メトロイドヴァニア専門レビューサイトでも前作からの改良点として武器同時所持が頻繁に言及された
- Steam 推定で累計約 100 万本規模、コンソール (PS5 / Xbox Series X/S / Switch) でも同時発売され、機種横断で日本語ローカライズが入った
- スペイン語圏 (The Game Kitchen はセビリア拠点) の宗教・絵画モチーフが独自色を保ち、Hollow Knight / Hades フォロワーが氾濫する 2023 年のインディーシーンで差別化に成功
- Team17 パブリッシングにより小売・サブスク (PS Plus / Game Pass) 経由の露出も確保

## 企画への示唆

- 主要アクションを 3 系統並立にしてホットスワップを許すと、戦闘・探索・パズルの 3 軸を 1 つの操作で兼用でき、ボタン総数を増やさずに判断量だけを増やせる
- 続編で「同じジャンル内で何を増やすか」迷ったら、**前作の中核 1 要素を多重化する** (1 武器 → 3 武器、1 ジャンプ → 二段＋瞬移) のが安全な拡張軸になる
- 宗教・神話モチーフを徹底的にアートディレクションに落とすと、機構が他作と似ていても固有名詞・固有絵で記憶される
- メトロイドヴァニアでビルド要素 (REW 的層) を入れる際は、装備収集を目的化させず「同じ操作の解像度を上げるモジュレータ」に留めるとジャンル純度が保てる
- ソロ専用でも、武器ごとに固有の地形ギミックを用意すれば「マップの再読み替え」だけで周回モチベーションは作れる

## 比較で見るべきタイトル

- Hollow Knight (2017) — メトロイドヴァニア + 高難度近接の現代的基準
- Blasphemous (2019) — 前作。1 武器・遅めテンポとの差分が本作設計を逆照射する
- Salt and Sanctuary / Salt and Sacrifice — 2D ソウルライク系の探索 + ビルド両立例
- Nine Sols (2024) — 東洋モチーフ + パリィ特化メトロイドヴァニアの近接ライバル
- Ender Lilies / Ender Magnolia — 召喚式攻撃で「武器多重化」を別解で実装した同世代作
