# Celeste

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ピクセル単位の入力精度で山を登りきる体験そのもの」にある。物語要素（不安障害を抱える Madeline の自己受容）が高く評価されているため NAR 寄りに見えるが、実プレイ時間の大半は「同じ部屋を 30 回 40 回死に戻り、ようやくクリアした瞬間の解放感」に費やされる。物語は登攀という体験を裏打ちするフレームであり、駆動装置ではない。

social_axis は「ソロ」。マルチや非同期競争を排し、各部屋に Death Counter（死亡回数）を表示してプレイヤーと画面の二者で完結する閉じた構造を採る。一方で speedrun コミュニティと連動した競技的サブ文化が育っており、secondary に **REW** が付くのは「Strawberry 175 個 / B-Side / C-Side / Golden Strawberry」という収集・到達層が独立した目標として機能しているため。コア体験は EXP、その上に REW が乗る構造となっている。

## 体験の核 / 設計の発明

| 層 | 設計要素 | 何を発明したか |
|---|---|---|
| 入力 | 8 方向ダッシュ + 壁掴み + ジャンプの 3 動作 | ボタン数を最小化し、組み合わせの深さを部屋ごとに増幅 |
| 部屋 | 1 画面 = 1 ステージ、即時リスポーン | 死亡コストを 1 秒未満に圧縮し、試行回数を稼ぐ |
| 難度 | Assist Mode（無敵 / ダッシュ無限 / 速度 50%） | アクセシビリティと「正規難度の純度」を両立させた |
| 収集 | Strawberry / Crystal Heart / B-Side カセット | クリア線とは別軌道の到達指標を二重化 |
| 物語 | チャプター末尾の対話と内省演出 | 体験の達成感と Madeline の登攀メタファーを同期 |

前世代の精密プラットフォーマー（Super Meat Boy、N+）と比較した発明は二つ。第一に、死亡からのリトライを「画面遷移なし・効果音なし・即フレーム」で繋ぎ、フロー断絶をゼロに近づけたこと。第二に、Assist Mode を「初心者向けモード」ではなく独立した設定群として提示し、難度の選択をスティグマ化しない設計にしたこと。後者は以降のインディー作品の標準仕様となった。

## バズ要因 / 人気の根拠

- 世界累計 1.7M+ 本（2025年1月）。インディー精密ジャンルでは Hollow Knight、Cuphead に次ぐ規模。
- メンタルヘルスを正面から扱った物語が GDC・The Game Awards Best Independent Game 等で評価され、批評文脈と一般プレイヤー文脈の両方を取り込んだ。
- speedrun シーンの定番タイトル化。Any% / 100% / All Red Berries 等カテゴリが GDQ 系イベントで継続的に走られ、配信視聴の長尾効果を生んだ。
- 開発者 Maddy Thorson のオープンな開発姿勢（ソースコード公開、Mod 環境 Everest の容認）がコミュニティ作品（Strawberry Jam 等数千部屋規模の有志 Mod）を育てた。

## 企画への示唆

- **リトライコストを 1 秒未満に圧縮すると、難度は「不快」から「テンポ」に変質する**。死亡演出を削ることはペナルティの削除ではなく試行回数の獲得である。
- **アクセシビリティ設定を「弱者救済」ではなく「設定群」として提示すると、難度選択がスティグマ化しない**。Assist Mode の名称設計は語彙レベルの発明。
- **収集要素を「クリア軌道」と別レイヤーに置くと、EXP コア体験を壊さず REW 層を積める**。Strawberry はクリアに必須ではないが熟練度の可視化として機能する。
- **物語をメカニクスのメタファーと同期させると、ゲームプレイそのものが感情曲線になる**。「登る = 自己と向き合う」を 8 章構造で反復した。
- **Mod・speedrun を許容するコミットメントが、発売後 5 年以上の長尾を作る**。コアループが堅牢であれば外部制作物が露出を再生産する。

## 比較で見るべきタイトル

- **Super Meat Boy**（2010）— 精密プラットフォーマーの直接の前世代。即時リスポーンの祖。
- **Hollow Knight**（2017）— 同時期インディー、難度設計と物語のメタファー同期の別解。
- **Hades**（2020）— 高頻度試行 + 物語層の重ね方の別アプローチ（ローグライク版）。
- **A Short Hike**（2019）— 同じ「登山」モチーフを EXP の方向ではなく探索の方向に振った対照例。
- **Cuphead**（2017）— 同世代ハードコア・アクション、Assist Mode を持たない設計の対比。
