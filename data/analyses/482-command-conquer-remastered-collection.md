# Command & Conquer Remastered Collection (2020)

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「資源を採って軍を組み、相手より速く読み合って制圧する」体験そのものを反復する** ことにある。1995 年の Tiberian Dawn と 1996 年の Red Alert を 4K 化＋オリジナルの 8 ビットインターフェース感を残したまま再構築した本作は、シングルキャンペーン（FMV 含む）の再走と、マルチプレイの対戦・観戦という二つの遊び方を内包する。social_axis は **対戦** で、ラダー / クイックマッチ / カスタムルールの上で「ハーベスター護衛 vs ラッシュ」「テスラコイル vs 物量」の択を回す行為自体がコアの楽しみになる。

secondary に **REW** が入っているのは、ミッション 100 本以上＋FMV アーカイブ＋サウンドトラックという「収集される素材」が同梱されており、シリーズファンが資料的価値として完走 / 解放を目指す動機が副次的に効くため。物語消費（FMV）は触媒であって主動機ではない。

## 体験の核 / 設計の発明

| 層 | 中核要素 | 前世代（1995–96 オリジナル）との差分 |
|---|---|---|
| 視覚 | 4K リマスター、スプライト 8 倍書き直し | スペースバーで原画 320x200 に即時切替可能 |
| 操作 | スクロール速度 / ホットキー / カメラピン現代化 | ユニット選択上限・キュー管理が現代 RTS 並 |
| マルチ | Steam / EA アカウント直結ラダー、リプレイ・観戦 | LAN / IPX 専用、観戦概念なし |
| 改造 | 公開 Map Editor、ゲームソース（TD / RA）GPL で開放 | 改造はバイナリパッチで非公式 |
| 資産 | FMV リマスター、サントラ全曲、コンセプトアート 4 時間 | ディスク容量制約で圧縮 FMV |

設計の発明は **「ゲームコードの GPL 公開」と「原画↔リマスター瞬時切替」** の 2 点に集約される。前者は modding コミュニティを公式が制度化した珍しい事例で、対戦バランスパッチも MOD 経由で恒常的に供給される土壌を作った。後者は「思い出補正の主観と現実のギャップ」を可視化する UI として機能し、配信視聴者にも刺さる。

## バズ要因 / 人気の根拠

- EA が自社の 25 年前タイトルのソースコードを GPL で開放した点が、RTS / 改造文化の双方で話題化し、reddit / Hacker News で拡散
- Petroglyph（Westwood の元スタッフが在籍）と Lemon Sky による「原作スタッフが触る」体制が公示され、ファンの信頼を担保
- Steam owners 50万〜100万本規模の推定（中堅 RTS リバイバルとしては成功ライン）、リリース当日に Steam Top Seller 上位入り
- FMV シーン全編の 4K 再エンコードと公式 YouTube 公開が「90年代 B 級 SF 演出」のミーム的再評価を促した

## 企画への示唆

- **ソースコードを開放すると、コミュニティが恒久バランスパッチャーに化ける**（公式メンテ寿命を 3〜5 倍に伸ばせる）
- **原画と高解像度を 1 ボタンで切替できると、配信映え＋ノスタルジア検証の両方が同時に成立**する
- **元開発スタッフが在籍するスタジオを起用したと公示すると、リマスター企画の最大の地雷である「魂の不在」批判を先回り回避できる**
- **オリジナルが含む B 級 FMV・サントラをアーカイブ商品として束ねると、ゲームを遊ばないファンも資料目的で買う層が顕在化**する
- **対戦ラダー＋観戦＋リプレイの 3 点セットを後付けで実装すると、25 年前タイトルでも e スポーツ的二次活動が可能になる**

## 比較で見るべきタイトル

- Age of Empires II: Definitive Edition（2019、Microsoft）— 同時期の RTS リマスター、観戦＋ラダー＋常時バランスパッチの先行例
- StarCraft: Remastered（2017、Blizzard）— 韓国 e スポーツ層へのアプローチ、原画↔HD 切替の発明元
- Warcraft III: Reforged（2020、Blizzard）— 失敗側のサンプル、リマスター企画における「原作改変の過剰」が招く反発
- OpenRA — C&C / RA / Dune 2000 を再実装するオープンソースプロジェクト、公式 GPL 化が後押しした派生エコシステム
