# コアキーパー（Sunken Sea / Cherished Memory アップデート）

## なぜ EXP 主分類なのか

プレイヤーの主動機は「真っ暗な地下洞窟をランタンの灯で少しずつ可視化し、掘る・建てる・戦う」という探索体験そのものである。Pugstorm の Core Keeper は 2D 俯瞰のボクセル・サバイバル／クラフトで、Terraria と Stardew Valley の中間に位置づけられるが、コアタイトルが既にカタログ収録済みのため、本エントリは 2024 年以降の大型無料アップデート（Sunken Sea = 水中バイオーム、Cherished Memory = ボス／装備再構成）を中心に扱う。social_axis は「ソロ」だが最大 8 人マルチに開いており、目的駆動ではなく**プレイヤーが自分でゴールを切り出す**サンドボックス設計が一貫している。

secondary に REW が立つのは、新装備・新素材・新スキルツリーといった「集める／強くする」副ループがアップデートのたびに増築され、エンドゲーム勢のリテンションを支えているため。ただし主動機はあくまで「未踏のバイオームに踏み込んだ瞬間の景色」にあり、収集は探索の燃料として機能している。

## 体験の核 / 設計の発明

Core Keeper の中核は「**中心の Core から放射状に広がる円形ダンジョン**」という独自の地形構造で、Terraria の横スクロールでも Valheim のオープンワールドでもない、**極座標的に難易度が上がる**マップを採用している。Sunken Sea / Cherished Memory アップデートはこの軸構造を壊さず、新バイオームを「外周」に追加する形で延長した。

| 要素 | 設計 | プレイヤー体験への効果 |
|---|---|---|
| Core 中心地形 | 中央から外周へ進むほど敵・素材の階層が上がる | 進捗の方向感覚が直感的に分かる |
| Sunken Sea | 水中バイオーム＋舟・釣りメカ | 移動手段の刷新で既知マップが再走査対象になる |
| Cherished Memory | ボス再戦／レガシー装備リワーク | 上位プレイヤー向けの second hump を提供 |
| 8 人協力 | ソロでも成立しつつ協力で建築が加速 | social_axis ソロのまま、コミュニティ運営が可能 |
| 自動化要素 | コンベア・ドリル・ファーム自動化 | サバイバルから「工場ゲー」への滑らかな移行 |

前世代との差分は、Terraria が「水平に世界が広がる」のに対し Core Keeper は「下方向＋外周方向に世界が掘られる」点、そして Stardew Valley の時間制カレンダーを廃して**プレイヤーのテンポで進める**点にある。社会軸はソロ既定だが、Discord 主導のコミュニティが MOD・建築共有を強く回している。

## バズ要因 / 人気の根拠

- 1.0 リリース時点で Core Keeper はカタログ表記で世界累計 **500 万本超**に到達しており、Early Access 期からのコア層が大型無料アップデートのたびに復帰する「波打つ DAU」モデルを定着させた。
- Sunken Sea / Cherished Memory は **DLC 課金ではなく無料アップデート**として配信されたため、Steam レビュー数の再加速と SNS 上のスクリーンショット拡散を同時に引き起こした。
- 4 プラットフォーム同時運用（PC / Switch / PS5 / Xbox Series X/S）で、ゲームパッド需要と Switch の携帯機需要を同時に取り込んでいる。
- ジャンル的に Terraria・Don't Starve・Valheim とプレイヤー層が重なるため、配信者の「次に何やる？」枠で循環的に取り上げられやすい。

## 企画への示唆

- **マップの幾何形状を独自にすると、それ自体が記憶される構造になる**：極座標型の Core Keeper、上下スクロールの Terraria、島構造の Valheim は、構造そのものがブランドとして機能する。
- **無料大型アップデートを年 1〜2 回の波として設計すると、サブスク不要の継続的話題化が成立する**：DLC 課金より低 ARPU だが、SNS 上の再露出と新規取り込みコストを相殺できる。
- **social_axis を「ソロ既定／協力可」にすると、コミュニティ運営コストを下げつつ口コミを最大化できる**：必須マルチではないので導入摩擦が低く、配信では協力プレイ映像が回る。
- **サバイバルから自動化への滑らかな勾配を用意すると、エンドゲームの離脱を遅らせられる**：手作業 → 半自動 → 完全自動の 3 段階を素材ツリーに織り込む設計が、Factorio 系の長期勢を取り込む。
- **既知マップを再走査させるアップデート（新移動手段／新探知器）は、新規バイオーム追加より低コストで継続率を稼げる**：水中・空中・地下深部のような「軸の追加」は資産再利用率が高い。

## 比較で見るべきタイトル

- Terraria（横スクロール／対戦・協力／2011）— ボクセル系サバクラの長期運営モデルの原典
- Valheim（オープン 3D／協力／2021）— 北欧神話×無料アプデで波を作る運営の参照
- Stardew Valley（俯瞰／ソロ＋協力／2016）— 個人開発由来の長期サポート文化の前例
- Don't Starve Together（俯瞰／協力／2016）— ソロ既定→協力拡張の参照
- Factorio（俯瞰／自動化／2020）— エンドゲームの自動化勾配の参照
