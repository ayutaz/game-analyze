## なぜ EXP 主分類なのか

プレイヤーの主動機は「壊れた美意識のステージを、規定外のムーブで突破する」体験そのものにある。`genre` に書かれた immersive sim / FPS / sicko-core が示すとおり、本作は Deus Ex / Hitman 系の「ステルス・暗殺・解法の多重化」を継承しつつ、UI とテクスチャを意図的に破壊して提示する。`social_axis` がソロであることも整合的で、プレイヤーは他者と協調するのではなく、ステージごとの最適解（あるいは最も狂った解）を自分で発見していく。

`secondary` に NAR と REW が並ぶ理由は明確だ。NAR はサイバーディストピアの寓話（株を売って蘇生／臓器を売却して資金繰り）が散文ではなく環境とシステムで語られているから、REW は武器・インプラント・株式の収集と装備差分による解法拡張があるからで、いずれも単独では成立せず、体験を補強する副軸として効いている。

## 体験の核 / 設計の発明

中核は「immersive sim の語彙を、UX 設計の常識から切り離した」点にある。前世代（Deus Ex, Dishonored, Prey）は、可読性の高い UI と明快な世界設定の上に複数解法を載せていた。Cruelty Squad は同じ複数解法を残しつつ、可読性そのものを攻撃対象にした。

| 設計要素 | 前世代の immersive sim | Cruelty Squad |
|---|---|---|
| UI | 機能的に整理された HUD | 蛍光色・歪み・点滅ノイズ |
| 死亡ペナルティ | リトライ／チェックポイント | 蘇生ループ＋株価変動と接続 |
| 装備獲得 | ミッション進行と物語連動 | 株式市場で買う／臓器を売る |
| 解法の正当化 | ステルス・非殺害が高評価 | 全員殺害も「合法」、誘導なし |
| 世界の語り | NPC ダイアログ + テキスト | 環境と UI そのものが寓話 |

social_axis がソロでも観戦消費を生んでいるのは、この「常識崩しの解法」が配信映像として強い差別化を持つからで、ソロ体験が二次的にバズ装置として機能している。

## バズ要因 / 人気の根拠

`popularity` の "Cult anti-aesthetic immersive sim; reframed the genre for the post-2020 indie scene." を分解する。

- Steam 売上は約 19.7M ドル相当、コピー販売 50 万本超と推定され、1 人開発のニッチタイトルとしては突出した規模に到達した。
- 「醜い UI / 過剰な色彩 / 不条理な世界観」を意図的な美学として確立し、post-2020 のインディー immersive sim（後続の Anthology of the Killer や同系シューターの語彙形成）に波及した。
- 配信・実況映像との親和性が高く、SNS で UI スクリーンショットと「株式市場で蘇生する」プレイループのスクリーン共有が定期的に拡散している。
- Consumer Softproducts（実質 Ville Kallio 個人）がパブリッシャーも兼ねており、価格・更新・別タイトル（Psycho Patrol R）への横展開を作家性主導で動かしている。

## 企画への示唆

- 既存ジャンルの「快適さの規約」を破ると、メカニクスは同じでも別ジャンルとして認識される。
- UI の醜さを「不具合」ではなく「世界観の語り手」に転用すると、テキスト量を増やさず寓話性を上げられる。
- 死亡を株価などのメタゲーム指標に接続すると、ロスがランダムウォークの一部に変換され、リトライ忌避が下がる。
- 解法に「正しい解」を設定しないと、プレイヤーの実況・スクリーンショットが多様化し、自然な UGC バズが起きやすい。
- 1 人開発でも価格・トーン・ビジュアルを作家性として一貫させれば、年間 1〜2 万本ペースのロングテールに乗せられる。

## 比較で見るべきタイトル

- Deus Ex (2000) — immersive sim の規範。Cruelty Squad が何を継承し何を破壊したかの基準点。
- Hitman: Codename 47〜WoA — 暗殺ターゲット制と複数解法の語彙を共有。
- ULTRAKILL / DUSK — post-2020 のレトロ FPS 美学リバイバル文脈。
- Anthology of the Killer / Mouthwashing — Cruelty Squad 以降の「醜さを意匠化する」インディー潮流。
- Disco Elysium — 単独・小規模作家性で世界観の語りを変形した同時代の例。
