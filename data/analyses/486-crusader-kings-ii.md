# クルセイダーキングス II / Crusader Kings II

## なぜ EXP 主分類なのか

このゲームのプレイヤーの主動機は「中世王朝の当主として家系を継承し続ける体験そのものを駆動する」ことにある。`social_axis: ソロ` が示すとおり、対戦や協力ではなく、シングルプレイヤーが自分の意思で生成した「ロールプレイの物語」をシステムから引き出すゲームだ。

ジャンルは Medieval grand strategy だが、4X や Civilization のように国家を最適化するゲームではなく、特定の人物（伯爵・公爵・王・皇帝）を操作し、その人物が死ぬと相続権を持つ次代に視点が移る。プレイヤーは国家ではなくダイナスティ（血統）を運営し、結婚・暗殺・宗教改宗・反乱・十字軍といった出来事のなかで「面白い物語が自然発生する」のを待つ。Paradox 自身が「stories engine」と呼ぶ設計で、`popularity` 文の「made dynastic roleplay a viable genre niche」が示すのは、報酬や物語を上から与えるのではなく、シミュレーションが副産物として吐き出すエピソードを楽しむ受容形式を成立させた点である。`target` 想定層は中世史・系図・ロールプレイに耐性のあるストラテジー愛好家で、勝利条件は明示されない。`secondary` が空なのは、収集（REW）や脚本化された物語（NAR）よりも、シミュレーションを回す体験そのものが圧倒的に主であることを反映している。

## 体験の核 / 設計の発明

CK2 の発明は、グランドストラテジーの粒度を「国家」から「人物」へ落とし、人物に属性・特性・人間関係を持たせたことにある。前作 CK1（2004）からの大きな差分は、(a) UI の使用に堪える刷新、(b) DLC による地理・宗教・文化の段階的拡張、(c) イベント密度の倍化である。

| 設計要素 | 機能 | 体験への効果 |
|---|---|---|
| 人物単位の継承 | 当主死亡で次代へ視点が移動 | 永続的な「目的」を強制し、世代を跨ぐ長期物語が生まれる |
| 特性・性格システム | 勇敢／臆病、貪欲／敬虔等のフラグ | NPC が一貫した行動原理を持ち、人格として記憶される |
| 王朝・血統スコア | 婚姻と継承で名声が累積 | 国家サイズではなく「家系の隆盛」が暗黙の勝利条件になる |
| イベントトリガ | 無数の条件分岐ポップアップ | プレイヤーがログを後から物語化できる素材を供給する |
| DLC レイヤ拡張 | 共和制／遊牧／インド／ムスリム／異教等 | 同じ地図でも文化ごとに別ゲームになる長期運営モデル |

`social_axis: ソロ` だが、プレイヤーは生成された物語をフォーラム・Reddit・配信で共有し、AAR（After Action Report）文化を生んだ。体験の核は「プレイ中」だけでなく「プレイ後に物語化して人に語る」までを含む。

## バズ要因 / 人気の根拠

`popularity`：「Paradox を戦略ジャンルで広く知られる存在に押し上げ、王朝ロールプレイをジャンルニッチとして成立させた」を分解する。

- 2019 年に F2P へ移行し、Steam での推定オーナー数は 600 万本超。base game を無料化し DLC で収益化する Paradox の運営モデルを定着させた
- 「馬の大公」「私生児が皇帝になる」「キャラ同士の歪な人間関係」など、プレイヤーが生成したエピソードが SNS・掲示板でミーム化し、ゲーム外の言説が新規流入を駆動した
- 12 年（2012–2024）以上にわたるサポートと十数本の major DLC により、購入後に止められないコンテンツ更新サイクルが形成された
- 続編 Crusader Kings III（2020）のヒットは CK2 で培ったコミュニティが土台で、Paradox の grand strategy 群（EU4、HoI4、Stellaris、Victoria 3）の集客導線として機能した

## 企画への示唆

- 国家ではなく人物を主語にすると、シミュレーションが自動で「キャラクター主導の物語」を吐き出すようになる
- 勝利条件を明示しないと、プレイヤーは「自分の家系の繁栄」を勝手に目標化し、自己駆動の長期セッションが成立する
- NPC に強い特性タグを持たせると、AI が一貫性のある奇行をするため、プレイヤーが記憶・命名・物語化する余地が生まれる
- base game 無料化 + DLC レイヤ販売モデルは、ニッチジャンルでも長期 LTV を確保する手段になる
- ゲーム外のユーザー生成コンテンツ（AAR・配信・ミーム）が新規流入の主導線になるなら、エピソードのスクショ・共有導線を設計に組み込む価値がある

## 比較で見るべきタイトル

- Crusader Kings III（2020）: UI と 3D 化で間口を拡げた直接の後継、ストーリーエンジン思想は同一
- Europa Universalis IV: 同じ Paradox の grand strategy だが、主語が国家に戻り、人物 RP の濃度は薄い
- Dwarf Fortress: 「事象を生成し、プレイヤーが物語化する」エンジン設計の哲学的祖
- RimWorld: コロニーシミュにキャラ特性 + イベント生成を組み合わせ、CK2 の「物語が勝手に生まれる」体験を別ジャンルで再現
- Mount & Blade II: Bannerlord: 中世＋封建制＋王朝継承を 3D アクションに翻訳したライバル系譜
