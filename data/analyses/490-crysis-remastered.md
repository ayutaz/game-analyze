# クライシス リマスター（Crysis Remastered, 2020 / Crytek）

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ナノスーツの能力（Armor / Strength / Speed / Cloak）を切り替えながら、ジャングルの広い交戦エリアを自分の戦術で攻略する」こと、つまり**戦う**ことそのものである。物語は北朝鮮軍とエイリアンを背景に置く SF ミリタリーだが、章ごとに開かれる地形・敵配置・索敵範囲がプレイの中心で、ストーリーは「次の戦闘エリアへの導線」として機能している。`social_axis: ソロ` の通り、判断と試行錯誤を完結させるのはプレイヤー単独であり、対戦・協力の文脈は持たない。

副次的に `NAR` を持つのは、Nomad を主人公とした直線的なミリタリー SF プロットがミッション間の動機付けを担っているため。ただし NAR は EXP を駆動するための薄い被膜であり、章間ムービー・無線指示が「次に何のためにジャングルへ降りるか」を補強する役割にとどまる。

## 体験の核 / 設計の発明

中核は **ナノスーツ 4 モードの即時切り替え**である。攻略の単位が「武器選択」ではなく「モード選択 × 地形」に置き換わっている点が、同時代の直線型 FPS との差分を生んだ。

| 設計要素 | 何を可能にしているか | EXP への効果 |
|---|---|---|
| Armor モード | 被弾を吸収しエネルギー消費で耐える | 正面突破ルートの選択肢化 |
| Strength モード | ジャンプ高度・近接威力・反動制御を強化 | 高所・遮蔽物を「破壊して進む」道に変える |
| Speed モード | 短時間の高速移動・近接の手数を担保 | 索敵→離脱の機動戦を成立させる |
| Cloak モード | 光学迷彩で発見距離を一時的に切る | 同一敵集団に潜入／側面攻撃／撤退の 3 解を与える |
| サンドボックス型ジャングル | 1km 超の交戦エリアを章単位で開く | 「攻略ルートが 1 本に収束しない」状態を維持 |

リマスターでは Ray Tracing、SSDO、テクスチャ高解像度化、`Can it run Crysis?` モードを実装し、2007 年版の構造を保ったままハードウェア負荷を再演している。**ゲームメカニクスを書き換えるのではなく、ベンチマーク文脈ごとリブートした**点が企画として特徴的である。

## バズ要因 / 人気の根拠

`Crysis (2007)` が PC ベンチマークの代名詞となった文脈をそのままリマスターが回収している。

- 2007 年の原版が「But can it run Crysis?」というミーム化したフレーズで PC ゲーミング史に固定され、ハイエンド GPU 評価の事実上の基準曲線になった。
- リマスターは Ray Tracing 実装と「Can It Run Crysis?」モードを搭載し、**13 年経っても重い**ことを売りに据える逆張りのプロモーション設計を取った。
- シリーズ累計 800 万本超という規模（`sales_world: Franchise 8M+`）を踏まえた上で、リマスター単体は Steam で 50 万〜100 万本規模に収まっており、AAA 復刻としては中規模に着地している。
- PC コミュニティ・ベンチマーク系メディア・GPU レビュアーが定常的に動画素材として使うことで、長期的な検索流入と再評価サイクルを生んだ。

## 企画への示唆

- **遊びの単位を「武器」から「モード」に上げると、同じ地形が複数攻略の場に化ける。**選択軸の粒度を 1 段抽象化することで、ステージ再利用率が上がる。
- **ハードウェア負荷をネガではなくフィーチャーとして売る**と、技術メディアが恒久的に取り上げる広告チャネルになる（ベンチマーク文脈の自家発電）。
- **リマスターはメカニクスを触らず、文脈ごと再生する**選択肢がある。原典が文化的アイコン化している場合、ゲームプレイ改修は逆効果になりやすい。
- **オープンレベル × 4 モードのような直交軸**は、3〜4 章で消費し切らないと飽きやすい。ボリュームと自由度のトレードオフを章ごとに設計する必要がある。
- **シングル特化の FPS でも、ソロ EXP として成立させるには「敵 AI の索敵半径」と「プレイヤーのステルス能力」の数値関係が体験設計の主軸**になる。能力設計ではなく検知設計を主にする発想が転用可能。

## 比較で見るべきタイトル

- Far Cry（2004, Crytek）— Crysis の祖型。ジャングル × オープンレベル FPS の原典。
- Deus Ex: Human Revolution / Mankind Divided — ステルス・突破・社会工学の 3 解を許す EXP 設計の系譜。
- Metal Gear Solid V: The Phantom Pain — オープンエリア × 能力切り替えで攻略軸を分岐させる発想の発展系。
- Half-Life 2 — 同世代の PC FPS ベンチマーク的存在で、技術アピールと体験設計のバランスが対照的。
