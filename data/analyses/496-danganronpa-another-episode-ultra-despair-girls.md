# 絶対絶望少女 ダンガンロンパ Another Episode

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ダンガンロンパ世界線の "間" を読み、本編 1 と 2 のあいだで何が起きていたのかを追体験すること」である。本作は『ダンガンロンパ』本編のサウンドノベル＋学級裁判という枠組みを離れ、三人称シューター（メガホン銃「ハッキング銃」）に持ち替えたスピンオフだが、ジャンル換装の目的はあくまで **苗木こまる＋腐川冬子の二人視点で語られる "絶望の街" の物語を駆動するため** の装置である。シューティング部分は探索・脱出・章間の演出として機能しており、撃ち合いの上達や報酬収集よりも、章をまたぐ謎の解け方／キャラクターの掘り下げ／本編との接続が体験の中心にある。

`concept`/`target` フィールドが空白のため厳密には推定だが、`genre` が "Action-Adventure / Narrative Spin-off" と明示され、`popularity` も "narrative spin-off; expands Danganronpa lore" と「lore 拡張」を主軸に置いている。これらは「物語消費の動詞」を指しており、REW（収集／育成）や EXP（撃ち合いの上達）よりも NAR が妥当である。`secondary` は未記載だが、機能的にはアクション要素が副次層として下支えしている。

## 物語の駆動構造

スピンオフが本編の物語消費をどう拡張しているかを、駆動レイヤーで分解する。

| レイヤー | 仕掛け | 物語推進への効果 |
|---|---|---|
| 視点キャラ | 苗木こまる（本編主人公の妹）＋腐川冬子（本編の超高校級の文学少女） | 本編主人公不在の状況下で「家族／読者代理」の二重視点を確保 |
| 時系列 | 本編 1 と 2 の間（希望ヶ峰絶望事件直後の "絶望の街"） | プレイヤーが既に知っている世界観の "空白" を埋める動機 |
| 章立て | 5 章＋幕間、各章ボス（モノクマキッズ五人衆） | キャラクター個別エピソード＝群像劇として読める |
| メカ翻訳 | ハッキング銃の弾種＝感情／命令（PARALYZE / KNOCKBACK / MOVE 等） | 戦闘行為が "謎解き＋キャラ感情の代弁" になり、撃ち合いが物語の文法に翻訳される |
| 接続装置 | 本編・続編キャラのカメオ、回想、書き下ろし設定資料 | "本編を遊んだ者だけが拾える伏線" を多層配置 |

ジャンル換装にもかかわらず NAR のままでいられるのは、「アクションメカニクスを物語のメタファーに従属させた」設計のため。撃つ＝命令を上書きする、という言語化が、サウンドノベル時代の "ことだま" 議論と地続きに置かれている。

## バズ要因 / 人気の根拠

`popularity` の「Steam とコンソールで Danganronpa の lore を広げた三人称シューター」を出発点に分解する。

- 累計売上はおよそ 50 万本規模（`sales_world` に "~500k worldwide" 記載）で、これは 2021 時点のシリーズ累計 500 万本のうち約 10% にあたる。本編より小さいがスピンオフとしては成立する規模。
- 元は PS Vita 専用（2014）として日本で先行、その後 PS4 / Steam に展開（本 JSON はマルチ展開後の `year:2017` を採用）。「Vita で遊んだ人にも、Vita を持たない海外勢にも届く」マルチ展開戦略が認知を底上げした。
- 本編 1→2 のあいだの "空白の年代史" を埋める唯一の公式正史であるため、シリーズファンの「全部触りたい」需要を確実に拾った。
- 主役を本編主人公ではなく「妹＋既存サブキャラ」に振った構成は、二次創作的な好奇心（本編で語られなかった視点）と相性が良く、配信／実況においても "ダンガンロンパ世界を別角度から見る" コンテンツとして消費された。

## 企画への示唆

- 既存 IP の "本編と本編の間" を切り出すスピンオフは、新規 IP より低リスクでファン需要を回収できる（前提：本編に時系列の空白があること）。
- ジャンルを大きく変える（ノベル → TPS）ときは、新ジャンルのメカニクスを **既存 IP の言語体系（本作なら "ことだま"／命令／論破）に翻訳** すると、ファンの違和感が拒否反応に転びにくい。
- 主役を本編主人公ではなく「主人公の家族 ＋ 既存サブキャラ」に置くと、本編との連続性と新規視点を両立できる。サブキャラの掘り下げが副次的な動機として効く。
- マルチ視点（こまる＋冬子の二人組）にすると、シリアス担当＋ボケ担当の役割分担で長時間の探索パートが冗長化しにくい。
- 「シリーズ累計の 10% 程度」というスピンオフのスケール感を設計目標に据えると、本編と食い合わない予算規模・期待値を逆算できる。
