# ディープロックギャラクティック (Deep Rock Galactic)

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「4 人のドワーフで手分けして洞窟を掘り、虫を撃ち、ピックアップに間に合うように脱出する」体験そのもの** である。物語（NAR）は背景として存在し、Ghost Ship Games が積み上げる季節制シーズン・新ボス・新バイオームのナラティブはあるが、それはプレイヤーが毎晩ボイスチャットで合流する理由を作るための舞台装置でしかなく、ストーリーを読み進めるために起動する人はほぼいない。

`social_axis: 協力` がここでは支配的で、4 クラス（Scout / Gunner / Engineer / Driller）の **役割が地形操作と垂直性によって物理的に絡み合う** ことが核になっている。Scout がグラップリングフックで天井のミネラルを撃ち、Engineer がプラットフォームを生成して足場を作り、Driller が壁を掘り抜いて帰路を作り、Gunner がジップラインで全員を引き上げる ── この **「協力しないと地形に殺される」設計** が、4 人いれば 4 人それぞれの仕事がある状態を生み出している。

secondary に NAR が付いているのは、シーズンごとの新敵・新ロアと、ドワーフ文化（"Rock and Stone!" コール、酒場 Abyss Bar、ピザ）が継続プレイの理由として副次的に効いているため。

## 体験の核 / 設計の発明

| レイヤー | 仕掛け | 何を生んでいるか |
|---|---|---|
| 地形 | 完全破壊可能な手続き生成ケイブ | 毎回違う洞窟、Driller の掘削に意味が出る |
| クラス | 4 クラスの非対称ツール（フック / プラットフォーム / 掘削 / ジップ） | 移動・地形操作で役割が物理的に分かれる |
| ミッション | Mining / Egg Hunt / Escort / Salvage 等の Mission Type | 同じマップ生成でも目的で動きが変わる |
| 危険度 | Hazard 1-5 + Deep Dives | 自分の練度で難易度を選べる、同卓内でも揃えやすい |
| 文化 | "Rock and Stone" / ビール / ピザ | ボイチャ無しでも仲間感が出る低コスト儀式 |

前世代の Left 4 Dead / Vermintide が「横スクロール的に一本道を走る協力 FPS」だったのに対し、DRG は **空間が 3 次元に開いて掘削で書き換わる** ことで、ルートそのものをプレイヤーが作る。これが「同じ COOP シューターを 200 時間遊ぶ」リテンションを支えている。

## バズ要因 / 人気の根拠

popularity の "the gold-standard hangout-co-op shooter of the decade" を分解すると：

- **2023 年時点で 8M+ プレイヤー**（sales_world）。Game Pass 配信と Steam セール常連で長期に裾野を伸ばし続けた
- **"Rock and Stone!" ミーム** がコミュニティ標準のあいさつになり、Twitch / Reddit / Discord の入口として機能している
- **シーズン制アップデートが無料配布**（バトルパスも全コンテンツ恒久解放）で、新規参入時に「過去シーズンが壁にならない」設計
- 開発元 Ghost Ship Games が小規模スタジオを保ち、**派生作 Deep Rock Galactic: Survivor / Rogue Core を別チームで展開** することで IP を疲弊させずに横展開している

## 企画への示唆

- **地形を破壊可能にすると、プレイヤーが空間そのものを成果物として扱う** → 完成形の地形が毎回違うので、同じマップで飽きが来づらい
- **4 クラスの移動手段を「それぞれが他クラスを助ける」形に直交させる** → 役割分担が会話せずとも自然に発生し、ボイチャ無しでも協力感が出る
- **協力 PvE では「儀式的なコール（あいさつ / 称賛）」を 1 個ゲームに仕込む** → コミュニティが外に広めるミームを内製できる
- **バトルパスを期限後に恒久解放する** → 復帰プレイヤーが疎外感を抱かず、長期 DAU が積み上がる
- **本編は破壊しないまま、派生ジャンルを別チームで展開する** → IP の鮮度を保ったまま、Survivor / Roguelike など別の遊び方で別レイヤーの新規を取れる

## 比較で見るべきタイトル

- **Left 4 Dead 2 / Back 4 Blood** — 一本道型 4 人協力 FPS の前世代、地形が固定であることとの比較
- **Vermintide 2 / Darktide** — Tide シリーズ、4 クラス非対称協力の近縁
- **Helldivers 2** — 2024 年の協力 PvE 文化現象、惑星規模のメタ進行で DRG の儀式設計を別軸で発展させた
- **Risk of Rain 2** — 同じ Coffee Stain 配下、4 人協力 + 手続き生成の系譜
- **Lethal Company** — DRG 文脈を継ぐ 2023 年の "hangout 協力 PvE" 現象
