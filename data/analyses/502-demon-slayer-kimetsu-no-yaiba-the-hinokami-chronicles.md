## なぜ REW 主分類なのか

このタイトルでプレイヤーが繰り返し起動する主動機は「鬼滅の刃のキャラクターを操作可能な手駒として揃え、強化し、彼らで戦う」ことに置かれている。3D アリーナファイティングというジャンルの皮を被ってはいるが、CyberConnect2 が長年積み上げてきたナルト『ナルティメットストーム』系の設計を踏襲しており、アーケード対戦の腕試しよりも、原作の主要場面を「自分のロスター」で再演することがコアループになっている。アニメ放送と同期した IP 消費の文脈で買われたタイトルである以上、購入後にプレイヤーが期待するのは戦闘システムを極めることではなく、推しキャラの解放・コスチューム・必殺技演出を一通り「集め切る」ことであり、ここに REW の重力が働いている。

副次的に EXP が効いているのは事実で、2vs2 のタッグ戦と各キャラ固有の血鬼術アクションは触れて気持ち良い手触りを供給する。ただし対戦ゲームとしてのメタゲームやコミュニティ競技性は浅く、戦闘体験は「集めたキャラを試走するための場」として REW を支える役割に留まる。

## 報酬ループの組成

3D アリーナファイティングを REW として解剖すると、ループは三層に分かれている。

| 層 | 報酬の種類 | プレイヤーへの効用 |
|---|---|---|
| 章モード（無限列車編まで） | キャラ解放 + 場面回想 | 原作未読層への入口、IP 体験の完走感 |
| ヴァーサスモード | 全キャラ使用権 + 技演出 | 推しキャラを実際に操作する満足 |
| コスチューム / カラー / 称号 | 装飾コンテンツ | コレクション欲求の継続供給 |

章モードが「原作の追体験」という入場券を売り、ヴァーサスとカスタマイズが「IP との接触時間を伸ばす」役割を担う。F2P モバイルガチャのような確率ループは存在しないが、構造はコンソール買い切り版の収集型 IP ゲームに典型的なもので、「全キャラ解放までの所要時間 × アニメ放送同期の話題性」がリテンションを規定している。CyberConnect2 はこの設計を 2010 年代から繰り返し最適化しており、ストーリー → 解放 → 演出鑑賞という導線は鬼滅の刃の章構成と相性が良い。

## バズ要因 / 人気の根拠

popularity は「Demon Slayer の記録的 IP を Steam に持ち込んだアニメファイター大ヒット」と要約されている。これを分解すると：

- アニメ『鬼滅の刃』が劇場版『無限列車編』で日本歴代興収 1 位を更新した直後（2021）のリリースで、IP 露出のピークと発売タイミングがほぼ重なった。
- Steam で 16.5K レビュー / 87% 好評、世界累計 3M 本超という規模に到達。日本産アニメ IP の格闘ゲームが Steam で 300 万本級に届いた事例は限られる。
- CyberConnect2 が培ったナルト『ナルティメットストーム』系演出が「原作の決め技」をそのまま動かす期待値を裏切らなかった。
- PC（Steam）に投下されたことで、コンソール中心のアニメ IP 市場に対し PC 洋圏ファンへのリーチが追加された。

## 企画への示唆

- アニメ放送・劇場版のピーク半年〜1 年以内にリリースを置くと、IP 想起と購入動機が一致して売上が前倒しで立つ。
- 戦闘システムの深さより「決め技演出の再現精度」を優先すると、競技勢ではなく原作ファンの満足が最大化される。
- 章モードを原作のクライマックスで切り、続編 DLC で次章を売る運用にすると、アニメの放送スケジュールに合わせて再点火できる。
- コスチューム / カラーをガチャではなく解放型にすると、F2P モバイルの世界観と被らず、買い切り IP ゲームとしてのブランドを守れる。
- 開発を「アニメ IP ファイター職人スタジオ」（CyberConnect2 / Arc System Works）に発注すると、演出資産の流用で開発期間と品質の両立が現実的になる。
