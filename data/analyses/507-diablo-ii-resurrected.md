# ディアブロ II リザレクテッド (2021)

## なぜ REW 主分類なのか

プレイヤーの主動機は「より良いアイテムを集め、ビルドを完成させる」こと、つまり**集める**である。2000 年のオリジナル版が確立した「ハック＆スラッシュ → ドロップ → ビルド再構築」の三段ループを 4K テクスチャと現行コンソール対応で再パッケージしたのが本作で、ゲームプレイのデータ構造そのものはほぼ凍結されている。`concept` 上は ARPG リマスターだが、`target` は「シリーズ既存ファン＋現代の Path of Exile / Diablo IV 世代に旧作体験を提供する」層であり、両層ともプレイの目的は **Unique / Set / Runeword アイテムの収集と、それを土台にした 7 クラス × 多ビルドの育成** にある。物語は短く章立ても薄く、本編は 4 ACT＋拡張 1 ACT を Normal → Nightmare → Hell の三周ループする構造で、ストーリー消費よりも「Hell 難易度で何が落ちるか」が継続動機を支配する。よって主分類は REW、secondary に EXP が入るのは、操作精度・スキル選択・モンスター密度といった戦闘体験そのものも遊びの核を成しているためで、Diablo II は「報酬を取りに行く手触り」自体が報酬構造の不可分な一部になっている稀なタイトルである。

## 報酬ループの組成

| 層 | 内容 | 体感サイクル |
|---|---|---|
| 短期 | モンスター撃破 → 即時ドロップ（金 / Gem / 雑魚 Magic 装備） | 数十秒 |
| 中期 | ボス・スーパーユニーク周回 → Rare / Set / Unique 入手 | 10〜30 分 |
| 長期 | Runeword 素材集め、Hell Cow Level 周回、Magic Find ビルド転向 | 数十〜数百時間 |
| メタ | ラダーシーズン更新でのリセット + 新 Runeword 投入 | 半年単位 |

設計上の要点は三つ。第一に **アイテム etymology の階層化**（White → Magic → Rare → Set → Unique → Runeword）が一目で価値序列を伝え、ドロップのたびに「上位 tier かどうか」を即時判定できる。第二に **Magic Find ステータス**を独立パラメータとして露出させたことで、装備＝戦闘力ではなく「装備＝次の装備を引く確率」へと役割が二重化し、収集自体が自己強化ループを形成する。第三に **オフライン非対応のラダー制**で、シーズン区切りに資産がリセットされるため、長期プレイヤーが「もう一周する理由」を運営側から供給できる。リマスター版でこれらは温存され、追加されたのは UI / グラフィック / クロスプログレッションのみで、報酬経済そのものは触っていない。

## バズ要因 / 人気の根拠

- 発売 7 ヶ月未満で全世界 500 万本超を販売（2022 年 4 月時点、`sales_world`）。リマスター作品としては Blizzard 自社内でも上位規模。
- オリジナルがジャンルを定義したタイトルであり、Path of Exile / Last Epoch / Diablo III・IV といった後続 ARPG のプレイヤーが「原典に戻る」需要を抱えていた。
- 旧グラフィック ⇄ 新グラフィックを **TAB キーで瞬時切替**できる演出が話題化し、配信・SNS でビフォーアフター比較が拡散した。
- ラダーシーズンの再開で旧来コミュニティ（取引フォーラム、ビルド計算機、Path of Diablo 系 MOD コミュニティ）が再活性化し、ストリーマー周回が再開された。

## 企画への示唆

- **報酬経済を凍結したままリパッケージすると、長期プレイヤーの資産設計が損なわれず復帰しやすい** — 触るべきは UI / 解像度 / 入力レイヤーで、ドロップテーブルと数式は触らない方が信頼を失わない。
- **新旧グラフィック切替ボタンを残すと、リマスターそのものが SNS 拡散素材になる** — 比較スクリーンショットの生産コストを 1 キーまで下げる UI が拡散効率を決める。
- **ラダーシーズン制をリマスターに継承すれば、運営は本編に手を入れずに「再開理由」を半年ごとに配れる** — シーズン報酬とアイテム再投入はメタ層のコンテンツとして極めて低コスト。
- **アイテム tier の色分けを抽象化したまま温存すると、20 年前のプレイヤーと新規が同じ語彙で会話できる** — Unique / Set / Runeword という固有名詞は世代を越えるブランド資産。
- **「物語 = 短いお膳立て、収集 = 長い本編」と割り切ったゲームは、難易度ループ＋シーズン制で 20 年売れる** — ナラティブ拡張に投資せず、ドロップ確率の調整だけで延命できる構造を最初から組むという選択肢。

## 比較で見るべきタイトル

- Path of Exile（2013、PoE）— D2 の精神的後継。Currency Orb 経済とアクト構造を引き継ぎつつスキルジェムで分岐させた。
- Diablo III（2012）/ Diablo IV（2023）— 同シリーズ後続。Paragon / Greater Rift / シーズンと、報酬ループの現代化方向。
- Last Epoch（2024）— Affix 強化システムでアイテム工芸を前面化した最新世代 ARPG。
- Grim Dawn（2016）— Titan Quest 系譜のオフライン重視 ARPG。D2 直系の質感を維持する代替候補。
