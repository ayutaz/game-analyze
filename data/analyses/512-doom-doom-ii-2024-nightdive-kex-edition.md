## なぜ EXP 主分類なのか

プレイヤーの主動機は「30 年前のレベルデザインを現代のハードと操作感で**走り抜けて撃つ**」ことに尽きる。Kex Edition は 1993/1994 年のオリジナル WAD をそのまま動かしつつ、4K / 120fps / 広視野角 / クロスプラットフォーム協力 / MOD ブラウザを上に被せた「保存と上書きの両立」レイヤである。物語（NAR）は最低限のテキストブリーフィングのみ、報酬（REW）も新エピソード Legacy of Rust の武器追加に留まり、ループの中心は依然として「部屋に入る→敵が湧く→走って撃って鍵を拾う」という秒単位の体験そのものにある。social_axis は「ソロ」だが、co-op / Deathmatch の追加サーバーが副次的に効く。secondary に NAR / REW を置いているのは、Legacy of Rust（id+Nightdive+MachineGames の新規 18 マップ）が章立てと新銃 2 種を導入し、わずかに物語と収集の薄皮を被せているためである。

## 体験の核 / 設計の発明

Kex Edition は「オリジナル EXE の体験を 1 ピクセルもずらさず、外側のレイヤだけ 2024 年水準に持ち上げる」構造を取る。下表のように、コア（マップ / AI / 武器バランス）は触らず、シェル（描画 / ネット / MOD 配信 / 入力）だけを差し替えている。

| レイヤ | 1993/94 原典 | 2024 Kex Edition | 効果 |
|---|---|---|---|
| 描画 | 320×200 / 35Hz | 4K / 最大 120fps / ワイドスクリーン | 視認情報量↑、ピクセル単位の回避が可能に |
| 入力 | キーボードのみ前提 | デュアルスティック・ジャイロ・マウスルック | 速度上限が腕前で決まるようになる |
| ネット | LAN / シリアル | クロスプラットフォーム co-op / DM (最大 16 人) | ソロ EXP に非同期/対戦軸が同居 |
| コンテンツ | IWAD 固定 | 公式 MOD/WAD ブラウザ内蔵、Legacy of Rust 同梱 | ループが終わらない（外部供給で延命） |
| 物語 | テキスト幕間 | 新エピソード 18 マップ + 2 武器 | secondary に NAR/REW を薄く積む |

差分は「コアに触らないこと」にこそ価値がある設計で、保存型リマスターのテンプレートとして機能する。

## バズ要因 / 人気の根拠

popularity の 1 文（id+Nightdive+MachineGames の Kex エンジン版で Legacy of Rust 新エピソードを同梱した、現代版クラシック Doom の決定版）を分解すると以下になる。

- 既存の Doom (2019) 版 Doom / Doom II 所有者へ**無料アップグレード**として配布され、初動の同時接続者を一気に押し上げた。
- Steam owners は 500K–1M レンジに乗り、クラシック FPS リマスターとしては上位。
- 開発体制が id Software（原典）/ Nightdive（Kex エンジン）/ MachineGames（新エピソード）の三社合議という珍しい構成で、発表時点でメディア露出が広範囲に走った。
- 公式 MOD ブラウザ搭載により、Boomer FPS / DOOM MOD コミュニティ（Brutal Doom, MyHouse.wad 等）の話題が常時供給され続けている。

## 企画への示唆

- **コアを凍結しシェルだけ差し替えると、過去 IP は新規ハード世代ごとに再販売できる**（解像度・入力・ネットの 3 層）。
- **既存購入者への無料アップグレード**は初動 DAU を即座に作る代わりに、新規ペイドユーザー数を犠牲にする。話題化と物販を分けて評価する設計が要る。
- **MOD ブラウザを公式に内蔵**すると、ライブサービス的なコンテンツ供給コストを外部化できる。秒単位のループがあるゲームほど効く。
- **原典開発元 + 移植スタジオ + 続編スタジオの 3 社共作**は、保存（authenticity）と新規制作（freshness）の責任分界を綺麗に切れる。クレジットそのものがマーケになる。
- **新エピソードを「同梱」して別売しない**ことで、リマスター本体の価値訴求を厚くできる。DLC 切り出しより総体評価が安定する。

## 比較で見るべきタイトル

- Quake (2021 Nightdive Remaster) — 同じ Kex エンジン・同じ無料アップグレード戦略の先行例。
- Quake II (2023 Nightdive Remaster) — MachineGames が新エピソードを書き下ろした直接の前例。
- Doom 64 (2020) — Nightdive による初期の Kex 移植、保存志向の原点。
- Half-Life (Valve, 2020 25 周年アップデート) — IP オーナー自身が原典をモダン化するパターンとの比較対象。
- System Shock (2023 Remake) — Nightdive がコアごと作り直した対極の手法、保存ではなく再構築。
