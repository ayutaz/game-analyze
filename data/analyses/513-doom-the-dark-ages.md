# DOOM: The Dark Ages — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「中世風の鎧を着たスレイヤーで重装兵として殴り合う」——つまり**戦闘の手触りを味わう**ことにある。social_axis は「ソロ」、ジャンルは Single-player FPS、concept にあたる popularity 文も「shield-saw mechanic と slower combat が id の戦闘公式をリフレーミングする」と明記しており、軸は完全に体験側に振れている。

DOOM (2016) / DOOM Eternal が確立した「動き続けて弾薬を殴り取る高速ダンス」を、本作は意図的に減速させ、**シールドを軸にしたパリィ・ブロック・近接の地上戦**へ作り替えている。プレイヤーが手に入れるのは新しい物語消費体験ではなく「鈍重なのに気持ちのいい殴り合い」という新しい筋肉の使い方であり、これは典型的に EXP（体験型）として扱う。

secondary に NAR を置いているのは、prequel として「Slayer の起源」を描く章立てとカットシーン量が前2作より厚く、Argent D'Nur／Maykrs の神話に踏み込む語り口が副次的な動機として機能するため。ただし物語は戦闘ステージを連結する装置であり、主従関係は EXP > NAR で動かない。

## 体験の核 / 設計の発明

本作は前作のメカニクスを一旦壊し、要素を入れ替えるかたちで再構築している。中核を分解する。

| レイヤー | DOOM Eternal (2020) | DOOM: The Dark Ages (2025) |
|---|---|---|
| 移動 | 空中ダッシュ・グラップル中心の縦運動 | 重量ある地上機動、シールドチャージで突進 |
| 攻撃の核 | チェーンソー / グローリーキルで資源を回収 | シールドソーでブロック→パリィ→近接の三拍子 |
| 敵処理ロジック | 弱点別に武器をスワップするパズル | 飛来弾を見切ってパリィ、ガード硬直で反撃 |
| 大型遭遇 | アリーナでの空中乱戦 | メック搭乗・ドラゴン騎乗の章単位の演出戦闘 |
| トーン | サイバー / ヘル混成のメタル | 中世ゴシック＋ヘヴィメタルの prequel 美術 |

設計上の発明はシールドソーを「防御 + 移動 + 攻撃」の三役にまとめた点で、Eternal が要求した武器スワップの認知負荷を、**パリィのタイミング判断ひとつ**に圧縮している。social_axis がソロに固定されているのも、この入力反射ループを純粋に磨くため。マルチ要素は意図的に外され、体験は「自分の反応速度と戦場の読み」のみに帰着する。

## バズ要因 / 人気の根拠

- 初週で **300 万プレイヤー** に到達し、id Software 史上最大のローンチ規模となった（売り切りに加え Xbox Game Pass / PC Game Pass 同梱が DAU を底上げ）。
- 一方で paid copy は ~80 万本と推定され、Game Pass モデルに移行した Microsoft 第一陣タイトルとして「**プレイヤー数 vs 売上の乖離**」が議論された代表例になっている。
- 中世＋メック＋ドラゴンというビジュアルが既存の DOOM ファン層を超えてストリーマー受けし、ゴア表現と重量感のあるシールドアクションが配信映え素材になった。
- Eternal 期に分かれた「コンボ系シューター好き / 旧 DOOM 好き」の議論を、減速とパリィで一度合流させ直した点が言説上のフックになっている。

## 企画への示唆

- **続編で機能追加ではなく機能差し替えにする**と、シリーズ疲労を抜けつつ既存の世界観資産を温存できる（Eternal の縦運動を捨て、横の地上戦へ）。
- **シールドのような単一ガジェットに防御・移動・攻撃を集約する**と、戦闘の認知負荷が下がり、入門者と熟練者の両方に同じレバーで深さを提供できる。
- **prequel 構造を採用する**と、シリーズ世界の起源描写が物語のフックになり、本編シリーズの前提を知らない新規も入り口を持てる。
- **サブスクと売り切りを併走させる**場合、初週プレイヤー数と paid copy を**別 KPI として設計**しないと、評価会議で数字の解釈が割れる。
- **ソロ専用に振り切る**と、サービス運営コストを切り離せる代わりに、リテンションは DLC / 章追加に集約することを最初から設計しておく必要がある。

## 比較で見るべきタイトル

- DOOM Eternal (2020) — 前作。本作の「減速」が何を捨てたかを測る基準線。
- DOOM (2016) — 三部作の起点で、グローリーキル中心の戦闘ループ。
- Wolfenstein: The New Order (2014) — 同じ MachineGames / Bethesda 系列のシングル FPS、物語駆動の比較対象。
- Warhammer 40,000: Space Marine 2 (2024) — 重装兵によるソロ／協力シューターで「重量感のある近接＋射撃」の同時代比較。
- Lords of the Fallen (2023) — 中世ダークファンタジー × パリィ駆動戦闘という、トーンとメカニクスの隣接点。
