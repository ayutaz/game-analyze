# Dusk (2018) 分析

## なぜ EXP 主分類なのか

Dusk のプレイヤーが動かしている動詞は明確に「撃つ・走る・跳ぶ」であり、物語や収集ではなく **高速で動き続けるシューティング体験そのもの** が主動機になっている。social_axis が「ソロ」であることは、対戦・協力に頼らず **AI 敵密度とレベルデザインだけで体験を成立** させていることを意味する。1990 年代の Quake / Blood の DNA を、David Szymanski 個人開発というスケールに圧縮しなおした作品で、プレイヤーはマルチプレイ抜きで「素早い射撃と移動の手応え」を味わうために起動する。

secondary に NAR が入っているのは、東欧風カルト・コズミックホラーの雰囲気テキストや環境物語が体験を底上げしているため。ただし筋を追う作品ではなく、物語は **撃ち続ける動機としての世界観装飾** に留まる。

## 体験の核 / 設計の発明

| 設計レイヤー | Dusk の選択 | 効果 |
|---|---|---|
| 移動 | 低摩擦のスライディング、ストレイフジャンプ許容 | 撃ち合い前に位置取りを楽しめる |
| 武器 | 二刀流ショットガン / リベットガン / 剣など物理感の強い ローテク武器 | リロード待ち時間ゼロで撃ち続けられる |
| 敵設計 | ヒルビリー、カルト信者、内臓モンスターを混在配置 | 距離別に複数武器を切り替える意味が生まれる |
| ビジュアル | PS1 級ローポリ + 低解像度テクスチャ | 描画負荷が低く 144Hz でも 60Hz でも滑らかに撃てる |
| レベル | 廃工場・カルト村・地下迷宮の 33 マップ | 学習→突破→新ギミックのリズムが章ごとに刻まれる |

Quake 系の「fast-FPS」が AAA から消えていた 2010 年代後半に、**ローポリ表現で開発コストを抑えつつ操作応答性だけは現代水準** という配分にした点が発明的で、後続の Amid Evil / Ultrakill / Prodeus が踏襲する型を定義した。

## バズ要因 / 人気の根拠

- Steam owners 50 万〜100 万本規模で、インディー FPS としては突出した到達。
- 2018 年の Steam レビューで「Overwhelmingly Positive」を維持し、口コミで広告費を肩代わりさせる構図。
- New Blood Interactive がブランドとして信頼を獲得するきっかけになり、Amid Evil / Gloomwood の販路を切り開いた。
- John Romero 系の元 id Software 人脈や FPS YouTuber 層が継続的に推奨し続け、boomer-shooter というジャンル名そのものを定着させた。

## 企画への示唆

- **AAA が空けた席を狙うと、技術的負荷の低いジャンルでも復興市場が立ち上がる**：Quake 系を 2018 年に再提示しただけで需要を顕在化させた。
- **ローポリは「妥協」ではなく「武器」に転換できる**：描画負荷を下げると操作応答が上がり、ジャンル本質（撃ち心地）に予算を集中できる。
- **個人開発でも 30 マップ以上を組めば、価格に対する物量説得力が生まれる**：体験密度はマップ数 × ギミック数で測られる。
- **物語は「装飾」として配置すると主動機を邪魔せず雰囲気だけ上げられる**：NAR を secondary に留める設計判断。
- **パブリッシャー（New Blood）がジャンル旗艦になると後続作の販路が開く**：1 本目を「ジャンルの基準」に据えると 2 本目以降のマーケ費が下がる。

## 比較で見るべきタイトル

- Ultrakill（2020、Arsi Patala / New Blood）— Dusk の後継、テンポをさらに加速。
- Amid Evil（2019、Indefatigable / New Blood）— ファンタジー版 boomer-shooter。
- Quake（1996、id Software）— 起源にあたる fast-FPS の原典。
- Prodeus（2022、Bounding Box Software）— モダンライティング × レトロ感覚のハイブリッド比較対象。
