# ELEX

## なぜ EXP 主分類なのか

ELEX を遊ぶプレイヤーの主動機は「ジェットパックを背負って、SF と中世ファンタジーが衝突したオープンワールドを物理的に踏み分ける」ことにある。Piranha Bytes 製 RPG の系譜（Gothic / Risen）と同じく、地形が単純に通行可能 / 不可能で分かれておらず、敵レベルが地域ごとに固定されているため、プレイヤーは「弱い自分が強い土地を歩いて回る」体験そのものを楽しむ。社会軸は「ソロ」で、誰かとの関わりではなく、世界そのものとプレイヤーの摩擦が中核に据えられている。

物語（NAR）は副次的に効いており、Berserkers / Clerics / Outlaws の三派閥のどれに与するかが進行を分岐させる。ただし物語消費が主動機ではなく、派閥選択はあくまで「どの地形・どの装備系統で世界を踏破するか」という体験面の選択肢として機能する点が、本作を NAR ではなく EXP に置く根拠になる。

## 体験の核 / 設計の発明

ELEX の設計を構成要素に分解すると以下のレイヤーに整理できる。

| レイヤー | 中身 | プレイヤー体験への効果 |
|---|---|---|
| 移動 | ジェットパックによる垂直探索 | 崖・塔・尾根が「壁」ではなく「ショートカット」に転化する |
| 地形 | レベルスケーリングなしの帯状危険度 | 「行けない場所」が常に視界内にあり、成長の方向が地理的に可視化される |
| 派閥 | Berserkers（魔法）/ Clerics（エネルギー兵器）/ Outlaws（火薬） | 戦闘ビルドと装備調達ルートが土地ごとに差別化される |
| 倫理 | 感情を抑制する物質 Elex の摂取量 | 会話選択肢と一部スキルが解放され、ロールプレイ軸が変動する |

前世代の Piranha Bytes 作品（Gothic / Risen）との差分は「縦方向移動の常時可用化」に集約される。中世ファンタジーの地形パズルを SF ガジェットで踏み越える構図は、後発のオープンワールドにも反復可能なモジュールになっている。

## バズ要因 / 人気の根拠

popularity の原文「Piranha Bytes' post-Risen original IP; jetpack-equipped Eurojank RPG with a loyal following」を分解すると以下の柱に整理できる。

- Piranha Bytes の Gothic / Risen 経由のコア層が、新規 IP でも継続的にプロジェクトを支持した
- Steam owners 推定 50 万〜100 万本という規模は AAA ではないが、ニッチ RPG として継続的に話題化されるロングテール圏に入っている
- 「Eurojank」と呼ばれる、洗練度より独自性を優先する欧州 RPG 文化圏のシンボル的な一本として参照され続けている
- SF とファンタジーを単純な世界観融合ではなく「派閥のテクノロジー観の対立」として処理した点が、レビュー記事や RPG コミュニティでの語り種になっている

## 企画への示唆

- 縦方向移動ガジェットを序盤から無制限に渡すと、オープンワールドの「視界＝目標」ループが地形パズルとして強化される
- レベルスケーリングを廃して敵の強さを地理に固定すると、プレイヤーの成長曲線が「数値」ではなく「地図上の塗りつぶし」として可視化される
- 派閥選択を会話分岐ではなく装備系統・移動系統・スキル系統に紐づけると、物語選択がそのまま体験設計の分岐になる
- 世界観のジャンル衝突（SF × ファンタジー）を背景設定で済ませず、派閥の技術観として表現すると、プレイヤーが衝突の論理を体験的に把握できる
- 洗練を捨てて独自性を取ると、規模は小さいが熱量の高いコア層が継続的に作品を支える経済圏が成立する
