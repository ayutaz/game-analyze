# 鋼鉄の追憶 シャドウ・トーチ / F.I.S.T.: Forged in Shadow Torch

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ディーゼルパンクな機械都市を、巨大なパワーグローブで殴り抜けながら探索する」体験そのものである。物語（人間と動物のディストピア寓話）も収集要素（HP/SP 拡張、武器強化）も存在するが、それらは戦闘と探索の手触りを補強するための裏付けであり、プレイヤーが何時間もコントローラを握る理由ではない。

ジャンルは Metroidvania、social_axis は「ソロ」。他者と関わる余地は基本的にない閉じた世界で、プレイヤーは「うさぎの主人公が機械の腕を振るう」運動感覚を獲得していく。secondary に REW が入っているのは、武器三種（拳・ドリル・鞭）の段階解放と能力強化ツリーがメトロイドヴァニア典型のゲートロックと密接に絡んでいるためで、これは中華インディーが伝統的にトロフィー的に積み上げる「コレクション系の安心感」を残した設計と読める。だが報酬は探索動機の燃料であって、終着点ではない。

## 体験の核 / 設計の発明

中核は「3 種の重火器を切り替えるアクション」と「ディーゼルパンク × 中華 IP の美術」のレイヤーが重なった点にある。前世代の 2D メトロイドヴァニア（Hollow Knight / Ori / Bloodstained）と比べると、F.I.S.T. はキャラクターと武器のスケール差を強調する方向に寄せている。

| 要素 | F.I.S.T. の選択 | 前世代の標準 |
|---|---|---|
| 主人公スケール | 小柄なうさぎ + 自身より大きい機械腕 | 等身大の剣士・呪われた人形 |
| 武器系統 | 拳（近距離高速）/ ドリル（突進）/ 鞭（中距離面） | 単一武器 + サブウェポン |
| 美術方向 | ディーゼルパンク + 動物擬人化 + 中華近代 | 暗黒ファンタジー / 神話 / ゴシック |
| マップ構造 | エリア間メトロ移動 + 各区画にハブ | ゲートロック型大マップ単一 |
| 難度設計 | 普通～歯ごたえあり、ボス処刑型ではない | Hollow Knight 系の即死リズム |

ソロ社会軸を採るアクション・探索型は「武器バリエーション × マップ進行」の掛け算で時間あたりの新規体験量を確保する必要があり、F.I.S.T. は武器 3 種それぞれに移動アビリティを紐付けた（鞭で振り子移動、ドリルで掘削など）ことで、戦闘トリガーと探索トリガーを単一資源で兼用させた。

## バズ要因 / 人気の根拠

popularity 文「中華発ディーゼルパンク・メトロイドヴァニア、上り調子の中華インディーシーンの代表格」を分解すると：

- 中国のインディーが家庭用機・PC のグローバル舞台で評価された比較的早期の事例で、後の Nine Sols / 黒神話：悟空 等の流れの中で参照される位置にいる
- Steam の推定オーナー数は 50 万〜100 万本レンジ、ジャンル・新規 IP・新興スタジオの 3 条件下では十分な規模
- PS5 ローンチ近辺で PS Plus フリープレイ枠に入ったことが認知拡大に寄与したと広く知られている
- ディーゼルパンク × 動物擬人化 × 中華近代史的モチーフという美術の組み合わせ自体が珍しく、スクリーンショットだけで認知される

## 企画への示唆

- 武器切り替えに「戦闘技」と「移動アビリティ」を同時にバインドすると、ジャンル定数である「武器追加」と「探索ゲート解除」を一資源で二重消費でき、開発コストに対する遊びの密度を高くできる
- 動物擬人化の主人公を採用するとスケール差の演出（小さい主人公 × 巨大武器）が成立し、戦闘の見映えがフレーム単価以上に強く出る
- 新興地域のスタジオがグローバルで認知されたい場合、土地固有の美術文脈（中華近代・ディーゼルパンク）を尖らせる方が、AAA 風グラフィックを目指すよりコスト効率が高い
- メトロイドヴァニアで social_axis をソロに閉じる場合、SNS でシェアされる「絵が立つ瞬間」をボスや特殊移動アクションに集中設計すると配信動画経由の発見が増える
- 武器 3 種程度の有限セットに絞り、それぞれにキャラクター性（重さ・モーション・SE）を付与する方が、武器が 20 種以上ある「装備品」設計より体験の核を作りやすい
