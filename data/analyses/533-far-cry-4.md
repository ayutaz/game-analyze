# ファークライ4

## なぜ EXP 主分類なのか

プレイヤーの主動機は「ヒマラヤ山岳国家 Kyrat のオープンワールドで、武器・乗り物・動物を組み合わせて遊撃戦の体験を組み立てること」である。物語と収集要素も濃いが、プレイヤーがセッション中に最も時間を割くのは前哨基地の解放・象に乗っての侵入・自動小銃と弓のスイッチング・捕食動物の乱入を利用した三つ巴戦闘の即興であり、`primary: EXP` がふさわしい。

`social_axis: ソロ` の通り Co-op はあるもののメインの動線は単独プレイヤーが Kyrat 全土を踏破するソロ FPS であり、Pagan Min を巡る政治劇は「動機を与える額縁」として機能する。`secondary: [NAR, REW]` は妥当で、NAR は Vaas 系譜のカリスマ悪役による物語牽引、REW は皮剥ぎ・植物採取・スキルツリー・武器アンロックという成長収集ループとして体験の合間に挿入される。

## 体験の核 / 設計の発明

| レイヤー | 仕掛け | 効果 |
|---|---|---|
| 地形 | ヒマラヤの高低差・峡谷・吊り橋 | グラップル＋ウィングスーツによる垂直方向の侵入経路を増やす |
| 移動 | 象・ジャイロコプター・小型機・ATV | Far Cry 3 の車・船・ハングライダーから「飛行」「巨大動物騎乗」を追加 |
| 戦闘 | 弓・自動小銃・火炎放射器・スローモーダーツ | 静音/騒音の選択がそのまま接敵距離と動物乱入の確率を変える |
| 介入 | 虎・鷲・サイ・象などの捕食動物 | プレイヤーと敵 NPC の二項関係を三つ巴に拡張 |
| 解放 | 前哨基地 24 箇所＋鐘塔解放 | エリア単位で fast travel / ミッション開放のリワードに変換 |

Far Cry 3 の Rook Islands と比較した発明は、(1) 象という「乗れる兵器」の追加、(2) ウィングスーツ＋グラップルの組み合わせで侵入経路を 2D 平面から 3D 立体に拡張、(3) Golden Path の Sabal / Amita 二派分岐で同じ前哨基地に異なる演出を被せる、の 3 点に整理できる。`social_axis: ソロ` 主体ながら、Co-op で 2 人目が空挺降下してくる「Guns for Hire」が前哨基地侵入の即興性を担保している。

## バズ要因 / 人気の根拠

- popularity の通り、ヒマラヤを舞台にした Kyrat の景観は当時のオープンワールド FPS において希少で、ピンクの桜・雪山・寺院・川下りが Twitch/YouTube 配信映えを生んだ。
- Pagan Min は Vaas（Far Cry 3）が確立した「カリスマ悪役が冒頭でプレイヤーに語りかけるオープニング」を継承し、シリーズの語り口を定型化した。
- `sales_world: 7M+ copies (2015)` — 発売翌年で 700 万本超を達成し、Ubisoft の AAA シリーズの中で最も収益性の高いラインの一つに位置付けた。
- 「画面を見ずに 15 分待つと真エンディングに到達する」隠し演出が SNS で拡散し、メタ的なフックとしてミーム化した。

## 企画への示唆

- 前作のヒット型メカニクスを保ったまま舞台気候だけ大きく変えると、同一ジャンル内でブランドの拡張余地が生まれる（熱帯島 → ヒマラヤ山岳）。
- プレイヤーと敵の二項対立に「環境からの第三者（捕食動物）」を挿入すると、毎回の戦闘が即興劇になりリプレイ価値が上がる。
- カリスマ悪役を冒頭の数分間だけ前面に出すと、本人の登場頻度が低くてもゲーム全体の語りを支配できる（Vaas → Pagan Min の様式継承）。
- 前哨基地のような「解放可能ノード」を 20〜30 個敷くと、オープンワールドの広さを「達成カウンタ」に変換できる。
- 政治派閥の選択肢を物語末尾ではなく中盤の小ミッションに分散させると、プレイヤーの態度がじわじわ可視化されエンディング選択が自然化する。
