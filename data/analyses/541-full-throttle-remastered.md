# Full Throttle Remastered（541）分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「Tim Schafer のバイカー・ノワール世界をクリック操作で踏破する体験」である。1995 年原作の LucasArts ポイント＆クリック・アドベンチャーを、Double Fine が 2017 年にリマスターした作品で、本質的にはストーリードリブンだが、本カタログでは **アドベンチャージャンルを「謎解きと探索の体験ゲーム」** として EXP に置く運用に従う。social_axis は「ソロ」で、Ben のバイク・ギャングを操作して 1980 年代風ロードムービーを進行する単独完結型である。

副次的には NAR（脚本・キャラクター造形・ハードボイルド調の語り口が骨格）と REW（背景アート差分・ロスレス再生のリマスター比較自体がコレクション的快感）が効いており、特に NAR は「秒で買い直す動機」として強い。

## 体験の核 / 設計の発明

リマスター版の設計を分解すると、1995 年原作のメカニクスに対して「描画レイヤーと音声レイヤーだけを差し替える」という最小侵襲の手法を採っている。

| レイヤー | 1995 オリジナル | 2017 リマスター | 設計の意図 |
|---|---|---|---|
| 解像度 | 320×200 ピクセル | 4K 対応の手描き再リマスター | 拡大しても線が割れない |
| サウンド | iMUSE + 低ビット | 再収録／再マスター BGM | バイカー BGM の質感を保ったまま現代機で鳴らす |
| 操作系 | 動詞アイコン UI | 簡素化されたアクションホイール | タッチ／パッドへの移植可能性 |
| 切替 | なし | ボタン一つで原作⇄リマスター比較 | ノスタルジー観賞用モード |

中核は「動詞アイコン UI を簡素化しつつ、原作のテンポを壊さない」点であり、Day of the Tentacle Remastered / Grim Fandango Remastered と同じ Double Fine のリマスター手法の系譜に属する。social_axis が「ソロ」のまま機能するのは、謎解きの間合いとセリフ送りが個人鑑賞に最適化されているためで、配信向けに引き伸ばすチューンはほぼ入っていない。

## バズ要因 / 人気の根拠

popularity は「Tim Schafer のバイカー・ノワール LucasArts クラシックの復刻、アドベンチャー復活の鍵作品の 1 つ」。これを分解する。

- 原作は 1995 年に 100 万本超を売り上げた LucasArts アドベンチャー黄金期の代表作で、Schafer の脚本・Peter Chan のアートに対する熱狂的ファン層が 22 年後の購買予約層として既に存在していた。
- Double Fine は Day of the Tentacle Remastered（2016）／Grim Fandango Remastered（2015）に続く「LucasArts 復刻三部作」の最後の 1 本として本作をリリースし、シリーズ的な期待値が積み上がっていた。
- Steam owner 数は十万本台規模に留まるが、PS4 / PS Vita / iOS / Switch / Xbox One へ広く移植され、モバイル・携帯機での「短時間で読み切れる映画的アドベンチャー」需要を確保している。
- 2010 年代後半のアドベンチャー再評価（Thimbleweed Park / Return of the Obra Dinn / Disco Elysium 周辺）の文脈と接続し、「クラシック復刻」を語る際の定番引用作になった。

## 企画への示唆

- 既存 IP のリマスターで **「動詞アイコン UI のような時代の刻印」を完全消去せず、簡素化に留める** と、原作ファンの怒りを買わずに現代機への移植解像度を上げられる。
- **原作⇄リマスター切替機能**を入れると、ノスタルジー観賞用途と SNS スクリーンショット文化の両方に乗り、配信時の比較動画が無料宣伝として走る。
- 1〜2 時間で読み切れる短尺アドベンチャーは、**PS Vita / Switch / iOS のような携帯機展開**で「映画的体験を 1 セッション完結」できる枠を取りやすい。
- 同じ作家（ここでは Tim Schafer）の他リマスターと**三部作的にまとめて出す**と、各作の販売が相互参照され、ジャンル全体の再評価ムーブメントを作れる。
- 短編・ノワール・ロードムービーといった **映画フォーマットを借用** すると、ゲーム尺の短さがむしろ「映画 1 本分」として肯定される構造に変わる。

## 比較で見るべきタイトル

- Day of the Tentacle Remastered（2016, Double Fine）— 同じリマスター三部作・同じ簡素化 UI 思想。
- Grim Fandango Remastered（2015, Double Fine）— 三部作の起点。3D 化されたアドベンチャーの復刻。
- Thimbleweed Park（2017, Ron Gilbert）— 同年・同系譜の新作アドベンチャー、動詞 UI を完全保存する逆方向の設計。
- Broken Age（2014, Double Fine）— Schafer 自身の新作アドベンチャー、Kickstarter 発のジャンル復活の象徴。
