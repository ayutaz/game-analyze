## なぜ EXP 主分類なのか

プレイヤーの主動機は「3 アクション制のターンで処刑（Execution）ボーナスを連鎖させ、自軍を回し続けて敵を殲滅する」という戦闘体験そのものである。コンセプトが示す通り、Gears of War の IP を XCOM 形式のターン制タクティクスへ移植したタイトルで、ターゲットも「ターン制戦術と Gears 世界観の両方を求めるソロプレイヤー」に絞られている。社会軸はソロ完結であり、対人戦・観戦・協力の余地はない。盤面の最適化とリスクテイクを反復することが体験の中核なので EXP が妥当となる。

secondary に NAR / REW が立つ理由は明確で、Diaz / Reyna を主軸とした章立てのキャンペーン（NAR）と、装備ドロップ / 部隊カスタマイズ / クラス成長（REW）が、戦闘ループの外側に重ねられているからである。ただしどちらも EXP を駆動するための装飾で、物語のために遊ぶ／装備のために遊ぶ構造ではない。

## 体験の核 / 設計の発明

XCOM 系の「2 アクション + オーバーウォッチ」を 3 アクションへ拡張し、Execution（瀕死敵の処刑）でチーム全員に追加アクションを返す経済設計が中核である。これにより 1 ターンの可動量が膨らみ、Gears 本編の高速で攻撃的なテンポを盤面の上に再構築している。

| 設計要素 | 効果 | 前世代タクティクス（XCOM2 等）との差分 |
|---|---|---|
| 3 アクション / ターン | 移動 + 射撃 + 再配置を 1 体で完結 | 2 アクション制よりキャラあたりの選択肢が深い |
| Execution 連鎖 | 瀕死敵を処刑するとチーム全員に +1 AP | 行動経済を攻撃的プレイへ強制誘導 |
| カバー破壊が頻繁 | 留まるより前進する方が安全になる場面が増える | 「カバー裏に居座る」最適解を意図的に壊している |
| グリッドレス移動 | 円形 AOE 範囲指定 | XCOM のマス目から離脱、Gears の流動感を再現 |
| ヒーロー＋小隊構成 | 主人公 2 名は固定、補充兵は装備で個別化 | 物語キャラと REW 的兵士を 1 編成に同居 |

social_axis がソロであることは、Execution 経済が「テンポを上げて押し切る」設計と噛み合っており、対人で再現するとバランスが破綻する種類のメカニクスである点とも整合する。

## バズ要因 / 人気の根拠

- Xbox 第一党 IP のスピンオフが PC 発でリリースされ、Microsoft Store / Steam / Game Pass の day-one 配信で「PC タクティクス × 大作 IP」という珍しい組み合わせを成立させた。
- 累計は Game Pass + Steam で 1M+ 規模と推定され、AAA 本編の数字とは別軸で「Gears 世界観をターン制で消費したい」層を可視化した。
- The Coalition が本編 IP 監修に入り、ボス戦・カットシーン・敵デザインの作り込みが家庭用 AAA 水準で、タクティクスジャンルとしては production value が突出していた点が SNS / レビューでの差別化要因になった。
- ジャンル外 IP の参入例（XCOM フォロワー）として、Mario + Rabbids、Marvel's Midnight Suns と並んで「タクティクスの IP 化」文脈で参照されることが多い。

## 企画への示唆

- ターン経済を 2 → 3 に増やすと、「行動を惜しむゲーム」から「行動を消費するゲーム」へ性質が変わり、原作 IP の攻撃的テンポを盤面に持ち込みやすい。
- 処刑ボーナスのような「リスク行動が次のリスク行動の燃料になる」報酬経済を組むと、プレイヤーは自然と攻撃的なムーブに最適化する。守備偏重ジャンルの突破口になる。
- 固定ヒーロー（物語担当）と補充兵（カスタマイズ担当）を同一編成に同居させると、NAR と REW を 1 つの戦闘ループから同時に供給できる。
- 大作 IP のスピンオフをジャンル外に振る場合、「世界観の動詞（Gears なら処刑・突撃）」をメカニクスに翻訳する作業が成立条件で、見た目だけ移植すると IP ファンも新規ジャンルファンも掴めない。
- ソロ完結でも production value（カットシーン・ボス・敵デザイン）を AAA 水準に寄せると、対人不在を補ってジャンル内の話題性で勝負できる。

## 比較で見るべきタイトル

- XCOM 2 / War of the Chosen — 2 アクション制の参照原点。Execution 経済の差分が明確になる。
- Mario + Rabbids: Kingdom Battle — IP×タクティクス成立例。固定ヒーロー編成の先行事例。
- Marvel's Midnight Suns — IP×タクティクスにカード経済を足した派生形。3 アクション系の親戚。
- Phoenix Point — XCOM の正統派フォロワー。グリッドレス AOE 等の設計を共有する。
- Phantom Doctrine — ソロ完結タクティクスのストーリー駆動側の比較対象。
