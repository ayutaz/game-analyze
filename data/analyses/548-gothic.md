# ゴシック (Gothic, 2001)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「閉じた流刑コロニーの中で **生き延びる術を肌で覚える**」こと、つまり世界を歩き回って人間関係と戦闘技術を獲得していく **過程の体験** にある。物語の筋（旧キャンプ・新キャンプ・セクト派の三勢力の対立、眠れる神の影）も濃いが、それは「どの陣営に身を寄せ、どの NPC に弟子入りし、どの森でどの獣に殴り殺されるか」という日々の生存体験に没入させるための装置で、Skyrim 以降の主流である「物語をスキップしてサンドボックスを遊ぶ」構造の祖型に近い。

`social_axis` はソロ。マルチプレイは存在せず、AI NPC が時刻ごとに作業し、酒場で殴られ、罪を犯せば追放されるという **生活シミュレーション的な社会** が一人プレイの中に閉じている。`secondary: NAR` が効くのは、勢力選択（旧キャンプの傭兵 / 新キャンプの自由民 / セクト派の信徒）が後半の物語経路と装備カーストを同時に変える点で、体験の積層の上に物語が後から乗ってくる構図になっている。

## 体験の核 / 設計の発明

中核は「**最初は何もできない**」という設計思想。主人公はレベル 1 で槍を構えるモーションすらぎこちなく、最序盤のシャドウビーストや狼の群れに殴り殺される。プレイヤーは強くなる前に **NPC の生活パターンを観察し、誰が何を売り、誰に殴られても罪にならないか** を学習する。Bethesda 系のレベルスケーリング（敵が同等に強くなる）を採らず、地域ごとに固定難易度を置いたことで「危険地帯を遠回りに通る」という地理的知識そのものが報酬になる。

| 設計要素 | Gothic の解 | 後続 RPG への遺伝 |
|---|---|---|
| 主人公の初期状態 | 名もなき囚人、武器熟練度ゼロ | Witcher 1 のゲラルト、Risen の難破者 |
| 強さの獲得経路 | NPC 弟子入り＋学習ポイントで段階的解放 | Risen / ELEX のスキルトレーナー方式 |
| 世界の構造 | 単一マップに 3 勢力居住区、徒歩で踏破可能 | Kingdom Come: Deliverance のボヘミア |
| 時間 | NPC が独自スケジュール（睡眠・労働・食事） | Skyrim の Radiant AI、KCD |
| 戦闘 | 方向入力＋ステップで間合いを取る重量戦闘 | Risen、ELEX、Dragon's Dogma 系 |
| 物語進入 | 章切替で世界状態が変わる（不可逆） | Witcher 2 の章構造 |

特に「**章遷移で NPC が死に、ダンジョンが封鎖される**」不可逆性は、オープンワールドにありがちな「全てを後で回収できる」設計を意図的に拒否しており、プレイヤーに「いま動かないと失う」という時間圧を与えている。

## バズ要因 / 人気の根拠

- ドイツ語圏で 2001 年に発売され、Piranha Bytes は **Witcher / Risen / ELEX の系譜（欧州派 3D オープンワールド RPG）の直接的な祖** と評価されている（series 累計 500 万本超）。
- Bethesda 系（Morrowind、2002）と発売時期が近かったが、Gothic は「**マップを狭く、NPC を濃く**」という逆方向の解を選び、Eurojank と呼ばれるサブジャンルの原型を作った。
- ポーランドで熱狂的支持を集め、後の CD Projekt Red が Witcher を企画する際に「Gothic に憧れて作る」と明言した経緯がある（European RPG の系譜論で頻繁に引用される逸話）。
- 戦闘システムの硬さ、UI の独特さ、バグの多さといった「触りにくさ」がそのまま「**世界の質感**」として評価され、Dark Souls 受容以前から「不便さが没入を生む」というデザイン哲学を支持する層を獲得した。

## 企画への示唆

- **最初から弱くしておくと、地理が報酬になる** — レベルスケーリングを捨て、地域固定難易度にすれば「行ける場所が増える」こと自体が進行報酬になる。
- **勢力選択を物語＋装備＋スキルの三重で結ぶと、re-playability が一気に伸びる** — 単なる分岐エンディングより、毎日の作業の見た目が変わる方が周回動機が強い。
- **NPC にスケジュールを持たせ、罪システムを置くと「観察するゲーム」になる** — 戦闘の谷間を埋めるのが探索ではなく社会観察になる。
- **章遷移で世界を不可逆に変えると、サンドボックスに時間圧が生まれる** — オープンワールドの「後でやればいい」問題を構造で潰せる。
- **「触りにくさ」を世界の質感として正当化できれば、ニッチな熱狂層を獲得できる** — UX を磨くより、世界観との一貫性を優先する選択肢がある。

## 比較で見るべきタイトル

- **The Witcher (2007)** — Gothic の系譜を引き継ぎ、物語密度を強化した直接の後継
- **Risen (2009) / ELEX (2017)** — Piranha Bytes 自身による Gothic 設計思想の継承
- **Kingdom Come: Deliverance (2018)** — NPC スケジュール＋固定難易度の現代版
- **Morrowind (2002)** — 同時代の対極解（広い・薄い・自由）
- **Dark Souls (2011)** — 「不便さ＝世界の質感」という哲学の系譜
