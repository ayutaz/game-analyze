# Homeworld Remastered Collection

## なぜ EXP 主分類なのか

プレイヤーの主動機は「3D 空間で艦隊を編成・機動させ、敵艦隊と撃ち合って勝つ」体験そのものである。RTS の操作快感と艦隊単位の戦術判断が主であり、ストーリーモードのカットシーンや艦種コレクションは副次的にしか機能しない。

`concept` 不在ながら、本作の本体は 1999 年の Homeworld 1 と 2003 年の Homeworld 2 を 4K 解像度・モダンシェーディングで作り直した HD バンドルである。プレイヤーは Mothership を旗艦とした艦隊を 3 軸の空間で運用し、地形がない宇宙空間特有のフォーメーション戦闘と資源回収（Harvester）ループを回す。`social_axis: 対戦` が示すとおり、シングルキャンペーン以外に 1v1〜マルチプレイのスカーミッシュが長期遊技の中心であり、競うこと、特に上下軸を含めた立体機動で敵編隊を撃破する手応えが動機の核になる。`secondary` が空であることからもわかるように、収集要素や物語追体験は主導機にならず、純粋な戦闘体験で評価される作品である。

## 体験の核 / 設計の発明

3D 空間 RTS というジャンル自体が極めて少数派で、Homeworld シリーズはそのリファレンス実装として 20 年以上参照され続けている。Remastered Collection はその「設計の発明」を 2015 年の解像度で再演奏した版である。

| 要素 | 設計 | プレイヤー体験への効果 |
|---|---|---|
| 3 軸機動 | Z 軸（上下）まで指示可能なムーブコマンド | 平面 RTS の「囲む／側面取る」が「上下から挟む」に拡張 |
| 艦隊フォーメーション | Sphere / Claw / Wall / X など編成プリセット | 同じ艦種でも対戦相手の編成に応じて陣形でメタを取れる |
| 資源 = 艦隊本体 | 撃破された艦は素材として回収可能、Mothership が母艦兼工場 | 戦闘と経済が一つのオブジェクトに統合され、単一の戦線で資源と兵力が同時に動く |
| キャンペーン連続性 | 残存艦隊が次ミッションに持ち越し | 損耗を抑える保守的なプレイと「全力出して次面ジリ貧」の選択肢が連続する |

Remastered 化で追加されたのは主に視覚再構築（モデル・テクスチャ・ライティング）と Homeworld 1 を Homeworld 2 エンジン上で再実装したこと、そして Steam Workshop 対応である。後者は MOD コミュニティ（Star Wars: Warlords、Complex Simple など）の延命に直結した。`social_axis: 対戦` の現代的な機能は、この MOD と Workshop に支えられている。

## バズ要因 / 人気の根拠

`popularity` は「Homeworld 1/2 を生き延びさせ、最終的に Homeworld 3 への布石となった HD バンドル」と要約される。

- 1999/2003 の原作はとうに販売停止に近かったところを、Gearbox が THQ 倒産後に IP を取得し、2015 年に Remastered として再流通させた
- Steam 推定オーナー 100〜200 万本規模で、当時の RTS ニッチとしては十分な再ヒット
- Workshop と長期パッチサポートでスカーミッシュ対戦コミュニティが残存、配信・大会の細い回路が継続
- この延命がなければ 2024 年の Homeworld 3 もクラウドファンディング（Fig で 130 万ドル超）の支持母体を集められなかった可能性が高い

## 企画への示唆

- 過去 IP を所有しているなら、HD 化はストーリーやシステムを刷新するより「視覚と起動環境の現代化」に絞ると、原作ファンの再評価とコミュニティ再起動が同時に取れる
- スカーミッシュ向けの公式 MOD プラットフォーム（Workshop 対応）を出すと、開発側のコンテンツ供給が止まっても寿命が伸びる
- 3 軸機動・立体フォーメーションのような「ジャンル内で他に再現がない発明」を 1 つ持っていると、20 年後でも参照され続ける
- リマスター = 完全新作までの IP 観測気球として機能させる。Remastered の販売実績と Workshop アクティブ数が続編 GO/NO-GO の判断材料になる
- 経済と兵力を同一オブジェクト（Mothership）に統合する設計は、プレイヤーに「攻めるか守るかの単一決断」を強制する。意思決定密度を上げたいゲームで応用可能

## 比較で見るべきタイトル

- Homeworld 3（2024）— 本作が延命した IP の続編、3D RTS の現代版
- Sins of a Solar Empire シリーズ — 4X×RTS の宇宙空間戦闘、対戦設計の比較対象
- Nexus: The Jupiter Incident — 3D 宇宙戦術のオフライン版、Homeworld の文法を別角度で実装
- StarCraft II — 平面 RTS の到達点、3 軸機動を捨てた代わりにマイクロの密度を極めた対照例
