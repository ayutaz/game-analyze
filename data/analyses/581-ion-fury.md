# Ion Fury（アイオン・フューリー）

## なぜ EXP 主分類なのか

プレイヤーの主動機は「90年代型 FPS の射撃と移動の手触りを、現代のハードウェアで連射し続ける」という一点に集約される。ストーリーや収集物ではなく、Build エンジン由来のレベル構造を踏破するときの**撃つ／避ける／拾うのループ**そのものが目的化しているため、分類は EXP が妥当である。

social_axis は「ソロ」。マルチプレイ／協力モードは持たず、Shelly "Bombshell" Harrison 一人でネオサイバーフランシスコを駆け抜けるシングル特化型の設計になっている。secondary に NAR が入るのはレベル間のボイスログや一枚絵カットシーン、Duke Nukem 系統のワンライナーが体験に味付けをしているからで、物語そのものは主目的ではなく**ステージ間の幕間として機能する補助線**として効いている。

## 体験の核 / 設計の発明

中核メカニクスは「Build エンジン上で動く非リコイル・高速ストレイフ・即着弾武器」の組み合わせ。現代の FPS が定着させた ADS／カバー／リロードを意図的に外し、移動と視界中央エイムだけで戦闘が回るよう削ぎ落とされている。

| 設計要素 | 実装 | 体験への効果 |
|---|---|---|
| ヒットスキャン武器 | Loverboy リボルバー、Penetrator SMG | リロードや弾道計算を捨て、移動方向に集中させる |
| マップ構造 | Build エンジンの傾斜・破壊壁・隠し通路 | 縦方向探索と secret 発見が戦闘と並走 |
| 体力管理 | ヘルスパック手動回収（リジェネ無し） | 撤退と再突入のリズムを残し、緊張を維持 |
| Bowling Bomb / 自爆ドローン | 範囲攻撃の連鎖 | 群れ処理に「掃射より爆破」を選ばせる |
| Aftershock 拡張（2023） | 新章＋武器＋乗り物 | 4 年越しにキャンペーン尺を倍化 |

前世代との差分は「Duke Nukem 3D / Blood / Shadow Warrior の延長線を、20 年ぶりに**新規商業作品**として組み直した」点にある。レトロ FPS は MOD 改造シーンには存在し続けていたが、3D Realms 公式パブリッシュで現代の解像度・FoV・コントローラ対応を備えたタイトルは Ion Fury が事実上の再起動点となった。

## バズ要因 / 人気の根拠

- popularity が示す通り **「20 年ぶりの新作 Build エンジン FPS」** という歴史的位置取りそのものが話題化の燃料になった
- Steam 所有者推計 50 万〜100 万、4000+ レビューで 93% Positive と、ニッチジャンルとしては定着規模に達している
- 2023 年の Aftershock 拡張で再評価ウェーブが発生し、初動売り切り型ではない長期商品化に成功
- 開発元 Voidpoint＋パブリッシャ 3D Realms という「Build 系作家陣を集めた小規模スタジオ」体制が、boomer-shooter リバイバル（Dusk / Amid Evil / Prodeus）の文脈に組み込まれて語られた

## 企画への示唆

- レトロ復古は「同じエンジンを引き続き使う」と決め打つと、グラフィック競争から降りつつ独自の触感が立ち上がる
- 武器セットを ADS なし・即着弾で揃えると、戦闘の難度設計が「敵配置と移動経路」のレイヤーに移り、レベルデザインの寄与率が上がる
- secret 発見と戦闘を同じマップに重ねると、ソロ体験でも「探す動機」と「撃つ動機」が交互にトリガーされ、ループの単調化を抑えられる
- 数年後の拡張パック投下を初期から計画に入れると、コミュニティの再活性タイミングを自分で作れる
- パブリッシャの遺産 IP 文脈（3D Realms = Duke / Build）に乗ると、レビュー記事の冒頭で **「20 年ぶり」** のフレームを与えてもらえる

## 比較で見るべきタイトル

- Dusk（2018, New Blood Interactive）— boomer-shooter リバイバルの代表格、ローポリ表現で同ジャンルを牽引
- Amid Evil（2019, Indefatigable）— ヒロイック・ファンタジー版の同時代作、武器の手触りを軸にした設計の比較対象
- Prodeus（2022, Bounding Box Software）— 現代ピクセル表現で boomer-shooter を高解像度化した進化系
- Duke Nukem 3D（1996, 3D Realms）— Build エンジンの直接の先祖、レベル構造とユーモアの遺伝子の出所
- Wrath: Aeon of Ruin（2024, KillPixel）— Quake エンジン側からの同潮流、ジャンル内の派閥比較に有用
