## なぜ REW 主分類なのか

プレイヤーの主動機は「龍が如くスタジオの IP 資産（木村拓哉演じる八神隆之というキャラクター、神室町の街並み、サブストーリー群）を PC で**手に入れて所有する**こと」にある。`sales_world` が「初月 ~50K copies / ~$1M revenue」と相対的に小規模で、`popularity` が「数年に及ぶライセンス係争を経た悲願の Steam 移植」と語られていることが象徴的で、PC 版 Judgment が動かしたのは新規体験への欲求ではなく、**既存 IP を自分のライブラリに刻む**という収集動機だった。

`secondary: ["EXP"]` が示すとおり、本体は探偵アクション――尾行・追跡・ドローン捜査・喧嘩スタイルといった龍が如く派生のアクション体験である。ただし PC 版時点では「やっと遊べるようになった」というアクセス権そのものが商品価値となっており、体験の新規性より**プラットフォーム横断のコレクション完成**が購買を駆動した点で REW が主、EXP が副の構造になる。

## 報酬ループの組成

PC 移植版 Judgment の報酬構造は、ゲーム内ループと**メタな所有ループ**の二層に分けて見るとわかりやすい。

| 層 | 報酬の単位 | リテンションを駆動するもの |
|---|---|---|
| ゲーム内 | サブストーリー完走 / フレンドイベント / ヨコハマ・神室町の店舗開拓 / 喧嘩スタイル熟練度 | 龍が如く本編で確立された「街を埋める」収集快感 |
| ゲーム内 | ドローン / VR 双六 / ダーツ等のミニゲーム達成 | 一発当たりが軽く、コレクション欄が可視化される |
| メタ | Steam 実績 / カード / クラウドセーブ | PC 環境にライブラリを束ねたい層の所有欲 |
| メタ | 木村拓哉 IP の Steam 初登場 | 「やっと PC で遊べる」という稀少性プレミアム |

特筆すべきは、ライセンス係争で 3 年以上 PC 版がリリースできなかった経緯が、結果として**待たされた分だけ移植リリース自体が報酬イベント化**した点である。

## バズ要因 / 人気の根拠

- ライセンス係争（俳優肖像権・所属事務所との交渉）で長年 PC 版が出せなかった経緯が、リリース時に「ついに来た」という文脈付きで報じられた
- 龍が如くスタジオの探偵スピンオフという立て付けで、本編 IP 既プレイヤー層が**未プレイのスピンオフを補完購入**しやすい
- 初月 50K 本 / 約 100 万ドルという数字は PC 版日本産アクションとしては中規模だが、和ゲー Steam 移植の長期テールパターン（セール時にじわじわ伸びる）に乗りやすい
- 木村拓哉という実写スターをフェイススキャンしたキャラクターモデルが、PC スクリーンショット文化と相性が良く SNS 拡散しやすかった

## 企画への示唆

- 既存 IP の**未開拓プラットフォーム移植**は、新規体験を作らずに収集動機だけで売上を立てられる――特にライブラリ統合欲が強い PC ユーザー層に効く
- ライセンス係争や移植延期のような**ネガティブ前史**を抱える IP は、リリース時に「ようやく」という文脈報酬を上乗せできる。意図的にこの遅延を演出する逆張りもあり得る
- スピンオフ作品を本編 IP のファンに**補完購入させる動線**は、本編より作品単位が小さくても収益が立つ前提で企画可能
- 実写スター IP × Steam の組み合わせは、スクリーンショット時代の SNS 拡散を狙う際の有力な配置である
- 初月販売が小さくても、Steam の長期セールテールを織り込めば「中規模ヒット移植」は事業計画として成立しうる
