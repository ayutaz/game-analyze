## なぜ EXP 主分類なのか

プレイヤーの主動機は「恐竜という制御困難な資産を抱えながらパークを設計・運営する」体験そのものにある。経営シムの数字最適化は副次的な達成にすぎず、実態は **囲いを破った T-Rex がジープを蹴散らす光景を眺め、フェンスを引き直す** という因果ループの試行錯誤に時間が溶けていく。これは「集めるために回す」REW でも「物語を読む」NAR でもなく、自分が組んだ系の挙動を観察し直す **EXP（ソロ）** に該当する。

`social_axis: ソロ` は妥当で、本作は他プレイヤーとの対戦／協力を持たず、レンジャー視点でのジープ操作・ヘリ射撃などのアクション介入を含めて、すべてが「自分のパーク」内で完結する。`concept`/`target` 欄は未記載だが、Frontier の Planet Coaster で培った経営シム文法に IP を載せた構造は明確で、`secondary` が空なのは「経営 × ジオラマ × 怪獣鑑賞」が単線でつながっていて副動機を別建てする必要がないため、と読める。

## 体験の核 / 設計の発明

中核は **「コントロール可能なシム × コントロール不能な恐竜」** という非対称ループ。プレイヤーが直接操作するのは施設配置・遺伝子合成・契約ミッションといった上流の意思決定だけで、ひとたび恐竜を放した瞬間にシステムが自律的に振る舞いはじめ、ストームや脱走で計画を破壊する。

| 層 | プレイヤーが触る要素 | システムが返す出力 |
|---|---|---|
| 経営 | 施設配置、チケット価格、契約受託 | 来園者数、収益、ストーリー進行 |
| 科学 | 化石発掘、ゲノム抽出、種改変 | 恐竜のステータス、寿命、攻撃性 |
| 飼育 | 囲い設計、社会性／環境ニーズ調整 | 快適度、脱走確率、共食い発生 |
| 危機介入 | ジープ／ヘリの直接操作、麻酔銃 | 観光客被害の最小化、リカバリー時間 |

Planet Coaster との差分は、「客を喜ばせる装置」が同時に「客を殺しうる装置」でもあるという緊張で、これにより設計判断の重みが従来のテーマパーク経営シムの数倍に跳ね上がる。`ソロ` であることはこの緊張の前提でもあり、他人に責任を分散できない閉じた実験室として機能している。

## バズ要因 / 人気の根拠

- 世界累計 **6M+ units / $146M revenue**（JSON 記載）に到達し、Frontier が IP 経営シムというフォーマットを商業的に実証した。
- 2018 年の映画『ジュラシック・ワールド／炎の王国』公開と同期したタイアップで、映画ファンと PC 経営シム層の二重需要を掴んだ。
- Jeff Goldblum らオリジナル俳優陣のボイス収録による IP 純度の高さが、配信や YouTube の「Let's Play」と相性がよく可視化された。
- 続編 *Jurassic World Evolution 2* (2021) と長期 DLC 運営に接続でき、Frontier の **IP 経営シム＝Planet Coaster／Planet Zoo／JWE の三本柱** という戦略の一角を確立した。

## 企画への示唆

- **客を喜ばせる装置を、同時に客を脅かす装置として設計する**と、配置の意思決定に倫理的・経営的な重みが乗り、単純最適化ゲームから一段抜けた緊張を生む。
- **IP の「危険な魅力」を経営シムに翻訳する**と、ファン需要と経営シム層を二重に取り込めるが、IP の象徴的シーン（脱走・嵐・捕食）を必ずシステムで再現できるかが分水嶺になる。
- **プレイヤー操作を「上流の意思決定」と「危機介入」に二段化する**と、ふだんは俯瞰で遊びつつ事故時だけアクションに降りるリズムが生まれ、経営シムの単調さを破れる。
- **シリーズ／映画と公開時期を同期させる**と、購入動機が「ゲームの評判」だけでなく「IP との接触」にもなり、コア外の購買層を取り込める。
- **同フォーマットを IP 違いで連打する（Planet Coaster → JWE → Planet Zoo）** と、エンジン・UI 資産を償却しつつ別 IP ファンへ横展開でき、スタジオ単位の収益安定につながる。

## 比較で見るべきタイトル

- *Planet Coaster* / *Planet Zoo*（Frontier）— 同社の経営シム系譜、IP なし版との比較で「IP が乗ると何が変わるか」が見える。
- *Jurassic Park: Operation Genesis*（2003）— 同 IP の前世代経営シム、20 年越しの再解釈という観点での参照軸。
- *Two Point Hospital*（2018）— 同年の英国産タイアップ経営シム、ユーモア寄りのアプローチとの対比。
- *Zoo Tycoon*（Blue Fang）— 「危険な動物を飼う」シムの古典、JWE が更新した部分／継承した部分が見える。
