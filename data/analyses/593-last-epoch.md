# ラスト・エポック (Last Epoch) 分析

## なぜ REW 主分類なのか

プレイヤーの主動機は **「ビルドを組んで、より強い装備を集める」**。Diablo / Path of Exile 系 ARPG の本流に位置する作品で、ハクスラ＝報酬ループの最適化が遊びの中心にある。本作の `concept` は時間旅行ギミックを抱えた ARPG だが、ナラティブはあくまで装備掘りの背景設定として機能しており、プレイヤーが反復するのは「Echo（時間軸ダンジョン）を周回し、ユニーク / Legendary Potential 付き装備を獲得し、Idol / Glyph で再ロールする」というルートだ。

`secondary: ["EXP"]` が示すように、スキルツリーの組み合わせを試行錯誤するビルド遊びの体験性も強い。ただし主動機は「組んだビルドで効率よく装備を回収すること」であり、戦闘体験は装備獲得を加速するための手段として位置づけられる。`social_axis` 未設定は妥当で、Co-op マルチも実装されているがソロ周回で完結する設計が基本になっている。

## 報酬ループの組成

ARPG ジャンルの中で Last Epoch が「中間ポジション」を確保しているのは、報酬層の組成が Diablo IV と Path of Exile の中間に意図的に置かれているからだ。

| 層 | 主な要素 | 比較対象との差分 |
|---|---|---|
| 装備ドロップ | Unique / Set / Exalted / Legendary Potential 付き | PoE より読みやすく、Diablo IV より深い |
| クラフト | Glyph + Shard で affix を加算・再ロール | PoE の通貨経済を簡素化し、決定論的に近づけた |
| ビルド分岐 | 5 base class × 3 mastery、Skill ごとのスペシャリゼーションツリー | スキル自体がレベルアップし固有ツリーを持つのは ARPG では珍しい |
| エンドゲーム | Monolith of Fate（Echo 周回 + Timeline ボス） | PoE Atlas より導線が短く、Diablo IV Nightmare Dungeon より深い |
| シーズン | Cycle 制（ラダー + Offline 永続） | Offline モードを持つ点が ARPG では異例 |

リテンション設計の中核は「Legendary Potential」── Unique 装備に Exalted の affix を融合させる仕組みで、ドロップした Unique がそのまま終着点ではなく、「もう一度引いて affix を上書きする」二段階の射幸性を作っている。

## バズ要因 / 人気の根拠

`popularity` は「2024 年にインディー ARPG として市場を驚かせた、深いスキルスペシャリゼーションとオフラインモードを持つ Diablo IV / PoE の代替」と要約される。これを分解すると：

- 2024 年 2 月 21 日の 1.0 ローンチで Steam 同時接続 **264k CCU** に到達、ARPG ジャンルで Diablo IV 直撃後の空白期にハマった
- 同月までに **Steam 累計 2.1M 本超** を販売。Early Access 期間（2019〜）の地道な改善を経た「ローンチ時点で完成形」が評価された
- **オフラインモード**の存在が、常時オンライン必須の Diablo IV / PoE への明確な対抗軸として SNS で言及拡散
- 公式 / 個人問わずビルドガイドの YouTube / Reddit エコシステムが立ち上がり、Skill Specialization の組み合わせ可視化サイト（Last Epoch Tools / Build Planner）が事実上の標準として機能

## 企画への示唆

- **ジャンル飽和市場では「上位互換」より「中間ポジション」を狙う**と、両極の難点（PoE の複雑さ / Diablo の浅さ）を嫌うプレイヤーをまるごと吸収できる
- **Unique × 再ロール の二段ドロップ**を作ると、レア入手の達成感とエンドコンテンツのやり込みを 1 つの装備で両立できる
- **スキル自体にレベルとツリー**を与えると、装備依存のビルド多様性とは別軸で「スキル選択 = ビルド」という設計上の差別化が立つ
- **オフラインモードを残す**と、常時オンライン疲れのプレイヤーと配信者の安定再生環境を同時に取り込める（特に北米深夜帯）
- **長期 Early Access + 透明なロードマップ**でローンチ前にコミュニティ言語（クラス愛称・ビルド名）を形成しておくと、1.0 直後の同接ピークが伸びる

## 比較で見るべきタイトル

- Diablo IV（2023、Blizzard）── 簡素な報酬層 + AAA 演出の対極側
- Path of Exile / PoE 2（GGG）── 通貨経済とビルド自由度の極側
- Grim Dawn（2016、Crate）── デュアルクラス + オフライン重視の先行例
- Torchlight Infinite（2022、XD）── モバイル F2P 側からの ARPG 報酬層実装
