# マッドマックス (Mad Max, 2015)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「荒野を車で走り、襲い、改造する」という一連の運転＋格闘体験にある。映画 IP のフィクションを追体験する NAR 寄りの枠組みに見えながら、実際の遊びの 8 割は **車 (Magnum Opus) を走らせ・ぶつけ・パーツを剥がして強化する** ループに費やされる。social_axis が「ソロ」である通り、他者と関わる要素はなく、Avalanche Studios 流のオープンワールド「拠点制圧 → リソース回収 → 強化 → 次の拠点」というサンドボックスを一人で延々と回す構造になっている。

secondary に NAR と REW が立っているのは、映画と地続きの世界観・キャラクター描写（NAR）と、車パーツ・装備・スカベンジロケーションのコレクション要素（REW）が体験を補強しているため。ただしどちらも単独で成立しておらず、運転と格闘という EXP の核に従属する形で機能している。

## 体験の核 / 設計の発明

中核メカニクスは「車 vs 車のぶつけ合い」と「下車してのフィスト格闘 (Batman: Arkham 由来のフリーフローコンバット流用)」の二層構造。両者を Magnum Opus の改造軸でつなぐことで、ジャンル横断のループを成立させている。

| レイヤ | 主要メカニクス | プレイヤーの動詞 |
|---|---|---|
| 車上 | ハープーン・ショットガン・体当たり・ブースト | 走る・撃つ・引き剥がす |
| 徒歩 | フリーフロー格闘・処刑・ステルス | 殴る・奪う |
| 拠点 | 占拠・パーツ収集・水/燃料の確保 | 集める・整える |
| 改造 | エンジン/シャシ/装甲/ニトロのアップグレード | 育てる |
| 砂嵐 | 視界制限イベント、レアパーツ出現 | 待つ・突っ込む |

前世代の同社作 Just Cause 系が「破壊の祝祭」を空中グラップリングで実現したのに対し、Mad Max は同じ破壊志向を **地表 × 車** に再翻訳した点が設計の差分。社会軸が「ソロ」であることはマルチプレイの欠落というよりも、荒野の孤独そのものを体験設計の前提に据えた選択として読める。

## バズ要因 / 人気の根拠

JSON の popularity が示す通り、Fallout と並んで「ポストアポカリプス・オープンワールド設計の基盤」として後年も参照され続けている。

- 2015 年 5 月公開の映画『Fury Road』と同年 9 月発売という IP 連動タイミングで、映画ファンの流入を確保した
- Steam owners 2〜5M レンジ（JSON 記載）と、AAA 規模としては中堅だが long tail で売れ続けるカルト的支持
- Avalanche Studios の Apex Engine による広大な砂漠表現と、車両ダメージ表現がレビュアー間で評価された
- セール常連タイトルとして Steam / PSN で繰り返し値下げされ、後年プレイヤーが流入する構造になっている

## 企画への示唆

- 既存の映画 IP を扱うなら、物語の再現ではなく **世界観の手触り（環境・乗り物・暴力の質感）** を中核に置くと、IP 消費型ではなく体験型として独立できる
- 「車 × 徒歩」の二層をひとつの改造軸でつなぐと、ジャンル横断のプレイループが単一の進行感に統合される
- オープンワールドで social_axis をあえて「ソロ」に固定すると、孤独・寡黙・荒涼といったテーマ性が設計に乗せやすい（マルチ要素を入れると世界観が崩れる）
- フリーフロー格闘のような既成メカニクス（Arkham 系）を移植する際は、**主舞台と異なるサブ動作** に使うと盗用感が薄まる
- 環境イベント（砂嵐）にレア報酬を仕込むと、ランダム性とリスクテイクが体験に組み込まれる

## 比較で見るべきタイトル

- **Fallout 4 / New Vegas** — 同じポストアポカリプス・オープンワールドの NAR/REW 寄り対比
- **Just Cause 3** — Avalanche 自社の破壊志向オープンワールドで、空中 vs 地表の設計差
- **Rage 2** — id Software による車＋徒歩のポストアポカリプス FPS、改造軸の比較対象
- **Death Stranding** — 「ソロ × 荒野移動」を非同期で再設計した次世代解
