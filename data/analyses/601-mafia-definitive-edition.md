# マフィア：コンプリート・エディション (Mafia: Definitive Edition)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「1930 年代のロスト・ヘブンを **車で走り、撃ち、追われる**」体験そのものにある。物語（NAR）と段階的な装備・車両のアンロック（REW）も強い副軸として効くが、シーンごとの運転・銃撃・潜入の手触り — つまり時代再現された都市を移動する行為自体 — が遊びの中心で、ここが EXP に分類される根拠になる。

social_axis は「ソロ」。マルチプレイは存在せず、章立てで進む一本道の犯罪映画を一人称に近い距離で追体験する設計である。secondary に NAR が入るのは、トミー・アンジェロが運転手から幹部に登り詰め裏切るという犯罪ドラマが進行の駆動力になっているため。REW が入るのは、章を進めるとガレージで歴史的車両が解放され、武器がアップグレードされる「映画的アンロック」が報酬体験として機能しているため。ただし、いずれも体験本体を補強する役割で、主動機を奪うほどではない。

## 体験の核 / 設計の発明

2002 年原作は「ミッション間に都市を運転で接続する」構造で、当時の GTA III と比較された。2020 年版は **物語と章立てはほぼ忠実**に踏襲しつつ、UE4 系内製エンジン (Illusion Engine 派生) で都市・車・銃撃・カバーアクションを 2020 年水準まで作り直した、いわゆる *ground-up remake* である。

| 構成要素 | 2002 年原作 | 2020 年 Definitive |
|---|---|---|
| 走行 | リアル寄り、AI 車との交通法ルール | 「クラシック / シミュレーション」運転モード切替で当時の挙動を温存 |
| 銃撃 | ヒットスキャン中心、カバー薄 | カバーシューター文法を全面採用 |
| 都市 | 街区とミッション地点を結ぶ移動 | 同じ街区構造を維持、密度と光源を増強 |
| 物語 | 章 = 映画的フラッシュバック | 同構造のまま新撮ボイス・カットシーン |
| ループ | 章 → 自由走行 → 次章 | 同左、ただし fast travel と難易度層を追加 |

設計の発明は新規メカニクスではなく **「20 年前の章構造をそのまま生かし、操作層だけを差し替える」**という保存方針にある。オープンワールド犯罪劇の文法を Rockstar が一手に握ったあとで、Hangar 13 は「自由度を増やすのではなく、物語の密度で勝負する」ポジションを取り直した。

## バズ要因 / 人気の根拠

- 2002 年原作は PC 黎明期の犯罪劇として根強いカルト人気を持ち、リメイク発表自体がシリーズ復権の文脈で語られた。
- Steam では概ね 86K レビュー / 86% 好評と安定した評価帯にあり、推定 100〜200 万本の販売規模に達したと見られている。
- 「Rockstar の隣接ポジション (Rockstar-adjacent prestige crime franchise)」として、GTA / RDR の年間隙間を埋める犯罪ドラマ枠の需要を引き受けた。
- 続編 Mafia III (2016) の評価分裂後、シリーズの方向性を再定義する役割を担い、2026 年予定の前日譚 *Mafia: The Old Country* に繋がる「シリーズ再起動の起点」として位置付けられている。

## 企画への示唆

- **既存 IP のリメイクで「章構造」を温存し操作層だけ差し替える**と、ノスタルジーと現代基準のプレイ感を両立できる（自由度を増やすより、密度で勝負できる）。
- **運転モードを「クラシック / 現代」で切替可能**にすると、原作派と新規派を同一パッケージで吸収できる。
- **オープンワールドを「探索目的」ではなく「章間の移動装置」として使う**と、Rockstar 系の物量勝負を回避しつつ犯罪劇のテンポを保てる。
- **ソロ専用 / 一本道**を選ぶことで、マルチ運用コストを切り捨て、映画的演出（カットシーン尺・ボイス収録量）に予算を寄せられる。
- **報酬を「物語進行に伴うガレージ・武器アンロック」に統合する**と、REW 的満足を NAR の進行と一体化させ、別ループを増やさずに済む。

## 比較で見るべきタイトル

- Grand Theft Auto: Vice City — 同時代設定の犯罪オープンワールドの代表。自由度で対比する基準点。
- L.A. Noire — 章立てと映画的演出で進行するソロ犯罪劇。Mafia とのテンポ設計の比較対象。
- Mafia II / Mafia III — シリーズ内での章構造と都市設計の変遷を辿るためのリファレンス。
- Red Dead Redemption 2 — 「物語密度で勝負する」犯罪劇の上限を示す比較対象。
- Yakuza 0 — 章立て＋密度型都市の別解（日本側からの対比）。
