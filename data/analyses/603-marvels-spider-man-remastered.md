# Marvel's Spider-Man Remastered

## なぜ EXP 主分類なのか

プレイヤーの主動機を一文にすると「マンハッタンの摩天楼をウェブスイングで自在に飛び回り、地上に降りずに敵集団へ突入する移動と戦闘のフロー体験」である。物語が分厚く、装備・スーツ収集も豊富であるにもかかわらず、本作の中核には「都市そのものを遊具化した運動ゲーム」という発明がある。Insomniac は Ratchet & Clank で培った武器バリエーションと運動感覚の設計を、垂直方向に拡張された都市プレイグラウンドへ移植した。social_axis は「ソロ」でマルチ要素は持たないが、配信映え・スクリーンショット文化を介して間接的に他者と接続している。

secondary に NAR と REW を持つ通り、MJ パートのスニーキング章や Peter / Miles のドラマは映画的に質が高く、また 40 を超えるスーツとガジェットの解禁・MOD スロット強化は「集める動機」として独立して機能する。ただしこれらは「街を駆ける」運動を継続させるための燃料であり、主従関係は EXP → NAR/REW で揺るがない。

## 体験の核 / 設計の発明

中核メカニクスを分解すると三層になる。

| 層 | 要素 | 前世代（Spider-Man 2 / Web of Shadows 等）からの差分 |
|---|---|---|
| 移動 | Web Swing / Wall Run / Web Zip / Point Launch | 物理的に建物に紐が刺さる必要があり、摩天楼依存。地上では機能不全 = マンハッタン特化 |
| 戦闘 | Bat 系コンボ + ガジェット 8 種 + 環境利用 | Arkham 系のフリーフロー戦闘を空中軸に拡張。ウェブで吊るす・引き寄せる垂直処理 |
| 表現 | ボタン押下と慣性の同期、IK ベースのポーズ補正 | キャラがウェブを「掴みに行く」前傾モーションを毎回計算、止まる瞬間が映える |

この三層が同時に走ることで、戦闘から離脱して移動に戻る摩擦がゼロになり、ミッション間の街移動それ自体が報酬になる。social_axis がソロでも飽きが来ないのは、移動の物理エンジン自体が無限に試行可能な遊具だからである。Remastered 版では Miles Morales 主人公モデルの顔のキャプチャを Peter Parker にも適用し、表情演技の精度を上げた点が NAR 寄与として加算される。

## バズ要因 / 人気の根拠

- Insomniac の自社オープンワールド最大規模作で、Marvel IP × Sony first-party × PC 展開という三重の話題性を 2022 年に同時に獲得した
- 2022 年 8 月 PC 版リリースから 2023 年 2 月までに PC だけで 150 万本超を販売、Sony の PC 戦略にとって Horizon / Days Gone に続く第三の柱になった
- PS4 オリジナル版（2018）は世界累計 2,000 万本超を売上げ、PS5 ローンチタイトル Miles Morales、続編 Spider-Man 2（2023）へと続くフランチャイズの起点となった
- 映画 No Way Home / Across the Spider-Verse の興行と同時期に走ったため、IP 全体の注目を吸い上げる形でゲーム側の認知も増幅された

## 企画への示唆

- 「移動それ自体を報酬化」すると、ファストトラベルを実装しても誰も使わなくなる（本作はファストトラベル所要時間が街移動とほぼ同じになるよう調整されている）
- 「IP の象徴的アクション一つに全力投資する」と派生メカニクスは自動的に整流される（ウェブスイング → ウェブ戦闘 → ウェブ捕縛 → ウェブガジェット）
- 「都市を 1 つに絞って垂直方向を解放」すると、横長 GTA 型より体感面積が増え、コンテンツ密度を上げやすい
- 「主分類 EXP × secondary NAR を併存させる」とき、NAR を別キャラ視点（MJ / Miles）で挟むと運動ゲーム飽和を回避できる
- 「コスチューム解禁 = REW 副軸」を入れる場合、スキル木とガジェットとは別ラインで進行させると報酬の重複感を消せる

## 比較で見るべきタイトル

- Marvel's Spider-Man 2（2023、続編）— 二人主人公とより広い都市での進化形
- Batman: Arkham シリーズ — フリーフロー戦闘の祖。地上型 IP 適応の対照例
- inFAMOUS Second Son（Sucker Punch）— Sony first-party 製の超能力 × 都市移動の前世代
- Marvel's Spider-Man: Miles Morales（2020）— 同エンジン・同マップで主人公とトーンを変えた派生実験
