# マウント・アンド・ブレード II バナーロード — 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「中世カラディア大陸で、一兵卒から領主・王へと立場を変えながら戦う / 統治する」ことに集約される。固定の物語があってそれを追体験するのでも、ガチャを引いて強キャラを集めるのでもなく、軍を編成し、地形を読み、剣を振り、内政を回す——その**プロセスの体験そのもの**が中核である。

`social_axis: ソロ` が示すように、対戦・協力の人間関係は前提に置かれない。代わりに 200 体規模の AI 兵が同時に動くリアルタイム戦闘と、王国・氏族・経済の Grand Strategy 層が同居し、プレイヤーは「歩兵として刺突する自分」と「外交で属国を増やす自分」を行き来する。`concept` 的にはアクション＋大戦略のハイブリッド・サンドボックスで、ナラティブは固有シナリオではなく**プレイヤー自身が事後的に語る回想**として生まれる。

`secondary` の NAR は「自分の氏族の興亡譚」が emergent に立ち上がる効果、REW は装備・領地・部隊レベルの蓄積が走る効果として副次的に効いている。だが両者ともプレイヤーが意図的に選んだ戦術の結果である点で、EXP に従属している。

## 体験の核 / 設計の発明

| 層 | メカニクス | 前作・前世代との差分 |
|---|---|---|
| 戦闘 | 方向制御の近接剣戟、馬上槍、弓、攻城兵器 | Warband より兵数上限・地形破壊・攻城ハシゴ運用が拡張 |
| キャンペーン | カラディア大陸のオープン世界、村→城→王国の連鎖支配 | 氏族・親族・婚姻による継承システムが新規導入 |
| 経済 | 隊商・工房・関税・略奪の収支シミュレート | プレイヤーが商人専業でも成立する経済粒度 |
| 統治 | 評議会投票、文化ポリシー、家臣の忠誠 | Grand Strategy 寄りの政治レイヤを大幅拡張 |
| Mod | TaleWorlds 公式の Modding API と Steam Workshop | Early Access 期間に培われた巨大 Mod エコシステム |

戦術 RTS と FPS と Paradox 系大戦略を**一つの操作レイヤに地続きで載せた**ことが設計の発明で、`Total War` のように戦闘/戦略が画面遷移で切れない。ソロ axis ながら、プレイヤーは AI 諸侯と外交テーブルを共有することで擬似的な社交を得る。

## バズ要因 / 人気の根拠

`popularity` の核は「8 年の Early Access を経た Warband 続編が、リリース後にも怪物的に売れた」点にある。これを事実で分解する：

- 全世界累計 **800 万本超** に到達し、Steam の歴代ベストセラー上位に名を連ねる
- 2020 年 Early Access 開始時点で Steam 同時接続 17 万人超を記録、リリース時の地ならしが完成していた
- Warband 時代から続く Mod 文化が継続し、三国志・指輪物語・古代ローマなど大型 Total Conversion が常時更新
- 配信側では「縛りプレイ・歴史 RP・氏族存続記」など**プレイヤー自作の長期キャンペーン**が動画化されやすい

## 企画への示唆

- **Early Access を 8 年回すと、リリース時点で既に「待望の続編」になっている**——リリースを到達点ではなく長い助走の終端に置く設計
- **戦術アクションと大戦略を同一画面に積層すると、プレイヤーは「自分が将軍であり剣士でもある」物語を勝手に生成する**——固有シナリオを書かずに NAR を立ち上げる手口
- **AI 諸侯にパーソナリティと氏族・親族を持たせると、対人がいないソロ axis でも擬似社交が成立する**——マルチプレイ抜きでも社会性を演出
- **Mod API を公式で開放すると、本編コンテンツの賞味期限を 5〜10 年単位で延命できる**——プレイヤー＝開発者の境界を意図的に曖昧化
- **継承・婚姻・氏族存続を入れると、1 周のプレイ時間が「世代単位」に拡張される**——時間軸を縦に伸ばす REW 的蓄積の作り方

## 比較で見るべきタイトル

- Mount & Blade: Warband — 直系前作、軽量化された Bannerlord として今も併存
- Crusader Kings III — 親族・継承の Grand Strategy 純粋形、戦闘は抽象化
- Kingdom Come: Deliverance — 一人称中世 RPG、戦闘の手触りを物語で支える対極
- Total War: Three Kingdoms — 戦闘と戦略を画面遷移で分ける旧来型ハイブリッド
- Kenshi — オープン世界サンドボックス RPG、ソロ axis で氏族と経済を回すコンセプト近縁
