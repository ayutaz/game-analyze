# パス・オブ・エグザイル2 — 詳細分析

## なぜ REW 主分類なのか

プレイヤーの主動機は「ドロップを集め、ビルドを組み、より強い装備で次のリーグを駆け抜ける」こと、すなわち **アイテムとビルドの収集 / 蓄積** にある。ジャンル表記こそ ARPG だが、ゲームプレイの大半は「ボスを倒す体験」ではなく「落ちたアイテムを鑑定し、通貨でクラフトし、取引で価値化する」という収集 / 経済ループに費やされる。前作 PoE1 から続く Currency Item（スクロールやオーブ自体が通貨でありクラフト材料）と、3 ヶ月ごとに新リーグでキャラクターをリセットする運営モデルが、本作を「単発のクリア型 ARPG」ではなく「シーズン制ガチャ的収集ゲーム」に近づけている。

副次の EXP は、戦闘そのものの手応え — 回避ロール、WASD 操作、ボス戦のソウルライク化された読み合いがリブートされた点に効いている。戦闘体験のシビア化がアイテム獲得の達成感を増幅し、REW のループを駆動するエンジンになっている。

## 報酬ループの組成

PoE2 の報酬構造は、ARPG の「拾う」とライブサービスの「シーズンを回す」が多層に積み重なっている。

| 層 | 主な要素 | リテンション貢献 |
|---|---|---|
| ドロップ層 | レアリティ別アイテム / Currency Orb / ユニーク装備 | プレイ時間に比例した即時報酬 |
| クラフト層 | オーブによる Mod 付与 / リロール / 強化 | アイテムを「素材」に変換し再投資を促す |
| 経済層 | プレイヤー間トレード / 通貨レート / リーグ経済 | 自分が拾ったものに市場価値が付き続ける |
| ビルド層 | パッシブツリー（1500 ノード超規模） / スキルジェム / Ascendancy | 同じドロップでも組み方で別ゲームになる |
| シーズン層 | 3 ヶ月毎の新リーグ + キャラクターリセット | 累積疲労をリセットし新規参入機会を作る |
| 課金層 | コスメチック / Stash Tab（収納拡張） | ゲームプレイは無料、収集効率に課金 |

特筆すべきは **「最も価値ある報酬がコスメではなくクラフト用通貨」** という設計で、課金圧を経済の外側（見た目・収納）に逃がしている点。これにより F2P / 早期アクセス勢でも経済参加が可能で、リーグごとの「やり直し」が報酬感を毀損しない。

## バズ要因 / 人気の根拠

- Steam 史上 **有料 ARPG ジャンル最大の発売初動**を記録。早期アクセス開始数時間で **100 万本超**の販売、Steam 同時接続最高 **57.8 万**を計上した
- 2025 年 8 月のアップデート時にも再度 **35.3 万 CCU** をマークし、ローンチ後 8 ヶ月経過時点で「ライブサービスとしてのピーク回帰」を実証した
- 前作 PoE1 で 10 年以上積み上げた **複雑系ビルド愛好家コミュニティ** と、Diablo IV への失望を抱えた ARPG 難民層が同時に流入した文脈がある
- 開発元 Grinding Gear Games は Tencent 子会社化以降も **「Stash Tab 課金以外は完全無料」** ポリシーを維持しており、配信者・コアコミュニティが擁護論を語りやすい立て付けになっている

## 企画への示唆

- **3 ヶ月リーグ + キャラクターリセット制を採用すると**、累積疲労と「もう追いつけない」離脱を防ぎつつ、復帰インセンティブを定常生成できる
- **アイテムドロップ自体を通貨化（Currency Item）すると**、インフレを抑えつつクラフトと取引を同じレイヤーで成立させられる
- **課金対象をコスメ + 収納拡張に限定すると**、Pay-to-Win 批判を回避しながら、コア層の収集動機（Stash 不足）を直接マネタイズできる
- **戦闘の難度を意図的にソウルライク寄りに引き上げると**、報酬獲得時の達成感が増し、ドロップ単価あたりの心理的価値が高まる
- **早期アクセスを「永続課金」ではなく「先行アクセス権 + サポーターパック」と位置付けると**、完成前の段階でも有料販売の正当性を確保しやすい

## 比較で見るべきタイトル

- Diablo IV（同ジャンル / シーズン制 / よりカジュアル寄り、対照例として）
- Path of Exile 1（前世代 / 同社、ループ設計の継承と更新の関係）
- Last Epoch（同 ARPG / クラフト寄り / インディースタジオ）
- Destiny 2（別ジャンルだがシーズン + ライブ運営 + コア / カジュアル二重構造として近縁）
