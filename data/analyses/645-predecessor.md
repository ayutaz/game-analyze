## なぜ EXP 主分類なのか

プレイヤーの主動機は「5v5 の三人称視点 MOBA で、レーン戦・ガンク・タワー破壊の刻一刻の判断を**競う**」ことに尽きる。Predecessor は 2018 年に Epic Games が打ち切った *Paragon* の資産を Omeda Studios がコミュニティ主導でクラウドファンディング再生したタイトルで、ゲームプレイの中核は「ヒーロー選択 → カード（アイテム）構築 → 30〜40 分のオブジェクト戦」という MOBA の古典的なループそのものである。social_axis が「対戦」になるのは必然で、勝敗・KDA・ランクといった対人指標が体験の意味を作っている。

secondary に REW が立っているのは、Free-to-play モデルが採用されており、ヒーローのアンロック、スキン、バトルパス進行といった**収集・進捗の層**が対戦体験の外殻として機能しているからだ。ただし F2P 収益はあくまで運営持続のための装置で、主動機を引っ張っているのは試合内のスキル発揮であり、REW を主分類に置くのは誤読になる。

## 体験の核 / 設計の発明

Predecessor の特異性は「MOBA を三人称視点で遊ばせる」点にある。俯瞰視点の DotA / LoL と異なり、TPS 的なエイム要素とポジショニングが勝敗に直結し、シューター層と MOBA 層の中間市場を狙った設計になっている。前世代との差分を整理すると以下のようになる。

| 要素 | 俯瞰 MOBA (LoL/Dota2) | Predecessor (TPS MOBA) |
|---|---|---|
| カメラ | トップダウン俯瞰 | 三人称背後追従 |
| スキル指定 | クリック / 地点指定 | エイム / レティクル |
| マップ把握 | 視界が広く戦略的 | 視界が狭く戦術的 |
| 起源 | DotA 系の正統進化 | Paragon (2016-18) の資産継承 |
| 収益 | F2P + スキン | F2P + Founders Pack + スキン |

social_axis 「対戦」の効果は、視点変更によって**個人スキルの可視性が上がる**ことにある。エイムが効くため「キャリーが上手い／下手」が観戦者にも伝わりやすく、配信文化との親和性が高い。一方で MOBA 特有の戦略性（マクロ判断）は視界が狭いぶん劣化しがちで、その均衡を取るためのカード（アイテム）合成システムが Paragon から引き継がれ調整され続けている。

## バズ要因 / 人気の根拠

- **Paragon ノスタルジア市場の独占**：Epic が 2018 年に打ち切った Paragon の遺産を引き継いだ後継プロジェクトは複数あったが、2020 年代まで生き残ったのは Predecessor がほぼ唯一。失われた IP への愛着が初期ユーザー基盤になった。
- **クラウドファンディング起点のコミュニティ駆動開発**：Omeda Studios は Kickstarter / Founders Pack で資金調達し、開発初期からプレイヤーをステークホルダーに巻き込んだ。ロードマップ公開とパッチ頻度の高さが定着率を支えている。
- **2M+ プレイヤー、Steam ピーク 7.2K CCU**：MOBA としては中堅規模だが、ニッチな TPS MOBA カテゴリでは生存ライン上にある。LoL / Dota2 の数百分の一の規模だが、F2P + 低運営コストで継続可能な水準。
- **F2P + Founders Pack のハイブリッド収益**：早期アクセス段階で課金者を取り込み、正式 F2P 化後はスキン / バトルパスで継続課金。MOBA としては標準的だが、ジャンル衰退期に新規参入する際の生存戦略として機能した。

## 企画への示唆

- **打ち切られた IP の資産を引き継ぐと、ノスタルジア層を初期動員できる**。ただしライセンス / 法務クリアと、オリジナル開発元との関係構築が前提になる。
- **既存ジャンルのカメラ視点を 1 段ずらすと、隣接ジャンルの層を取り込める**。俯瞰 MOBA → 三人称 MOBA で、シューター層が流入可能になる。
- **クラウドファンディング + Founders Pack で開発資金とコミュニティを同時調達すると、開発初期のリスクが分散する**。ステークホルダー化したユーザーは離脱率が低い。
- **競合過密ジャンルでもニッチカテゴリ（TPS MOBA など）を切り出すと、トップタイトル不在の市場で生存できる**。LoL / Dota2 と正面競合せず、Paragon 後継という座を独占できた。
- **F2P + 早期 Founders 課金のハイブリッド収益は、運営持続コストの低いジャンルで有効**。MOBA のように継続コンテンツ追加が必要なジャンルでは特に。

## 比較で見るべきタイトル

- **SMITE** (Hi-Rez, 2014) — 三人称視点 MOBA の元祖。Predecessor の直接的な先行例。
- **Paragon** (Epic Games, 2016-2018) — 資産継承元。Predecessor の存在意義そのもの。
- **League of Legends** (Riot, 2009) — 俯瞰 MOBA の標準。比較対象として常に存在する。
- **Fault** (Strange Matter, 2020) — 同じく Paragon 後継を名乗ったプロジェクト。Predecessor との生存競争。
