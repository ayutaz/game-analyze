# バイオハザード3 ラストエスケープ (1999 / Classic PC)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「閉鎖空間で追跡者から逃げ続け、瞬間ごとの危機を切り抜ける」体験そのものにある。ラクーンシティの市街地を舞台にしたサバイバルホラーで、物語の節目を消費するためではなく、限られた弾薬と回復のなかで「どこに逃げて、どう撃って、どう隠れるか」という判断を回し続けることに動詞の中心がある。social_axis は完全な「ソロ」で、他プレイヤーや非同期要素は介在しない。あくまで個人が画面の前で恐怖と緊張のループに浸るための装置として組み上げられている。

物語性（ジルの脱出劇、アンブレラの陰謀）も濃いが、これは恐怖を成立させる文脈として機能しているのであって、選択肢分岐や章立てがプレイヤーを牽引する設計ではない。secondary は空で、ガチャ・収集・育成のような副次レイヤーもない。EXP の中でも「ソロ・体験純度の高いホラー」枠に位置付けるのが妥当。

## 体験の核 / 設計の発明

中核は「Nemesis（追跡型ボス）」の常駐化と、それを成立させるための市街地マップ設計、そして瞬時の交戦/回避を選ばせる新規メカニクスにある。RE2 までの固定配置の脅威に対し、3 は「予測不能に部屋を越えて追ってくる強敵」をホラー文法に持ち込んだ。

| 要素 | RE2 までの設計 | RE3 での更新 |
|---|---|---|
| 主要敵の挙動 | 部屋単位で固定配置／スクリプト遭遇 | Nemesis が複数エリアを跨いで追跡 |
| 交戦の選択 | 撃つ or 逃げるの二択 | 緊急回避 (Quick Turn / ダッジ) で第三の選択肢 |
| マップ設計 | 警察署の閉鎖空間中心 | 屋外市街地＋分岐ルート、Live Selection で動的に変化 |
| 弾薬経済 | 拾得型 | 火薬調合システムで「何を作るか」の判断を内包 |

social_axis がソロのため、緊張は「他プレイヤーへの説明」や「協力での緩和」によって希釈されない。Nemesis の不規則出現がそのままプレイヤーの心拍数に直結する設計で、後の RE2 Remake (2019) の Mr. X、Alien: Isolation の Xenomorph、Clock Tower 系のストーカー型恐怖へと系譜が引き継がれている。

## バズ要因 / 人気の根拠

popularity に記された「ジャンル横断で模倣されるストーカー型敵の元祖、RE2 Remake の Mr. X の直接の祖先」を出発点に分解すると：

- 発売直後に世界 100 万本超を売り上げ、長期的にもマルチプラットフォーム展開（PS1 / Dreamcast / GameCube / PC）で累計数百万本のロングテール
- Nemesis という固有名のボスがミーム化し、「STARS…」のボイスとともに SNS／配信時代でも引用が続いている
- 2020 年の RE3 Remake 発売時にオリジナル版が再評価され、Classic PC 版のストア配信も継続
- ホラー実況ジャンルで「追跡型敵 = 配信映え」のテンプレを早期に提供し、後の Outlast / Amnesia / Alien: Isolation 系の配信文化の土壌になった

## 企画への示唆

- 「固定配置の脅威」を「マップを跨ぐ追跡 AI」に置き換えると、同じアセットでも体感の予測不能性が跳ね上がる
- 戦闘と逃走の二択しかないシステムに「ダッジ／緊急回避」を足すと、プレイヤーが下す判断の粒度が一段細かくなり、リプレイ価値が増す
- 拾得型の弾薬経済に「調合（クラフト）」を 1 段乗せるだけで、リソース管理の能動性が大きく変わる
- ソロホラーは social_axis を増やさず、配信視聴という非同期の鑑賞経路を意識した方が長期的なバズに繋がる
- 名前のあるボス（固有名・固有ボイス）はミーム化の起点になりやすく、シリーズ後続作のフックとしても再利用が効く

## 比較で見るべきタイトル

- Resident Evil 2 (1998) — 同シリーズ前作。固定脅威モデルの完成形
- Resident Evil 2 Remake (2019) — Mr. X として Nemesis 文法を再導入したリメイク
- Alien: Isolation (2014) — 追跡 AI 一体に全体験を寄せた現代的継承例
- Clock Tower (1995) — シザーマンによる「追われる」ホラーの先行事例
- Dead Space (2008) — 三人称ホラー＋資源管理を別文脈で再構築した系譜
