## なぜ EXP 主分類なのか

このタイトルを遊ぶ主動機は「**馬鹿げた装備と挙動でオープンワールドの治安をかき乱す**」体験そのものにある。プレイヤーはストーリーを追う／コレクションを埋めるよりも先に、紫色のディルドバットで通行人を殴り、戦車を市街地に突っ込ませ、空挺降下からの市民投擲を試す。social_axis が「ソロ」であるのは、この体験が他者との競争ではなく **「自分のサンドボックスでどこまで馬鹿げたことができるか」を自分自身に問う一人遊び**として完結しているからである。

concept 上は GTA の「リアル系犯罪オープンワールド」の対義語に位置するコメディ犯罪劇で、target は GTA の重さに飽きた／GTA を真面目にロールプレイしたくないプレイヤー層。物語（NAR）や武器・服装・拠点アンロック（REW）も豊富だが、いずれも「次のおふざけ」を許可するチケットとして機能しており、目的ではなく手段。よって主分類は EXP、副次に NAR と REW が乗る構造となる。

## 体験の核 / 設計の発明

SR3 が GTA IV（2008）以降のリアル志向に対して打ち出した差分は、**「サンドボックスをコメディ用ステージとして再定義する」** ことだった。Remastered（2020）はそのコアを保ったままアセットを Volition / Sperasoft が刷新している。

| 層 | 設計 | プレイヤーへの効果 |
|---|---|---|
| 入力レイヤ | 殴打武器に巨大ディルド／ガス銃／タンクキャノン | 「予想外の武器選択」自体が笑いを生む |
| キャラ造形 | 顔・体型・声・走り方まで深いキャラクリ | 主人公が既に「ボケ」なので世界と接触するだけで物語が立つ |
| 世界応答 | NPC の過剰リアクション、警察の段階的エスカレーション | 行動の振れ幅を世界側が増幅して返す |
| 進行系 | ミッションと「Activity」（保険金詐欺・タンクラム・虎乗せ運転）の二層 | 物語進行とふざけ遊びが交互に挟まる |
| アンロック | キャッシュで武器強化・無敵・無限弾薬を解禁 | 中盤以降は「常識破壊権」を購入する遊びへ移行 |

social_axis が「ソロ」固定なのは、Co-op が存在するにもかかわらず **笑いの最小単位がプレイヤーの自由意志に閉じている**ためで、他者と同期した対戦／観戦ではなく「自分の悪ふざけログ」を生成する装置として最適化されている。

## バズ要因 / 人気の根拠

popularity の「GTA に対する不条理側オルタナティブとしてのカルト人気」が出発点。

- **シリーズ売上**: SR3 オリジナルは累計 5.5M+ 出荷、Remastered も Steam で 1M+ 規模で流通しており、シリーズ最高峰として参照され続けている
- **2020 リマスター文脈**: 旧世代タイトルの再販ラッシュ（2019–2021）の中で、グラフィック刷新＋全 DLC 同梱で「PC で SR3 を遊び直す唯一の正規ルート」になった
- **配信適性**: キャラクリ・ふざけ武器・ラグドール挙動が VTuber／配信切り抜き文化と相性が良く、リリースから時間が経っても短尺動画素材として消費される
- **2022 年の SR 新作（リブート）への失望**との対比で再評価が進み、「シリーズの頂点はやはり 3」というコミュニティ合意が固定化した

## 企画への示唆

- **リアル系の対極に振り切ると差別化が立つ**: ジャンル王者（GTA）が「重さ」を強化する方向に進んでいる時、同じ土俵で軽さに振ると独自ポジションが取れる
- **主人公をボケに置く**: シリアスな世界に変人主人公を投げ込むと、世界の挙動がすべてツッコミとして機能し、コンテンツ生成コストが下がる
- **アンロックを「常識破壊権」として設計する**: 強化＝数値上昇ではなく「警察を無視できる」「無限弾薬で乱射可」のような **ルール解除型報酬**にすると、終盤の遊びが質的に変わる
- **ミッション層とアクティビティ層を分離**: 物語進行と純粋な悪ふざけ遊びを別チャンネルにすると、プレイヤーが気分でモードを切り替えられる
- **リマスター戦略**: シリーズが迷走した時こそ「過去の頂点を現行機で遊べるようにする」事業判断がカタログ価値を保全する

## 比較で見るべきタイトル

- **Grand Theft Auto V** — リアル志向のオープンワールド犯罪の頂点。対比軸として必須
- **Just Cause 3 / 4** — 「物理いたずらサンドボックス」としての近縁。social_axis ソロ・EXP 設計が近い
- **Saints Row IV** — SR3 の続編で「スーパーパワー＋シミュレーション」に飛びすぎた事例。差分検証用
- **Goat Simulator 3** — ふざけサンドボックスをジャンル化した派生系
- **Saints Row (2022 リブート)** — 同 IP の失敗例。何を捨てると IP が壊れるかの教科書
