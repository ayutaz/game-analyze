# シャドウラン リターンズ

## なぜ EXP 主分類なのか

プレイヤーの主動機は「サイバーパンク×ファンタジーのテーブルトップ RPG 世界で、ラン（依頼）を組み立て、戦術級グリッド戦闘で切り抜けること」である。動詞で言えば「戦う／潜入する」が先頭に立ち、物語はその体験の枠組みとして機能する。

`concept` が示すとおり本作は 1989 年のテーブルトップ RPG『Shadowrun』を 2D アイソメトリック CRPG として復活させた作品で、`social_axis: ソロ` の通り完全シングルプレイで進む。アクションポイント制ターン戦闘、種族（ヒューマン／エルフ／ドワーフ／オーク／トロール）× 役職（ストリートサムライ／デッカー／シャーマン／リガー等）の構築、サイバーウェア改造といった「自分のランナーを設計し、戦術的に試す」サイクルが核であり、ここを最も長く触ることになる。`secondary: NAR` が示すように本編キャンペーン『Dead Man's Switch』とコミュニティ製モジュール（Berlin / Dragonfall / Hong Kong へと続く拡張）には濃密な物語が乗るが、物語は「ランの依頼書」として戦闘ミッションを駆動する位置づけで、物語消費そのものを主目的にした作りではない。

## 体験の核 / 設計の発明

旧 SNES / Genesis 版のアクション RPG 路線から、本作はテーブルトップ原典のターン制グリッドへ回帰した。設計上の核は次の三層に分解できる。

| レイヤー | 中核メカニクス | プレイヤーが操作する変数 |
|---|---|---|
| キャラクタ構築 | 種族 × クラス（アーキタイプ） × カルマ振り分け | サムライ／デッカー／シャーマン／リガー／アデプト等の比率、サイバーウェア vs 魔法のトレードオフ |
| ミッション戦闘 | AP（アクションポイント）制ターン戦闘、カバー、視線 | 武器選択、呪文、ドローン、召喚精霊、ハッキング順序 |
| メタ層 | マトリックス潜入（デッカー専用シーン） | 物理空間と並行する電脳空間の二重マップを行き来する |

前世代との差分は二点ある。第一に**マトリックス＝電脳空間を「物理戦場と並列の独立マップ」として再構成**したこと。デッカーがジャックインすると物理戦闘は別キャラで継続しつつ、本人はサイバー空間のターン戦闘を別個に処理する。第二に **Steam Workshop に対応した公式エディタを同梱**し、Berlin キャンペーンの先行同梱を含めユーザー製モジュールを CRPG 本編と同列で配信できる構造を取った。`social_axis: ソロ` ながらユーザー生成シナリオが体験の総量を肥大化させる、いわば「非同期な創作流通」がソロ体験を支える。

## バズ要因 / 人気の根拠

`popularity` が示す「Shadowrun ライセンスをインディー CRPG トリロジーとして蘇生させた」点を分解すると：

- 2012 年の Kickstarter で **約 180 万ドル**を調達。当時の CRPG 復権ブーム（Wasteland 2, Project Eternity, Torment: Tides of Numenera）と同時期に走り、ジャンル復活の象徴の一つになった。
- 累計販売は数十万本規模。本編単体より、続く **Dragonfall（2014）／Hong Kong（2015）の三部作トータル**で評価が確定し、Dragonfall 以降はメタクリティック・コミュニティ評価ともに大きく上昇した。
- **公式エディタ＋ Steam Workshop** により、コミュニティが Berlin など独立キャンペーンを継続的に供給し、リリース後数年にわたる long-tail を形成した。
- PC / Mac / Linux に加え **iOS / Android にも展開**し、CRPG として珍しくモバイルで Shadowrun を遊べる経路を確保した。

## 企画への示唆

- **休眠 IP × Kickstarter ファン直接調達**で前払いされた開発費は、ニッチジャンル復活の最短ルートになる（ライセンス保持者ではなくファンが資金を立てる構造）。
- **ターン制 CRPG にユーザー生成シナリオを公式同梱**すると、本編のボリューム不足を共同体が補い、寿命が「ゲーム」ではなく「プラットフォーム」化する。
- **本編＝最低限のショウケース、拡張＝代表作**という二段ロケット構成は、初動の批判（Dead Man's Switch の短さ・線形性）を後続作で回収できる時間差設計として参考になる。
- **物理戦場とサイバー戦場を別マップで並列**運用すると、単一の戦闘 UI に背景設定（Shadowrun ならマトリックス）を可視化でき、世界観の独自性を機械的に表現できる。
- **種族 × クラス × サイバーウェア／魔法のトレードオフ**という三軸構築は、TRPG 原典の「自分のランナーを作る」体験を CRPG に翻訳する最小公式として機能する。

## 比較で見るべきタイトル

- Shadowrun: Dragonfall (2014) / Hong Kong (2015) — 同シリーズ内での設計成熟版
- Wasteland 2 (2014) — 同時期 Kickstarter CRPG 復権の双子
- XCOM: Enemy Unknown (2012) — AP 制グリッド戦闘の同時代基準
- Battletech (2018) — Harebrained Schemes 自身による戦術級 IP リバイバル続編
- Disco Elysium (2019) — TRPG 文脈を CRPG に翻案する後続の到達点
