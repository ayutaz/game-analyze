## なぜ EXP 主分類なのか

プレイヤーの主動機は「香港の街路で殴り、車を奪い、警官と犯罪者の二重身分を行き来して暴れる」体験そのものにある。GTA 系オープンワールドの語彙を借りながら、本作の中核は射撃ではなく **近接格闘 + 環境キル + 香港ロケーション** であり、移動・戦闘・潜入の「やってみる」過程に時間が溶ける構造になっている。social_axis が「ソロ」なのは、マルチプレイ要素を持たず、街そのものとプレイヤーの肉体的アドリブが向き合う設計だからだ。

secondary に NAR と REW が入るのは、Wei Shen 編の潜入捜査ストーリー（章単位の任務進行・人物関係の構築）が背骨として効き、かつ服装・武術スタイル・車両・隠し祠の収集要素が街歩きの動機を補強しているため。ただし物語消費だけでクリアまで走るプレイヤーは少数で、収集も AAA 標準の補助レイヤーに留まる。主は「殴る・走る・撃つ」の身体性で、二次が物語と収集という階層関係が明確である。

## 体験の核 / 設計の発明

| 要素 | 設計 | 体験への効果 |
|---|---|---|
| 近接格闘 | Batman: Arkham 系のフリーフロー + 環境インタラクト（電話ボックス・氷塊・換気扇） | 銃よりも素手・関節技が主軸となる希少なオープンワールド犯罪アクション |
| 二重身分メーター | Cop XP と Triad XP を別軸で蓄積、ミッション中の振る舞いで増減 | 「警官として礼儀」と「ヤクザとして派手」を同一任務内で両立する判断負荷 |
| 香港ロケーション | 九龍城・北角・中環を縮約した実在感ある密度 | GTA の北米都市疲労を回避した観光的新鮮さ |
| 武術道場・面子 (Face) | 街中イベントで Face レベルが上がり店舗割引・服装が解放 | 寄り道が経済的・美術的リターンに結び付く |

前世代の Activision 開発打ち切り（元 True Crime: Hong Kong）を Square Enix が拾い上げた経緯もあり、GTA の枠組みを **アジア圏設定 + 体術主導** に局所最適化した点が発明である。Definitive Edition は 2014 年に PS4/Xbox One/PC へ移植し、全 DLC（Year of the Snake、Nightmare in North Point 等）同梱でカタログ寿命を延ばした。

## バズ要因 / 人気の根拠

- 「香港版 GTA」というキャッチコピーが言語化しやすく、Steam の犯罪オープンワールド需要を継続的に吸収（owners 約 3.5〜3.6M、推定 gross 約 $75M）
- 開発元 United Front Games の解散後もコミュニティ MOD・グラフィック改善 patch が継続し、Steam セールの常連カタログとして残存
- 武術アクション + ジョン・ウー的銃撃演出の組み合わせが、香港映画ファン層を独立した客層として取り込んだ
- 続編が出ないまま IP が凍結したことで、逆に「あの一作で完結した香港ゲーム」というカルト的位置取りが固定化

## 企画への示唆

- 既存ジャンル（GTA 型）の **舞台と身体動作だけを差し替える** と、ジャンル疲労を回避しながら新規客（映画ファン・地域文化ファン）を取り込める
- プレイヤーの社会的役割を **二軸メーターで同時計測** すると、同じ任務でも「どう振る舞ったか」を本人が意識化し、再走の動機になる
- オープンワールドに **観光的密度（実在地名・実在文化）** を持たせると、移動そのものが報酬になり世界探索の動機を底上げできる
- 戦闘を銃ではなく **近接・環境キル主導** に振ると、敵 1 体ごとの体感時間が伸び、低敵密度でも満腹感を出せる
- 開発元消滅後も **Definitive Edition + 全 DLC 同梱** にすることで、IP 凍結を「完結した一作」というブランド資産に転化できる

## 比較で見るべきタイトル

- Grand Theft Auto V（同 social_axis ソロ・前世代スタンダード、銃撃主導との対比）
- Yakuza 0 / 龍が如く 0（東アジア都市 + 近接格闘の系譜、章立て NAR 比重は高い）
- Mafia II / III（時代と都市の差し替えでジャンル疲労を回避した事例）
- Batman: Arkham City（フリーフロー格闘の設計原典）
- Shenmue II（香港九龍を舞台にした探索アドベンチャーの先行例）
