# システムショック2 / System Shock 2

## なぜ EXP 主分類なのか

プレイヤーの主動機は「閉鎖空間で資源を切り詰めながら、自分なりの戦い方を組み立てて生き延びる」ことに集約される。SHODAN という存在感のある AI が物語を引っ張り、サイバーモジュールで能力を伸ばす RPG レイヤーも積まれているが、これらは体験を駆動するための装置であって、消費の目的そのものではない。プレイヤーが画面の前で実際に行っているのは「次の通路で何が出るか分からないまま、残弾とハック技能とサイオニックの三択をどう配分するか」というリアルタイムの意思決定であり、これは典型的な EXP の負荷である。

social_axis はソロで、Von Braun 号という船内に閉じ込められたシングルプレイ前提の没入シム。secondary に NAR と REW を併記しているのは、音声ログ／無線越しの SHODAN／Polito の語り（NAR 的な物語消費）と、サイバーモジュール配分による能力ビルド（REW 的な数値成長）が体験の質を底上げしているためで、いずれも主従としては EXP の下に従属している。

## 体験の核 / 設計の発明

中核は「FPS の操作レイヤー」「RPG の成長レイヤー」「ホラーの圧力レイヤー」を 1 つのシームレスな船内空間に重ね合わせたことにある。前作 System Shock (1994) は操作 UI と探索が分厚すぎ、Doom 系の FPS は探索と成長が薄かった。本作はその中間に「リソース枯渇と選択の重み」を置いた。

| レイヤー | 主な負荷 | 後継への遺伝 |
|---|---|---|
| FPS 操作 | 弾薬種別の使い分け、武器耐久、ハック | BioShock のプラスミド／武器二択 |
| RPG 成長 | サイバーモジュール配分、3 兵科＋サイオニック | Deus Ex の augmentation、Prey の Neuromod |
| ホラー圧力 | 音声ログ、ハイブリッドの「I'm sorry」、SHODAN の介入 | BioShock の Andrew Ryan、Prey の Typhon |
| 環境物語 | 船内の死体・落書き・音声記録のみで物語を語る | 以降の immersive sim 全般 |

social_axis ソロが効くのは、Von Braun の閉鎖性と「無線越しに指示してくる存在は信用できるのか」という疑念が、ひとりで歩いているからこそ機能するため。協力プレイは後の MOD で実装されたが、SHODAN の語りかけが「画面の前のあなた」に向かなくなり、設計の核が崩れることが知られている。

## バズ要因 / 人気の根拠

popularity は「BioShock と Prey の直接的な DNA、SHODAN の正統続編」と要約されている。これを分解すると：

- BioShock (2007) の Ken Levine、Prey (2017) の Arkane Austin は、いずれも本作のスタッフ／思想を直接受け継ぐ系譜であり、ジャンル史の参照点として繰り返し引用される
- 売上は商業的にはミドル規模に留まったが、生涯で 100 万本を超え、Nightdive Studios のリマスターと Enhanced Edition で現在も販売が継続している
- SHODAN という AI 敵役は、ゲーム史の悪役ランキング上位常連で、声優 Terri Brosius の演技を含めミーム化されている
- 「環境物語（environmental storytelling）」と「immersive sim」という用語の代表例として、ゲームデザイン論や大学のゲーム研究で取り上げられる頻度が高い

## 企画への示唆

- 物語装置を「無線越しに指示してくる声」に集約すると、プレイヤーは単独で動きながらも常に対話圧力にさらされ、ソロ体験のテンションが維持される
- 成長通貨（サイバーモジュール）を「拾える総量に上限を設ける」と、ビルドの選択が排他的になり、周回プレイ動機と SNS 上のビルド議論が同時に発生する
- 武器耐久と弾薬種別を重ねると、戦闘そのものより「どれをどこで使うか」の判断が体験の中心に移り、ホラー圧力と相性が良くなる
- 環境物語（死体配置・落書き・音声ログ）で物語の 7 割を語ると、カットシーン制作コストを抑えつつ、プレイヤー側の解釈余地で語りの密度が水増しできる
- 序盤で味方として振る舞う存在を中盤で敵に転換すると、プレイヤーの「これまでの選択は何だったのか」という再解釈が起き、二周目需要が生まれる

## 比較で見るべきタイトル

- BioShock (2007) — 本作の思想を AAA 規模で大衆化したアダプテーション
- Prey (2017, Arkane Austin) — 同じく Von Braun 型の閉鎖船内 immersive sim を 2010 年代の操作感で再構築
- Deus Ex (2000, Ion Storm) — 同年代の immersive sim、RPG レイヤーをさらに前面化
- Dead Space (2008) — SF ホラー＋資源管理の系譜、ただし FPS でなく TPS
- SOMA (2015, Frictional) — 環境物語と AI 倫理の系譜を継ぐホラー寄りの後継
