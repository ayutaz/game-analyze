# シークレット・オブ・モンキーアイランド スペシャルエディション

## なぜ EXP 主分類なのか

プレイヤーの主動機は **「ピクセル世界の論理を解読し、突飛なアイテム組み合わせを試して笑う」** という、ポイント＆クリック特有のパズル体験そのものにある。読む（NAR）でも集める（REW）でもなく、画面上のオブジェクトを総当たり的に擦り合わせ、想定外の組み合わせが成立した瞬間に得られる「謎が解けた／ジョークが刺さった」二重の快感が中核にある。

`social_axis` はソロ。1990 年の原作を 2009 年に HD リマスター化した本作は、純粋に一人でテキストとピクセルアートを読み解く遊びであり、PvP も非同期社交も介在しない。`secondary` に NAR（コミカルな海賊ストーリー）と REW（インベントリにアイテムが揃っていく満足感）が入っているのは、Guybrush の三幕構成と、解いたパズルの副産物としてアイテム欄が育つ感覚が体験を補強しているため。ただし副次要素はあくまで EXP のパズル解体プロセスを駆動する燃料であり、主動機ではない。

## 体験の核 / 設計の発明

スペシャルエディションが 1990 年原作にもたらした差分は、ゲームプレイそのものではなく「**現代プレイヤーが古典アドベンチャーに触れ続けられるための UX 層**」を被せた点にある。

| 要素 | 1990 年 原作 | 2009 年 SE |
|---|---|---|
| グラフィック | EGA / VGA ピクセル | 手描き HD アート（ワンキーで原作表示にトグル可） |
| 音声 | MIDI + 無音セリフ | フルボイス + リオーケストレーション |
| 操作系 | SCUMM 動詞メニュー（Open / Look at / Use…） | コンテキストアイコン + ヒントシステム |
| 入力 | キーボード/マウス | ゲームパッド対応（XBLA / iOS） |

「ピクセルと HD を一瞬で切り替えられる」というメタ仕掛けは単なるノスタルジー演出ではなく、**詰まった時に原作ビジュアルへ戻ると当時のヒント記憶が蘇る**という攻略補助として機能している。ヒントシステムも段階開示型で、ジョーク台詞→ヒント→正解の順に押し付けがましさを抑えて提供される。social_axis ソロを徹底しつつ「現代の忍耐に合わせて摩擦だけ削る」設計判断が貫かれている。

## バズ要因 / 人気の根拠

- **LucasArts カタログの店頭復活契機**: 本作と続編 SE は、LucasArts の古典アドベンチャー群を Steam / XBLA / iOS の現行ストアフロントに乗せ直す象徴的な再投入だった。
- **Steam 所有数で数十万、SE2 合算で 100 万超**: `sales_world` に明記されている通り、リマスター単体としてはミドル規模だが「眠っていた IP を起こす」事例として参照頻度が高い。
- **配信文化との親和**: 90 年代テキストアドベンチャーは「視聴者が一緒に総当たりに参加できる」構造で、リマスター以降の配信プラットフォーム時代に再評価された。
- **iOS 移植によるタッチ世代の流入**: ポイント＆クリックはタッチ操作と相性が良く、PC 黎明期を知らない層への入口になった。

## 企画への示唆

- **原作トグルを実装すると、ノスタルジー層と新規層を 1 本のパッケージに同居させられる**（懐古勢の「原作の方が良い」批判を中和できる）。
- **段階開示ヒントを置くと、難度を下げずに離脱率だけを下げられる**（パズル設計を改変せずに体験完了率を上げる安価な手）。
- **古典アドベンチャーをリマスターする場合、グラフィックよりも操作系（動詞メニュー→コンテキスト化）の現代化が体験摩擦の主要因**。
- **既存 IP を眠らせるより、廉価 HD 再販で店頭に居続けさせる方が長期的な IP 価値が保てる**（SE 2 → Telltale 版 → Return to Monkey Island 2022 への布石になった）。
- **「笑える失敗」がパズル総当たりの心理コストを相殺する**——間違った組み合わせに専用ジョーク台詞を用意すると、行き詰まりがコンテンツに転化する。

## 比較で見るべきタイトル

- **Day of the Tentacle Remastered (2016)**: 同じ LucasArts SCUMM 系のリマスター。原作トグル踏襲。
- **Grim Fandango Remastered (2015)**: 3D 期 LucasArts の再投入事例。操作系の現代化アプローチで対照。
- **Return to Monkey Island (2022)**: Ron Gilbert 復帰の正統続編。SE の HD アート路線とは異なる方向の「現代化」事例。
- **Thimbleweed Park (2017)**: SCUMM の文法を新規 IP で再構築した同窓会的作品。
