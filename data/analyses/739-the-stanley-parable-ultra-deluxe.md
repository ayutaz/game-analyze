# ザ・スタンリー・パラブル ウルトラデラックス（2022 / Crows Crows Crows）

## なぜ NAR 主分類なのか

プレイヤーの主動機は「ナレーターの語りを聴き、分岐の網を踏み歩く」こと。FPS の操作系を借りているが、戦闘も収集もスコアも存在せず、残るのは語り手と選択行為そのものへのメタ的な対話だけになる。

concept は「会社員スタンリーがオフィスで指示通りに歩くか／逸脱するか」を巡る一人称メタフィクションで、target はナラティブデザインや実況視聴に親しんだプレイヤー層に寄っている。social_axis は未設定（事実上ソロ単独体験）だが、ナレーター（Kevan Brighting の音声）が常時話し相手として機能するため、孤独ではなく擬似的な二人芝居の構造を持つ。secondary は空欄で、副次動機（収集／競争）が意図的に切り落とされていることが、NAR 純度を高めている。

## 物語の駆動構造

Ultra Deluxe は 2013 年版 The Stanley Parable に対し「再リリース」自体を物語題材化した構造をとる。語りの層を表に整理する。

| 層 | 物語駆動の役割 | プレイヤー入力 |
|---|---|---|
| 一次ループ | 左右ドア、指示遵守／逸脱、エンディング群 | 進行方向の選択 |
| メタ層 | ナレーターがプレイ履歴に言及・茶化す | 反復・再開・無視 |
| Ultra Deluxe 増築層 | 「新コンテンツ」ボタン、賞バケツ、博物館的展示 | 増築扉の発見 |
| 制作者ジョーク層 | 続編・賞・批評文化への自己言及 | 既存ファンの再訪 |

エンディングは 20 以上に枝分かれするが、線形なシナリオ達成ではなく **「どのナレーション経路を引き出したか」を収集する形式** に置き換わっている点が肝で、章立ては「再起動による上書き」で表現される。Ultra Deluxe は旧作の構造に新通路を増設し、続編というメタ題材自体を語り直す入れ子を作っている。

## バズ要因 / 人気の根拠

- Steam 公開初月で 400K 本超、推定収益 $6M 規模（JSON 記載）と、ナラティブ単発作としては高水準のスタートを切った。
- 2013 年原作がメタ・ナラティブの参照点として既に共有資産化しており、Ultra Deluxe の発表自体が「9 年越しの増築」というニュース価値を持った。
- ナレーター主導の一人芝居構造は実況・配信と相性が良く、エンディング差分の比較が SNS / YouTube で二次拡散しやすい。
- PS5 / Xbox / Switch を含むマルチプラットフォーム同時展開で、PC コミュニティ外への到達経路が確保された。

## 企画への示唆

- ナレーターを常駐させると、UI / チュートリアル / 物語の三役を 1 つの音声系統に圧縮できる。
- プレイヤーの「指示に従わない自由」を物語の主題に置くと、操作ログそのものがシナリオ素材になる。
- 再リリースを「増築」として題材化すると、旧作ファンへの還元と新規告知を同じパッケージで両立できる。
- エンディングを線形達成ではなく収集対象に再定義すると、配信視聴と二週目モチベーションを同時に支えられる。
- 戦闘・スコアを意図的に削ぐと、語りの密度が相対的に上がり、短時間（5〜8 時間）でも記憶に残せる。

## 比較で見るべきタイトル

- The Beginner's Guide（2015 / Davey Wreden）— 同作者周辺、語り手主導メタ物語の延長線。
- The Magic Circle（2015 / Question）— 開発工程をメタ題材化した一人称ナラティブ。
- Pony Island（2016 / Daniel Mullins）— UI とゲームジャンルそのものへのメタ介入。
- Inscryption（2021 / Daniel Mullins）— メタ語りを別ジャンル（カード）に接合した事例。
- Portal 2（2011 / Valve）— 「環境＋語り手」一人称体験の前世代基準。
