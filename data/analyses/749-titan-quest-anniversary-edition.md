# タイタンクエスト アニバーサリーエディション (2016 / Iron Lore・THQ Nordic)

## なぜ REW 主分類なのか

プレイヤーの主動機は「ギリシャ／エジプト／東洋神話の世界で**装備とスキルを集め、ビルドを練り上げる**」点にある。Diablo 2 系 ARPG の文法を踏襲し、ランダム生成装備の orange / green / purple ティアを延々と掘り、2 つの mastery を組み合わせて自分だけのクラスを構築する遊びが本体である。Anniversary Edition は 2006 年版＋拡張 Immortal Throne を統合した無償アップグレード再販で、アイテム・スキル・MIプロパティの再バランスを含む。

`secondary: ["EXP"]` は妥当で、神話世界をオンライン Co-op で 6 人パーティ探索する「冒険の場の体験」が副次動機として強く効く。ただし装備掘り＝報酬獲得が動詞として支配的なので主分類は REW。`social_axis` 空白は仕様上の欠落だが、実態としては「ソロ + 任意 Co-op」のハイブリッドで、主要プレイ層はソロ志向が多い。

## 報酬ループの組成

| 層 | 入手物 | リテンション駆動 |
|---|---|---|
| 装備ティア | white → yellow → green (set) → blue (magical) → purple (epic) → orange (legendary) | レアリティ昇格時の「次がもっと出るかも」期待 |
| Monster Infrequent (MI) | 特定モンスター固有 drop。固有 affix 付き | ファーミング先がモンスター単位で固定される |
| Mastery × 2 | 9 系統から 2 つ選択 = 36 通りのデュアルクラス | 周回ごとに別ビルドで再走する動機 |
| 難易度層 | Normal → Epic → Legendary（敵 lv と drop quality 上昇） | エンドゲーム ＝ Legendary 周回での MI 厳選 |
| 神話舞台 | Greece → Egypt → Orient → Olympus (Immortal Throne) | 地域ごとに装備テーマが入れ替わる |

ループは「2 mastery 選定 → 装備で穴を埋める → 難易度上げて再周回 → MI 厳選」で閉じており、Diablo 2 と Path of Exile の中間に位置する。Grim Dawn 開発チーム (Crate Entertainment) の前身が Iron Lore であり、本作の MI 思想と 2-mastery 設計はそのまま Grim Dawn に継承された。

## バズ要因 / 人気の根拠

- 2006 年版がスタジオ倒産で更新停止していたところ、THQ Nordic が IP を取得し**既存オーナーに無償アップグレード**として再販した珍しい再生事例
- Steam owners 200〜500 万本規模に積み上がっており、ARPG ニッチとしては長期テールが太い
- 続編的位置にある **Grim Dawn** の人気がリバースで本作の評価を引き上げた（「Grim Dawn の祖」というナラティブ）
- 2018 年に Ragnarök、2019 年に Atlantis、2021 年に Eternal Embers と追加 DLC が継続投入され、10 年以上のサポートサイクルを確保

## 企画への示唆

- **倒産タイトルを買い戻して無償アップグレード配布する**と、休眠 IP の MAU を低コストで回復しコミュニティ忠誠を獲得できる
- **mastery 2 系統の組み合わせ**を軸にすると、クラス数を 9 で抑えながら 36 通りの体験差分を提供できる（クラス制と自由ビルドの折衷）
- **モンスター固有 drop (MI) を仕込む**と「どこを周回するか」が自明化し、エンドゲームの目的設計が players-led に委ねられる
- **神話 IP を地域ごとに切り替える**（ギリシャ → エジプト → 東洋）と、章構成と装備テーマ更新を同時に達成できる
- **長期 DLC 投入で 10 年運用**する場合、本編メカニクスを壊さない「追加マスタリー / 追加地域」が拡張の安全フォーマットになる

## 比較で見るべきタイトル

- Grim Dawn — 同チームが Iron Lore 解散後に作った精神的続編。MI 思想を継承
- Diablo 2: Resurrected — ランダム drop ARPG の原点。Anniversary 版と同じ「旧作リマスター × 既存層」戦略
- Path of Exile — ビルド多様性の極北。本作の mastery × 2 を passive tree に拡張した方向
- Torchlight II — 同世代 Diablo-like。ペットシステムで差別化
- Last Epoch — Mastery 系統選択の現代版。本作の DNA を 2020 年代に再構築
