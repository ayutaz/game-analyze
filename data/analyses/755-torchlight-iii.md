# トーチライト III (2020)

## なぜ REW 主分類なのか

プレイヤーの主動機は「ハック&スラッシュで湧き続けるアイテムを引き、ビルドを組み直す」こと、つまり**収集と育成のループ**である。Diablo クローン系 ARPG の中でもトーチライトはとりわけ「カラフルなアイテムドロップとペット同伴で町まで戻らずに換金させる」構造を売りにしてきたシリーズで、本作もその系譜上にある。戦闘そのものは EXP 的だが、戦う目的が常に「より良いレア／ユニーク／レジェンダリーを引くため」に従属しているため、主分類は REW が妥当である。

secondary に EXP を置いているのは、クラス固有の「リレイック」(2 つ目のスキルツリー) 選択や、フォージ／レイルマスター／シャープシューター等のクラス固有メカニクスの操作感が一定のプレイ体験価値を持つためで、ビルド構築の試行錯誤はプレイヤーが滞在する第二の動機になっている。social_axis は未設定だが、本作はオフライン専用化を経て「ソロ／フレンドコープ寄り」に着地している。

## 報酬ループの組成

| 層 | 要素 | リテンション寄与 |
|---|---|---|
| 戦利品 | 通常 / マジック / レア / ユニーク / レジェンダリー / セット | 1 セッション内のドーパミン頻度 |
| ビルド | クラス × リレイック (副ツリー) | 「次キャラ作りたい」動機 |
| ペット | 戦闘補助 + 町への戻り運搬 | ループ短縮による試行回数増 |
| フォート | プレイヤー拠点のクラフト&装飾 | セッション間の所属感 |
| エンドゲーム | Fazeer's Dun'djinn (随机ダンジョン) | レベリング後の周回先 |

開発経緯の特殊性として、本作は元々「Torchlight Frontiers」という基本無料 MMO ARPG として 2018 年に発表され、2020 年に MMO 要素を剥がして買い切りの「III」へとピボットしている。この経緯が報酬ループに歪みを残し、フォート／シーズン的要素が買い切り構造と噛み合わずレビューが割れた。

## バズ要因 / 人気の根拠

- popularity 欄が示す通り、本作は「Frontiers から III への転換」自体が話題化した作品で、Steam owners は推定 0.5〜1M レンジに留まる。
- シリーズ前作 Torchlight II (2012) が Diablo III と同時期に「カラフルで軽快な代替」として評価された遺産を背負って発売された。
- Echtra Games (元 Runic Games / Diablo 3 リードの Max Schaefer 設立) のブランド期待値が予約を牽引した。
- MMO ピボットによりオンライン前提だった設計をオフライン買い切りに作り直したため、Steam レビューは Mixed に着地し「ARPG ピボット失敗の教訓」として現代 ARPG 史の脚注になった。

## 企画への示唆

- **ジャンル横断ピボット (MMO→買い切り) は経済設計を全て作り直す必要がある** と、フォート / シーズン / エンドゲームの相互ロックを買い切り前提で再設計しないとレビューが割れる。
- **ペットによる「町に戻らず売却」設計を入れると周回密度が上がる** ため、ループの摩擦点 (インベントリ管理) を NPC に外注する仕組みは ARPG 以外でも転用できる。
- **クラス固有のサブシステム (フォージの熱量管理、レイルマスターの列車召喚) を差し込むと、クラス選択がビルド選択と二重に効く** ので、リプレイ動機が複線化する。
- **シリーズ前作の遺産だけで初動は取れるが、エンドゲームの層が薄いとリテンションは前作比で必ず落ちる** ため、続編は新規メカニクスより周回構造の厚みで勝負した方が安全。
- **「カラフル / カジュアル / 短時間」のポジショニングを Diablo / Path of Exile のハードコア層との差別化として明示的に守る** と、ニッチで生き残れる。

## 比較で見るべきタイトル

- Torchlight II (Runic Games, 2012) — シリーズの黄金期、買い切り ARPG として現在も評価。
- Diablo III (Blizzard, 2012) — 同時期の主戦場、シーズン制エンドゲームの参考。
- Path of Exile (Grinding Gear Games, 2013) — F2P ARPG として「リーグ」運営に成功した対照例。
- Wolcen: Lords of Mayhem (2020) — 同年発売、同じくローンチで賛否が割れた ARPG。
- Last Epoch (Eleventh Hour Games, 2024) — インディー ARPG が時間をかけて評価を確立した成功例。
