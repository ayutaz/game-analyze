## なぜ REW 主分類なのか

『Undecember』をプレイする主動機は「ルーンとアイテムを集め、ビルドを組み直し、より強い数値を求めて掘り続ける」ことにある。動詞で言えば**集める／育てる**であり、戦闘そのものよりもドロップとクラフトのループが時間を消費する。

LINE Games がパブリッシュした韓国製の Diablo / Path of Exile 系 ARPG であり、F2P でクロスプラットフォーム配信されたという出自が分類を決定づける。職業を固定せず**ルーンの組み合わせで任意のビルドを成立させる**設計は、収集ゲームの語彙そのままで、「次にどの Link Rune が落ちるか」「どこまで詰めれば自分のビルドが完成形に達するか」というハクスラ的・ガチャ的興奮が中心エンジンになる。`secondary: [EXP]` が示す通り、リアルタイムのスキル運用とエンドコンテンツの周回そのものは EXP 寄りだが、F2P の収益・継続設計が完全にドロップ＆強化の縦軸に積まれている時点で主分類は REW とみなすのが妥当。

## 報酬ループの組成

ハクスラ ARPG 特有の多層ループに、F2P モバイル流の課金軸を被せている。

| 層 | 内容 | プレイヤーが追うもの |
|---|---|---|
| 戦闘 | クリックアクションでモブを溶かす | ダメージ数値・APM |
| ドロップ | 装備・ルーンが大量落下 | レア度と接尾辞のロール |
| ルーンビルド | Skill Rune と Link Rune を盤面状に配置 | 自分専用のスキル構成 |
| 強化／クラフト | リロール、強化、合成 | 「あと 1 段階」 |
| シーズン / レイド | 期間制コンテンツとボス | ランキング・限定報酬 |
| 課金 | ルーンガチャ・パス | 時間短縮と確率の押し上げ |

縦軸を多段に積むことで「今日 30 分やる理由」が常に残り、横軸（シーズン）でビルド作り直しを促す。Diablo / PoE 系の伝統的設計をスマホ片手でも回せる粒度に落としたのが Undecember の立ち位置。

## バズ要因 / 人気の根拠

- **韓国発の本格 ARPG が PC とモバイルでクロスプレイ**という珍しい立て付けで、Diablo Immortal や Lost Ark に先行・並走する形で 2022 年に注目された
- Path of Exile に強く影響を受けた**ルーン盤ビルド**は「ジョブ縛りなしで好きなスキル構成を組める」と海外コミュニティで話題化
- Steam 同接・所有者ベースで F2P の上位帯に入り、`Steam owners 1-2M` レンジに到達
- 一方で長期運用フェーズでは課金圧と周回設計への評価が割れ、`active but waning` と表現される失速傾向に入っている

## 企画への示唆

- **職業ではなくルーン盤でビルドを定義**すると、プレイヤーは「自分の発明」に愛着を持ち、シーズン跨ぎでも戻ってくる
- **PC とモバイルでまったく同じ進行データを共有**すると、出社／帰宅の時間帯まで回遊時間に変えられる
- ハクスラに**シーズン制を被せる**と、装備リセットを正当化でき、上位プレイヤーと新規の差を周期的に均せる
- F2P ARPG の課金は**確率（ガチャ）×時短（パス）×外見（コスメ）**の 3 層に分けると、コア層と軽課金層を同時に取りに行ける
- ただし、**ドロップ確率を絞り過ぎる**と海外コア層に「Pay-to-Progress」と判定され、リリース直後のバズが半年でしぼむ

## 比較で見るべきタイトル

- Path of Exile（ルーン／パッシブツリー系ビルド設計の原典）
- Diablo Immortal（同時期の F2P クロスプラットフォーム ARPG、課金論争の比較対象）
- Lost Ark（同じ韓国発、PC 主体の ARPG/MMO ハイブリッド）
- Last Epoch（同世代のハクスラ、ビルド自由度の比較）
