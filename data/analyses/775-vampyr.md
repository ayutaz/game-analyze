# ヴァンパイア (Vampyr) — 詳細分析

## なぜ NAR 主分類なのか

このゲームの主動機は「1918 年ロンドンの市民を一人ずつ読み、誰を喰らい誰を生かすかを決め、その帰結としての街の物語を見届ける」ことにある。Action RPG の外形を持ちながら、プレイヤーが繰り返し戻ってくる動機は戦闘の手触りより、Jonathan Reid 医師というキャラクターを通じた道徳的選択と市民群像劇の消費にある。

開発元 DONTNOD は『Life is Strange』で選択肢駆動の物語ゲームを確立した studio で、Vampyr もその系譜にある。ロンドンの 4 地区に住む 60 人弱の市民それぞれに固有のバックストーリーと人間関係ネットワークがあり、市民を「Embrace（喰殺）」すれば大量経験値が得られ Reid は強くなるが、地区の健康度が下がり、関連 NPC のクエスト連鎖が永久に閉じる。これは戦闘 RPG の体験というより、「物語を切り落とす痛みを引き換えに進行を買う」物語ゲームの構造で、social_axis が空欄（実質ソロ）であることも、誰にも見られない密室での倫理選択というナラティブ体験を強化している。secondary は明示されていないが、副次的にはアクション RPG としての戦闘ビルドと、スペイン風邪流行下のロンドンという歴史考証された世界観の探索が効いている。

## 物語の駆動構造

物語消費を駆動しているのは、4 つの層が直列ではなく相互参照で組み上がっている構造にある。

| 層 | 仕組み | 物語駆動への寄与 |
|---|---|---|
| 個人 | 市民 60 名弱に Hint 階層（基本→深層）が用意され、会話・調査で開示される | 「Embrace するか」の判断材料が増えるほど、決定が重くなる |
| 地区 | 4 区画ごとに「健康度」が集約され、市民死亡や Embrace で劣化、治療で回復 | ミクロな選択がマクロな街の状態に集約される可視化 |
| 主筋 | Reid の出生・吸血鬼血統・科学的探究を巡る章立て | 倫理選択とは独立に進む縦軸を提供 |
| エンディング分岐 | 累計 Embrace 数と地区健康度の組み合わせで複数結末 | 周回プレイの動機、選択の事後評価を与える |

中核の発明は「経験値経済と倫理を直結させた」点にある。Embrace なしでクリアする pacifist run は最終戦が極端に難しくなる設計で、「物語的善行＝機械的ハードモード」という非対称が、選択の重みを数値で実感させる。

## バズ要因 / 人気の根拠

- 累計 200 万本超を売り上げ、DONTNOD の商業的代表作の一つ（『Life is Strange』に並ぶ）として位置づけられた
- スペイン風邪流行下の 1918 年ロンドンという、ゲームでは珍しい歴史背景・公衆衛生危機の描写が COVID-19 期に再評価された
- 「市民を全員生かす／全員喰らう」の両極端 run が配信文化に乗りやすく、複数の YouTube/Twitch で長時間プレイ動画が成立した
- Focus Home Interactive（現 Focus Entertainment）の AA タイトル戦略の成功例として、後の Sherlock Holmes Chapter One や Greedfall 系の参照点になった

## 企画への示唆

- 経験値の供給源を「殺してはいけない対象」に置くと、機械的進行と倫理が分離せず、選択が常に物語化される
- NPC を「クエスト発行者」ではなく「経済単位」として設計すると、削除コストが可視化され、群像劇の規模を錯覚的に大きく見せられる
- 主人公の職業（医師）を倫理的緊張の源泉に組み込むと、世界観と選択システムの整合が一段強くなる
- 健康度のような地区集約メトリクスを 1 つ用意すると、無数のミクロ判断を 4〜5 個のマクロ状態に圧縮でき、プレイヤーが自分の運営を把握できる
- pacifist run を「報酬を諦めれば可能」ではなく「機械的難度が上がる代償付き」にすると、善行プレイそのものが挑戦コンテンツになる

## 比較で見るべきタイトル

- Life is Strange (DONTNOD, 2015) — 同 studio の選択肢駆動 NAR、結果可視化の系譜
- Disco Elysium (ZA/UM, 2019) — 主人公の内面と街の社会調査を物語化した NAR の同時代作
- Pathologic 2 (Ice-Pick Lodge, 2019) — 疫病下の街を医師として救う倫理選択ゲーム、近い問題設定
- Dishonored (Arkane, 2012) — 殺すか見逃すかで世界が変容する Chaos system の先行例
