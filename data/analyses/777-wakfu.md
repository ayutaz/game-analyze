## なぜ REW 主分類なのか

プレイヤーの主動機は「18 クラスを横断して育成枠を増やし、職業 / 装備 / 領地 / 政治職を集める」こと。MMO の長期運営構造に乗りつつ、戦闘の面白さよりも **キャラクターと資源の蓄積を世代をまたいで積む** ことが継続のドライバーになっており、REW を主分類に置いた。

concept にある「DOFUS の sci-fantasy 姉妹タイトル」「Krosmoz トランスメディア宇宙への接続」は、Ankama がアニメ・コミック・カードゲーム横断で IP を膨らませる過程の **ゲーム側のコレクションハブ** として WAKFU を設計したことを示す。target はターン制タクティクス好きと DOFUS 経験者の重複層で、両者とも「次のクラスを始める」「次の装備セットを完成させる」という収集的反復に親しい。

secondary に EXP が立つのは、ターン制 AP/MP グリッド戦闘 = 操作介在の濃い「体験」自体が報酬ループの土台を支えているため。「集める」ための戦闘が退屈なら収集は崩れる、という相互依存構造になっている。social_axis は未設定だが、ギルド・国家政治・職業ギルドが絡む点では実質「非同期 + 協力」のハイブリッドに近い。

## 報酬ループの組成

| 層 | 中身 | リテンション効果 |
|---|---|---|
| 戦闘報酬 | AP/MP ターン制グリッドのドロップ・経験値 | 1 セッション内で完結する短期ループ |
| クラス収集 | 18 クラスを 1 アカウントで育成可能 | 「次のクラスを試す」=擬似的な新作体験 |
| 職業 / 領地 | 採集 / クラフト / 領地経営 | 戦闘外プレイの受け皿、非戦闘プレイヤーの貢献経路 |
| 政治職 | 国家統治・市長などの選挙制ポジション | コミュニティ内のソーシャル資本としての報酬 |
| トランスメディア | Krosmoz 世界（アニメ / TCG / DOFUS）の参照 | IP ファンが「読む / 観る」と「集める」を行き来 |

中核は「18 クラス × 装備セット × 職業」の三軸で、組み合わせ空間が広いため **コレクション完了までの距離が常に残る**。2024 年の国際サーバ統合は、過疎化した収集圏を 1 つに束ね直して、希少装備や政治職の市場流動性を回復させる延命策に当たる。

## バズ要因 / 人気の根拠

- 累計約 400 万登録プレイヤー。MMO としては中規模だが、ターン制タクティクスというニッチで十数年継続している点が稀少
- アニメ『Wakfu』（Ankama 自社制作、Netflix 配信あり）からの逆流入があり、ゲーム単体ではなく **IP エコシステムの一端** として認知された
- DOFUS の姉妹タイトルとして、既存ファン層を 3D 風 2D アイソメトリックの新世代世界へスライド誘導した
- 18 クラス・職業・領地・政治といった「やることの面の広さ」が、戦闘飽きを別ループに逃がす形でリテンションを延ばした

## 企画への示唆

- **クラス数を 2 桁後半まで増やすと、1 アカウント内で擬似的な新作体験が連続供給される** — 新規ローンチに頼らない長期運営の鍵になる
- **戦闘外（採集 / クラフト / 政治）に独立した報酬ループを置くと、戦闘飽き層が辞めずに横滑りする** — MMO の離脱率を吸収する設計
- **選挙制の政治職をゲーム内に組み込むと、コミュニティ内地位が報酬として機能する** — 金銭・装備に依存しない非物質的リテンション
- **トランスメディア IP の中にゲームを置くと、アニメ / コミック視聴者が新規流入経路になる** — マーケ予算を IP 側に集約できる
- **過疎化したらサーバ統合で市場流動性を再起動する** — 長期 MMO の終末期延命の定石
