## なぜ EXP 主分類なのか

プレイヤーの主動機は「ナチス占領下のディストピアで、重火器を二丁持ちして撃ち抜く」——つまり戦う行為そのものにある。social_axis が「ソロ」であることが象徴的で、本作はマルチ協力・対戦を一切持たず、シングルプレイ FPS の手触りに資源を集中している。

ただし secondary に NAR が入っているのは妥当で、MachineGames が本作で行った最大の発明は「B級アクションの皮を被った、戦友の喪失と抵抗運動の人間ドラマ」という物語層を、撃ち合いの間隙に挟み込んだことだった。物語は強い駆動力を持つが、プレイヤーが手を動かしている時間の大半は射撃・潜入・探索であり、物語は「撃つ理由」を補強する装置として機能している。よって主分類は EXP、副は NAR が筋となる。

## 体験の核 / 設計の発明

| 設計要素 | 内容 | 前世代（Raven 版 Wolfenstein 2009）との差分 |
|---|---|---|
| 戦闘ループ | 全武器の二丁持ち、ステルス→警報→殲滅の階段 | オープン市街→閉鎖チャプター式へ回帰、密度を優先 |
| 探索 | 章ごとのコレクタブル（Enigma コード／黄金／手紙） | 報酬ではなく世界観補強の文書類が中心 |
| パークシステム | プレイスタイル（ステルス・タクティカル・突撃・デモリッション）に紐付く実績解放型 | スキルツリーを廃し、行動が能力を育てる方式 |
| シナリオ分岐 | 序章の二択（フェルガス／ワイアット）で章単位の小道具・仲間スキルが変化 | 完全リニア構成から一段だけ枝分かれを導入 |
| ステージ間ハブ | クライサウ将軍の地下基地で仲間との会話・サイドミッション | ミッション選択画面のみだった構造に居場所を追加 |

設計の発明は「FPS の射撃気持ちよさを犠牲にせず、章間に長い静かなドラマ尺を挟む」配分にある。ソロ前提だからこそ、戦闘の緊張と地下基地の生活描写を交互配置する編集が成立する。

## バズ要因 / 人気の根拠

- popularity 文の通り「Nazi-killing icon の MachineGames によるリブート」という期待値設定。Raven 版（2009）の商業的失敗で半ば終わった IP を、id Tech 5 上で再起動した点が話題化した。
- 初月 200 万本超、累計マルチミリオンの実売で「シングル FPS は死んでいない」言説の旗艦となった。
- アルタナティブ歴史（1960 年代にナチスが世界を制覇した世界）の美術設定が、月面基地・ロンドン・ベルリンと章ごとに視覚的に転換し、トレーラー素材として強かった。
- 主人公 BJ Blazkowicz の独白と PTSD 描写、ロンドンの抵抗運動メンバーとの会話量が「B 級から作家性のあるドラマへ」とレビュー界隈で評価を集めた。

## 企画への示唆

- シングル FPS でも「章間ハブで仲間と会話できる居場所」を入れると、戦闘の緊張と物語の重さを両立できる。
- 序章で「殺される仲間を二択で決める」分岐を入れると、章ごとの小道具差で 2 周目動機が生まれる（フル分岐を作らず一段だけ枝を切る低コスト手法）。
- スキルツリーを廃し「ステルスキルを N 回でステルス系パーク解放」の行動駆動式にすると、プレイスタイルの自己発見がそのまま育成になる。
- コレクタブルを能力報酬ではなく世界観補強の文書類にすると、収集動機が物語層と接続し EXP+NAR のハイブリッドが成立する。
- 死に IP のリブートは「前作の路線を否定し、開発スタジオの作家性を前面に出す」方が市場の話題化に寄与する（Raven→MachineGames のケース）。

## 比較で見るべきタイトル

- Wolfenstein II: The New Colossus（2017）— 続編。物語層をさらに肥大化させた結果、戦闘テンポとの摩擦が議論された対照例。
- DOOM (2016)— 同じく id Software 系列の FPS リブート。物語を薄め体験に振り切った逆ベクトル。
- Dishonored（2012）— ステルス／殺戮の二択構造を能力育成と接続した先行設計。
- BioShock Infinite（2013）— 「シングル FPS × 作家性のある物語」路線の同時代旗艦。
