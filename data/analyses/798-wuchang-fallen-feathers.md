## なぜ REW 主分類なのか

プレイヤーの主動機は「ビルドを練り上げ、装備とスキルを集めて自分なりの戦型に育てる」こと。Soulslike ARPG という器の中で、本作は明朝末期・蜀地を舞台にした探索よりも、武器種ごとに分岐するスキルツリー（剣・刀・斧・双剣・銃槍など）と「内丹」「羽根」系の収集要素を回し続ける構造に重みがある。死亡時のロスト、ボスドロップ素材による武器強化、属性付与といった「叩く → 強くなる → 次のボスを叩く」の階段が中核にある以上、主動機は EXP（戦闘体験）より REW（育成・収集ループ）として整理する方が、プレイヤーが何時間も粘る理由を説明できる。

secondary に EXP を置くのは、当然ながら回避・パリィ・スタミナ管理という Soulslike の身体性が体験の土台にあるため。ただし Sekiro のような体験一発勝負ではなく、詰まったらビルドを練り直して突破する設計なので、副次扱いに留めた。social_axis は基本ソロプレイで、未設定のままにしている。

## 報酬ループの組成

| 層 | 構成要素 | リテンション役割 |
|---|---|---|
| 短期 (1 戦闘) | 通常敵ドロップの素材・経験値（内丹）・羽根 | 雑魚狩りを「素振り」ではなく「収穫」に変える |
| 中期 (1 セッション) | 篝火相当の拠点でスキル振り直し / 武器強化 / 属性付与 | ビルド変更コストを下げ、試行錯誤を促す |
| 長期 (1 周回) | ボスドロップ専用武器・遺物・スキン、エンディング分岐 | クリア後にビルド別 NG+ を回す動機 |
| メタ層 | 中国神話＋クトゥルー風味の世界設定、書物・碑文の収集 | 物語派にも収集動機を提供 |

Wukong が「中華 AAA ＋ 重厚演出」で勝負したのに対し、本作は AA 規模で「武器種数 × スキル再振り自由度」という反復遊技性に振っている。Day-One Game Pass 同時配信という流通設計もまた、初動の母数を稼ぎ、長期育成ループに乗せるための前段装置として効いている。

## バズ要因 / 人気の根拠

- ポストWukong の波に乗った中国産コンソール／PC アクションとして注目され、2026 年 3 月までに 100 万本超を販売
- Microsoft の Day-One Game Pass 採用により、買い切り購入を渋るユーザー層も初週から流入
- Soulslike の中でも「武器種ごとに別ゲー化する」ビルド多様性が配信者・解説動画の素材として機能
- Black Myth: Wukong (2024) と並べて語られる「中華アクション枠」のスロットを早期に押さえた

## 企画への示唆

- AAA の隣に AA を置くと、価格・物量で住み分けつつ同じ話題の波に乗れる（Wukong → WUCHANG の追走）
- スキルの再振りコストを限りなくゼロに近づけると、武器種の数がそのままプレイ時間に変換される
- Day-One サブスク配信は買い切り売上を一部食う代わりに、初動レビュー数とビルド共有文化の立ち上げを買える
- 文化的固有性（明末・道教・羽化＝堕落のモチーフ）を世界観の中心に据えると、Souls 系の量産感から差別化できる
- ロスト・ペナルティを「ビルド再構築の口実」にすると、死にゲーの苦痛が育成の動機に転化する

## 比較で見るべきタイトル

- Black Myth: Wukong（中華 AAA アクションの先行者、規模感の対比）
- Lies of P（韓国産 Soulslike、武器パーツ組み換えという別系統の REW 設計）
- Nioh 2（武器種 × スキル × 装備厳選で REW 寄りに振った Soulslike の先例）
- Code Vein（アジア系 Soulslike、ビルド多様性とキャラメイクで RPG 寄りに調整した例）
