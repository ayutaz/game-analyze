# 祈風 Inorikaze (2019)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「読む」——守矢一家が諏訪を離れて幻想郷へ移り住み、天狗との戦いを経て「信仰と絆」を取り戻す群像劇を、9〜12 時間かけて追体験することにある。

concept は 20 人超の東方キャラクター × 7 ルートのマルチエンディング、target は「信仰・家族・絆をテーマにした重めの群像劇ノベルをじっくり読みたい」層と明示されており、ゲームプレイの巧拙やスコアではなく**物語の解像度と関係性の網**を消費させる設計である。`social_axis` が未設定（≒ソロ完結）であることも、他者と遊ぶ体験ではなくテキストを読み進めるソロ読書体験であることを裏付ける。secondary に EXP が入っているのは、選択肢分岐とルート踏破という「物語空間の探索体験」が副次的に効いているからで、純粋なキネティックノベルではない点を補足している。

## 物語の駆動構造

東方 Project という巨大な二次創作生態系を母体に持つため、本作の物語駆動は「キャラクター記号の事前共有」と「ルート踏破による関係性差分の収集」の 2 層で動く。

| 層 | 仕掛け | 機能 |
|---|---|---|
| キャラクター記号層 | 守矢三柱（諏訪子・神奈子・早苗）+ 天狗・他 20 人超 | 東方ファンの既存知識をブートストラップとして使い、設定説明コストを圧縮 |
| ルート分岐層 | 7 ルートのマルチエンディング | 「同じ事件を別視点で読み直す」モチベーションを供給、踏破コンプ欲を喚起 |
| テーマ層 | 信仰 / 家族 / 絆 / 移住 | 個別エピソードを貫く縦串。原作の「神様と信仰」モチーフを再解釈 |
| 言語層 | 繁体字中国語のみ | 中華圏東方シーンを一次読者に固定、翻訳パッチ文化が二次波を生む |

選択肢は単に分岐するだけでなく、原作で読者が既に知っている「諏訪子と神奈子の関係」や「早苗の現人神性」に**追加のレイヤーを書き込む**ことで、原作プレイヤーほど読み込み甲斐が増す構造になっている。

## バズ要因 / 人気の根拠

- Steam レビュー 406 件中 99% 高評価で「圧倒的に好評」を獲得。少数だが極めて純度の高い熱量を集めた
- **繁体字中国語のみ**という極端に狭い言語サポートにも関わらず、東方同人 Steam ヒットとして国際的に名が伝播
- 東方 Project 公認二次創作という座組み。ZUN 公認ラインに乗ったことで同人圏での信頼が担保された
- 守矢神社という「東方原作内でもファンの厚いサブクラスタ」を主役に据えた点が、ニッチ刺し型のヒットを生んだ

## 企画への示唆

- **既存 IP の特定サブクラスタを主役に据えると、薄く広く狙うより濃いコア層を確保できる**（守矢神社 = 東方の中の一神社、を全編主役化）
- **言語を絞ることは弱点だけでなく文化的境界の演出にもなる**——繁体字のみ配信は中華圏東方シーンの内輪感を維持し、翻訳パッチ需要が二次拡散の燃料になる
- **7 ルート × 群像劇は「同じ世界を別視点で読み直す」消費を作る**ので、1 ルート長さを抑えてでもルート数を増やすほうがリテンションが伸びる
- **原作公認ラインに乗ると同人圏での評価耐性が跳ね上がる**（無許諾二次創作との温度差が Steam レビュー欄に直結する）
- **「信仰」「家族」のような抽象テーマを縦串に置くと、個別エピソードがバラバラでも 9〜12 時間の長尺を支えられる**

## 比較で見るべきタイトル

- 東方 Project 本編（ZUN / 上海アリス幻樂団）——原作 STG。本作はその物語層を抽出した派生形
- ひぐらしのなく頃に（07th Expansion）——同人ノベル × ループ／群像劇の先行モデル
- 月姫（TYPE-MOON）——同人ノベルからメジャー化した群像伝奇の祖
- Higurashi When They Cry の MangaGamer 版——同人ノベルが Steam で言語の壁を越えて広がる典型事例
