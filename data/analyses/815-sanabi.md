# SANABI (サナビ)

## なぜ EXP 主分類なのか

プレイヤーの主動機は「義手のチェーンで廃墟を一筆書きのように走り抜ける」体験そのものにある。`concept` が明示する通り、本作は 2D グラップリング・プラットフォーマーであり、ステージを通過する手段＝武器＝移動が一本のチェーンに集約される。プレイヤーは敵を倒したいから引っ掛けるのではなく、引っ掛けて移動した先で副次的に敵を貫通するため、「動かす快楽」が先にあり戦闘がそれに従う構造になっている。`social_axis` がソロであることも、この移動体験を他者と比較・共有するのではなく、個人の手癖と地形の対話として閉じている点を裏づける。

副次の NAR（父娘の絆と贖罪を描くシナリオ）は強力で、Famitsu / 4Gamer のレビューでも度々言及されるが、物語パートだけを抜き出してビジュアルノベル化しても本作の評価は成立しない。物語はチェーンを振るう手の止めどころ＝休符として機能しており、`secondary: [NAR]` は EXP を駆動する情緒的なフックという位置づけが妥当である。

## 体験の核 / 設計の発明

中核メカニクスはチェーン 1 本に集約された「移動・攻撃・状態遷移」の三役一体化である。前世代の 2D グラップリングアクション（Bionic Commando 系、Spider-Man 系）と比較すると、SANABI は「壁・足場・敵を同一の文法でフックする」点と「着地モーションを最小化し、空中での連結を前提にしている」点で異なる。

| 要素 | 設計 | 体験への効果 |
|---|---|---|
| チェーン射出 | ワンボタン、対象選別なし | 「移動か攻撃か」の判断コストをゼロ化し思考速度を上げる |
| 敵フック | 引っ掛けた瞬間に貫通ダッシュ | 戦闘＝移動の副作用として処理、テンポを切らない |
| 着地 | ほぼ無硬直、即再射出可 | エアコンボの「線」が途切れず、ステージが一筆書き化 |
| ボス戦 | 環境フック前提の地形パズル | アクション RTA 的な攻略の読み解き楽しさを付与 |
| 物語パート | アクションの合間の固定演出 | 高速移動の緊張を解き、情緒の余韻を確保 |

social_axis ソロの効果は明確で、リトライ間隔の短さ（即死→即再開）に振り切れた設計はマルチプレイ的な合議を一切要求しない。1 セッションあたりの「失敗→気づき→再トライ」のサイクルが秒単位で回るため、Hades / Celeste 系の「短い死を繰り返して上手くなる」感覚に近い学習曲線を持つ。

## バズ要因 / 人気の根拠

- 韓国の大学生 5 人チーム（Wonder Potion）による制作という背景が「K-indie」の象徴例として国内外メディアで取り上げられ、NEOWIZ パブリッシングの組み合わせが小規模 IP のメディア露出を底上げした。
- Steam レビューは「圧倒的に好評」、4 万超の母数を持ち、`sales_world` は Steam 推定で約 82 万本。インディー基準では明確な商業的成功圏。
- Famitsu / 4Gamer の特集記事と、配信者・レビュアー個人による GOTY 級評価が同時発生し、2023〜2024 年の「見逃されがちな良作」枠として日本のコア層に浸透。
- 父娘の物語が SNS の感想シェア（ネタバレ伏字つきの長文ポスト）を誘発し、アクションの上手下手に関わらず「最後まで見たい」動機を補強した。

## 企画への示唆

- 移動・攻撃・状態遷移を 1 ボタンに統合すると、プレイヤーの意思決定コストが消え、結果としてプレイ密度が上がる。
- 着地硬直を削ると、ステージ全体が「線」として記憶され、リプレイ動機（RTA・実況映え）が自動生成される。
- ソロ × 即死即再開のループは、短時間でも達成感が出るため、可処分時間の細切れな現代プレイヤーに刺さりやすい。
- アクション主体の作品でも、章間に情緒的な静止パートを挟むと、SNS シェアの単位がクリップから長文感想に拡張される。
- 小規模チーム × 中堅パブリッシャ（NEOWIZ）構成は、開発リスクを抑えつつメディア露出を確保できる現実解として再現性が高い。

## 比較で見るべきタイトル

- Ghostrunner（高速移動 × 一撃必殺、SANABI の 3D 兄弟分）
- Bionic Commando Rearmed（グラップリングを移動と攻撃に統合した先行例）
- ENDER MAGNOLIA（同時期の 2D アクション、情緒×手触りで比較）
