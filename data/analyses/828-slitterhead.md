# スリッターヘッド (Slitterhead) — 詳細分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「他者の身体を渡り歩きながら血の刃で擬態怪物を斬る」というアクション体験そのものにある。concept は霊的存在「憑鬼」として通行人に次々と憑依し、人間に化けたスリッターヘッドを討つボディホラー・アクションと定義されており、ここで快感の中心は物語の続きを読む行為ではなく、群衆の中から適切な肉体を選んで乗り換え、血液を武器化して斬撃を組み立てる手続き自体にある。target も「外山圭一郎作品のファン」「東アジア的湿度のあるホラー世界観を求める層」「作家性の強いアクションを好む層」と並んでおり、観賞より操作の比重が高い。social_axis は「ソロ」で、九龍城という閉じた箱庭を一人で踏破する構造が EXP・ソロ系のアクションホラーに収まる。secondary に NAR が立っているのは、外山作品らしい断片的な事件群と稀人（マレビト）の関係性が周回を駆動するため。物語は背景設計として機能するが、瞬間ごとの判断と斬撃のリズムが前景にあるため主分類は EXP となる。

## 体験の核 / 設計の発明

中核メカニクスは「憑依」と「血の武器化」の二層で、群衆 NPC を弾薬とみなして消費・回転させる設計になっている。

| 層 | 仕組み | 体験への効果 |
|---|---|---|
| 憑依層 | 通行人・店主・警官など任意の人間に乗り換え可能 | 戦闘中の即時リスポーン、群衆そのものが HP プールになる |
| 戦闘層 | 宿主の血を刃に変えて斬撃／ガード／回避 | 宿主の HP 消費と斬撃のリスクリターン管理 |
| 稀人層 | 固有スキルを持つ特別な宿主が複数存在 | キャラビルド要素、周回時のパーティ編成的な楽しみ |
| 探索層 | 九龍城モチーフの濃密な路地・看板・雨 | 視覚的圧と方向感覚喪失でホラー感を増幅 |

前世代との差分は、SIREN の「視界ジャック」が「他人の視界を借りる受動的覗き見」だったのに対し、本作は「他人の身体を能動的に乗り換えて戦う」運用層まで踏み込んだ点にある。social_axis「ソロ」の効果として、群衆 NPC は協力 AI ではなく消費資源として扱われ、孤独な憑鬼が街を渡り歩く感触が強調される。

## バズ要因 / 人気の根拠

- 外山圭一郎の新スタジオ Bokeh Game Studio の初作という制作体制そのものが、サイレントヒル／サイレン系列のファンに対する強力な発表時フックになった
- 90 年代香港九龍城という、現存しない実在モチーフを採用したことで美術が事前情報の段階から差別化された
- 「他者憑依型アクションホラー」という一行で説明可能なコンセプトが、トレーラー段階で SNS 拡散しやすかった
- インディー AA スケールでありながらフルプライス級の作家性 IP を成立させた点が、批評・同人で賛否両論の議論を呼んだ（評価が割れること自体が言及量を押し上げた）

## 企画への示唆

- 群衆 NPC を「弾薬」として再定義すると、リスポーンの説明コストを払わずに死亡ペナルティを設計できる
- 「視界共有」を「身体共有」まで一段強める軸変更を行うと、同じホラー作家性でも操作層が刷新される
- 実在モチーフ（九龍城など）を採用する際は、現存しない過去都市を選ぶと考証コストと表現規制リスクを同時に下げられる
- 作家性 IP は AAA フルスケールでなく AA スケールで出す方が、賛否両論の構造をむしろ言及量に変換できる
- 主分類を EXP に据えつつ secondary に NAR を置く構成は、物語消費だけでは届かないコア層に対し操作の手応えで訴求できる

## 比較で見るべきタイトル

- SIREN / SIREN 2 / SIREN: New Translation（外山圭一郎による視界ジャック系ホラーの前世代）
- サイレントヒル 1〜3（湿度の高い東アジア／日本的ホラー作家性の源流）
- The Medium（霊的二重視点を操作層に組み込んだ近作ホラー）
- Returnal（ソロ TPS の操作密度とアートディレクションの両立例）
