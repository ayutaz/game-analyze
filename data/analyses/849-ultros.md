# ウルトロス (Ultros) 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「探索する／ビルドし直す」であり、視覚刺激と園芸メカニクスを通じて Sarcophagus 内部を踏破する体験そのものが核に置かれている。`concept` が示すとおり、時間ループする宇宙子宮を舞台に、果実を育てて能力を組み替えながらマップを開拓していく構造は、目的地としての物語結末よりも「ループごとに違うルートを開く」プロセスを楽しむ設計であり、`social_axis: ソロ` の独我的な体験に最適化されている。

`target` も「実験的なゲーム構造を好むインディー嗜好層、Hollow Knight や Hyper Light Drifter 系の探索アクションを求めるプレイヤー」と明示しており、語られる物語より触感（操作・視覚・成長の組み替え）が前面に出る。ただし `secondary: [NAR]` が示すとおり、サイケデリックなビジュアルが醸す神話的世界観と Sarcophagus を巡る寓話的設定が体験の没入を支えており、純粋なメカニクスゲームではなく「世界を読み解く副次動機」が併走している。

## 体験の核 / 設計の発明

中核は **メトロイドヴァニア × 時間ループ × 園芸ビルド** の三層構造。前世代のメトロイドヴァニア（Hollow Knight、Ori 等）は「進めるほど能力が永続的に増える」一方向設計だったが、Ultros はループのたびに能力ツリーがリセットされ、栽培した果実の摂取で再ビルドする逆方向設計を採る。

| 層 | 仕掛け | 効果 |
|---|---|---|
| 探索層 | Sarcophagus の閉鎖空間 | 反復踏破で地形理解が深まる |
| ループ層 | 時間ループによる能力リセット | プレイヤーに毎回「ビルド選択」を強制 |
| 園芸層 | 果実の栽培・摂取で能力獲得 | 探索ルートが栽培行為と結合する |
| 視覚層 | El Huervo のサイケデリック2D | 同じ地形でも知覚が更新される |

`social_axis: ソロ` であるため、ビルドの最適解共有や PvP は前提にされず、プレイヤー個人が「ループ間で何を捨て、何を残すか」を選ぶ意思決定そのものが遊びになる。

## バズ要因 / 人気の根拠

- Hotline Miami のアートディレクター El Huervo がビジュアル全般を担当し、過去作のファン層を直接呼び込めた
- 極彩色サイケデリック 2D という「スクリーンショットが流通する」絵づくりで SNS / 配信映え（2024 年で「最も視覚的に特異な作品」として欧米メディアに広く露出）
- 「時間ループ × 園芸 × メトロイドヴァニア」というジャンル接続の珍しさが、レビュー文に書き出しやすい引きを生んだ
- Publisher の Kepler Interactive（Sifu、Pacific Drive 等を抱える新興インディー専業 publisher）の流通網に乗り、PC / PS4 / PS5 同時展開で初動の話題が分散しにくかった

## 企画への示唆

- 著名作の特定スタッフ（アーティスト、コンポーザー）を前面に立てると、ジャンルが違っても旧作ファンを横断流入できる
- メトロイドヴァニアに「リセットされる成長」を持ち込むと、能力獲得の永続性に依存した設計通念を壊し、新しい意思決定を生める
- 探索行動と栽培行動を結合すると、移動の往復が単なる移動でなくなり、リトレース疲労を緩和できる
- アートスタイルを「ジャンルの慣習色」から外すと、ストアページ・配信サムネで識別性が跳ね、CPI（認知単価）が下がる
- ループ構造はナラティブの説明責任を軽くするので、世界設定を寓話レベルの抽象に留めても破綻しにくい

## 比較で見るべきタイトル

- Hollow Knight — 同 social_axis / 探索深度の上限値としての比較対象
- Hyper Light Drifter — 寓話的世界観と独自アートディレクションの先行例
- Outer Wilds — 時間ループと探索の組合せ設計
- Hades — ループごとのビルド組み替えという快感設計
- Returnal — ループ × アクションの三次元版、ビルド組替の意思決定構造
