# Crypt Custodian / クリプト・カストディアン 分析

## なぜ EXP 主分類なのか

プレイヤーの主動機は「死後の世界の地形を **掃き清めながら歩き回る** こと自体の体験」にある。`concept` の通り、主人公は清掃員の猫であり、能力解放と地形開放が一体化した探索行為そのものが報酬になる古典的メトロイドヴァニアの動詞構造を踏襲している。

`target` に書かれている通り、想定プレイヤーは Hollow Knight / Islets / Tunic 系の「難易度より雰囲気・密度」を求める層であり、`social_axis` がソロであることもこれを裏付ける。物語要素（住民との対話、追放された霊界の世界観）は確実に効いているが、章立てやカレンダーで強く駆動するわけではなく、あくまで探索の合間に積層する位置づけのため `secondary: NAR` 止まりが妥当。死後の住民群との小エピソード集積はキャラクター愛着を生むが、主要動機は「次の通路を通れるようにする能力解放」のループにある。

## 体験の核 / 設計の発明

Kyle Thompson 3 作目として、前 2 作（Sheepo / Islets）で蓄積した「攻撃よりも移動・パズル寄りのメトロイドヴァニア」設計を継承しつつ、清掃という日常動詞をマップ探索に重ねた点が中核の発明。

| 層 | 要素 | プレイヤーに残る感触 |
|---|---|---|
| 動詞 | 掃く / 移動アビリティ取得 / NPC 会話 | 探索が「清掃という労働」に意味づけされる |
| 空間 | 追放された霊界の周辺領域、相互接続マップ | 部屋単位の作家性、密度が高い |
| 難易度 | 戦闘は控えめ、パズル / プラットフォーミング寄り | 雰囲気を妨げない優しさ |
| 物語 | 死後の住民との対話、サブクエスト群 | 探索の合間に NAR が積層する |

Sheepo（射撃を捨てたメトロイドヴァニア）、Islets（島の物理移動）と並べると、本作は「攻撃の重みをさらに下げ、世界との交流密度を上げた」位置にある。

## バズ要因 / 人気の根拠

`popularity` の「2024 年メトロイドヴァニアの代表的スリーパーヒット」を分解すると：

- 開発者 Kyle Thompson が **Sheepo / Islets** のファン層を保持しており、3 作目のリリース時点で既存コミュニティが起動装置になった
- 「死後の世界で **清掃員として働く猫**」というワンライナー化しやすいフックがソーシャル / 配信文脈で拡散しやすかった
- Steam で「圧倒的に好評」評価を維持し、レビュー由来のロングテール流入が継続
- 推定 owners は ~200k-500k 規模（Steam 推定）で、AAA ではないが個人開発メトロイドヴァニアとしては上位群

## 企画への示唆

- 日常動詞（掃く・撮る・運ぶ）をジャンル動詞（探索・戦闘）に重ねると、ジャンル疲労層に新規性として刺さる
- 攻撃の比重を下げ移動アビリティとパズルに置き換えると、難易度耐性が低い層まで顧客層が広がる
- 主人公を「小動物 + 異世界の労働者」にすると、ワンライナー拡散力が上がり配信文脈で初動を稼げる
- 作家性のある個人開発者は、前作のコミュニティを 3 作目以降の起動装置として再利用できる（Sheepo → Islets → Crypt Custodian の累積効果）
- ソロ探索 + キャラ密度の高い NPC 群を組み合わせると、ボリュームを膨らませずに「居心地」で滞在時間を伸ばせる

## 比較で見るべきタイトル

- **Hollow Knight**（2017）：ソロ探索型メトロイドヴァニアの近代標準。難易度の対極として参照される
- **Islets**（2022）：同 Kyle Thompson 前作。動詞設計の進化線が読める
- **Tunic**（2022）：ソロ + 雰囲気重視 + パズル寄りという target 層が重なる
- **Animal Well**（2024）：同年メトロイドヴァニアのスリーパーヒット枠の比較対象
