# ペッパーグラインダー — Pepper Grinder

## なぜ EXP 主分類なのか

このゲームの主動機は「掘る／飛ぶ／滑るの手触りそのものを味わう」ことである。物語の起伏や恒久的な収集要素ではなく、ドリル "ペッパー" を地中に突き刺し、慣性を保ったまま土を泳ぎ、勢いを乗せて空中へ射出されるという**ロコモーション（移動感）**が中心にある。`concept` の「掘削＝移動という新感覚 2D アクション」がそのまま体験の核を言い当てており、プレイヤーは地形を読み、ドリルの突入角度と離脱タイミングを最適化することで進行する。

`social_axis: ソロ` が示す通り、対戦・協力の社交要素は持たず、ステージごとに完結する**自分対地形**の構造で、`target` も「短時間で濃密な手触りを求める 2D platformer 愛好家」に絞られている。secondary が空であることは、本作が物語消費（NAR）にもメタ進行（REW）にも寄せず、純粋に体験の質で勝負していることを補強する。

## 体験の核 / 設計の発明

中核メカニクスを分解すると、従来 2D プラットフォーマーが前提としていた「地面＝障害物／空中＝移動先」という二項関係を反転させた点にある。地面が**追加の移動レイヤー**になり、空中ジャンプは "地中から射出された結果" として副次的に得られる。

| 設計要素 | 一般的な 2D プラットフォーマー | Pepper Grinder |
|---|---|---|
| 地面の役割 | 立つ・障害になる | 泳ぐ・加速する媒体 |
| 移動の主動詞 | 走る・跳ぶ | 掘る・射出される |
| ジャンプの位置づけ | 主動作 | 掘削の結果 |
| ステージ設計 | 足場の配置 | 掘れる土と掘れない岩の塗り分け |
| 失敗の質 | タイミングミス | 角度・慣性の読み違い |
| 1 ステージの長さ | 中〜長 | 数分の短編集 |

`ソロ` 軸かつ短編構成で**1 ステージ＝1 つの掘削パズル**として閉じており、Ahr Ech の 1 人開発という制作体制が、機能の絞り込みとモチーフの統一に直結している。

## バズ要因 / 人気の根拠

- 1 人開発（Ahr Ech）ながら Devolver Digital が配給し、流通とプロモを補完する**インディー × パブリッシャー組み合わせ**の典型例として注目された
- 「ドリルで地中を泳ぐ」というワンアイデアが SNS の短尺動画やトレーラーで**画面写りの即時理解性**を持ち、プレイ映像が拡散しやすかった
- 2024 年プラットフォーマーの隠れた名作として**複数メディアの年間ベスト**に掲載され、レビュー側からも手触りが評価された
- PC（Steam）と Switch の同時展開で、PC インディー文脈と任天堂機の 2D アクション文脈の双方からプレイヤーを取り込んでいる

## 企画への示唆

- **既存メカニクスの "前提" を 1 つ反転させる** と、UI もアセットも大きく増やさずに新感覚を作れる（地面＝障害 → 地面＝媒体）
- **1 つの動詞に全機能を集約** すると（本作なら "掘る"）、1 人開発でもステージ設計が量産可能になる
- **画面写りで仕掛けが 3 秒で伝わる** メカニクスは、トレーラー／GIF／TikTok の伝播コストが下がり広告予算の代替になる
- **短編集型のステージ構成** にすると、1 セッション 5〜10 分でも満足感が出るためモバイル隣接の可処分時間にも刺さる
- **Devolver のような目利き型パブリッシャーと組む** と、小品でも年間ベスト圏に押し上げる露出を確保しやすい

## 比較で見るべきタイトル

- **Downwell**（2015）— 縦方向ロコモーションを 1 つの動詞に集約した先行例
- **Celeste**（2018）— ソロ軸 2D アクションで手触りの密度を最大化した参照点
- **Drill Dozer**（2005）— ドリル＝移動という発想の系譜（GBA / Game Freak）
- **Tiny Thor / Nubby's Number Factory** 系の Devolver 小品 — 同パブリッシャーの絞り込み型インディー
- **Pizza Tower**（2023）— 慣性と勢いの維持を中核に据えた 2D 動感系の近年作
