# 超探偵事件簿 レインコード

## なぜ NAR 主分類なのか

プレイヤーの主動機は「読む／真相を見届ける」である。雨の止まないカナイ区という閉鎖都市を舞台に、記憶喪失の探偵見習いユーマと死神少女シーリの関係を軸に章ごとの事件を解いていく構造で、操作の巧拙やキャラ収集ではなく **物語の章を読み進めて事件と都市の秘密を暴く** ことが牽引力になっている。concept にある「謎迷宮」も推理ロジック・物的証拠・容疑者証言を読み合わせる装置で、target に挙げられたダンガンロンパ／逆転裁判系 ADV のフォロワー層が「真犯人と動機を読む」目的で接続している点でも NAR が妥当。

secondary に EXP が立つのは、謎迷宮内での「推理デスマッチ」（解答ソードで矛盾を斬る）「謎ロイド」（証言を組み合わせ犯人を再構成）などミニゲーム的アクションが章クライマックスに置かれているため。読み物の合間にプレイヤーの操作が解答儀式として挟まる構造で、ノベルゲーの中盤テンポを補強している。

## 物語の駆動構造

章立ては「都市怪奇 → 個別事件 → 上層部の陰謀」の三層で、各章が独立した謎を完結させつつメタプロットを 1 ノッチ進める王道ミステリー連作の形を取る。下表に駆動要素を分解する。

| 駆動要素 | 役割 | 効果 |
|---|---|---|
| 章クライマックスの謎迷宮 | 通常ノベルパートの「解決儀式」 | 読書を中断させずアクションで快感ピークを作る |
| 死神シーリの能力アンロック | 章ごとに新ギミック解禁 | 後半章のパズル難度と演出の派手さを底上げ |
| 超探偵の固有能力 | キャラ別の推理ロール | 群像劇の濃度確保と章ごとの主役交代 |
| カナイ区の都市伝承 | 通底するメタ謎 | 個別事件を「都市の秘密への踏み台」に変換 |
| 雨という環境設定 | 視覚・聴覚の常時演出 | ダークファンタジー基調の固定化と没入 |

物語の進行と推理アクションが「読み → 操作 → 読み」のサンドイッチで回るため、純文芸 ADV が陥りがちな中だるみを避けている。

## バズ要因 / 人気の根拠

- ダンガンロンパシリーズの中核スタッフ（小高和剛のシナリオ、小松崎類のキャラデザ、高田雅史の音楽）が TooKyo Games で再集結した「精神的続編」として、シリーズ未完を惜しんだファン層が初動を押し上げた
- 2023 年に Switch 独占でリリースされ、家庭用 ADV の話題作が少なかったタイミングと噛み合った
- 2024 年に Plus 版で PS5 / PC / Xbox Series X/S にマルチ展開し、Switch 独占を理由に保留していた海外コアファン層が合流した
- ダークファンタジー×ミステリーという比較的薄いジャンル交差で、配信向けに映える謎迷宮シーンとシーリのキャラクター性が SNS・実況シーンで切り取られた

## 企画への示唆

- 既存 IP の中核スタッフを別レーベルで再結集すると、IP 権利を持たなくても「精神的続編」ブランドが立ち上がる
- 純ノベル ADV に章クライマックスの操作儀式を挟むと、読書テンポを切らずに快感ピークを生成できる
- 異界化された事件現場（謎迷宮）を導入すると、現実考証の制約を外して推理パズルの自由度を確保できる
- 雨のような単一の環境設定を全編固定すると、低コストで世界観の統一感とトーン管理が成立する
- 連作形式で「個別事件 → 都市の秘密」の二層構造を組むと、1 章で離脱しても次章への引きが残る
