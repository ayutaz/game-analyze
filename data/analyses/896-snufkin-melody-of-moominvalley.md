# スナフキン：ムーミン谷のメロディ

## なぜ EXP 主分類なのか

プレイヤーの主動機は「歩く・聴く・対話する」というフィールド体験そのものにある。スナフキンとしてムーミン谷を巡り、ハーモニカで自然の音と応答しながら、谷を「公園」に作り変えようとする park keeper を静かに退ける。物語の進行は明確に存在するが、消費されているのはカットシーンや選択肢ではなく、フィールドを移動しながら音色と景観を採取するその過程そのものである。

`social_axis: ソロ` は本作の設計と整合している。マルチプレイ要素はなく、谷の住人との一対一の対話、自然との対話が孤独な歩行体験として閉じている。`secondary: NAR` はトーベ・ヤンソン原作の世界観と park keeper を巡る筋立てとして副次的に効いており、物語は体験を意味づける額縁として機能するが、主導権は常にフィールド側にある。

## 体験の核 / 設計の発明

中核は「音楽 × 探索 × コージー」の三層で組み立てられている。Sigur Rós 書き下ろし楽曲が常時のサウンドスケープを担い、プレイヤーの行動（歩行、対話、ハーモニカ演奏）が環境音と即時に絡む。テンポは戦闘ゲームの 1/3 以下に抑えられ、失敗してもペナルティが小さい。

| レイヤー | 担当する刺激 | 主な行動 |
|---|---|---|
| サウンドスケープ | Sigur Rós 楽曲 + 環境音 | 歩く・立ち止まる |
| フィールド探索 | 北欧的な手描き風アート | ムーミン谷の各所を踏破 |
| 対話 / ミニゲーム | 原作キャラとの一対一 | 依頼・楽曲採取・解決 |
| 抵抗の構造 | park keeper の施設 | 非暴力的な妨害・撤去 |

前世代のコージーゲーム（A Short Hike, Cozy Grove, Spiritfarer）からの差分は二点ある。第一に、音楽が BGM ではなくゲーム機構の中心にある点。第二に、原作 IP の世界観を「破壊」ではなく「保存」する側として動かす倫理設計が、プレイヤーの行動原理を強化している。

## バズ要因 / 人気の根拠

- Sigur Rós の書き下ろし楽曲という、ゲーム外の音楽ファン層を強く引き込むキャスティング
- トーベ・ヤンソンのムーミン IP（特に北欧・日本・東アジアで世代横断的な認知）を、ノスタルジーに頼らず現代的なコージーゲーム文法で再解釈
- Steam で「圧倒的に好評」評価、PC / Switch / PS / Xbox / iOS / Android に加え Netflix 版・Apple Arcade 版を同時展開した多プラットフォーム同時露出
- 2024 年コージーゲーム文脈（Cozy Grove: Camp Spirit, Tales of the Shire 等）の中で、音楽性と IP の組み合わせで明確に差別化

## 企画への示唆

- 既存 IP を扱うなら、ノスタルジーではなく「現代的な遊びの文法（コージー、音楽連動）」に翻訳すると新規層に届く
- 音楽家を BGM 担当ではなくゲーム機構の共同設計者として迎えると、音そのものが UX の中心になる
- 主人公の行動原理を「破壊／戦闘」ではなく「保存／対話」に置くと、ストレスフリーな探索ゲームの倫理的説得力が増す
- マルチプラットフォーム同時配信（コンソール + モバイル + サブスク）は、コージーゲームのターゲット（カジュアル成人層）の生活時間に侵入するのに有効
- 短時間プレイで完走可能な尺（数時間〜10 時間程度）に絞ると、忙しい大人ターゲットの完走率と口コミが両立する

## 比較で見るべきタイトル

- A Short Hike（2019, adamgryu）— ソロ探索コージーの祖型
- Cozy Grove / Cozy Grove: Camp Spirit — IP なしの純コージー設計
- Spiritfarer（2020, Thunder Lotus）— 物語を額縁にした EXP / NAR ハイブリッド
- Tchia（2023, Awaceb）— 音楽 + 自然 + 文化 IP の組み合わせ事例
- Tales of the Shire（2024, Wētā Workshop）— 既存大型 IP のコージー翻訳事例
