# ボタニー・マナー / Botany Manor

## なぜ EXP 主分類なのか

プレイヤーの主動機は「読み解く」体験そのものにある。1890 年の英国植物学者アラベラ・グリーンとなり、屋敷に散らばる手紙・絵画・標本ノートから「この植物は満月の光が必要」「あの種子は塩水に浸けると発芽する」といった条件を組み立て、各部屋の枯れかけた植物を蘇らせていく。動詞は「集める」ではなく「推理して実行する」で、コレクション欲よりも仮説検証のプロセス自体が報酬になっている。

`social_axis` はソロで、第三者との競争や協力は完全に排除されている。`concept` が示す通り暴力ゼロ・タイマーゼロ・失敗ペナルティゼロのコージー設計であり、`target` の Unpacking / A Short Hike 系プレイヤー像と合致する。`secondary` に NAR が入っているのは、女性研究者が論文発表を拒まれ続けた 19 世紀末の科学コミュニティという背景物語が、パズル進行の動機付けとして埋め込まれているため。物語は消費対象ではなく、パズルを解く文脈の壁紙として機能する。

## 体験の核 / 設計の発明

中核メカニクスは「ヒント収集 → 条件推定 → 環境セットアップ → 開花観察」の 4 ステップが部屋単位で完結するミニマル構造。Myst 系一人称パズルの語彙を借りつつ、解錠や暗号ではなく「植物学的な栽培条件」を解として要求する点が前世代との差分となる。

| レイヤー | 具体内容 | 設計上の効果 |
|---|---|---|
| 探索 | 屋敷の部屋を巡回、本・手紙・絵を拾う | 受動的読書ではなく能動的な情報統合を強制 |
| 推論 | pH / 光量 / 温度 / 水質などの条件を組合せ | 「正解は一つ」のロジック型と「文脈読解」型を混合 |
| 実行 | プランターに種を置き条件を整える | 手順ミスはやり直しコストゼロ＝試行回数を増やす設計 |
| 観察 | 花が開きジャーナルに標本として記録 | 完了報酬を視覚＋ドキュメント化で二重化 |

ソロ軸の効果としては、SNS で攻略動画を見ても自分の頭で組み立てる工程が圧縮されない点が大きい。RTA や PvP に最適化されない代わりに、配信者と視聴者が同じ情報量で謎に向き合える構図が生まれている。

## バズ要因 / 人気の根拠

- Day-1 Xbox Game Pass 投入により初週リーチが買い切り単独より一段高く、Whitethorn Games の発売モデルとして定着している配給戦略をそのまま踏襲した
- Netflix Games 経由のモバイル無償提供と Epic Games Store の無料配布で、コア PC ゲーマー以外の層にもタイトル名が浸透した
- 暴力描写ゼロ・短時間（4〜6 時間）クリアの設計が、配信者の単発実況やコージーゲーム特集記事と親和性が高かった
- 「女性科学者の追体験」というテーマがレビュー媒体・教育系メディアで取り上げられ、純粋なゲーム文脈の外側に話題が伸びた

## 企画への示唆

- パズルの解を「文化的・学術的知識」に紐付けると、攻略情報の即時拡散性が下がり、プレイヤー個人の体験密度が保たれる
- 失敗ペナルティをゼロにすると、試行回数が増えてプレイヤーの仮説形成サイクルが回りやすくなる（コージージャンルの基礎設計）
- 屋敷という閉じた空間を「部屋＝1 パズル単位」で章割りすると、短セッションプレイと長時間プレイの両方を同じ構造で吸収できる
- Day-1 サブスク + モバイル別配信 + 後追い無料配布を多段で組むと、買い切り小規模インディーでも 3 回バズの機会を作れる
- 主題（女性研究者の科学史）を UI に押し付けず、手紙・絵画のフレーバーに溶かすと、説教臭さを避けつつメッセージが伝わる

## 比較で見るべきタイトル

- Strange Horticulture（植物同定 × オカルト、推理を解の中核に据えた先行例）
- The Witness（一人称環境パズル、ヒントを世界に埋め込む手法）
- Unpacking（暴力ゼロ・ソロ・物語をオブジェクトに溶かす設計）
- A Short Hike（短編コージーの完成形、target 層が重なる）
- Return of the Obra Dinn（演繹推理＋ジャーナル記入による完了感の二重化）
