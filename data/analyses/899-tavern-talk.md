# タバーン・トーク (Tavern Talk)

## なぜ NAR 主分類なのか

プレイヤーの主動機は「酒場に立ち寄る冒険者たちの物語を聞き、ドリンクを介して彼らのクエストの結末を読む」ことにある。動詞でいえば**読む**であり、戦うでも集めるでもない。

concept にある「ドリンクを調合し冒険者たちの相談に乗ることで彼らのクエストの結末を左右する」は、調合行為そのものを楽しむ EXP 軸より、調合という選択肢が**他キャラの運命分岐に接続される**点に重心がある。target が「D&D・クリティカルロールに親しむ TRPG 文化圏」と「Coffee Talk 系の会話シム愛好層」の交差として定義されているのも、両者を貫く共通項が「他人の物語に立ち会う快楽」だからだ。プレイヤー自身は店主＝聞き役に固定され、舞台の中央には常に冒険者たちのクエスト譚が置かれる。

副次的に EXP が効くのは、ドリンク調合 UI が会話と独立したミニループとして成立しているため。ただしこの調合自体に成績や巧拙の評価軸はなく、あくまで「どの効能を渡すか」という**物語選択の手触りを物体化する装置**として機能する。social_axis は未設定（シングルプレイ・非対戦）で、観戦も対戦も介在しない閉じた一人称物語体験である。

## 物語の駆動構造

Coffee Talk 系コージー VN の構造を踏襲しつつ、TRPG 的「クエスト発注主」の視点に置き換えたのが本作の駆動装置。物語を進める力学は以下のように層化されている。

| 層 | 仕組み | 物語駆動への寄与 |
|---|---|---|
| 来客レイヤー | 固有 NPC 冒険者が日替わりで来店、TRPG 種族・職業のステレオタイプを背負って入店 | 「次は誰が来るか」を期待装置にし、来店自体が章立てになる |
| 対話レイヤー | 噂話・悩み・依頼を聞き出す分岐ダイアログ | TRPG 卓の DM が PC の身の上を聞き出す手触りを再現 |
| 調合レイヤー | 効能（勇気・癒し・記憶喚起など）別のドリンクをレシピで作成 | 「どの効能を渡すか」が他キャラのクエスト判定に直結 |
| クエスト帰還レイヤー | 冒険者が後日談を持って再来店、結末が変化 | 選択の帰結を**時間差で**回収する遅延報酬構造 |
| メタ・カレンダー | 営業日の積み重ねで複数キャラの線が並走 | 群像劇として複線進行、再読価値を担保 |

肝は、調合という「物質的な選択肢」がテキスト分岐の代替として置かれている点である。Coffee Talk が「ラテアートで気持ちを伝える」ことで会話の温度を可視化したのに対し、本作は「ポーションの効能が他者の冒険結果に作用する」という TRPG の DM 役割を商業簡略化している。

## バズ要因 / 人気の根拠

- popularity に記された通り、**TRPG 文化圏とコージーゲーマーという二つのニッチコミュニティの交差点**に作品を位置取りした。どちらも単独では小規模だが、重なる層は欧米英語圏で活発な配信文化を持つ。
- D&D 第 5 版および Critical Role 由来のファンタジー世界観リテラシーが 2020 年代前半に欧米一般層まで広がったことで、「酒場の店主視点」というメタ的設定が説明不要で受け入れられる土壌が整っていた。
- Coffee Talk（2020）/ Necrobarista / Va-11 Hall-A 系の「カウンター越し会話シム」ジャンルが既に成立していたため、**ジャンル文法の理解コストが低く**、レビュアー側もカテゴリ評価軸を持っていた。
- 売上は非公開だが、欧米メディアレビュー・小〜中規模配信者からの取り上げが先行し、Switch 同時展開でコージーゲーム棚に着地した点が中長期セールスに寄与している。

## 企画への示唆

- **二つの隣接ニッチの交差点に作品を置くと、単独ニッチより広い射程を取れる**（TRPG × コージー VN）。どちらにも片足を残すことで、片方の文脈で語れる枠が二倍になる。
- **プレイヤーを物語の中心ではなく聞き役に固定すると、群像劇の複線進行が成立する**。主人公の成長曲線を設計しなくて済むため、低予算で物語密度を稼げる。
- **テキスト選択肢を「物体（ドリンク／料理／道具）」に置換すると、選択の重みが手触りとして残る**。Coffee Talk のラテアートや本作のポーションのように、UI 上の選択を物質に翻訳する設計は記憶定着率を上げる。
- **選択の帰結を即時ではなく時間差で回収すると、来店ループ自体に期待値が宿る**。「次に彼が来店するとき、結末が判明する」構造は短時間プレイの動機を補強する。
- **既存ジャンル文法（ここではコージー会話シム）に忠実であるほど、レビュアー側の評価軸が安定し露出が伸びる**。発明よりも文法遵守＋舞台差し替えで勝てる領域がある。

## 比較で見るべきタイトル

- Coffee Talk（2020）— カウンター越し会話シムの直系祖、ラテアート＝選択肢の物体化
- Va-11 Hall-A（2016）— サイバーパンクバーテンダー、調合と会話の接続原型
- Necrobarista（2020）— 死後の世界カフェ、コージー VN の隣接事例
- Disco Elysium（2019）— TRPG リテラシーを前提にした VN 設計の参照点
- Wildermyth（2021）— 同じく群像 TRPG 文法を採用するインディー、進行構造比較に有用
