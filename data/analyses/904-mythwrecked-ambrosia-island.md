## なぜ EXP 主分類なのか

プレイヤーの主動機は「戦わずに島を歩き、記憶喪失の神々と話しトリンケットを渡して関係を築く」という探索＋対話の体験そのものにある。concept にある「戦闘なしの探索アドベンチャー」、target に並ぶ『A Short Hike』『Spiritfarer』『Tchia』はいずれも EXP（ソロ探索）系の cozy 文脈で、プレイヤーは謎解きの結末よりも「散歩と会話のリズム」を味わいに来る。social_axis が「ソロ」なのは、対戦も協力もなく、神々という NPC との一対一の関係構築だけで他者性が成立しているため。

secondary に NAR が入るのは、記憶喪失の神々の正体と島の謎が一本の筋として収束する 9–12 時間の物語駆動が確かに存在するからで、ただし章立てやマルチエンディングで「読ませる」設計ではなく、探索ループの副産物として物語が立ち上がる構造になっている。

## 体験の核 / 設計の発明

中核は「島の散策 → 神を見つける → トリンケット（神話的小物）を集めて手渡す → 関係値が上がり記憶が戻る → 新しい場所と神への手がかりが開く」というループ。戦闘を取り除き、報酬を「関係性の進展」と「神話的小物の意味付け」に置き換えている点が、cozy / 探索ジャンルの中での差分。

| 設計要素 | 機能 | 効果 |
|---|---|---|
| 戦闘の不在 | 失敗で進行が止まる場面を排除 | 9–12 時間の短尺で完走率を担保 |
| トリンケット収集 | 探索の即時報酬 | 散策に「集める」動機を重ねる |
| 神々との対話 | 関係値が記憶喪失の解除条件 | NPC が「鍵」ではなく「友達」になる |
| ギリシャ神話モチーフ | 既知 IP の認知を借用 | 神の名前と属性で世界観の説明コストを削減 |

ソロ social_axis だが、神々という固定 NPC ロスターが「他者がいる感覚」を肩代わりしているのが構造上の発明で、観光ゲームというよりはキャラ友情シミュレーターに近い手触りを作っている。

## バズ要因 / 人気の根拠

- 開発元 Polygon Treehouse は前作『RÖKI』で BAFTA ノミネート（Family / British Game）を獲得済みで、cozy / 北欧神話系の手描き美術に対する信頼が下敷きになっている
- パブリッシャー Whitethorn Games は cozy ジャンル特化の流通網を持ち、2024年12月のマルチプラットフォーム同時配信（PC / Switch / PS4 / PS5 / Xbox One / Xbox Series X/S）で導線を広く張った
- Xbox Game Pass への day-one 投入により、買い切り判断を留保したライトユーザーへもリーチ
- 9–12 時間でクリアできる短尺＋戦闘なしという仕様が、年末年始の cozy ゲーマー需要と Greek mythology IP 復権（『Hades』系）の余波に乗った

## 企画への示唆

- 戦闘を抜くと進行リスクが消える → 完走率が上がり、短尺タイトルの「最後まで遊んでもらう」KPI が伸びる
- 報酬を「アイテム」ではなく「NPC との関係値」に置き換える → 収集動機と物語動機を 1 つのループに畳める
- 既存神話 IP を借りる → キャラ設定の説明文を削れて、9–12 時間尺でもキャラ数を確保できる
- ソロ social_axis でも固定 NPC ロスターを置く → 「他者がいる」感覚は対戦・協力なしでも作れる
- cozy 特化パブリッシャー × Game Pass の組み合わせ → 戦闘なしタイトルの初動を「ジャンル文脈」で押し上げられる

## 比較で見るべきタイトル

- 『A Short Hike』（2019）— ソロ探索 + NPC 会話 + 短尺の原型
- 『Spiritfarer』（2020）— NPC との関係性が進行の鍵になる cozy 設計
- 『RÖKI』（2020）— 同スタジオの前作、北欧神話 × 手描き美術
- 『Tchia』（2023）— 短尺の島探索アドベンチャー、文化モチーフの借用
- 『Hades』（2020）— ギリシャ神話 NPC との関係値設計の参照点（戦闘ありだが対比軸）
