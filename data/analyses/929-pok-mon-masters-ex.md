# ポケモンマスターズ EX

## なぜ REW 主分類なのか

プレイヤーの主動機は「歴代トレーナー＋ポケモンの組（バディーズ）をフェス限定ガチャで集め、星上げ・才能開花で育てる」こと。3vs3 のリアルタイムバトルや本編シナリオを再演する物語パートも揃っているが、それらは「集めたバディーズの活躍する場」として機能しており、戦闘体験そのものや物語消費が単独で動機を成立させているとは言いづらい。

concept にある「フェス限定バディーズ＋週次イベント」、target に明記された「歴代トレーナーへの思い入れ」「キャラ収集型ガチャ RPG を長期で遊ぶ層」という記述は、典型的な日本型キャラガチャ運営の動機設計そのもので、secondary が NAR / EXP の順に控えるのは、本編トレーナー再登場という IP の語り直しと、3vs3 リアルタイムの操作性が「コレクションを引き立てる装置」として副次的に効いているからだ。social_axis は非同期（イベントランキング、フレンドの応援バディーズ参照、共闘イベントの貢献度集計）で、対人 PvP に踏み込まず、ソロ＋ゆるい比較に閉じている点も REW の典型である。

## 報酬ループの組成

| 層 | 内容 | リテンションへの寄与 |
|---|---|---|
| ガチャ層 | バディーズ（トレーナー×ポケモン）ガチャ。フェス限・マスターフェア限定枠で星 6 EX 進化、専用技持ち | 新規キャラ実装ごとに引く動機が再生成 |
| 育成層 | 才能開花、星上げ素材、技マシン、サポート/アタッカー/テクニカルのロール最適化 | 1 バディーズで数十時間の周回が発生 |
| イベント層 | 共闘イベント（協力周回）、伝説バトル、ストーリーイベント、エクストラランク | 週次/月次の出現周期で出戻りを誘導 |
| IP 層 | 歴代主人公・四天王・ジムリーダー・チャンピオン・悪の組織ボスを継続実装 | 思い入れのあるトレーナー登場で復帰トリガー |
| 比較層 | バトルヴィラのソロタワー、共闘の貢献度ランキング | 非同期に「上手い人と比べる」場を提供 |

ガチャでキャラを引いた瞬間に終わらず、そこから星上げ→才能開花→ロール固有の周回素材集めへと長い育成パイプラインが伸びる構造で、1 体の最終形に到達するまでに複数イベントの素材を跨ぐ。これが「フェス限実装 → 復帰 → 既存バディーズの育成見直し → 周回」という循環を生む。

## バズ要因 / 人気の根拠

- 配信 4 日で全世界 1,000 万 DL・1,000 万ドルを達成したスタートダッシュ型のヒット（Sensor Tower）。
- 初年度収益約 7,500 万ドル、初年度 DL 2,270 万という規模で、ポケモン IP のキャラ収集需要の厚さを実証。
- 2023 年 4 月時点で累計 5,000 万 DL を突破し、日本国内では 2022 年 35.97 億円→2023 年 27.4 億円（w3g 推計）と緩やかな減衰を見せつつも長期運営に成功。
- 歴代トレーナー（カスミ、シロナ、N、ククイ博士、ライバル系など）を継続的にフェス限で実装し、本編アニバーサリーと連動した IP 露出で復帰サイクルを作っている。

## 企画への示唆

- IP の語り直しをガチャ単位に細分化すると、本編クリア後の長尾需要を収益化できる（歴代トレーナーを月次新キャラ枠として再投入）。
- キャラ＝トレーナー単体ではなく「トレーナー×ポケモン」のペア単位にすると、同一トレーナーの別衣装/別ポケモン版が自然な追加実装になりライフタイムが伸びる。
- 3 ロール（アタッカー／サポート／テクニカル）に分けて編成を 3 枠固定化すると、同ロール内で新規キャラが旧キャラを陳腐化させない設計が組める。
- PvP を入れず非同期ランキングと共闘イベントに閉じると、IP の温度感（ポケモン世界の協調的トーン）を壊さずに長期運営できる。
- フェス限と通常ピックアップを明確に分離し、フェス限のみ EX 進化対応とすると「引くべきガチャ」のシグナルが明確化し、ガチャ間の自己カニバリを抑制できる。

## 比較で見るべきタイトル

- Fire Emblem Heroes — 同じく歴代シリーズキャラを月次フェス限で再投入し、ロール制で長期運営する任天堂 IP の隣接事例。
- 遊戯王 デュエルリンクス — IP × カード収集 × 非同期＋簡略 PvP という近い構造を、対人寄りに振った比較対象。
- ドラゴンクエストウォーク／ドラクエタクト — 国民的 RPG IP のキャラ＆装備ガチャを別レイヤー（位置情報／SRPG）で展開した参考例。
- Pokémon GO — 同じポケモン IP だが REW の動機を「位置情報による収集体験」に寄せた対照例。
