# 妖怪ウォッチ ぷにぷに

## なぜ REW 主分類なのか

プレイヤーの主動機は「コラボで現れる限定妖怪を集める」ことであり、パズルそのものは収集と育成を回すための作業として機能している。concept にある通り盤面をなぞるマッチ 3 はあくまで召喚・育成のリソース生成手段で、target も「コラボ重視のカジュアル課金層」を明示している。10 年運営の長寿タイトルが新規 DL とセルランを維持できているのは、毎月差し替わる外部 IP コラボの限定キャラを取り逃さないように回す、というガチャ／イベント収集ループが軸になっているからである。

secondary に EXP が入るのは、スワイプで同色ぷにを長く繋げてコンボを伸ばすパズル操作自体に手応えがあるためで、social_axis「非同期」はランキングイベント・フレンド助っ人・コラボ復刻のお祭り感がプレイヤー同士を緩く結びつける構造を指している。直接的な対人戦はなく、他者は「同じ期間に同じ妖怪を狙っている存在」として可視化される。

## 報酬ループの組成

| 層 | 要素 | リテンション効果 |
|---|---|---|
| 短期 (1 プレイ) | スワイプでぷにを繋げる → 妖怪召喚 → 敵撃破 | パズル操作の即時フィードバック |
| 中期 (週次) | コラボ期間限定ガチャ・期間限定ステージでメダル収集 | 「今しか取れない」希少性で課金圧 |
| 長期 (年次) | 周年・正月・夏・ハロウィン等の大型コラボ、復刻、進化合成 | IP コラボの相手切替で飽きを延命 |
| メタ (10 年) | 累計妖怪コレクション、シリーズ既プレイヤーのノスタルジー | IP 自体への愛着が乗り換えコストになる |

中核は「限定妖怪を逃すと次の復刻まで取り戻せない」という時間制ガチャと、外部 IP（他作品キャラを「ぷに」化して登場させる）との常時コラボの組み合わせである。コラボ相手を切り替えることで毎月入口を更新でき、休眠ユーザーを「今月のコラボだけ」で呼び戻せる設計になっている。

## バズ要因 / 人気の根拠

- 妖怪ウォッチ本編 IP のピーク（3DS 期）に育ったプレイヤーがそのままモバイルへ移行し、10 年以上継続運営されている長寿タイトルである
- 2023 年通年売上 58.67 億円（前年比 115%）、世界累計収益 6 億ドル超、累計 2,900 万 DL という規模を維持
- 2023 年の 10 周年記念施策では前年同期比で収益 3 倍・DL 8 倍を記録し、長寿タイトルでも記念施策で再活性化できる事例となった
- コラボイベントを軸にセルラン上位へ周期的に返り咲く運用が定着しており、外部 IP を「ぷに」化する変換が定番化している

## 企画への示唆

- パズルを「報酬を引き出すスロット」として薄く設計すると、IP コラボの収集圧と相性が良い（操作が軽いほどコラボ頻度を上げられる）
- 外部 IP を自タイトルの絵柄に変換するルール（ぷに化）を確立すると、コラボ相手の幅が広がりつつ世界観が崩れない
- 10 周年で前年比収益 3 倍を出した実績は、長寿サービスでも「記念タイミング × 大型復刻」で再活性化できることを示している
- 直接対戦を入れず非同期ランキング止まりにすると、操作が軽いパズルでも IP ファン層の離脱を抑えられる
- 親 IP（妖怪ウォッチ本編）のピークが過ぎた後にモバイル側が IP の主接点になるパターンは、IP ホルダーにとって長期収益化のテンプレになる
