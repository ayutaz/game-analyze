# スターオーシャン:アナムネシス

## なぜ REW 主分類なのか

プレイヤーの主動機は「シリーズ歴代キャラをガチャで集めて編成し、育てる」ことに置かれている。本作は tri-Ace の本編アクションRPG（キャンセルボーナス、ステップ回避）をモバイルに移植する形を取りながら、コンテンツ供給の中心は20周年記念の「歴代オールスター」を切り出した期間限定召喚にあった。プレイヤーは新ステージを攻略するために既存編成を磨くのではなく、新キャラ実装ごとに編成を組み替え、属性別の星4を回収していく。これは典型的な収集×育成ループであり、social_axis が「協力」でありつつも、協力先で必要になるのは結局「強いキャラとビルド」である点で、報酬獲得が動機の上位に立つ。

secondary に EXP（アクションバトルの操作快感）と NAR（本編世界観・キャラとの再会）が入るのは妥当で、アクション継承が SO ファン残留の前提、IP 起点のキャラ愛着がガチャ単価を支える構造になっている。

## 報酬ループの組成

オールスター IP × 協力 PvE × ガチャの三層構造でリテンションを設計している。

| 層 | 内容 | 主動機 |
|---|---|---|
| 召喚（ガチャ） | 歴代主人公・ヒロイン・ボスのキャラ・武器ガチャ。属性別・期間限定で実装 | コレクション充足、最強編成更新 |
| 育成 | レベル / 限界突破 / 武器強化 / 因子 | キャラを「使える状態」にする労力 |
| 協力ミッション | 最大4人マルチ PvE、属性相性で編成最適化 | 育てた編成の発表会、素材回収 |
| イベント / コラボ | 期間限定ボス、外部 IP 衣装、復刻 | 取り逃し防止プレッシャー |

新キャラの実装テンポが SO 本編のキャストを「枯渇させる」までは収益を駆動できたが、IP 枠を消費した後の継続的なキャラ供給が課題になり、グローバル版（2019）→日本版（2021）の順でサービスを閉じている。報酬層は深いが、IP 在庫の有限性が長期運営の天井になった事例として読める。

## バズ要因 / 人気の根拠

- 20周年記念タイトルとして、SO1〜SO5 のキャラを一堂に集めた「再会装置」を提示し、事前登録53万人を獲得（JSON 記載値）
- 配信半年で500万 DL に到達（2017）。tri-Ace 監修のアクション継承がモバイルアクション RPG としての品質保証になった
- Sensor Tower 集計で全世界 Google Play 単体で1000万ドル超収益（2018 早期）。iOS 込みではさらに上振れ
- 一方でグローバル版は2019年11月、日本版は2021年6月にサービス終了。IP 内キャラ在庫の消費スピードと、本編側の SO6（2023）に向けたリソース再配分が要因と読める

## 企画への示唆

- 周年記念で「歴代キャラ全部出す」をやると、序盤の課金圧は最大化するが、IP 在庫を先食いするので2〜3年後の供給設計を最初に決めておく必要がある
- 本編譲りのアクション（キャンセル、回避）をモバイル UI に落とすと、IP ファンの操作快感への期待値を満たせるが、片手操作の限界からエンドゲームではキャラ性能ガチャに帰着するため、操作差で勝てる PvP は設計から外した方が筋がよい
- 協力 PvE を social_axis に据えると、ガチャの「最強編成」が他者に見せる動機になり、ARPU を引き上げられる
- ナンバリングタイトルのモバイル分家は、本編新作の発表タイミングと寿命が衝突しやすい。本編 SO6 開発時期と重なるとリソースが本編に寄せられる前提で、サービス終了の出口設計まで企画段階で描いておく
- サービス終了後にオフライン版を提供するのはファン離反緩和に有効。IP の継続的好意度を守る後始末として参照価値がある

## 比較で見るべきタイトル

- 白猫プロジェクト（コロプラ） — アクション RPG × ガチャ × マルチ協力の先行モデル
- ロマンシング サガ リ・ユニバース — 周年・歴代キャラ集結型モバイル RPG の長期運営成功例
- FINAL FANTASY ブレイブエクスヴィアス — 同じスクエニ配給、IP 横断キャラ召喚型ガチャの代表
- テイルズ オブ ザ レイズ — シリーズ歴代キャラ集結型でサービス終了に至った同類事例
