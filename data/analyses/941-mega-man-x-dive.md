# ロックマンX DiVE

## なぜ REW 主分類なのか

このタイトルでプレイヤーが最も時間を投下する動詞は「集める」である。歴代ロックマンシリーズ 100 体超を横断するハンタープログラム（キャラ）ガチャと武器ガチャを 2 軸で回し、900 以上のステージを周回して育成素材を抽出する構造になっており、横スクロール／全方向シュー視点のアクション操作それ自体は「ガチャ資産を可視化するためのテスト場」として機能する。

concept に示された「100 体超キャラ収集・900 ステージ攻略」、target に示された「歴代ボスやヒロインの if コラボを愛でるノスタルジー層」という記述は、IP の歴史的厚みをガチャ in-pool に換金する設計を明示している。social_axis は「非同期」で、PvP ランキングや協力レイドはあるが、主動機は他者と即時に対戦することではなくランキング比較とフレンド助っ人で回るリテンション構造に寄っている。

secondary に EXP が入るのは、ロックマン X 由来のジャンプ・ダッシュ・チャージショットのアクション手触りが「ガチャを引き続ける理由」を支える副次レイヤとして効いているためで、純粋なオート戦闘ガチャとは別線にいる。

## 報酬ループの組成

| レイヤ | 中身 | リテンション効果 |
|---|---|---|
| キャラガチャ | 歴代 100 体超、ヒロイン if 衣装、S クラス「オメガ」等の節目排出 | IP 横断による掘り起こし需要 |
| 武器ガチャ | キャラと別線。組み合わせで火力が伸びる | 課金二重線でガチャ回転を二倍化 |
| ステージ周回 | 900+ ステージ、素材ドロップ | 育成上限解放のための毎日タスク |
| PvP / 協力レイド | 非同期ランキング、フレンド助っ人 | 「育てた資産の見せ場」を供給 |
| 期間限定コラボ | 歴代ボス・ヒロイン if | 旧作ファンの再訪トリガー |

特徴は「キャラ × 武器」の二重ガチャで一発の引きあたり期待値を分散させていること、そして 900 ステージという尋常でない量のステージ供給によって、ガチャで得たキャラを試す場と素材ファームの場の両方を確保している点にある。S クラス「オメガ」追加で売上ランキングが再びトップ 30 入りした事実は、節目キャラのピックアップが集客の主導因であることを示している。

## バズ要因 / 人気の根拠

- ロックマン X シリーズ初の本格モバイル展開という IP 文脈で、配信前から事前登録 10 万超を集めた。
- 日本配信直後に App Store 売上ランキングが 40 位 → 28 位まで上昇し、ローンチピークを可視化（gamebiz 2020）。
- S クラスキャラ「オメガ」追加時にも売上トップ 30 入りし、節目キャラ実装での再加熱を実証。
- カプコンモバイル売上を 2021 年度に約 47 億円（US$45M 相当）押し上げたと報告されており、IP モバイル展開としては一定の収益貢献。
- 一方で約 3 年でオンライン版終了（2023 年 9 月アジア・日本、2024 年 7 月北米欧州）、買い切り型オフライン版（Steam/iOS/Android）へ移行した経緯はガチャ運営の長期持続性に課題を残す。

## 企画への示唆

- レガシー IP × ガチャでは「歴代キャラ in-pool」を最大の差別化資産にすると、ファンの過去資産が課金動機に直結する。
- キャラガチャと武器ガチャを別線で持つと、一発の天井で課金を止められず長期で回し続ける動機を作れる。
- ステージ数を 900+ まで積むと、ガチャ報酬の試運転場と素材ファームを同一画面で同居させられる。
- 節目キャラ（S クラス、コラボ）を 3 〜 6 か月周期で打つと、ランキング再加熱が観測可能な KPI として残る。
- IP モバイルで 3 年でクローズ → 買い切り版移行というルートは、運営継続不能になった際の IP 棄損リスクを抑える出口として参考になる。
