# アイドルランドプリパラ

## なぜ REW 主分類なのか

プレイヤーの主動機は「推しのコーデを集めて自分のマイキャラを着飾る」こと、つまり**集める**である。concept が明示する通り中核ループは「コーデガチャ＋育成イベント周回」であり、ライブ演出やキャラとの会話は集めた装備を可視化するための舞台装置として機能している。target にあるアーケード版プリパラ／プリチャン世代の旧ファンは、もともと筐体でカードを引き当てて衣装を組み替える体験を消費してきた層であり、その消費構造をスマホのガチャ＋クローゼットに転写したのが本作である。

secondary に EXP が入るのは、social_axis「非同期」によるテーマパーク内のすれ違い・他プレイヤーのマイキャラ観覧・コーデショーへの参加といった「他人に自分のコーデを見せる」体験が、収集の出口として機能しているためだ。ただし主動詞はあくまで「集めて着せ替える」であり、対戦や協力で勝つことが目的化していないので REW が主分類になる。

## 報酬ループの組成

アイドル運営型 REW は単純なガチャ → 育成では完結せず、複数のレイヤーが並走する。本作の構造は概ね以下に分解できる。

| レイヤー | 内容 | リテンションの効き方 |
|---|---|---|
| コーデガチャ | レアリティ付きの衣装一式（トップス・ボトムス・シューズ等を含むセット） | 旧 IP の象徴的衣装が周期的に復刻し、世代ノスタルジー需要を回す |
| マイキャラ育成 | プレイヤー自身のアバターをコーデで構成し、ライブで運用 | コーデ＝資産が「自分の分身」に紐づくため離脱コストが高い |
| イベント周回 | 期間限定ストーリー／キャラ覚醒素材／限定コーデ報酬 | 旧 IP キャラ復活イベントが「待望感」をピーク化させる装置 |
| 非同期ショー／観覧 | 他プレイヤーのマイキャラと擦れ違う「アイドルランド」内回遊 | コーデを「見せる」出口を作り、ガチャ動機を再強化 |
| 長期 IP 運営 | 2024 年マギシステムへの運営移管後の継続コラボ | 旧筐体のサ終で行き場を失った IP 資産を回収する受け皿 |

筐体プリパラがやっていた「カード排出→筐体内ファッションショー」を、ガチャ＋クローゼット＋非同期テーマパークに分解して再構築したのが設計上の特徴である。

## バズ要因 / 人気の根拠

popularity の「2 年以上の延期を経て 2023 年 8 月リリース」「リリース直後は入場制限」「2024 年 7 月マギシステム移管」を分解すると以下のようになる。

- 2021 年発表 → 2023 年 8 月 17 日リリース、約 2 年の延期がむしろ旧プリパラ／プリチャン勢の「待望感」を蓄積させた
- リリース直後はアクセス集中で**入場制限**がかかる事態が発生、初動の話題性を可視化した
- 音楽ゲーム無料 DL ランキングで平均 76 位／最高 63 位（APPLION・iDayo.jp 2024 集計）と、メガヒットではないが女児・旧ファン特化型として一定の継続的流入を確保
- 2024 年 7 月にタカラトミーアーツからマギシステムへ運営移管され、IP の長期運営フェーズへ移行（短期収益より資産延命の局面）

## 企画への示唆

- **筐体サ終 IP をスマホで延命する** → ファンが筐体カード資産を失った直後にスマホ版を出すと、ノスタルジー需要が「待望感」として圧縮されてリリース時に解放される
- **長期延期はネガティブだけではない** → 2 年の延期でも IP の核ファンに対しては期待値の蓄積として作用しうる（ただし新規層への訴求力は確実に減衰する点とのトレードオフ）
- **コーデを「資産」に昇格させる** → ガチャ排出物を消耗品ではなく自分のアバターに紐づくクローゼット資産にすると、離脱コストが上がる
- **非同期テーマパーク UI を採用する** → 他プレイヤーと「同時にいる必要がない擦れ違い空間」を作ると、コーデ＝集めた資産の出口になり再ガチャ動機を生む
- **運営移管を IP 戦略として設計に組み込む** → タカラトミーアーツ → マギシステム移管のように、初期パブリッシャーの体力が尽きる前に IP 専業の運営会社へ受け渡す前提でコスト構造を組むと、長期運営が成立しやすい
