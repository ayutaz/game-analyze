- 2026年6月に Claude Code の Workflow ツールで8つの並列リサーチエージェント（年別＋モバイル＋インディー）→ consolidate → 各候補に対する敵対的検証 → エンリッチの4フェーズで生成・検証した101件。

- 一部は2014–2019年発売だが、2020年以降も日本国内で継続的な人気・話題があったため含めた（雀魂、Identity V、グラブル、デレステ、シャニマス、刀剣乱舞、まほやく、アナデン、プリコネ、BLEACH Brave Souls、BanG Dream!、コンパス、荒野行動）。

### 追加分の出典・調査メモ

- 2026-06-24 に Claude Code の Workflow + deep-research ツールで **22 並列 discover エージェント**（Western CRPG / Grand Strategy & 4X / RTS / ARPG / MMO / FPS 競技 / FPS 単独 / シミュレーション / コロニーサバイバル / 古典イマーシブシム / タクティカル / VN・同人 / Roguelike & Deckbuilder / ホラー / Action Soulslike / Metroidvania / レーシング / ナラティブ / Open World / Asia PC / Puzzle・Card / クラシック PC）→ ローカル dedup → ヒューリスティック品質スコアの3フェーズで生成・採択した500件。
- discover フェーズで 1271 raw 候補を取得後、既存 303 件との重複を除外し 1140 ユニーク、quality score（売上推定・批評受賞シグナル等から算出）上位 500 を採用。
- Workflow は consolidate フェーズへ到達する前に western-crpg エージェントが 6 連続 stall でクラッシュしたため、verify+enrich は走らず **discover の raw 出力を直接構造化** している。`sales_world` の値はソースエージェントの推定がそのまま入っており、`Steam owners 1-2M` / `Metacritic 8X` / `Twitch peak XK` 等の表現ゆれが残る。
- `social_axis` はジャンル＋angle ベースのルール（CRPG=ソロ、MOBA=対戦、MMO=協力、サンドボックス=ソロまたは協力 等）で機械的に決定。個別タイトルの遊び方判定は未実施。
- 補強対象は Western CRPG（Mass Effect / Pillars / Pathfinder / Wasteland / KOTOR / Skyrim / Fallout）、Grand Strategy / 4X（Civilization / Crusader Kings / Stellaris / Total War）、RTS（StarCraft 2 / Age of Empires / CoH）、MMO（WoW / Lost Ark / ESO / Black Desert）、FPS（Counter-Strike 2 / Hunt: Showdown / Tarkov / Hell Let Loose）、ARPG（Path of Exile / Last Epoch / Grim Dawn）、シム（Cities Skylines / Factorio / MSFS / Football Manager）、サンドボックス（RimWorld / Kenshi / Rust / ARK / Project Zomboid）、古典 PC（Half-Life / Portal / Bioshock / Dishonored / Deus Ex / Hitman WOA / Outer Worlds）、VN・同人（Tsukihime Remake / Fate/stay night / Umineko / ATRI / DDLC）、レーシング（Forza Horizon / ACC / iRacing / F1）など、これまでカタログに不足していた PC 主軸ジャンル全般。
